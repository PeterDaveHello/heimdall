## 2023 年 6 月 30 日

### [最高法院推翻了大学招生中的平权行动](https://www.latimes.com/politics/story/2023-06-29/supreme-court-strikes-down-affirmative-action-in-college-admissions)

- 最高法院驳回了高校将种族作为录取决定因素的平权行动政策，裁定这些政策是非法的种族歧视，违反了宪法第 14 修正案。
- 首席大法官小约翰-G-罗伯茨表示，宪法禁止根据种族对人进行区别对待，应该根据学生的个人经历而不是根据种族来对待他们。
- 该裁决在加州的影响可能有限，因为该州已经禁止将种族作为公立大学的录取因素，但该裁决也将该禁令扩展到私立大学。

### [行业反应](http://news.ycombinator.com/item?id=36520658)

- 最高法院对大学录取中的平权行动的裁决引起了关于其在解决不平等问题上的有效性的辩论。
- 一些人认为，种族不应该是唯一的焦点，而应该考虑社会经济因素。
- 讨论强调了解决历史不公正问题的复杂性，理解歧视背景的重要性，以及改善弱势社区的教育和机会的必要性。

### [Valve 不愿意再发布有 AI 生成内容的游戏了？](https://old.reddit.com/r/aigamedev/comments/142j3yt/valve_is_not_willing_to_publish_games_with_ai/)

- 阀门公司不再愿意发布含有 AI 生成内容的游戏，理由是与知识产权有关的潜在法律问题。
- 这一决定是在一个开发者试图发布含有人工智能生成的资产的游戏，并收到来自 Valve 的消息，称该开发者没有必要的艺术资产的权利。
- 阀门公司的这一举动表明，他们对围绕人工智能生成内容的潜在版权侵权问题持谨慎态度，这可能对人工智能在游戏开发中的未来产生影响。

### [行业反应](http://news.ycombinator.com/item?id=36522665)

- 如果没有训练数据的所有权证明，Valve 将不再发布带有 AI 生成的内容的游戏。
- 该决定是为了避免侵犯版权的法律责任。
- 阀门公司的政策反映了他们对版权保护的关注，并提出了关于人类和机器生成的内容之间的区别问题。

### [Awk 编程语言，第二版](https://awk.dev/)

- 第二版 "AWK 编程语言 "即将发布，反映了自 1988 年第一版以来 Awk 语言和计算机世界的变化。
- 该书将包括历史文件、代码片段以及关于 Awk 和相关主题的文章，为读者提供更多有趣和有用的材料。
- Awk 有多种实现方式，读者可以在 GitHub 上访问 Awk 源代码，在 GNU 网站上访问 Gawk 发布，还可以找到 Arnold Robbins 编制的其他 Awk 实现方式的列表。

### [行业反应](http://news.ycombinator.com/item?id=36517525)

- 第二版 "Awk 编程语言 "正在发布，由 Brian Kernighan 完成了更新和重组。
- 该书的早期章节涉及实践探索性的数据处理，特别是 CSV 文件。
- 在 Gawk 和 awk 中会有一个新的"--csv "选项，允许正确的 CSV 输入模式，解决引号和多行字段的解析。

### [隐藏在 Python 标准库中的 CLI 工具](https://til.simonwillison.net/python/stdlib-cli-tools)

- Python 标准库包含隐藏的 CLI 工具，可以使用`python -m`命令从终端直接访问。

### [行业反应](http://news.ycombinator.com/item?id=36515531)

- Python 的标准库包含了一些隐藏的、不广为人知的 CLI 工具。
- 一个例子是 re.Scanner 工具，它是一个基于 regex 的标记器，在官方文档中缺失。
- 这些 CLI 工具提供了方便而强大的功能，可以用于各种任务，如漂亮地打印 JSON 或运行一个简单的 HTTP 服务器。

### [首次发现怪物引力波](https://www.nature.com/articles/d41586-023-02167-7)

- 研究人员利用脉冲星，即银河系附近的信标星，首次发现了怪物引力波。
- 这些波比 2015 年探测到的波要大得多，也长得多，波长可达几十光年。
- 这一发现可以为宇宙中成千上万对超大质量黑洞的存在及其合并提供证据，激光干涉仪空间天线(LISA)任务将对此进行进一步研究。

### [行业反应](http://news.ycombinator.com/item?id=36520925)

- 科学家们检测到了引力波，证实了爱因斯坦广义相对论的一个重要预测。
- 使用干涉仪对这些波的探测为黑洞和中子星的存在提供了强有力的证据。
- 这一突破使我们可以研究宇宙中的暴力和能量过程，影响我们对宇宙的理解。

### [据报道，阿斯巴甜甜味剂将被世卫组织宣布可能有致癌风险](https://www.theguardian.com/society/2023/jun/29/aspartame-artificial-sweetener-possible-cancer-risk-carcinogenic)

- 据报道，世界卫生组织的癌症研究部门正准备宣布阿斯巴甜(一种用于成千上万种产品的人工甜味剂)为 "可能对人类致癌"。
- 自 20 世纪 80 年代以来，阿斯巴甜已被广泛使用，并被审查了现有证据的监管机构授权在全球使用。
- 现有的证据对阿斯巴甜对癌症风险的潜在影响提出了疑问，但需要进一步的研究来做出确定的结论。

### [行业反应](http://news.ycombinator.com/item?id=36519942)

- 阿斯巴甜是一种被广泛消费的人工甜味剂，根据法国的一项研究，它可能被世界卫生组织列为可能的癌症风险。
- 专家们对阿斯巴甜的致癌潜力和该研究的结果存在争议。
- 正在进行的讨论强调了对阿斯巴甜和其他物质的风险水平进行明确的指导和沟通的重要性。

### [C-18 法案：谷歌因网络新闻法在加拿大删除新闻链接](https://www.ctvnews.ca/politics/google-set-to-remove-news-links-in-canada-meta-ends-contract-for-journalism-fellowship-as-bill-c-18-fallout-continues-1.6461557)

- 由于一项新的在线新闻法，即 C-18 法案，谷歌将删除加拿大的新闻链接。
- 谷歌此举是对该法律要求的回应，该法律将强制要求科技公司为其内容向新闻出版商付费。
- 谷歌删除新闻链接的做法引发了精通技术的人的兴趣和关注。

### [行业反应](http://news.ycombinator.com/item?id=36524662)

- 加拿大的 C-18 法案针对谷歌和 Meta，要求它们删除新闻链接并向新闻出版物支付版税。
- 批评者认为，该法律损害了互联网的开放性，干扰了信息的获取。
- 该法提出了关于 "新闻内容 "的定义、对加拿大新闻机构的影响、以及对他人作品进行货币化的道德影响等问题。

### [编程语言的超越](https://whileydave.com/2023/06/27/programming-languages-going-above-and-beyond/)

- 作者讨论了现有编程语言的局限性，并强调了编程语言的改进潜力。
- 作者介绍了 Dafny，一种提供高级静态检查能力的语言，允许在编译时验证后置条件。
- Dafny 能够开发出可验证为正确的函数，而不需要进行大量的测试，这使它成为软件开发中一个独特而强大的工具。

### [行业反应](http://news.ycombinator.com/item?id=36519497)

- Dafny 是一种支持形式验证的编程语言，可以证明有关程序的属性，提高软件的可靠性和正确性。
- 其他工具和语言如 Ada、SPARK 和 ACL2 也支持形式化验证，并在现实世界中得到应用。
- 关于函数式编程、静态类型化和编写健壮代码的实用性和好处的辩论。
