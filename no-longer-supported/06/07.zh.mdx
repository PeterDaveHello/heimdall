## 2023 年 6 月 7 日

### [Reddit 杀死第三方应用程序的计划引发了广泛的抗议](https://arstechnica.com/gadgets/2023/06/reddits-plan-to-kill-third-party-apps-sparks-widespread-protests/)

- Reddit 对第三方应用征收 API 费用的计划引发了用户的抗议和反感，数百个子版块计划在 6 月 12 日进入黑暗期，持续 48 小时。- 第三方应用程序面临着高额的 API 定价，许多开发者说这远远高于其他网站的收费，可能会迫使他们关闭。- Reddit 的官方应用被普遍认为不如第三方选项，而且 API 条款还要求在第三方应用中屏蔽广告，并排除成人内容，这些都是该网站的主要吸引力。

### [行业反应](http://news.ycombinator.com/item?id=36210805)

- Reddit 被指责人为地提高子版块的本地化版本的参与度，批评者认为这符合其战略，即引导用户使用官方应用程序，并在上市前抽出感知价值。- 用户建议实施类似于 Discord 和 Twitch 的资助模式，允许用户为子版块的升级付费，并将部分资金用于资助审核。- 帖子讨论了 Reddit 的移动应用和网站，用户对没有优先修复问题表示不满，并对公司的优先事项提出了批评。

### [GGML - AI at the Edge](http://ggml.ai)

- GGML 是一个用于机器学习的张量库，能够在普通硬件上实现大型模型和高性能，并针对苹果硅架构进行了优化。- 它支持 16 位浮点和整数量化，有内置的优化算法，并通过 WebAssembly 和 WASM SIMD 提供网络支持。- 它包括两个项目，用于语音到文本的 whisper.cpp 和用于语言建模推理的 llama.cpp，这两个项目都在苹果硅硬件上展示了高效的性能。

### [行业反应](http://news.ycombinator.com/item?id=36215651)

- GGML 是一个在 CPU 上运行机器学习模型的库，已经得到了风险投资公司的大量资金支持，并允许在商品硬件上进行高效计算。- 人们对开放的核心商业模式和公司与社区之间潜在的利益不一致表示关切。- Willow 和 Home Assistant 是 GGML 的替代方案，用于本地命令检测和语音识别，需要进行基准测试以比较不同的选择。

### [OpenGL 3.1 on Asahi Linux](https://asahilinux.org/2023/06/opengl-3-1-on-asahi-linux/)

- Asahi Linux 已经将其图形驱动从 OpenGL 2.1 升级到 OpenGL 3.1，并将 OpenGL ES 2.0 的支持增加到 OpenGL ES 3.0。这意味着更好的渲染质量和更多的功能应用。- 然而，这些驱动程序是试验性的，还不符合 OpenGL 或 OpenGL ES 规范，因此用户可能会遇到错误。- 升级的亮点包括多个渲染目标、多采样、转换反馈和纹理缓冲器对象。具体来说，多重采样是一种有效的方法，可以减少由于在分辨率不足的情况下进行渲染而导致的对角线边缘锯齿状的混杂。它为每个像素创建多个样本，并对它们进行平均，以创建一个规则的图像。然而，苹果公司的 AGX GPU 需要一种独特的方法来进行采样着色，当混合和多采样被启用但采样着色没有被启用时，混合的性能会退步。

### [行业反应](http://news.ycombinator.com/item?id=36212674)

- Asahi Linux 发布了一个具有各种功能的更新，包括对 M2 Pro/Max/Ultra SoC 的基本支持和一种新的基于固件的电池充电控制方法。- 苹果硬件的用户对在其上运行 Linux 的自由表示感谢，但有些人对 Asahi 的可持续性以及是否值得为运行它而购买一台 Mac 进行辩论。- Asahi Linux 已经取得了重大进展，但仍有一些限制和与 GPU 驱动的不兼容，会影响性能。

### [英国法律承认八爪鱼、螃蟹和龙虾为有生命的生物(2021)](https://www.lse.ac.uk/News/Latest-news-from-LSE/2021/k-November-21/Octopuses-crabs-and-lobsters-welfare-protection)

- 根据英国法律，八爪鱼、螃蟹和龙虾已被承认为有生命的生物。- 这意味着政府已经承认这些动物能够感受到疼痛和痛苦，并且必须得到相应的对待。- 新法律还禁止销售和分销供人食用的活甲壳类动物，包括龙虾、螃蟹和小龙虾。

### [行业反应](http://news.ycombinator.com/item?id=36216959)

- 根据新的环境措施，英国已经承认章鱼、螃蟹和龙虾是有生命的生物，该措施旨在改善动物福利，特别是无脊椎动物的福利。- 新的法律禁止某些做法，如不经绝育的活体水煮，向未经训练的处理者活体出售十足类甲壳动物，以及去爪。- 新的分类法最终可能导致减少所有动物的痛苦，尽管有些人对禁止活体甲壳类动物消费或改善肉类行业的做法持怀疑态度。
