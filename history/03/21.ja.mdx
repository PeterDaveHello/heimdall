### [ウェブフィンガープリントは思ったより悪い](https://www.bitestring.com/posts/2023-03-19-web-fingerprinting-is-worse-than-I-thought.html?ref=hackernewsgpt.com)

- Web フィンガープリンティングとは、Web サイトが、Cookie やその他の永続的なストレージに依存することなく、Web ブラウザやデバイス構成に関する情報を利用して、個々のユーザーを識別・追跡するために使用する手法です。
- フィンガープリンティングは、詐欺防止やウェブサイトのセキュリティのために、e コマースウェブサイトにサービスとして販売されています。
- この記事では、FingerprintJS という会社がフィンガープリントを他のウェブサイトにサービスとして販売していることを詳しく説明し、フィンガープリントの有効性を実証するテストと、フィンガープリントから保護するための最善の戦略を紹介します。
- Tor ブラウザや Firefox を resistFingerprinting=true で使用すると、Google Chrome や Brave のような Chromium ベースのブラウザを使用するよりも保護効果が高くなります。
- VPN はフィンガープリンティングからの保護を提供しません。

### [Asahi Linux、パフォーマンスと互換性を向上させた GPU ドライバのアップデートをリリース](https://asahilinux.org/2023/03/road-to-vulkan/?ref=hackernewsgpt.com)

- Asahi Linux チームは、GPU ドライバのバージョン 0.3 をリリースしました。このバージョンでは、パフォーマンスの大幅な改善、Mesa との互換性の向上、明示的な同期サポートが含まれており、これにより CPU-GPU 並列性が向上しています。
- Asahi Linux は、過去 1 年間、Apple Silicon 用のオープンソース GPU ドライバに取り組んでおり、最近、ドライバのカーネル部分とユーザー空間部分のインターフェースである、各クラスの GPU 用のユーザー空間 API を設計しました。
- バッチ追跡システムは、アクティブでないにもかかわらず提出されるバッチの処理に対応し、効率を高めるために開発されました。
- 新しいドライバは、マルチ VM、マルチジョブサブミッション、コンピュートシェーダの実行機能、blit コマンドのスタブサポートもサポートし、DRM スケジューラコンポーネントのメモリ安全性のバグが修正されました。

### [Mailchimp がブラウザの開発ツールを開いたユーザーをブラックリストに登録](http://news.ycombinator.com/item?id=35235732&ref=hackernewsgpt.com)

- MailChimp のユーザーが、電子メールの編集中にブラウザの開発ツールを開いた後、アカウントからブロックされたことを報告しています。
- コメントでは、この禁止は、コンソールでエスケープされていない、または不正な HTML を使用することによって引き起こされるクロスサイトスクリプティングの脆弱性、またはシステムのウェブアプリケーションファイアウォールと相互作用する API に関連しているかもしれないと指摘されました。
- Mailchimp は、ユーザーがメールテンプレートを検査したり、メールキャンペーンをリバースエンジニアリングすることを防ごうとしているのではないか、あるいは、ブロックの仕組みは、ユーザーが広告ブロッカーを使用することを防ぐためではないかと推測する人もいました。
- この問題は、開発ツールに対する制限や、過度に攻撃的または慎重なセキュリティ機構によって仕事に不可欠なツールを拒否される可能性のある開発者への影響について懸念を抱かせています。

### [Petals は 100B 以上の言語モデルの共同利用を可能にする](https://petals.ml/?ref=hackernewsgpt.com)

- Petals は、大規模な言語モデル(BLOOM-176B など)を、モデルの一部をロードし、他の部分を提供する他の人とチームを組んで共同利用することを可能にします。
- ユーザーは推論や微調整を行うことができ、シングルバッチ推論は 1 トークンあたり ≈1 秒 ∽、並列推論は数十トークン/秒に達する。
- Petals は、従来の言語モデル API よりも柔軟性が高く、任意の微調整やサンプリング手法の採用、モデルのカスタムパスの実行、モデルの隠れた状態の確認が可能です。PyTorch を使用し、柔軟性を高めています。
- 現在は待機中ですが、ユーザーは GitHub の ColabDocs で試したり、Discord チャンネルに参加して開発をフォローすることができます。

### [それで `fzf` をインストールしました - 次は何ですか](https://andrew-quinn.me/fzf/?ref=hackernewsgpt.com)

- Fzf は、ファイルやディレクトリのファジー検索を可能にするコマンドラインツールです。
- Ctrl+R や Alt+C などのキーボードショートカットで、検索効率を向上させることができます。
- Fzf は、grep や vi などの他の UNIX ツールと組み合わせて使用することで、より優れた検索体験を提供します。
- 高速な再帰的 grep ツールである rg と組み合わせることで、より迅速な検索が可能になります。
