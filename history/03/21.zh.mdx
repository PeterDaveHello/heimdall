## 2023 年 3 月 21 日

### [在伦敦市，骑自行车的人现在比开车的人多](https://www.forbes.com/sites/carltonreid/2023/03/01/cyclists-now-outnumber-motorists-in-city-of-london/?ref=hackernewsgpt.com)

- 骑自行车的人现在是城市街道上高峰期最大的车辆模式，占城市道路交通的 40%，全天占 27%。
- 在过去的十年中，机动车的使用在英国的金融中心越来越受到限制。
- 目前，行人的数量低于大流行前的数字，机动车的数量也是 2019 年的 80%，但骑自行车的人数是大流行前水平的 102%。

### [亚马逊在先前裁员的基础上再裁员 9000 人](https://www.cnbc.com/2023/03/20/amazon-layoffs-company-to-cut-off-9000-more-workers.html?ref=hackernewsgpt.com)

### [网络指纹识别比我想象的要糟糕](https://www.bitestring.com/posts/2023-03-19-web-fingerprinting-is-worse-than-I-thought.html?ref=hackernewsgpt.com)

- 网络指纹是一种技术，网站通过使用网络浏览器和设备配置的信息来识别和跟踪个人用户，而不依靠 cookies 或其他持久性存储。
- 指纹技术作为一项服务出售给电子商务网站，用于防止欺诈和网站安全。
- 本文详细介绍了一家名为 FingerprintJS 的公司如何将指纹识别作为一项服务出售给其他网站，并介绍了证明指纹识别功效的测试以及防止指纹识别的最佳策略。
- 使用 Tor 浏览器或 Firefox 的 resistFingerprinting=true，比使用基于 Chromium 的浏览器如谷歌浏览器或 Brave 提供更多的保护。
- VPN 不提供对指纹的保护。

### [Adobe Firefly: AI Art Generator](https://www.adobe.com/sensei/generative-ai/firefly.html?ref=hackernewsgpt.com)

- Adobe 公布了 Firefly，这是一个创造性的生成性人工智能模型，让用户使用日常语言生成一系列创造性的设计和设置。
- 用户可以上传图形以生成海报和横幅设计，在视频上尝试不同的编辑，混合两张或更多照片的元素，并最终创建 3D 模型。
- Firefly 将被整合到 Adobe Creative Cloud 中，旨在通过提供生成性人工智能工具，满足创意社区的需求和工作流程，显著提高创造力。
- Adobe 致力于以负责任的方式开发创造性的生成性人工智能，并通过 "内容真实性倡议 "和 "内容来源和真实性联盟"，努力支持生成性人工智能的问责制、责任制和透明度。

### [Asahi Linux 发布更新的 GPU 驱动，性能和兼容性得到改善](https://asahilinux.org/2023/03/road-to-vulkan/?ref=hackernewsgpt.com)

- Asahi Linux 团队发布了 0.3 版本的 GPU 驱动，其中包括巨大的性能改进，改进了与 Mesa 的兼容性，以及明确的同步支持，从而提高了 CPU-GPU 的并行性。
- Asahi Linux 在过去的一年里一直在为 Apple Silicon 开发开源的 GPU 驱动，最近为每一类 GPU 设计了一个用户空间 API，这是驱动的内核和用户空间部分的接口。
- 批量跟踪系统的开发是为了处理那些并不活跃但被提交的批次，以提高工作效率。
- 新驱动程序还支持多虚拟机、多任务提交、执行计算着色器的能力，并包括对 blit 命令的存根支持，而 DRM 调度器组件中的内存安全错误已被修复。

### [流行的摄影评论网站 DPReview.com 将被关闭](https://www.dpreview.com/news/5901145460/dpreview-com-to-close?ref=hackernewsgpt.com)

- 关闭 DPReview 的决定是其母公司对网站年度运营计划审查的一部分。
- 该网站将以只读模式提供服务，直到 4 月 10 日之后的有限时间，之后将无法使用。
- 用户可以在 4 月 6 日之前要求下载他们上传的所有内容。
- 该网站的编辑团队将继续从事评论工作，旨在关闭前提供一些有史以来最好的内容。
- DPReview 近 25 年来一直为世界各地的摄影师提供信息丰富的内容、评论和新闻。
- DPReview 发布了一份由相机和镜头租赁公司 Lensrentals 出租的最受欢迎的点对点相机的清单，揭示了拥有先进相机系统的智能手机可能已经使 "点对点 "相机的概念过时。然而，一些摄影师仍然喜欢专用相机的简单性和可靠性。

### [Mailchimp 将打开浏览器开发工具的用户列入黑名单](http://news.ycombinator.com/item?id=35235732&ref=hackernewsgpt.com)

- MailChimp 的用户报告说，他们在编辑邮件时打开浏览器的开发工具后，被封锁了他们的账户。
- 评论者指出，该禁令可能与控制台中使用未转义或畸形的 HTML 造成的跨站脚本漏洞有关，或者与与系统的网络应用程序防火墙交互的 API 有关。
- 一些人猜测，Mailchimp 是为了防止用户检查电子邮件模板或逆向工程的电子邮件活动，或者阻止机制是为了防止用户使用广告屏蔽器。
- 这个问题引起了人们对开发工具限制的关注，以及对开发者的影响，他们可能因为过于激进或谨慎的安全机制而被剥夺了工作的基本工具。

### [Petals 允许协作使用 100B 以上的语言模型](https://petals.ml/?ref=hackernewsgpt.com)

- Petals 允许协作使用大型语言模型(如 BLOOM-176B)，方法是加载模型的一小部分，并与其他服务于其他部分的团队合作。
- 用户可以运行推理或微调，单批推理的运行速度为每个标记 ≈1 秒，而并行推理则达到数百个标记/秒。
- 与经典的语言模型 API 相比，Petals 提供了更多的灵活性，能够采用任何微调和采样方法，通过模型执行自定义路径，或查看其隐藏状态。它使用 PyTorch 来增加灵活性。
- 目前在等待名单上，但用户可以在 GitHub 上的 ColabDocs 中试用它，或加入 Discord 频道来关注它的发展。

### [So you've installed `fzf` - now what?](https://andrew-quinn.me/fzf/?ref=hackernewsgpt.com)

- Fzf 是一个命令行工具，可以实现文件和目录的模糊查找。
- 键盘快捷键如 Ctrl+R 和 Alt+C 可以提高搜索效率。
- Fzf 可以与其他 UNIX 工具如 grep 和 vi 结合使用，以改善搜索体验。
- 将 fzf 与 rg(一种快速递归的 grep 工具)搭配使用，可以提供更快速的搜索。
