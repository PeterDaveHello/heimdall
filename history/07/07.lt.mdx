## 2023 m. liepos 7 d.

#### Beveik visi likę "Evernote" darbuotojai buvo atleisti

- Populiari užrašų programėlė "Evernote" atleido didelę dalį savo darbuotojų.
- Bendrovė susidūrė su finansiniais sunkumais ir stengėsi konkuruoti su kitomis užrašų programėlėmis.
- Ši žinia svarbi, nes leidžia manyti, kad "Evernote" gali patirti sunkumų, o programėlės naudotojams gali tekti apsvarstyti alternatyvias galimybes.

#### [Pramonės reakcijos](http://news.ycombinator.com/item?id=36609641)

- Po to, kai "Evernote" perėmė "Bending Spoons", ji atleido didelę dalį likusių darbuotojų, todėl kyla susirūpinimas dėl bendrovės gebėjimo finansuoti naujas funkcijas.
- Vartotojai išreiškia nusivylimą dėl to, kad "Evernote" stengiasi nemokamus vartotojus paversti mokamais klientais, ir pabrėžia paprastumo, patikimumo ir nešiojamumo svarbą užrašų programėlėje.
- Alternatyvios užrašų programėlės, tokios kaip "Obsidian" ir "OneNote", minimos kaip sėkmingi "Evernote" konkurentai, o naudotojai aptaria rizikos veiksnius ir trūkumus, susijusius su rizikos kapitalo finansuojamomis pradedančiosiomis įmonėmis.

#### [GPT-4 API General Availability](https://openai.com/blog/gpt-4-api-general-availability)

- Kompanija "OpenAI" paskelbė, kad visiems mokantiems API klientams visuotinai prieinama GPT-4 API, jų pajėgiausias modelis. Ši labai pageidaujama API suteikia kūrėjams prieigą prie naujoviškų produktų, kuriuose naudojamas GPT-4.
- OpenAI taip pat pristatė pokalbių užbaigimo API, kuri pasirodė esanti galinga ir lanksti atliekant įvairias užbaigimo užduotis. Ji sudaro 97 % API naudojimo atvejų ir siūlo geresnius rezultatus su struktūrizuota raginimo sąsaja.
- Senesni užbaigimo API modeliai bus pašalinti ir nuo 2024 m. sausio 4 d. pakeisti naujais modeliais. Programuotojams, naudojantiems šiuos senesnius modelius, reikės atnaujinti savo integraciją, kad būtų užtikrinta tolesnė prieiga prie API.

#### [Pramonės reakcijos](http://news.ycombinator.com/item?id=36621120)

- OpenAI išleido GPT-4 API, kuri dabar prieinama visiems mokamiems naudotojams.
- Tokie "ChatGPT" modeliai kaip gpt-3.5-turbo ir text-davinci-003 ilgainiui bus pašalinti GPT-4 API naudai.
- GPT-4, palyginti su ankstesnėmis versijomis, pasižymi geresniu našumu ir galimybėmis, o kūrėjai giria jos kokybės šuolį įvairiose srityse.

#### [Debesyje palaikomas SQLite](https://sqlite.org/cloudsqlite/doc/trunk/www/index.wiki)

- Sistema "Cloud Backed SQLite" (CBS) leidžia saugoti duomenų bazes debesų saugyklų paskyrose, todėl saugyklų klientai gali jas skaityti ir rašyti neatsisiuntę visos duomenų bazės.
- Šiuo metu CBS palaiko "Azure Blob Storage" ir "Google Cloud Storage", gali būti, kad bus pridėta ir kitų debesų saugyklų sistemų palaikymas.
- CBS naudoja fiksuoto dydžio blokus SQLite duomenų bazėms debesyje saugoti, o manifest faile aprašoma, kaip blokai sujungiami į duomenų bazes. Taip pat teikiamos API ir komandinės eilutės priemonės, skirtos duomenų bazėms debesyje įkelti, atsisiųsti ir valdyti.

#### [Pramonės reakcijos](http://news.ycombinator.com/item?id=36610595)

- "Cloud Backed SQLite" leidžia teikti dideles "SQLite" duomenų bazes dalimis ir teikti joms užklausas HTTP diapazono užklausomis.
- Biblioteka sql.js-httpvfs atlieka didžiąją dalį darbo, susijusio su duomenų bazės suskaidymu į dalis ir įkėlimu į "GitHub Pages".
- Žmonės džiaugiasi "SQLite" greičiu ir efektyvumu, nes tai yra tik skaitymui skirta duomenų bazė, skirta dideliems duomenų rinkiniams be serverio infrastruktūros pridėtinių išlaidų.
