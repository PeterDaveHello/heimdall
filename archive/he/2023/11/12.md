## [Flyscrape: כלי גירוד אינטרנט רב עוצמה וידידותי למשתמש בגו](https://github.com/philippta/flyscrape)

- המגרד מספק תיעוד יסודי ודוגמאות להתקנה ושימוש קלים.
- הוא כולל גם API של שאילתות לטיפול ברכיבי HTML ו- API של HTTP לביצוע בקשות.
- אנו מעודדים את המשתמשים לדווח על בעיות ולהציע שיפורים עבור הכלי.

### [תגובות](https://news.ycombinator.com/item?id=38230476)

- Flyscrape הוא כלי גירוד אינטרנט שנכתב בגו ומתאים למשימות גירוד בסיסיות, אך עשוי שלא לעבוד טוב עם פרויקטים מורכבים יותר.
- יש לו מגבלות בגירוד אתרים בעולם האמיתי וחסר שוויון תכונות מלא עם קולי.
- עם זאת, זה נוח למשתמשים שמכירים JavaScript וניתן להשתמש בו עם שירותי צד שלישי כמו ScrapingBee לתמיכה בדפדפן.
