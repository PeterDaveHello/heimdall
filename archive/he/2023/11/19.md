## [התקדמות לקראת מהדר חלודה משולב עם GCC](https://lwn.net/SubscriberLink/954787/41470c731eda02a4/)

- פרויקט gccrs עובד על יצירת מהדר Rust להכללה במהדורת GCC 14, המכוונת לגרסת Rust 1.49.
- הפרויקט מתמקד כיום בתמיכה בהרכבת ארגזי ליבה והקצאה, אך מתמודד עם אתגרים ברזולוציית מאקרו ומאקרו דקורטור.
- פרויקט זה נועד לאפשר הידור קוד Rust באמצעות GCC, התורם לפיתוח Rust ותומך במערכות משובצות. פרויקט מתקדם נוסף, rustc_codegen_gcc, כבר מאפשר הידור של Rust עבור לינוקס ללא תיקונים, והדיון כולל שילוב Polonius במהדר Rust ואת היתרונות של יישום מהדר Rust מבוסס GCC כגון חשיפה לבאגים ו-bootstrapping.

### [תגובות](https://news.ycombinator.com/item?id=38684102)

- הפיתוח של gccrs, מהדר Rust מבוסס GCC, נדון בקהילת הטכנולוגיה.
- יש ויכוח על הטמעות מרובות של Rust, כאשר חלק טוענים שזה יכול לעזור לחשוף באגים ולשפר את השפה, בעוד שאחרים מודאגים לגבי תאימות וכפילויות.
- מהדרים מתמודדים עם אתגרים בשמירה על תקני שפה, ושימוש במהדרים מרובים יכול לעזור לשפר את איכות הקוד ולזהות באגים. השיחה גם בוחנת את הצורך במפרט רשמי עבור Rust ואת היתרונות והחסרונות הפוטנציאליים של סטנדרטיזציה.
