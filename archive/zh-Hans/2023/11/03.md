## [通过节奏和音调为婴儿唱歌有助于语言学习](https://www.theguardian.com/science/2023/dec/01/singing-to-babies-is-vital-to-help-them-understand-language-say-scientists)

- 剑桥大学的一项研究表明，给婴儿唱歌是帮助他们学习语言的一个重要因素。
- 研究表明，婴儿最初是通过节奏和音调而不是单个声音来理解语言的。
- 这项研究对 "处理语音信息是语言学习的主要方面 "这一观点提出了质疑，并表明阅读障碍和发育性语言障碍可能与节奏感有关。

### [反应](https://news.ycombinator.com/item?id=38500906)

- 婴儿是通过节奏和音调而不是单个声音来理解语言的，这就对语言发展所需的 "婴儿语言 "提出了挑战。
- 语言匮乏会阻碍语言的习得，但只要婴儿没有被剥夺或虐待，他们就会正常发展语言技能。
- 养育策略并不是影响儿童语言发展的唯一因素，个人经历和遗传差异也在其中发挥着作用。

## [GQL：Git 文件查询语言](https://github.com/AmrDeveloper/GQL)

- GQL 是一种类似于 SQL 的查询语言，可直接查询 .git 文件，无需单独的数据库。
- 它提供一系列查询操作，包括选择、聚合、过滤和排序。
- GQL 不区分大小写，语法与 SQL 相似。该软件采用 MIT 许可发布。

### [反应](https://news.ycombinator.com/item?id=38498688)

- 这篇文章探讨了 GQL（一种用于 git 仓库的数据分析工具）的使用，并提出需要更方便用户的分析查询。
- 报告讨论了 Git 的类 SQL 查询语言可能带来的好处和缺点，并对其实用性提出了不同意见。
- 讨论了 GQL 和 GraphQL 之间的混淆，以及替代接口和精炼 SQL 方法。

## [光缆能 "听到 "蝉的叫声，提供新的昆虫监测方法](https://www.wired.com/story/cicadas-are-so-loud-fiber-optic-cables-can-hear-them/)

- 科学家们发现，光纤电缆可以捕捉到蝉声，从而为监测昆虫种群提供了一种潜在的新方法。
- 利用分布式声学传感技术（DAS），研究人员可以分析通过电缆发射的激光反弹回来的光线，从而探测到巨大声响或地震活动造成的干扰。
- 这项技术可以让昆虫学家利用已经非常发达的光纤电缆远程监测蝉，并收集有关其种群数量和位置的数据。不过，利用 DAS 监测较安静的昆虫物种可能更具挑战性。

### [反应](https://news.ycombinator.com/item?id=38500065)

- 在入侵检测和振动监测等应用中，光纤电缆可用作传感器来检测弯曲情况并收集相关信息。
- 光时域反射仪（OTDR）通常用于测量光缆中的反射，但不确定的光缆速度会影响这些测量的准确性。
- 电缆的扭曲率等各种因素会导致电缆速度的不确定性，并影响传感技术的精度。

## [Clang 的默认行为导致在原始 Pi B+ 上运行二进制文件时出现问题](https://rachelbythebay.com/w/2023/11/30/armv6/)

- 作者分享了在 Raspberry Pi B+ 上使用 clang 编译 C++ 程序的经验。
- 由于默认行为的改变，使用 clang 生成的二进制文件无法在 B+ 上运行。
- 作者指出了一种通过指定目标架构的变通方法，但质疑为什么这种默认行为的改变没有被注意到，并认为较旧的 Raspberry Pi 型号已不再广泛使用。

### [反应](https://news.ycombinator.com/item?id=38504134)

- 由于 Raspberry Pi B+ 采用了较老的 ARM1176 内核，因此讨论了 Clang 编译器与原始 Raspberry Pi B+ 之间的兼容性问题。
- 通过指定目标为 armv6，解决了 LLVM 编译器中的一个错误。
- 对话中还谈到了在 Linux 发行版中支持旧硬件所面临的挑战、自由软件的概念，以及使用 QEMU 和 Docker 在 x86_64 上模拟 ARM64。会议强调，正确配置工具箱容器的默认映像至关重要。

## [Excalidraw：手绘风格图表的开源绘图工具](https://github.com/excalidraw/excalidraw)

- Excalidraw 是一款开源虚拟白板，用于创建手绘风格的图表和线框图。
- 其功能包括无限画布、可定制工具、图像支持和端到端加密协作。
- Excalidraw.com 应用程序提供 PWA 支持、实时协作和本地优先功能，而 Excalidraw npm 软件包则提供其他功能。该项目由赞助商赞助，并与各种平台和服务集成。

### [反应](https://news.ycombinator.com/item?id=38499375)

- Excalidraw 是一款流行的开源绘图工具，以其协作功能、友好的用户界面和手绘风格而著称。
- 它被广泛用于系统设计访谈和远程协作，尤其是初创企业，他们非常欣赏它的简洁性和对设计的专注。
