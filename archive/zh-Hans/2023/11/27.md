# 2023-12-27

## [日本采取行动打破苹果和谷歌应用商店垄断](https://asia.nikkei.com/Business/Technology/Japan-to-crack-down-on-Apple-and-Google-app-store-monopolies)

- 日本正计划出台相关法规，迫使苹果和谷歌等科技巨头在其移动操作系统上允许第三方应用程序商店和其他支付方式。
- 其目的是解决滥用日本市场支配地位的指控。
- 该立法将涵盖应用程序商店和支付、搜索、浏览器和操作系统，预计将于 2024 年提交议会。

### [反应](https://news.ycombinator.com/item?id=38773429)

- 日本以可能违反反垄断法为由，对苹果和谷歌的应用程序商店垄断表示担忧。
- 讨论的重点是与应用程序商店和游戏机市场的竞争、开放性和用户体验有关的问题。
- 关于应用程序商店的监管、游戏机的未来，以及竞争法对保护消费者利益的重要性，都存在争论。

## [mRNA-4157 癌症疫苗促进 Keytruda 治疗，复发率降低 492b 期研究](https://www.freethink.com/health/cancer-vaccine)

- Moderna 和默克公司合作开发了一种基于 mRNA 的癌症疫苗，名为 mRNA-4157。
- 该疫苗与默克公司的 Keytruda 治疗相结合，在预防黑色素瘤复发或死亡方面取得了良好的效果。
- 在一项 2b 期研究中，与 Keytruda 单药相比，联合疗法可将三年后的复发或死亡风险降低 49%。
- 这些积极的研究结果促使该联合疗法启动了第三阶段试验。
- 到 2025 年，这种疫苗有可能在一些国家获得加速批准。
- Moderna 和默克公司还在探索将这种疫苗用于其他类型的癌症，如非小细胞肺癌。

### [反应](https://news.ycombinator.com/item?id=38775439)

- 讨论深入到疫苗的各个方面，例如 Moderna 的 mRNA 癌症疫苗的疗效和潜在副作用。
- 报告还谈到了目前围绕疫苗定义的争论，以及对 COVID-19 疫苗安全性和长期影响的担忧。
- 讨论进一步探讨了凝血问题、COVID-19 疫苗接种的意义、疫苗开发的资助模式、用于癌症治疗的个性化疫苗、药物管线的进展与挑战以及制药业内怀疑论的作用等话题。

## [亚马逊 Prime Video 将引入广告，为无广告体验收取额外费用](https://www.theverge.com/2023/12/26/24015595/amazon-prime-video-ads-coming-january-29)

- 从 1 月 29 日开始，亚马逊 Prime Video 将引入有限的广告，除非用户选择每月额外支付 2.99 美元来获得无广告体验。
- 此举的目的是让亚马逊能够投资内容，更好地与其他流媒体服务竞争。
- 虽然亚马逊 Prime 的月租费将保持不变，但无广告流媒体选项将使总费用增加到略低于 18 美元，这与迪斯尼 Plus 和 Netflix 等其他流媒体服务的做法一致，后者在其更便宜的计划中包含广告。

### [反应](https://news.ycombinator.com/item?id=38777516)

- 亚马逊 Prime Video 将于 1 月 29 日开始播放广告，这让使用广告屏蔽器的用户感到担忧。
- Netflix 和 Prime Video 此前曾在检测到广告屏蔽软件时阻止用户观看视频。
- 亚马逊的这一举动可能会影响喜欢无广告环境的用户的观看体验。

## [游戏男孩彩色版：改进与创新](https://www.copetti.org/writings/consoles/game-boy/)

- Game Boy 是 NES 的便携式版本，于 1989 年发布并广受欢迎。
- 它经历了两次改版和两代产品，包括单色 Game Boy 和 Game Boy Color。
- Game Boy Color 的特点是时钟速度提高了一倍，向后兼容，并增加了银行切换和扩展内存等功能。
- 该系统的图形处理能力有限，但包括用于视觉效果的调色板和色块。
- 它的音频系统有四个声道，可提供不同的波形。
- 启动过程包括初始化 RAM、复制标识和检查盒式磁带。
- Game Boy Color 有不同的游戏运行模式，并能与其他游戏机通信。
- 实施了反盗版措施以控制发行。

### [反应](https://news.ycombinator.com/item?id=38775904)

- 这篇文章探讨了 Game Boy 和 Game Boy Color 的诞生过程，这两款便携式游戏的概念最初曾遭到质疑。
- 该团队对便携式游戏设备的设想彻底改变了整个行业，并使移动游戏得到了普及。
- 文章还深入介绍了便携式游戏设备的历史，并重点介绍了 Game Boy 的功能。

## [研究发现，私募股权收购损害了患者护理](https://medicalxpress.com/news/2023-12-quality-declines-private-equity-hospitals.html)

- 哈佛大学医学院进行的一项全国性研究显示，医院被私募股权公司收购后，医疗质量会下降。
- 这些医院的病人发生可预防疾病和不良后果（如跌倒和感染）的风险较高。
- 研究结果凸显了利润驱动动机与患者护理和安全之间的潜在冲突，引发了人们对私募股权在医疗保健领域的影响的担忧。

### [反应](https://news.ycombinator.com/item?id=38773426)

- 私募股权公司对医院和医疗行业产生了影响，人们担心并购后医疗质量会下降。
- 关于包括医疗保健在内的各行各业私有化的利弊以及政府监管在其中的作用，存在着争论。
- 此外，还讨论了私募股权的效率和在医疗保健领域实施自由市场模式所面临的挑战，以及该行业利润驱动动机可能带来的负面影响。

## [英国实现历史性碳减排，却未被公众和政界人士注意到](https://www.spectator.co.uk/article/uk-becomes-first-g20-country-to-halve-its-carbon-emissions/)

- 英国已成为二十国集团（G20）中第一个将碳排放量减半的主要国家，目前的碳排放量低于 3.2 亿吨。
- 这一成就意义重大，因为它表明，尽管该国人口和经济规模较大，但在减少碳足迹方面仍取得了进展。
- 令人惊讶的是，这一成就很少受到运动团体和政治家的关注，也没有得到积极推广。这些数据可在《旁观者》数据中心获取，由全球碳项目跟踪。

### [反应](https://news.ycombinator.com/item?id=38770693)

- 批评者认为，英国减少碳排放的部分原因是外包了制造业，并向服务型经济转型，这引起了人们对英国消费商品排放出口的担忧。
- 讨论了碳核算做法、英国脱欧对制造业的影响以及重复计算绿色能源生产的可能性。
- 文章探讨了核能、风能和其他可再生能源的作用，以及公共交通的有效性和便利性。文章就环保措施对经济的影响发表了不同意见，并对政治偏见和消息来源的可信度表示担忧。

## [使用 Make：简化 Linux 中的 Makefile 进程](https://text.causal.agency/001-make.txt)

- Linux 中的 make(1) 命令可用于编译 C、C++ 和 Fortran 文件，而无需 Makefile。
- CFLAGS 和 LDLIBS 等环境变量可用于向 make 命令传递编译器和链接器标志。
- 在 Makefile 中加入一条规则，可以在头文件发生变化时自动重建源文件，而添加一个干净目标则可以删除目标文件和二进制文件。

### [反应](https://news.ycombinator.com/item?id=38774165)

- 讨论还深入探讨了依赖管理、并行化和 Makefile 组织等问题。
