## [Představujeme SSH3: Rychlejší a bezpečnější Shell s HTTP/3 a QUIC](https://github.com/francoismichel/ssh3)

- SSH3 je přepracovaná verze protokolu SSH, která využívá mechanismy HTTP a nabízí rychlejší a bezpečnější prostředí shellu.
- Pro bezpečné navázání kanálu používá QUIC+TLS1.3 a pro ověření uživatele HTTP Authorization.
- SSH3 poskytuje rychlejší navazování relací, vylepšené metody ověřování (OAuth2.0, OpenID Connect), odolnost proti útokům skenováním portů, přesměrování portů UDP a funkce moderního protokolu QUIC.
- Umožňuje skrýt server SSH a podporuje certifikáty X.509 pro ověřování.
- SSH3 je v současné době experimentální a lze jej nainstalovat stažením binárních souborů nebo kompilací kódu ze zdrojových kódů.
- Pro použití SSH3 je třeba nasadit server SSH3 a klient SSH3 podporuje různé metody ověřování.

### [Reakce](https://news.ycombinator.com/item?id=38664729)

- Komentáře se týkají řady témat souvisejících s SSH, například kompatibility s protokoly jako HTTP/3 a QUIC, používání certifikátů pro ověřování a potenciální nové verze SSH.
- Diskutuje se také o alternativních protokolech, jako je Mosh.
- Diskuse zahrnuje různé pohledy a obavy týkající se bezpečnosti, účinnosti, implementace, cenzury sítě, skrývání protokolů a správy zařízení.
