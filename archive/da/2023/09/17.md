# 2023-09-17

## [Horcrux: Opdel din fil i krypterede fragmenter](https://github.com/jesseduffield/horcrux)

- Horcrux er et open source-værktøj, der opdeler en fil i krypterede sektioner, så brugerne ikke behøver at huske en adgangskode for at sikre distribueret lagring.
- Teknologien fungerer ved hjælp af Shamir Secret Sharing Scheme, som fragmenterer en krypteringsnøgle på en måde, så kun et foruddefineret antal dele kan rekonstruere den.
- Værktøjet henvender sig til en særlig målgruppe, der skal kryptere store, følsomme filer og sprede dem via flere kanaler for at mindske risikoen for aflytning.

### [Reaktioner](https://news.ycombinator.com/item?id=37534615)

- Diskussionen er centreret omkring flere kryptografiske værktøjer og teknikker såsom Horcrux, Shamir Secret Sharing (SSS), HTTP Basic Authentication, polynomial samples og Oblivious Transfer (OT).
- Samtalen omhandler bredt disse værktøjers begrænsninger, sikkerhedsproblemer og potentielle brugssituationer og understreger behovet for forbedringer og brugertilgængelighed.
- Nogle ikke-relaterede emner, såsom Harry Potter, J.K. Rowlings politiske synspunkter og transkønnedes rettigheder, inddrages også kort i diskussionen.

## [Java 21 får mig til at kunne lide Java igen](https://wscp.dev/posts/tech/java-pattern-matching/)

- Java 21, som forventes lanceret i september 2023, har til formål at bringe record patterns ind i switch blocks og expressions og dermed muliggøre funktionelle programmeringsmønstre i Java.
- De diskuterede funktioner omfatter produkttyper, der er relevante for Javas record class, fordelene ved switch patterns og sealed classes fra Java 17, samt pattern matching med sealed interfaces og indførelsen af guard clauses.
- Denne kommende udgave, nemlig Java 21, fremhæves som en betydelig udvikling i retning af at forbedre Javas værktøjer til funktionel programmering.

### [Reaktioner](https://news.ycombinator.com/item?id=37538333)

- Diskussionerne er centreret om flere facetter af programmeringssprog som Java, Go, Kotlin og C#, herunder nye tilføjelser som virtuelle tråde og forseglede grænseflader i Java.
- Forskellige sprog blev undersøgt for deres enkelhed, værktøjer og brugervenlighed, samt forhindringerne ved at skifte til opdaterede Java-versioner.
- Det blev bemærket, at udviklernes meninger varierede betydeligt, hvor nogle roste robustheden og økosystemet i visse sprog, mens andre påpegede deres begrænsninger og manglende funktioner, hvilket understreger den subjektive karakter af sprogpræferencer.
