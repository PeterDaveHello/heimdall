## [Code Llama, a legkorszerűbb nagy nyelvi modell a kódoláshoz](https://ai.meta.com/blog/code-llama-large-language-model-coding/)

- A Code Llama egy új nagy nyelvi modell (LLM), amelyet kifejezetten kódolási feladatokhoz terveztek.
- Képes kódot és kódról szóló természetes nyelvet generálni kódból vagy természetes nyelvi kérésekből.
- A Code Llama három modellben kapható: Code Llama, Codel Llama - Python és Code Llama - Instruct.
- A kódolási feladatokban felülmúlta más nyilvánosan elérhető LLM-ek teljesítményét.
- A modellek a Llama 2-re épülnek, és kutatási és kereskedelmi célokra szabadon felhasználhatók.
- A Code Llama képes javítani a fejlesztői munkafolyamatokat és elérhetőbbé tenni a kódolást.
- Támogatja a népszerű programozási nyelveket, és használható kódkiegészítésre és hibakeresésre.
- A Code Llama biztonságos és felelősségteljes használatát hangsúlyozzuk, és a modellek biztonsági értékelésen estek át.
- A Code Llama kiadása ösztönzi az innovációt és az együttműködést a mesterséges intelligencia közösségében.
