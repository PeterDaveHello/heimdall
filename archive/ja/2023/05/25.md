## 2023 年 5 月 25 日

## [PyPI Was Subpoenaed](https://blog.pypi.org/posts/2023-05-24-pypi-was-subpoenaed/)

- Python Software Foundation は、2023 年 3 月と 4 月に米国司法省から PyPI ユーザーデータの召喚状を 3 回受け取りました。
- PSF は、5 つの PyPI ユーザー名に関する要求されたデータを提供する必要があると判断しました。
- PyPI は、ユーザーのプライバシーとセキュリティを確保するために、新しいデータ保持と開示のポリシーを開発することを約束します。

#### [業界の反応](http://news.ycombinator.com/item?id=36061407)

- Python パッケージの PyPI リポジトリは、2023 年 3 月と 4 月に召喚状を受け取り、監視と説明責任の欠如に関する懸念が高まっています。
- この投稿では、米国、オランダ、カナダ、ドイツ、欧州の裁判例や法律など、各国における言論の自由とその法的限界について論じています。
- また、憲法修正第 1 条と米国最高裁によるその発展的な解釈、捜査における箝口令の使用と倫理に関する議論も含まれています。

## [胃食道逆流を解消する簡単エクササイズ(2022 年)](https://www.ncbi.nlm.nih.gov/pmc/articles/PMC9106553/)

- 胃食道逆流を解消するための下部食道括約筋(LES)の抵抗トレーニングのための新規エクササイズの概要が症例報告されています。
- この運動は、頭を胃より低く下げた膝立ちの姿勢で食物を飲み込むことで、LES に重力に対する抵抗力を与えるものである。
- 数ヶ月間、毎日繰り返した結果、胃食道逆流の症状は止まり、運動は再発することなく中止されたことから、この問題の永久的な解決策の可能性が示されました。

#### [業界の反応](http://news.ycombinator.com/item?id=36059247)

- ユーザーは個人の経験を共有し、胃食道逆流症(GERD)を管理するための食生活の改善、投薬、運動など様々な方法を提案しています。
- 記事では、GERD の症状を緩和するのに役立つかもしれない「スクワット＆ゴックン」エクササイズを紹介していますが、潜在的なリスクがあるため、注意が必要です。
- また、ヨガのポーズ、呼吸法、アルギン酸ナトリウムなどの代替療法も、個人によっては成功を収めていますが、PPI などの薬物療法はマイナスの副作用がある場合があります。

## [テープドライブから Frogger 2 のソースを復元するための長い道](https://github.com/Kneesnap/onstream-data-recovery/blob/main/info/INTRO.MD)

## [詳細をモーダルウィンドウで開かず、別ページにさせる](https://youdontneedamodalwindow.dev/)

- 詳細なコンテンツにモーダルウィンドウを使用する必要はありません。代わりに、別のページを使用してください。
- モーダルウィンドウは、ブックマークやリンクとして共有することができませんし、新しいタブで開くと混乱することがあります。
- モーダルウィンドウを正確に実装するのは難しく、必ずしも最良の選択とは限りません。代替案を検討し、Web アプリが「リフレッシュテスト」に合格するようにしましょう。

#### [業界の反応](http://news.ycombinator.com/item?id=36056376)

- この記事では、UI デザインにおけるモーダルウィンドウのユースケースと潜在的な問題について考察し、アラートやフォームなどの特定のタスクにのみ使用すべきであると提案しています。
- アクセシビリティは、スクリーンリーダーだけでなく、ズームやフォーカス/リーダーモードなど他の機能についても、モーダルウィンドウを設計する際に重要な考慮事項です。
- モーダルウィンドウの有用性と潜在的な欠点については、開発者の間で議論があり、混乱を招くという意見もあれば、特定の状況での使用を支持する意見もある。

## [SectorC: 512 バイトの C コンパイラ](https://xorvoid.com/sectorc.html)

- SectorC は x86-16 アセンブリで書かれた C コンパイラで、x86 マシンの 512 バイトのブートセクタに収まるもので、おそらくこれまでに書かれた C コンパイラの中で最も小さいものです。
- その小さなサイズにもかかわらず、SectorC は、グローバル変数、関数、if 文、while 文、多くの演算子、ポインタ非参照、インラインマシンコード、コメント、および多くを含む、実際の興味深いプログラムを書くのに十分に大きい C 言語のサブセットをサポートしています。
- SectorC は、最小限のレキサとパーサを実装し、コードサイズを最小化するために atoi のハッシュ関数を利用し、適切な C になることを可能にします。

#### [業界の反応](http://news.ycombinator.com/item?id=36064971)

- SectorC は、わずか 512 バイトの x86 アセンブリを使った C サブセットコンパイラであり、一般的な前提を疑うことの価値を実証している。
- 機械命令に JIT コンパイルし、コンパイルされた出力を実行することで、AOT コンパイラよりも多くの最適化を行うことができ、Javascript などの一部の言語やプロファイルガイド付き最適化の動的アナログとして使用されています。
- Hacker News に掲載されたディスカッションでは、仮想マシンに C コンパイラを展開するツールである Bootstrap Project の潜在的な用途を探っており、深宇宙への応用や小型マイクロコントローラ向けなど、この最小環境における構造体の使用に関する議論が行われています。

## 自分の作品に対する批判を個人的に受け止めないようにするにはどうしたらいいのでしょうか？

- Hacker News (Discussion Service) コミュニティのメンバーから、自分の仕事に対する批判を個人的に受け止めないようにするにはどうしたらいいかという質問がありました。
- 多くの Discussion Service ユーザーが、批判に対処するための自身の経験やテクニックを共有しました。
- ディスカッションでは、個人的な価値と作品の価値を切り離すこと、自己の価値に影響を与えることなくフィードバックから学ぶことに焦点が当てられました。

#### [業界の反応](http://news.ycombinator.com/item?id=36056299)

- 建設的な批判を個人的に受け取るのは普通のことですが、反脆弱な態度を身につけることで、批判をうまく処理し、そこからより良い人間になることができます。
- 自己価値をアウトプットやデリバリーから切り離し、ポジティブな意図を想定し、内省を実践し、セルフコンパッションに集中することで、プロフェッショナルな場でのネガティブなフィードバックに対応できるようになります。
- 悪いフィードバックを認識し、会社の目標に向かって努力し、楽観的に批判を予期しながら間違いを認めることが、ソフトウェア業界における個人的・職業的成長につながるのです。

## [脳脊髄インターフェイスを用いた脊髄損傷後の自然歩行](https://www.nature.com/articles/s41586-023-06094-5)

- 科学者たちは、3 人の脊髄損傷患者が自然でバランスの取れた歩行で再び歩けるようにする脳-脊椎インターフェースを開発しました。
- このインターフェースは、埋め込み型センサーを用いて脳からの信号を記録し、それを腰部に装着した皮膚の電極パッチを通じて脚の筋肉に電気刺激を与えるというデコードを行うものです。
- 3 人は、長年の麻痺の後、脚の装具を使って歩くのと同じような努力で、訓練と若干の補助で歩けるようになったそうです。

#### [業界の反応](http://news.ycombinator.com/item?id=36059429)

- デジタル脳脊髄インターフェースにより、脊髄損傷患者が補助なしで歩けるようになった
- この技術は、完全に脊髄を損傷した人の動きを回復させ
- 著者らは、麻痺を持つ幅広い人々に適用できることに自信を示している

## [極超音速ミサイルは誤解されている](https://medium.com/@ToryBrunoULA/hypersonic-missiles-are-just-misunderstood-1a35c8e3dd0)

- 極超音速ミサイルは新しいものではなく、何十年も前から持っていたものです。
- 極超音速ミサイルの本当の問題は、機動性があるため、弾道ミサイル用に設計された従来のミサイル防衛システムで迎撃することが困難なことです。
- レーザーのような指向性エネルギー(DE)は、その高速性とレーザーの視野全体をカバーする戦闘空間のために、機動性の高い極超音速ミサイルを迎撃するための潜在的なソリューションとなる可能性があります。

#### [業界の反応](http://news.ycombinator.com/item?id=36057675)

- この記事では、極超音速ミサイルを破壊するために衛星からのレーザーを使用する可能性について論じているが、展開のコストや複雑さについて懐疑的であり、また宇宙の兵器化に対する懸念もある。
- パトリオットのようなミサイル防衛システムの迎撃効果については、強力な防衛を主張する人もいれば、限界を指摘する人もいて、議論が交わされています。
- 専門家は、軍用レーザーの最近の技術的進歩とその潜在的な用途について議論するが、プラズマシールドに対する有効性や宇宙空間での武器展開の禁止など、考慮すべき注意点や制限もある。
