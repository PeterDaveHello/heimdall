## [Horcrux: Membagi file Anda menjadi beberapa bagian yang dienkripsi](https://github.com/jesseduffield/horcrux)

### [Reaksi](https://news.ycombinator.com/item?id=37534615)

### [Reaksi](https://news.ycombinator.com/item?id=37536103)

- Let's Encrypt telah meningkatkan server basis datanya dengan prosesor AMD EPYC dan SSD NVMe, meningkatkan performa dan skalabilitas, dan secara signifikan meningkatkan penerbitan sertifikat harian.
- Perbandingan dengan database lain dan penghematan biaya dibandingkan hosting AWS juga dibahas, memicu perdebatan antara menggunakan layanan cloud atau berinvestasi pada perangkat keras fisik.
- Strategi alternatif seperti menyewa server khusus atau menggunakan layanan colocation disarankan, dengan menekankan perlunya mempertimbangkan keadaan dan beban kerja yang unik saat mengambil keputusan.
