# 2023-08-19

## [詢問與猜測文化](https://jeanhsu.substack.com/p/ask-vs-guess-culture)

- 本文探討了提問文化與猜測文化的概念，以及它如何影響人際關係和互動。
- 詢問文化涉及直接詢問你想要什麼，而猜測文化依賴於閱讀間接線索，只詢問是否有信心得到積極的回應。
- 本文提供了這些文化規範之間衝突的示例，並提供了在主要詢問文化工作環境中導航猜測文化的策略。

### [反應](https://news.ycombinator.com/item?id=37176703)

- 本文討論了溝通和社交禮儀方面的文化差異，特別是“詢問”文化和“猜測”文化之間的差異。
- 分享個人經歷和軼事，以說明人際關係中的文化衝突。
- 這篇文章涉及的主題包括童年創傷、治療效果、輕微創傷對人際關係的影響、復原力以及理解和尊重文化差異的重要性。

## [羅傑斯先生說得有道理——經常向六個鄰居打招呼可以最大限度地提高幸福感](https://news.gallup.com/poll/509543/saying-hello-linked-higher-wellbeing-limits.aspx)

- 根據蓋洛普的一項研究，向多個鄰居打招呼與美國成年人的更高幸福感有關。
- 美國人的幸福感得分隨著迎接鄰居數量的增加而增加，最多六個鄰居。
- 問候鄰居在老年人和高收入人群中更為常見，它與社會、社區、職業、身體和財務狀況的改善有關。

### [反應](https://news.ycombinator.com/item?id=37175432)

- 社交互動，包括問候和與鄰居的互動，對於整體福祉和社區參與至關重要。
- 本文探討了社交焦慮和問候語中的文化差異的不同觀點，以及大流行對社交互動的影響。
- 建立聯繫和培養社區意識有很多好處，並受到物理和社會環境的影響。有意識地與他人互動至關重要。

## [聯邦調查局再次證明它不能被第702條信任](https://www.eff.org/deeplinks/2023/07/deja-vu-fbi-proves-again-it-cant-be-trusted-section-702)

- 聯邦調查局被發現在沒有遵循適當規則和限制的情況下訪問州參議員和美國參議員的私人通信。
- 外國情報監視法院（FISC）在解密意見中披露了這一資訊，強調聯邦調查局無視隱私和公民自由。
- 兩黨都支持對授權收集海外通信的第702條進行重大修改或終止。電子前沿基金會（EFF）主張關閉對美國通信的無證訪問，並提高第702條更新的透明度。

### [反應](https://news.ycombinator.com/item?id=37176717)

- 這次談話引發了對聯邦調查局未經授權的搜查和聯邦政府內部腐敗的擔憂。
- 它還強調了情報機構濫用權力和侵犯隱私權的行為。
- 討論強調執法問責制、改革、加強監督和保護公民自由的保障措施的必要性。

## [Windows 11 將愉快地執行 30 年前編譯的二進位檔](https://twitter.com/mikko/status/1692503249595584526)

- Windows 11 展示了令人印象深刻的向後相容性，因為它可以運行 30 年前編譯的二進位檔，特別是從 1993 年 8 月 18 日開始。
- 這突顯了Microsoft致力於確保與傳統軟體的相容性。
- 它表示使用者仍然可以在最新版本的 Windows 上訪問和運行較舊的應用程式，而不會出現任何問題。

### [反應](https://news.ycombinator.com/item?id=37176666)

- 討論圍繞著操作系統中向後相容性的概念，特別是對於Windows和Linux。
- 參與者探討了圍繞向後相容性的挑戰和辯論，例如保護文化價值和相關成本。
- 提到了強制軟體更新的問題、Microsoft的聲譽以及在平臺之間移植軟體的困難，以及不同操作系統之間不同程度的向後相容性。建議組織需要維護舊版本的系統庫以確保相容性。

## [文本：Python 的快速應用程式開發框架](https://github.com/Textualize/textual)

- Textualize 是一個用於 Python 的快速應用程式開發框架，它使開發人員能夠在終端中創建高級使用者介面，並且將來還可以在 Web 瀏覽器中創建高級用戶介面。
- 它提供了各種功能，例如滑鼠支援、1670 萬種顏色和流暢的動畫。
- Textualize 與 Linux、macOS 和 Windows 相容，需要 Python 3.7 或更高版本。它可以通過 pip 輕鬆安裝，並提供文檔、範例和 Discord 社區以支援。

### [反應](https://news.ycombinator.com/item?id=37174657)

- 文本是用於創建終端應用程式的Python框架，並且正在討論即將推出的Web瀏覽器版本。
- 使用者正在談論諸如「目錄樹」支援以及文本編輯器和編碼實踐的共用首選項等功能。
- 爭論集中在軟體開發中使用框架的優缺點，同時考慮到專案要求和技能水準。討論還涵蓋了軟體升級和框架使用的挑戰和好處，重點是安全性、資源管理和靈活性。

## [C 和 C++ 優先考慮性能而不是正確性](https://research.swtch.com/ub)

- 在 C 和 C++ 程式設計語言中，將性能置於正確性之上可能會導致程式損壞。
- 由於這種優先順序，未初始化變數、空指標和未定義行為的問題在 C 和 C++ 中很常見。
- Go 程式設計語言被呈現為具有不同理念的替代方案，優先考慮安全和保障。

### [反應](https://news.ycombinator.com/item?id=37178009)

- 討論圍繞程式設計語言及其對不同任務的適用性展開，例如優先考慮性能或安全功能。
- C 和 C++ 被認為優先考慮性能，但與 Rust 等語言相比可能缺乏安全功能。
- 關於 Rust 與 C++ 相比的局限性和優勢以及解決高級語言和硬體之間差距的重要性存在爭議。

## [警方正在從那些認為自己退出宣告的人那裡獲得DNA數據。](https://theintercept.com/2023/08/18/gedmatch-dna-police-forensic-genetic-genealogy/)

- 法醫遺傳系譜學家正在利用GEDmatch DNA資料庫中的一個漏洞來訪問那些選擇不與執法部門共用的個人的DNA數據。
- 這種技術有助於解決冷案，但它引發了對隱私和缺乏監管的擔憂。
- Ancestry和23andMe不允許執法部門訪問，但MyHeritage和GEDmatch在使用者同意的情況下進行，GEDmatch在受保護的個人資料方面存在漏洞。

### [反應](https://news.ycombinator.com/item?id=37179094)

- 執法部門使用DNA數據引起了對侵犯隱私和濫用遺傳信息的擔憂。
- 有一個漏洞允許與執法部門合作的系譜學家訪問選擇不共用其DNA的個人的個人資料。
- 公眾對這些侵犯隱私行為缺乏憤怒和政治關注令人擔憂，並引發了有關政府監督，個人自由以及與DNA資料庫相關的道德問題的更廣泛問題。

## [記錄顯示，世通元無視肯亞停止虹膜掃描的初始命令](https://techcrunch.com/2023/08/15/worldcoin-in-kenya/)
