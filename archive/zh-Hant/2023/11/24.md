# 2023-11-24

## [蘋果刪除開發者帳戶，引發法律戰和損失](https://seraleev.notion.site/Our-developer-account-was-removed-by-Apple-and-they-haven-t-paid-out-108-878-b61192711c74487480373badc70d42c0)

- 照片和視頻應用程式開發商 Sarafan Mobile Limited 關閉了他們的 Apple 開發者帳戶，導致公司及其員工遭受經濟損失和困難。
- 關閉導致 108,878 澳元的未付收入，Sarafan Mobile Limited 正在尋求追回。
- 蘋果公司以違反其開發者計劃許可協定為由關閉了該帳戶，但沒有提供具體細節或解決步驟。Sarafan Mobile Limited的上訴被駁回。

### [反應](https://news.ycombinator.com/item?id=38394364)

- 該系列涵蓋了有關蘋果對其應用程式生態系統控制的各種討論，包括刪除開發者帳戶和對虛假評論的猜測。
- 主題包括需要更好的審核和透明度，關於允許側載入和第三方應用商店的爭論，對壓迫性技術和監控的擔憂，以及開放操作系統平臺的潛在風險。
- 討論還包括應用商店及其政策之間的比較、App Store 開發者面臨的挑戰、虛假評論和付費服務的價值問題、對帳戶關閉和缺乏透明度的擔憂，以及平臺控制所涉及的風險和原則。

## [美國政府向AT&T支付無證電話記錄搜索費用，引發擔憂](https://www.theregister.com/2023/11/22/wyden_hemisphere_letter/)

- 參議員羅恩·懷登（Ron Wyden）對“半球計劃”（Hemisphere Project）表示擔憂，這是一項秘密監視計劃，允許美國執法機構在沒有搜查令的情況下搜索數萬億條電話記錄。
- 該計劃涉及AT&T提供可追溯到1987年的電話記錄，每天增加40億條通話記錄，白宮國家藥物管制政策辦公室支付這種訪問費用。
- 懷登呼籲提供有關該計劃的更多資訊，並質疑其合法性，因為它被指定為“執法敏感”，不受聯邦隱私影響評估的約束。

### [反應](https://news.ycombinator.com/item?id=38393237)

- 討論引發了對政府監控和侵犯隱私的擔憂，以及缺乏執法問責制。
- 它探討了諸如在沒有搜查令的情況下訪問電話記錄、使用元數據以及潛在的濫用權力等問題。
- 對話還涉及第四修正案、加密消息應用程式以及個人隱私與政府監控之間的平衡等主題。

## [根除脊髓灰質炎：防止復發和潛在益處](https://www.nature.com/articles/d41586-023-03602-5)

- 全球根除脊髓灰質炎的運動幾乎取得了成功，但人們擔心該病毒在根除后仍會在實驗室和個人中徘徊。
- 由於脊髓灰質炎病毒能夠悄無聲息地感染人類，而且在口服脊髓灰質炎疫苗（OPV）停用期間病毒有重新出現的風險，因此根除脊髓灰質炎的努力一直具有挑戰性。
- 為了防止脊髓灰質炎在根除后捲土重來，高人群免疫力、強有力的監測、更好的實驗室生物安全性以及開發不含活病毒的替代疫苗至關重要。

### [反應](https://news.ycombinator.com/item?id=38392277)

- 本文討論了根除脊髓灰質炎的進展和挑戰，包括中央情報局的干預和巴基斯坦的文化障礙。
- 對話涵蓋了各種主題，例如為脊髓灰質炎疫苗申請專利的決定，尋找脊髓灰質炎引起的癱瘓的治療方法，以及發達國家幫助衛生和供水基礎設施的重要性。
- 關於政府支出、人口增長、疫苗效力、陰謀論以及殺死奧薩馬·本·拉登影響巴基斯坦疫苗推廣的道德問題存在爭議。

## [Chrome 的 Manifest V3 威脅廣告攔截器，可能會將使用者推向 Firefox](https://www.malwarebytes.com/blog/news/2023/11/chrome-pushes-forward-with-plans-to-limit-ad-blockers-in-the-future)

- 谷歌計劃從 Manifest V2 過渡到 Manifest V3，這是其 Chrome 擴展規範的最新版本，儘管有人批評它限制了廣告攔截器
- Manifest V3 對過濾規則集引入了新的限制，從而降低了廣告攔截器和隱私保護跟蹤器攔截器的功能
- 電子前沿基金會（EFF）批評Manifest V3具有欺騙性和威脅性，而Firefox計劃採用它來實現跨瀏覽器相容性，這可能使Firefox成為需要完整擴展功能的使用者的更好選擇。

### [反應](https://news.ycombinator.com/item?id=38394613)

- 谷歌正在考慮限制其Chrome瀏覽器中的廣告攔截器，這引起了使用者和其他瀏覽器開發人員的擔憂。
- Brave、Vivaldi 和 Opera 等瀏覽器已承諾支援當前的廣告攔截功能，同時正在討論 Manifest V3 中可能引入的新功能。
- 使用者正在考慮切換到Firefox或其他注重隱私的瀏覽器作為替代方案，並正在討論廣告攔截器的局限性，Google對Chromium代碼庫的控制，以及對廣告攔截品質和瀏覽器性能的影響。還表達了對過度廣告的挫敗感、某些設備上缺乏廣告攔截器以及倡導用戶興趣的重要性。
