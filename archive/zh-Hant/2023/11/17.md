## [SSH3 簡介：使用 HTTP/3 和 QUIC 提供更快、更安全的 shell 體驗](https://github.com/francoismichel/ssh3)

- SSH3 是 SSH 協定的修訂版本，它使用 HTTP 機制來提供更快、更安全的shell體驗。
- 它採用 QUIC+TLS1.3 進行安全通道建立，並使用 HTTP 授權進行使用者身份驗證。
- SSH3 提供更快的會話建立、改進的身份驗證方法（OAuth2.0、OpenID Connect）、對埠掃描攻擊的魯棒性、UDP 埠轉發以及現代 QUIC 協定的功能。
- 它允許隱藏 SSH 伺服器，並支援 X.509 證書進行身份驗證。
- SSH3 目前處於實驗階段，可以通過下載二進位檔或從原始程式碼編譯代碼來安裝。
- 需要部署SSH3伺服器才能使用SSH3，SSH3用戶端支援多種認證方式。

### [反應](https://news.ycombinator.com/item?id=38664729)

- 評論線程涵蓋了一系列與 SSH 相關的主題，例如與 HTTP/3 和 QUIC 等協定的相容性、使用證書進行身份驗證以及新版本 SSH 的可能性。
- 還討論了Mosh等替代協定。
- 討論包括有關安全性、效率、實施、網路審查、協定隱藏和設備管理的各種觀點和關注點。

## [農作物營養價值下降：原因和解決方案](https://jeroenvanbaar.substack.com/p/data-dispatch-4-the-falling-nutritional)

- 在過去的六十年中，由於綠色革命、氣候變化和雜交作物的使用等因素，小麥和水稻等作物的營養價值有所下降。
- 這些因素導致作物產量較大，礦物質含量較低，導致許多人出現“微量營養素營養不良”，特別是在低收入國家。
- 為了彌補營養價值的下降，建議遵循飲食指南，食用更高比例的水果和蔬菜。然而，隨著二氧化碳水準的上升，可能需要更多數量的這些食物才能獲得足夠的礦物質。

### [反應](https://news.ycombinator.com/item?id=38664875)

- 人們擔心農作物營養價值的下降及其對人類健康的潛在影響。
- 討論的潛在解決方案包括有機農業、再生農業和使用人類排泄物作為肥料。
- 辯論還涉及單一栽培農業的影響、消費模式的變化以及宏量營養素和微量營養素在維持均衡飲食中的重要性。進一步的研究和糧食系統的複雜性被強調。

## [英特爾、三星和台積電在IEEE會議上展示3D堆疊晶體管](https://spectrum.ieee.org/cfet-intel-samsung-tsmc)

- 關於是否優先增加緩存大小或使用更大的 CPU 寄存器，潛在的解決方案包括寄存器視窗或特殊重命名，一直存在爭議。

## [三星和ASML合作開發2nm晶元](https://www.koreaherald.com/view.php?ud=20231215000518)

- 三星電子董事長李在鎔對他的荷蘭之行感到高興，在那裡他與ASML達成了一項協定，在韓國建立了一個先進的EUV技術研究機構。
- 該合作夥伴關係旨在説明三星獲得用於生產 DRAM 存儲晶片和邏輯晶片的高數值孔徑 EUV 光刻掃描器設備。

- ASML和三星合作生產2nm晶元，展示了晶元生產的進步和限制。
