# 2023-10-05

## [บทนําแบบโต้ตอบเกี่ยวกับ CRDTs](https://jakelazaroff.com/words/an-interactive-intro-to-crdts/)

- โพสต์เจาะลึกแนวคิดของประเภทข้อมูลจําลองที่ปราศจากความขัดแย้ง (CRDTs) และบทบาทของพวกเขาในการพัฒนาแอปพลิเคชันการทํางานร่วมกัน
- บล็อกมีศูนย์กลางอยู่ที่ CRDT ของรัฐและเสนอตัวอย่างเช่น Last Write Wins Register และ Last Write Wins Map
- บทความนี้อธิบายเพิ่มเติมเกี่ยวกับการใช้งานคลาสแผนที่ Last Write Wins (LWW) พร้อมตัวอย่างโค้ดที่รวมไว้สําหรับการจัดการข้อมูลในรีจิสเตอร์ LWW ซึ่งบ่งบอกถึงโพสต์ในอนาคตที่เป็นไปได้

### [ปฏิกิริยา](https://news.ycombinator.com/item?id=37764581)

- บทความนี้เจาะลึก CRDTs (ประเภทข้อมูลที่จําลองแบบที่ปราศจากความขัดแย้ง) และบทบาทในการแก้ไขข้อความและระบบเอกสารที่ทํางานร่วมกัน
- เน้นย้ําถึงความสําคัญของการทําความเข้าใจการดําเนินงานพื้นฐานของ CRDT ผ่านตัวอย่างโค้ดจริงและกล่าวถึงความท้าทายและการแลกเปลี่ยนของการใช้ CRDT ในการตั้งค่าครั้งแรกในท้องถิ่น
- ในการอภิปรายฟอรัมประกอบมีการถกเถียงกันเกี่ยวกับการใช้ CRDTs เทียบกับการแปลงการดําเนินงานในซอฟต์แวร์แก้ไขร่วมกันซึ่งสนับสนุนความเกี่ยวข้องของ CRDTs ในแอปพลิเคชันยอดนิยม

## [FTC ฟ้องร้องเพื่อเลิก Amazon เกี่ยวกับ "ภาษีที่ซ่อนอยู่" ทั่วทั้งเศรษฐกิจ](https://www.thebignewsletter.com/p/the-ftc-sues-to-break-up-amazon-over)

- คณะกรรมาธิการการค้าแห่งสหพันธรัฐ (FTC) และ 17 รัฐได้เปิดตัวคดีต่อต้านการผูกขาดกับ Amazon โดยกล่าวหาว่าเป็นการผูกขาดและการแข่งขันที่ไม่เป็นธรรมเนื่องจากโปรแกรมสมาชิกระดับไพร์มเป็นหลัก
- ข้อกล่าวหาที่ปรากฏในคําฟ้องรวมถึงค่าธรรมเนียมสูงที่เรียกเก็บจากผู้ขายบุคคลที่สามซึ่งนําไปสู่ราคาผู้บริโภคที่เพิ่มขึ้นกีดกันผู้ขายจากการเสนอราคาที่ถูกกว่าบนแพลตฟอร์มอื่น ๆ และทําให้ประสบการณ์ของผู้ใช้แย่ลง
- หากชุดสูทประสบความสําเร็จอาจส่งผลให้ราคาลดลงการแข่งขันที่สูงขึ้นและการเยียวยาที่เป็นไปได้เช่นการยุติมาตรการลดราคาหรือการแยก Amazon การร้องเรียนยังนํามาซึ่งลักษณะที่เป็นความลับของศาลต่อต้านการผูกขาดและการทําลายเอกสารของ Amazon

### [ปฏิกิริยา](https://news.ycombinator.com/item?id=37763424)

- คณะกรรมาธิการการค้าแห่งสหพันธรัฐ (FTC) กําลังฟ้อง Amazon โดยกล่าวหาว่าภาษีที่ซ่อนอยู่ผ่านรูปแบบการโฆษณาที่เพิ่มต้นทุน ในทํานองเดียวกันอัยการสูงสุดแห่งแคลิฟอร์เนียกล่าวหาว่า Amazon มีแนวทางปฏิบัติต่อต้านการแข่งขัน
- มีการหารือเกี่ยวกับประเด็นสําคัญเช่นการจัดการราคาการแข่งขันและผลกระทบ 'กล่องซื้อ' ของ Amazon ต่อทางเลือกของลูกค้าและการแข่งขัน ความคิดเห็นของ Reddit มุ่งเน้นไปที่ประโยค Most Favored Nation (MFN) ของ Amazon ซึ่งจุดประกายความกังวลเกี่ยวกับความเป็นธรรมของตลาด
- การอภิปรายยังครอบคลุมถึงการควบคุมตลาดอีคอมเมิร์ซของ Amazon ความยากลําบากในการสร้างการแข่งขันสําหรับแพลตฟอร์มเช่น Amazon นโยบายที่ต้องการราคาต่ําสุดเมื่อเทียบกับแพลตฟอร์มอื่น ๆ และการแทรกแซงของรัฐบาลที่อาจเกิดขึ้น

## [ทฤษฎีบทพีทาโกรัสที่พบบนเม็ดดินเหนียวเก่ากว่าพีทาโกรัส 1k ปี](https://link.springer.com/article/10.1057/jt.2009.16)

- ทฤษฎีบทพีทาโกรัสในขณะที่มักมาจากพีทาโกรัส อาจเป็นที่รู้จักของนักคณิตศาสตร์ชาวบาบิโลนก่อนหน้านี้
- ทฤษฎีบทนี้มีค่าสําคัญทั้งในด้านฟิสิกส์และคณิตศาสตร์ซึ่งยังคงถูกนําไปใช้ต่อไป
- นักคณิตศาสตร์ที่มีชื่อเสียงเช่น Euclid และ Pierre de Fermat ก็มีส่วนสําคัญเช่นกันโดยขยายความสําคัญและความเกี่ยวข้องของทฤษฎีบทในปัจจุบัน

### [ปฏิกิริยา](https://news.ycombinator.com/item?id=37763143)

- ตัวแก้ไขสาธารณะของ Cloudflare (1.1.1.1) และบริการ WARP กําลังเผชิญกับความท้าทายกับลิงก์ archive.is เนื่องจากการสนับสนุนส่วนขยาย DNS ที่เข้ากันไม่ได้ ดูเหมือนว่า Archive.is จะจงใจส่งคืนผลลัพธ์ที่ผิดพลาดให้กับตัวแก้ไขของ Cloudflare ซึ่งอาจกังวลเรื่องความปลอดภัย
- Cloudflare กําลังพัฒนาโซลูชันทางเลือกเพื่อแก้ไขปัญหาเหล่านี้ สถานการณ์นี้นําไปสู่การสนทนาที่กว้างขึ้นเกี่ยวกับความปลอดภัยและความเป็นส่วนตัวของ DNS รวมถึงการใช้งาน DNS-over-HTTPS (DoH)
- ประเด็นอื่น ๆ ของการอภิปราย ได้แก่ บทบาทของตัวแก้ไข DNS ในการให้การควบคุมโดยผู้ปกครองและคุณสมบัติการบล็อกโฆษณานโยบายการเก็บรักษาข้อมูลการใช้บริการแก้ไข DNS ทางเลือกและความต้องการที่สําคัญสําหรับความซ้ําซ้อนในเซิร์ฟเวอร์ DNS
