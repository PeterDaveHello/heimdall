## Извлечение заметок Apple в формате Json с помощью Liberator

- Apple Notes Liberator - это новый инструмент, который позволяет пользователям извлекать данные из Apple Notes и сохранять их в формате JSON.
- Этот инструмент помогает исследователям, использующим Apple Notes в качестве основного приложения для ведения заметок, собирать неструктурированные заметки.
- Пользователи отметили эволюцию Apple Notes от простого приложения для ведения заметок до его нынешней структуры базы данных с алгоритмами, подобными CRDT, для общих заметок и нескольких одновременных правок.
- Инструмент решает проблему Apple Notes, которая затрудняет массовый экспорт заметок или их извлечение в формат, который можно использовать вне экосистемы Notes.
- Программа сохраняет извлеченную информацию в файле notes.json, который содержит массив объектов, представляющих Apple Notes, а в будущем планируется поддержка других типов данных и форматов вывода, таких как CSV и HTML.

## Управление Blender с помощью команд на естественном языке с помощью Blendergpt

- BlenderGPT - это инструмент искусственного интеллекта, который позволяет пользователям управлять Blender'ом с помощью команд на естественном языке, генерируемых OpenAI's GPT-4, но сгенерированный код не всегда может быть правильным и требует итераций.
- Этот инструмент полезен для непрограммистов для создания простых дизайнов и анимации, но может оказаться недостаточным для более сложных и точных задач, требующих знания API Blender.
- Развитие ИИ и автоматизации может привести к автоматизации разработки программного обеспечения, при этом неквалифицированные работники могут быть заменены машинами, что поднимает вопросы о будущем работы и навыках, необходимых для успешной работы в индустрии программного обеспечения.
- Некоторые утверждают, что автоматизация определенных задач может высвободить время для более сложной и творческой работы, что принесет пользу отрасли и экономике в целом.
- Обсуждение потенциала интерфейсов естественного языка для программирования вращается вокруг отсутствия необходимой конкретики и точности, но некоторые предлагают использовать явный символьный подход или использовать LLM для сшивания более крупных блоков бизнес-логики, "санкционированных" создателем фреймворка.
- GPT от OpenAI теперь может генерировать команды Blender из запросов на естественном языке с помощью ChatGPT, потенциально революционизируя пользовательский интерфейс для проектирования в программах 3D-моделирования и других случаях использования.
- В репозитории BlenderGPT на GitHub размещено расширение, позволяющее использовать команды на естественном языке для управления Blender с помощью OpenAI's GPT-4, однако пользователям может потребоваться повторный запуск кода и действительный ключ OpenAI API.

## Руководство: Плагины Chatgpt от Openai с помощью Llama

- OpenAI выпустила поддержку плагинов для ChatGPT, позволяющих языковой модели взаимодействовать с внешним миром с помощью API.
- Это руководство демонстрирует доказательство концепции с использованием плагина chatgpt-retrieval-plugin от OpenAI с языковой моделью LLaMA от Meta.

- Команда lastmile ai изучает необходимость создания стандарта Foundation Model Plugin Protocol для стандартизации взаимодействия между моделями Foundation и плагинами/расширениями.

## Использование Sqlite для частичной дедупликации файлов с помощью Blake3

- Один из пользователей поделился своим нестандартным использованием SQLite для частичной дедупликации файлов, что вызвало дискуссию об альтернативных методах дедупликации и проблемах обработки обновлений файлов для игр.
- Возможные решения включали кластерную репликацию SQLite и определяемое содержимым разбиение на части, а также был представлен анализ дедупликации для сравнения с хранением целых файлов.
- Был проявлен интерес к возможным вариантам использования для разработчиков, обновляющих контент, и обсуждались технические детали, такие как дедупликация на уровне блоков в сравнении с дедупликацией всего файла и соображения по выравниванию для разработчиков Steamworks.
- Авторский подход, включающий использование хэш-функции BLAKE3 и сложного запроса для сопоставления похожих файлов, оказался более компактным, чем более простой вариант.
- Другие комментаторы поделились собственными нестандартными случаями использования SQLite, включая дедупликацию файлов при превышении объема памяти и хранение данных системного администрирования.
