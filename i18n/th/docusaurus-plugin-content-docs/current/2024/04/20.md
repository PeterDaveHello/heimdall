## [นําไปใช้กับ Y Combinator สําหรับ S24 - หมดเขต 22 เมษายน](https://news.ycombinator.com/item?id=40091622)

## [การเรียนรู้การเขียนโปรแกรมด้วย C++: ฉบับล่าสุดของ Stroustrup](https://www.stroustrup.com/programming.html)

- หนังสือเล่มนี้สํารวจแนวคิดการเขียนโปรแกรมพื้นฐาน เน้นการเขียนโค้ดที่มีประสิทธิภาพและบํารุงรักษาได้ และใช้มาตรฐาน C++ สมัยใหม่ เช่น C++20 และ C++23
- ครอบคลุมการเขียนโปรแกรมเชิงขั้นตอน เชิงวัตถุ และทั่วไป ตลอดจนเทคนิคการเขียนโปรแกรมเชิงปฏิบัติ คุณลักษณะภาษา และไลบรารีที่จําเป็นสําหรับการใช้งานในโลกแห่งความเป็นจริง

### [ปฏิกิริยา](https://news.ycombinator.com/item?id=40086779)

- มีการกล่าวถึงการตั้งค่าสําหรับภาษาต่างๆ เช่น C++, Python, Rust และ Swift พร้อมกับแหล่งข้อมูลการเรียนรู้ แนวทางปฏิบัติสมัยใหม่ ความจําเป็นในการเรียนรู้ C++ สําหรับการพัฒนาขั้นสูง ปัญหาด้านประสิทธิภาพ และทางเลือกอื่นเพื่อเอาชนะข้อจํากัดด้านภาษา

- MPTCP รวมแบนด์วิดท์จัดลําดับความสําคัญของเส้นทางที่มีเวลาแฝงต่ํากว่าและเปลี่ยนเส้นทางได้อย่างราบรื่นหากล้มเหลวตรวจพบผ่านตัวเลือกเพิ่มเติมในส่วนหัว TCP และดูแลโดย Path Manager และ Packet Scheduler
- รองรับการจัดการเส้นทางประเภทต่างๆ ตัวเลือกซ็อกเก็ต และคุณสมบัติการดีบัก ซึ่งดูแลโดยชุมชน MPTCP ผ่านเครื่องมือต่างๆ เช่น MPTCP daemon และ Packetdrill

### [ปฏิกิริยา](https://news.ycombinator.com/item?id=40089609)

- Multipath TCP (MPTCP) สําหรับ Linux ต้องเผชิญกับการยอมรับที่ช้าในช่วงทศวรรษที่ผ่านมา เนื่องจากความซับซ้อนในการใช้งานบนเซิร์ฟเวอร์จริงและข้อกังวลเกี่ยวกับมิดเดิลบ็อกซ์
- ความพยายามมุ่งเน้นไปที่การเพิ่มประสิทธิภาพเครือข่ายโดยใช้หลายเส้นทางสําหรับการส่งข้อมูลด้วยโซลูชันที่เป็นนวัตกรรมเช่น ECMP, Maglev, QUIC และ MPQUIC
