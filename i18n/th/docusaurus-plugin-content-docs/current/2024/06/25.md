# 2024-06-25

## [จูเลียน อัสซานจ์ได้บรรลุข้อตกลงในการรับสารภาพกับสหรัฐฯ ทำให้เขาได้รับการปล่อยตัว](https://www.nbcnews.com/politics/justice-department/julian-assange-reached-plea-deal-us-allowing-go-free-rcna158695)

- จูเลียน อัสซานจ์ ผู้ก่อตั้งวิกิลีกส์ ได้รับการปล่อยตัวจากเรือนจำในอังกฤษหลังจากถูกคุมขังเป็นเวลา 5 ปี และกำลังมุ่งหน้าไปยังหมู่เกาะนอร์เทิร์นมาเรียนาเพื่อรับสารภาพในข้อหาสมรู้ร่วมคิดตามข้อตกลงยอมความกับสหรัฐฯ
- ข้อตกลงการรับสารภาพซึ่งรวมถึงโทษจำคุก 62 เดือนโดยมีการนับเวลาที่ถูกคุมขังแล้ว จะยุติการต่อสู้ทางกฎหมายเกี่ยวกับการเผยแพร่เอกสารลับ ทำให้ Assange เป็นอิสระเมื่อได้รับการอนุมัติ
- อัสซานจ์คาดว่าจะกลับไปออสเตรเลียหลังจากกระบวนการเสร็จสิ้น โดยที่ภรรยาของเขา สเตลลา และลูกสองคนของพวกเขากำลังรอการกลับมาของเขา

### [ปฏิกิริยา](https://news.ycombinator.com/item?id=40782130)

- จูเลียน อัสซานจ์ได้บรรลุข้อตกลงในการรับสารภาพกับสหรัฐฯ ส่งผลให้เขาได้รับการปล่อยตัวจากเรือนจำเบลมาร์ชหลังจากถูกคุมขังเป็นเวลา 1901 วัน
- ผลลัพธ์นี้เกิดขึ้นจากการรณรงค์ระดับโลกและการเจรจากับกระทรวงยุติธรรมของสหรัฐอเมริกา ซึ่งนำไปสู่การออกจากสหราชอาณาจักรของเขา
- อัสซานจ์ ผู้เป็นที่รู้จักจากการเผยแพร่ข้อมูลของ WikiLeaks ที่เปิดโปงการทุจริตของรัฐบาล จะได้กลับมาพบกับครอบครัวของเขาในไม่ช้า

## [ท้องถิ่นก่อนเสมอ](https://tonsky.me/blog/crdt-filesync/)

- ซอฟต์แวร์ที่เน้นท้องถิ่นให้ความสำคัญกับการเก็บข้อมูลบนอุปกรณ์ของผู้ใช้ในขณะที่ซิงค์ออนไลน์เป็นครั้งคราว มอบการควบคุมและความเป็นเจ้าของข้อมูลให้กับผู้ใช้
- ความท้าทายรวมถึงการพึ่งพาส่วนประกอบออนไลน์สำหรับการซิงค์ ซึ่งสามารถบรรเทาได้โดยการใช้บริการคลาวด์ที่มีอยู่ทั่วไป เช่น Dropbox, iCloud Drive, OneDrive, Google Drive หรือ Syncthing
- การรวมข้อมูลที่ปราศจากความขัดแย้งสามารถทำได้โดยใช้ CRDTs (ประเภทข้อมูลที่จำลองแบบปราศจากความขัดแย้ง) โดยมีการใช้งานจริงที่แสดงให้เห็นผ่านเครื่องมือเช่น Automerge บน GitHub

### [ปฏิกิริยา](https://news.ycombinator.com/item?id=40786425)

- การสนทนามุ่งเน้นไปที่แนวคิดของซอฟต์แวร์ "local-first" ซึ่งให้ความสำคัญกับการเก็บข้อมูลไว้ในอุปกรณ์ของผู้ใช้แทนที่จะพึ่งพาบริการคลาวด์
- ประเด็นสำคัญคือความท้าทายในการสร้างรายได้จากแอปพลิเคชันที่เน้นท้องถิ่น เนื่องจากโมเดลการสมัครสมาชิกแบบดั้งเดิมไม่ค่อยเหมาะสม และนักพัฒนาต้องหาช่องทางรายได้ทางเลือกอื่น ๆ
- การสนทนานี้เน้นถึงความสำคัญของการโฮสต์เซิร์ฟเวอร์ซิงค์แบ็กเอนด์ด้วยตนเองเพื่อให้แน่ใจว่าการเข้าถึงข้อมูลในระยะยาวและการควบคุมข้อมูล โดยมีตัวอย่างของโครงการปัจจุบันและแนวทางแก้ไขที่เป็นไปได้

## [สี่เหลี่ยมมุมมนมีอยู่ทุกที่](https://www.folklore.org/Round_Rects_Are_Everywhere.html)

- สตีฟ จ็อบส์ได้สร้างแรงบันดาลใจให้บิล แอทคินสันนำรูปสี่เหลี่ยมมุมมนมาใช้ใน QuickDraw โดยชี้ให้เห็นว่ารูปทรงนี้พบได้ทั่วไปในวัตถุที่เราใช้ในชีวิตประจำวัน
- ในตอนแรก บิล แอตคินสันมีความสงสัย แต่เขาได้พัฒนาเดโมใหม่ที่สามารถวาดสี่เหลี่ยมมุมมนได้เกือบเร็วเท่ากับสี่เหลี่ยมธรรมดา ซึ่งทำให้ฟีเจอร์ "RoundRects" กลายเป็นส่วนสำคัญของอินเทอร์เฟซผู้ใช้ของแมคอินทอช
- เรื่องนี้เน้นย้ำถึงความสำคัญของการออกแบบส่วนติดต่อผู้ใช้และจิตวิญญาณแห่งนวัตกรรมที่ Apple ในช่วงการพัฒนา Macintosh รุ่นแรก

- ผู้ใช้หลายคนแสดงความคิดถึงต่อเวอร์ชันเก่าของ Windows เช่น Windows 2000 และ Windows Server 2003 โดยอ้างถึงความเรียบง่ายและความสม่ำเสมอของพวกมัน
- การสนทนาขยายไปถึงเหตุผลด้านการใช้งานและความสวยงามของมุมโค้งในบริบทต่างๆ รวมถึงสถาปัตยกรรมและการออกแบบผลิตภัณฑ์

## [Windows 11 กำลังเปิดใช้งานการสำรองข้อมูลโฟลเดอร์ OneDrive โดยไม่ต้องขออนุญาต](https://www.neowin.net/news/windows-11-is-now-automatically-enabling-onedrive-folder-backup-without-asking-permission/)

### [ปฏิกิริยา](https://news.ycombinator.com/item?id=40781048)

- Windows 11 ตอนนี้เปิดใช้งานการสำรองข้อมูลโฟลเดอร์ OneDrive โดยอัตโนมัติโดยไม่ได้รับความยินยอมจากผู้ใช้ ทำให้เกิดปัญหาต่างๆ เช่น ความเร็วอินเทอร์เน็ตช้าลงและเส้นทางไดเรกทอรีเปลี่ยนแปลง
- ผู้ใช้พบว่าการย้อนกลับการเปลี่ยนแปลงเหล่านี้เป็นเรื่องท้าทาย ทำให้เกิดความหงุดหงิดและความไม่ไว้วางใจใน Microsoft มากขึ้น โดยบางคนกำลังพิจารณาทางเลือกอื่นเช่น Linux หรือ macOS
- การสำรองข้อมูลอัตโนมัติทำให้เกิดความกังวลเกี่ยวกับความเป็นส่วนตัว โดยเฉพาะอย่างยิ่งสำหรับข้อมูลที่มีความละเอียดอ่อน ทำให้ผู้ใช้รายงานการปฏิบัติเหล่านี้ต่อหน่วยงานเช่น FTC

## [เหตุผลที่ไม่ควรทำให้เทพนิยายสะอาดเกินไป](https://www.plough.com/en/topics/culture/literature/the-case-for-not-sanitizing-fairy-tales)

- เฮลีย์ สจ๊วต โต้แย้งว่าการทำให้เรื่องราวเหล่านี้สะอาดสะอ้าน อย่างที่เห็นในเวอร์ชันสมัยใหม่เช่นเวอร์ชันของดิสนีย์ ทำให้เด็กๆ ขาดโอกาสในการประมวลผลความกลัวของพวกเขาและเข้าใจว่าความดีสามารถชนะได้
- สจ๊วตสนับสนุนการรักษาเวอร์ชันดั้งเดิมที่ไม่ถูกแก้ไขของนิทานเพื่อช่วยให้เด็กๆ พัฒนาความยืดหยุ่นและสามารถรับมือกับความซับซ้อนของโลกได้

- การดัดแปลงของดิสนีย์ เช่น "The Little Mermaid" มักถูกยกมาเป็นตัวอย่างของการปรับปรุงเรื่องราวเก่าให้ทันสมัย ซึ่งก่อให้เกิดการถกเถียงว่าการเปลี่ยนแปลงเหล่านี้จำเป็นหรือเป็นการเซ็นเซอร์
- ผู้สนับสนุนโต้แย้งว่าการพัฒนาเรื่องราวสามารถทำให้เข้าถึงได้ง่ายขึ้นและมีผลกระทบมากขึ้นสำหรับผู้ชมสมัยใหม่ ในขณะที่นักวิจารณ์กังวลเกี่ยวกับการเจือจางเจตนาเดิมและความสำคัญทางวัฒนธรรมของเรื่องราวเหล่านั้น

## [การเร่งความเร็วการประมวลผลภาพด้วย SIMD บนไมโครคอนโทรลเลอร์ราคา $2](https://shraiwi.github.io/read.html?md=blog/simd-fast-esp32s3.md)

- การอภิปรายยังครอบคลุมถึงศักยภาพของ TPU (Tensor Processing Units) ที่มีราคาย่อมเยา, บอร์ด FPGA (Field-Programmable Gate Array), และการใช้ภาษาโปรแกรม Rust สำหรับการพัฒนา ESP32

## [iDOS 3 ถูกปฏิเสธโดย Apple](https://litchie.com/2024/04/new-hope)

- Apple ได้ขึ้นบัญชีดำ iDOS 2 ทำให้นักพัฒนาต้องส่งมันใหม่ในชื่อ "iDOS 3" ซึ่งถูกปฏิเสธในตอนแรกเนื่องจากมีความคล้ายคลึงกับรุ่นก่อนหน้าเกินไปและเนื่องจาก "การออกแบบที่เป็นสแปม"
- แม้จะมีคำอธิบายจากนักพัฒนา แต่ Apple ยังไม่ได้แก้ไขปัญหา และแอปยังคงอยู่ในระหว่างการตรวจสอบ โดย Apple ให้แนวทางที่คลุมเครือเกี่ยวกับสิ่งที่ถือว่าเป็นเครื่องเล่นเกมย้อนยุค
- นักพัฒนากำลังรอการอัปเดตเพิ่มเติมและหวังว่าจะสามารถชี้แจงการเพิ่มเวอร์ชันให้กับผู้ใช้ได้ในเร็วๆ นี้

### [ปฏิกิริยา](https://news.ycombinator.com/item?id=40782541)

- แอปเปิลปฏิเสธแอป iDOS 3 โดยอ้างว่าไม่ใช่เครื่องเล่นเกมย้อนยุค และให้คำแนะนำที่ไม่ชัดเจนสำหรับการเปลี่ยนแปลงที่จำเป็น
