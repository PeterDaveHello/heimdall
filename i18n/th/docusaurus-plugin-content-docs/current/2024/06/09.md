# 2024-06-09

## [ยีนบําบัดฟื้นฟูการได้ยินในเด็กที่หูหนวกที่สืบทอดมา](https://cosmosmagazine.com/science/biology/gene-therapy-restores-hearing-to-children-with-inherited-deafness/)

- การทดลองทางคลินิกประสบความสําเร็จในการฟื้นฟูการได้ยินในเด็ก 5 คนที่มีอาการหูหนวกที่สืบทอดมาโดยใช้ยีนบําบัด โดยเฉพาะการส่งสําเนาการทํางานของยีน OTOF ผ่านไวรัสที่เกี่ยวข้องกับอะดีโน (AAV)
- มีการปรับปรุงที่สําคัญในการได้ยินการรับรู้คําพูดและการแปลเสียงโดยเด็กสองคนพัฒนาความซาบซึ้งในดนตรี
- ตีพิมพ์ใน Nature Medicine การศึกษาเน้นย้ําถึงศักยภาพของแนวทางนี้และสนับสนุนการทดลองระดับนานาชาติที่ใหญ่ขึ้นโดยสังเกตความปลอดภัยและประสิทธิผลของการบําบัดแม้จะมีเหตุการณ์ไม่พึงประสงค์เล็กน้อย

### [ปฏิกิริยา](https://news.ycombinator.com/item?id=40620056)

- การบําบัดด้วยยีนได้ฟื้นฟูการได้ยินในเด็กที่หูหนวกที่สืบทอดมาโดยการฉีดไวรัสที่เกี่ยวข้องกับอะดีโนที่มียีน OTOF ที่ใช้งานได้เข้าไปในหูชั้นใน เพื่อให้แน่ใจว่ามีการแสดงออกของยีนในระยะยาว
- การอภิปรายสํารวจศักยภาพของเทคโนโลยีการแก้ไขยีน เช่น CRISPR สําหรับการดัดแปลงของมนุษย์ที่รุนแรง และผลกระทบทางจริยธรรมของการจัดลําดับความสําคัญของพันธุวิศวกรรมเครื่องสําอางมากกว่าการใช้งานทางการแพทย์
- มีข้อกังวลด้านจริยธรรมเกี่ยวกับการใช้ยีนบําบัดเพื่อกําจัดอาการหูหนวกทางพันธุกรรมในเด็ก โดยเน้นที่ความยินยอมและการโต้เถียงภายในชุมชนคนหูหนวกเกี่ยวกับประสาทหูเทียม

## [Piku: ลดความซับซ้อนของการปรับใช้ Git Push สําหรับเซิร์ฟเวอร์ส่วนบุคคลในหลายภาษา](https://github.com/piku/piku)

- Piku ซึ่งได้รับแรงบันดาลใจจาก Dokku อํานวยความสะดวกในการปรับใช้ git push ไปยังเซิร์ฟเวอร์ส่วนบุคคล โดยรองรับเวิร์กโฟลว์ที่เหมือน Heroku สําหรับภาษาโปรแกรมต่างๆ (Python, Node, Go, Java, Clojure, Ruby)
- สามารถปรับใช้ จัดการ และปรับขนาดแอปพลิเคชันหลายตัวทั้งบนสถาปัตยกรรม ARM และ Intel เข้ากันได้กับผู้ให้บริการคลาวด์หรือการตั้งค่าแบบ Bare Metal โดยใช้ Python, nginx และ uwsgi
- Piku มีความเสถียร บํารุงรักษาอย่างแข็งขัน และออกแบบมาสําหรับอุปกรณ์ระดับล่าง มือสมัครเล่น และการใช้งานด้านการศึกษา โดยเน้นการพึ่งพา ความเรียบง่าย และความเข้ากันได้น้อยที่สุด

### [ปฏิกิริยา](https://news.ycombinator.com/item?id=40622704)

- การสนทนาเน้นย้ําถึง Piku ซึ่งเป็นเครื่องมือที่เรียบง่ายสําหรับการปรับใช้ Git push ไปยังเซิร์ฟเวอร์ส่วนบุคคล โดยเปรียบเทียบกับ Heroku, Dokku และ Kubernetes เนื่องจากความเรียบง่ายและใช้งานง่าย
- บทช่วยสอนสําหรับ Piku ซึ่งได้รับการยกย่องในด้านความชัดเจนตอนนี้เป็นส่วนหนึ่งขององค์กร Piku GitHub อย่างเป็นทางการโดยมีแผนสําหรับการปรับปรุงเพิ่มเติม
- ผู้ใช้พูดคุยเกี่ยวกับการตั้งค่าการปรับใช้ต่างๆ รวมถึง systemd nspawn สําหรับ sandboxing และ NixOS เพื่อความเสถียร และจัดการกับข้อกังวลเกี่ยวกับการจัดการการพึ่งพา ความปลอดภัย และค่าใช้จ่ายของ Docker ซึ่งชอบโซลูชัน PaaS ที่มีน้ําหนักเบาและปรับแต่งได้

## [ไวอากร้าแสดงให้เห็นถึงคํามั่นสัญญาในการปรับปรุงการไหลเวียนของเลือดในสมองเพื่อป้องกันภาวะสมองเสื่อม](https://www.ox.ac.uk/news/2024-06-07-sildenafil-viagra-improves-brain-blood-flow-and-could-help-prevent-dementia)

- การศึกษาของมหาวิทยาลัยออกซ์ฟอร์ดที่ตีพิมพ์ใน Circulation Research ระบุว่าซิลเดนาฟิล (ไวอากร้า) ช่วยเพิ่มการไหลเวียนของเลือดในสมองและการทํางานในผู้ป่วยที่มีความเสี่ยงต่อการเกิดภาวะสมองเสื่อมจากหลอดเลือด
- การทดลอง OxHARP ซึ่งเป็นการศึกษาแบบอําพรางสองฝ่ายที่ควบคุมด้วยยาหลอกกับผู้เข้าร่วม 75 คนพบว่าซิลเดนาฟิลช่วยเพิ่มการทํางานของหลอดเลือดสมองโดยมีผลข้างเคียงน้อยกว่าเมื่อเทียบกับ cilostazol
- การค้นพบนี้ชี้ให้เห็นถึงศักยภาพของซิลเดนาฟิลในการป้องกันภาวะสมองเสื่อมจากหลอดเลือด

### [ปฏิกิริยา](https://news.ycombinator.com/item?id=40620739)

- การศึกษาเมื่อเร็ว ๆ นี้ชี้ให้เห็นว่าไวอากร้าในขั้นต้นสําหรับปัญหาหัวใจและหลอดเลือดและการหย่อนสมรรถภาพทางเพศ (ED) อาจช่วยเพิ่มการไหลเวียนของเลือดในสมองและช่วยป้องกันภาวะสมองเสื่อม
- ผู้ใช้รายงานผลข้างเคียงที่หลากหลาย เช่น ปวดศีรษะ คลื่นไส้ และคัดจมูก ซึ่งมักจะลดลงเมื่อเวลาผ่านไป และเน้นย้ําถึงความจําเป็นของการกระตุ้นทางเพศเพื่อให้ยาทํางาน
- การอภิปรายครอบคลุมถึงประโยชน์ต่อสุขภาพในวงกว้างความคาดเดาไม่ได้ของการวิจัยยาและความจําเป็นในการศึกษาอย่างเข้มงวดเพื่อยืนยันประโยชน์สําหรับเงื่อนไขเช่นอัลไซเมอร์และภาวะสมองเสื่อมจากหลอดเลือด

## [Ruxandra Teslo สนับสนุน 'Weird Nerds' ในสถาบันการศึกษาท่ามกลางฟันเฟือง Karikó](https://www.writingruxandrabio.com/p/the-weird-nerd-comes-with-trade-offs)

- โดยเน้นถึงประโยชน์ของเซิร์ฟเวอร์ถาวรในเกมอย่าง "Squad" และ "Counter-Strike" สําหรับการสร้างความสัมพันธ์และให้เครดิตบล็อกของ Glenn Fiedler สําหรับการใช้งานเน็ตโค้ดที่ประสบความสําเร็จโดยใช้ WebRTC และ enet
- โพสต์ให้รายละเอียดเกี่ยวกับเทคนิคการจับคู่ที่เกี่ยวข้องกับเน็ตโค้ดที่เชื่อถือได้ของเซิร์ฟเวอร์พร้อมการคาดคะเนฝั่งไคลเอ็นต์และการย้อนกลับ โดยเน้นการเชื่อมต่อที่มีเวลาแฝงต่ํา และกล่าวถึงความท้าทายในการสร้างสมดุลระหว่างระดับทักษะและเวลาแฝง ซอร์สโค้ดมีอยู่ใน GitHub พร้อมตัวเลือกในการปิดใช้งานการสับเปลี่ยนผู้เล่น

## [ทีมตัวแทน LLM มีประสิทธิภาพเหนือกว่าในการใช้ประโยชน์จากช่องโหว่ซีโร่เดย์ด้วยระบบ HPTSA](https://arxiv.org/abs/2406.01637)

- บทความ "ทีมตัวแทน LLM สามารถใช้ประโยชน์จากช่องโหว่ซีโร่เดย์" โดย Richard Fang และคณะ ตรวจสอบการใช้เอเจนต์โมเดลภาษาขนาดใหญ่ (LLM) ในการรักษาความปลอดภัยทางไซเบอร์ โดยเน้นที่ช่องโหว่ซีโร่เดย์
- ผู้เขียนแนะนํา HPTSA ซึ่งเป็นระบบที่ตัวแทนการวางแผนประสานงานตัวแทนย่อยเพื่อเอาชนะข้อจํากัดของตัวแทน LLM แต่ละรายการในการจัดการช่องโหว่ที่ไม่รู้จักและการวางแผนระยะยาว
- การเปรียบเทียบช่องโหว่ในโลกแห่งความเป็นจริง 15 จุดการศึกษาแสดงให้เห็นว่าวิธีการแบบทีมของ HPTSA มีประสิทธิภาพเหนือกว่าวิธีการก่อนหน้านี้อย่างมีนัยสําคัญโดยเพิ่มประสิทธิภาพได้ถึง 4.5 เท่า

### [ปฏิกิริยา](https://news.ycombinator.com/item?id=40624563)

- การอภิปรายจะตรวจสอบจุดแข็งและจุดอ่อนของโมเดลภาษาขนาดใหญ่ (LLM) เช่น Claude-opus และ ChatGPT ในการระบุช่องโหว่ซีโร่เดย์ในโค้ด
- แม้ว่า LLM จะสามารถระบุปัญหาทางเทคนิคที่ถูกต้องได้ แต่ความเกี่ยวข้องในวงกว้างก็ยังเป็นที่ถกเถียงกัน และเครื่องมือพิเศษและความรู้จากผู้เชี่ยวชาญยังคงจําเป็นสําหรับช่องโหว่ที่ซับซ้อน
- ข้อกังวลด้านจริยธรรมและกฎหมาย ค่าใช้จ่ายสูง และปัญหาด้านการปฏิบัติจริง โดยเฉพาะอย่างยิ่งสําหรับโครงการโอเพ่นซอร์ส เน้นย้ําถึงความสงสัยเกี่ยวกับมูลค่าเพิ่มของ LLM เหนือเทคนิคที่มีอยู่ในความปลอดภัยทางไซเบอร์
