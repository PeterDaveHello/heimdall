## [Super Heavy ของ SpaceX ลงจอดในอ่าวเม็กซิโกได้สําเร็จ](https://twitter.com/SpaceX/status/1798701489097183286)

- SpaceX รายงานว่าจรวด Super Heavy ลงจอดในอ่าวเม็กซิโกได้สําเร็จเมื่อวันที่ 6 มิถุนายน พ.ศ. 2024
- นี่นับเป็นก้าวสําคัญสําหรับ SpaceX ซึ่งแสดงให้เห็นถึงความสามารถในการกู้คืนและนําบูสเตอร์จรวดที่ใหญ่ที่สุดกลับมาใช้ใหม่
- การลงจอดที่ประสบความสําเร็จคาดว่าจะทําให้เป้าหมายของ SpaceX ก้าวหน้าสําหรับการเดินทางในอวกาศที่คุ้มค่าและภารกิจเพิ่มเติมไปยังดาวอังคาร

### [ปฏิกิริยา](https://news.ycombinator.com/item?id=40596883)

- ความก้าวหน้าทางเทคโนโลยีล่าสุดของ SpaceX รวมถึงการสาดน้ําที่ประสบความสําเร็จของ Super Heavy ในอ่าวเม็กซิโกและความคาดหวังของศูนย์ปล่อยจรวดแห่งใหม่ในฟลอริดาตอนกลาง
- การอภิปรายเน้นย้ําถึงความน่าเชื่อถือของอินเทอร์เน็ต Starlink ระหว่างการกลับเข้าสู่อวกาศ การลงจอดของจรวดที่แม่นยํา และความยืดหยุ่นของ Starship ระหว่างการกลับเข้ามาใหม่ แม้ว่าจะมีความเสียหายอยู่บ้าง
- การสนทนาครอบคลุมถึงความเป็นไปได้ทางเศรษฐกิจของการปรับใช้ดาวเทียมความสําคัญของยานปล่อยขนาดใหญ่และตลาดในอนาคตเช่นการท่องเที่ยวในอวกาศและการผลิตที่ไม่มีแรงโน้มถ่วง

## [จาก OpenGL ถึง Vulkan: การสร้างเอ็นจิ้นเกมแบบกําหนดเองในสามเดือน](https://edw.is/learning-vulkan/)

- ผู้เขียนบันทึกการเดินทางสามเดือนของการเรียนรู้ Vulkan และพัฒนาเอ็นจิ้นเกมขนาดเล็ก แม้ว่าจะไม่มีประสบการณ์ Vulkan มาก่อน แต่มีพื้นฐานบางอย่างใน OpenGL และเอ็นจิ้นเกม
- เอ็นจิ้นนี้เหมาะสําหรับเกมระดับเล็ก ๆ ประกอบด้วยการสาธิตสองรายการและมีอยู่ใน GitHub พร้อมคําแนะนําที่เป็นประโยชน์สําหรับผู้เริ่มต้นที่เปลี่ยนจาก OpenGL เป็น Vulkan
- บทความนี้ครอบคลุมด้านเทคนิค เช่น เฉดสี การซิงโครไนซ์ และการรวม UI และกล่าวถึงข้อดีของ Vulkan สําหรับเกมเดสก์ท็อป 3 มิติขนาดเล็ก ในขณะเดียวกันก็สะท้อนถึงอนาคตที่ไม่แน่นอนของ OpenGL และศักยภาพของ WebGPU

### [ปฏิกิริยา](https://news.ycombinator.com/item?id=40595741)

- บทความนี้สํารวจประสบการณ์ของผู้เขียนในการเรียนรู้ Vulkan และพัฒนาเอ็นจิ้นเกม โดยให้คําแนะนําการเขียนโปรแกรมเชิงปฏิบัติและเน้นหลักการ YAGNI (You Aren't Gonna Need It)
- เน้นย้ําถึงความท้าทายของการใช้ Rust กับ Vulkan, WGPU และ Rend3 รวมถึงปัญหาด้านประสิทธิภาพ
- การอภิปรายครอบคลุมความซับซ้อนของบทช่วยสอน Vulkan เปรียบเทียบ API กราฟิกต่างๆ เช่น OpenGL และ Direct3D และตรวจสอบประโยชน์ของสิ่งที่เป็นนามธรรมและเครื่องมือระดับสูง โดยสังเกตช่วงการเรียนรู้ที่สูงชันของ Vulkan แต่มีคุณสมบัติขั้นสูง

## [หน่วยงานกํากับดูแลของสหรัฐฯ เปิดตัว Antitrust Probes ใน Microsoft, OpenAI และ Nvidia](https://www.nytimes.com/2024/06/05/technology/nvidia-microsoft-openai-antitrust-doj-ftc.html)

- FTC จะมุ่งเน้นไปที่ Microsoft และ OpenAI ในขณะที่กระทรวงยุติธรรมจะตรวจสอบ Nvidia ซึ่งส่งสัญญาณถึงการตรวจสอบด้านกฎระเบียบที่เพิ่มขึ้นของเทคโนโลยี AI
- การสืบสวนเหล่านี้มีจุดมุ่งหมายเพื่อจัดการกับการละเมิดการต่อต้านการผูกขาดที่อาจเกิดขึ้นและอันตรายต่อผู้บริโภค ซึ่งสะท้อนถึงความพยายามที่เพิ่มขึ้นในสหรัฐอเมริกาในการดูแลอุตสาหกรรม AI ที่ก้าวหน้าอย่างรวดเร็ว

- การสืบสวนเน้นย้ําถึงความซับซ้อนของกฎหมายต่อต้านการผูกขาดความจําเป็นในการปรับปรุงและผลกระทบที่กว้างขึ้นสําหรับกฎระเบียบของอุตสาหกรรมเทคโนโลยีและการคุ้มครองผู้บริโภค
