# 2024-06-20

## [สภาสหภาพยุโรปได้ถอนการลงคะแนนเกี่ยวกับการควบคุมแชท](https://stackdiary.com/eu-council-has-withdrawn-the-vote-on-chat-control/)

- บทความนี้กล่าวถึงเว็บเบราว์เซอร์ชั้นนำที่ให้ความสำคัญกับความเป็นส่วนตัวและความปลอดภัยของผู้ใช้ โดยเน้นคุณสมบัติและประโยชน์ของพวกเขา
- มันให้การเปรียบเทียบของเบราว์เซอร์เช่น Brave, Firefox, และ Tor ซึ่งเป็นที่รู้จักกันดีในเรื่องการปกป้องความเป็นส่วนตัวและมาตรการความปลอดภัยที่แข็งแกร่ง
- จุดสนใจอยู่ที่วิธีที่เบราว์เซอร์เหล่านี้จัดการข้อมูลผู้ใช้, บล็อกตัวติดตาม, และเสนอคุณสมบัติความปลอดภัยที่เพิ่มขึ้นเพื่อป้องกันภัยคุกคามออนไลน์

### [ปฏิกิริยา](https://news.ycombinator.com/item?id=40736771)

## [เททริสเป็นฟอนต์](https://erikdemaine.org/fonts/tetris/)

- Erik และ Martin Demaine ได้ออกแบบฟอนต์ที่ได้รับแรงบันดาลใจจากเกม Tetris โดยที่แต่ละตัวอักษรและตัวเลขถูกสร้างขึ้นจากชิ้นส่วนของ Tetris (I, J, L, O, S, T, Z).
- การออกแบบฟอนต์ใช้ซอฟต์แวร์ BurrTools และปฏิบัติตามกฎของเกม Tetris รวมถึงระบบ Super Rotation System สำหรับการหมุนชิ้นส่วนเริ่มต้น
- ฟอนต์นี้เป็นส่วนหนึ่งของการวิเคราะห์ทางคณิตศาสตร์ของเกม Tetris ที่เน้นถึงลักษณะ NP-complete ของมัน และได้รับแรงบันดาลใจจากการร่วมมือกันในงาน BRIDGES 2017

### [ปฏิกิริยา](https://news.ycombinator.com/item?id=40737294)

- Erik Demaine ได้สร้างฟอนต์ที่แต่ละตัวอักษรและตัวเลขประกอบด้วยชิ้นส่วนของเกม Tetris โดยออกแบบด้วยตนเองโดยใช้ซอฟต์แวร์ BurrTools ช่วย
- มีการแชร์การสาธิตฟอนต์ที่เล่นเกม Tetris โดยใช้ Harfbuzz shaper กับ WebAssembly ซึ่งแสดงให้เห็นการใช้เทคโนโลยีเว็บอย่างสร้างสรรค์
- โครงการนี้ได้รับความสนใจและคำติชมเชิงบวกอย่างมากจากชุมชนเทคโนโลยี โดยเน้นถึงความแปลกใหม่และการผสมผสานที่สร้างสรรค์ระหว่างการออกแบบตัวอักษรและการเล่นเกม

## [โซเน็ตของโคลด 3.5](https://www.anthropic.com/news/claude-3-5-sonnet)

- โมเดลนี้มีความเชี่ยวชาญในด้านการให้เหตุผลระดับบัณฑิตศึกษา ความสามารถในการเขียนโค้ด และการให้เหตุผลเชิงภาพ ทำให้เหมาะสมสำหรับงานที่ซับซ้อน เช่น การสนับสนุนลูกค้าและการทำงานหลายขั้นตอน และยังมีการแนะนำคุณสมบัติใหม่ ๆ เช่น Artifacts สำหรับการสร้างและแก้ไขเนื้อหาแบบไดนามิก
- ความปลอดภัยและความเป็นส่วนตัวได้รับการให้ความสำคัญ โดยมีการทดสอบอย่างเข้มงวดและการประเมินจากภายนอก และแผนในอนาคตรวมถึงการเปิดตัว Claude 3.5 Haiku และ Claude 3.5 Opus พร้อมกับฟีเจอร์ใหม่ ๆ เช่น Memory สำหรับประสบการณ์ที่ปรับแต่งได้

## [X เปิดตัวเมื่อ 40 ปีที่แล้ว (1984)](https://www.talisman.org/x-debut.shtml)

- ระบบ X Window ซึ่งพัฒนาโดย Robert W. Scheifler เปิดตัวเมื่อวันที่ 19 มิถุนายน 1984 และถูกออกแบบมาสำหรับ VS100 โดยมีความเร็วเป็นสองเท่าของรุ่นก่อนหน้า W.
- ห้องปฏิบัติการวิทยาการคอมพิวเตอร์ (LCS) ได้ยอมรับ X โดยสร้างแอปพลิเคชันบนมัน และสนับสนุนให้ผู้อื่นที่ใช้ W เปลี่ยนมาใช้ X เพื่อประสิทธิภาพและการทดลองที่ดียิ่งขึ้น
- อินเทอร์เฟซปัจจุบันประกอบด้วย CLU และ Argus โดยมีอินเทอร์เฟซ C กำลังพัฒนา; แอปพลิเคชันที่มีอยู่ได้แก่ โปรแกรมแก้ไขข้อความ (TED), อินเทอร์เฟซ I/O ของ Argus และตัวจัดการหน้าต่างพื้นฐาน แม้ว่าจะยังต้องการเอกสารประกอบอยู่ก็ตาม

### [ปฏิกิริยา](https://news.ycombinator.com/item?id=40731922)

- X11, ระบบหน้าต่างสำหรับการแสดงผลแบบบิตแมป, ฉลองครบรอบ 40 ปี โดยเปิดตัวครั้งแรกในปี 1984.
- ผู้ใช้ได้แบ่งปันประสบการณ์ที่น่าคิดถึง ตั้งแต่การกำหนดค่าโมเดลไลน์และการจัดการระบบ Unix ไปจนถึงวิวัฒนาการของห้องปฏิบัติการคอมพิวเตอร์และการเปลี่ยนไปใช้ Linux
- การอภิปรายเน้นย้ำถึงการลดลงของห้องปฏิบัติการคอมพิวเตอร์แบบดั้งเดิมเนื่องจากการเพิ่มขึ้นของแล็ปท็อปราคาย่อมเยาและผลกระทบที่สำคัญของ X11 ต่ออาชีพหลาย ๆ คน

## [ระบบไฮเปอร์มีเดีย](https://hypermedia.systems/)

- htmx และ Hyperview เสนอวิธีการปฏิวัติในการทำให้การพัฒนาเว็บแอปพลิเคชันง่ายขึ้น โดยไม่จำเป็นต้องใช้เฟรมเวิร์กของ Single Page Application (SPA)
- เครื่องมือเหล่านี้เหมาะสำหรับนักพัฒนาเว็บที่ต้องการเน้นความเรียบง่าย เสริมความเข้าใจในพื้นฐานของเว็บ หรือสำรวจไฮเปอร์มีเดียและ REST (Representational State Transfer)
- มีทรัพยากรออนไลน์ให้ใช้ฟรี พร้อมตัวเลือกในการซื้อสำเนาฉบับพิมพ์หรืออีบุ๊คบน Amazon

### [ปฏิกิริยา](https://news.ycombinator.com/item?id=40733160)

- เอกสาร HTMX มีประโยชน์สำหรับการอ้างอิง แต่ขาดบทเรียนที่ครอบคลุม ซึ่งหนังสือเล่มนี้มุ่งหวังที่จะให้
- แนะนำให้ใช้ HTMX สำหรับเครื่องมือภายในและเว็บไซต์ที่มีความไดนามิกเล็กน้อย แต่ไม่เหมาะสำหรับการใช้งานบนมือถือที่ยาวนานเนื่องจากการปรับแต่งของเบราว์เซอร์
- การผสมผสานระหว่าง JavaScript (JS) แบบวานิลลาหรือ Alpine.js สามารถเพิ่มประสิทธิภาพให้กับเว็บไซต์ HTMX ได้ และบางครั้งการใช้เฟรมเวิร์กเต็มรูปแบบอย่าง Vue หรือ React ก็จำเป็นสำหรับส่วนที่มีความไดนามิกสูง

## [ซอฟต์แวร์ฟรีได้เข้าครอบงำชีวิตของ Philip Hazel](https://lwn.net/SubscriberLink/978463/608c876c1153fd31/)

- ฟิลิป เฮเซล ผู้สร้าง Exim MTA และ PCRE ได้ดูแล PCRE และผู้สืบทอดของมัน PCRE2 มานานกว่า 27 ปี และขณะนี้กำลังมองหาผู้สืบทอดสำหรับ PCRE2
- ไฮไลท์ในอาชีพของเฮเซลรวมถึงการมีส่วนร่วมที่สำคัญในระบบ Unix และการพัฒนาซอฟต์แวร์ที่ได้รับการยอมรับอย่างกว้างขวาง เช่น Exim และ PCRE ซึ่งถูกใช้ในโครงการใหญ่ ๆ เช่น Apache HTTPD และ Postfix
- ชุมชนชื่นชมบทความและเรียกร้องให้มีการนำเสนอเพิ่มเติมเกี่ยวกับนักพัฒนาซอฟต์แวร์โอเพนซอร์ส โดยเน้นถึงผลกระทบที่เป็นรูปธรรมและความท้าทายในการดูแลรักษาโครงการดังกล่าว

### [ปฏิกิริยา](https://news.ycombinator.com/item?id=40736577)

- ประสบการณ์ของ Philip Hazel เน้นย้ำถึงความท้าทายในการหาผู้ดูแลใหม่สำหรับโครงการซอฟต์แวร์เสรี โดยเฉพาะอย่างยิ่งสำหรับโครงการที่ได้รับการดูแลอย่างดีเช่น PCRE2
- ความกังวลรวมถึงผู้ดูแลใหม่ที่อาจเพิ่มฟีเจอร์ที่ไม่จำเป็นและการขาดแรงจูงใจในการดูแลซอฟต์แวร์ที่ "เสร็จสมบูรณ์" ซึ่งนำไปสู่การหมดไฟและความเครียดของอาสาสมัคร
- การอภิปรายยังกล่าวถึงความเป็นไปได้ที่หน่วยข่าวกรองจะมุ่งเป้าไปที่โครงการโอเพนซอร์สและความยากลำบากในการตรวจสอบผู้ดูแลใหม่

## [ไม่มีใครรู้ว่าเกิดอะไรขึ้น](https://www.raptitude.com/2024/06/nobody-knows-whats-going-on/)

- ความผิดพลาดของนักข่าวเกี่ยวกับการเกษียณอายุของผู้เขียนเมื่ออายุ 33 ปี แสดงให้เห็นว่าข้อมูลที่ผิดพลาดสามารถแพร่กระจายได้ง่ายเพียงใด โดยเฉพาะทางออนไลน์
- ปรากฏการณ์ Gell-Mann Amnesia อธิบายว่าทำไมผู้คนถึงเชื่อถือสื่อในหัวข้อที่ไม่คุ้นเคยแม้จะรับรู้ถึงความไม่ถูกต้องในพื้นที่ที่คุ้นเคย
- ความรู้ที่เชื่อถือได้มักจะหายากและมาจากผู้ที่มีความทุ่มเทในสาขาเฉพาะ แต่โดยทั่วไปแล้วมนุษย์มักถูกขับเคลื่อนด้วยแรงจูงใจทางสังคมและอารมณ์มากกว่าความถูกต้อง

### [ปฏิกิริยา](https://news.ycombinator.com/item?id=40733615)

- การสนทนานี้เน้นปรากฏการณ์ที่ผู้คนตระหนักถึงความไม่ถูกต้องของบทความในหัวข้อที่พวกเขารู้ดี ซึ่งนำไปสู่ความสงสัยเกี่ยวกับบทความในหัวข้อที่พวกเขาไม่รู้
- สิ่งนี้มักถูกเรียกว่าเอฟเฟกต์ Gell-Mann Amnesia ซึ่งผู้คนลืมความไม่ถูกต้องในบทความหนึ่งเมื่ออ่านบทความอื่นในสิ่งพิมพ์เดียวกัน
- การสนทนายังครอบคลุมถึงผลกระทบที่กว้างขึ้นของข้อมูลที่ผิดพลาดและความท้าทายในการแยกแยะข้อมูลที่ถูกต้องในสื่อและการสนทนาออนไลน์
