## [การทดแทน SQS แบบ Drop-In โดยใช้ SQLite](https://github.com/poundifdef/SmoothMQ)

- มีการสร้างโปรแกรมทดแทน Amazon SQS (Simple Queue Service) ที่เป็นโอเพนซอร์สและเข้ากันได้กับ API โดยเขียนด้วยภาษา Go และแจกจ่ายเป็นไฟล์ไบนารีเดียวโดยใช้ SQLite สำหรับการจัดเก็บข้อมูล
- โซลูชันนี้มีฟีเจอร์ขั้นสูงเช่น การค้นหา การตั้งเวลา การสังเกตการณ์ และการจำกัดอัตรา และสามารถใช้งานร่วมกับแอปพลิเคชัน SQS ที่มีอยู่ได้โดยเพียงแค่เปลี่ยนจุดสิ้นสุด
- มันรองรับโปรโตคอลหลายแบบ (AMQP, PubSub) และสามารถปรับให้เข้ากับระบบจัดเก็บข้อมูลต่างๆ (RocksDB, Postgres) โดยมีแผนในอนาคตสำหรับการทำงานแบบกระจายและการปรับขนาดอัตโนมัติ และมุ่งหวังที่จะเสนอระบบคิวที่มีค่าใช้จ่ายต่ำ

### [ปฏิกิริยา](https://news.ycombinator.com/item?id=40837610)

- มีการพัฒนาทางเลือกแบบโอเพนซอร์สที่เข้ากันได้กับ API ของ Amazon SQS โดยใช้ Go และ SQLite ซึ่งมีคุณสมบัติเช่น การค้นหา การตั้งเวลา การสังเกตการณ์ และการจำกัดอัตรา โดยไม่ต้องเขียนแอปใหม่
- โซลูชันนี้รองรับ Celery, ให้การมองเห็นข้อความที่ดีกว่า SQS, และสามารถเชื่อมต่อกับ backend ที่เป็นไปได้เช่น RocksDB หรือ Postgres, ซึ่งสามารถใช้งานโปรโตคอลหลายแบบได้
- มันง่ายต่อการปรับใช้เป็นไบนารี Go เดี่ยว โดยมีแผนในอนาคตสำหรับการกระจายและการปรับขนาดอัตโนมัติ และยินดีรับฟังความคิดเห็น

## [ไทรโลไบต์ถูกฆ่าโดยเถ้าภูเขาไฟ](https://www.bristol.ac.uk/news/2024/june/prehistoric-pompeii.html)

- ฟอสซิลที่ถูกเก็บรักษาโดยเถ้าภูเขาไฟเมื่อกว่า 500 ล้านปีก่อน แสดงให้เห็นว่ามีอวัยวะส่วนหัวสี่คู่แทนที่จะเป็นสามคู่ตามที่เคยคิดไว้ก่อนหน้านี้ ซึ่งให้ข้อมูลเชิงลึกใหม่เกี่ยวกับกลไกการกินของไทรโลไบต์
- การถ่ายภาพเอกซเรย์ไมโครโทโมกราฟีความละเอียดสูง (XRµCT) ถูกใช้เพื่อสร้างแบบจำลอง 3 มิติที่มีรายละเอียดสูง ซึ่งเน้นถึงศักยภาพของการสะสมเถ้าภูเขาไฟในการค้นพบฟอสซิลที่ได้รับการอนุรักษ์ไว้อย่างดี

## [OpenDNS ระงับบริการในฝรั่งเศสเนื่องจากคำสั่งบล็อกการละเมิดลิขสิทธิ์ของ Canal+](https://torrentfreak.com/opendns-suspends-service-in-france-due-to-canal-piracy-blocking-order-240629/)

- ศาลฝรั่งเศสได้สั่งให้ Google, Cloudflare และ Cisco บล็อกบันทึก DNS ที่หลีกเลี่ยงการปิดกั้นเว็บไซต์ละเมิดลิขสิทธิ์ที่มีอยู่ ตามคำร้องขอจาก Canal+.
- ในการตอบสนอง Cisco ได้ถอนบริการ OpenDNS ออกจากฝรั่งเศส ซึ่งเน้นให้เห็นถึงผลกระทบของการดำเนินการทางกฎหมายต่อบริการอินเทอร์เน็ต
- คำสั่งศาลนี้เป็นส่วนหนึ่งของแคมเปญต่อต้านการละเมิดลิขสิทธิ์ของ Canal+ ซึ่งก่อนหน้านี้ได้กำหนดให้ผู้ให้บริการอินเทอร์เน็ตบล็อกเว็บไซต์สตรีมมิ่งกีฬาที่ละเมิดลิขสิทธิ์กว่า 100 แห่ง ทำให้ผู้ใช้ต้องหันไปใช้เซิร์ฟเวอร์ DNS ของบุคคลที่สาม

### [ปฏิกิริยา](https://news.ycombinator.com/item?id=40835117)

- OpenDNS ได้ระงับการให้บริการในฝรั่งเศสเนื่องจากคำสั่งศาลที่บังคับให้บล็อกเว็บไซต์ละเมิดลิขสิทธิ์ ทำให้เกิดการถกเถียงเกี่ยวกับความเป็นกลางของ DNS และการปฏิบัติตามกฎหมาย
- การปฏิบัติตามคำสั่งของบริษัทใหญ่ ๆ อย่าง Google ทำให้เกิดความกังวลเกี่ยวกับการสร้างแบบอย่างสำหรับการปรับเปลี่ยน DNS และการแทรกแซงของรัฐบาลที่อาจเกิดขึ้น
- ผู้ใช้อาจมองหาผู้ให้บริการ DNS ทางเลือกหรือ VPN เพื่อหลีกเลี่ยงข้อจำกัดเหล่านี้ ซึ่งเน้นถึงความท้าทายทางเทคนิคและผลกระทบของการบังคับใช้การบล็อก DNS

- นักวิจัยกำลังสำรวจเครือข่ายแอนะล็อกของทรานซิสเตอร์ ซึ่งถูกจำกัดให้อยู่ในโหมดโอห์มิก เพื่อทำงานด้านการเรียนรู้ของเครื่องโดยไม่ใช้โปรเซสเซอร์แบบดั้งเดิม
- วิธีการนี้อาจเสนอการดำเนินงานที่มีประสิทธิภาพและขนานกันมากกว่าระบบดิจิทัล แม้ว่าจะมีความท้าทายในการใช้งานจริง เช่น การรักษาประจุบนตัวเก็บประจุของเกต
- แนวคิดนี้กำลังได้รับความสนใจเนื่องจากมีศักยภาพในการปฏิวัติการคำนวณของเครือข่ายประสาทโดยการใช้ส่วนประกอบแบบแอนะล็อก ซึ่งอาจนำไปสู่ความก้าวหน้าที่สำคัญในด้านประสิทธิภาพการใช้พลังงานและความเร็วในการประมวลผล
