## [สิ่งที่คุณได้รับหลังจากรัน SSH Honeypot เป็นเวลา 30 วัน](https://blog.sofiane.cc/ssh_honeypot/)

### [ปฏิกิริยา](https://news.ycombinator.com/item?id=40694768)

- ผู้ใช้ที่รัน SSH honeypot เป็นเวลา 30 วัน สังเกตเห็นการจราจรที่ไม่ถูกต้องอย่างมีนัยสำคัญ ส่วนใหญ่เป็นจาก "บริษัทความปลอดภัยทางอินเทอร์เน็ต" ที่สแกนพื้นที่ IPv4
- การบล็อกเครือข่ายเหล่านี้ลดการจราจรที่ไม่ต้องการลงกว่า 50% แต่การอภิปรายนี้เน้นย้ำถึงข้อจำกัดของการบล็อก IP เนื่องจาก CGNAT (Carrier-Grade Network Address Translation)
- เน้นการใช้เครื่องมือรักษาความปลอดภัยเช่น fail2ban, VPNs และการยืนยันตัวตนด้วยกุญแจสาธารณะเพื่อปกป้องเซิร์ฟเวอร์ โดยผู้ใช้แบ่งปันประสบการณ์การโฮสต์ด้วยตนเองและความท้าทายจากการโจมตีอย่างต่อเนื่อง

## [ผู้พิพากษา NLRB ประกาศว่าข้อกำหนดไม่แข่งขันเป็นการปฏิบัติด้านแรงงานที่ไม่เป็นธรรม](https://www.nlrbedge.com/p/in-first-case-of-its-kind-nlrb-judge)

- ผู้พิพากษา NLRB ซาราห์ คาร์พิเนน ตัดสินว่าข้อกำหนดไม่แข่งขันและข้อกำหนดไม่ชักชวนเพื่อนร่วมงานเป็นการปฏิบัติที่ไม่เป็นธรรมต่อแรงงาน ซึ่งเป็นการตัดสินที่สำคัญ
- กรณีนี้เกี่ยวข้องกับบริษัท J.O. Mory, Inc. ซึ่งมีการไล่พนักงานจัดตั้งสหภาพออกเนื่องจากการ "salting" (การจัดตั้งเพื่อนร่วมงาน) และผู้พิพากษาได้สั่งให้พนักงานจัดตั้งสหภาพกลับเข้าทำงานพร้อมกับจ่ายค่าจ้างย้อนหลัง
- นี่เป็นครั้งแรกที่ผู้พิพากษากฎหมายปกครอง (ALJ) ได้ตัดสินว่าข้อกำหนดเหล่านี้เป็นการปฏิบัติที่ไม่เป็นธรรมต่อแรงงาน โดยเน้นย้ำถึงจุดยืนของ NLRB ที่ต่อต้านข้อกำหนดดังกล่าวสำหรับแรงงานที่ไม่ใช่ผู้ควบคุม

### [ปฏิกิริยา](https://news.ycombinator.com/item?id=40696992)

- ผู้พิพากษา NLRB ได้ตัดสินว่าข้อกำหนดไม่แข่งขันเป็นการปฏิบัติที่ไม่เป็นธรรมในการจ้างงาน ทำให้เกิดการถกเถียงเกี่ยวกับการปฏิบัติการจ้างงานที่จำกัดอื่น ๆ
- คำตัดสินดังกล่าวได้นำไปสู่การถกเถียงเกี่ยวกับข้อกำหนดการไม่เรียกร้องและนโยบายของบริษัทที่จำกัดผู้จัดการไม่ให้การอ้างอิง ซึ่งบางคนโต้แย้งว่าขัดขวางการเคลื่อนย้ายงาน
- การตัดสินใจนี้เป็นส่วนหนึ่งของการสนทนาที่กว้างขึ้นเกี่ยวกับการสร้างสมดุลอำนาจระหว่างนายจ้างและลูกจ้าง โดยมีการเรียกร้องให้เพิ่มการคุ้มครองแรงงานเพื่อให้แน่ใจว่ามีการปฏิบัติที่เป็นธรรมและความคล่องตัวในตลาดงาน

## [ศาลฝรั่งเศสสั่งให้ Google, Cloudflare, Cisco ปลอมแปลง DNS เพื่อหยุดการละเมิดลิขสิทธิ์](https://torrentfreak.com/google-cloudflare-cisco-will-poison-dns-to-stop-piracy-block-circumvention-240613/)

- ศาลฝรั่งเศสได้สั่งให้ Google, Cloudflare และ Cisco บล็อกการเข้าถึงโดเมนสตรีมมิ่งกีฬาละเมิดลิขสิทธิ์ประมาณ 117 โดเมนผ่านตัวแก้ไข DNS ของพวกเขา ซึ่งเป็นการเพิ่มความเข้มข้นในการต่อต้านการละเมิดลิขสิทธิ์สำหรับผู้แพร่ภาพ Canal+.
- Canal+ ยังได้รับอนุญาตให้ลบเว็บไซต์เหล่านี้ออกจากผลการค้นหา โดยใช้กฎหมายการบล็อกเว็บไซต์ที่มีอยู่ในฝรั่งเศสเพื่อต่อสู้กับการละเมิดลิขสิทธิ์
- แม้จะมีการโต้แย้งเกี่ยวกับผลกระทบที่น้อยนิดต่ออัตราการละเมิดลิขสิทธิ์ ศาลยังคงยืนยันสิทธิของ Canal+ และ Google ได้ยืนยันว่าจะปฏิบัติตามคำสั่งดังกล่าว ซึ่งทำให้เกิดคำถามเกี่ยวกับกลยุทธ์ต่อต้านการละเมิดลิขสิทธิ์ในอนาคต

### [ปฏิกิริยา](https://news.ycombinator.com/item?id=40693451)

- ศาลฝรั่งเศสได้สั่งให้ Google, Cloudflare และ Cisco เปลี่ยนตัวแก้ไข DNS ของพวกเขาเพื่อบล็อกการเข้าถึงโดเมนสตรีมมิ่งกีฬาละเมิดลิขสิทธิ์ประมาณ 117 โดเมน
- คำตัดสินนี้แตกต่างจากการต่อต้านก่อนหน้านี้ของบริษัทเทคโนโลยีต่อกฎหมายที่คล้ายกันของสหรัฐฯ โดยเน้นถึงการเปลี่ยนแปลงในท่าทีของพวกเขาเพื่อรักษาการควบคุมและหลีกเลี่ยงปัญหาทางกฎหมาย
- นักวิจารณ์เสนอให้ใช้ตัวแก้ไข DNS ส่วนบุคคลเพื่อหลีกเลี่ยงการบล็อกเหล่านี้ ซึ่งก่อให้เกิดการถกเถียงอย่างต่อเนื่องเกี่ยวกับความสมดุลระหว่างมาตรการต่อต้านการละเมิดลิขสิทธิ์และเสรีภาพทางอินเทอร์เน็ต

## [Raspberry Pi 5 ไม่สามารถเทียบกับ Tini-Mini-Micro PC ได้](https://louwrentius.com/the-raspberry-pi-5-is-no-match-for-a-tini-mini-micro-pc.html)

- Raspberry Pi 5 ซึ่งเปิดตัวในฤดูใบไม้ร่วงปี 2023 มีประสิทธิภาพที่ดีขึ้นแต่ยังคงถูกจำกัดด้วยการเก็บข้อมูลบนการ์ด SD และมีต้นทุนที่สูงกว่ารุ่นก่อนหน้า
- มินิพีซีมือสองขนาด 1 ลิตรจากแบรนด์อย่าง Dell, Lenovo และ HP มีประสิทธิภาพเหนือกว่า Pi 5 ในด้านพลังการประมวลผล ความจุ RAM และตัวเลือกการจัดเก็บข้อมูล (SATA และ NVME SSDs)
- แม้ว่า Raspberry Pi 5 จะประหยัดพลังงาน แต่คอมพิวเตอร์ขนาดเล็กมือสองเหมาะสมกว่าสำหรับการใช้งานเซิร์ฟเวอร์ที่บ้านเนื่องจากมีประสิทธิภาพที่เหนือกว่าและรองรับ SSD/NVME ในตัว

### [ปฏิกิริยา](https://news.ycombinator.com/item?id=40697831)

- Raspberry Pi 5 ได้สูญเสียความคุ้มค่าเนื่องจากสิ้นสุดข้อตกลงพิเศษกับ Broadcom ทำให้มีความสามารถในการแข่งขันน้อยลงในตลาดปัจจุบัน
- ผู้ใช้พบว่ามินิพีซีมือสองมีความคุ้มค่ามากกว่า เนื่องจากมีพลังและฟีเจอร์มากขึ้นในราคาที่ใกล้เคียงกัน
- แม้ว่า Pi 5 จะยังคงแข็งแกร่งในด้านการศึกษาและการใช้งาน GPIO (General-Purpose Input/Output) แต่ทางเลือกอื่น ๆ เช่น Intel NUCs มือสองหรือ N100 mini PCs ใหม่ ๆ นั้นน่าสนใจกว่าสำหรับงานคอมพิวเตอร์ทั่วไป

## [สถาปัตยกรรมเบื้องหลังสตาร์ทอัพเทคโนโลยีที่มีคนเดียว (2021)](https://anthonynsimon.com/blog/one-man-saas-architecture/)

- องค์ประกอบหลักประกอบด้วยการจัดการการจราจรด้วย Cloudflare และ AWS, การปรับใช้แบบอัตโนมัติผ่าน GitHub Actions และ Flux, และความน่าเชื่อถือผ่านการปรับขนาดอัตโนมัติและการตรวจสอบความมีชีวิตของ Kubernetes

- ให้ความสำคัญกับการปรับใช้ที่รวดเร็วและพิจารณาการปรับโครงสร้างในภายหลัง; ผู้ใช้สนใจผลิตภัณฑ์ที่ใช้งานได้มากกว่าที่จะสนใจเทคโนโลยีที่อยู่เบื้องหลัง
- ในขณะที่เทคโนโลยีใหม่บางอย่างเช่น Kubernetes อาจจะเกินความจำเป็น โซลูชันที่ง่ายกว่าเช่น Docker Swarm หรือบริการที่มีการจัดการอาจเหมาะสมกว่าสำหรับการสร้างสมดุลระหว่างความเรียบง่ายและความสามารถในการขยายตัว

- เทคนิคการก่อวินาศกรรมที่แนะนำสำหรับ CTO ได้แก่ การกำหนดให้มีการเขียนโค้ดใหม่ที่ยาวนาน, ส่งเสริมการใช้ภาษาที่หลากหลาย, และการแยกระบบเพื่อเพิ่มความซับซ้อน
- วิธีการอื่นๆ รวมถึงการละเลยตัวชี้วัดที่มีประโยชน์ การเชื่อมโยงค่าตอบแทนกับตำแหน่งและขนาดทีม และการสร้างกระบวนการจ้างงานที่เป็นอัตวิสัย ทั้งหมดนี้มีเป้าหมายเพื่อชะลอประสิทธิภาพการทำงานอย่างแยบยลผ่านการกระทำที่ดูสมเหตุสมผล

### [ปฏิกิริยา](https://news.ycombinator.com/item?id=40695839)

- การอภิปรายเกี่ยวกับการก่อวินาศกรรมอย่างง่ายในซอฟต์แวร์ โดยอ้างอิงจากบทความในปี 2023 โดย Erik Bernhardsson ตั้งคำถามถึงประสิทธิภาพของกลยุทธ์การก่อวินาศกรรมดั้งเดิมของ CIA
- ผู้แสดงความคิดเห็นถกเถียงถึงความเป็นไปได้และผลกระทบของกลยุทธ์เหล่านี้ โดยชี้ให้เห็นว่าการบริหารจัดการที่ไม่ดีและการเพิ่มผลกำไรที่ผิดพลาดสามารถทำให้องค์กรล้มเหลวได้โดยธรรมชาติ
- การสนทนานี้รวมถึงบริบททางประวัติศาสตร์ บทบาทของความไร้ความสามารถ และแนวปฏิบัติสมัยใหม่ เช่น ระบบบริการตนเองที่ซับซ้อนและนโยบายการกลับมาทำงานที่สำนักงานที่อาจสอดคล้องกับหลักการก่อวินาศกรรมโดยไม่ตั้งใจ

## [ทางเลือกยุโรป](https://european-alternatives.eu)

- โพสต์นี้เน้นทางเลือกในยุโรปสำหรับบริการและผลิตภัณฑ์ดิจิทัลต่างๆ โดยเน้นการสนับสนุนธุรกิจท้องถิ่นและการปฏิบัติตาม GDPR (กฎระเบียบการคุ้มครองข้อมูลทั่วไป)
- มันแสดงรายการหมวดหมู่ของบริการพร้อมกับจำนวนทางเลือกที่มีอยู่ เช่น การวิเคราะห์เว็บ การประมวลผลแบบคลาวด์ และผู้ให้บริการอีเมล เป็นต้น
- โพสต์นี้เน้นย้ำถึงประโยชน์ของการเลือกผู้ให้บริการในยุโรป ซึ่งรวมถึงการคืนภาษีมูลค่าเพิ่ม วิธีการชำระเงินที่เป็นที่นิยมในยุโรป และการบังคับใช้สิทธิภายในสหภาพยุโรปที่ง่ายขึ้นเนื่องจากกฎหมายที่เป็นเอกภาพ

### [ปฏิกิริยา](https://news.ycombinator.com/item?id=40695754)

- European Alternatives แสดงรายการผลิตภัณฑ์และบริการจากบริษัทที่ตั้งอยู่ในประเทศสมาชิกของ EU, EEA, EFTA, หรือ DCFTA โดยไม่รวมผู้ให้บริการโฮสติ้งย่อยที่ไม่ใช่ EU/EFTA
- เว็บไซต์ครอบคลุมหมวดหมู่ต่างๆ เช่น การจัดการตัวตนและการเข้าถึง ผู้ให้บริการ VPS และเครื่องมือค้นหา พร้อมการอภิปรายเกี่ยวกับการยกเว้นสหราชอาณาจักรหลัง Brexit และการปฏิบัติตามกฎระเบียบของสหภาพยุโรป
- ผู้ใช้ยังถกเถียงกันถึงคุณภาพของบริการอย่าง Hetzner และ Tuta รวมถึงผลกระทบทางภูมิรัฐศาสตร์ของความเป็นส่วนตัวของข้อมูลและการแข่งขันในภูมิทัศน์เทคโนโลยีของยุโรป

## [การสร้าง SimCity: วิธีการใส่โลกลงในเครื่องจักร](https://mitpress.mit.edu/9780262547482/building-simcity/)

## [SQLite น่าจะถูกใช้งานมากกว่าฐานข้อมูลอื่น ๆ ทั้งหมดรวมกัน](https://sqlite.org/mostdeployed.html)

- SQLite เป็นเครื่องมือฐานข้อมูลที่ถูกใช้งานและติดตั้งมากที่สุดในโลก โดยมีอยู่ในอุปกรณ์และแอปพลิเคชันนับพันล้าน รวมถึง Android, iOS, Mac, Windows 10, เว็บเบราว์เซอร์หลัก ๆ และแอปพลิเคชันยอดนิยมต่าง ๆ
- ด้วยสมาร์ทโฟนมากกว่า 4 พันล้านเครื่องที่แต่ละเครื่องมีฐานข้อมูล SQLite หลายร้อยฐานข้อมูล มีความเป็นไปได้ว่ามีฐานข้อมูล SQLite ที่ใช้งานอยู่ทั่วโลกมากกว่าหนึ่งล้านล้านฐานข้อมูล
- SQLite เป็นหนึ่งในห้าโมดูลซอฟต์แวร์ที่ถูกใช้งานมากที่สุด อาจเป็นไลบรารีซอฟต์แวร์ที่ถูกใช้งานมากที่สุดเป็นอันดับสองรองจาก zlib และอาจเป็นส่วนประกอบซอฟต์แวร์ที่ถูกใช้งานมากที่สุดเนื่องจากมีการใช้งานหลายอินสแตนซ์ต่ออุปกรณ์

### [ปฏิกิริยา](https://news.ycombinator.com/item?id=40693500)

- SQLite ถูกใช้งานอย่างกว้างขวาง น่าจะมากกว่าฐานข้อมูลอื่น ๆ ทั้งหมดรวมกัน เนื่องจากความง่ายในการใช้งาน ความน่าเชื่อถือ และการทำงานแบบออฟไลน์
- ผู้ใช้กำลังเปลี่ยนจาก json+zip ไปเป็น SQLite ที่ถูกบีบอัดด้วย gzip เพื่อการบังคับใช้สคีมาที่ดียิ่งขึ้นเมื่อดาวน์โหลดข้อมูลสำหรับการสำรองข้อมูลหรือการแชร์
- แม้ว่า SQLite จะเป็นที่นิยม แต่ก็ไม่ได้เปิดรับการมีส่วนร่วมจากผู้อื่น อย่างไรก็ตาม รูปแบบไฟล์ของมันได้รับการบันทึกไว้อย่างดีและมีความเสถียร ทำให้เหมาะสำหรับการใช้งานหลายประเภท รวมถึง iOS และ Android

## [การสร้างระบบลินุกซ์ฝังตัวครั้งแรกของฉัน](https://popovicu.com/posts/making-my-first-embedded-linux-system/)

- โพสต์นี้บอกเล่าเรื่องราวการเดินทางของผู้เขียนในการสร้างระบบ Linux จากศูนย์ โดยเริ่มต้นจากการไม่มีประสบการณ์เกี่ยวกับ PCB (แผงวงจรพิมพ์) มาก่อน
- ขั้นตอนสำคัญรวมถึงการโปรแกรมไมโครคอนโทรลเลอร์เช่น Atmel ATtiny, การออกแบบ PCB ที่กำหนดเอง, และการสร้างบอร์ด Linux แบบฝังตัวโดยใช้ F1C100s SoC (System on Chip).
- โครงการนี้เกี่ยวข้องกับการใช้เครื่องมือเช่น avrdude สำหรับการโปรแกรม, แผงวงจรพิมพ์ 4 ชั้นสำหรับการเดินสายที่ซับซ้อน, และการปรับแต่งภาพซอฟต์แวร์ Lichee Pi สำหรับบอร์ดที่กำหนดเอง

### [ปฏิกิริยา](https://news.ycombinator.com/item?id=40695165)

- ผู้ใช้คนหนึ่งได้แบ่งปันประสบการณ์ในการสร้างระบบ Linux ฝังตัวครั้งแรกของพวกเขา ซึ่งได้จุดประกายการสนทนาเกี่ยวกับรายละเอียดทางเทคนิคและการปรับปรุงต่างๆ
- จุดสำคัญรวมถึงการใช้แฟลชขนาด 16 เมกะบิต, การเดินสายแบบดิฟเฟอเรนเชียล, และประโยชน์ที่อาจเกิดขึ้นจากการเพิ่มตัวเชื่อมต่อการ์ด SD
- โครงการนี้ใช้ Allwinner F1C100S ซึ่งเป็น ARMv7 แบบคอร์เดียวที่มีความเร็ว 533MHz พร้อม DRAM บนบอร์ดขนาด 32MB ทำให้เกิดการเปรียบเทียบกับโปรเซสเซอร์อื่นๆ ที่คล้ายกัน
