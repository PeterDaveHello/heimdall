# 2024-06-22

## [คอมพิวเตอร์ Windows ของฉันไม่รู้สึกเหมือนเป็นของฉันอีกต่อไป](https://www.howtogeek.com/my-windows-computer-just-doesnt-feel-like-mine-anymore/)

- Windows ได้กลายเป็นเชิงพาณิชย์มากขึ้นและมีความเป็นส่วนตัวน้อยลง โดยการผสานโฆษณาทำให้รู้สึกว่ามันดูราคาถูก
- การอัปเดตที่บังคับและการเปลี่ยนแปลงที่ล่วงล้ำมักทำให้ระบบเสียหายและเปลี่ยนแปลงการตั้งค่าของผู้ใช้โดยไม่ได้รับความยินยอม ส่งผลกระทบต่อความน่าเชื่อถือและประสบการณ์ของผู้ใช้
- ทางเลือกอย่าง macOS และ Linux ได้รับการยกย่องว่าไม่มีโฆษณาและให้การควบคุมผู้ใช้มากขึ้น แม้ว่า Linux อาจจะซับซ้อน

### [ปฏิกิริยา](https://news.ycombinator.com/item?id=40755444)

- ผู้ใช้กำลังแสดงความไม่พอใจเกี่ยวกับการควบคุมที่เพิ่มขึ้นและคุณสมบัติที่ล่วงล้ำใน Windows เช่น การไม่สามารถลบ 'แนะนำ' ออกจากเมนูเริ่มและการแจ้งเตือน OneDrive ที่เกิดขึ้นอย่างต่อเนื่อง
- ข้อร้องเรียนรวมถึงปัญหาต่างๆ เช่น การค้นหาเว็บในเมนูเริ่มที่ไม่สามารถปิดได้ การตั้งค่าเบราว์เซอร์ Edge ที่ถูกรีเซ็ต และ Windows ติดตั้งเวอร์ชันไดรเวอร์เก่าโดยอัตโนมัติ
- ผู้ใช้บางคนกำลังแนะนำทางเลือกอื่น เช่น การใช้ลินุกซ์ดิสทริบิวชัน (เช่น Fedora, Linux Mint) หรือเครื่องมือในการจัดการและลดความซับซ้อนของ Windows ซึ่งเน้นถึงความไม่พอใจที่เพิ่มขึ้นกับประสบการณ์การใช้งาน Windows ในปัจจุบัน

## [Internet Archive ถูกบังคับให้ลบหนังสือ 500,000 เล่มหลังจากผู้จัดพิมพ์ชนะคดีในศาล](https://arstechnica.com/tech-policy/2024/06/internet-archive-forced-to-remove-500000-books-after-publishers-court-win/)

- Internet Archive (IA) ถูกบังคับให้ลบหนังสือ 500,000 เล่มหลังจากแพ้คดีความกับผู้จัดพิมพ์หนังสือ ซึ่งส่งผลกระทบอย่างมากต่อผู้อ่านที่พึ่งพา IA ในการเข้าถึงหนังสือหายาก
- IA กำลังยื่นอุทธรณ์ต่อการตัดสินใจ โดยโต้แย้งว่าการให้ยืมหนังสือดิจิทัลที่ควบคุมโดย IA ควรถือเป็นการใช้งานที่เป็นธรรมภายใต้กฎหมายลิขสิทธิ์ โดยเน้นบทบาทของห้องสมุดในการให้การเข้าถึงความรู้
- จดหมายเปิดผนึกที่มีลายเซ็นเกือบ 19,000 รายชื่อเน้นถึงผลกระทบเชิงลบต่อการศึกษาและชุมชนที่ขาดแคลน โดยผู้สนับสนุน IA เรียกร้องให้ผู้จัดพิมพ์คืนสิทธิ์การเข้าถึงหนังสือที่ถูกลบออกไป

### [ปฏิกิริยา](https://news.ycombinator.com/item?id=40754229)

- Internet Archive (IA) ได้รับคำสั่งให้ลบหนังสือจำนวน 500,000 เล่มหลังจากคำตัดสินของศาลที่สนับสนุนผู้จัดพิมพ์ ทำให้เกิดการถกเถียงเกี่ยวกับกฎหมายลิขสิทธิ์
- นักวิจารณ์โต้แย้งว่าการกระทำที่มีเจตนาดีของ IA นั้นมีความเสี่ยงทางกฎหมายและอาจทำให้องค์กรตกอยู่ในอันตราย ซึ่งเน้นย้ำถึงความจำเป็นในการปฏิรูปกฎหมายลิขสิทธิ์
- กรณีนี้เน้นย้ำถึงความตึงเครียดระหว่างการรักษาการเข้าถึงความรู้และการปฏิบัติตามกฎหมายลิขสิทธิ์ในปัจจุบัน โดยมีหลายฝ่ายเรียกร้องให้มีการปรับสมดุลที่ดียิ่งขึ้น

## [โฟลเดอร์คงที่ของตัวเองใน C/C++](https://www.neilhenning.dev/posts/yourownconstantfolder/)

- การใช้ `sqrtps` intrinsic ใน C/C++ ร่วมกับแฟล็ก `-ffast-math` ของ clang อาจนำไปสู่ปัญหาความแม่นยำเนื่องจากมันแทนที่ `sqrtps` ด้วย `rsqrtps`.
- อินไลน์แอสเซมบลีสามารถบังคับการใช้ `sqrtps` ได้ แต่ไม่รองรับการพับค่าคงที่เมื่อฟังก์ชันถูกอินไลน์
- เพื่อให้แน่ใจว่ามีการพับค่าคงที่และพฤติกรรมที่ถูกต้อง ให้ใช้ `__builtin_constant_p` เพื่อตรวจสอบว่าค่าของเวกเตอร์เป็นค่าคงที่หรือไม่ และใช้ `_mm_sqrt_ps` ตามความเหมาะสม

### [ปฏิกิริยา](https://news.ycombinator.com/item?id=40757787)

- ผู้ใช้ Hacker News ได้ชี้ให้เห็นถึงความไม่มีประสิทธิภาพในกระบวนการคำนวณทางคณิตศาสตร์ที่ปลอดภัยต่อประเภทของ Rust เมื่อทำการปรับปรุงโค้ดการเรียนรู้ของเครื่อง โดยเฉพาะอย่างยิ่งกับการหาส่วนเหลือแบบยุคลิดที่เกี่ยวข้องกับตัวเลขทศนิยมและตัวหารจำนวนเต็ม
- ผู้ใช้สร้างฟังก์ชันที่กำหนดเองเพื่อเพิ่มประสิทธิภาพ แต่พบข้อจำกัดเนื่องจากข้อจำกัดของประเภทในภาษา Rust โดยเสนอแนะว่าการสนับสนุนจากคอมไพเลอร์อาจช่วยให้การปรับแต่งดังกล่าวเป็นไปได้
- การอภิปรายขยายไปถึงการพิจารณาถึงความเป็นไปได้ในการสร้างกล่องเฉพาะ ความซับซ้อนของการปรับแต่ง C/C++ และความสำคัญของการเข้าใจพฤติกรรมของคอมไพเลอร์และการแลกเปลี่ยนในด้านการปรับแต่งประสิทธิภาพ

## [แอนดรูว์ เอส. ทาเนนบอม ได้รับรางวัลระบบซอฟต์แวร์จาก ACM](https://vu.nl/en/news/2024/andrew-s-tanenbaum-receives-acm-software-system-award)

- เมื่อวันที่ 20 มิถุนายน 2024, Andrew S. Tanenbaum ได้รับรางวัล ACM Software System Award สำหรับผลงานของเขาใน MINIX ซึ่งเป็นระบบปฏิบัติการ UNIX ที่ใช้ไมโครเคอร์เนลที่สร้างขึ้นในปี 1987
- MINIX, เริ่มต้นด้วยโค้ด 12,000 บรรทัดและเปิดเป็นโอเพนซอร์สในปี 2000, มีอิทธิพลอย่างมากต่อการสอนระบบปฏิบัติการและการออกแบบระบบเช่น Linux และ MeikOS.
- รางวัลนี้ซึ่งได้รับการสนับสนุนจาก IBM และมีเงินรางวัล $35,000 ยกย่องผลกระทบของ Tanenbaum ต่อการออกแบบระบบปฏิบัติการ โดยเฉพาะการสนับสนุนของเขาต่อสถาปัตยกรรมไมโครเคอร์เนล

## [เรื่องราวที่สี่ (1995)](https://groups.google.com/g/comp.lang.forth/c/ThWiGXwKqv4)

- อัลเลน เซโคริช ผู้สำเร็จการศึกษาด้านฟิสิกส์ เริ่มต้นอาชีพของเขาในปี 1975 ที่บริษัท MDH Industries โดยทำงานเกี่ยวกับการวัดปริมาณกำมะถันในถ่านหินโดยใช้แคลิฟอร์เนียม 252 และในช่วงแรกใช้ภาษาโปรแกรม Fortran สำหรับการจำลอง
- ต่อมาเขาได้นำภาษาโปรแกรมมิ่ง Forth มาใช้ แม้จะมีความสับสนในตอนแรก และใช้มันในการพัฒนาเครื่องวัดกำมะถันต้นแบบ โดยต้องเผชิญกับการต่อต้านจากเพื่อนร่วมงานที่มองว่า Forth เป็นสิ่งที่ไม่ธรรมดา
- แม้ว่า Forth จะมีพลังและประสิทธิภาพ แต่สถานะเฉพาะกลุ่มของมันและความชอบของอุตสาหกรรมที่มีต่อภาษาหลักทำให้การนำไปใช้ถูกจำกัด ส่งผลให้ Cekorich หยุดการเขียนโปรแกรมในที่สุด

### [ปฏิกิริยา](https://news.ycombinator.com/item?id=40755688)

- ซีพียู NOVIX 2000 ของ Chuck Moore ซึ่งมีพื้นฐานมาจาก Forth ใช้สแต็กสองตัวและดำเนินการหลายคำสั่งต่อรอบนาฬิกาด้วยเกตที่น้อยที่สุด แสดงให้เห็นถึงสถาปัตยกรรมซีพียูทางเลือก
- ชิป GA144 ที่ได้รับแรงบันดาลใจจาก Forth มีความเรียบง่ายและอาจแทนที่ System on Chips (SoCs) ด้วยความยืดหยุ่นมากขึ้นและการใช้พลังงานที่ต่ำลง แม้ว่ามันอาจจะท้าทายในการใช้งานหากไม่มีไลบรารีอุปกรณ์ต่อพ่วงทั่วไป
- ความเรียบง่ายของ Forth ช่วยให้สามารถสร้างสรรค์ได้อย่างรวดเร็วและเป็นส่วนตัว แต่ก็อาจไม่เป็นมาตรฐานและยากต่อการเข้าใจ ทำให้ไม่ค่อยได้รับการยอมรับจากทีมงานแม้ว่าจะมีความสามารถสูงก็ตาม
