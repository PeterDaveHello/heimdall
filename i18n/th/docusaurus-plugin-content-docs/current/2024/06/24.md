## [แอปเปิลพบว่าละเมิดกฎการแข่งขันของสหภาพยุโรป](https://www.theguardian.com/technology/article/2024/jun/24/apple-breach-eu-competition-rules-digital-markets-act)

- คณะกรรมาธิการยุโรปพบว่า Apple ละเมิดกฎหมายใหม่ของสหภาพยุโรปที่มุ่งปกป้องคู่แข่งรายย่อยและช่วยให้ผู้บริโภคสามารถหาแอปที่ราคาถูกกว่าได้
- Apple มีเวลา 12 เดือนในการปฏิบัติตามข้อบังคับ มิฉะนั้นจะต้องเผชิญกับค่าปรับสูงสุดถึง 10% ของรายได้ทั่วโลกของบริษัท หลังจากการสอบสวนที่เริ่มขึ้นในเดือนมีนาคม
- นี่เป็นการสอบสวนการไม่ปฏิบัติตามข้อกำหนดครั้งที่สามของ Apple นับตั้งแต่กฎหมายตลาดดิจิทัล (DMA) ถูกบังคับใช้ โดยเน้นไปที่ค่าธรรมเนียมและแนวปฏิบัติของ Apple ที่จำกัดนักพัฒนาจากการชี้นำผู้บริโภคไปยังตัวเลือกการซื้อทางเลือกอื่น

## [อัยการสหรัฐฯ แนะนำให้กระทรวงยุติธรรมตั้งข้อหาทางอาญาต่อบริษัทโบอิ้ง](https://www.voanews.com/a/us-prosecutors-recommend-justice-department-criminally-charge-boeing-as-deadline-looms/7667194.html)

- อัยการสหรัฐฯ เสนอข้อกล่าวหาทางอาญาต่อบริษัทโบอิ้งในข้อหาละเมิดข้อตกลงปี 2021 ที่เกี่ยวข้องกับเหตุการณ์เครื่องบิน 737 MAX ตกสองครั้งที่มีผู้เสียชีวิต
- กระทรวงยุติธรรมมีเวลาจนถึงวันที่ 7 กรกฎาคมในการตัดสินใจเกี่ยวกับการดำเนินคดี โดยที่โบอิ้งโต้แย้งข้อกล่าวหาการละเมิดและมีการเจรจาอย่างต่อเนื่อง
- การตั้งข้อหาทางอาญาอาจทำให้วิกฤตของโบอิ้งรุนแรงขึ้น ส่งผลกระทบต่อสัญญากับรัฐบาลสหรัฐฯ ขณะที่ครอบครัวของเหยื่อเรียกร้องค่าปรับ 25 พันล้านดอลลาร์และการดำเนินคดี

## [คอสโมโพลิแทน v3.5.0](https://github.com/jart/cosmopolitan/releases/tag/3.5.0)

- Cosmopolitan Libc v3.5.0 ได้ถูกปล่อยออกมาแล้ว ทำให้ภาษา C กลายเป็นภาษาที่สามารถสร้างครั้งเดียวและรันได้ทุกที่โดยไม่ต้องใช้ตัวแปลภาษา (interpreter) หรือเครื่องเสมือน (virtual machine)
- การปรับปรุงที่สำคัญรวมถึงการปรับใช้แผนที่หน่วยความจำที่มีคุณภาพสูงขึ้น การเขียนโปรแกรมจัดการหน่วยความจำใหม่ และการอัปเดต Lua เป็นเวอร์ชัน 5.4.6
- มีการแนะนำคุณสมบัติใหม่ใน C++ STL เช่น การเพิ่มประสิทธิภาพของ ctl::string สำหรับสตริงขนาดเล็ก, ctl::set, ctl::map, และ ctl::unique_ptr พร้อมกับการแก้ไขข้อบกพร่องและการปรับปรุงประสิทธิภาพต่างๆ

### [ปฏิกิริยา](https://news.ycombinator.com/item?id=40773139)

- Cosmopolitan v3.5.0 ได้ถูกปล่อยออกมาแล้ว สร้างความสนใจอย่างมากเนื่องจากความสามารถในการสร้างไฟล์ปฏิบัติการที่พกพาได้สูง ซึ่งสามารถทำงานบนระบบปฏิบัติการและสถาปัตยกรรมหลายแบบโดยไม่ต้องแก้ไข
- เวอร์ชันใหม่มีประสิทธิภาพที่ดีขึ้น โดยมีการอ้างว่า Cosmopolitan Libc มีความเร็วเป็นสองเท่าของ Musl Libc สำหรับโปรแกรมหลายตัวที่ใช้บรรทัดคำสั่ง (CLI) และมีการสนับสนุนเธรดที่แข็งแกร่ง ทำให้เหมาะสมสำหรับซอฟต์แวร์ AI ที่มีประสิทธิภาพสูง
- การเปิดตัวนี้รวมถึงการสนับสนุนสำหรับ WebAssembly (WASM) และฟีเจอร์ต่างๆ เช่น ส่วนหัว "Actually Portable Executable" (APE) ซึ่งช่วยเพิ่มความเข้ากันได้ข้ามแพลตฟอร์ม รวมถึงการสนับสนุนสำหรับ Android ด้วย

## [LINQPad – สนามเด็กเล่นของโปรแกรมเมอร์ .NET](https://www.linqpad.net/)

- LINQPad เป็นเครื่องมือที่หลากหลายสำหรับโปรแกรมเมอร์ .NET ในการทดสอบโค้ด C#/F#/VB ทันที, สอบถามฐานข้อมูล, และทำงานอัตโนมัติด้วยประโยชน์ของ REPL (Read-Eval-Print Loop).
- รองรับ C# 12 และ .NET 8, ช่วยให้สามารถทำการสืบค้นฐานข้อมูลแบบโต้ตอบใน LINQ หรือ SQL และมีไดรเวอร์สำหรับฐานข้อมูลหลายประเภท เช่น SQL Server, Oracle, และ MySQL
- LINQPad มีฟีเจอร์ขั้นสูงเช่น การเติมข้อความอัตโนมัติด้วย AI, การดีบักแบบบูรณาการ, และการจัดรูปแบบผลลัพธ์ที่หลากหลาย ทำให้เป็นเครื่องมือที่ทรงพลังสำหรับการเขียนโค้ดอย่างรวดเร็วและการพัฒนาแบบไดนามิก

### [ปฏิกิริยา](https://news.ycombinator.com/item?id=40770171)

- LINQPad ได้รับการยกย่องอย่างสูงในเรื่องความง่ายในการใช้งาน โดยเฉพาะในหมู่นักพัฒนา .NET สำหรับการเขียนสคริปต์ การดีบัก และการรันคำสั่ง LINQ
- ผู้ใช้ชื่นชมคุณสมบัติต่าง ๆ เช่น การเชื่อมต่อฐานข้อมูล SQL Server แต่แสดงความไม่พอใจเกี่ยวกับการใช้งานที่จำกัดเฉพาะ Windows และค่าใช้จ่ายที่สูง
- ทางเลือกที่กล่าวถึงได้แก่ winmerge, NetPad, dotnetfiddle.net, และ RoslynPad โดยมีผู้ใช้บางคนต้องการการสนับสนุนข้ามแพลตฟอร์มและการรวมเข้ากับ Visual Studio

## [ตัวแทนจำหน่ายรถยนต์กลับมาใช้ปากกาและกระดาษหลังจากการโจมตีทางไซเบอร์ต่อผู้ให้บริการซอฟต์แวร์](https://apnews.com/article/car-dealerships-cyberattack-cdk-outage-3f7c81f6be0e212172b33cdc9f49feba)

- ตัวแทนจำหน่ายรถยนต์ในอเมริกาเหนือประสบกับการหยุดชะงักอย่างมากเนื่องจากการโจมตีทางไซเบอร์ต่อ CDK Global ซึ่งเป็นผู้ให้บริการซอฟต์แวร์หลักสำหรับตัวแทนจำหน่ายรถยนต์
- การโจมตีเมื่อวันที่ 19 มิถุนายน 2024 ทำให้เกิดการหยุดชะงักอย่างต่อเนื่อง ส่งผลให้เกิดความล่าช้าและต้องดำเนินการด้วยมือที่ตัวแทนจำหน่ายที่ได้รับผลกระทบ
- บริษัทรถยนต์รายใหญ่เช่น Stellantis, Ford และ BMW ยืนยันถึงผลกระทบแต่ยังคงดำเนินงานต่อไปโดยใช้วิธีการทางเลือก ในขณะที่ CDK Global กำลังทำงานเพื่อฟื้นฟูระบบของตน

### [ปฏิกิริยา](https://news.ycombinator.com/item?id=40776951)

- ตัวแทนจำหน่ายรถยนต์กลับมาใช้ปากกาและกระดาษหลังจากการโจมตีทางไซเบอร์ต่อผู้ให้บริการซอฟต์แวร์ของพวกเขา CDK Global ซึ่งทำให้การดำเนินงานดิจิทัลของพวกเขาถูกขัดจังหวะ
- การโจมตีทางไซเบอร์ได้ส่งผลกระทบอย่างรุนแรงต่อแผนกชิ้นส่วนและบริการมากกว่ายอดขายรถยนต์ใหม่ เนื่องจากข้อมูลสินค้าคงคลังของพวกเขาถูกเก็บไว้ในระบบ CDK ที่ถูกโจมตี
