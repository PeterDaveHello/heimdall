# 2024-01-30

## [การสร้างโปรแกรม C โดยใช้ libcurl ด้วย --libcurl Option](https://everything.curl.dev/libcurl/libcurl)

- ตัวเลือกบรรทัดคําสั่ง --libcurl ใน curl ช่วยให้ผู้ใช้สามารถสร้างโปรแกรม C ที่ใช้ libcurl สําหรับการถ่ายโอน
- โปรแกรมที่สร้างขึ้นทําหน้าที่เป็นข้อมูลอ้างอิงสําหรับการใช้ตัวเลือก libcurl และเพิ่มอาร์กิวเมนต์เพิ่มเติม
- แม้ว่ารหัสโปรแกรมจะมีตัวเลือก curl_easy_setopt ต่างๆ แต่บางตัวเลือกอาจต้องมีการใช้งานด้วยตนเอง

### [ปฏิกิริยา](https://news.ycombinator.com/item?id=39175873)

- การอภิปรายมุ่งเน้นไปที่ข้อดีของการใช้คําสั่ง curl เมื่อเทียบกับ HTTP ดิบและทางเลือกอื่นสําหรับบริการออนไลน์
- ผู้เข้าร่วมพูดคุยเกี่ยวกับคุณลักษณะและเครื่องมือ "คัดลอกเป็น cURL" สําหรับการแปลงคําสั่ง cURL ตลอดจนการดูและแก้ไขโค้ด
- การสนทนายังครอบคลุมถึงการใช้ Invoke-WebRequest ใน PowerShell ข้อจํากัดของโมเดล UNIX/POSIX ความสามารถของ libcurl และการตั้งค่าสําหรับเครื่องมือที่กําหนดมากกว่า AI

- Boardzilla เป็นเฟรมเวิร์กใหม่ที่ออกแบบมาเพื่อพัฒนาเกมกระดานบนเว็บ ซึ่งสร้างขึ้นจากความไม่พอใจกับเครื่องมือที่มีอยู่
- เฟรมเวิร์กทําให้กระบวนการพัฒนาง่ายขึ้นโดยอนุญาตให้นักพัฒนาใช้โค้ดเดียวกันสําหรับทั้งไคลเอนต์และเซิร์ฟเวอร์ ขจัดความกังวลเกี่ยวกับการจัดการสถานะ การคงอยู่ และเครือข่าย
- แม้ว่าจะยังอยู่ในระหว่างดําเนินการ แต่ Boardzilla ได้เปิดตัวเอกสารสําหรับนักพัฒนาและเกมตัวอย่างบน Github โดยเพิ่มคุณสมบัติและเกมอย่างแข็งขัน และยินดีรับคําติชมจากผู้ใช้

- หัวข้อที่กล่าวถึง ได้แก่ ความง่ายในการใช้งานความเข้ากันได้การพัฒนาแบบเนทีฟบริการของบุคคลที่สามแอปบนเว็บและประสิทธิภาพ
