## [Zed Code Editor เปิดตัวโอเพ่นซอร์ส เน้นการทํางานร่วมกัน](https://zed.dev/blog/zed-is-now-open-source)

- Zed ซึ่งเป็นโปรแกรมแก้ไขโค้ดได้เปลี่ยนเป็นโครงการโอเพ่นซอร์สเพื่อปรับปรุงผลิตภัณฑ์และมีส่วนร่วมกับชุมชนการเขียนโปรแกรม
- ขณะนี้ Zed พร้อมใช้งานบน GitHub ภายใต้ใบอนุญาต copyleft และ Apache 2 ทําให้นักพัฒนาสามารถเข้าถึงและแก้ไขโค้ดได้
