# 2024-01-27

## [Oasis: ระบบ Linux ที่เชื่อมโยงแบบคงที่โดยเน้นความเรียบง่ายและการปรับแต่ง](https://github.com/oasislinux/oasis)

- Oasis เป็นระบบปฏิบัติการบน Linux ที่คล้ายกับ BSD และเน้นความเรียบง่ายและการปรับแต่ง
- ใช้ซอฟต์แวร์ที่เชื่อมโยงแบบคงที่อย่างสมบูรณ์บิลด์ที่รวดเร็วและทําซ้ําได้และการพึ่งพาบูตสแตรปน้อยที่สุด
- Oasis ไม่มีตัวจัดการแพ็คเกจ แต่เข้ากันได้กับ pkgsrc และ nix และใช้ BearSSL เป็น TLS และไลบรารีการเข้ารหัสลับ

### [ปฏิกิริยา](https://news.ycombinator.com/item?id=39142748)

- การอภิปรายครอบคลุมหัวข้อที่เกี่ยวข้องกับระบบ Linux รวมถึงการใช้ Oasis เพื่อสร้างอิมเมจระบบปฏิบัติการที่ไม่เปลี่ยนรูป และประโยชน์ของการเชื่อมโยงแบบคงที่และแบบไดนามิกในการพัฒนาซอฟต์แวร์
- ผู้ใช้สํารวจการจัดการการพึ่งพา การคอมไพล์แบบกระจาย และความท้าทายของการสนับสนุน GPU
- มีการหารือเกี่ยวกับไลบรารีคอมไพเลอร์และโซลูชันทางเลือกต่างๆเช่น Docker และ Nix โดยมีความคิดเห็นและข้อเสนอแนะที่แตกต่างกันเกี่ยวกับการเพิ่มประสิทธิภาพไลบรารีและไบนารีการจัดการการพึ่งพาและปรับปรุงความเสถียรของระบบ

## [ICJ สั่งอิสราเอลป้องกันการฆ่าล้างเผ่าพันธุ์ในฉนวนกาซา ไม่สั่งหยุดยิง](https://apnews.com/article/israel-gaza-genocide-court-south-africa-27cf84e16082cde798395a95e9143c06)

- คําตัดสินของศาลถูกมองว่าเป็นการวิพากษ์วิจารณ์พฤติกรรมของอิสราเอลในช่วงความขัดแย้ง และได้เพิ่มแรงกดดันจากนานาชาติให้หยุดการรุกราน
- นายกรัฐมนตรีเบนจามิน เนทันยาฮู ของอิสราเอลได้ปฏิเสธคําตัดสินและให้คํามั่นว่าจะดําเนินสงครามต่อไป ซึ่งทําให้เกิดความไม่แน่นอนว่าอิสราเอลจะปฏิบัติตามคําตัดสินที่มีผลผูกพันทางกฎหมายหรือไม่
- ศาลยังเรียกร้องให้กลุ่มฮามาสปล่อยตัวประกันและเรียกร้องให้ประชาคมระหว่างประเทศรับรองว่าอิสราเอลปฏิบัติตามคําตัดสิน
- คําตัดสินนี้ตรงกับวันรําลึกถึงความหายนะสากล

## [Rclone: เครื่องมืออเนกประสงค์สําหรับจัดการไฟล์บน Cloud Storage](https://rclone.org/)

- Rclone เป็นโปรแกรมบรรทัดคําสั่งสําหรับจัดการไฟล์บนที่เก็บข้อมูลบนคลาวด์พร้อมรองรับผู้ให้บริการมากกว่า 70 ราย
- มีคุณสมบัติต่างๆ เช่น การเข้ารหัส การบีบอัด และการแฮช ทําให้เหมาะสําหรับการสํารองข้อมูล การกู้คืนข้อมูล การย้ายข้อมูล และการซิงค์ไฟล์
- Rclone รักษาการประทับเวลา ตรวจสอบเช็คซัม และสามารถใช้เป็นดิสก์เพื่อเข้าถึงที่เก็บข้อมูลบนคลาวด์ โดยให้บริการไฟล์ผ่านโปรโตคอลต่างๆ เป็นโอเพ่นซอร์สและมีชุมชนที่เป็นประโยชน์

### [ปฏิกิริยา](https://news.ycombinator.com/item?id=39151937)

- มีการกล่าวถึงเครื่องมือซิงค์ทางเลือกและตัวเลือกการเข้ารหัสสําหรับที่เก็บข้อมูลบนคลาวด์
