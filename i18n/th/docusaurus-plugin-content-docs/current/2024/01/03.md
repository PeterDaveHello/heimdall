# 2024-01-03

## [ผู้เล่น Tetris สร้างสถิติโลกใหม่ทําให้เกิดความผิดพลาดของเกมหลังจาก 34 ปี](https://www.youtube.com/watch?v=GuJ5UuknsHU)

- ผู้เล่น Tetris Blue Scuti สร้างสถิติโลกใหม่ด้วยการไปถึงเลเวล 157 และทําให้เกิดความผิดพลาดของเกมที่เรียกว่า Killscreen "True"
- ความสําเร็จนี้เหนือกว่าบันทึกและเทคนิคก่อนหน้านี้ซึ่งแสดงให้เห็นถึงความก้าวหน้าในการเล่นเกมและกลยุทธ์
- ระดับสีที่ผิดพลาดที่เข้าถึงโดย AI ชื่อ stackrabbit และการผ่านความผิดพลาดของเกมโดยไม่ได้ตั้งใจของ Blue Scuti ได้ฟื้นฟูชุมชน Tetris ซึ่งสร้างแรงบันดาลใจให้กับกลยุทธ์ใหม่และการบันทึกการไล่ล่า

### [ปฏิกิริยา](https://news.ycombinator.com/item?id=38841080)

- ความคิดเห็นครอบคลุมหัวข้อต่างๆที่เกี่ยวข้องกับเกม Tetris เช่นกลยุทธ์การเล่นการบรรลุเป้าหมายและการอธิบายคําศัพท์เฉพาะกลุ่ม
- มีการกล่าวถึงความสําเร็จของ Arbitrary Code Execution ใน NES Tetris ซึ่งแสดงให้เห็นถึงความเฉลียวฉลาดและทักษะของผู้เล่น
- หัวข้ออื่น ๆ ได้แก่ แนวคิดในการเข้าถึง "kill screen" การแสวงหาประโยชน์จากความผิดพลาดในเกมข้อ จํากัด ของฮาร์ดแวร์และซอฟต์แวร์และความเพลิดเพลินในการเล่นเกมเป็นกิจกรรมสันทนาการ

## [ซ่อม DIY Blind: แก้ไขมู่ลี่ของคุณได้อย่างง่ายดายด้วยชิ้นส่วนอะไหล่และวิดีโอแนะนํา](https://fixmyblinds.com/)

### [ปฏิกิริยา](https://news.ycombinator.com/item?id=38844274)

- การอภิปรายครอบคลุมหัวข้อต่างๆเช่นมู่ลี่บานประตูหน้าต่างและความพร้อมของชิ้นส่วนอะไหล่สําหรับการซ่อมแซมเครื่องใช้ไฟฟ้า
- ผู้เข้าร่วมแบ่งปันประสบการณ์ของพวกเขากับการซ่อมแซมด้วยตัวเองและแสดงความหงุดหงิดกับเครื่องใช้ไฟฟ้าที่ทันสมัย
- การสนทนายังรวมถึงการอภิปรายเกี่ยวกับมูลค่าของการซ่อมเครื่องใช้ไฟฟ้าเทียบกับการซื้อใหม่และกล่าวถึงเว็บไซต์เฉพาะเช่นตัวเลือก fixmyblinds.com และตาบอดที่ใช้เครื่องยนต์

## [การฟื้นคืนชีพเบราว์เซอร์ Dillo: ผู้ใช้ฟื้นโครงการหลังจากโดเมนหมดอายุ](https://dillo-browser.github.io/)

- โฮสต์ dillo.org หมดอายุในกลางปี 2022 ส่งผลให้เว็บไซต์ Dillo, repo ปรอท, รายชื่อผู้รับจดหมายและเซิร์ฟเวอร์อีเมลสําหรับนักพัฒนา Dillo หยุดทํางาน
- ผู้ใช้ก้าวเข้ามาเพื่อป้องกันไม่ให้โครงการตายโดยการอัปโหลดสําเนาของ repo ไปยัง GitHub ดําเนินการบํารุงรักษาโค้ดและสร้างเว็บเพจใหม่
- ผู้ใช้ได้ทําการปรับปรุง Dillo เช่นการเพิ่มการสนับสนุนสําหรับโปรโตคอลหลายตัวและการใช้ CI ที่เหมาะสมด้วยการทดสอบการแสดงผล พวกเขาวางแผนที่จะปล่อยเวอร์ชัน 3.1 หลังจากการทดสอบและกําลังมองหาข้อเสนอแนะและการมีส่วนร่วมจากผู้ใช้

### [ปฏิกิริยา](https://news.ycombinator.com/item?id=38847613)

- เบราว์เซอร์ Dillo ซึ่งถูกลบออกไปก่อนหน้านี้ได้รับการฟื้นฟูโดยนักพัฒนาที่อัปโหลดรหัสไปยัง GitHub
- นักพัฒนาได้ทําการปรับปรุงเบราว์เซอร์รวมถึงการสนับสนุน OpenSSL และ mbedTLS และกําลังมองหาข้อเสนอแนะและการมีส่วนร่วมจากผู้ใช้
- ผู้ใช้ชื่นชมประสิทธิภาพและความเร็วของ Dillo ในเครื่องระดับล่าง แต่พูดถึงข้อ จํากัด เช่นปัญหาการเล่นวิดีโอและช่องโหว่ในโค้ด พวกเขายังหารือเกี่ยวกับความเข้ากันได้ของเบราว์เซอร์ที่มีน้ําหนักเบากับระบบปฏิบัติการที่แตกต่างกัน

## [การเพิ่มขึ้นของ LLM และการเขียนโปรแกรมในปี 2024](http://antirez.com/news/140)

- โปรแกรม LLMs (Master of Laws) และทักษะการเขียนโปรแกรมกําลังได้รับความนิยมในต้นปี 2024
- บทความนี้กล่าวถึงเหตุผลที่อยู่เบื้องหลังความสนใจที่เพิ่มขึ้นใน LLMs และทักษะการเขียนโปรแกรม
- มันให้ข้อมูลเชิงลึกเกี่ยวกับประโยชน์และโอกาสในการทํางานที่อาจเกิดขึ้นที่เกี่ยวข้องกับพื้นที่การศึกษาเหล่านี้

### [ปฏิกิริยา](https://news.ycombinator.com/item?id=38840626)

- ผู้ใช้มีประสบการณ์ที่แตกต่างกันกับโมเดลภาษาในความช่วยเหลือด้านโค้ดพบว่ามีประโยชน์สําหรับการสร้างโค้ดและระบบอัตโนมัติ แต่ยังสังเกตเห็นข้อ จํากัด ในการครอบคลุมและความน่าเชื่อถือ
- มีการถกเถียงกันเกี่ยวกับผลกระทบของความช่วยเหลือรหัสอัตโนมัติต่อการเคลื่อนย้ายงานโดยมีความคิดเห็นที่แตกต่างกันเกี่ยวกับผลประโยชน์และความกังวลเกี่ยวกับการสูญเสียงาน
- มีการหารือเกี่ยวกับโมเดลภาษาในการทํางานร่วมกันและเครื่องมือแก้ไขโค้ด โดยผู้ใช้จะแบ่งปันประสบการณ์และแนะนําการปรับปรุง โดยรวมแล้วโมเดลภาษาช่วยเพิ่มผลผลิตและประสิทธิภาพของการเขียนโปรแกรม แต่ผู้ใช้เน้นย้ําถึงความสําคัญของความเชี่ยวชาญของมนุษย์และความระมัดระวังต่อการพึ่งพาเครื่องมือ AI มากเกินไป

## [ซีอีโอของ Mozilla เพิ่มขึ้น 20% แม้จะมีรายได้ลดลง แต่การมุ่งเน้นไปที่ AI ทําให้เกิดคําถาม](https://www.theregister.com/2024/01/02/mozilla_in_2024_ai_privacy/)

- Mitchell Baker ซีอีโอของ Mozilla ได้รับค่าตอบแทนที่สูงขึ้นแม้จะมีรายได้ลดลงทําให้เกิดการเรียกร้องให้บริษัทก้าวเร็วขึ้น
- รายงานระบุว่า AI โดยเฉพาะอย่างยิ่งในภาคการดูแลสุขภาพเป็นจุดสนใจหลักสําหรับแผนในอนาคตของ Mozilla
- มีความกังวลว่าการให้ความสําคัญกับ AI จะส่งผลให้เกิดนวัตกรรมที่แท้จริงหรือเป็นเพียงการเพิ่มข้อเสนอของ บริษัท อย่างตื้นเขิน รายงานยังเน้นย้ําถึงความทุ่มเทของ Mozilla ต่อความเป็นส่วนตัว การเข้ารหัส และความปลอดภัยของข้อมูล

### [ปฏิกิริยา](https://news.ycombinator.com/item?id=38849580)

- การอภิปรายหมุนรอบหลายประเด็นเกี่ยวกับ Mozilla และซีอีโอเช่นการวิจารณ์การจัดการความกังวลด้านรายได้ที่ลดลงความไม่พอใจกับคุณสมบัติของ Firefox การอภิปรายค่าตอบแทนซีอีโอและอนาคตของตลาดเบราว์เซอร์
- มีความคิดเห็นที่หลากหลายเกี่ยวกับ Mozilla และข้อเสนอโดยมีบางส่วนแสดงการสนับสนุนและคนอื่น ๆ แสดงความไม่พอใจและเรียกร้องให้มีการเปลี่ยนแปลง

## [การปรับปรุงการหักเงินเดือนด้วยแผนภูมิการพึ่งพา: ประสบการณ์ของเจ้าของธุรกิจชาวแคนาดา](https://www.daemonology.net/blog/2023-12-31-Canadian-payroll-dependency-chart.html)

- พวกเขาอธิบายว่าเอกสารมีความซับซ้อนและต้องการการพลิกระหว่างส่วนต่างๆอย่างต่อเนื่องเพื่อค้นหาการคํานวณที่จําเป็น

- มีการอภิปรายเกี่ยวกับความท้าทายของการนําทางกฎหมายภาษีการยื่นภาษีในฐานะผู้รับเหมาอิสระและการทําความเข้าใจเงินสมทบของนายจ้างในบัญชีเกษียณอายุ

## [ดําเนินการ OCR บน Mac โดยใช้ CLI หรือ Python ฟรี](https://blog.greg.technology/2024/01/02/how-do-you-ocr-on-a-mac.html)

- ผู้เขียนแบ่งปันคําแนะนําเกี่ยวกับวิธีการดําเนินการ OCR บน Mac โดยใช้อินเทอร์เฟซบรรทัดคําสั่งหรือ Python โดยไม่มีค่าใช้จ่ายใด ๆ ที่เกี่ยวข้อง
- มีคําแนะนําทีละขั้นตอนสําหรับการตั้งค่าบริการ OCR ภายในเครื่องโดยใช้แอพคําสั่งลัดและเรียกใช้ OCR บนรูปภาพโดยใช้ทางลัดที่สร้างขึ้น
- โพสต์ยังมีข้อมูลโค้ด Python สําหรับดําเนินการ OCR บนไฟล์

### [ปฏิกิริยา](https://news.ycombinator.com/item?id=38844943)

- ผู้ใช้พูดคุยเกี่ยวกับ OCR (การรู้จําอักขระด้วยแสง) บนแพลตฟอร์มต่างๆเช่น Mac, iOS และ macOS แบ่งปันประสบการณ์กับเครื่องมือเช่น Apple Vision และ Tesseract
- หัวข้อที่ครอบคลุม ได้แก่ ความถูกต้องการสนับสนุนภาษาการแยกตารางการรู้จําลายมือและการใช้ทางลัด

## [The Sea Wolf: การผจญภัยของ Thomas Cochrane ในกองบัญชาการกองทัพเรือและการเคลื่อนไหวทางการเมือง](https://sergey.substack.com/p/thomas-cochrane)

- Thomas Cochrane เอิร์ลที่ 10 แห่ง Dundonald มีอาชีพที่โดดเด่นในฐานะผู้บัญชาการทหารเรือและนักเคลื่อนไหวทางการเมือง
- แม้จะมีการต่อสู้ในศาลในช่วงต้นอาชีพของเขา Cochrane กลายเป็นที่รู้จักสําหรับกลยุทธ์ที่ชาญฉลาดและการซ้อมรบที่กล้าหาญในการจับเรือศัตรู
- นอกจากนี้ Cochrane ยังมีส่วนร่วมในกิจกรรมทางการเมืองรณรงค์ต่อต้านการทุจริตและดํารงตําแหน่งสมาชิกรัฐสภา
- Cochrane มีบทบาทสําคัญในสงครามประกาศอิสรภาพต่างๆ รวมถึงชิลี บราซิล และกรีซ
- แม้จะมีการโต้เถียงการจําคุกและความพ่ายแพ้ส่วนตัวตัวละครที่กล้าหาญของ Cochrane ก็ทิ้งมรดกที่ยั่งยืนไว้
- เรื่องราวของ Cochrane สอนบทเรียนสําคัญสองประการ: การใช้วัตถุระเบิดอย่างชาญฉลาดและศักยภาพในการประสบความสําเร็จในความพยายามที่ดูเหมือนสิ้นหวัง

### [ปฏิกิริยา](https://news.ycombinator.com/item?id=38842571)

- โพสต์นี้สํารวจทางเลือกของ Thomas Cochrane ในด้านการศึกษาและการเมือง ซึ่งตรงกันข้ามกับปาร์ตี้และหารือเกี่ยวกับกลยุทธ์การรณรงค์ของเขาในเขตที่ทุจริต
- มันกล่าวถึงพระราชบัญญัติการปฏิรูปปี 1832 และผลกระทบต่อรัฐสภาเช่นเดียวกับพระราชบัญญัติเลิกทาสปี 1833 และการเชื่อมต่อของ Lord Cochrane กับกองทัพเรือชิลี
- บทสนทนาเกี่ยวกับบทบาทของชาวไอริชพลัดถิ่นในการเคลื่อนไหวเพื่อเอกราชและเจาะลึกหนังสือเกี่ยวกับประวัติศาสตร์กองทัพเรือรวมถึงการเปรียบเทียบระหว่างซีรีส์ Hornblower และ Aubrey / Maturin การอภิปรายเกี่ยวกับการแสดงตัวละครและความแม่นยําทางประวัติศาสตร์และการกล่าวถึงขนาดและน้ําหนักของปืนที่ Cochrane ถือ
