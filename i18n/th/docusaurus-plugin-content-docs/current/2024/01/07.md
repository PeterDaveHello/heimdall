# 2024-01-07

## [FAA สั่งตรวจสอบเครื่องบินโบอิ้ง 737 Max 9 กว่า 170 ลําหลังเกิดเหตุแผงกลางเที่ยวบิน](https://www.cnbc.com/2024/01/06/boeing-737-max-9-grounding-after-alaska-airlines-door-blows-midflight.html)

- เอฟเอเอได้สั่งให้นําเครื่องบินโบอิ้ง 737 แม็กซ์ 9 กว่า 170 ลําลงจอดเพื่อตรวจสอบหลังจากคณะผู้วิจัยถูกปลดออกจากเที่ยวบินของสายการบินอะแลสกาแอร์ไลน์ในช่วงกลางเที่ยวบิน
- Alaska Airlines และ United Airlines ได้นําเครื่องบินโบอิ้ง 737 Max 9 ทั้งหมดลงจอดเพื่อเป็นการป้องกันไว้ก่อน
- เหตุการณ์ดังกล่าวอยู่ระหว่างการสอบสวนโดย NTSB และโชคดีที่ไม่มีรายงานการบาดเจ็บสาหัส เหตุการณ์ดังกล่าวทําให้เกิดความกังวลเกี่ยวกับความล้มเหลวของโครงสร้างที่อาจเกิดขึ้นในโบอิ้ง 737 Max ซึ่งต้องเผชิญกับการตรวจสอบอย่างเข้มข้นหลังจากเกิดอุบัติเหตุร้ายแรงสองครั้งในปี 2019

### [ปฏิกิริยา](https://news.ycombinator.com/item?id=38893909)

- การอภิปรายครอบคลุมหลายหัวข้อที่เกี่ยวข้องกับความปลอดภัยด้านการบิน รวมถึงการลงจอดของเครื่องบินโบอิ้ง 737 Max 9 และความกังวลด้านความปลอดภัยกับ Alaska Airlines
- นอกจากนี้ยังเน้นย้ําถึงความจําเป็นในการควบคุมที่เข้มงวดขึ้นในอุตสาหกรรมการบินและวิพากษ์วิจารณ์โบอิ้งว่าละเลยความปลอดภัยและการอัปเดตที่จําเป็น
- การอภิปรายขยายไปไกลกว่าการบินเพื่อรวมการอภิปรายเกี่ยวกับความสัมพันธ์ระหว่างรัฐบาลกับองค์กรการใช้จ่ายทางทหารวัฒนธรรมของ บริษัท และการลดลงของธุรกิจบางอย่าง

## [กรมสรรพากรทดลองระบบการยื่นภาษีฟรีเผชิญกับการต่อต้านและการสนับสนุนที่แบ่งแยก](https://www.nytimes.com/2024/01/05/your-money/irs-tax-filing-free-online.html)

- กรมสรรพากรกําลังทดสอบระบบการยื่นภาษีฟรีที่ช่วยให้ผู้อยู่อาศัยใน 12 รัฐสามารถยื่นแบบแสดงรายการภาษีของรัฐบาลกลางทางออนไลน์ได้โดยตรงโดยไม่มีค่าใช้จ่ายใด ๆ
- โครงการนําร่องนี้ออกแบบมาสําหรับผู้เสียภาษีที่มีรายได้น้อยและปานกลางที่มีผลตอบแทนที่ไม่ซับซ้อนโดยมีเป้าหมายเพื่อลดความซับซ้อนของขั้นตอนการยื่นภาษี
- บริษัท เตรียมภาษีเชิงพาณิชย์กําลังคัดค้านความคิดริเริ่มนี้ในขณะที่สมาชิกสภาคองเกรสจากทั้งสองฝ่ายมีความคิดเห็นที่แตกต่างกันเกี่ยวกับการสนับสนุน กรมสรรพากรจะเผยแพร่รายงานสาธารณะเกี่ยวกับการพิจารณาคดีก่อนที่จะตัดสินใจเกี่ยวกับการขยายระบบการยื่นโดยตรงไปยังผู้เสียภาษีจํานวนมาก

### [ปฏิกิริยา](https://news.ycombinator.com/item?id=38892044)

- ระบบภาษีของสหรัฐอเมริกามีความซับซ้อนและไม่มีประสิทธิภาพทําให้เกิดการอภิปรายเกี่ยวกับความจําเป็นในการลดความซับซ้อนและการปรับปรุง
- การอภิปรายมุ่งเน้นไปที่การมีส่วนร่วมของรัฐบาลในการยื่นภาษีการใช้ซอร์สโค้ดสาธารณสมบัติและการเปรียบเทียบกับระบบภาษีของประเทศอื่น ๆ
- การวิเคราะห์รวมถึงประสิทธิภาพของบริการภาครัฐการจัดสรรงบประมาณและบทบาทของหน่วยงานเอกชนพร้อมกับมุมมองเกี่ยวกับการปฏิบัติตามภาษี IRS ความปลอดภัยของข้อมูลและการศึกษาผู้เสียภาษี

## [สํารวจ Playground รหัสในเบราว์เซอร์: Codapi และพลังของรันไทม์ WASI น้ําหนักเบา](https://antonz.org/in-browser-code-playgrounds/)

- มีการกล่าวถึงสนามเด็กเล่นรหัสในเบราว์เซอร์โดยเฉพาะอย่างยิ่งที่เกี่ยวข้องกับเครื่องมือโอเพ่นซอร์สที่เรียกว่า Codapi
- รันไทม์ภาษาสามารถคอมไพล์ลงในไบนารี WebAssembly System Interface (WASI) ทําให้สามารถรันในเบราว์เซอร์ด้วยรันไทม์ WASI ที่มีน้ําหนักเบา
- Playground แสดงตัวอย่างโค้ดแบบโต้ตอบใน Python, PHP, Ruby, Lua, JavaScript, Fetch และ SQLite พร้อมด้วยคุณสมบัติขั้นสูง เช่น เทมเพลตและเซลล์โค้ด
- มีคําแนะนําเกี่ยวกับวิธีใช้สนามเด็กเล่นและประโยชน์ของการเรียกใช้โค้ดในเบราว์เซอร์จะถูกเน้น

### [ปฏิกิริยา](https://news.ycombinator.com/item?id=38891177)

- บทความนี้สํารวจสนามเด็กเล่นโค้ดในเบราว์เซอร์ต่างๆ ที่ช่วยให้ผู้ใช้สามารถเขียนและรันโค้ดได้โดยตรงในเว็บเบราว์เซอร์ของตน
- ตัวอย่างเช่น สนามเด็กเล่นสําหรับภาษาแบบแผนและสนามเด็กเล่น Python พร้อมคุณสมบัติที่มุ่งสู่การศึกษาการเขียนโปรแกรมสําหรับผู้เริ่มต้น
- บทความนี้เน้นถึงความยากลําบากในการเรียกใช้โค้ดด้วยเวลาดําเนินการที่กว้างขวางหรือลูปที่ไม่มีที่สิ้นสุดภายในเบราว์เซอร์และกล่าวถึงศักยภาพของสนามเด็กเล่นบนเว็บแอปเพื่อแสดงผลิตภัณฑ์แซนด์บ็อกซ์เพื่อวัตถุประสงค์ในการสาธิต

## [ขอแนะนํา Swieros: CPU Emulator อเนกประสงค์, C Compiler และ OS](https://github.com/rswier/swieros)

- Swieros เป็นเคอร์เนลคอมไพเลอร์และแพลตฟอร์ม userland ที่มีลักษณะคล้าย Unix ที่มีน้ําหนักเบาและรวดเร็วซึ่งใช้ xv6
- มันถูกออกแบบมาเพื่อนําเสนอสภาพแวดล้อมที่สนุกสนานและการศึกษาเพื่อวัตถุประสงค์ในการวิจัย
- Swieros มี CPU เสมือนที่รองรับโหมดผู้ใช้ / หัวหน้างานและหน่วยความจําเสมือนรวมถึงคอมไพเลอร์ที่รวดเร็วสําหรับการรวบรวมและการดําเนินการได้ทันที
- คุณสมบัติอื่น ๆ ของ Swieros ได้แก่ ไคลเอนต์เครือข่ายและเซิร์ฟเวอร์ OpenGL ระยะไกลแอปพลิเคชัน GUI และอื่น ๆ
- สามารถทํางานได้ทั้งบนระบบปฏิบัติการ Windows และ Linux
