# 2024-03-22

## [Apple เผชิญกับคดีต่อต้านการผูกขาดเรื่องการผูกขาด iPhone](https://www.nytimes.com/2024/03/21/technology/apple-doj-lawsuit-antitrust.html)

- กระทรวงยุติธรรมสหรัฐฯ และ 16 รัฐ รวมถึงดิสตริกต์ออฟโคลัมเบีย ได้ยื่นฟ้องต่อต้านการผูกขาดกับ Apple เนื่องจากถูกกล่าวหาว่าผูกขาดตลาดสมาร์ทโฟน
- คดีนี้อ้างว่าแนวทางปฏิบัติของ Apple ยับยั้งการแข่งขันและขัดขวางการพัฒนาทางเลือกกระเป๋าเงินดิจิทัล ซึ่งส่งผลกระทบต่อผู้บริโภคและนักพัฒนา
- Apple ปกป้องการกระทําของตนโดยระบุว่าพวกเขาเพิ่มความปลอดภัยของอุปกรณ์และส่งเสริมผลประโยชน์ของผู้บริโภค แต่การลงมติของคดีอาจส่งผลกระทบอย่างลึกซึ้งต่อธุรกิจของ Apple และภาคเทคโนโลยีโดยรวม

## [Difftastic: เครื่องมือ Diff โครงสร้าง Syntax-Aware](https://difftastic.wilfred.me.uk/)

- Difftastic เป็นเครื่องมือ CLI diff ที่เปรียบเทียบไฟล์ตามไวยากรณ์ ไม่ใช่ทีละบรรทัด ซึ่งช่วยให้มนุษย์ระบุความแตกต่างของโค้ดได้อย่างแม่นยําและมีประสิทธิภาพมากขึ้น
- ใช้ผู้ดูแลต้นไม้เพื่อวิเคราะห์โค้ด ทําให้สามารถรับรู้การเปลี่ยนแปลงในนิพจน์ภายในในขณะที่ไม่สนใจการเปลี่ยนแปลงการจัดรูปแบบ
- Difftastic รองรับภาษาโปรแกรมและรูปแบบไฟล์ต่างๆให้หมายเลขบรรทัดจริงแสดงภาพการปรับเปลี่ยนการห่อรวมเข้ากับ Git และเป็นโอเพ่นซอร์สทั้งหมดภายใต้ใบอนุญาต MIT ที่พัฒนาโดย Wilfred Hughes กับ Emacs และกาแฟ

## [ความฉลาดที่โดดเด่นของ. coli: ไข chemotaxis](https://jsomers.net/e-coli-chemotaxis/)

- ข้อความนี้อธิบายเคมีบําบัดในอีโคไล ซึ่งเซลล์เคลื่อนที่เข้าหาตัวดึงดูดผ่านสัญญาณเคมี ซึ่งเกี่ยวข้องกับโปรตีน เช่น CheY และ CheA เพื่อควบคุมการเคลื่อนไหวและการส่งสัญญาณ
- มอเตอร์แฟลเจลลาร์ในอีโคไลเป็นสิ่งจําเป็นสําหรับการเคลื่อนไหว โดยต้องใช้โปรตีน CheY-p หลายตัวเพื่อเปลี่ยนทิศทาง และความแตกต่างของพฤติกรรมแบคทีเรียแต่ละตัวเกิดจากระดับโปรตีนมากกว่าพันธุกรรม
- เทคนิคต่างๆ เช่น กล้องจุลทรรศน์เรืองแสงและการสร้างแบบจําลองคอมพิวเตอร์ถูกนํามาใช้เพื่อศึกษาเคมีบําบัดของแบคทีเรีย โดยมีผลทางการแพทย์ที่เป็นไปได้ในการพัฒนายาปฏิชีวนะ ช่วยเพิ่มความเข้าใจในหน้าที่ทางชีวภาพ

## [Dropflow: เอ็นจิ้นเค้าโครง CSS ขั้นสูงสําหรับคุณสมบัติสมัยใหม่](https://github.com/chearon/dropflow)

- อย่างไรก็ตาม เอ็นจิ้นยังไม่รองรับคุณลักษณะเค้าโครงขั้นสูง เช่น flexbox หรือกริด โดยจัดลําดับความสําคัญของเค้าโครงข้อความและการสนับสนุนสคริปต์ที่ไม่ใช่ภาษาละติน
- ปัจจุบันใช้ใน CellEngine สําหรับไลบรารีสเปรดชีตแคนวาสการใช้งานในอนาคตของเอ็นจิ้นเกี่ยวข้องกับการเรนเดอร์ PDF อย่างมีประสิทธิภาพ

- ไมโครพลาสติกที่ย่อยสลายได้ทางชีวภาพเหล่านี้ปราศจากเศษที่เป็นอันตรายมีศักยภาพในการต่อสู้กับมลภาวะต่อสิ่งแวดล้อมด้วยความพยายามอย่างต่อเนื่องในการทําให้สามารถใช้งานได้จริงสําหรับการผลิตในวงกว้าง

### [ปฏิกิริยา](https://news.ycombinator.com/item?id=39777898)

- พอลิเมอร์จากพืช โดยเฉพาะโพลีเมอร์จากสาหร่ายจาก UCSD สามารถย่อยสลายได้ทางชีวภาพภายใน 7 เดือน ซึ่งแตกต่างจากพลาสติกทั่วไปซึ่งอาจใช้เวลาหลายร้อยถึงหลายพันปีในการย่อยสลาย
- การอภิปรายเกี่ยวกับพลาสติกที่ย่อยสลายได้ทางชีวภาพเกี่ยวกับผลกระทบต่อมลพิษไมโครพลาสติกและเงื่อนไขที่จําเป็นสําหรับการย่อยสลายอย่างมีประสิทธิภาพ
- การอภิปรายเกี่ยวข้องกับบทบาทของรัฐบาลประชาธิปไตยการตัดสินใจของแต่ละบุคคลและธุรกิจในการต่อสู้กับมลพิษพลาสติกการส่งเสริมความยั่งยืนการจัดการกับผลกระทบต่อสิ่งแวดล้อมของพลาสติกความล้าสมัยตามแผนและการหาทางเลือกในการจัดการกับการสะสมของขยะพลาสติก
