# 2024-03-30

## [แบ็คดอร์ที่สําคัญค้นพบใน xz/liblzma คุกคามเซิร์ฟเวอร์ SSH](https://www.openwall.com/lists/oss-security/2024/03/29/4)

- โครงการ Openwall นําเสนอผลิตภัณฑ์โอเพ่นซอร์สฟรีเพื่อความปลอดภัยของเซิร์ฟเวอร์ เช่น Linux OS, แคร็กเกอร์รหัสผ่าน และเครื่องมือแฮชรหัสผ่าน
- มีการค้นพบแบ็คดอร์ในแพ็คเกจ xz/liblzma ซึ่งส่งผลกระทบต่อเซิร์ฟเวอร์ SSH บนระบบ x86-64 Linux ที่ใช้ glibc ทําให้เกิดความเสี่ยงต่อการเข้าถึงโดยไม่ได้รับอนุญาตหรือการเรียกใช้โค้ดจากระยะไกล
- โค้ด Exploit กําลังกําหนดเป้าหมายไลบรารีเฉพาะ โดยแจ้งให้การแจกจ่าย เช่น Debian และ Red Hat แก้ไขปัญหา ระบบที่มีช่องโหว่จําเป็นต้องได้รับการอัพเกรดอย่างเร่งด่วน

### [ปฏิกิริยา](https://news.ycombinator.com/item?id=39865810)

- แบ็คดอร์ในต้นน้ํา xz/liblzma นําไปสู่การประนีประนอมของเซิร์ฟเวอร์ SSH ส่งผลให้มีการระงับบัญชีและการลบคีย์ของผู้เขียนออกจากที่เก็บ ทําให้เกิดการอภิปรายเกี่ยวกับความซับซ้อนของโค้ดและการมีส่วนร่วมของนักแสดงของรัฐที่อาจเกิดขึ้นในโครงการโอเพ่นซอร์ส
- ความกังวลเกิดขึ้นเกี่ยวกับความปลอดภัยของรูปแบบไฟล์ XZ ในลีนุกซ์ดิสทริบิวชันขององค์กรที่ขาดการอัปเดตในทันที โดยเน้นย้ําถึงความสําคัญของการตรวจสอบสิทธิ์แบบหลายปัจจัยด้วยวิธีการต่างๆ เช่น YubiKeys และการจัดเก็บรหัสผ่านการกู้คืน TOTP สําหรับกรณีฉุกเฉิน
- การอภิปรายเกี่ยวกับประสิทธิภาพของการรับรองความถูกต้องด้วยสองปัจจัยข้อ จํากัด ของ MFA ความเสี่ยงในการจัดเก็บรหัสผ่านและโทเค็นบนอุปกรณ์เดียวกันการอภิปรายเกี่ยวกับการใช้งานพาสคีย์คีย์ฮาร์ดแวร์สําหรับการตรวจสอบสิทธิ์และความโปร่งใสในแนวทางปฏิบัติในการเข้ารหัสพร้อมกับการเรียกร้องของชุมชนให้ระมัดระวังและจัดการกับช่องโหว่ด้านความปลอดภัยอย่างรวดเร็ว

## [Apache Guacamole: เข้าถึงเดสก์ท็อปได้ทุกที่ด้วย Clientless Gateway](https://guacamole.apache.org/)

- Apache Guacamole เป็นเกตเวย์เดสก์ท็อประยะไกลแบบไร้ไคลเอ็นต์ที่รองรับโปรโตคอล เช่น VNC, RDP และ SSH ซึ่งสามารถเข้าถึงได้ผ่านเว็บเบราว์เซอร์สําหรับการเข้าถึงเดสก์ท็อประยะไกล
- ซอฟต์แวร์นี้เป็นโอเพ่นซอร์สภายใต้ใบอนุญาต Apache ซึ่งได้รับการปรับปรุงอย่างต่อเนื่องโดยชุมชนนักพัฒนา โดยมี API ที่เป็นเอกสารสําหรับการผสานรวมกับแอปพลิเคชันต่างๆ อย่างราบรื่น
- มีตัวเลือกการสนับสนุนทั้งชุมชนและเชิงพาณิชย์สําหรับ Apache Guacamole

### [ปฏิกิริยา](https://news.ycombinator.com/item?id=39867702)

- ผู้ใช้ได้รายงานปัญหาต่างๆ เช่น ความล่าช้าในการป้อนข้อมูลและคุณภาพเสียงขณะใช้ Guacamole แต่ได้แบ่งปันประสบการณ์เชิงบวกในการศึกษาและสถานที่ทํางาน

## [การรั่วไหลของปุ๋ยในไอโอวาทําลายปลาในแม่น้ํา 60 ไมล์](https://www.nytimes.com/2024/03/29/us/iowa-spill-fish-kill.html)

- การรั่วไหลเน้นย้ําถึงความยากลําบากในการบังคับใช้กฎระเบียบที่เข้มงวดขึ้นในรัฐเกษตรกรรม โดยให้ความกระจ่างเกี่ยวกับความท้าทายด้านมลพิษทางน้ําที่กําลังดําเนินอยู่

## [เปิดเผย xz-utils Backdoor: คําแนะนําด้านความปลอดภัยเร่งด่วน](https://gist.github.com/thesamesam/223949d5a074ebc3dce9ee78baad9e27)

- พบแบ็คดอร์ใน xz-utils เมื่อวันที่ 29 มีนาคม 2024 ซึ่งส่งผลกระทบต่อระบบที่มีเวอร์ชัน 5.6.0 หรือ 5.6.1 ของ xz หรือ liblzma ซึ่งถูกกระตุ้นโดยระบบที่ไม่มีสิทธิพิเศษระยะไกลที่เชื่อมต่อกับพอร์ต SSH สาธารณะ

- ผู้ดูแล xz-utils กําลังทํางานอย่างแข็งขันกับแพตช์ โดยเน้นย้ําถึงความเร่งด่วนสําหรับผู้ใช้ที่มี SSH ที่เข้าถึงได้แบบสาธารณะเพื่ออัปเดตระบบของตนทันที

### [ปฏิกิริยา](https://news.ycombinator.com/item?id=39869068)
