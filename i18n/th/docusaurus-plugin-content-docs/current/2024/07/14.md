## [หลังจากที่ปฏิเสธในตอนแรก Apple ได้อนุมัติตัวจำลอง PC ตัวแรกสำหรับ iOS](https://www.theverge.com/2024/7/13/24198015/apple-utm-se-pc-os-emulator-for-ios)

- Apple ได้อนุมัติ UTM SE ซึ่งเป็นโปรแกรมจำลองพีซีตัวแรกสำหรับ iOS ที่ช่วยให้ผู้ใช้สามารถรันซอฟต์แวร์คลาสสิกและเกมเก่าๆ สำหรับ Windows, Mac OS 9 และ Linux บน iPhone ได้
- UTM SE ซึ่งสร้างจาก QEMU รองรับโหมด VGA และเทอร์มินัล และจำลองสถาปัตยกรรม x86, PPC และ RISC-V แต่ไม่รวมระบบปฏิบัติการใด ๆ
- การอนุมัติแอปหลังจากการปฏิเสธครั้งแรกได้รับความช่วยเหลือจากทีม AltStore และการนำ QEMU TCTI ของนักพัฒนาอีกคนหนึ่งมาใช้ และตอนนี้แอปนี้สามารถใช้งานได้ฟรีบน iOS, iPadOS และ visionOS

### [ปฏิกิริยา](https://news.ycombinator.com/item?id=40958465)

- แอปเปิลได้อนุมัติอีมูเลเตอร์พีซีตัวแรกสำหรับ iOS ชื่อ UTM หลังจากที่เคยปฏิเสธไปในตอนแรก ทำให้ผู้ใช้ตื่นเต้นกันอย่างมาก
- การอนุมัตินี้เกิดจากกฎระเบียบของสหภาพยุโรป ซึ่งเน้นถึงการอภิปรายอย่างต่อเนื่องเกี่ยวกับสิทธิของผู้บริโภค ความสามารถในการซ่อมแซม และการเป็นเจ้าของในอุปกรณ์อิเล็กทรอนิกส์สมัยใหม่
- UTM ไม่มีการคอมไพล์แบบ Just-In-Time (JIT) เนื่องจากข้อจำกัดของ Apple ทำให้ประสิทธิภาพช้าลง แต่ยังสามารถรันระบบปฏิบัติการต่างๆ รวมถึง Linux 64 บิต บน iPhone ที่ไม่ได้เจลเบรกได้

## [ไม่มีความคาดหวังที่สมเหตุสมผลเกี่ยวกับความเป็นส่วนตัวในข้อมูลตำแหน่งของ Google](https://fourthamendment.com/?p=58338)

- ศาลวงจรที่สี่ตัดสินว่าไม่มีความคาดหวังที่สมเหตุสมผลในเรื่องความเป็นส่วนตัว (REP) ในข้อมูลตำแหน่งของ Google ของบุคคล เนื่องจากผู้ใช้ยินยอมที่จะแบ่งปันข้อมูลนี้กับ Google
- คำตัดสินของศาลเน้นย้ำว่าการใช้ geofencing ซึ่งเป็นวิธีที่หน่วยงานบังคับใช้กฎหมายใช้ในการรวบรวมข้อมูลตำแหน่ง ไม่ถือว่าเป็นการค้นหาภายใต้การแก้ไขครั้งที่สี่
- คำตัดสินนี้มีความสำคัญเนื่องจากเป็นการตอบสนองต่อการใช้หมายค้นพื้นที่ทางภูมิศาสตร์ที่เพิ่มขึ้นโดยหน่วยงานบังคับใช้กฎหมายตั้งแต่ปี 2016 และขั้นตอนของ Google ในการปกป้องความเป็นส่วนตัวของผู้ใช้

### [ปฏิกิริยา](https://news.ycombinator.com/item?id=40958458)

- Google กำลังเปลี่ยนวิธีการจัดการข้อมูลตำแหน่งโดยการส่งข้อมูลไปยังอุปกรณ์และลบออกจากเซิร์ฟเวอร์ ซึ่งหมายความว่าจะไม่ตอบสนองต่อหมายค้นพื้นที่จากหน่วยงานบังคับใช้กฎหมายอีกต่อไป
- การเปลี่ยนแปลงนี้ได้ก่อให้เกิดปฏิกิริยาที่หลากหลาย โดยบางผู้ใช้ชื่นชมความเป็นส่วนตัวที่เพิ่มขึ้น ในขณะที่บางคนกังวลเกี่ยวกับการสูญเสียการเข้าถึงข้อมูลไทม์ไลน์ของตน
- การสำรองข้อมูลที่เข้ารหัสสำหรับข้อมูลตำแหน่งมีให้ใช้งานแต่ถูกปิดใช้งานโดยค่าเริ่มต้น ซึ่งต้องเปิดใช้งานด้วยตนเอง ซึ่งผู้ใช้บางคนพบว่าไม่เป็นธรรมชาติ

## [ทรัมป์ถูกยิงที่การชุมนุมในเพนซิลเวเนีย](https://apnews.com/article/trump-vp-vance-rubio-7c7ba6b99b5f38d2d840ed95b2fdc3e5)

- โดนัลด์ ทรัมป์ ถูกรีบนำตัวออกจากเวทีที่การชุมนุมในบัตเลอร์ รัฐเพนซิลเวเนีย หลังจากมีเสียงปืนดังขึ้น; เขาถูกยิงที่หูแต่รายงานระบุว่าเขาปลอดภัยดี
- มือปืน โทมัส แมทธิว ครุกส์ ถูกเจ้าหน้าที่หน่วยสืบราชการลับยิงเสียชีวิต; มีผู้เข้าร่วมงานเสียชีวิตหนึ่งราย และบาดเจ็บสาหัสสองราย
- เหตุการณ์นี้ ซึ่งเป็นความพยายามลอบสังหารที่ร้ายแรงที่สุดนับตั้งแต่เหตุการณ์ของโรนัลด์ เรแกนในปี 1981 ได้เพิ่มความกังวลเกี่ยวกับความรุนแรงทางการเมืองก่อนการเลือกตั้งประธานาธิบดี

### [ปฏิกิริยา](https://news.ycombinator.com/item?id=40957564)

- อดีตประธานาธิบดีทรัมป์ถูกยิงที่การชุมนุมในเพนซิลเวเนีย จุดประกายการถกเถียงเกี่ยวกับประชาธิปไตย ความรุนแรงทางการเมือง และสิทธิในการครอบครองอาวุธปืน
- การอภิปรายรวมถึงบริบททางประวัติศาสตร์ของความรุนแรงทางการเมืองในอเมริกา การแก้ไขเพิ่มเติมครั้งที่สอง และบทบาทของสื่อสังคมออนไลน์ในการแบ่งแยกทางการเมือง
- มือปืนซึ่งรายงานว่าเป็นสมาชิกพรรครีพับลิกัน ได้สร้างความกังวลเกี่ยวกับความรุนแรงที่เพิ่มขึ้นและผลกระทบที่อาจเกิดขึ้นต่อการเลือกตั้งที่กำลังจะมาถึง

## [Talos: ระบบปฏิบัติการลินุกซ์ที่ปลอดภัย ไม่เปลี่ยนแปลง และมีขนาดเล็กสำหรับการรัน Kubernetes](https://www.talos.dev)

- Talos Linux เป็นระบบปฏิบัติการลินุกซ์ที่ปลอดภัย, ไม่เปลี่ยนแปลง, และมีขนาดเล็ก ซึ่งออกแบบมาเฉพาะสำหรับ Kubernetes รองรับแพลตฟอร์มคลาวด์, ฮาร์ดแวร์เปล่า, และแพลตฟอร์มการจำลองเสมือน โดยการจัดการระบบทำผ่าน API
- มันเน้นความปลอดภัย, ความสามารถในการคาดการณ์, และความสามารถในการพัฒนาโดยการเป็นระบบที่เรียบง่าย, แข็งแกร่ง, และไม่เปลี่ยนแปลง, โดยการเข้าถึง API ทั้งหมดจะถูกป้องกันด้วยการยืนยันตัวตนแบบ mutual TLS (mTLS).
- Talos Linux พร้อมใช้งานในระดับการผลิต รองรับคลัสเตอร์ Kubernetes ขนาดใหญ่ และสามารถเปิดใช้งานบนแล็ปท็อปภายใน Docker ได้ในเวลาเพียง 3 นาที ทำให้มันเข้าถึงได้ง่ายและมีประสิทธิภาพสูงสำหรับนักพัฒนา

### [ปฏิกิริยา](https://news.ycombinator.com/item?id=40956546)

- Talos เป็นระบบปฏิบัติการ Linux ที่ปลอดภัย ไม่เปลี่ยนแปลง และมีขนาดเล็ก ซึ่งออกแบบมาเฉพาะสำหรับการรัน Kubernetes โดยทำให้การบำรุงรักษาระบบง่ายขึ้นผ่านการกำหนดค่า การอัปเกรด และการดีบักที่ใช้ API
- Talos ต้องการดิสก์เต็มสำหรับการติดตั้ง ซึ่งอาจเป็นเรื่องท้าทายสำหรับเซิร์ฟเวอร์ที่มีพื้นที่เก็บข้อมูลจำกัด แต่ผู้ใช้ได้สำรวจวิธีแก้ปัญหาเช่น PXE, การจำลองเสมือน และการเก็บข้อมูลภายนอก
- Talos ได้รับการยกย่องในเรื่องความง่ายในการตั้งค่าและการบำรุงรักษา รองรับ Raspberry Pi 4b และมีบทเรียนบนช่อง YouTube ของตน
