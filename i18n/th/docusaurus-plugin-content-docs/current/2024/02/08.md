## [John Walker ผู้ก่อตั้ง Autodesk เสียชีวิต](https://scanalyst.fourmilab.ch/t/john-walker-1949-2024/4305)

- จอห์นเป็นนักวิทยาศาสตร์คอมพิวเตอร์และวิศวกรไฟฟ้าที่เสียชีวิตเมื่อวันที่ 2 กุมภาพันธ์ พ.ศ. 2024
- เขาทํางานที่ศูนย์คอมพิวเตอร์ของ Case Western Reserve University และต่อมาได้ก่อตั้ง Autodesk
- การเดินทางและการทํางานของจอห์นสามารถสํารวจได้ผ่านหนังสือของเขา The Autodesk File และเว็บไซต์ Fourmilab ของเขา

### [ปฏิกิริยา](https://news.ycombinator.com/item?id=39297185)

- John Walker ผู้ก่อตั้ง Autodesk เสียชีวิตแล้ว โดยทิ้งมรดกในการพัฒนาซอฟต์แวร์และหนังสือ "The Hackers Diet" ของเขาไว้เบื้องหลัง
- ผู้แสดงความคิดเห็นพูดคุยเกี่ยวกับปีแรก ๆ ของ Autodesk และการเปลี่ยนแปลงไปสู่อุตสาหกรรมที่เน้นธุรกิจมากขึ้น
- ผลกระทบของวอล์คเกอร์ต่อชุมชนเทคโนโลยีได้รับการยอมรับและมีการสนทนาเกี่ยวกับ CICO สําหรับการลดน้ําหนักการติดตามความคืบหน้าและความสําคัญของการออกกําลังกาย

## [การสร้างเครื่องมือค้นหา Python ด้วยรหัสอะซิงโครนัสและอัลกอริทึม BM25](https://www.alexmolas.com/2024/02/05/a-search-engine-in-80-lines.html)

- แอป FastAPI ขนาดเล็กถูกสร้างขึ้นเพื่อเชื่อมต่อกับเครื่องมือค้นหา แม้ว่าคุณลักษณะขั้นสูงบางอย่างจะหายไป
- แผนในอนาคตรวมถึงการเพิ่มความสามารถในการค้นหาความหมายและสนับสนุนให้ผู้อื่นเพิ่มฟีด RSS ลงในบล็อกของพวกเขา
- ผู้เขียนได้รับความเข้าใจที่ดีขึ้นเกี่ยวกับ Solr และประโยชน์ของโค้ดแบบอะซิงโครนัส

### [ปฏิกิริยา](https://news.ycombinator.com/item?id=39293050)
