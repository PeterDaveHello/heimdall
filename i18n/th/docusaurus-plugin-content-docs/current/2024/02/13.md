# 2024-02-13

## [Cloudflare มีชัยเหนือ Sable โทรลล์สิทธิบัตรในการพิจารณาคดี](https://blog.cloudflare.com/cloudflare-defeats-patent-troll-sable-at-trial)

- Cloudflare ได้รับชัยชนะในการพิจารณาคดีกับโทรลล์สิทธิบัตร Sable IP และ Sable Networks
- คณะลูกขุนสรุปว่า Cloudflare ไม่ได้ละเมิดสิทธิบัตรที่ยืนยันและประกาศว่าการเรียกร้องสิทธิบัตรของ Sable ไม่ถูกต้อง
- ชัยชนะดังกล่าวอาจเกิดจากความพยายามของทีมกฎหมาย ที่ปรึกษาภายนอก และผู้เข้าร่วมในโครงการริเริ่ม Project Jengo ของ Cloudflare ซึ่งให้รางวัลแก่บุคคลสําหรับการให้หลักฐานที่ทําให้การเรียกร้องสิทธิบัตรของโทรลล์เป็นโมฆะ
- ความสําเร็จของ Cloudflare ในการทําให้สิทธิบัตร Sable สามส่วนเป็นโมฆะได้จํากัดความสามารถในการยื่นฟ้องบริษัทอื่น
- คดีนี้เน้นย้ําถึงค่าใช้จ่ายและความเสี่ยงสูงที่เกี่ยวข้องกับการฟ้องร้องสิทธิบัตร และส่งคําเตือนไปยังโทรลล์สิทธิบัตรว่า Cloudflare จะไม่ถูกข่มขู่โดยง่าย

### [ปฏิกิริยา](https://news.ycombinator.com/item?id=39344956)

- การอภิปรายครอบคลุมหลายแง่มุมของระบบสิทธิบัตร เช่น คุณภาพสิทธิบัตร โทรลล์สิทธิบัตร สิทธิบัตรซอฟต์แวร์ การเก็บภาษีทรัพย์สินทางปัญญา ระยะเวลาสิทธิบัตร และบทบาทของสิทธิบัตรในนวัตกรรม
- ข้อเสนอแนะสําหรับการปรับปรุงรวมถึงการจูงใจให้ บริษัท ต่างๆป้องกันสิทธิบัตรที่ไม่ถูกต้องเพิ่มเงินทุนสําหรับสํานักงานสิทธิบัตรลงโทษการยื่นจดสิทธิบัตรโดยไม่สุจริตและแก้ไขปัญหาของ บริษัท ที่สร้างหน่วยงานเชลล์
- มีการถกเถียงกันเกี่ยวกับวัตถุประสงค์และประสิทธิผลของสิทธิบัตร โดยบางคนสนับสนุนการกําจัดสิทธิบัตร และบางคนเน้นย้ําถึงบทบาทในการปกป้องนักประดิษฐ์และส่งเสริมการพัฒนา นอกจากนี้ยังมีการกล่าวถึงผลกระทบของสิทธิบัตรในอุตสาหกรรมต่างๆ เช่น ยาและเทคโนโลยี

## [AMD ให้ทุนสนับสนุนการใช้งาน CUDA แบบโอเพ่นซอร์สสําหรับ AMD GPUs บน ROCm](https://www.phoronix.com/review/radeon-cuda-zluda)

- AMD ได้สนับสนุนทางการเงินในการพัฒนาการใช้งาน CUDA แบบดรอปอินแบบโอเพ่นซอร์สที่เรียกว่า ZLUDA ซึ่งช่วยให้แอปพลิเคชัน NVIDIA CUDA ทํางานบน GPU AMD Radeon โดยไม่ต้องแก้ไขซอร์สโค้ด
- เดิมทีพัฒนาขึ้นสําหรับกราฟิก Intel ZLUDA ได้รับการดัดแปลงเพื่อใช้กับ GPU ของ AMD แต่ก็ไม่ปลอดภัยอย่างสมบูรณ์และขาดการรองรับ NVIDIA OptiX อย่างเต็มที่
- การเปิดตัวโอเพนซอร์สของ ZLUDA ช่วยให้สามารถระบุสตริงกราฟิกการ์ด Radeon จริงแทนที่จะเป็นป้ายกํากับ "อุปกรณ์กราฟิก" ทั่วไป

### [ปฏิกิริยา](https://news.ycombinator.com/item?id=39344815)

- การอภิปรายเกี่ยวกับความพยายามของ AMD ในการแข่งขันกับ NVIDIA ในตลาด GPU ส่วนใหญ่อยู่ในการเรียนรู้ของเครื่องและความเข้ากันได้ของซอฟต์แวร์
- มีความกังวลเกี่ยวกับการครอบงําอย่างท่วมท้นของ CUDA และความยากลําบากในการเลียนแบบโดยเน้นย้ําถึงความจําเป็นในการใช้มาตรฐานแบบเปิด
- การสนทนายังกล่าวถึงความท้าทายด้านซอฟต์แวร์และการสนับสนุนไดรเวอร์ของ AMD และเน้นย้ําถึงความสําคัญของระบบปฏิบัติการในด้าน AI มีการเรียกร้องทั่วไปเพื่อเพิ่มการเข้าถึง การทํางานร่วมกัน และการแข่งขันในอุตสาหกรรม

- ผู้เขียนยอมรับความผิดพลาดในการมองข้ามเทคโนโลยี Zigbee และ Z-Wave ในการตั้งค่าบ้านอัจฉริยะ และยกย่องความน่าเชื่อถือเมื่อเทียบกับมาตรฐานอื่นๆ
- พวกเขาเน้นหลอดไฟ Philips Hue เป็นตัวอย่างของอุปกรณ์ที่ทํางานบนเครือข่ายของตัวเองอย่างสม่ําเสมอและราบรื่น
- ผู้เขียนยังแบ่งปันประสบการณ์เชิงบวกในการตรวจสอบฮับ Homey Pro ซึ่งใช้เทคโนโลยี Zigbee และ Z-Wave

### [ปฏิกิริยา](https://news.ycombinator.com/item?id=39344786)

- การสนทนาเกี่ยวกับข้อดีและข้อเสียของการใช้ Zigbee, Z-Wave, WiFi, Thread และ Matter สําหรับอุปกรณ์สมาร์ทโฮม
- ผู้ใช้แบ่งปันประสบการณ์และความคิดเห็นเกี่ยวกับความน่าเชื่อถือการทํางานร่วมกันความเข้ากันได้และข้อกังวลด้านความเป็นส่วนตัวที่เกี่ยวข้องกับโปรโตคอลและอุปกรณ์ต่างๆ
- นอกจากนี้ยังมีการหารือเกี่ยวกับปัจจัยสําคัญ เช่น เครือข่ายแบบเมช อายุการใช้งานแบตเตอรี่ และการควบคุมในพื้นที่ แต่ไม่มีข้อตกลงเป็นเอกฉันท์ในหมู่ผู้ใช้ เนื่องจากความชอบและประสบการณ์ของพวกเขากับแต่ละเทคโนโลยีแตกต่างกันไป
