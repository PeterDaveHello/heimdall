# 2024-05-03

## [การสร้าง "ShapeUp": ตัวสร้างแบบจําลอง 3 มิติใน C](https://danielchasehooper.com/posts/shapeup/)

- Daniel Hooper พัฒนาตัวสร้างแบบจําลอง 3 มิติชื่อ "ShapeUp" ในระหว่างกิจกรรมการเขียนโปรแกรมเป็นเวลาหนึ่งสัปดาห์โดยใช้ฟิลด์ระยะทางที่ลงนาม C และ ray marched เพื่อความเป็นไปได้
- เขาจัดการรูปร่างด้วยการจัดสรรหน่วยความจําแบบคงที่และรวมอินเทอร์เฟซผู้ใช้เครื่องกําเนิด GLSL shader การเลือกเมาส์และการส่งออกลูกบาศก์การเดินขบวน
- แม้จะมีความท้าทายกับไลบรารี raylib แต่ Hooper ก็เสร็จสิ้นโครงการ โดยผลิตตัวสร้างแบบจําลอง 3 มิติที่ใช้งานได้ด้วยโค้ด C 2300 บรรทัดและโค้ด GLSL 250 บรรทัด ซึ่งพร้อมใช้งานออนไลน์สําหรับการดูและทดสอบ

### [ปฏิกิริยา](https://news.ycombinator.com/item?id=40239164)

- โพสต์เจาะลึกถึงข้อจํากัด ความท้าทาย และข้อดีของการใช้ Raylib ในการพัฒนาเกม สัมผัสกับปัญหาแบบเต็มหน้าจอ
- ผู้ใช้แสดงความต้องการโหมดเต็มหน้าจอแบบไร้ขอบเมื่อเทียบกับโหมดเต็มหน้าจอจริงในขณะที่พูดคุยเกี่ยวกับการเลื่อนที่ราบรื่นในกราฟิก 2 มิติบนเบราว์เซอร์ด้วย WebAssembly
- หัวข้อต่างๆ เช่น การตรวจสอบพารามิเตอร์ใน C/C++ ความสําคัญของเอกสารที่ชัดเจนในการเขียนโปรแกรม C ประโยชน์ของ C ในการสร้างแบบจําลอง/การแสดงผล และการใช้ Signed Distance Functions (SDF) ยังครอบคลุม พร้อมด้วยการกล่าวถึงเครื่องมือ CAD ขั้นสูง การจัดการหน่วยความจําใน Java กิจกรรมการพัฒนาเกมที่กําลังจะมีขึ้น การเรนเดอร์ SDF ใน Solar Ash และข้อดีของการใช้ C/C++ ในการพัฒนาเกม

## [คาร์ดิโอฟิตเนส: ตัวทํานายที่สําคัญของความเสี่ยงต่อสุขภาพ](https://bjsm.bmj.com/content/58/10/556)

- บทความนี้ประเมินการศึกษาจํานวนมากที่ตรวจสอบว่าสมรรถภาพหัวใจและหลอดเลือด (CRF) ส่งผลต่อการเสียชีวิตโรคเรื้อรังและความเป็นอยู่โดยรวมอย่างไรโดยเน้นย้ําถึงความสําคัญของการออกกําลังกายเป็นประจําในการรักษา CRF ที่ดี
- โดยเน้นย้ําถึงลักษณะการทํานายที่แข็งแกร่งของ CRF สําหรับผลลัพธ์ด้านสุขภาพต่างๆ โดยแนะนําการวัด CRF ตามปกติในการตั้งค่าทางคลินิกสําหรับการประเมินสุขภาพและการแทรกแซง
- จําเป็นต้องมีการวิจัยเพิ่มเติมเพื่อแก้ไขช่องว่างที่มีอยู่ในวรรณกรรมและกําหนดความเชื่อมโยงเชิงสาเหตุระหว่าง CRF กับผลลัพธ์ด้านสุขภาพ

### [ปฏิกิริยา](https://news.ycombinator.com/item?id=40243238)

- เน้นความสําคัญของการออกกําลังกายแบบคาร์ดิโอเพื่อสุขภาพโดยรวมและอายุยืน โดยแนะนําการฝึกโซน 2 เพื่อสร้างสมรรถภาพแบบแอโรบิคอย่างต่อเนื่อง
- คาร์ดิโอที่สม่ําเสมอในจังหวะที่ง่ายสามารถป้องกันการบาดเจ็บเพิ่มความอดทนและเพิ่มประโยชน์สูงสุด กิจวัตรที่แตกต่างกันและการหลีกเลี่ยงการผสมความเข้มข้นภายในเซสชั่นเดียวกันเป็นสิ่งสําคัญ
- การถกเถียงกันอย่างต่อเนื่องรวมถึงประสิทธิภาพของการสวมหมวกกันน็อคขณะปั่นจักรยานสํารวจความเชื่อมโยงระหว่างการออกกําลังกายระดับวิตามินดีและการเสียชีวิตผ่านการทดลองที่มีการควบคุมโดยเน้นย้ําถึงความจําเป็นในการใช้แนวทางที่ครอบคลุมในการออกกําลังกายและสุขภาพ

## [การพัฒนา "sqlite-vec": โซลูชันการค้นหาเวกเตอร์ใหม่ใน SQLite](https://alexgarcia.xyz/blog/2024/building-new-vector-search-sqlite/index.html)

- Alex Garcia แนะนํา sqlite-vec ซึ่งเป็นส่วนขยาย SQLite ใหม่สําหรับการค้นหาเวกเตอร์ที่ออกแบบมาเพื่อแทนที่ sqlite-vss โดยนําเสนอฟังก์ชัน SQL แบบกําหนดเองและตารางเสมือน
- sqlite-vec เขียนด้วย C เพียงอย่างเดียวกําหนดเป้าหมายความเร็วและความสามารถในการฝังตัวโดยมีการสนับสนุนข้ามแพลตฟอร์มสําหรับ WebAssembly และอุปกรณ์มือถือพร้อมกับการจัดการหน่วยความจําที่ดีขึ้น
- ส่วนขยายที่กําลังดําเนินการอยู่มีกําหนดสําหรับการเปิดตัวครั้งแรกที่กําลังจะมาถึง นักพัฒนากําลังมองหาโอกาสในการสนับสนุนเพื่อพัฒนาโครงการ

### [ปฏิกิริยา](https://news.ycombinator.com/item?id=40243168)

- ผู้เขียนกําลังพัฒนาส่วนขยาย SQLite การค้นหาเวกเตอร์ใหม่ที่เน้นการสแกนเชิงเส้นในขั้นต้นโดยมีแผนในอนาคตสําหรับการจัดทําดัชนีขั้นสูงการประมวลผลเวกเตอร์จํานวนมากอย่างรวดเร็วและการตรวจสอบปริมาณเพื่อการค้นหาที่รวดเร็วยิ่งขึ้น
- ผู้ใช้กําลังแลกเปลี่ยนความคิดเห็นเกี่ยวกับโครงการที่เกี่ยวข้องและการเผชิญหน้ากับโซลูชันการจัดเก็บเวกเตอร์ในขณะที่โพสต์ครอบคลุมเวกเตอร์ไบนารีการหาปริมาณโมเดลการเรียนรู้ของเครื่องและข้อดีของการค้นหาเวกเตอร์ในฐานข้อมูลเช่น SQLite
- ข้อเสนอแนะเกี่ยวกับโครงการเป็นไปในเชิงบวกโดยผู้ใช้กระตือรือร้นที่จะมีส่วนร่วมและเพิ่มพูนความรู้เกี่ยวกับส่วนขยาย SQLite

- Newchip ซึ่งเป็นตัวเร่งสตาร์ทอัพประกาศล้มละลายนําไปสู่การประมูลใบสําคัญแสดงสิทธิจากสตาร์ทอัพกว่า 1,000 รายทําให้ผู้ก่อตั้งอย่าง Lacey Hunter แห่ง TechAid สูญเสียบริษัทของพวกเขา
- แอนดรูว์ ไรอัน ซีอีโอได้รับการวิพากษ์วิจารณ์เกี่ยวกับรูปแบบการจัดการของเขา และพลาดโอกาสในการจัดการใบสําคัญแสดงสิทธิ ซึ่งมีมูลค่า 500 ล้านดอลลาร์
- Deal Syndicate ได้รับใบสําคัญแสดงสิทธิจํานวนมาก โดยคาดว่าจะขายได้อีกในไม่ช้า โดยเน้นย้ําถึงความเสี่ยงที่เกี่ยวข้องกับโปรแกรมเร่งความเร็วผ่านประสบการณ์ของผู้ก่อตั้งอย่าง Garrett Temple

### [ปฏิกิริยา](https://news.ycombinator.com/item?id=40241525)

- ผู้ประกอบการในอุตสาหกรรมสตาร์ทอัพต้องเผชิญกับความท้าทาย เช่น ตัวเร่งความเร็วที่เข้าซื้อบริษัทด้วยเงินจํานวนน้อยที่สุด และความกังวลเกี่ยวกับมูลค่าที่เสนอโดยตัวเร่งความเร็ว
- การถกเถียงเกิดขึ้นเกี่ยวกับการมีส่วนร่วมของผู้ร่วมทุนในการระดมทุนให้กับสตาร์ทอัพ
- บุคคลสําคัญอย่าง นาวาล รวิกานต์ มีอิทธิพล การเป็นเจ้าของใบสําคัญแสดงสิทธิเป็นสิ่งสําคัญในการลงทุนเริ่มต้น โดยเน้นย้ําถึงผลกระทบที่อาจเกิดขึ้นกับทั้งผู้ก่อตั้งและนักลงทุน
