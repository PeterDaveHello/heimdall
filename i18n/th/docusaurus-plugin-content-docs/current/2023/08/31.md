## [Microsoft ใช้ป๊อปอัปที่เหมือนมัลแวร์ใน Win11 เพื่อให้ผู้คนทิ้ง Chrome](https://www.theverge.com/2023/8/30/23851902/microsoft-bing-popups-windows-11-malware)

- Microsoft กําลังแจ้งให้ผู้ใช้ Windows 11 มีข้อความป๊อปอัปแนะนําให้เปลี่ยนเครื่องมือค้นหาเริ่มต้นจาก Google เป็น Bing แม้ว่าพวกเขาจะใช้ Chrome ก็ตาม
- กลยุทธ์เหล่านี้ทําให้ผู้ใช้หงุดหงิดเนื่องจากพวกเขาคิดว่าป๊อปอัปดังกล่าวล่วงล้ําและเชื่อว่าไม่ควรอยู่ภายใต้โฆษณาภายในระบบปฏิบัติการที่พวกเขาซื้อ
- Microsoft ได้ตระหนักถึงฟันเฟืองและได้หยุดการแจ้งเตือนเหล่านี้ชั่วคราวในขณะที่พวกเขาพยายามแก้ไขปัญหา

### [ปฏิกิริยา](https://news.ycombinator.com/item?id=37321002)

- การสนทนามุ่งเน้นไปที่ความผิดหวังของผู้ใช้กับ Windows โดยเฉพาะ Windows 11 ทําให้บางคนพิจารณาทางเลือกอื่นเช่น Mac หรือ Linux เนื่องจากปัญหาเกี่ยวกับป๊อปอัปบัญชีผู้ใช้และการมองเห็นโปรแกรม
- ในบรรดาตัวเลือกระบบ Linux ได้รับการยกย่องว่าใช้งานง่ายและใช้งานได้ แต่ได้สังเกตเห็นข้อเสียสําหรับการรันเกมเฉพาะ พื้นที่ที่กล่าวถึงอื่น ๆ ได้แก่ การตั้งค่าฮาร์ดแวร์ความเข้ากันได้ของซอฟต์แวร์และข้อ จํากัด ของระบบย่อย Windows สําหรับ Linux (WSL)
- มีความรู้สึกไม่พอใจอย่างกว้างขวางกับการหยุดชะงักและโฆษณาในผลิตภัณฑ์เทคโนโลยีโดยมีการถกเถียงกันเกี่ยวกับข้อ จํากัด ของแพลตฟอร์มอนาคตของ Windows พฤติกรรมของ Microsoft และการดําเนินการทางกฎหมายหรือกฎระเบียบที่อาจเกิดขึ้น ความผิดหวังบางอย่างกับแอป iPhone สําหรับ Gmail และ Google Maps ยังบ่งบอกถึงความไม่พอใจที่อาจเกิดขึ้นกับผลิตภัณฑ์ของ Google

## [FCC ปฏิเสธที่จะยกเลิกกฎที่กําหนดให้ ISP แสดงรายการค่าธรรมเนียมรายเดือนทุกรายการ](https://arstechnica.com/tech-policy/2023/08/fcc-says-too-bad-to-isps-complaining-that-listing-every-fee-is-too-hard/)

- Federal Communications Commission (FCC) ได้ปฏิเสธการอุทธรณ์จาก Comcast และ ISP อื่น ๆ เพื่อลบกฎที่บังคับให้พวกเขาเปิดเผยค่าธรรมเนียมรายเดือนทั้งหมด
- ผู้ให้บริการอินเทอร์เน็ตแย้งว่าข้อกําหนดนี้ก่อให้เกิดความท้าทายด้านการบริหารและความซับซ้อนที่ไม่อาจหลีกเลี่ยงได้ แต่ FCC เน้นย้ําถึงความต้องการความโปร่งใสสําหรับผู้บริโภคเมื่อเลือกบริการอินเทอร์เน็ต
- ตามกฎหมาย FCC จะบังคับใช้กฎเหล่านี้โดยกําหนดให้ ISP รวมค่าธรรมเนียมที่รัฐบาลกําหนดทั้งหมดบนฉลากบรอดแบนด์

### [ปฏิกิริยา](https://news.ycombinator.com/item?id=37326806)

- Federal Communications Commission (FCC) ได้ยึดกฎที่กําหนดให้ผู้ให้บริการอินเทอร์เน็ต (ISP) เปิดเผยค่าธรรมเนียมรายเดือนทั้งหมดให้กับลูกค้าเพื่อเพิ่มความโปร่งใสและอํานวยความสะดวกในการเปรียบเทียบระหว่างผู้ให้บริการ
- ผู้สนับสนุนยืนยันว่ากฎนี้จะเปิดเผยค่าใช้จ่ายที่ซ่อนอยู่กระตุ้นการแข่งขันระหว่างผู้ให้บริการอินเทอร์เน็ตและรับผิดชอบ การอภิปรายยังทําให้เกิดคําถามว่าค่าธรรมเนียมการส่งผ่านที่กําหนดโดยรัฐบาลท้องถิ่นควรได้รับการพิจารณาเป็นภาษีหรือไม่และผู้ให้บริการอินเทอร์เน็ตควรส่งต่อสิ่งเหล่านี้ให้กับลูกค้าหรือไม่
- นอกจากนี้ FCC ยังตัดสินว่าผู้ให้บริการอินเทอร์เน็ตต้องเปิดเผยค่าธรรมเนียมและค่าใช้จ่ายทั้งหมดล่วงหน้าในโฆษณาของตนเพื่อให้เกิดความโปร่งใสมากขึ้นสะท้อนความรู้สึกของสาธารณชนเกี่ยวกับความซื่อสัตย์ในการกําหนดราคาและต่อต้านค่าธรรมเนียมที่ซ่อนอยู่

## [การแฮ็ก EDID ของ LG Monitor](https://gist.github.com/kj800x/be3001c07c49fdb36970633b0bc6defb)
