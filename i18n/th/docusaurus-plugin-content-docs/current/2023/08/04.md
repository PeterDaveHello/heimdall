## [AWS จะเริ่มเรียกเก็บเงินสําหรับที่อยู่ IPv4 สาธารณะ](https://aws.amazon.com/blogs/aws/new-aws-public-ipv4-address-charge-public-ip-insights/)

- ตั้งแต่วันที่ 1 กุมภาพันธ์ 2024 AWS จะใช้ค่าบริการ 0.005 USD ต่อที่อยู่ IP ต่อชั่วโมงสําหรับที่อยู่ IPv4 สาธารณะในบริการของ AWS ทั้งหมด
- การเปลี่ยนแปลงนี้ได้รับแรงบันดาลใจจากการขาดแคลนที่อยู่ IPv4 ที่เพิ่มขึ้นและมีวัตถุประสงค์เพื่อส่งเสริมการนํา IPv6 มาใช้
- นอกจากนี้ AWS ยังแนะนํา Public IP Insights ซึ่งเป็นคุณสมบัติฟรีที่ช่วยให้ผู้ใช้สามารถตรวจสอบและวิเคราะห์การใช้ที่อยู่ IPv4 สาธารณะได้
- ในช่วง 12 เดือนแรกตั้งแต่วันที่ 1 กุมภาพันธ์ 2024 AWS Free Tier for EC2 จะรวมการใช้งานที่อยู่ IPv4 สาธารณะ 750 ชั่วโมงต่อเดือน

- AWS ได้ใช้ค่าบริการสําหรับที่อยู่ IPv4 สาธารณะ ซึ่งนําไปสู่ความไม่พอใจในหมู่ผู้ใช้
- ผู้ใช้ชื่นชมโซลูชันของ Google Cloud และแนะนําทางเลือกอื่น เช่น การใช้อินสแตนซ์ NAT หรือ Linux VM
- มีความยุ่งยากเกี่ยวกับต้นทุนและการออกแบบภายใน VPC ใน AWS รวมถึงความกังวลเกี่ยวกับการเพิ่มขึ้นของราคาของที่อยู่ Elastic IP

## [FCC ปรับ robocaller เป็นประวัติการณ์ $ 300M หลังจากบล็อกการโทรหลอกลวงหลายพันล้านครั้ง](https://techcrunch.com/2023/08/03/fcc-fines-robocaller-a-record-300m-after-blocking-billions-of-their-scam-calls/)
