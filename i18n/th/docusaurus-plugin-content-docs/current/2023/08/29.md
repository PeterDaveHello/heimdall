# 2023-08-29

## [ChatGPT เอ็นเตอร์ไพรส์](https://openai.com/blog/introducing-chatgpt-enterprise)

- OpenAI ได้เปิดตัว ChatGPT Enterprise ซึ่งเป็นแพลตฟอร์มใหม่ที่ให้ความปลอดภัยความเป็นส่วนตัวการประมวลผลที่เร็วขึ้นหน้าต่างบริบทที่ยาวขึ้นและความสามารถในการวิเคราะห์ข้อมูล
- แพลตฟอร์มนี้ซึ่งได้รับการยอมรับอย่างดีจากบริษัทที่ติดอันดับ Fortune 500 กว่า 80% ให้การเข้าถึง GPT-4 (โมเดล AI ขั้นสูง) อย่างเต็มรูปแบบ และช่วยให้องค์กรสามารถปรับใช้ผู้ช่วย AI ที่กําหนดเองได้โดยไม่กระทบต่อความเป็นส่วนตัวของข้อมูล
- OpenAI ให้ความสําคัญกับความเป็นส่วนตัวของข้อมูล ซึ่งเป็นการส่งสัญญาณว่า ChatGPT Enterprise เป็นไปตามข้อกําหนด SOC 2 และให้การสนทนาที่เข้ารหัส โดยวางแผนที่จะแนะนําคุณสมบัติเพิ่มเติมในอนาคต

### [ปฏิกิริยา](https://news.ycombinator.com/item?id=37297304)

- การสนทนาอย่างต่อเนื่องหมุนรอบการใช้ ChatGPT ของ OpenAI ในการตั้งค่าทางธุรกิจโดยมีข้อกังวลที่สําคัญเกี่ยวกับความเป็นส่วนตัวของข้อมูลการละเมิดทรัพย์สินทางปัญญาที่อาจเกิดขึ้นและประสิทธิภาพของเครื่องมือ
- OpenAI ได้เปิดตัว ChatGPT เวอร์ชันองค์กรเพื่อแก้ไขปัญหาบางอย่าง แต่ความสงสัยยังคงมีอยู่และความท้าทายเช่นผลทางกฎหมายของเนื้อหาที่สร้างโดย AI และแอปพลิเคชันเชิงพาณิชย์ที่เป็นไปได้ก็กําลังถูกกล่าวถึงเช่นกัน
- มีการเก็งกําไรเพิ่มขึ้นเกี่ยวกับความสัมพันธ์ระหว่าง OpenAI และ Microsoft ซึ่งทั้งสองอย่างนี้มีเครื่องมือ AI สําหรับธุรกิจและคําถามเกี่ยวกับลิขสิทธิ์ของเอาต์พุตที่สร้างจากแมชชีนเลิร์นนิงยังก่อให้เกิดความคลุมเครือเกี่ยวกับโมเดล AI เช่น ChatGPT

## [Amazon เข้าซื้อกิจการ Fig](https://fig.io/blog/post/fig-joins-aws)

- Fig ซึ่งเป็นบริษัทที่สร้างเครื่องมือสําหรับนักพัฒนา ถูกซื้อกิจการโดย Amazon Web Services (AWS) ซึ่งเป็นบริษัทย่อยของ Amazon ที่ให้บริการประมวลผลบนระบบคลาวด์
- ผู้ใช้ปัจจุบันของ Fig จะยังคงได้รับการสนับสนุนและคุณสมบัติของทีมแบบชําระเงินจะพร้อมใช้งานฟรี
- การลงทะเบียนผู้ใช้ใหม่สําหรับผลิตภัณฑ์ของ Fig จะหยุดชั่วคราว เนื่องจากบริษัทมีเป้าหมายที่จะเพิ่มประสิทธิภาพผลิตภัณฑ์สําหรับลูกค้าปัจจุบันและดําเนินการผสานรวมกับ AWS

### [ปฏิกิริยา](https://news.ycombinator.com/item?id=37296401)

- Amazon ได้ซื้อ Fig ซึ่งเป็นเครื่องมือที่ให้การเติมข้อความอัตโนมัติสําหรับอินเทอร์เฟซบรรทัดคําสั่งทําให้เกิดการถกเถียงกันเกี่ยวกับอนาคตของผลิตภัณฑ์
- ผู้ใช้กําลังตั้งคําถามกับรูปแบบการสมัครสมาชิกและความเป็นส่วนตัวของข้อมูลของ Fig ตั้งแต่การได้มาและดึงความสนใจไปที่ความต้องการทางเลือกโอเพ่นซอร์ส
- ชุมชนถูกแยกออกจากการเข้าซื้อกิจการโดยมีบางส่วนคาดการณ์ถึงการปรับปรุงที่อาจเกิดขึ้นในขณะที่คนอื่น ๆ แสดงความสงสัยและกังวลเกี่ยวกับอนาคตของ Fig

## [OpenTelemetry ใน 2023](https://bit.kevinslin.com/p/opentelemetry-in-2023)

- OpenTelemetry (OTEL) ก่อตั้งขึ้นในปี 2019 โดยเป็นการควบรวมกิจการของโครงการติดตามโอเพ่นซอร์สสองโครงการโดยมีเป้าหมายเพื่อให้การวัดและส่งข้อมูลทางไกลชั้นยอดสําหรับองค์กร
- OTEL ซึ่งมีการใช้งานมากที่สุดเป็นอันดับสองใน Cloud Native Computing Foundation (CNCF) ได้รับการยอมรับอย่างกว้างขวางในหมู่ผู้ให้บริการด้านการสังเกตการณ์และประกอบด้วยข้อกําหนดและการใช้งานสําหรับเมตริกบันทึกการติดตามและการจัดการตัวแทน
- มีการขยายและใช้กันอย่างแพร่หลายมี SDK สําหรับรหัสเครื่องมือในภาษายอดนิยมและนักสะสมเพื่อรับและส่งออกการวัดและส่งข้อมูลทางไกลพร้อมแผนการขยายและปรับปรุงในอนาคต

### [ปฏิกิริยา](https://news.ycombinator.com/item?id=37295097)

- การอภิปรายเกี่ยวข้องกับ OpenTelemetry ซึ่งเป็นเครื่องมือซอฟต์แวร์เพื่อเพิ่มความสามารถในการสังเกตของแอปพลิเคชันโดยเน้นถึงข้อดีเช่นความคุ้มค่าและการจัดการข้อมูลขนาดใหญ่และข้อเสียเช่นความซับซ้อนและคําจํากัดความที่คลุมเครือ
- หัวข้อสําคัญที่กล่าวถึง ได้แก่ การดูการติดตามช่วงรากการสุ่มตัวอย่างการติดตามข้อผิดพลาดการโอเวอร์โหลดระบบการใช้งาน SQL และโซลูชันที่โฮสต์ด้วยตนเองซึ่งแสดงมุมมองที่หลากหลายเกี่ยวกับการใช้งานและประสิทธิภาพของเครื่องมือ
- ผู้เข้าร่วมยังสํารวจทางเลือกอื่นสําหรับ OpenTelemetry พูดคุยเกี่ยวกับประโยชน์และข้อเสียของการติดตามเหตุการณ์บันทึกและเครื่องมือและเทคโนโลยีต่างๆ

## [การค้นพบที่ไม่คาดคิดที่ทําให้พื้นที่ดัชนีที่ไม่ได้ใช้ว่าง 20GB (2021) ว่าง](https://hakibenita.com/postgresql-unused-index-size)
