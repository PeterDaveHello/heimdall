# 2023-08-22

## [uBlock Origin Lite พร้อมให้เล่นแล้วบน Firefox](https://addons.mozilla.org/en-US/firefox/addon/ublock-origin-lite/)

- uBlock Origin Lite เป็นส่วนขยายตัวบล็อกเนื้อหาสําหรับ Firefox ที่บล็อกโฆษณาตัวติดตามและนักขุด
- ส่วนขยายใช้วิธีการประกาศสําหรับการกรองที่มีประสิทธิภาพและไม่ต้องการสิทธิ์ในวงกว้างในการติดตั้ง
- ผู้ใช้มีตัวเลือกในการให้สิทธิ์เพิ่มเติมในไซต์เฉพาะเพื่อการกรองที่แม่นยํายิ่งขึ้น

### [ปฏิกิริยา](https://news.ycombinator.com/item?id=37215557)

- การอภิปรายมีศูนย์กลางอยู่ที่การใช้งานและจริยธรรมของส่วนขยายการบล็อกโฆษณาโดยเน้นที่ uBlock Origin และเวอร์ชันต่างๆ
- ประโยชน์ด้านความปลอดภัยและข้อ จํากัด ของส่วนขยายเหล่านี้กําลังเป็นที่ถกเถียงกันรวมถึงความสามารถในการลดเวกเตอร์การโจมตีที่อาจเกิดขึ้นและความกังวลเกี่ยวกับการอัปเดตที่ถูกบุกรุก
- นอกจากนี้ยังมีการกล่าวถึงผลกระทบทางจริยธรรมของการบล็อกโฆษณาลักษณะการรุกรานของการโฆษณาออนไลน์และผลกระทบของการเปลี่ยนแปลงการทํางานของเบราว์เซอร์ต่อตัวบล็อกโฆษณา

## [GNU Parallel คุณอยู่ที่ไหนมาตลอดชีวิตของฉัน?](https://alexplescan.com/posts/2023/08/20/gnu-parallel/)

- โพสต์บล็อกกล่าวถึงประสบการณ์ของผู้เขียนในการใช้ GNU Parallel เพื่อรวบรวมสถิติเกี่ยวกับอัตราการผ่าน / ไม่ผ่านการทดสอบแบบ end-to-end
- การดําเนินการแบบขนานโดยใช้ GNU Parallel ปรับปรุงประสิทธิภาพและนําเสนอฟังก์ชันเพิ่มเติม
- โพสต์เน้นคุณสมบัติของ GNU Parallel เช่นการควบคุมการทํางานพร้อมกันการจับเอาต์พุตและความสามารถขั้นสูงที่อธิบายไว้ในหนังสือของผู้เขียน

### [ปฏิกิริยา](https://news.ycombinator.com/item?id=37208083)

- การอภิปรายสํารวจข้อดีและประโยชน์ของ GNU Parallel เมื่อเทียบกับคําสั่งเชลล์และเครื่องมือทางเลือกเช่น YSH, Nushell และ Fish ในการคํานวณแบบขนาน
- มีการกล่าวถึงภาษาทางเลือกเช่น Perl, Python และ Haskell รวมถึงเครื่องมือเช่น zargs ใน zsh และ DASK ใน Python สําหรับการประมวลผลแบบขนาน
- การอภิปรายรวมถึงการเปรียบเทียบระหว่างแกน CPU และ GPU WGP / SM การรวมประกาศการอ้างอิงในใบอนุญาต GPL และบทบาทของ Richard Stallman ในการตีความ GPL ความสําคัญของการอ้างอิงในการวิจัยซอฟต์แวร์และการโต้เถียงเกี่ยวกับผู้เขียนของ GNU Parallel ที่ขอการอ้างอิงหรือการชําระเงินได้รับการแก้ไขแล้ว มีการกล่าวถึงข้อดีข้อเสียของการใช้ xargs และ parallel สําหรับการดําเนินการแบบขนานและประโยชน์ของการใช้กลุ่มเธรด / คนงาน GNU Parallel ได้รับการยกย่องในด้านความเก่งกาจและประสิทธิภาพในการประมวลผลแบบขนาน

## [อย่ายิงนักวาดภาพประกอบของคุณ](https://sambleckley.com/writing/dont-fire-your-illustrator.html)

- มีการอธิบายแนวคิดของการแพร่กระจายที่เสถียรพร้อมกับความท้าทายในการใช้ข้อความเพื่อสร้างภาพโดยตรง
- คาดการณ์ว่า AI จะเข้ามาแทนที่งานภาพประกอบและการเขียนบางประเภท แต่การบรรลุผลลัพธ์ที่เฉพาะเจาะจงอาจเป็นเรื่องที่ท้าทาย และแนะนําให้ใช้ข้อจํากัดของรูปภาพ ผู้เขียนเน้นย้ําถึงความสําคัญของผลิตภัณฑ์ AI ที่รวมเข้ากับเวิร์กโฟลว์ของนักวาดภาพประกอบได้อย่างราบรื่นและแสดงความหวังสําหรับผลิตภัณฑ์ AI ในอนาคตที่รองรับภาพประกอบเชิงพาณิชย์

## [ช่วยชีวิต (2004)](https://www.folklore.org/StoryView.py?story=Saving_Lives.txt)

- สตีฟจ็อบส์กระตุ้นให้วิศวกรที่ทํางานบนคอมพิวเตอร์แมคอินทอชปรับปรุงเวลาบูตโดยบอกพวกเขาว่าทุก ๆ 10 วินาทีที่บันทึกไว้จะช่วยชีวิตโหล
- วิศวกรประสบความสําเร็จในการลดเวลาบูตลงมากกว่า 10 วินาทีในเดือนต่อ ๆ มา
- เรื่องนี้เน้นสไตล์ความเป็นผู้นําของสตีฟจ็อบส์และความทุ่มเทของวิศวกรในการตอบสนองความคาดหวังของเขา

### [ปฏิกิริยา](https://news.ycombinator.com/item?id=37211230)

- การทําความเข้าใจความต้องการของลูกค้าและการแก้ไขปัญหาพื้นฐานเป็นสิ่งสําคัญสําหรับนักพัฒนาซอฟต์แวร์ในการจัดลําดับความสําคัญของความพึงพอใจของผู้ใช้
- ซอฟต์แวร์ที่ช้าอาจส่งผลเสียต่อประสบการณ์ของผู้ใช้และควรได้รับการปรับให้เหมาะสมเพื่อประสิทธิภาพที่เร็วขึ้น
- ความก้าวหน้าในการเพิ่มประสิทธิภาพฮาร์ดแวร์และซอฟต์แวร์ทําให้เวลาบูตเร็วขึ้นปรับปรุงประสบการณ์การใช้งานโดยรวม
- ระบบปฏิบัติการที่ใช้ ROM มีทั้งข้อดีและข้อเสียและควรพิจารณาอย่างรอบคอบสําหรับกรณีการใช้งานเฉพาะ
- ภาพเคลื่อนไหว UI สามารถปรับปรุงประสบการณ์ของผู้ใช้ได้ แต่ควรใช้อย่างรอบคอบเพื่อหลีกเลี่ยงการครอบงําหรือทําให้ผู้ใช้เสียสมาธิ
- SSD ให้ประโยชน์อย่างมากกับ HDD รวมถึงเวลาในการเข้าถึงข้อมูลที่เร็วขึ้นและการตอบสนองของระบบที่ดีขึ้น
- การประหยัดเวลาในการพัฒนาซอฟต์แวร์เป็นสิ่งจําเป็นสําหรับประสิทธิภาพและประสิทธิผล
- การอัปเดตที่ช้าและแพลตฟอร์มการสื่อสารที่ไม่เพียงพออาจนําไปสู่ความยุ่งยากในหมู่ผู้ใช้ซอฟต์แวร์และนักพัฒนา
- Stack Overflow ได้รับการวิพากษ์วิจารณ์และข้อกังวลโดยเน้นถึงความต้องการแพลตฟอร์มทางเลือกและการกลั่นกรองที่ดีขึ้น
- เมื่อขอความช่วยเหลือสิ่งสําคัญคือต้องให้ข้อมูลที่ชัดเจนและสามารถดําเนินการได้เพื่อรับความช่วยเหลือที่มีประสิทธิภาพ

## [ฉันทําให้การแพร่กระจายที่เสถียร XL ฉลาดขึ้นโดยการปรับแต่งภาพที่สร้างขึ้นโดย AI ที่ไม่ดี](https://minimaxir.com/2023/08/stable-diffusion-xl-wrong/)

- โพสต์สํารวจการใช้ LoRAs สําหรับการปรับแต่ง SDXL แสดงตัวอย่างการสร้างภาพที่มีและไม่มี LoRAs และกล่าวถึงเทคนิค "แจ้งผิด" เพื่อปรับปรุงภาพที่สร้างโดย AI

## [5 ปีที่แล้ว Valve เปิดตัว Proton](https://www.gamingonlinux.com/2023/08/5-years-ago-valve-released-proton-forever-changing-linux-gaming/)

- การเปิดตัว Proton ของ Valve ซึ่งเป็นเวอร์ชันดัดแปลงของ Wine ได้เปลี่ยนการเล่นเกม Linux เมื่อห้าปีก่อนโดยทําให้เกมยอดนิยมทํางานบนแพลตฟอร์ม
- สิ่งนี้จุดประกายให้เกม Linux เพิ่มขึ้นและมีส่วนช่วยในการสร้าง Steam Deck
- การปรับปรุงอย่างต่อเนื่องของ Proton ได้ปรับปรุงความเข้ากันได้และประสิทธิภาพทําให้นักเล่นเกม Linux สามารถเข้าถึงเกมที่หลากหลายซึ่งนําไปสู่ความคิดเห็นของผู้ใช้ที่กระตือรือร้น

### [ปฏิกิริยา](https://news.ycombinator.com/item?id=37213479)

- แพลตฟอร์มเกม Proton ของ Valve ได้รับความนิยมในหมู่ผู้ใช้ในฐานะทางเลือกที่ทํางานได้สําหรับ Windows สําหรับการเล่นเกมบน Linux
- Valve ได้รับการยกย่องสําหรับการสนับสนุนลูกค้าและการอุทิศตนให้กับเดสก์ท็อปโอเพ่นซอร์สในขณะที่ Epic ถูกวิพากษ์วิจารณ์ว่าขาดในด้านเหล่านี้
- มีความกังวลเกี่ยวกับข้อตกลงการผูกขาดของ Epic และการลดลงของแพลตฟอร์มที่ไม่มี DRM

- เน้นความต้องการการออกแบบที่รอบคอบและปรับเปลี่ยนได้เพื่อปรับปรุงประสบการณ์ของผู้ใช้บนอุปกรณ์ต่างๆพร้อมกับข้อ จํากัด ของเครื่องมือออกแบบในปัจจุบันและการต่อสู้ในการสร้างเลย์เอาต์ที่ลื่นไหลและยืดหยุ่น

- การสํารวจล่าสุดโดย Pew Research Center แสดงให้เห็นว่า 57% ของชาวอเมริกันสนับสนุนการขยายพลังงานนิวเคลียร์ในสหรัฐอเมริกาเพิ่มขึ้นจาก 43% ในปี 2020
- อย่างไรก็ตามพลังงานแสงอาทิตย์ (82%) และพลังงานลม (75%) ยังคงเป็นที่ชื่นชอบของชาวอเมริกันมากกว่าเมื่อเทียบกับพลังงานนิวเคลียร์
- การสนับสนุนพลังงานนิวเคลียร์เพิ่มขึ้นในหมู่ทั้งพรรคเดโมแครตและรีพับลิกัน โดยผู้ชายและรีพับลิกันมีแนวโน้มที่จะสนับสนุนและเชื่อว่ารัฐบาลควรส่งเสริมการผลิต ปัจจุบันสหรัฐอเมริกามีเครื่องปฏิกรณ์พลังงานนิวเคลียร์ 93 เครื่องซึ่งส่วนใหญ่ตั้งอยู่ในภาคใต้ แต่จํานวนลดลงนับตั้งแต่จุดสูงสุดในปี 1990 ซึ่งอาจเกิดจากความกังวลเกี่ยวกับอุบัติเหตุและความตึงเครียดระหว่างประเทศ

### [ปฏิกิริยา](https://news.ycombinator.com/item?id=37216647)

- การถกเถียงเกี่ยวกับพลังงานนิวเคลียร์มีหลายแง่มุมโดยบางคนโต้เถียงกันเพื่อประโยชน์ของมันและคนอื่น ๆ ที่สนับสนุนแหล่งพลังงานหมุนเวียนเช่นพลังงานแสงอาทิตย์และลม
- ผู้เสนอพลังงานนิวเคลียร์เน้นความคุ้มค่าและเป็นมิตรต่อสิ่งแวดล้อมเมื่อเทียบกับพลังงานรูปแบบอื่น ๆ
- ฝ่ายตรงข้ามของพลังงานนิวเคลียร์ทําให้เกิดความกังวลเกี่ยวกับค่าใช้จ่ายสูงอันตรายที่อาจเกิดขึ้นปัญหาด้านความปลอดภัยกฎระเบียบการจัดเก็บกากนิวเคลียร์และอิทธิพลของการเมืองและอุตสาหกรรมต่อการรับรู้ของสาธารณชน
