# 2023-09-13

## [ปรับแต่ง Llama 2 ของคุณเองเพื่อแทนที่ GPT-3.5/4](https://news.ycombinator.com/item?id=37484135)

- ผู้เขียนกล่าวถึงการปฏิบัติจริงและประโยชน์ของการปรับแต่ง LLM แบบโอเพนซอร์ส (Language Learning Models) ซึ่งเกี่ยวข้องกับการฝึกอบรมโมเดลที่มีอยู่ก่อนแล้วในงานเฉพาะที่นําไปสู่ประสิทธิภาพของโมเดลที่มีการควบคุมมากขึ้น
- การปรับจูนอย่างละเอียดเมื่อเปรียบเทียบกับการกระตุ้นเตือนอาจส่งผลให้การตอบสนองเร็วขึ้นและลดค่าใช้จ่ายในการอนุมาน ผู้เขียนอ้างว่าโมเดลที่ปรับแต่งมาอย่างดีของพวกเขาทํางานคล้ายกับ GPT-4 ซึ่งเป็นโมเดลภาษาที่มีประสิทธิภาพสูงในขณะที่ประหยัดค่าใช้จ่ายมากกว่าอย่างมีนัยสําคัญ
- ผู้เขียนอ้างอิงเครื่องมือชื่อ 'OpenPipe' ซึ่งเป็นผลิตภัณฑ์โอเพ่นซอร์สที่มุ่งช่วยเหลือวิศวกรในการนําแนวทางปฏิบัติในการปรับแต่งมาใช้ได้อย่างง่ายดาย

### [ปฏิกิริยา](https://news.ycombinator.com/item?id=37484135)

- การสนทนามุ่งเน้นไปที่การประยุกต์ใช้รูปแบบการเรียนรู้ภาษาแบบโอเพนซอร์ส (LLMs) ซึ่งส่วนใหญ่เป็นแบบจําลองลามะสําหรับงานที่หลากหลายโดยเน้นความคุ้มค่าและประสิทธิภาพเมื่อเทียบกับ GPT-3.5

- ความสําคัญของคู่อินพุต/เอาต์พุตสําหรับชุดข้อมูลการปรับแต่งอย่างละเอียดและข้อดีที่คาดหวังของ LLM ที่สร้างขึ้นตามวัตถุประสงค์จะถูกนําเข้าสู่แสงเช่นกัน

## [iPhone 15 และ iPhone 15 Plus](https://www.apple.com/newsroom/2023/09/apple-debuts-iphone-15-and-iphone-15-plus/)

- Apple ได้เปิดเผยการเปิดตัว iPhone 15 และ iPhone 15 Plus ซึ่งมีการออกแบบใหม่กล้องที่ได้รับการอัพเกรดคุณสมบัติด้านประสิทธิภาพและความปลอดภัยขั้นสูงตัวเลือกการชาร์จและองค์ประกอบด้านความยั่งยืน
- การสั่งซื้อล่วงหน้าจะวางจําหน่ายในเดือนกันยายนโดยมีให้เลือกห้าสีพร้อมข้อเสนอแลกเปลี่ยนและอุปกรณ์เสริมเพิ่มเติมสําหรับรุ่นใหม่เหล่านี้
- นอกจากโทรศัพท์แล้ว Apple ยังได้เปิดตัวอุปกรณ์เสริมใหม่ การอัปเดตซอฟต์แวร์ และเน้นย้ําถึงความคิดริเริ่มด้านสิ่งแวดล้อมของพวกเขา รายละเอียดเกี่ยวกับราคา การเชื่อมต่อ และความพร้อมจําหน่ายในภูมิภาครวมอยู่ด้วย

### [ปฏิกิริยา](https://news.ycombinator.com/item?id=37485290)

- อิทธิพลที่อาจเกิดขึ้นจากกฎระเบียบของรัฐบาลเกี่ยวกับมาตรฐานการชาร์จพอร์ตยังกลายเป็นจุดโฟกัสของการอภิปรายซึ่งแสดงให้เห็นถึงมุมมองของผู้บริโภคในวงกว้างเกี่ยวกับการใช้งานเทคโนโลยี

- มีการนําเสนอคําแนะนําสําหรับการปรับปรุง แต่ความซับซ้อนของปัญหาเป็นอุปสรรคต่อความก้าวหน้า

## [เราสร้าง CI ที่เร็วที่สุดและล้มเหลว](https://earthly.dev/blog/shutting-down-earthly-ci/)

- Earthly บริษัท ที่เชี่ยวชาญด้านเครื่องมือ CI / CD (Continuous Integration / Continuous Delivery) ต้องเผชิญกับอุปสรรคในขณะที่แนะนําผลิตภัณฑ์ Earthly CI ส่วนใหญ่เนื่องจากค่าใช้จ่ายในการย้ายถิ่นฐานและความสงสัยจากผู้มีโอกาสเป็นลูกค้า
- บริษัท ได้เรียนรู้ว่ากลยุทธ์การเข้าสู่ตลาดโดยตรงนั้นไม่ได้ผลแทนที่จะตระหนักว่านักพัฒนาจําเป็นต้องค้นพบและนําเครื่องมือของตนมาใช้อย่างอิสระซึ่งนําไปสู่การเปลี่ยนแปลงแนวทาง
- เนื่องจากข้อผิดพลาดเช่นการไม่สร้างทีละน้อยและเพิกเฉยต่อข้อเสนอแนะในช่วงต้นพวกเขาจึงต้องปิดส่วนประกอบ CI (Continuous Integration) ซึ่งสะท้อนให้เห็นถึงความสําคัญของความคิดเห็นของผู้ใช้และการพัฒนาที่เพิ่มขึ้นในการเปิดตัวผลิตภัณฑ์เทคโนโลยี

### [ปฏิกิริยา](https://news.ycombinator.com/item?id=37481513)

- วาทกรรมมุ่งเน้นไปที่ความท้าทายและข้อควรพิจารณาที่สําคัญที่เกี่ยวข้องกับระบบบูรณาการอย่างต่อเนื่อง (CI) รวมถึงความล้มเหลวของ บริษัท ที่ชื่อว่า Earthly
- มีการเปรียบเทียบคุณสมบัติและการใช้งานภายในแพลตฟอร์ม CI/CD แบบโอเพนซอร์ส เช่น GitLab, GitHub และ Azure DevOps โดยมีหัวข้อต่างๆ เช่น การแคชในระบบบิลด์ ข้อเสนอแนะของกระบวนการ CI และการตัดสินใจสร้างใหม่
- ฉันทามติถูกแบ่งออกโดยเน้นความสมดุลที่สําคัญระหว่างคุณสมบัติเฉพาะและความเรียบง่ายในระบบ CI / CD การอภิปรายยังพิจารณาถึงประสิทธิภาพของการโฆษณาการสาธิตและการสร้างความไว้วางใจในการส่งเสริมเครื่องมือสําหรับนักพัฒนาใหม่

## [การอัปเดตราคาและบรรจุภัณฑ์ของแผน Unity](https://blog.unity.com/news/plan-pricing-and-packaging-updates)

- ภายในเดือนพฤศจิกายนจะมีการเพิ่มเครื่องมือและบริการใหม่ ๆ ในแผนการสมัครสมาชิกของ Unity โดยไม่มีราคาที่นั่งเพิ่มขึ้น Unity Plus จะถูกยกเลิกสําหรับสมาชิกใหม่อย่างไรก็ตามสมาชิกปัจจุบันจะมีโอกาสอัปเกรดเป็น Unity Pro
