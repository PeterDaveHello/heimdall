## [Stirling-PDF: เครื่องมือจัดการ PDF ที่อัดแน่นไปด้วยคุณสมบัติสําหรับการใช้งานในท้องถิ่น](https://github.com/Frooodle/Stirling-PDF)

- Stirling-PDF เป็นเครื่องมือบนเว็บที่โฮสต์ในเครื่องสําหรับจัดการไฟล์ PDF โดยนําเสนอคุณสมบัติต่างๆเช่นการแยกการรวมการแปลงและอื่น ๆ
- เครื่องมือนี้ช่วยให้มั่นใจได้ถึงความปลอดภัยและความเป็นส่วนตัวของไฟล์โดยไม่ต้องโทรออกเพื่อติดตามหรือเก็บบันทึก
- ใช้เทคโนโลยีเช่น Spring Boot, Thymeleaf, PDFBox, LibreOffice และอื่น ๆ และสามารถเข้าถึงได้ผ่านอินสแตนซ์ที่โฮสต์หรือเรียกใช้ในเครื่องโดยใช้ Docker หรือ Podman

### [ปฏิกิริยา](https://news.ycombinator.com/item?id=38765627)

- ผู้ใช้กําลังพูดถึงเครื่องมือแก้ไข PDF ต่างๆรวมถึงความเข้ากันได้ข้อ จํากัด และตัวเลือกอื่น ๆ
- การสนทนาครอบคลุมกระบวนการอัตโนมัติเช่นการแปลงอีเมลเป็น PDF และการจัดระเบียบคอลัมน์ PDF
- ผู้ใช้กล่าวถึงซอฟต์แวร์คุณสมบัติข้อกังวลด้านราคาการใช้เทคโนโลยี AI ข้อบกพร่องที่พบและคําแนะนําสําหรับเครื่องมือ PDF ทางเลือก

## [การปรับปรุงการตรวจสอบรหัสผ่าน SSH ด้วย Password Manager และ Zsh Wrapper](https://vincent.bernat.ch/en/blog/2023-sshpass-without-sshpass)

- บทความนี้สํารวจการตรวจสอบรหัสผ่าน SSH แบบไม่โต้ตอบและเสนอ Zsh wrapper สําหรับดึงรหัสผ่านจากตัวจัดการรหัสผ่าน
- มันนําเสนอวิธีการที่จะทําให้ scp ใช้ฟังก์ชัน ssh ที่กําหนดเอง
- โพสต์เน้นถึงประโยชน์ของการใช้คีย์สาธารณะแทนรหัสผ่านและกล่าวถึงความท้าทายที่เกี่ยวข้องกับการรับรองความถูกต้องตามใบรับรองและวิธีการอนุญาตที่ละเอียดยิ่งขึ้น

- หัวข้ออื่น ๆ ได้แก่ ปัญหาเกี่ยวกับเชลล์สคริปต์และ API การถอดรหัสคีย์ SSH ความกังวลเกี่ยวกับการจัดเก็บรหัสผ่านอย่างถาวรและการจัดการรหัสผ่านในสภาพแวดล้อมขนาดใหญ่

## [โรงงานประกวดใช้สิทธิบัตรซอฟต์แวร์ในวงกว้างเพื่อฟ้องร้องผู้จัดประกวดออนไลน์ซึ่งท้าทายโดย EFF](https://www.eff.org/deeplinks/2023/12/stupid-patent-month-selfie-contests)

- Contest Factory ได้ยื่นฟ้องบริษัทต่างๆ ที่จัดการแข่งขันออนไลน์หลายครั้ง โดยยืนยันว่าสิทธิบัตรของพวกเขาครอบคลุมการแข่งขันออนไลน์ที่หลากหลาย
- Electronic Frontier Foundation (EFF) ให้เหตุผลว่าสิทธิบัตรซอฟต์แวร์เหล่านี้เป็นอันตรายต่อสิทธิของผู้คนในการแสดงออกและมีส่วนร่วมในวัฒนธรรมออนไลน์
- EFF เชื่อว่าสิทธิบัตรดังกล่าวไม่ควรได้รับและสนับสนุนกระบวนการที่เข้าถึงได้ง่ายขึ้นเพื่อท้าทายสิทธิบัตรเหล่านี้

### [ปฏิกิริยา](https://news.ycombinator.com/item?id=38763566)

- Electronic Frontier Foundation (EFF) ได้แสดงความกังวลเกี่ยวกับสิทธิบัตรที่เรียกว่า "การประกวดเซลฟี่" และโต้แย้งว่าการเพิ่มคําว่า "บนคอมพิวเตอร์" ลงในกิจกรรมประจําวันไม่ควรสามารถจดสิทธิบัตรได้
- การอภิปรายมุ่งเน้นไปที่ประเด็นต่างๆเช่นศิลปะก่อนหน้าความชัดเจนการหลอกสิทธิบัตรและกระบวนการทําให้สิทธิบัตรเป็นโมฆะ
- มีข้อเสนอแนะต่าง ๆ รวมถึงการใช้ทนายความหรือกระบวนการอย่างเป็นทางการในการขอยกเลิก แต่ความซับซ้อนและค่าใช้จ่ายที่เกี่ยวข้องได้รับการยอมรับ นอกจากนี้ยังมีการกล่าวถึงตัวอย่างของศิลปะก่อนหน้าและการอนุมัติสิทธิบัตรที่น่าสงสัย

- Tillitis TKey เป็นคอมพิวเตอร์แท่ง USB ที่สร้างขึ้นโดยเฉพาะสําหรับการเรียกใช้แอปพลิเคชันที่ปลอดภัยซึ่งมีคุณสมบัติด้านความปลอดภัยต่างๆเช่นการเข้ารหัสและการสร้างโทเค็น
