## [อะไรที่ผ่านมานานกว่าเมื่อวานก็ควรจะพูดวันที่จริง](https://grumpy.website/1389)

## [ประสิทธิภาพของ Linux](https://www.brendangregg.com/linuxperf.html)

- เว็บไซต์ของ Brendan Gregg เป็นแหล่งที่ครอบคลุมสําหรับประสิทธิภาพของระบบและการวิเคราะห์ประสิทธิภาพของ Linux รวมถึงบล็อกโพสต์หนังสือและเครื่องมือสําหรับการวิเคราะห์ประสิทธิภาพการปรับแต่งและการเปรียบเทียบ
- ไซต์นี้มีการพูดคุยและการนําเสนอของ Gregg ในหัวข้อต่างๆเช่นการวิเคราะห์ประสิทธิภาพของคอนเทนเนอร์เครื่องมือติดตาม Linux และรายการตรวจสอบประสิทธิภาพของ Site Reliability Engineering (SRE)
- นอกเหนือจากเนื้อหาของตัวเองแล้วเว็บไซต์ยังแนะนําแหล่งข้อมูลที่มีค่าอื่น ๆ เพื่อความเข้าใจอย่างลึกซึ้งเกี่ยวกับประสิทธิภาพของ Linux

### [ปฏิกิริยา](https://news.ycombinator.com/item?id=37879706)

- บทความนี้เจาะลึกเครื่องมือและเทคนิคการปรับแต่งประสิทธิภาพของ Linux โดยเน้นบทบาทสําคัญในการเพิ่มประสิทธิภาพแอปพลิเคชันแบบเรียลไทม์บน Debian และ Ubuntu Virtual Private Servers (VPSs)
- ผู้เขียนเน้นย้ําถึงการมีส่วนร่วมอย่างมากของ Brendan Gregg ในสาขานี้โดยแนะนําหนังสือของเขาว่าเป็นแหล่งข้อมูลที่จําเป็นและเน้นย้ําถึงความสําคัญของการเพิ่มประสิทธิภาพเวลาแฝง
- นอกจากนี้ยังสนับสนุนการเรียนรู้ในงานและการตัดสินใจอย่างชาญฉลาดเมื่อกําหนดค่าและปรับแต่งเซิร์ฟเวอร์ฐานข้อมูลโดยเน้นถึงความซับซ้อนของการปรับแต่งประสิทธิภาพของ Linux และความสําคัญของเอกสารความรู้

## [ตัวกรอง Anti-Anti-Adblocker uBlock เพื่อกําจัดข้อความ YouTube ที่น่ารําคาญ](https://twitter.com/endermanch/status/1713129854818742684)

- ตัวกรองใหม่ได้ถูกรวมเข้ากับ uBlock Origin ซึ่งมีฟังก์ชั่นปิดการใช้งานคุณลักษณะต่อต้านการบล็อกโฆษณา JavaScript บน YouTube
- นวัตกรรมนี้ให้ทางออกสําหรับผู้ใช้โดยขจัดปัญหาที่เกิดซ้ําของข้อความที่น่ารําคาญ
- UBlock Origin เป็นเครื่องมือยอดนิยมที่ช่วยให้ผู้ใช้อินเทอร์เน็ตสามารถควบคุมเนื้อหาที่แสดงบนหน้าจอได้

### [ปฏิกิริยา](https://news.ycombinator.com/item?id=37884683)

- จุดยืนแตกต่างกันไปในเรื่องเหล่านี้ตั้งแต่ความไม่พอใจในตําแหน่งโฆษณาและการละเมิดความเป็นส่วนตัวไปจนถึงข้อเสนอในการสมัครสมาชิกบริการระดับพรีเมียมหรือการยอมรับการแสดงตนโฆษณาเป็นความจริงของแพลตฟอร์ม

## [ตัวกรอง Kalman จากพื้นดินขึ้น](https://www.kalmanfilter.net/default.aspx)

- บทสรุปนี้เน้นบทช่วยสอนและหนังสือที่อุทิศให้กับอัลกอริทึม Kalman Filter ศัพท์แสงทางเทคนิคสําหรับวิธีการที่ใช้ในการประมาณการและทํานายสถานะของระบบท่ามกลางความไม่แน่นอน
- แหล่งข้อมูลการศึกษามีจุดมุ่งหมายเพื่ออธิบายหัวข้อที่ซับซ้อนโดยให้ตัวอย่างที่เป็นประโยชน์และคําอธิบายที่ชัดเจนจัดการแนวคิดพื้นฐานในบทช่วยสอนและสํารวจหัวข้อขั้นสูงเช่นตัวกรองที่ไม่ใช่เชิงเส้นในหนังสือ
- เขียนโดยมืออาชีพที่มีประสบการณ์ 20 ปีวัสดุนําเสนอคําอธิบายที่ใช้งานง่ายและตัวอย่างภาพประกอบเพื่อทําความเข้าใจ Kalman Filter ซึ่งเป็นอัลกอริทึมที่สําคัญสําหรับการประมาณสถานะที่ซ่อนอยู่และทํานายสถานะในอนาคตแม้จะมีการวัดและความไม่แน่นอนของกระบวนการ

### [ปฏิกิริยา](https://news.ycombinator.com/item?id=37879715)

- การอภิปรายส่วนใหญ่เกี่ยวกับความสับสนและความไม่พอใจเกี่ยวกับบทช่วยสอนที่มองข้ามการชี้แจงที่สําคัญหรือสันนิษฐานความรู้ก่อนหน้านี้โดยมุ่งเน้นไปที่การใช้งานตัวกรอง Kalman
- ผู้มีส่วนร่วมในการอภิปรายเปรียบเทียบการใช้ตัวกรอง Kalman และโมเดลแมชชีนเลิร์นนิงโดยยอมรับว่าเทคนิคทั้งสองมีข้อดีและข้อเสีย
- กระทู้ตั้งข้อสังเกตว่าตัวกรอง Kalman มีการใช้กันอย่างแพร่หลายในหลายอุตสาหกรรม แต่ต้องการความรู้เกี่ยวกับข้อ จํากัด ของพวกเขาโดยผู้แสดงความคิดเห็นคนหนึ่งเล่าถึงความยากลําบากในการประยุกต์ใช้ในสภาพแวดล้อมการผลิต

## [การตรวจสอบซิลิคอนตายของโปรเซสเซอร์ Intel 386](http://www.righto.com/2023/10/intel-386-die-versions.html)

- โพสต์บล็อกให้รายละเอียดเกี่ยวกับบริบททางประวัติศาสตร์และคุณสมบัติของโปรเซสเซอร์ Intel 386 โดยเน้นย้ําถึงบทบาทในการเปลี่ยนไปใช้สถาปัตยกรรม 32 บิต
- มันเน้นบทบาทที่สําคัญของโปรเซสเซอร์ 386 ในการสร้างการครอบงํา x86 และการเปลี่ยนแปลงอุตสาหกรรมคอมพิวเตอร์
- นอกจากนี้ยังมีข้อมูลเพิ่มเติมเกี่ยวกับแง่มุมต่างๆเช่นจํานวนทรานซิสเตอร์การเพิ่มประสิทธิภาพไปป์ไลน์หน่วยการทํางานและความลังเลของ IBM เพื่อสนับสนุน 386

### [ปฏิกิริยา](https://news.ycombinator.com/item?id=37881656)

- โพสต์นี้กล่าวถึงโปรเซสเซอร์ Intel 386 โดยมุ่งเน้นไปที่ความสําคัญทางประวัติศาสตร์การเพิ่มประสิทธิภาพที่เป็นเอกลักษณ์และความตรงกันข้ามกับโปรเซสเซอร์ร่วมสมัย
- มันให้ข้อมูลเชิงลึกเกี่ยวกับบทบาทของไมโครโค้ดในโปรเซสเซอร์และเจาะลึกกระบวนการออกแบบของ Intel
- เน้นอิทธิพลของโปรเซสเซอร์ 386 ต่อการประมวลผล 32 บิตและรวมถึงเกร็ดเล็กเกร็ดน้อยส่วนบุคคลที่แบ่งปันโดยผู้แสดงความคิดเห็นที่เกี่ยวข้องกับชิป

## [Firefox add-on เพื่อเปิดวิดีโอ YouTube ในส่วนหน้าทางเลือก](https://github.com/d3vr/yt-siphon)

- ผู้ใช้ได้พัฒนา Firefox Add-On เพื่อตอบสนองต่อการถูกบล็อกโดย YouTube สําหรับการใช้ adblocker
- Add-On นี้ช่วยให้พวกเขาเปิดวิดีโอโดยใช้ส่วนหน้าทางเลือกรวมถึง piped.video
- ช่วยให้ผู้ใช้สามารถปรับแต่งส่วนหน้าเริ่มต้นและใช้งาน keybindings เพื่อเปิดวิดีโอในส่วนหน้าที่กําหนดค่าไว้

### [ปฏิกิริยา](https://news.ycombinator.com/item?id=37881771)

- มีการสร้างส่วนเสริม Firefox ที่ช่วยให้ผู้ใช้สามารถเปิดวิดีโอ YouTube บนแพลตฟอร์มทางเลือกหลังจากการบล็อกผู้ใช้ของ YouTube ด้วยตัวบล็อกโฆษณา
- ผู้ใช้แนะนําให้ใช้ส่วนขยายเพื่อเปลี่ยนเส้นทาง URL ของ YouTube ไปยังแพลตฟอร์มเช่น Invidious หรือ Piped เพื่อประสบการณ์การท่องเว็บที่เป็นส่วนตัวมากขึ้น แต่สิ่งนี้ทําให้เกิดความกังวลเกี่ยวกับผลกระทบต่อการแสดงผลโฆษณาและเงินทุนสําหรับเบราว์เซอร์เช่น Firefox
- การสนทนาถกเถียงกันว่าเบราว์เซอร์ควรมีความสามารถในตัวเพื่อหลีกเลี่ยงมาตรการบล็อกหรือไม่ในขณะที่ตระหนักถึงความสําคัญของโฆษณาและการสมัครสมาชิกเพื่อชดเชยต้นทุนพื้นที่เก็บข้อมูลและแบนด์วิดท์

## [คลื่นเสียงทําลายเนื้องอกได้รับการอนุมัติจาก FDA สําหรับการรักษาตับในมนุษย์](https://news.umich.edu/tumor-destroying-sound-waves-receive-fda-approval-for-liver-treatment-in-humans/)
