# 2023-10-29

## [OpenSign - ทางเลือกโอเพ่นซอร์สสําหรับ DocuSign](https://github.com/OpenSignLabs/OpenSign)

- OpenSign เป็นทางเลือกโอเพ่นซอร์สสําหรับแพลตฟอร์มการลงนามเอกสารเชิงพาณิชย์เช่น DocuSign, PandaDoc และ SignNow
- คุณสมบัติที่นําเสนอโดย OpenSign รวมถึงการลงนามในเอกสารที่ปลอดภัยอินเทอร์เฟซที่ใช้งานง่ายการสนับสนุนหลายแพลตฟอร์มสิ่งอํานวยความสะดวกการทํางานร่วมกันเส้นทางการตรวจสอบใบรับรองการเสร็จสมบูรณ์และการรวม API
- พัฒนาโดย OpenSignLabs, OpenSign ได้รับอนุญาตภายใต้ใบอนุญาต AGPL, ยินดีต้อนรับการมีส่วนร่วมของชุมชน, และได้รับการสนับสนุนและข้อเสนอแนะจากผู้ใช้และผู้มีส่วนร่วม.

### [ปฏิกิริยา](https://news.ycombinator.com/item?id=38052344)

- OpenSign Labs ได้เปิดตัว OpenSign ซึ่งเป็นทางเลือกโอเพ่นซอร์สสําหรับ DocuSign ซึ่งจุดประกายการถกเถียงเกี่ยวกับความถูกต้องตามกฎหมายของลายเซ็นดิจิทัลและความลึกของการวิจัยทางกฎหมายของ OpenSign
- การเสวนายังเน้นถึงความต้องการเครื่องแบบสําหรับลายเซ็นดิจิทัลทั่วภูมิภาค โดยอ้างถึง IT Act 2000 ในอินเดียและ eIDAS ในยุโรป
- ควบคู่ไปกับการอภิปรายเกี่ยวกับความปลอดภัยและความน่าเชื่อถือของ OpenSign ข้อดีและข้อเสียของซอฟต์แวร์โอเพ่นซอร์สและบทบาทของความไว้วางใจและความเป็นไปได้ในระยะยาวในผลิตภัณฑ์ทางธุรกิจที่จัดการเอกสารที่ลงนามแล้ว

## [FCC ต้องการสนับสนุนวิทยุสมัครเล่น](https://www.radioworld.com/news-and-business/business-and-law/fcc-wants-to-bolster-amateur-radio)

- Federal Communications Commission (FCC) ตั้งใจที่จะปรับปรุงคลื่นวิทยุสมัครเล่นให้ทันสมัยโดยแทนที่ข้อ จํากัด อัตราบอด (ความเร็วในการส่งข้อมูล) ด้วยการ จํากัด แบนด์วิดท์โดยมีเป้าหมายเพื่อส่งเสริมนวัตกรรมและปรับปรุงการสื่อสารในกรณีฉุกเฉิน
- American Radio Relay League (ARRL) ซึ่งเป็นองค์กรสําหรับผู้ที่ชื่นชอบวิทยุสมัครเล่นสนับสนุนความคิดริเริ่มนี้
- FCC กําลังพิจารณาที่จะลบข้อ จํากัด อัตราบอดในย่านความถี่อื่น ๆ ด้วยและเปิดรับข้อเสนอแนะเกี่ยวกับการตั้งค่าขีด จํากัด แบนด์วิดท์ที่เหมาะสม การอัปเดตเหล่านี้คาดว่าจะเป็นประโยชน์อย่างมากต่อผู้ประกอบการวิทยุสมัครเล่นและสนับสนุนการสื่อสารข้อมูล

### [ปฏิกิริยา](https://news.ycombinator.com/item?id=38052577)

- Federal Communications Commission (FCC) กําลังแก้ไขข้อบังคับวิทยุสมัครเล่นตามคําร้องจาก American Radio Relay League (ARRL) ซึ่งอาจส่งผลให้มีการทดลองเพิ่มขึ้นและเทคนิคการมอดูเลตใหม่
- หัวข้อสําคัญภายใต้การพิจารณาของ FCC ได้แก่ รหัสมอร์สขีด จํากัด อัตราบอดการเข้ารหัสประโยชน์ที่อาจเกิดขึ้นจากการเข้ารหัสในวิทยุสมัครเล่นและความถี่ที่เพิ่มขึ้นและอัตราการส่งข้อมูลสําหรับผู้ปฏิบัติงาน
- ในขณะที่การปรับปรุงนํามาซึ่งความตื่นเต้นในชุมชนความกังวลเกี่ยวกับการแทรกแซงที่เป็นไปได้และข้อ จํากัด เกี่ยวกับกิจกรรมวิทยุที่ไม่ใช่เชิงพาณิชย์และการทดลองก็แสดงออกเช่นกัน

## [MagicaVoxel - โปรแกรมแก้ไขศิลปะ voxel ฟรีและตัวแสดงการติดตามเส้นทางแบบโต้ตอบ](https://ephtracy.github.io/)

- MagicaVoxel เป็นโปรแกรมแก้ไขศิลปะ voxel ฟรีและ renderer นําเสนอฟังก์ชันการทํางานและทางลัดหลายอย่างที่ออกแบบมาสําหรับการสร้างโมเดล 3 มิติและภาพเคลื่อนไหว อย่างไรก็ตามไม่สามารถขายในเชิงพาณิชย์หรือรวมกับซอฟต์แวร์อื่นได้
- การอัปเดตล่าสุดได้เพิ่มประสิทธิภาพการสร้างภาพเคลื่อนไหวการแสดงผลการแก้ไขการเปลี่ยนแปลงระบบวัสดุและตัวเลือกแปรง เวอร์ชันล่าสุดนํามาซึ่งการปรับปรุงรวมถึงโหมดการเลือกใหม่ตัวเลือกแปรงที่ปรับได้และคุณภาพการเรนเดอร์ที่ดีขึ้น
- เครื่องมือนี้รวมคุณสมบัติที่เป็นเอกลักษณ์เพิ่มเติมเช่นการควบคุมกล้องชุดรูปแบบการแปลภาษาและรองรับการส่งออกโมเดล voxel ในรูปแบบต่างๆด้วยคุณสมบัติต่างๆเช่นการกระเจิงหมอกและเฉดสี voxel

### [ปฏิกิริยา](https://news.ycombinator.com/item?id=38050106)

- MagicaVoxel เป็นโปรแกรมแก้ไขศิลปะ voxel ฟรียอดนิยมและตัวแสดงการติดตามเส้นทางที่ใช้กันอย่างแพร่หลายในการสร้างแบบจําลอง voxel และทิวทัศน์เช่นใน voxel MMORPG แบบโอเพนซอร์ส Veloren
- แม้ว่าจะไม่ใช่โอเพ่นซอร์ส แต่ก็มีการติดตามที่แข็งแกร่งเนื่องจากความเรียบง่ายประสบการณ์การใช้งานที่สนุกสนานและความเข้ากันได้กับการพิมพ์ 3 มิติ มันยังกระตุ้นการพัฒนาเครื่องมือที่คล้ายกัน
- ผู้ใช้บางคนได้แสดงความปรารถนาที่จะสนับสนุน Linux และ Apple Silicon ในอนาคตซึ่งบ่งบอกถึงพื้นที่ที่มีศักยภาพของซอฟต์แวร์สําหรับการเติบโตและการขยายตัว

## [รับบัตรห้องสมุดของฉันลงในโทรศัพท์ของฉันวิธีที่ยาก](https://iliana.fyi/blog/ios-wallet-library-card/)

## [ไนโตรเจนอิเล็กโทรดรีดิชั่นที่มีประสิทธิภาพจากกระแสสู่แอมโมเนียเกือบ 100% (2022)](https://www.nature.com/articles/s41586-022-05108-y)

- การศึกษาล่าสุดที่ตีพิมพ์ใน Nature มุ่งเน้นไปที่การลดอิเล็กโทรดรอชั่นของไนโตรเจนเป็นแอมโมเนียซึ่งใช้ได้ในภาคส่วนต่างๆเช่นพลังงานหมุนเวียนการขนส่งและการผลิตปุ๋ย
- นักวิจัยจากมหาวิทยาลัยโมนาชค้นพบว่าอิเล็กโทรไลต์ลิเธียมเกลือลิเธียมที่มีความเข้มข้นสูงทําให้อัตราผลตอบแทนแอมโมเนียสูงมีประสิทธิภาพจากกระแสต่อแอมโมเนียเกือบ 100%
- นวัตกรรมเหล่านี้อาจส่งผลให้วิธีการผลิตแอมโมเนียมีประสิทธิภาพและเป็นมิตรกับสิ่งแวดล้อมมากขึ้นซึ่งอาจมีการใช้งานเชิงพาณิชย์ผ่าน Jupiter Ionics

### [ปฏิกิริยา](https://news.ycombinator.com/item?id=38053586)

- รายงานการวิจัยใหม่เน้นความก้าวหน้าในไนโตรเจนอิเล็กโทรรีดิชั่นให้ประสิทธิภาพสูงในการผลิตแอมโมเนียผ่านการแยกตัวของลิเธียมออกไซด์แบบมัลติโฟตอนที่เกิดจากเลเซอร์
- วิธีนี้ปูทางสําหรับการผลิตแอมโมเนียในท้องถิ่นและทางเลือกเชื้อเพลิงที่ปราศจากคาร์บอนแม้จะต่อสู้กับการใช้พลังงานสูงและความกังวลด้านความปลอดภัย
- การศึกษายังสํารวจการแปลงไฟฟ้าเป็นแอมโมเนียอย่างมีประสิทธิภาพสําหรับการจัดเก็บพลังงานปุ๋ยและการผลิตเชื้อเพลิงในขณะที่กล่าวถึงการปฏิบัติจริงและการพิจารณาด้านความปลอดภัยโดยมีการอภิปรายเกี่ยวกับความสามารถในการปรับขนาดความเป็นพิษและการใช้แอมโมเนียที่อาจเกิดขึ้นสําหรับการผลิตระเบิด

- กฎหมายดังกล่าวได้ริเริ่มการอภิปรายในวงกว้างเกี่ยวกับการจัดการกับข้อมูลที่ผิดทางออนไลน์และผลกระทบต่อเสรีภาพในการแสดงออก การถกเถียงรอบนอกยังเกิดขึ้นเกี่ยวกับการวางผังเมืองการขนส่งสาธารณะและเผด็จการ

## [ตัวติดตั้ง Windows ของ ImageMagick จะไม่ถูกเซ็นชื่ออีกต่อไป](https://github.com/ImageMagick/ImageMagick/discussions/6826)

- ตัวติดตั้ง Windows ของ ImageMagick จะไม่ถูกเซ็นชื่ออีกต่อไปเนื่องจากใบรับรองการลงนามรหัสหมดอายุซึ่งพวกเขาไม่สามารถส่งออกและใช้ในการกระทําของ GitHub ได้อีกต่อไป
- ขณะนี้ทีมกําลังพิจารณาโซลูชันคลาวด์โดยมี Digicert เป็นตัวเลือกเดียวในปัจจุบันโดยเรียกเก็บเงิน 629 ดอลลาร์สําหรับปีเดียว
- ในขณะที่พวกเขากําลังเชิญผู้สนับสนุนเพื่อรับใบรับรองการลงนามรหัสผู้ใช้บางคนได้แนะนําทางเลือกอื่นเช่น SignPath และ Azure KeyVault

### [ปฏิกิริยา](https://news.ycombinator.com/item?id=38055816)

- บทสนทนามุ่งเน้นไปที่การต่อสู้โครงการโอเพ่นซอร์สที่เผชิญในการซื้อและต่ออายุใบรับรองการลงนามรหัสสําหรับซอฟต์แวร์ของพวกเขาโดยมีการวิพากษ์วิจารณ์เกี่ยวกับต้นทุนที่สูงและความไร้ประสิทธิภาพ
- การอภิปรายเกี่ยวกับความยั่งยืนทางการเงินข้อกําหนดสําหรับระบบการระดมทุนและความปลอดภัยที่ได้รับการปรับปรุงและโซลูชันทางเลือกเช่น Azure Key Vault ซึ่งเป็นบริการคลาวด์สําหรับการจัดเก็บและเข้าถึงคีย์การเข้ารหัสอย่างปลอดภัยก็แพร่หลายเช่นกัน
- เน้นความสําคัญของการยืนยันตัวตนความไว้วางใจในรหัสและความกังวลเกี่ยวกับซอฟต์แวร์ที่เป็นกรรมสิทธิ์และใบรับรองที่ลงนามด้วยตนเอง คุณค่าและจุดอ่อนของ ImageMagick ซึ่งเป็นชุดซอฟต์แวร์โอเพ่นซอร์สสําหรับการแก้ไขภาพก็ถูกบันทึกไว้เช่นกัน
