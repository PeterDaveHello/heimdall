## [Phind Model เอาชนะ GPT-4 ในการเข้ารหัสด้วยความเร็ว GPT-3.5 และบริบท 16k](https://www.phind.com/phindmodelhn)

- Phind ได้ปรับปรุงโมเดลรุ่นที่ 7 เพื่อให้มีประสิทธิภาพเหนือกว่าความสามารถในการเขียนโค้ดของ GPT-4 และให้คําตอบสําหรับคําถามทางเทคนิคเร็วขึ้นห้าเท่า
- โมเดลที่อัปเดตซึ่งใช้ CodeLlama-34B แบบโอเพนซอร์สได้คะแนน HumanEval 74.7%
- แม้จะมีปัญหาบางอย่างเกี่ยวกับการจัดการการสืบค้นที่ซับซ้อน แต่โมเดลนี้ได้รับการยอมรับอย่างอบอุ่นจากชุมชนผู้ใช้ และสามารถประมวลผลโทเค็นได้มากถึง 100 โทเค็นต่อวินาทีและรองรับโทเค็นสูงสุด 16k

## [macOS Sonoma Boot ล้มเหลว](https://github.com/AsahiLinux/docs/wiki/macOS-Sonoma-Boot-Failures)

- ผู้ใช้ MacBook Pro ประสบกับความล้มเหลวในการบูตอย่างรุนแรงเนื่องจากข้อบกพร่องที่สําคัญใน macOS Sonoma และ macOS Ventura 13.6 ซึ่งนําไปสู่หน้าจอสีดําอย่างต่อเนื่อง
- ข้อบกพร่องส่งผลกระทบต่อผู้ใช้ทั้งหมดรวมถึงผู้ที่มีและไม่ได้ติดตั้ง Asahi Linux โดยขณะนี้ตัวติดตั้งจะตรวจสอบปัญหาเหล่านี้โดยอัตโนมัติ
- ผู้ใช้ควรละเว้นจากการอัพเกรดระบบจนกว่า Apple จะแก้ไขข้อบกพร่อง ผู้ที่ได้รับผลกระทบแล้วสามารถใช้ตัวติดตั้ง Asahi Linux เพื่อการวินิจฉัยและแนวทางแก้ไขในขณะที่ผู้ที่ไม่สามารถบูตได้อาจต้องใช้โหมด DFU (Device Firmware Update) สําหรับการกู้คืน

### [ปฏิกิริยา](https://news.ycombinator.com/item?id=38089342)

- การตัดสินใจของ Apple ในการจัดเก็บอัตราการรีเฟรชจอแสดงผลในหน่วยความจําเข้าถึงแบบสุ่ม (NVRAM) แบบไม่ลบเลือนของอุปกรณ์อาจทําให้เกิดความเสียหายในการบูตใน macOS ตามการสนทนาของผู้ใช้ใน Hacker News
- ปัญหานี้ดูเหมือนจะเกี่ยวข้องกับการเปลี่ยนแปลงโหมดการแสดงผลซอฟต์แวร์ bootloader ที่ล้าสมัยและปัญหาที่อาจเกิดขึ้นกับการอัปเดต macOS บางอย่าง ผู้ใช้บางคนต้องเผชิญกับความล้มเหลวของฮาร์ดแวร์และมีความกังวลเกี่ยวกับการตอบสนองของ Apple
- ผู้ใช้ได้ถกเถียงกันระหว่าง Extensible Firmware Interfaces แบบกราฟิก (EFI) และทางเลือกแบบข้อความเท่านั้นโดยมีการสนทนาที่กว้างขึ้นเกี่ยวกับอินเทอร์เฟซผู้ใช้ที่แตกต่างกันตัวโหลดการบูตและความละเอียดหน้าจอส่งผลต่อประสิทธิภาพของคอมพิวเตอร์อย่างไร

## [Firefox เร็วขึ้นสําหรับผู้ใช้จริงในปี 2023](https://hacks.mozilla.org/2023/10/down-and-to-the-right-firefox-got-faster-for-real-users-in-2023/)

- เบราว์เซอร์ Firefox ของ Mozilla ในปี 2023 ได้รับการปรับปรุงประสิทธิภาพซึ่งเป็นส่วนหนึ่งของความพยายามของ Speedometer 3 โดยมีเป้าหมายเพื่อจําลองประสบการณ์ผู้ใช้ในโลกแห่งความเป็นจริงและการปรับปรุงไดรฟ์
- การปรับปรุงรวมถึงการลดเวลาในการโหลดวิชวลแรกลง 15% จากการรับไบต์แรกของข้อมูลจากเครือข่ายและลดเวลาดําเนินการ JavaScript ลง 20%
- นอกจากนี้ยังมีการปรับปรุง 10% ที่บันทึกไว้ใน "เวลาแฝงในการกดปุ่มในปัจจุบัน" ซึ่งให้ข้อเสนอแนะแก่ผู้ใช้เร็วขึ้นเมื่อพิมพ์ซึ่งแสดงให้เห็นถึงความคืบหน้าของ Firefox ในการจัดลําดับความสําคัญของประสบการณ์ผู้ใช้

### [ปฏิกิริยา](https://news.ycombinator.com/item?id=38087573)

- ผู้ใช้และนักพัฒนา Firefox ได้ประเมินความเร็วความก้าวหน้าและปัญหาด้านประสิทธิภาพของเบราว์เซอร์ตั้งแต่ปี 2023 เวอร์ชันล่าสุดและส่วนเสริมได้รับการจัดอันดับสูงแม้จะมีปัญหาความไม่ลงรอยกันของซอฟต์แวร์และการยอมรับมาตรฐานใหม่อย่างช้าๆ
- การจัดการ JavaScript และแนวทางปฏิบัติในการรวบรวมข้อมูลเป็นประเด็นสําคัญของการอภิปรายโดยผู้ใช้บางคนเน้นว่าทั้งสองอย่างมีความสําคัญต่อการปรับปรุงของ Firefox แม้จะมีข้อกังวลด้านความเป็นส่วนตัวก็ตาม
- การใช้ข้อมูลการวัดและส่งข้อมูลทางไกลของเบราว์เซอร์ซึ่งเป็นการละเมิดความเป็นส่วนตัวที่อาจเกิดขึ้นได้รับการถกเถียงกันโดยผู้ใช้เน้นบทบาทในการปรับปรุงซอฟต์แวร์และสนับสนุนระบบการเลือกใช้

## [ต้นกําเนิด uBlock 1.53](https://github.com/gorhill/uBlock/releases/tag/1.53.0)

- โครงการ Gorhill / uBlock บน GitHub ได้เปิดตัวการอัปเดตเวอร์ชัน 1.53.0 ซึ่งรวมถึงการปรับปรุงการแก้ไขข้อบกพร่องและนิพจน์การกรองใหม่สําหรับเอาต์พุตคนตัดไม้

- ผู้สนับสนุน kzar, ryanbr และ peace2000 มีความสําคัญในการดําเนินการอัปเดตเหล่านี้ ซึ่งรวมถึงการปรับปรุงสคริปต์ที่มีอยู่และการเปลี่ยนแปลงตามความคิดเห็นของผู้ใช้

## [การแบนโฆษณาเชิงพฤติกรรมของ Meta ของนอร์เวย์ขยายไปถึงทั้งสหภาพยุโรป](https://www.datatilsynet.no/aktuelt/aktuelle-nyheter-2023/datatilsynets-vedtak-mot-meta-utvides-til-eueos-og-gjores-permanent/)

- คณะกรรมการคุ้มครองข้อมูลแห่งยุโรปได้ขยายการห้ามทําการตลาดตามพฤติกรรมบน Facebook และ Instagram ของนอร์เวย์ทั่วทั้งสหภาพยุโรป/EEA การตัดสินใจนี้กล่าวถึงการติดตามการเฝ้าระวังและการจัดทําโปรไฟล์ที่ผิดกฎหมายบนแพลตฟอร์มเหล่านี้
- แม้จะมีข้อห้ามในอดีต Meta (เดิมชื่อ Facebook) ยังคงจัดการข้อมูลผู้ใช้อย่างผิดกฎหมาย ดังนั้นคณะกรรมการจึงได้ส่งต่อการตัดสินใจนี้ไปยังหน่วยงานคุ้มครองข้อมูลของไอร์แลนด์เพื่อบังคับใช้การห้ามสํานักงานใหญ่ในยุโรปของ Meta ในไอร์แลนด์
- แม้ว่า Meta วางแผนที่จะรักษาความปลอดภัยความยินยอมของผู้ใช้สําหรับการใช้ข้อมูลสําหรับการตลาดตามพฤติกรรมในอนาคต แต่ยังไม่ได้ดําเนินการเปลี่ยนแปลงใด ๆ คณะกรรมการยังแสดงความสงสัยต่อวิธีแก้ปัญหาที่แนะนําของ Meta ในการเรียกเก็บเงินจากผู้ใช้ที่ปฏิเสธที่จะยินยอม

- สหภาพยุโรปได้ขยายการห้ามการโฆษณาตามพฤติกรรมของ Meta เนื่องจากความล้มเหลวในการขอความยินยอมจากผู้ใช้และวางแผนที่จะเรียกเก็บเงินจากผู้ใช้ที่เลือกไม่แชร์ข้อมูล
- การแบนยังคงมีผลจนกว่า Meta จะแก้ไขปัญหาเหล่านี้กระตุ้นให้เกิดการอภิปรายเกี่ยวกับความเป็นส่วนตัวของข้อมูลผลกระทบจากการปิดกั้นโฆษณาและประสิทธิภาพของการโฆษณาในหมู่ผู้ใช้
- คณะกรรมการคุ้มครองข้อมูลแห่งยุโรปได้สั่งห้าม Disqus จากการประมวลผลข้อมูลส่วนบุคคลในนอร์เวย์เป็นการชั่วคราว โดยอ้างถึงการละเมิดกฎระเบียบคุ้มครองข้อมูลทั่วไป (GDPR) ของสหภาพยุโรป

## [MicroTCP สแต็ค TCP/IP ที่น้อยที่สุด](https://github.com/cozis/microtcp)

- MicroTCP เป็นสแต็คเครือข่าย TCP / IP ที่น้อยที่สุด แต่ครอบคลุมซึ่งสร้างโดย "cozis" ซึ่งเป็นส่วนหนึ่งของโครงการการเรียนรู้ส่วนบุคคลในระหว่างหลักสูตรเครือข่ายคอมพิวเตอร์ที่ Università degli Studi di Napoli Federico II
- สแต็กที่เข้ากันได้กับทั้ง Windows และ Linux มีฟังก์ชันการทํางานเต็มรูปแบบสําหรับการจัดการการสื่อสาร ARP, IPv4, ICMP และ TCP และอาจปรับให้เข้ากับไมโครคอนโทรลเลอร์ได้
- แม้จะมีการทดสอบอย่างต่อเนื่อง MicroTCP มีความสามารถในการจัดการการรับส่งข้อมูล HTTP ของเครือข่ายท้องถิ่นอยู่แล้ว

### [ปฏิกิริยา](https://news.ycombinator.com/item?id=38093353)

- OpenAI ได้เปิดตัวเครื่องมือรู้จําเสียงพูด AI เวอร์ชันที่มีประสิทธิภาพมากขึ้น Whisper ที่เรียกว่า Distil-Whisper เร็วกว่าหกเท่าและเล็กกว่า 49% ทําให้เหมาะสําหรับอุปกรณ์และเว็บเบราว์เซอร์ที่มีพลังงานจํากัด
- Distil-Whisper มีอัตราข้อผิดพลาด 1-2% สําหรับการถอดความภาษาอังกฤษส่วนใหญ่เกิดจากปัญหาเกี่ยวกับเครื่องหมายวรรคตอนและคําเติม
- ผู้ใช้บางคนกําลังมองหาการปรับแต่งโมเดลนี้และอาจรวมเข้ากับเครื่องมืออื่น ๆ สําหรับการสร้างผู้ช่วยเสียงส่วนตัว
