# 2023-11-21

## [YouTube ชะลอเวลาในการโหลดวิดีโอสําหรับผู้ใช้ Firefox ทําให้เกิดความกังวลเกี่ยวกับการละเมิดการต่อต้านการผูกขาด](https://old.reddit.com/r/youtube/comments/17z8hsz/youtube_has_started_to_artificially_slow_down/)

- ผู้ใช้ใน subreddit r / youtube กําลังเพิ่มความกังวลเกี่ยวกับ YouTube เช่นการโหลดวิดีโอช้าบน Firefox และการละเมิดการต่อต้านการผูกขาดที่อาจเกิดขึ้น
- มีการอภิปรายเกี่ยวกับ "Web Integrity API" ที่อาจจํากัดการเข้าถึงเว็บไซต์สําหรับเบราว์เซอร์บางตัว ซึ่งจุดประกายให้มีการกํากับดูแลการต่อต้านการผูกขาดและดําเนินการทางกฎหมายกับ YouTube
- ผู้ใช้ประสบกับความล่าช้าการบัฟเฟอร์และปัญหาเกี่ยวกับตัวบล็อกโฆษณาบนแพลตฟอร์มทําให้บางคนต้องหาวิธีแก้ไขในขณะที่คนอื่นแสดงความไม่พอใจกับการกระทําของ YouTube

### [ปฏิกิริยา](https://news.ycombinator.com/item?id=38345858)

- ผู้ใช้รายงานเวลาในการโหลดวิดีโอที่ช้าบน YouTube เมื่อใช้ Firefox ซึ่งอาจเกิดจากโค้ดโดยเจตนาทําให้การโหลดวิดีโอล่าช้าเพื่อตรวจจับการใช้งาน adblocker
- การอภิปรายกําลังดําเนินอยู่เกี่ยวกับความตั้งใจที่อยู่เบื้องหลังคุณลักษณะบางอย่างของเว็บไซต์ความสําคัญของเวลาในการโหลดและการใช้ฟังก์ชั่นการนอนหลับในการเขียนโปรแกรม
- ผู้ใช้กําลังสํารวจแพลตฟอร์มทางเลือกสําหรับการดูและดาวน์โหลดวิดีโอแสดงความผิดหวังกับบริการสตรีมมิ่งและแสดงความกังวลเกี่ยวกับการเลือกปฏิบัติและการโฆษณาที่เป็นเท็จ
- นอกจากนี้ยังมีการอภิปรายเกี่ยวกับอิทธิพลของ Google ต่อเบราว์เซอร์ประสิทธิภาพและความเข้ากันได้ของแพลตฟอร์ม Google บน Firefox และพฤติกรรมต่อต้านการแข่งขันที่อาจเกิดขึ้นของ Google

## [พนักงาน OpenAI เรียกร้องให้คณะกรรมการลาออกและคุกคามการอพยพจํานวนมาก](https://www.wired.com/story/openai-staff-walk-protest-sam-altman/)

- พนักงานกว่า 730 คนของ OpenAI ได้ลงนามในจดหมายเปิดผนึกเรียกร้องให้คณะกรรมการของสตาร์ทอัพลาออกและคืนสถานะซีอีโอที่ถูกขับไล่ Sam Altman
- พนักงานเชื่อว่าการตัดสินใจของคณะกรรมการในการเลิกจ้าง Altman และถอด Greg Brockman ออกจากคณะกรรมการทําให้งานและภารกิจของ บริษัท ตกอยู่ในความเสี่ยง
- จดหมายยังเรียกร้องให้มีการแต่งตั้งสมาชิกคณะกรรมการอิสระใหม่สองคน และมีความเป็นไปได้ที่พนักงานอาจเข้าร่วม Altman ที่ Microsoft ซึ่งเขาและ Brockman ได้รับการแต่งตั้งให้เป็นผู้นําหน่วยวิจัย AI ขั้นสูงใหม่

### [ปฏิกิริยา](https://news.ycombinator.com/item?id=38347868)

- OpenAI กําลังเผชิญกับความขัดแย้งภายในและความเป็นไปได้ของการลาออกเนื่องจากความไม่เห็นด้วยกับคณะกรรมการซึ่งอาจเกี่ยวข้องกับอิทธิพลของ Microsoft และการเข้าถึงรหัส
- การเข้าซื้อกิจการ OpenAI โดย Microsoft และการใช้ชุดข้อมูลของ Quora ที่อาจเกิดขึ้นทําให้เกิดความกังวลเกี่ยวกับการควบคุมเงินทุนของ OpenAI ของ Microsoft และการลดค่าเครดิต Azure
- มีการถกเถียงกันเกี่ยวกับอนาคตของปัญญาประดิษฐ์ทั่วไป (AGI) การใช้แบบจําลองภาษาในการก้าวไปสู่ AGI และผลกระทบทางจริยธรรมของ บริษัท ขนาดใหญ่ที่มีอํานาจเหนือกว่าบริษัทขนาดเล็ก การอภิปรายยังตั้งคําถามถึงความเป็นผู้นํา ความน่าเชื่อถือ และชื่อเสียงของ OpenAI

## [อดีตซีอีโอของ OpenAI Sam Altman ยังคงผลักดันให้กลับมาแม้จะถูกไล่ออก](https://www.theverge.com/2023/11/20/23969586/sam-altman-plotting-return-open-ai-microsoft)

- อดีตซีอีโอของ OpenAI Sam Altman และผู้ร่วมก่อตั้ง Greg Brockman กําลังมองหาที่จะกลับไปที่ บริษัท หากสมาชิกคณะกรรมการที่ไล่ Altman ออกจากกัน
- การแต่งตั้งซีอีโอคนใหม่เมื่อเร็ว ๆ นี้ Emmett Shear ในตอนแรกดูเหมือนจะยุติโอกาสในการกลับมาของ Altman แต่มีแรงกดดันอย่างต่อเนื่องต่อคณะกรรมการจากพนักงานและสมาชิกคณะกรรมการ Ilya Sutskever ที่สนับสนุน Altman
- การย้าย Altman ไปยัง Microsoft ยังไม่เสร็จสิ้น และการเจรจายังคงดําเนินการเพื่อกําหนดลักษณะของการทํางานร่วมกันในอนาคต การแย่งชิงอํานาจภายใน OpenAI เกิดขึ้นซึ่งนําไปสู่ความโกรธของพนักงานและการปลดพนักงานในการประชุมของ บริษัท Altman และ Microsoft กําลังทํางานเพื่อลงมติที่รักษาศักดิ์ศรีของคณะกรรมการ

### [ปฏิกิริยา](https://news.ycombinator.com/item?id=38352891)

- Sam Altman อดีตประธานของ Y Combinator และอดีต CEO ของ OpenAI กําลังมองหาที่จะกลับมาเป็น CEO หลังจากเรียกร้องให้คณะกรรมการลาออกโดยพนักงาน 700 คน
- คณะกรรมการของ OpenAI ต้องเผชิญกับการวิพากษ์วิจารณ์ว่าชอบผลกําไรมากกว่าความปลอดภัยทําให้เกิดการถกเถียงกันเกี่ยวกับจริยธรรมของเทคโนโลยีการป้องกันประเทศและบทบาทของสหรัฐอเมริกาในการเผยแพร่คุณค่าทางวัฒนธรรม
- การเก็งกําไรล้อมรอบอนาคตของ OpenAI การมีส่วนร่วมของ Microsoft และแรงจูงใจของบุคคลที่เกี่ยวข้องในขณะที่ความกังวลเกี่ยวกับความรับผิดทางกฎหมายและการล่มสลายของ บริษัท ที่อาจเกิดขึ้น นอกจากนี้ยังมีการกล่าวถึงผลกระทบของการจากไปของ Altman ต่อพนักงาน OpenAI และองค์กร

## [พนักงาน OpenAI พร้อมที่จะลาออกเนื่องจากการขับไล่ของ Sam Altman ทําให้เกิดฟันเฟือง](https://www.businessinsider.com/openais-employees-given-explanations-why-sam-altman-out-2023-11)
