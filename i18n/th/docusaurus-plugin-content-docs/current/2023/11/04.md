## [นักพัฒนาที่เกษียณอายุเพื่อปล่อยซอร์สโค้ดสี่ทศวรรษรวมถึงระบบปฏิบัติการที่หายาก](https://dunfield.themindfactory.com/dnldsrc.htm)

- Dave นักพัฒนาซอฟต์แวร์ที่เกษียณอายุแล้วกําลังสร้างซอร์สโค้ดความยาว 40 ปีสําหรับระบบปฏิบัติการสาธารณูปโภคเครื่องมือและโปรแกรมต่างๆต่อสาธารณะโดยมีเจตนาที่จะเป็นประโยชน์ต่อผู้อื่น
- ผลงานที่โดดเด่นของเขา ได้แก่ "ARMOS" เคอร์เนลหลายกระบวนการ/งานสําหรับ ARM, "CUBIX" ระบบปฏิบัติการที่ออกแบบมาสําหรับ CPU 6809 และ "DMS" ซึ่งเป็นระบบปฏิบัติการแรกของเขาที่พัฒนาขึ้นในปี 1980
- รุ่นนี้ให้โอกาสพิเศษสําหรับบุคคลเทคโนโลยีที่จะเกิดขึ้นหรือสนใจที่จะเรียนรู้จากงานของโปรแกรมเมอร์ที่มีประสบการณ์และมรดกรหัส

### [ปฏิกิริยา](https://news.ycombinator.com/item?id=38132016)

- Dave นักพัฒนาซอฟต์แวร์ที่มีประสบการณ์กําลังปล่อยซอร์สโค้ดมูลค่า 40 ปีของระบบปฏิบัติการยูทิลิตี้เครื่องมือและโปรแกรมที่หลากหลายเพื่อประโยชน์ของนักพัฒนารายอื่น
- โปรแกรมสําคัญที่เขามีส่วนร่วม ได้แก่ "ARMOS" เคอร์เนลที่อํานวยความสะดวกในกระบวนการ/งานหลายอย่างสําหรับโปรเซสเซอร์ ARM "CUBIX" ระบบปฏิบัติการสําหรับ CPU 6809 และ "DMS" ระบบปฏิบัติการแรกสุดของเขาที่พัฒนาขึ้นในปี 1980
- การกระทําที่ไม่เหมือนใครนี้ส่งเสริมการเรียนรู้และการแบ่งปันความรู้ในชุมชนการพัฒนาซอฟต์แวร์และให้ทรัพยากรอันล้ําค่าสําหรับนักพัฒนาที่มีประสบการณ์และมือใหม่

## [การหยุดทํางานตลอด 24 ชั่วโมงของ Cloudflare เน้นย้ําถึงความสําคัญของการจัดการพลังงานในศูนย์ข้อมูลและความเสี่ยงของการพึ่งพาบริษัทเดียว](https://news.ycombinator.com/item?id=38112515)

- Cloudflare ซึ่งเป็น บริษัท ที่ให้บริการออนไลน์ที่หลากหลายประสบกับการหยุดทํางานของ API ซึ่งอาจเกิดจากการสูญเสียพลังงานในศูนย์ข้อมูลของพวกเขา
- ผู้ใช้เน้นย้ําถึงความสําคัญของเครื่องสํารองไฟ (UPS) และลานเครื่องกําเนิดไฟฟ้า และความจําเป็นในการวางแผนฉุกเฉินด้านพลังงานที่ครอบคลุมและ Availability Zone เพื่อลดความเสียหายจากเหตุการณ์ดังกล่าว
- ในขณะที่สังเกตเห็นการพึ่งพา บริษัท ขนาดใหญ่เพียงแห่งเดียวเช่น Amazon, Google, Microsoft และ Cloudflare สําหรับบริการหลักผู้ใช้ยังแสดงความกังวลเกี่ยวกับความเสี่ยงด้านความปลอดภัยและการหยุดทํางานที่อาจเกิดขึ้น อย่างไรก็ตาม Cloudflare ได้รับการยกย่องสําหรับบริการป้องกันและความโปร่งใส

### [ปฏิกิริยา](https://news.ycombinator.com/item?id=38127313)

- Cloudflare ประสบปัญหาการหยุดทํางานของ API เนื่องจากอาจสูญเสียพลังงานในศูนย์ข้อมูล ซึ่งส่งผลกระทบต่อบริการต่างๆ
- เหตุการณ์ดังกล่าวได้กระตุ้นให้เกิดการอภิปรายเกี่ยวกับความสําคัญของแผนฉุกเฉินด้านพลังงานที่ครอบคลุม Uninterruptable Power Supply (UPS) และ Availability Zone ในศูนย์ข้อมูลซึ่งเกิดจากเหตุการณ์ที่คล้ายกันในอดีต
- การถกเถียงยังล้อมรอบการพึ่งพา บริษัท เดียวเช่น Amazon, Google, Microsoft และ Cloudflare สําหรับบริการที่จําเป็นควบคู่ไปกับความกังวลเกี่ยวกับความเสี่ยงด้านความปลอดภัยและการหยุดชะงักที่อาจเกิดขึ้นแม้จะมีความสามารถในการป้องกันและความโปร่งใสที่ได้รับการยอมรับ

- ศาลอุทธรณ์ภาค 4 ของสหรัฐอเมริกาตัดสินให้แมทธิว กิบสัน กับหลุยส์ โกลด์สตัน อดีตผู้พิพากษาศาลครอบครัว โดยตัดสินว่าเธอก้าวล่วงอํานาจตุลาการด้วยการค้นบ้านของกิ๊บสันโดยไม่มีหมายค้นท่ามกลางการหย่าร้าง
- ก่อนหน้านี้โกลด์สตันต้องเผชิญกับการตําหนิ ค่าปรับ และการประณามโดยศาลสูงเวสต์เวอร์จิเนีย ส่งผลให้เธอเกษียณอายุระหว่างการดําเนินคดีฟ้องร้อง
- แม้ว่าโกลด์สตันจะเกษียณอายุ แต่กิบสันและสถาบันเพื่อความยุติธรรม (สํานักงานกฎหมายที่ไม่แสวงหาผลกําไรระดับชาติที่ปกป้องสิทธิตามรัฐธรรมนูญ) ก็ดําเนินคดีเพื่อขจัดการละเมิดที่คล้ายคลึงกันและ จํากัด การแพร่กระจายของหลักคําสอนเรื่องภูมิคุ้มกันของตุลาการ

### [ปฏิกิริยา](https://news.ycombinator.com/item?id=38133279)

- ศาลอุทธรณ์ภาค 4 ของสหรัฐฯ มีมติเป็นเอกฉันท์ให้หลุยส์ โกลด์สตัน อดีตผู้พิพากษาศาลครอบครัว โดยยอมรับว่าเธอดําเนินการค้นบ้านของแมทธิว กิบสัน โดยไม่มีหมายค้นระหว่างกระบวนการหย่าร้าง ซึ่งเกินอํานาจตุลาการของเธอ
- โกลด์สตันซึ่งก่อนหน้านี้ถูกตําหนิปรับและประณามโดยศาลสูงเวสต์เวอร์จิเนียเกษียณอายุท่ามกลางความพยายามในการฟ้องร้อง แต่ Gibson และ Institute for Justice (IJ) ซึ่งเป็น บริษัท กฎหมายที่ไม่แสวงหาผลกําไรระดับชาติที่ปกป้องสิทธิตามรัฐธรรมนูญดําเนินการฟ้องร้องเพื่อยับยั้งการละเมิดที่คล้ายคลึงกัน
- Gibson และ IJ ติดตามการฟ้องร้อง Goldston แม้ว่าเธอจะเกษียณอายุแล้ว แต่ก็มีผลกระทบที่อาจเกิดขึ้นกับหลักคําสอนเรื่องภูมิคุ้มกันของตุลาการซึ่งพยายามป้องกันไม่ให้ผู้พิพากษาเข้าถึงมากเกินไป

## [คนงานโดยเฉพาะกลุ่มประชากรที่อายุน้อยกว่าเปิดรับการเสียสละสําหรับการทํางานสี่วันต่อสัปดาห์การสํารวจพบว่า](https://www.cnbc.com/2023/11/01/what-young-people-would-give-up-for-a-4-day-workweek.html)

- การสํารวจ Bankrate ใหม่ระบุว่า 92% ของคนงานอายุน้อยและ 86% ของคนงานสูงอายุเปิดกว้างที่จะประนีประนอมบางอย่างเพื่อประโยชน์ในการทํางานสี่วันต่อสัปดาห์
- การเสียสละของคนงานเหล่านี้พร้อมที่จะทํามีตั้งแต่การเปลี่ยนไปสู่การทํางานด้วยตนเองและใช้เวลาทํางานนานขึ้นไปจนถึงการทํางานในวันหยุดสุดสัปดาห์หรือตอนเย็นและแม้แต่การรับค่าจ้างที่ต่ํากว่า
- ด้วยนายจ้างในสหรัฐอเมริกาเพียง 15% ที่เสนอการทํางานสี่วันต่อสัปดาห์เพื่อรับผลประโยชน์ด้านสุขภาพจิตในปี 2023 สิ่งนี้นําเสนอพื้นที่การเติบโตที่มีศักยภาพและโอกาสในการเปลี่ยนแปลงแนวทางปฏิบัติขององค์กร

### [ปฏิกิริยา](https://news.ycombinator.com/item?id=38127203)

- การสํารวจใหม่ของ Bankrate เผยให้เห็นว่า 92% ของคนงานอายุน้อยและ 86% ของคนงานสูงอายุต้องการทํางานสี่วันต่อสัปดาห์แม้ว่าจะจําเป็นต้องเสียสละเช่นชั่วโมงการทํางานที่ยาวนานขึ้นการทํางานในช่วงสุดสัปดาห์และการลดเงินเดือน
- การสํารวจยังแสดงให้เห็นถึงความโน้มเอียงในหมู่พนักงานในการเปลี่ยนงานหรือกลับไปทํางานด้วยตนเองเพื่อให้เป็นไปตามตารางเวลาสี่วัน
- แม้จะมีความสนใจ แต่มีเพียง 15% ของนายจ้างในสหรัฐอเมริกาที่เสนอตัวเลือกสัปดาห์ละสี่วันสําหรับผลประโยชน์ด้านสุขภาพจิตซึ่งชี้ให้เห็นถึงโอกาสที่ไม่ได้รับการตอบสนองสําหรับองค์กรในการดึงดูดและรักษาผู้มีความสามารถ
