# 2023-11-29

## [ความผิดพลาดของยานสํารวจดาวอังคารมูลค่า 500 ล้านดอลลาร์ของฉัน: บทเรียนในความล้มเหลวและนวัตกรรม](https://www.chrislewicki.com/articles/failurestory)

- ในปี 2003 ในระหว่างขั้นตอนการประกอบและทดสอบของ Spirit Mars Rover ของ NASA กระแสไฟฟ้ากระชากทําลายมอเตอร์ของ Rock Abrasion Tool (RAT) ที่ติดอยู่กับแขนหุ่นยนต์ของโรเวอร์ส่งผลให้สูญเสียเงิน 500 ล้านดอลลาร์
- เหตุการณ์นี้เน้นย้ําถึงความสําคัญของการเรียนรู้จากความล้มเหลวและความสําคัญของการสนับสนุนจากเพื่อนร่วมงานเนื่องจากในที่สุดการเชื่อมต่อก็ได้รับการฟื้นฟูและการทดสอบยังคงดําเนินต่อไป
- ผู้เขียน Chris Lewicki สนับสนุนให้ผู้อื่นแบ่งปันเรื่องราวความล้มเหลวของตนเองเพื่อส่งเสริมภูมิปัญญาโดยรวมและส่งเสริมนวัตกรรมในอุตสาหกรรมอวกาศ

## [Microsoft Open-Sources ThreadX RTOS, ชุดพัฒนา Azure RTOS](https://www.theregister.com/2023/11/28/microsoft_opens_sources_threadx/)

- Microsoft มี ThreadX RTOS แบบโอเพนซอร์สและชุดพัฒนา Azure RTOS ภายใต้สิทธิ์การใช้งาน MIT
- ThreadX เป็น RTOS ยอดนิยมที่ทํางานบนอุปกรณ์หลายพันล้านเครื่องรวมถึง Raspberry Pis
- Microsoft บริจาคซอฟต์แวร์ให้กับ Eclipse Foundation ซึ่งตอนนี้จะเรียกว่า Eclipse ThreadX

### [ปฏิกิริยา](https://news.ycombinator.com/item?id=38445020)

- Microsoft ได้สร้างโอเพ่นซอร์ส ThreadX ของระบบปฏิบัติการแบบเรียลไทม์แบบฝังตัว (RTOS) ซึ่งบ่งชี้ถึงการเปลี่ยนแปลงจากแผนงานสําหรับ Azure RTOS
- การเข้าซื้อกิจการ FreeRTOS โดย Amazon ทําให้ Microsoft เข้าซื้อกิจการ ThreadX แต่ดูเหมือนว่าตอนนี้พวกเขากําลังมุ่งเน้นไปที่พื้นที่อื่น ๆ
- บทความนี้กล่าวถึงความสําคัญของการรับรองความปลอดภัยสําหรับ RTOS ความท้าทายในการปล่อยเฟิร์มแวร์และไดรเวอร์ GPU และการขาดแคลนพนักงานพัฒนาแบบฝังตัว

- ภักดียังคงต้องปฏิบัติตามข้อกําหนดด้านความปลอดภัยและการผลิตและทําการศึกษายืนยัน แต่สัตวแพทย์สามารถสั่งจ่าย LOY-001 ได้แล้วด้วยกระบวนการอนุมัติแบบมีเงื่อนไขของ FDA

## [SDXL Turbo: การสร้างข้อความเป็นภาพคุณภาพสูงแบบเรียลไทม์](https://stability.ai/news/stability-ai-sdxl-turbo)

- SDXL Turbo เป็นโมเดลการสร้างข้อความเป็นภาพแบบเรียลไทม์ที่ใช้เทคโนโลยีการกลั่นใหม่เพื่อให้ได้การสร้างภาพคุณภาพสูง
- ด้วยกระบวนการสร้างภาพแบบขั้นตอนเดียว SDXL Turbo จะลดขั้นตอนที่จําเป็นจาก 50 เหลือเพียงขั้นตอนเดียวทําให้มีประสิทธิภาพมากขึ้น
- แบบจําลองนี้ประกอบด้วย Adversarial Diffusion Distillation (ADD) ซึ่งเป็นเทคนิคการกลั่นที่รักษาความเที่ยงตรงของการสุ่มตัวอย่างสูงและแก้ไขปัญหาทั่วไปที่พบในวิธีการกลั่นอื่น ๆ

### [ปฏิกิริยา](https://news.ycombinator.com/item?id=38450390)

- บทความนี้สํารวจ SDXL Turbo ซึ่งเป็นโมเดลการสร้างข้อความเป็นภาพและทําให้เกิดความกังวลเกี่ยวกับการใช้งานที่ไม่ใช่เชิงพาณิชย์และการหลีกเลี่ยงตลาดบางแห่งของ OpenAI เช่นสื่อลามก
- มันกล่าวถึงการใช้บริการสร้างข้อความ AI แบบชําระเงินและฟรีและข้อกําหนดการคํานวณที่สูงสําหรับการปรับแต่งโมเดลอย่างละเอียด
- บทความนี้เจาะลึกหัวข้อต่างๆเช่นการสร้างรายได้จากโมเดล AI ที่ทํางานในท้องถิ่นความสามารถในการทํากําไรของ OpenAI สื่อลามกที่สร้างโดย AI การครอบงําของ Google ปัญหาทางกฎหมายเกี่ยวกับเนื้อหาที่สร้างโดย AI การใช้และข้อ จํากัด ของภาพที่สร้างโดย AI สถานะลิขสิทธิ์ของผลงานที่สร้างโดย AI และบทบาทของวิศวกรและศิลปิน AI ในกระบวนการสร้างสรรค์
