## [¿Qué pasa, Python? El GIL eliminado, un nuevo compilador, optparse obsoleto](https://www.bitecode.dev/p/whats-up-python-the-gil-removed-a)

- Python tiene ahora un nuevo compilador llamado LPython.
- Pydantic 2 ha sido lanzado con mejoras en velocidad y estabilidad.
- Los módulos getopt y optparse han quedado obsoletos.
- Cython 3.0 tiene ahora mejor soporte para Python puro.
- Se ha hecho una nueva propuesta llamada PEP 722 para especificar dependencias en scripts de un solo archivo.
- Se ha mejorado el soporte de Python VSCode para un rendimiento más rápido.
- Se ha introducido una nueva aplicación de pintura basada en el terminal llamada textual-paint.

- Se estudian alternativas como el uso de lenguajes como C++ o Rust para mejorar el rendimiento.

- Los posibles beneficios y desventajas de eliminar el GIL en Python son objeto de debate, con diversas opiniones sobre las mejoras de rendimiento y los posibles problemas de concurrencia.
