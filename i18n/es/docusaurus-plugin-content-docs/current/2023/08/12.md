## [Tailscale vs. Narrowlink](https://narrowlink.com/docs/comparisons/tailscale)

- Narrowlink y Tailscale son soluciones de código abierto para el acceso remoto seguro y la conectividad a través de redes.
- Tailscale es una plataforma de software como servicio (SaaS) que proporciona una VPN de configuración cero mediante el protocolo WireGuard, con una arquitectura de malla entre pares.
- Narrowlink es una plataforma autoalojada con un modelo cliente-agente-puerta de enlace, que ofrece flexibilidad, transparencia y seguridad, junto con un cifrado opcional de extremo a extremo.
- Narrowlink admite el control de acceso granular, puede desplegarse detrás de una CDN y ofrece más opciones para publicar servicios web que Tailscale.
- Tailscale se centra en la conectividad de los dispositivos, la facilidad de uso y el rendimiento bruto, mientras que Narrowlink da prioridad a la personalización, el control de acceso y la seguridad a través de una solución autoalojada.
- Narrowlink es más eficiente en términos de tamaño binario, uso de memoria y uso de CPU que Tailscale.

### [Reacciones](https://news.ycombinator.com/item?id=37089739)

- El debate compara Tailscale y Narrowlink, dos soluciones de código abierto para el acceso remoto y la conectividad.
- Existe un debate en torno a la naturaleza de código abierto de Tailscale debido a su servidor cerrado, aunque existe una implementación de código abierto llamada Headscale.
- Tailscale es elogiado por su interfaz web fácil de usar, su propuesta de valor y sus funciones adicionales como MagicDNS y la gestión de claves SSH.
