## [Phi-3 Mini: Výkonný jazykový model pro mobilní zařízení](https://arxiv.org/abs/2404.14219)

- Jazykový model phi-3-mini s 3,8 miliardami parametrů natrénovaných na 3,3 bilionech tokenů konkuruje větším modelům, jako jsou Mixtral 8x7B a GPT-3.5, a nabízí vysoký výkon a zároveň možnost nasazení v telefonu.
- Tréninková sada dat modelu je vylepšenou verzí sady použité pro phi-2, která obsahuje rozsáhle filtrovaná webová a syntetická data, přičemž prioritou je robustnost, bezpečnost a formát chatu.
- Výsledky také odhalují škálovatelnost modelů phi-3-small a phi-3-medium a dokazují jejich lepší schopnosti ve srovnání s modelem phi-3-mini při experimentech s počátečním škálováním parametrů.

### [Reakce](https://news.ycombinator.com/item?id=40127806)

- Technická zpráva Phi-3 se zabývá srovnávacími údaji a praktickým výkonem nového modelu, řeší problémy s nadměrným přizpůsobením a výkonem každodenních úloh.
- Porovnává model s jinými modely, jako jsou Llama 3 a GPT-4, a zdůrazňuje jejich silné a slabé stránky.
- Diskuse se věnují efektivitě benchmarků, obavám o kvalitu dat a využití syntetických dat při trénování modelů a osvětlují konkurenci a inovace v odvětví umělé inteligence, stejně jako skepsi a naději ohledně pokroku v oblasti jazykových modelů od technologických gigantů, jako jsou Apple a Microsoft.

## [Py2wasm zvyšuje výkon Pythonu na WebAssembly](https://wasmer.io/posts/py2wasm-a-python-to-wasm-compiler)

### [Reakce](https://news.ycombinator.com/item?id=40114567)

- Py2wasm je kompilátor jazyka Python do WebAssembly určený ke zvýšení výkonu programů v jazyce Python ve webových prohlížečích.
- Výsledky srovnávacích testů uživatelů jsou rozporuplné, někteří zaznamenali snížení rychlosti ve srovnání s nativním CPythonem.
- Projekt slibuje zlepšení kompatibility s jazyky Python a WebAssembly, hovoří se o integraci s nástroji jako Nuitka, což nabízí zajímavé vyhlídky pro vývoj webových aplikací, zejména v oblasti strojového učení.

## [Představujeme systém Parquet-WASM: Rust pro WebAssembly Parquet I/O](https://github.com/kylebarron/parquet-wasm)

- Vazby WASM Parquet WebAssembly umožňují čtení a zápis souborů Apache Parquet v Apache Arrow prostřednictvím schránek Rust parquet a arrow.
- Knihovna nabízí synchronní a asynchronní rozhraní API, klade důraz na malou velikost svazku a přizpůsobené sestavení pro různé kompresní kodeky.
- Uživatelé mohou získat parquet-wasm od NPM pro sestavení ESM, bundler a Node s funkcemi ladění, optimalizace výkonu, podpory komprese a zmenšené velikosti svazků, přičemž v budoucnu se plánuje přidat predikát pushdown a filtrování sloupců.

### [Reakce](https://news.ycombinator.com/item?id=40115155)

- Parquet-WASM je vazba WebAssembly napsaná v jazyce Rust pro efektivní čtení a zápis dat Parquet.
- Ve srovnání s alternativami, jako je DuckDB-WASM, je lehký, takže je dobrou volbou pro zpracování operací přesahujících hranice WASMJS.
- Uživatelé mohou číst data z adres HTTP URL, ale musí sami zpracovávat podepisování a nástroj v současné době zapisuje data do ArrayBuffer pro odesílání nebo ukládání.
