# 2024-01-21

## [Spotube: Open-source klient Spotify s integrací YouTube](https://github.com/KRTirtho/spotube)

- Spotube je klient Spotify s otevřeným zdrojovým kódem, který lze používat bez předplatného Spotify Premium.
- Využívá datové rozhraní API služby Spotify a nabízí alternativní zdroje zvuku, jako je YouTube, Piped.video nebo JioSaavn.
- Mezi funkce patří žádné reklamy, možnost stahování skladeb, podpora různých platforem, malá velikost, nízká spotřeba dat, anonymní přihlášení, časově synchronizované texty a nativní výkon. Neshromažďuje uživatelská data ani nepodporuje Spotify Shows & Podcasts.

### [Reakce](https://news.ycombinator.com/item?id=39066136)

- Diskuse zahrnuje několik aspektů platforem pro streamování hudby, jako jsou klíče API, omezení podcastů Spotify, obavy ohledně DRM, kvalita zvuku, srovnání Spotify a YouTube, frustrace z náhodného přehrávání, dopady na příjmy umělců, alternativní platformy jako Bandcamp a návrhy na finanční podporu hudebníků.
- Rozhovor zdůrazňuje význam lepšího odměňování umělců a zkoumání alternativních modelů streamování.
- Tato obsáhlá diskuse osvětluje současné problémy a možná řešení v odvětví streamování hudby.

## [Představujeme Bun Shell: Zjednodušení skriptování shellu v JavaScriptu](https://bun.sh/blog/the-bun-shell)

- Spouštění skriptů shellu v jazyce JavaScript může být náročné kvůli rozdílům v syntaxi a příkazech na různých platformách.
- Bun Shell je jazyk a interpret zabudovaný do jazyka Bun, který umožňuje multiplatformní skriptování shellu v jazycích JavaScript a TypeScript.
- Cílem shellu Bun Shell je poskytnout zjednodušené a efektivní řešení pro spouštění shellových skriptů v jazyce JavaScript, které řeší složitost a nekonzistenci stávajících shellů.

### [Reakce](https://news.ycombinator.com/item?id=39071688)

- Uživatelé diskutují o novém shellu Bun Shell a vyjadřují obavy ohledně kompatibility, nahrazení stávajících příkazů a přebujelých coreutils.
- Nástroj je oceňován pro svou jednoduchost, inovativnost a užitečnost při skriptování úloh.
- Rozhovor se týká také dostupnosti různých skořápek a doporučují se alternativní možnosti.
- Vývojář aplikace Bun odpovídá na dotazy a zpětnou vazbu a řeší obavy týkající se názvů importů a stability.
- Diskuse se rozšiřuje o programovací jazyk Zig, uživatelé si vyměňují zkušenosti a diskutují o jeho stabilitě.
- Další témata zahrnují skriptování v jazyce JavaScript, výkonnostní problémy s příkazy shellu v Node.js, označené šablony, Lit Element a knihovnu rimraf.
- Objevují se obavy ohledně bezpečnostních rizik a omezení při používání Bun Shell.
- Diskuse se zabývá také ZSH, šablonovacími jazyky, konfiguračními soubory JSON a YAML a kompatibilitou napříč platformami.
- Vzniká debata o synchronním vs. asynchronním provádění a negativních zkušenostech s poskytovatelem hostingu.
