## [„MicroMac, Macintosh za méně než 5 liber“](https://axio.ms/projects/2024/06/16/MicroMac.html)

- „Projekt s názvem Pico-mac používá mikrokontrolér Raspberry Pi RP2040 k emulaci Macintosh 128K, což mu umožňuje spouštět starý software pro Macintosh.“
- „264 KB RAM a 2 MB flash paměti RP2040 jsou dostatečné pro uložení 128 KB paměti Macu, emulátoru a obraz disku s operačním systémem a softwarem.“
- „Projekt zahrnoval vytvoření emulátoru Mac 128K, vytvoření obvodu pro VGA video výstup a optimalizaci výkonu prostřednictvím přetaktování a vylepšení interpretu, čímž bylo dosaženo přibližně 1,4 MIPS.“

### [Reakce](https://news.ycombinator.com/item?id=40699684)

- „MicroMac, Macintosh za méně než 5 liber, získal značnou pozornost, což zdůrazňuje nízkonákladový, kutilský přístup k retro výpočetní technice.“
- „Projekt zahrnuje použití mikrokontroléru RP2040 k emulaci 68k Macintosh, což nabízí autentičtější zážitek než pouhé spuštění emulátoru na Linuxu.“
- „Diskuse se také dotýká výzev a kreativních řešení při získávání starých technologických komponentů, zdůrazňuje vynalézavost komunity a zájem o retro počítačové projekty.“

## [„NumPy 2.0“](https://numpy.org/devdocs/release/2.0.0-notes.html)

- „NumPy 2.0.0 je první hlavní vydání od roku 2006, vyvinuté během 11 měsíců 212 přispěvateli, které přináší významné nové funkce, vylepšení výkonu a zásadní změny v API pro Python i C.“
- „Klíčové aktualizace zahrnují nový datový typ řetězce s proměnnou délkou, zrychlené třídicí funkce a jasné rozdělení mezi veřejnými a soukromými API, spolu s novými veřejnými C API pro vytváření vlastních datových typů.“
- „Uživatelé by si měli být vědomi zásadních změn a problémů se zpětnou kompatibilitou, přičemž jsou k dispozici migrační příručky, které pomohou s aktualizací kódu.“

### [Reakce](https://news.ycombinator.com/item?id=40699470)

- „Byla vydána verze NumPy 2.0, která přináší významné změny, včetně zastarání některých veřejných API a nové podpory pro práci s řetězci.“
- „Cílem vydání je řešit historické slabiny v operacích s řetězci v rámci NumPy, což by mohlo potenciálně prospět případům použití NLP (zpracování přirozeného jazyka) a LLM (velké jazykové modely).“
- „Kompatibilita s NumPy 1.x je většinou zachována, s výjimkou funkce "byte_bounds", a uživatelům se doporučuje připnout závislosti, aby se předešlo problémům během přechodu.“

- „Efekt Bouba-Kiki, kde děti přirozeně propojují smyslové modality, je zdůrazněn s poznámkou, že tato schopnost často u dospělých slábne.“
