## [Raspberry Pi 5](https://www.raspberrypi.com/products/raspberry-pi-5/)

- Raspberry Pi 5 je nově vydaný upgrade oblíbeného počítače Raspberry Pi, který díky optimalizovanému křemíkovému designu poskytuje 2-3krát vyšší výkon.
- Jeho rozšířené specifikace, které jsou doplněny různým příslušenstvím, zahrnují čtyřjádrový 64bitový procesor, duální výstup HDMI 4K, dvoupásmovou Wi-Fi, porty USB, gigabitový Ethernet, vysílače pro kameru/displej a hodiny reálného času.
- Pro optimální provoz s nejnovějším operačním systémem Raspberry Pi a kvalitním zdrojem napájení doporučuje používat aktivní chlazení pro optimální výkon. Předobjednávky mohou být dostupné prostřednictvím autorizovaných prodejců.

### [Reakce](https://news.ycombinator.com/item?id=37685580)

- Uživatelé vznášejí obavy ohledně ceny, funkcí a dostupnosti a srovnávají jej s konkurenčními počítači malých rozměrů, jako jsou HP Elitedesk 800 G3 Minis a Dell Wyse 5070. Vyzdvihují významné možnosti GPIO a softwarovou podporu Raspberry Pi, přestože mají obavy ohledně jeho výkonu a síly.
- Mezi diskutované očekávané funkce Raspberry Pi 5 patří nové rozhraní PCI Express pro rychlejší periferie, které může zvýšit jeho výkon. Předmětem diskuse je také nutnost a možná omezení aktivního chlazení.

## [Carton - Spouštění libovolného modelu ML z libovolného programovacího jazyka](https://carton.run)
