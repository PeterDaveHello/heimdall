## [Společnost Unity zřejmě potichu odstranila své repozitáře GitHub, které sledují změny ToS.](https://www.gamerbraves.com/unity-silently-deletes-github-repo-that-tracks-terms-of-service-changes-and-updated-its-license/)

### [Reakce](https://news.ycombinator.com/item?id=37516523)

- Společnost Unity smazala svůj repozitář na GitHubu, který sledoval změny jejích podmínek služby (ToS), což vyvolalo obavy o transparentnost a důvěryhodnost.
- Tyto retroaktivní změny smluvních podmínek naštvaly vývojáře her, kteří při své práci vycházeli z předchozí struktury nákladů.

## [Tails je přenosný operační systém, který chrání před dohledem a cenzurou](https://tails.net/)

- Tails je přenosný operační systém určený k ochraně před sledováním a cenzurou, který využívá síť Tor pro zajištění soukromí a odolnosti vůči cenzuře online.
- Tails je navržen tak, aby se dal nainstalovat na USB flash disk, a poskytuje bezpečné počítačové prostředí na libovolném počítači vybavené aplikacemi pro zpracování citlivých dokumentů a zabezpečenou komunikaci.
- Tails, který podpořil Edward Snowden a který financují organizace zabývající se svobodou internetu, je založen na systému Debian GNU/Linux. Je volně dostupný a často jej využívají novináři, aktivisté, osoby, které přežily domácí násilí, a další, kteří potřebují větší soukromí.

### [Reakce](https://news.ycombinator.com/item?id=37509560)

- Diskuse se zaměřuje na Tails, operační systém (OS) zaměřený na ochranu soukromí, který byl vyvinut na ochranu před sledováním a cenzurou a který obsahuje funkce, jako je blokování stop po předchozích relacích a ochrana před malwarem.
- Konverzace také poukazuje na některá omezení operačního systému Tails, včetně rizika úniku IP adresy, a jako možnosti bezpečného procházení jsou zmíněny alternativy jako Whonix, Heads a TENS.
- Mezi další témata dialogu patří historie a růst internetu, spolehlivost open-source softwaru a financování open-source iniciativ.

## [Kde se v úložišti Git nacházejí vaše soubory?](https://jvns.ca/blog/2023/09/14/in-a-git-repository--where-do-your-files-live-/)

- Systém Git ukládá soubory jako samostatné objekty v adresáři .git/objects uvnitř úložiště a používá ukládání s adresou obsahu, kde názvy souborů představují hash jejich obsahu.
- Systém Git neukládá rozdíly pro jednotlivé verze; každá verze je uložena jako kompletní soubor. K vyhledání starších verzí lze použít příkaz 'git log', který najde ID revizí související se změnami v souboru.
- Git komprimuje objekty pomocí formátu zlib. Autor uvádí praktický příklad načtení starší verze příspěvku na blogu pomocí systému Git, doplněný úryvky kódu.

### [Reakce](https://news.ycombinator.com/item?id=37512888)

- Článek poskytuje ucelený přehled systému Git, který zahrnuje jeho jedinečné funkce, metody řešení problémů, vnitřní fungování a využití směrovaného acyklického grafu (DAG) pro sledování změn.
- Kritické připomínky k uživatelskému rozhraní systému Git jsou zkoumány spolu s možnými vylepšeními, přestože argumenty podporují jeho flexibilitu a konzistenci v oblasti systémů pro správu verzí.
- Článek se také zabývá složitostí a použitelností systému Git, porovnává jeho popularitu s jinými možnostmi a nastiňuje potenciál pro budoucí inovace a rozšíření.
