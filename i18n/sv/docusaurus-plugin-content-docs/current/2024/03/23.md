## [Pack: Nästa generations containerformat för högeffektiv filkomprimering](https://pack.ac/note/pack)

- Pack är ett containerformat som fokuserar på att effektivt packa filer med hög komprimering och modern hårdvaruoptimering, med målet att uppnå bästa möjliga förhållande mellan komprimering och tid.
- Den erbjuder funktioner som att utforska och packa upp specifika filer eller mappar, och betonar användarnas integritet genom att inte samla in personuppgifter.
- Pack är öppen källkod, licensierad under Apache 2.0, baserad på SQLite3 och Zstandard, med kryptering, grafiskt gränssnitt och plattformsstöd som planeras för framtiden.

### [Reaktioner](https://news.ycombinator.com/item?id=39793805)

- Diskussionen introducerar det nya containerformatet Pack, som syftar till att ersätta äldre format som tar och zip, med betoning på zstd-komprimering, snabb uppackning av en enda fil och korskompatibilitet.
- Befintliga filformat som tar möter begränsningar som diskuteras för bättre datahantering, med SQLite som föreslås som ett fil- och utbytesformat trots begränsningar och nackdelar som noterats av vissa användare.
- Sammantaget handlar diskussionen om att förbättra komplexiteten, kompatibiliteten och prestandan hos arkivformat mot ett mer effektivt och användarvänligt format som Pack.

## [Oöverträffade 2K jordbävningar upptäckta utanför Kanadas kust](https://www.livescience.com/planet-earth/earthquakes/2000-earthquakes-in-1-day-off-canada-coast-suggest-the-ocean-floor-is-ripping-apart-scientists-say)
