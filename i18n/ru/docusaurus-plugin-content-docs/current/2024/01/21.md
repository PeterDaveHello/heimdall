## [Spotube: Клиент Spotify с открытым исходным кодом и интеграцией с YouTube](https://github.com/KRTirtho/spotube)

- Spotube - это клиент Spotify с открытым исходным кодом, который можно использовать без подписки Spotify Premium.
- Он использует API данных Spotify и предлагает альтернативные источники звука, такие как YouTube, Piped.video или JioSaavn.
