## [Команда Notify успешно перенесла базу данных PostgreSQL всего за 11 секунд простоя](https://gds.blog.gov.uk/2024/01/17/how-we-migrated-our-postgresql-database-with-11-seconds-downtime/)

- Команда GOV.UK Notify успешно перенесла свою базу данных PostgreSQL с минимальным временем простоя с помощью сервиса AWS Database Migration Service (DMS).
- Процесс миграции включал в себя настройку экземпляра DMS, создание целевой базы данных, полную загрузку данных и осуществление текущей репликации.
- В результате миграции время простоя составило всего 11 секунд, что превысило целевой показатель, составлявший менее 5 минут. Команда планирует перенести свои приложения на AWS Elastic Container Service (ECS) в будущем.

### [Реакции](https://news.ycombinator.com/item?id=39048317)

- Обсуждаются такие темы, как развертывание AWS RDS Blue-Green Deployments и ограничения на понижение базы данных для ее миграции.
- Также обсуждаются проблемы с фреймворками для миграции баз данных и проблемы с AWS Data Migration Service (DMS).
- В ходе дискуссии также затрагиваются вопросы облачных сервисов для правительства, безопасности данных и суверенитета, а также плюсы и минусы передачи ИТ-услуг иностранным компаниям, особенно в государственном секторе.
