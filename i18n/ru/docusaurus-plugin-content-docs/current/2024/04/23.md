## [Представляем Parquet-WASM: Rust для паркетного ввода-вывода WebAssembly](https://github.com/kylebarron/parquet-wasm)

- Связка WASM Parquet WebAssembly позволяет читать и записывать файлы Apache Parquet в Apache Arrow с помощью крейтов Rust parquet и arrow.
- Библиотека предлагает синхронные и асинхронные API, уделяя особое внимание небольшому размеру пакета и адаптированным сборкам для различных кодеков сжатия.
- Пользователи могут приобрести parquet-wasm от NPM для сборок ESM, bundler и Node, с функциями отладки, оптимизации производительности, поддержки сжатия и уменьшения размера пакета. В будущем планируется добавить предикаты pushdown и возможности фильтрации столбцов.

### [Реакции](https://news.ycombinator.com/item?id=40115155)

- Parquet-WASM - это привязка WebAssembly, написанная на Rust, для эффективного чтения и записи данных Parquet.
- По сравнению с такими альтернативами, как DuckDB-WASM, он имеет небольшой вес, что делает его хорошим выбором для обработки операций через границу WASMJS.
- Пользователи могут читать данные из HTTP URL, но они должны самостоятельно обрабатывать подпись, и в настоящее время инструмент записывает данные в ArrayBuffer для загрузки или сохранения.
