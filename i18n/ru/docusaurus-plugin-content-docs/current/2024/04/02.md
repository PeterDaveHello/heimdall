## [Rain показывает решение для борьбы с помехами Wi-Fi](https://predr.ag/blog/wifi-only-works-when-its-raining/)

- Автор делится историей о том, как у них дома Wi-Fi работал только во время дождя, и в итоге обнаружил, что помехой для сигнала является соседское дерево.

- Bun 1.1 - это надежный набор инструментов для приложений на JavaScript и TypeScript, в который внесено более 1700 исправлений, повышающих стабильность и совместимость с Node.js.

- Преимущества использования Bun включают более быстрое взаимодействие с TypeScript, быструю установку и улучшенную поддержку модулей ES по сравнению с Node.js и Deno, что заставляет сравнивать их с такими инструментами, как Jest, Node.js и npm.
- В ходе беседы подчеркиваются потенциальные преимущества и проблемы интеграции Bun и Deno в проекты разработки программного обеспечения, поднимаются вопросы совместимости, долговечности проектов, сбора телеметрии, подписания кода и недокументированных API-интерфейсов Windows.

## [Wireproxy: Клиент WireGuard в качестве HTTP/SOCKS5-прокси](https://github.com/pufferffish/wireproxy)

- Wireproxy - это приложение пользовательского пространства, подключающееся к аналогу Wireguard и служащее в качестве SOCKS5/HTTP-прокси или туннеля без прав root.
- Он поддерживает статическую маршрутизацию TCP, UDP, несколько пиров и специальные настройки маршрутизации, что повышает гибкость и контроль над проксированием трафика.
- Пользователи могут легко настроить Wireproxy, используя примерный файл конфигурации, что делает его универсальным инструментом для использования Wireguard для безопасного и эффективного сетевого взаимодействия.

### [Реакции](https://news.ycombinator.com/item?id=39900329)

- Wireproxy - это клиент WireGuard, работающий в качестве HTTP/SOCKS5-прокси, позволяющий пользователям выборочно направлять интернет-трафик через соединения WireGuard.
- Пользователи рекомендуют различные инструменты и настройки для выполнения аналогичных функций, обсуждают различные прокси-клиенты и VPN-решения.
- Получивший высокую оценку за удобный дизайн и эффективность, Wireproxy является одним из наиболее предпочтительных вариантов для направления определенного сетевого трафика через соединения WireGuard.
