# 2024-07-16

## [Запускайте CUDA, без изменений, на графических процессорах AMD](https://docs.scale-lang.com/)

- Система SCALE от Spectral Compute представляет собой набор инструментов для программирования GPGPU (общего назначения на графических процессорах), который позволяет нативную компиляцию приложений CUDA для графических процессоров AMD без изменения исходного кода CUDA или системы сборки.
- Система SCALE выступает в качестве замены компилятора NVIDIA nvcc, обеспечивая бесшовную интеграцию с существующими инструментами сборки и скриптами, и поддерживает различные открытые проекты CUDA, такие как NVIDIA Thrust и Blender Cycles.
- В настоящее время SCALE поддерживает графические процессоры AMD, такие как gfx1030 (Navi 21, RDNA 2.0) и gfx1100 (Navi 31, RDNA 3.0), при этом ведется разработка для дополнительных архитектур графических процессоров.

### [Реакции](https://news.ycombinator.com/item?id=40970560)

- Обсуждение сосредоточено на возможности и последствиях запуска CUDA (Compute Unified Device Architecture) на графических процессорах AMD, подчеркивая технические и юридические проблемы.
- Некоторые утверждают, что поддержка слоев перевода для CUDA на графических процессорах AMD непрактична и предлагают сосредоточиться на таких проектах с открытым исходным кодом, как PyTorch.
- Упоминается, что HIP (Heterogeneous-Compute Interface for Portability) является потенциальным решением с низкими затратами для переноса кода CUDA на графические процессоры AMD, хотя это не совсем решение без усилий.

## [«Для рекламы Firefox теперь по умолчанию собирает данные пользователей»](https://www.heise.de/en/news/For-advertising-Firefox-now-collects-user-data-by-default-9801345.html)

- Последняя версия Firefox теперь по умолчанию собирает данные пользователей, вводя атрибуцию с сохранением конфиденциальности (PPA) для анонимного измерения рекламы, что вызвало споры.
- Критики утверждают, что этот сбор данных по умолчанию подрывает доверие пользователей и ставит под сомнение намерения Mozilla по увеличению доходов, что бросает вызов репутации Firefox в области конфиденциальности.
- ППА использует сервер агрегации для анонимизации данных, но сохраняются опасения по поводу передачи данных с компьютеров пользователей и возможного будущего доступа рекламодателей.

### [Реакции](https://news.ycombinator.com/item?id=40974112)

- Firefox начал собирать данные пользователей по умолчанию для рекламных целей, что можно отключить в настройках в разделе Конфиденциальность и безопасность.
- Пользователи сообщили, что настройка для отключения этой функции не легко находится и не синхронизируется между устройствами.
- Технический директор Mozilla утверждает, что эта функция уравновешивает конфиденциальность и измерение рекламы, но скептицизм и критика со стороны пользователей сохраняются.

## [KUtrace: низконакладная система трассировки ядра Linux](https://github.com/dicksites/KUtrace)

- KUtrace — это инструмент трассировки ядра Linux с низкими накладными расходами, предназначенный для анализа производительности путем отслеживания переходов между ядром и пользователем.
- Он создает детализированные временные шкалы для каждого ядра процессора, которые можно увеличивать до наносекунды, что помогает решать проблемы с производительностью в дата-центрах и системах автономного вождения в реальном времени.
- Инструмент поддерживает процессоры x86 и ARM, является открытым исходным кодом под различными лицензиями и включает файлы из книги «Понимание динамики программного обеспечения» Ричарда Л. Сайтса.

### [Реакции](https://news.ycombinator.com/item?id=40972099)

- KUtrace — это низконагрузочное средство трассировки ядра Linux, предназначенное для понимания узких мест в производительности, особенно в многомашинных системах.
- «В отличие от инструментов eBPF, KUtrace требует патчей ядра и доступа root, что требует активного обслуживания.»
- Инструмент хвалят за его эффективность, с рекомендациями книги «Понимание динамики программного обеспечения» автора Дика Сайтса для получения дополнительных сведений о производительности системы.
