## [Osquery: Улучшение видимости и безопасности конечных точек](https://osquery.io/)

- Osquery - это инструмент, позволяющий пользователям выполнять запросы к своим устройствам, подобно базе данных, с помощью простых команд SQL, обеспечивая видимость конечных точек и соответствие требованиям безопасности путем обнаружения процессов, запущенных без двоичного файла на диске, что может сигнализировать о подозрительном поведении.

- Osquery, виртуальная таблица sqlite3, предоставляет данные ОС для SQL, несмотря на свою техническую неполноценность, и широко используется для таких задач, как листинг сокетов MacOS, а также в качестве основы для платформы EDR компании Sophos.
