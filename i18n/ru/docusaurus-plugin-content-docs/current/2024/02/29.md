## [Развязывание эффективности: Геометрическая алгебра в трехмерной графике](https://enkimute.github.io/LookMaNoMatrices/)

- Проект "Look, Ma, No Matrices!" на GitHub исследует использование геометрической алгебры и евклидовой АПГ в компьютерной графике для создания безматричного 3D-движка, нацеленного на более эффективные вращения, переводы и трансформации.
- Проект подчеркивает важность понимания PGA-двигателей, нормализации и эффективной реализации PGA-продуктов, таких как операторы композиции и сэндвича, для улучшения преобразований, а также подробно описывает преобразование ортогональных матриц в PGA-двигатели и масштабирование в 3D-моделях.
- В этом проекте рассматриваются касательные роторы для отображения нормалей в касательном пространстве и моторный скиннинг для скелетной анимации, демонстрирующие потенциал значительного увеличения занимаемой памяти и эффективности компьютерной графики с помощью PGA.

### [Реакции](https://news.ycombinator.com/item?id=39538670)

- В этом посте рассматривается геометрическая алгебра в 3D-графике, особое внимание уделяется сплайнам/кривым Безье и применению алгебры для решения геометрических задач.
- В нем упоминаются образовательные ресурсы по геометрической алгебре, библиотека ganja.js и методы умножения многочленов.
- Обсуждаются различия между кросс-продуктом и экстерном, использование геометрической алгебры для комплексных чисел, кватернионы, теория представлений Ли в графике, описание электромагнитных полей и ее эффективность в программировании.

## [Pql: Язык конвейерных запросов с открытым исходным кодом, компилируемый в SQL](https://pql.dev/)

- Недавно выпущенная версия pql 0.1.0 - это язык запросов с открытым исходным кодом, написанный на Go, который компилируется в SQL, следуя философии Unix для эффективной разработки запросов.
- Разработанный для преодоления разрыва между собственными языками запросов, такими как Splunk, и базами данных с открытым исходным кодом, поддерживающими SQL, pql предлагает простой интерфейс, основанный на синтаксисе kql от Microsoft.
- pql поддерживает различные табличные операторы и специфические функции, при этом неизвестные функции передаются базовому SQL-движку, а его исходный код доступен на GitHub под лицензией Apache 2.0.
