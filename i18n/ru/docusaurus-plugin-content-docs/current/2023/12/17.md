## [Представляем SSH3: Более быстрый и безопасный Shell с HTTP/3 и QUIC](https://github.com/francoismichel/ssh3)

- SSH3 - это пересмотренная версия протокола SSH, использующая механизмы HTTP для обеспечения более быстрой и безопасной работы с оболочкой.
- Он использует QUIC+TLS1.3 для создания безопасного канала и HTTP Authorization для аутентификации пользователей.
- SSH3 обеспечивает более быстрое установление сеанса, улучшенные методы аутентификации (OAuth2.0, OpenID Connect), устойчивость к атакам сканирования портов, переадресацию портов UDP и возможности современного протокола QUIC.
- Он позволяет скрыть SSH-сервер и поддерживает сертификаты X.509 для аутентификации.
- В настоящее время SSH3 является экспериментальным и может быть установлен путем загрузки двоичных файлов или компиляции кода из исходных текстов.
- Для использования SSH3 необходимо развернуть сервер SSH3, а клиент SSH3 поддерживает различные методы аутентификации.

### [Реакции](https://news.ycombinator.com/item?id=38664729)

- В ветке комментариев затрагивается целый ряд тем, связанных с SSH, таких как совместимость с такими протоколами, как HTTP/3 и QUIC, использование сертификатов для аутентификации и возможность создания новой версии SSH.
- Также обсуждаются альтернативные протоколы, такие как Mosh.
- Обсуждаются различные точки зрения и проблемы, касающиеся безопасности, эффективности, внедрения, сетевой цензуры, скрытия протоколов и управления устройствами.
