## [תחליף SQS מבוסס SQLite שניתן להכניס](https://github.com/poundifdef/SmoothMQ)

- ״נוצר תחליף בקוד פתוח, תואם API, עבור Amazon SQS (Simple Queue Service), שנכתב ב-Go ומופץ כקובץ בינארי יחיד המשתמש ב-SQLite לאחסון.״
- ״הפתרון מציע תכונות מתקדמות כמו חיפוש, תזמון, ניתור והגבלת קצב, והוא תואם ליישומי SQS קיימים על ידי שינוי פשוט של נקודת הקצה.״

### [תגובות](https://news.ycombinator.com/item?id=40837610)

- ״תחליף קוד פתוח, תואם API לאמזון SQS פותח באמצעות Go ו-SQLite, ומציע תכונות כמו חיפוש, תזמון, ניתור והגבלת קצב ללא צורך בשכתוב אפליקציות.״
- ״הפתרון תומך ב-Celery, מספק נראות הודעות טובה יותר מ-SQS, וניתן לחיבור עם מערכות אחסון פוטנציאליות כמו RocksDB או Postgres, המסוגלות ליישם פרוטוקולים מרובים.״
- ״קל לפרוס אותו כקובץ בינארי יחיד של Go, עם תוכניות עתידיות לפונקציונליות מבוזרת ואוטומטית, ועידוד משוב.״
