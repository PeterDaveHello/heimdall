## [Transformers.js – מודלים מאומנים מראש בדפדפן למשימות AI](https://github.com/xenova/transformers.js)

- Transformers.js היא ספריית למידת מכונה המאפשרת למשתמשים להשתמש במודלים מאומנים מראש ישירות בדפדפן עבור NLP, ראייה ממוחשבת, אודיו ומשימות רב-מודאליות ללא שרת.
- משתמשים יכולים להמיר מודלים של PyTorch, TensorFlow או JAX לפורמט ONNX, הנגיש באמצעות NPM או וניל JS, ובכך לשפר את הבנת השפה, עיבוד התמונה והיעילות הכוללת.
- ההתקדמות האחרונה במחקר מונחה בינה מלאכותית מציגה מודלים חדשים של טרנספורמטורים כמו Vision Transformers ו- T5, תוך התמקדות בלמידת העברת טקסט לטקסט, זיהוי אובייקטים, זיהוי תמונה ולמידת ייצוג דיבור, כולל מודל YOLOS החדשני לזיהוי אובייקטים בראייה על ידי אוניברסיטת Huazhong למדע וטכנולוגיה.

### [תגובות](https://news.ycombinator.com/item?id=40001193)

- הוא מציע שיטה חלופית למנועי החיפוש המסורתיים, ומפחית את ההסתמכות על פלטפורמות שעשויות להגביל את הגישה לתוכן מסוים.

## [Vortex 2.0: התקדמות בפיתוח קוד פתוח RISC-V GPGPU](https://vortex.cc.gatech.edu/)

- וורטקס הוא פרויקט קוד פתוח התומך ב-GPGPU באמצעות הרחבות RISC-V ISA, התואמות ל-OpenCL, הפועל כיום על FPGA.
- הפלטפורמה מציעה התאמה אישית ומדרגיות, ומציגה מחסנית תוכנה מלאה בקוד פתוח למחקר ארכיטקטורת GPU.
- שחרורו של Vortex 2.0 והפרסומים האחרונים מציגים התקדמות משמעותית בפיתוח GPU חומרה בקוד פתוח.

### [תגובות](https://news.ycombinator.com/item?id=40003868)

- הדיון מתעמק במורכבות ובפיצול של מחשוב GPU מודרני, ומכסה ממשקי API שונים כמו OpenCL, OpenGL, Vulkan, Metal, DirectX, CUDA, HIP, SYCL ו- WebGPU.
- הוא משווה את היסטוריית הפיתוח של מעבדים ומעבדים גרפיים, תוך שימת דגש על אתגרים והזדמנויות ברגולציה בתעשייה.
- השיחה מדגישה את ההבדלים בממשקי API של GPU, שפות תכנות, כלים ואת היעילות של כלי OpenCL, ומתייחסת לאי ודאות לגבי העתיד של OpenCL בפלטפורמות של אפל.
