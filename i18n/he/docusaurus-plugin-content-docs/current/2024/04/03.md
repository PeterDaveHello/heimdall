## [אמזון עוברת מ-'Just Walk Out' ל-Dash Carts בחנויות מכולת](https://gizmodo.com/amazon-reportedly-ditches-just-walk-out-grocery-stores-1851381116)

- טכנולוגיית "Just Walk Out" נותרה בחנויות Fresh נבחרות בבריטניה ובחנויות הנוחות Amazon Go.

## [Renderlet: מהפכה בגרפיקה עם WebAssembly חוצה פלטפורמות](https://news.ycombinator.com/item?id=39907845)

- המחבר עבד בעבר בחברת אדובי על תשתיות ליישומים כגון פוטושופ ואקרובט, ועסק במורכבויות של בסיסי קוד בפלטפורמות שונות.
- הם עזבו את אדובי כדי להקים את Renderlet, מסגרת גרפיקה ניידת המבוססת על WebAssembly, המשלבת מנוע וקטורי דו-ממדי.
- גרסת אלפא של Renderlet צפויה לקבל משוב, עם תוכניות עתידיות לשפר את המהדר והאינטגרציה במספר פלטפורמות.

### [תגובות](https://news.ycombinator.com/item?id=39907845)

- Renderlet, מסגרת גרפיקה ניידת מבוססת WebAssembly, פותחה על ידי עובד לשעבר של Adobe, המאפשרת יישומים בעלי ביצועים גבוהים בפלטפורמות מגוונות ומפשטת את יצירת המודולים הגרפיים.
- השילוב של מנוע הווקטור הדו-ממדי של ריב ב-API התלת-ממדי מודגם, עם תוכניות לשחרור ציבורי קרוב, מה שסולל את הדרך למשחקי AAA פוטנציאליים מבוססי דפדפן באמצעות WebGPU.
- הדיונים עוסקים גם במגבלות של WebGPU/WASM לגיימינג, התקדמות בפיתוח אפליקציות אינטרנט ראשונות עם גרפיקת תלת-ממד, החלפת אלקטרונים פוטנציאלית באמצעות WebAssembly, Flutter עבור ממשק משתמש חוצה פלטפורמות, והיתרונות והמגבלות של Haxe כמטא-קומפיילר ביצירת יישומים חזותיים חוצי פלטפורמות.

## [תמיכה משופרת ב-Python בעובדי Cloudflare](https://blog.cloudflare.com/python-workers)

- עובדי Cloudflare תומכים כעת ב- Python באמצעות Pyodide ו- WebAssembly, ומבטלים את הצורך בשלבי בנייה נוספים לשימוש בחבילות Python, המאפשרים אינטראקציה חלקה בין Python ו- JavaScript.
- ספריות לקוח פופולריות של Python HTTP כמו aiohttp ו-httpx תואמות, ו-FastAPI יכול לרוץ בעובדים באמצעות שרתי ASGI, במטרה לשפר את הביצועים והתאימות.
- Cloudflare מעודדת משוב מקהילת Python כדי לשפר את התמיכה ב-Pyodide, תוך שימת דגש על אבטחת רשת, מהירות האתר, ומציעה משאבים נוספים ומשרות פתוחות.

### [תגובות](https://news.ycombinator.com/item?id=39905441)

- עובדי Cloudflare תומכים כעת ב-Python at the Edge באמצעות Pyodide שעבר הידור ל-WebAssembly דרך Emscripten, ומאפשר הטמעות של Python בסביבת Edge.
- חסרונות כוללים להיות קשור לגרסת Python ספציפית, קשיים ברזולוציית חבילות ואתגרים אדריכליים הקשורים ל- JS/V8.
- הדיונים עוסקים בהשפעת תמונות V8, אופטימיזציה של התחלות קרות באפליקציות JS, היתרונות של Wasm עבור Python, שירותי Cloudflare כמו Durable Objects, הפעלת מחשבים ניידים של Jupyter ב- WASM, והיעילות של V8 מבודד בשירותי ענן.
