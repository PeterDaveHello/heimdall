## [Servo, równoległy silnik przeglądarki napisany w Rust](https://servo.org/)

- Servo to silnik renderowania stron internetowych napisany w języku programowania Rust.
- Silnik układu Servo, zwany Layout, został przepisany w 2020 roku.
- Niedawne ulepszenia procesu testowania Servo ułatwiły programistom wnoszenie wkładu.

#### [Reakcje branży](http://news.ycombinator.com/item?id=36092852)

- Rust i Servo są ze sobą ściśle powiązane, a Rust jest rozwijany równolegle z Servo i wykorzystuje Servo jako swoją aplikację.
- Istnieją alternatywy dla obecnego stosu sieciowego, takie jak Svelte, Astro i Gemini, które oferują lekkie i energooszczędne rozwiązania.
- Servo było niezbędne do wczesnego rozwoju Rusta i pracuje nad ulepszeniem Layout 2020, aby wspierać flexbox, siatki i niezawodną implementację float.
