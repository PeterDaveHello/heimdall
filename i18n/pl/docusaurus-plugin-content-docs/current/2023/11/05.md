## [Cztery luki typu zero-day wykryte w Microsoft Exchange umożliwiające RCE i kradzież danych](https://www.bleepingcomputer.com/news/microsoft/new-microsoft-exchange-zero-days-allow-rce-data-theft-attacks/)

- W Microsoft Exchange wykryto cztery luki typu zero-day, które można wykorzystać zdalnie w celu wykonania dowolnego kodu lub ujawnienia poufnych danych.
- Microsoft został poinformowany o tych lukach przez Trend Micro's Zero Day Initiative (ZDI), ale zgodnie z jego oceną, nie uznał ich za wystarczająco poważne, aby nakazać natychmiastowe działanie.

Microsoft twierdzi, że podjął środki ochronne i zajmie się tymi kwestiami w przyszłych aktualizacjach i produktach.
