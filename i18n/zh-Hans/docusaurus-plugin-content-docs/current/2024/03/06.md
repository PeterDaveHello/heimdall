## [Kagi 通过整合 Wolfram|Alpha 增强搜索功能](https://blog.kagi.com/kagi-wolfram)

- Kagi 的目标是利用人工智能增强搜索引擎结果，而不是生成结果，重点关注真实内容，并与 Wolfram|Alpha 合作提高能力。
- 与 Wolfram|Alpha 集成后，用户可以快速访问数学、时间和单位换算，同时还可以使用知识图表小部件和即时摘要框。
- 用户可以就整合情况提出反馈意见，并期待即将推出的功能，如增强型计算器小部件、体育比赛比分和可定制的搜索小部件 API。

### [反应](https://news.ycombinator.com/item?id=39606048)

- 讨论内容包括对 Kagi 关闭的担忧、性能问题和改进建议，将其与 Google 和 Wolfram|Alpha 等竞争对手进行比较，以及讨论付费订阅和终身会员制的好处。

## [ZLUDA：利用开源项目在 AMD GPU 上运行 CUDA 应用程序](https://www.cgchannel.com/2024/02/open-source-project-zluda-lets-cuda-apps-run-on-amd-gpus/)

- ZLUDA 是 Andrzej Janik 的一个开源项目，它能让英伟达™（NVIDIA®）GPU 的 CUDA 应用程序在 AMD GPU 上运行，特别是针对版本 3，允许不经修改的应用程序迁移到新硬件。
- ZLUDA 最初是为英特尔 GPU 设计的，现在已被重新用于 AMD GPU，并在 Blender、摄影测量应用程序和 Arnold 渲染器等软件上进行了测试，尽管存在一些限制，如某些应用程序的性能较慢以及对 OptiX 的支持不完整，但它仍能在 AMD GPU 上为 CUDA 应用程序提供接近原生的性能。
- Janik 提到，ZLUDA 的进一步开发需要主要 GPU 制造商的支持，他强调，尽管 ZLUDA 是一种非通用解决方案，对某些 CG 应用程序的成功可能有限，但开发人员仍可以使用它。

### [反应](https://news.ycombinator.com/item?id=39604745)

- 讨论围绕 ZLUDA 展开，这是一个促进 CUDA 应用程序在 AMD GPU 上运行的开源项目，引发了关于在非英伟达 GPU 上使用 CUDA 软件翻译层的合法性、实用性和影响的争论。
- 用户强调了 AMD 用户的重要性以及为不同 GPU 模型定制库的必要性，并阐明了 AMD 在 GPGPU 方面的努力、相对于英伟达 CUDA 的局限性以及为 AMD GPU 创建类似 CUDA 的编译器所面临的困难。
- 谈话的最后，大家一致认为，在科技公司中，高质量的软件对硬件起着至关重要的作用。

## [Cloudflare 推出人工智能防火墙，增强模型安全性](https://blog.cloudflare.com/firewall-for-ai)

- Cloudflare 推出了一款人工智能防火墙，以打击模型滥用，引发了关于审查、准确性和业务影响的讨论，同时还讨论了网络应用防火墙（WAF）的网络安全和技术债务问题。

## [人工智能发现椭圆曲线中的 "杂音](https://www.quantamagazine.org/elliptic-curve-murmurations-found-with-ai-take-flight-20240305/)

- 数学家利用人工智能在椭圆曲线中发现了令人惊讶的模式，称之为 "杂音"（murmurations），对数论和密码学产生了影响。
- 椭圆曲线、L 函数和模数形式之间的联系被确定下来，从而加深了对数学的理解。
- 机器学习算法在预测椭圆曲线特性方面大有可为，有可能引发数论的进步。

### [反应](https://news.ycombinator.com/item?id=39604600)

- 研究人员利用人工智能分析了 LMFDB 数据库中的椭圆曲线数据，重点研究了 Dirichlet 系数和复乘法，从而发现了欧拉系数的规律和见解。
- 讨论深入探讨了机器学习在组合学和数论中的应用，展示了其在启发式和预测方面的潜力。
- 探讨了人工智能在改变科学研究方面的潜力，可能带来新的物理学发现，以及它对科学和人类在技术进步中的推理能力的影响，同时批判了有关人工智能讨论中的自信，并认识到机器学习与人类能力相比的局限性。

## [欧洲法院通过加密裁决维护隐私权](https://www.washingtonpost.com/technology/2024/03/05/encryption-eu-human-rights-privacy-ruling/)
