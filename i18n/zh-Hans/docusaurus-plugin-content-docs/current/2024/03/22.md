# 2024-03-22

## [苹果公司因 iPhone 垄断面临反垄断诉讼](https://www.nytimes.com/2024/03/21/technology/apple-doj-lawsuit-antitrust.html)

- 美国司法部和包括哥伦比亚特区在内的 16 个州对苹果公司提起反垄断诉讼，指控其垄断智能手机市场。
- 诉讼称，苹果公司的做法扼杀了竞争，阻碍了数字钱包替代品的发展，对消费者和开发者造成了影响。
- 苹果公司为自己的行为辩护，称这些行为增强了设备的安全性，促进了消费者利益，但诉讼的解决可能会对苹果公司的业务和整个科技行业产生深远影响。

### [反应](https://news.ycombinator.com/item?id=39778999)

- 讨论的重点是反垄断问题，涉及苹果公司对 iPhone 生态系统、App Store 的控制，以及对第三方应用程序的限制，从而影响消费者的选择和市场竞争。
- 关于苹果公司封闭生态系统的利弊、垄断问题，以及科技平台需要更多互操作性和用户控制权的呼声，都引发了争论。
- 对话深入探讨了替代信息标准、去中心化网络的挑战，以及在应用程序商店的安全性和开放性之间取得平衡，并揭示了苹果公司的技术主导地位和正在进行的反垄断辩论。

## [Difftastic：语法感知结构差分工具](https://difftastic.wilfred.me.uk/)

- Difftastic 是一款 CLI diff 工具，可根据语法而非逐行比较文件，从而帮助人类更有效地准确识别代码差异。
- 它使用树形监视器分析代码，能够识别内部表达式的变化，同时忽略格式变化。
- Difftastic 支持多种编程语言和文件格式，提供真实的行号，可视化包装修改，与 Git 集成，完全开源，遵循 MIT 许可，由 Wilfred Hughes 使用 Emacs 和咖啡开发。

### [反应](https://news.ycombinator.com/item?id=39778412)

- 用户讨论了在各种工具（如 Difftastic、neovim 和 Github 代码搜索）中使用 tree-sitter 进行语法分析的问题，强调了其中的优势和挑战。
- 对话内容包括为树坐标编写语法、增量解析的局限性、差异可视化以及编程中语义换行的概念。
- 讨论延伸到比较 Difftastic、Semantic Merge 和 SemanticDiff 等工具，以了解代码变更、合并以及在 diff 工具中自定义颜色的情况，强调代码分析工具的效率和潜在进步。

## [回忆基于 Nextcloud 的 FOSS 照片平台](https://memories.gallery/)

- Memories 是一个免费的开源选项，类似于 Google Photos，通过 Nextcloud 自行托管，侧重于高性能和各种功能，如时间轴视图、基于人工智能的标签和元数据编辑。

### [反应](https://news.ycombinator.com/item?id=39783223)

- 用户将 Memories、Immich 和 Photo Prism 等其他照片存储解决方案与谷歌照片进行比较，强调谷歌照片的功能、稳定性和隐私性，包括元数据、搜索和面部识别。
- Nextcloud 因其在照片存储方面的多功能性、兼容性、易用性和人工智能功能而备受推崇，引发了用户的讨论。
- 讨论还涉及部署挑战、定价和用户反馈，强调了摄影管理软件开源项目的复杂性和优势。

## [大肠杆菌的非凡智慧：揭开趋化性的神秘面纱](https://jsomers.net/e-coli-chemotaxis/)

- 这段话解释了大肠杆菌的趋化作用，即细胞通过化学信号向吸引物移动，这涉及到控制移动和信号传递的 CheY 和 CheA 等蛋白质。
- 大肠杆菌的鞭毛马达对运动至关重要，它需要多种 CheY-p 蛋白质来改变方向，而细菌个体的行为差异源于蛋白质水平而非遗传。
- 利用荧光显微镜和计算机建模等技术来研究细菌的趋化性，这可能对抗生素的开发产生医学影响，并增强对生物功能的理解。

### [反应](https://news.ycombinator.com/item?id=39777229)

- 文章讨论了大肠杆菌等单细胞生物在趋化过程中的智能和适应能力，以及复杂适应系统和大脑功能。
- 它探讨了传统教育方法在理解生物学方面的局限性，以及创世论和进化论之间的争论。
- 文中重点介绍了研究细菌行为的计算能力的进步，强调了进化的复杂性以及人类在理解自然世界方面的知识极限。

## [黑客利用 Saflok 酒店钥匙卡锁的漏洞](https://www.wired.com/story/saflok-hotel-lock-unsaflok-hack-technique/)

- 黑客发现了 Unsaflok，这种方法能够迅速解锁全球酒店中的 300 万个 Saflok 品牌 RFID 密码锁。
- Saflok 背后的公司 Dormakaba 正在缓慢地更新和更换易损锁具，自 2022 年发现问题以来，仅有 36% 的锁具得到了解决。
- 研究人员建议酒店住客保持警惕，利用 NFC Taginfo 应用程序评估他们的钥匙卡是否仍有风险，因为这些不安全的门锁始终存在危险。

### [反应](https://news.ycombinator.com/item?id=39779291)

- 黑客已经发现了酒店钥匙卡锁的漏洞，原因包括系统过时、成本低廉、标准不安全，以及频繁补发钥匙卡带来的风险，以及公司将成本置于安全之上而不承担责任。
- 当钥匙卡系统失灵时，维修工人使用非常规方法打开酒店房门，引发了对应急钥匙卡和酒店政策的讨论，也引起了人们对公寓楼内 RFID 钥匙和物理锁安全性的担忧。
- 讨论还涉及 Dormakaba Saflok 门锁的漏洞、RFID 技术问题、不同门锁系统在防止未经授权进入方面的有效性，以及关于更清晰的通信和缩写标准的建议。

## [DuckDB：使用 SQL 简化 JSON 查询](https://www.pgrs.net/2024/03/21/duckdb-as-the-new-jq/)

- DuckDB 是一种类似于 SQLite 的数据库工具，能够无缝读取和解析 JSON 数据，是数据分析的重要工具。
- 与 jq 的复杂性相比，DuckDB 的 SQL 语法更简单，因此作者更倾向于使用 DuckDB 来查询 JSON 数据，这表明使用 SQL 进行数据分析非常容易。
- DuckDB 的突出特点是能够直接从 URL 读取 JSON 数据，从而增强了其在各种数据分析任务中的通用性。

### [反应](https://news.ycombinator.com/item?id=39782356)

- DuckDB 是作为 jq 数据处理的替代方案而提出的，强调了 jq 的局限性和可能的改进。
- 建议包括利用现代 OLAP DB 工具、shell 脚本、有意编码、最小化安装以及 shellcheck 等工具来高效处理数据。
- 对话内容包括用于 JSON 解析的 ClickHouse 等工具、用于数据处理的 Benthos 与 filebeat，以及 Nushell、jq 和 yq 的比较，突出了个人偏好和对命令行界面熟悉程度的影响。

## [GoFetch：针对苹果 CPU 的新型侧信道攻击](https://gofetch.fail)

- GoFetch 是一种侧信道攻击，目标是带有依赖于数据内存的预取器的苹果 CPU，攻击者可以从恒时加密实现中获取密钥。
- 该漏洞通过利用 DMP 功能进行缓存定时分析，影响苹果 M1、M2 和 M3 处理器以及英特尔第 13 代 Raptor Lake 微体系结构。
- 用户可以通过应用最新更新、在易受影响的 CPU 上停用 DMP 以及在加密协议中集成输入盲法来保护自己的系统，从而降低 GoFetch 攻击带来的风险。

### [反应](https://news.ycombinator.com/item?id=39779195)

- 对话强调了安全在当代硬件和软件开发中的重要性，特别是在侧信道攻击的脆弱性方面。
- 尽管在平衡安全性和性能优化方面存在一些问题，但还是提出了采用安全内核和加密扩展来管理风险的建议。
- 强调安全措施的优先性，如安全运行不受信任的代码，对于防止网络入侵造成的泄密和经济损失至关重要。

## [植物聚合物可在 7 个月内生物降解，提供环保解决方案](https://today.ucsd.edu/story/biodegradable-microplastics)

- 微塑料，即微小的塑料碎片，需要长达 1000 年的时间才能分解，对环境造成危害。
- 加州大学圣迭戈分校和 Algenesis 公司的科学家们创造出了基于植物的聚合物，可在七个月内完成生物降解，提供了一种可持续的解决方案。
- 这些可生物降解的微塑料不含有害碎片，具有治理环境污染的潜力，目前正在努力使其在生产中得到广泛应用。

### [反应](https://news.ycombinator.com/item?id=39777898)

- 植物基聚合物，尤其是加州大学圣地亚哥分校的藻类基聚合物，可在 7 个月内完成生物降解，而传统塑料可能需要数百至数千年才能分解。
- 围绕生物降解塑料对微塑料污染的影响以及有效降解所需的条件展开了辩论。
- 讨论涉及民主政府、个人决策和企业在打击塑料污染、促进可持续发展、解决塑料对环境造成的后果、计划报废以及寻找替代品解决塑料垃圾堆积问题方面的作用。
