# 2024-03-28

## [使用 Swift 更轻松地开发 GNOME 应用程序](https://www.swift.org/blog/adwaita-swift/)

- 这篇文章探讨了如何利用 Swift 开发 GNOME 应用程序，重点介绍了 Adwaita for Swift 软件包。
- 与 Python 等语言相比，Swift 的优点包括语法更简洁，代码可读性更高。
- Adwaita for Swift 可简化 GNOME 应用程序的开发，促进以数据为中心的 UI 设计，支持跨平台开发，并与 Flathub 集成用于应用程序的发布。

### [反应](https://news.ycombinator.com/item?id=39844936)

- 这篇文章深入探讨了使用 Swift 开发 Gnome 应用程序的障碍，强调为 Gnome 功能提供类似 SwiftUI 的封装，解决并发、数据绑定、跨平台支持和长期项目维护等难题。
- 用户分享了他们在使用 macOS 时遇到的 UI 更新和处理导航分割视图的困难，并阐明了 MVVM 等架构模式以及制定 UI 开发指南的意义。
- 讨论将延伸到微软技术对 Linux 的潜在影响，探索用于图形用户界面开发的各种语言和框架，同时概述以代码为中心的用户界面编程的利弊。

## [NotepadNext：跨平台 Notepad++ 重实现](https://github.com/dail8859/NotepadNext)

## [解密 Apple Pay 之外的数字钱包安全问题](https://birchtree.me/blog/digital-wallets-and-the-only-apple-pay-does-this-mythology/)

- 对话还涉及离线交易、智能卡的使用以及欧盟法规如何影响苹果公司的运营。

## [法学硕士揭开人工智能聊天机器人的知识检索面纱](https://news.mit.edu/2024/large-language-models-use-surprisingly-simple-mechanism-retrieve-stored-knowledge-0325)

- 大型语言模型，如人工智能聊天机器人背后的模型，利用基本的线性函数来访问各种主题的存储数据，使研究人员能够调查模型并纠正不准确的地方。
- 通过识别这些功能，研究人员可以纠正模型中的错误信息，加强对知识存储的理解，并有可能提高人工智能聊天机器人的准确性和可靠性。
- 来自麻省理工学院、东北大学、哈佛大学和以色列理工学院的一组科学家开展了这项研究，并将在学习表征国际会议上展示。

### [反应](https://news.ycombinator.com/item?id=39852118)

- 与会者深入探讨人工智能技术中大型语言模型（LLM）和转换器的挑战、进步和局限性，重点关注知识检索机制、计算能力和实际实施成本。
- 关于变压器是否已达到顶峰或是否还有未开发的发展潜力，以及对 LLM 压缩的有损性和模型全面掌握概念的能力的担忧，都存在争议。
- 讨论内容包括语言模型的复杂性、线性函数在人工智能中的作用、训练数据的重要性、优化函数，以及语言之间的知识转移和微分几何中的 "沉浸"。

## [ST-DOS：兼容 MS-DOS 的高级内核](http://sininenankka.dy.fi/~sami/fdshell/doskernel.php)

- ST-DOS 是使用 Open Watcom 编译器编译的内核，具有支持大磁盘、动态缓存和兼容 MS-DOS 等功能。
- 引导加载程序会在 FAT 文件系统中查找 KERNEL.BIN，并按照特定要求加载到内存中。
- 内核更新日志可在线获取，内核源代码也可在线访问。

### [反应](https://news.ycombinator.com/item?id=39847728)

- "ST-DOS "是 Sami Tikkanen 开发的个人 MS-DOS 克隆版，结合了 MS-DOS 的概念和独特元素。
- 讨论内容涉及 Atari ST、DR-DOS，以及对使用 CRT 显示器和拨号美学的老式计算机的回忆。
- 它涉及 MS-DOS 源代码和 DOS 操作系统的演变。

## [了解长期记忆形成过程中的 DNA 修复](https://www.nature.com/articles/d41586-024-00930-y)

- 发表在《自然》杂志上的一项关于小鼠的研究揭示，长期记忆是通过帮助 DNA 损伤修复的炎症反应建立起来的。
- 研究表明，在记忆形成过程中，强烈的电活动会导致脑细胞的DNA断裂，从而引发免疫反应进行修复，这可能会影响阿尔茨海默氏症等神经退行性疾病。
- 这项研究强调了理解细胞内记忆产生和维持机制的重要性。

### [反应](https://news.ycombinator.com/item?id=39849126)

- 记忆存储在大脑的不同部位，包括神经元的表观遗传变化，突触附近的 DNA 被修改，以加强神经连接。
- 重点是 DNA 在创造记忆中的意义，探索自然选择之外的机制，以及研究物质和精神状态如何影响记忆。
- 讨论强调了生物发育的复杂性，暗示了这一过程中潜在的未被发现的机制和复杂性。

## [AI21 实验室推出 Jamba：基于 Mamba 架构的尖端人工智能模型](https://www.maginative.com/article/ai21-labs-unveils-jamba-the-first-production-grade-mamba-based-ai-model/)

- AI21 实验室推出了首个基于 Mamba 架构的人工智能模型 Jamba，用于生产。
- Jamba 融合了 Mamba 的结构化状态空间模型和 Transformer 架构，提高了性能和效率。
- 该模型包括用于扩展上下文窗口和提高吞吐量的 MoE 层，可显示令人印象深刻的基准结果，并可根据 Apache 2.0 许可进行研究，计划不久将推出更适合商业用途的版本。

### [反应](https://news.ycombinator.com/item?id=39853958)

- Jamba 是一种生产级人工智能模型，源自 AI21 开发的 Mamba，融合了变压器层和 Mamba 层，以提高效率和性能。
- 该模型拥有宽泛的上下文窗口，并采用了专家组合，在推理过程中使用了约 120 亿个参数，但一些用户在使用特定 GPU 在 Linux 上运行该模型时遇到了难题。
- 讨论强调了变换器和状态空间模型层之间的权衡，以及广泛的上下文窗口的潜力和限制。Jamba 采用 Apache 2.0 许可。

## [Endlessh-go：用于捕获机器人和可视化指标的 Golang SSH Tarpit](https://github.com/shizunge/endlessh-go)

- Endlessh-go 是 Endlessh 的 Golang 版本，提供 Prometheus 指标，并将其显示在 Grafana 面板上，同时捕获 SSH 攻击者并提供地理位置数据。
- 它可以通过从源代码构建或利用 Docker 镜像进行设置，并允许通过不同的 CLI 参数进行自定义。
- 导出的指标涵盖 Endlessh 上的客户端连接、数据传输和使用时间，而 Grafana 面板需要 8.2 版本，可使用特定 ID 集成，并可通过 GitHub 获得咨询或问题支持。

### [反应](https://news.ycombinator.com/item?id=39848268)

- 争论围绕着采用非标准 SSH 端口、防火墙配置和额外的安全措施来阻止机器人和扫描程序。
- 策略包括利用 Endlessh 等工具、在 iptables 中实施防火墙规则以及验证码和端口隐藏等策略来阻止自动攻击。
- 用户讨论了他们在 Golang 中重新实施 Endlessh 等工具以提高效率和效益时遇到的问题。
