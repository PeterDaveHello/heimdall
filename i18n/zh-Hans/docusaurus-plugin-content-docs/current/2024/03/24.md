## [在圣安德烈斯-特特皮尔科发现新的阿兹特克纂本](https://tlacuilolli.com/2024/03/21/new-aztec-codices-the-codices-of-san-andres-tetepilco/)

- 国家人类学和历史研究所的专家在墨西哥城圣安德烈斯-特特皮尔科发现了三部新的阿兹特克手抄本，揭示了库尔瓦坎和特诺奇蒂特兰政体直至 1603 年的历史细节。
- 这一发现展示了新的图标和阿兹特克文字，澄清了以前模糊不清的字形和交流方式。
- 墨西哥政府目前将这些手抄本存放在国家人类学和历史研究所，激发了人们对未来出版这些手抄本的热情。

### [反应](https://news.ycombinator.com/item?id=39803508)

- 在墨西哥圣安德烈斯-特特皮尔科发现了新的阿兹特克古抄本，是墨西哥政府从当地一个家族手中获得的，据信其中包含的以前被抹去的阿兹特克文字在多光谱成像下清晰可见。
- 这一发现扩充了现有的阿兹台克手抄本收藏，提供了有关阿兹台克历史和文化的全新视角，包括有关圣安德烈斯-特特皮尔科的建立和特诺奇蒂特兰政体的细节。
- 研究人员计划在未来几年对这些手抄本进行进一步的深入分析，揭示阿兹特克人征服前和征服后的历史事件。

## [从 1990 年到 2019 年，早发性癌症发病率激增 80](https://bmjoncology.bmj.com/content/2/1/e000049#DC1)

- 文章深入探讨了 1990 年至 2019 年早发性癌症的全球趋势，展示了不断上升的负担以及饮食和遗传等关键风险因素。
- 强调健康生活和早期筛查计划对有效遏制这些趋势的重要意义。
- 讨论生活方式、空气质量和基因变异如何影响乳腺癌和结肠直肠癌，以及对糖尿病、胃癌和幽门螺旋杆菌感染流行病学的见解。

### [反应](https://news.ycombinator.com/item?id=39803845)

- 讨论强调了 50 岁以下人群癌症发病率的激增，将其归咎于污染和过度使用抗生素等因素，并强调了医疗部门的谨慎态度。
- 它探讨了微塑料对人类健康的影响、发展中国家肥胖症和癌症发病率的上升以及西方化饮食的影响。
- 对话还深入探讨了现代医疗保健在预防癌症方面的功效、环境因素在早发性癌症中的作用，以及雌激素、红肉摄入量和癌症易感性之间的联系。

## [Lapdev：简化远程开发环境管理](https://github.com/lapce/lapdev)

- Lapdev 是一款自托管应用程序，使用 Devcontainer 开放规范，便于管理个人服务器或云上的远程开发环境。
- 它将开发环境简化为代码，确保可扩展性和简单的复制，强调最低的安装和维护要求。
- Lapdev 的路线图包括潜在的增强功能，如支持各种工作区类型和操作系统，其网站上有详细的安装说明和贡献指南。

### [反应](https://news.ycombinator.com/item?id=39801399)

- 与 docker-compose 和 JetBrains 的远程 SSH dev 相比，Github 的重点是新的开源远程开发环境管理软件 Lapdev。
- 用户们讨论了远程开发的益处和疑虑，并探索了 devcontainers 和 Visual Studio Code 等远程开发选项，强调要改善体验和自动设置。
- 对话中强调了手动配置 Docker 设置所面临的挑战、利用配置文件进行容器化，以及为开发人员简化部署流程的意义。

## [老式旋转电话通过互动体验让现代艺术博物馆展览焕发生机](https://github.com/mnutt/rotary)

- 旋转电话项目通过使用与 Asterisk 和 Grandstream 设备连接的复古旋转电话来重现纽约现代艺术博物馆的展览，从而实现互动体验。
- 它包括在服务器上设置 Asterisk、连接旋转电话、播放音频文件和配置 Twilio 电话。
- 该项目提供了检查 MTA 列车状态、接受输入和分享笑话等功能，在给作者的儿子带来欢乐的同时，也带来了调试 SIP 连接等挑战。

### [反应](https://news.ycombinator.com/item?id=39801991)

- 旋转电话项目用 Nvidia Jetson 等现代技术更新了 20 世纪 70 年代的旋转电话，提供互动功能，如回答问题、阅读睡前故事和提供治疗。
- 怀旧情怀与尖端人工智能技术的完美融合，造就了与众不同、引人入胜的用户体验。
- 该项目引起了许多用户的浓厚兴趣，他们希望复制或改进该项目，这表明该项目很受欢迎，并具有进一步定制的潜力。

## [英特尔 8088：拆解指令预取电路](https://www.righto.com/2024/03/8088-prefetch-circuitry.html)

- Ken Shirriff 在博客中对英特尔 8088 微处理器进行了深入分析，尤其关注其预取电路和性能影响。
- 该博客对 8086 和 8088 进行了比较，阐述了 IBM PC 选用 8088 的决策过程，并解释了总线接口单元和执行单元的作用。
- 主题包括预取队列同步、指令解码器、触发器和控制线等处理器设计元素，以及高速缓存在内存操作中的重要性。自 2008 年以来，该博客还涉及电子学、编程和太空探索。

### [反应](https://news.ycombinator.com/item?id=39801963)

- 文章涉及的主题从英特尔 8088 处理器的指令预取电路到 IBM PC 基本内存配置的历史发展。
- 它探讨了定时测试在电子产品中的重要性、在互联网普及之前技术故障诊断所面临的挑战，以及 8086/8088 等处理器的设计选择和成本影响。
- 讨论还深入探讨了 CPUID 指令的历史、奔腾等处理器的高速缓冲存储器配置，以及 CPUID 推出前用于识别处理器类型的技术。
