# 2024-02-11

## [亚马逊因员工公开谈论同事之死而非法解雇员工](https://jacobin.com/2023/11/i-was-illegally-fired-by-amazon-for-speaking-out-about-a-coworkers-death/)

- 波兰一家法院裁定，亚马逊非法解雇了一名就同事死亡事件发表言论的工人，并强调了非人道待遇、侵犯工会权利和危险工作条件等问题。
- 法院认定亚马逊侵犯了工会的权利，但该公司仍可在三周内对裁决提出上诉。目前仍无法确定这一裁决是否会导致亚马逊的政策发生重大变化。
- 由于雇用合同的不同，工人在组织起来方面面临着挑战，不过也出现了一些积极的事态发展，例如对工会官员的法律保护以及新政府可能带来的变化。

### [反应](https://news.ycombinator.com/item?id=39326559)

- 亚马逊在波兰的一名员工声称自己因直言同事之死而被非法解雇，这引发了人们对美国员工可能遭到报复的担忧。
- 对话内容包括各州道路状况的差异、佛罗里达州和加利福尼亚州税收制度的不同、员工在工作场所庆祝活动和人力资源部门的经历以及雇主和员工之间的权力动态。
- 报告还深入讨论了工会工作、亚马逊的盈利能力以及雇主确保工作环境安全的义务。

## [OpenTTD：终极开源运输大亨游戏](https://www.openttd.org/about)

- OpenTTD 是一款开源模拟游戏，与经典游戏 "豪华运输大亨 "非常相似，同时还加入了新的功能。
- 它提供更大的地图、可容纳多达 255 名玩家的多人游戏模式、先进的寻路算法和丰富的自定义选项。

## [Svelte：一个神奇的、以用户为中心的网络框架，用于构建很酷的东西](https://github.com/sveltejs/svelte/discussions/10085)

- Svelte 是一个重视网络的网络框架，专注于优化积极的体验，而不是追逐流行度。
- Svelte 优先将 HTML 作为描述用户界面的主要语言，并鼓励网络开发的进步和创新。
- 该框架强调定性设计，旨在让 Svelte 感觉神奇，让用户无需丰富的技术知识就能创建很酷的东西。它由社区驱动，重视共识和反馈，旨在成为适用于任何类型应用程序的最佳框架。

### [反应](https://news.ycombinator.com/item?id=39327113)

- 目前正在讨论 Svelte 框架，用户对其优缺点意见不一。
- 一些用户喜欢它，因为它与 HTML 相似，对用户友好；而另一些用户则批评它偏离了 HTML，增加了复杂性。
- 讨论内容包括使用 HTML 与其他语言进行用户界面描述的利弊、HTML 在动态界面方面的局限性以及了解底层技术的重要性。还将与 React、NextJS 和 Vue 等其他框架进行比较，并讨论语法、编译器指令、性能优化和用户体验等问题。

## [少年涉嫌经营针对学校和教堂的特警服务，使数千人处于危险之中](https://www.independent.co.uk/news/world/americas/crime/swatting-nikki-haley-trump-fbi-stalkers-b2494097.html)

- 来自加利福尼亚州的少年艾伦-温斯顿-费里恩（Alan Winston Filion）涉嫌通过加密信息应用程序 Telegram 经营 "扫荡 "服务，提供虚假炸弹威胁和大规模枪击服务，并收取费用。
- Filion 将学校、教堂甚至最高法院作为这些虚假电话的目标，给成千上万的人带来了严重威胁。
- 作家帕特里克-汤姆林森（Patrick S. Tomlinson）曾在多起拍击事件中受害，他主张制定更严格的法律，并将拍击者视为国内恐怖分子。

- 人们对受害者和警官面临的危险表示关切。
- 建议的解决方案包括对拍击者进行更严格的处罚，以及在派出特警队之前改进验证方法。

## [纽约市租金稳定的一居室：高额中介费威胁可负担性](https://gothamist.com/news/a-rent-stabilized-1-bedroom-apartment-for-1100-in-nyc-the-brokers-fee-is-15k)

- 纽约皇后区的一家中介公司挂出了一套低于市场价的一居室公寓，月租金为 1100 美元，但要收取 15000 美元的中介费，由于需要预付现金，租户很难租到。
- 纽约市房地产市场过高的中介费给想要搬迁的租户造成了困难。
- 市议会提出了一项新法案，将支付中介费的责任从租户转移到雇佣中介的一方，但批评者认为这可能导致租金上涨。

### [反应](https://news.ycombinator.com/item?id=39326675)

- 讨论涉及各种与住房有关的话题，如租金价格、房产税、住房可负担性以及信用记录在租房中的作用。
- 其中提到了不同国家和城市的具体问题，如德国的住房短缺问题和美国的中介费问题。
- 对话强调了围绕住房问题的挑战和不同观点，并强调了公平和负担得起的住房的重要性。

## [成为地下城主：一家初创公司创新的支持工程师面试流程](https://www.propelauth.com/post/interview-dungeon-master)

- 一家初创公司为支持工程师开发了一套独特的面试流程，其中包括一个类似于日常工作任务的模拟环境。
- 应聘者会看到一个虚假的产品、文档和一连串的电子邮件/问题，他们的任务是像在现实生活中一样解决这些问题。
- 该流程包括每张票据的附加资产，以模拟在没有来回沟通的情况下解决问题，类似于《龙与地下城》中的地下城主。
- 模拟有助于评估候选人解决问题的技能，并为了解他们的能力提供宝贵的信息。
- 不过，在为候选人设定界限和确保面试官的一致性方面也存在挑战。
- 总体而言，这一面试流程得到了积极的反馈，为应聘者提供了更具吸引力的体验。

### [反应](https://news.ycombinator.com/item?id=39328007)

- 软件工程师的面试流程正在发生变化，重点是算法问题，而不是实际经验。
- 有经验的候选人在准备这类问题时可能会很吃力，从而导致对公平性的担忧。
- 其他讨论话题还包括大型科技公司的影响、leetcode 式问题的使用、评估实用技能的局限性以及经济体系对获取先进工具的影响。
- 对于不同访谈方法（如带回家作业和标准化访谈）的有效性，以及生成数据和让多人参与审查的重要性，都存在争议。
- 讨论了软件开发人员的有效招聘策略，包括确定工作要求、提出相关问题、评估好奇心和热情，以及提供有竞争力的薪酬。

## [.GOV顶级域名从Verisign过渡到Cloudflare：采用独特的 DNSSEC 签名方法](https://indico.dns-oarc.net/event/48/contributions/1038/)

- 演讲讨论了 .GOV 顶级域从 Verisign 到 Cloudflare 的过渡及其 DNSSEC 签名方法。
- Verisign 使用 RSA 和 NSEC3 进行离线签名，而 Cloudflare 则使用 ECDSA 和 NSEC 进行在线签名。
- 广泛的测试发现，同时使用 NSEC 和 NSEC3 没有任何问题，因此在过渡时期使用了这两种设备。

### [反应](https://news.ycombinator.com/item?id=39326092)

- Verisign 已将 .GOV 顶级域名的运营转移到 Cloudflare，引发了有关 DNS 复杂性和 .nz 域名宕机的讨论。
- 人们对政府与 Cloudflare 的合作、美国国家安全局的潜在监控以及 DNS 服务器管理和验证码的混乱产生了猜测。
- 此举使人们对 Cloudflare 的价值主张及其在管理关键互联网基础设施方面的作用产生了疑问。
