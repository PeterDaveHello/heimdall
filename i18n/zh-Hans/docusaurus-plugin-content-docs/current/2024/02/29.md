## [释放效率：三维图形中的几何代数](https://enkimute.github.io/LookMaNoMatrices/)

- GitHub 上的 "看，妈，没有矩阵！"项目探索在计算机图形中使用几何代数和欧几里得 PGA 来创建无矩阵 3D 引擎，旨在实现更高效的旋转、平移和变换。
- 该项目强调了解 PGA 电机、归一化和有效实施 PGA 产品（如用于改进变换的合成和三明治运算符）的重要性，同时还详细介绍了正交矩阵到 PGA 电机的转换以及三维模型中的缩放。
- 本项目重点介绍了用于切线空间法线映射的切线转子和用于骨骼动画的电机蒙皮，展示了通过 PGA 大幅改善计算机图形的内存占用和效率的潜力。

### [反应](https://news.ycombinator.com/item?id=39538670)

- 这篇文章探讨了 3D 图形中的几何代数，强调了花键/贝塞尔曲线和应用代数解决几何问题。
- 它提到了有关几何代数、ganja.js 库和多项式乘法技术的教育资源。
- 讨论内容包括交叉积和外积的区别、几何代数在复数中的应用、四元数、图形中的Lie表示理论、电磁场描述及其在编程中的效率。

## [调查六行星系统中的外星技术](https://www.space.com/alien-technosignatures-exoplanet-mathematically-perfect-orbits)

- 天文学家探测到一个恒星系统包含六颗海王星以下的行星，它们的轨道完全一致，这激发了人们探索可能的外星技术的兴趣。
- 尽管没有在 HD 11067 系统中发现先进的地外生命，但研究人员仍迫切希望进行进一步调查，因为它的边缘位置增加了识别来自拥有先进技术的文明的信号的机会。
- 科学家们坚持不懈地探索太阳系外的生命，以解决存在其他文明这一古老的问题。

### [反应](https://news.ycombinator.com/item?id=39538425)

- 对话深入探讨了数学上完美的恒星系统可能被外星人操纵的概念，涉及轨道共振、行星排列和能量操纵。
- 各种理论讨论了先进文明、人工智能和外星生命，并引用科幻小说作品来支持各种猜测。
- 讨论探讨了工程恒星系统的可行性、探测外星生命信号以及宇宙中智能设计的哲学含义。

## [HDMI 论坛拒绝为 AMD 的开源 HDMI 2.1 驱动程序提供支持](https://www.phoronix.com/news/HDMI-2.1-OSS-Rejected)

## [探索 SymPy：符号数学 Python 库](https://www.sympy.org/en/index.html)

- SymPy 是一个免费的 Python 符号数学库，轻量级且用户友好，适用于从张量代数到量子场论的各种任务。
- 它可以嵌入到不同的应用程序中，并通过自定义函数进行扩展，如 Cadabra、ChemPy、devito 和 EinsteinPy 等项目。
- SymPy 被集成到各种工具中，用于线性电路分析、数学优化和多体动力学等任务，也是 SageMath 和 Spyder 等系统的特色功能，用于数学计算和分析。

## [Pql：编译为 SQL 的开源管道式查询语言](https://pql.dev/)

- 新发布的 pql 0.1.0 版是一种开源的流水线查询语言，由 Go 语言编写，可编译为 SQL，遵循 Unix 的高效查询开发理念。
- pql 的开发是为了缩小 Splunk 等专有查询语言与支持 SQL 的开源数据库之间的差距，它提供了一个基于微软 kql 语法的简单界面。
- pql 支持各种表格运算符和特定函数，未知函数会转发给底层 SQL 引擎，其源代码可在 GitHub 上以 Apache 2.0 许可的方式访问。

## [儿童游戏：拥抱风险和刺激](https://www.afterbabel.com/p/why-children-need-risk-fear-and-excitement)

- 玛丽安娜-布鲁索尼强调了儿童游戏中风险和刺激的重要性，主张采取必要而非过度的安全措施来促进儿童的最佳发展。
- 她批评了户外游戏的减少和屏幕时间的增加，提倡攀爬和探索等对儿童身体和认知成长至关重要的活动。
- 布鲁索尼建议通过鼓励独立游戏、促进户外游戏、创建吸引人的游戏区、游说增加学校课间休息等方式来对抗密集型养育方式，同时强调过多的屏幕时间对儿童健康的不利影响。

### [反应](https://news.ycombinator.com/item?id=39544286)

- 辩论探讨了让儿童参与冒险游戏的利弊，强调了各代人对安全游戏定义的不同看法。
- 讨论内容包括骑马等活动参与度的降低、为人父母者规避风险意识的增强以及儿童发展过程中安全与冒险之间的平衡。
- 主题还包括有组织游戏和无组织游戏的影响、美国和欧洲在儿童监管方面的差异、不断演变的儿童规范以及父母对儿童行为的影响。
