# 2024-02-04

## [美国政府后门企图简史](https://www.atlasobscura.com/articles/a-brief-history-of-the-nsa-attempting-to-insert-backdoors-into-encrypted-data)

- 文章概述了美国政府试图在加密数据中引入后门的历史。
- 它强调了政府官员、网络安全专家和科技行业之间就消费者数据保护和政府获取信息之间的平衡问题正在进行的辩论。
- 文章提到了以前尝试插入后门的例子，包括英尼码机器和 Dual_EC_DRBG 算法。
- 荷兰和法国等一些国家的政府拒绝采用后门，而专家们则主张采用真正安全的加密技术。
- 文章最后指出，尽管苹果公司诉联邦调查局一案有了结果，但政府可能会继续努力破坏密码学。

### [反应](https://news.ycombinator.com/item?id=39244254)

- 讨论主题涉及加密、政府后门、间谍活动、硬件漏洞和隐私问题等多个方面。
- 它探讨了试图在加密数据中添加后门的历史，以及利用 ITAR 法规限制加密软件出口的情况。
- 该主题还讨论了有关中央情报局对 Crypto AG 所有权的披露、英特尔 ME 和 AMD 芯片的安全漏洞、关于多加密层对比特币安全性影响的辩论，以及加密后门和政府访问加密数据的风险。

## [苹果公司推出可编程配置语言 Pkl](https://pkl-lang.org/index.html)

- Pkl 是一种编程语言，使开发人员能够定义静态配置格式，并生成 JSON、YAML 和属性列表等不同格式的输出。
- 开发人员可以使用 Pkl 在运行时进行集成应用配置，并以 Java、Kotlin、Swift 和 Go 等语言生成代码。
- Pkl 提供集成开发环境集成、代码编写工具以及类型和验证系统，可在部署前防止出现配置错误，是开发人员安全、可扩展的配置解决方案。

### [反应](https://news.ycombinator.com/item?id=39239265)

- 苹果公司推出了旨在生成 JSON 和 YAML 的配置即代码语言 Pkl。
- 考虑到 Java 也能完成类似的任务，该语言的发布引起了用户的怀疑，他们质疑是否有必要开发一种新语言。
- 不过，有人强调，Pkl 可能对处理大量文件的组织有益。

## [探索火箭在高重力行星上的局限性](https://space.stackexchange.com/questions/14383/how-much-bigger-could-earth-be-before-rockets-wouldnt-work)

- 作者考虑了火箭尺寸、质量和发动机推重比等因素，研究了高重力环境下火箭和太空旅行的限制因素。
- 宜居世界的引力强度有一个极限，超过这个极限，使用传统化学火箭达到必要的逃逸速度就变得更加困难。
- 这段话还强调了替代发射系统的潜力，并强调了在考虑空间探索方案时资金和材料的重要性。

### [反应](https://news.ycombinator.com/item?id=39243303)

- 以太空为重点的论坛讨论涉及与太空旅行有关的各种话题，包括有效发射火箭的地球尺寸限制以及与火箭推进剂和轨道火箭有关的挑战。
- 它探讨了技术进步的重要性、无法进行太空采矿或绘制详细航空地图的文明所面临的限制，以及低地球轨道太空旅行的潜在益处。
- 对话还深入探讨了量子粒子运动、在高重力行星上建造高大建筑、从高点发射火箭、大气成分和磁场、宇宙中智慧生命的稀有性、星际旅行的挑战以及核推进和替代太空旅行方法等话题。

## [Quora 的衰落：人工智能生成的内容和缺乏监管困扰着用户](https://slate.com/technology/2024/02/quora-what-happened-ai-decline.html)

- 由于存在低质量、重复性和人工智能生成的内容，Quora 的用户正在减少
- 用户对平台缺乏管理和功能表示不满
- 此外，Quora 还面临着一些问题，如网站上充斥着大量不相关的文章，浏览量优先于质量，以及应对巨魔和账户冒名顶替者等。

### [反应](https://news.ycombinator.com/item?id=39241448)

- 用户正在讨论 Quora、StackOverflow 和 Reddit 等平台的衰落和挑战，以及维基百科等网站的缺陷。
- 令人担忧的问题包括缺乏对贡献者的关注、收入优先于用户满意度、内容有偏见、版主咄咄逼人以及负面的用户体验。
- 对话还涉及货币化战略、用户体验、隐私问题以及人工智能和机器学习对这些平台的影响等话题。总体而言，人们对这些网站的方向和管理感到不满。

## [发现M4恒星周围可能适宜居住的系外行星TOI-715 b](https://academic.oup.com/mnras/article/527/1/35/7172075)

- 一项研究发现了一颗可能适合居住的系外行星，名为TOI-715 b，它围绕一颗M4恒星运行，研究人员分析了它的特性及其主恒星的特性。
- 这项研究考察了该星系中第二颗行星的存在，并讨论了研究低质量恒星周围小行星的重要性。
- 利用不同的仪器和天文台进行了后续观测和数据分析，研究建议利用未来的仪器（如 JWST）进一步研究系外行星及其大气层。

### [反应](https://news.ycombinator.com/item?id=39241113)

- 讨论主题围绕着TOI-715 b的发现展开，这是一颗位于恒星宜居带的潜在宜居行星。
- 学员们探讨较小、较冷恒星的优缺点，并讨论行星的引力和特征。
- 寻找类地行星、超光速旅行以及星际探索对我们了解地球的影响也是讨论的主题。

## [苹果 Vision Pro VR 头显：雄心勃勃的设计，但也有不足之处](https://www.ifixit.com/News/90137/vision-pro-teardown-why-those-fake-eyes-look-so-weird)

- 苹果公司即将推出的 VR 头显 Vision Pro 凭借其独特的气泡玻璃正面和光栅屏幕设计脱颖而出，但一些科技记者对其功能性提出了批评。
- 耳机重量超过 1 千克，配有可更换的外置电池组，符合欧盟电池法规，采用模块化设计，便于维修。
- 不过，这款耳机在设计上存在一些缺陷，包括显示屏分辨率较低，以及因出汗导致密封条上的化妆品结块等问题。它采用 M2 Mac 芯片和 R1 芯片，用于摄像头和传感器输入。苹果公司在将其推向市场之前要解决这些缺陷，面临着挑战。

### [反应](https://news.ycombinator.com/item?id=39246664)

- 文章从多个方面探讨了苹果公司的 Vision Pro 设备和虚拟现实头盔，包括假眼的出现、将处理元件和风扇移至外部设备的可能性，以及在 VR 中使用无线流媒体的可能性。
- 文章还讨论了有关专有连接器和电池插头、USB-C 电缆的使用以及 Vision Pro 的局限性和问题。
- 此外，文章还涉及噪音消除技术、用于光纤的ulexite岩石、用于VR的透明有机发光二极管以及不同VR头显之间的比较等主题。文章还提出了对隐私和技术导致的注意力不集中的担忧，并就这些主题提供了全面的观点和意见。

## [优化 Docker 映像大小：从 ugit 工具中汲取的经验教训](https://bhupesh.me/publishing-my-first-ever-dockerfile-optimization-ugit/)

## [微软寻求 Rust 开发人员重写 C# 核心代码](https://www.theregister.com/2024/01/31/microsoft_seeks_rust_developers/)

- 作为采用 Rust 编程语言的一部分，微软正在寻找 Rust 开发人员重写核心 C# 代码。
- 该公司正在组建一个团队，以摆脱 C#，用 Rust 重新实现现有的基于 C# 的服务。
- 此举表明了微软对内存安全编程的执着追求，以及对未来选择的开放态度。

### [反应](https://news.ycombinator.com/item?id=39240205)

- 微软正在积极寻找 Rust 开发人员重写核心 C# 代码，这凸显了市场对 Rust 工作岗位的需求日益增长。
- Rust 因其复杂的概念而被认为在学习和使用上具有挑战性，但它在网络基础设施和安全领域的使用却在不断增加。
- 在拥有现有代码库的大公司中采用 Rust 可能是一项重大投资，需要熟练的开发人员以及与遗留代码的互操作性。
- Rust 可提高性能、减少内存使用量并具有可移植性，是防止安全漏洞的潜在解决方案。
- 微软之所以对 Rust 感兴趣，是因为在处理每天数十亿次的请求时需要高性能和成本优化。这涉及到性能、安全性、可扩展性和减少正确性错误等方面的考虑。
