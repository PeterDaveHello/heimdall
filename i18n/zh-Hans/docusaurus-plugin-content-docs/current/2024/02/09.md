# 2024-02-09

## [美国联邦通信委员会禁止在机器人电话中使用人工智能声音](https://www.fcc.gov/document/fcc-makes-ai-generated-voices-robocalls-illegal)

- 美国联邦通信委员会已宣布，根据 TCPA，在抢答电话中使用人工智能生成的声音是非法的。
- 联邦通信委员会一致通过了这一裁决，以解决机器人电话和诈骗短信中使用的深度伪造音频和视频技术问题。
- 该决定旨在打击人工语音的使用，以保护消费者免受欺诈活动的侵害。

### [反应](https://news.ycombinator.com/item?id=39304736)

- 人们正在讨论人工智能生成的机器人电话声音是否合法，有支持监管的，也有反对监管的。
- 雪佛龙推崇原则正受到与机器人电话中的人工智能声音有关的审查。
- 更广泛的讨论包括行政机构与立法机构之间的权力平衡、监管新技术的挑战以及在有争议的问题上缺乏共识。

## [Mozilla 任命新首席执行官，转向数据隐私保护](https://fortune.com/2024/02/08/mozilla-firefox-ceo-laura-chambers-mitchell-baker-leadership-transition/)

- Mozilla 公司宣布领导层发生变动，首席执行官米切尔-贝克（Mitchell Baker）卸任，专注于人工智能和互联网安全。
- 曾在 Airbnb 和 PayPal 等公司任职的 Mozilla 董事会成员劳拉-钱伯斯（Laura Chambers）将担任临时首席执行官。
- Mozilla 此举的目的是解决大科技和人工智能带来的隐私问题和挑战，同时还致力于开发优先考虑用户数据控制的产品和政策，并为现有商业模式提供替代方案。

### [反应](https://news.ycombinator.com/item?id=39302744)

- Mozilla 任命了一位关注数据隐私的新首席执行官，这引发了有关领导力和公司市场份额萎缩的讨论。
- 围绕前任首席执行官的高薪以及 Mozilla 在与谷歌 Chrome 浏览器竞争时面临的挑战，人们议论纷纷。
- 讨论的其他话题包括高管薪酬、FirefoxOS 的失败以及吸引用户使用 Firefox 浏览器的策略。

## [Cyberus Technology 发布开源版本，VirtualBox 现可支持 KVM 虚拟机管理程序](https://cyberus-technology.de/articles/vbox-kvm-public-release)

- 主题包括对虚拟化图形用户界面选项的不满，对 GPU 直通、virt-manager 和不同虚拟化技术（如 UML 和 KVM）的讨论，以及对安全功能、许可证问题和潜在未来发展的提及。

## [微软在 Windows 11 Insider 预览版中推出 Windows 版 Sudo](https://devblogs.microsoft.com/commandline/introducing-sudo-for-windows/)

- 微软在 Windows 11 Insider Preview Build 26052 中推出了适用于 Windows 的 Sudo，允许用户从未升级的控制台会话中运行升高的命令。
- 适用于 Windows 的 Sudo 是 GitHub 上的一个开源项目，它为希望在不打开新的提升控制台的情况下提升命令的用户提供了一种符合人体工程学的解决方案。
- 当前版本的 Windows 版 Sudo 支持三种配置选项，需要用户通过 UAC 对话框确认才能提升。微软计划扩充文档，并在未来分享更多有关该项目的安全影响和开源计划的信息。

### [反应](https://news.ycombinator.com/item?id=39305452)

- 讨论涉及多个与 Windows 有关的话题，如为 Windows 引入 Sudo 和整合团队以提供有效的解决方案。
- 有人对 "sudo "名称的使用及其引起混淆的可能性表示关切。
- 此外，还讨论了 PowerShell 在图形用户界面开发方面的功能、Windows 开发人员体验方面的挫折以及安全措施和软件部署方面的困难。

## [索尼 Funimation 将删除订阅者的数字图书馆，引发沮丧情绪](https://arstechnica.com/culture/2024/02/funimation-dvds-included-forever-available-digital-copies-forever-ends-april-2/)

- 由于与 Crunchyroll 合并，索尼旗下的动画流媒体服务 Funimation 将于 4 月 2 日后停止访问用户的数字图书馆。
- 用户最初被告知，他们可以无限期地流畅播放他们购买的电影和节目，但合并后，这些数字拷贝失去了支持。
- 客户感到沮丧的是，没有提及 DVD 或蓝光等实体拷贝的退款问题，如果他们知道失去流媒体访问权，可能就不会购买这些拷贝。

### [反应](https://news.ycombinator.com/item?id=39308679)

- 索尼公司因删除数字图书馆而遭到反弹，这引发了一场关于消费者保护、DRM 和数字内容提供商责任的讨论。
- 用户提出的解决方案包括实施最短购买期限、保险或代管制度，以及将获取与版权保护挂钩。
- 讨论还探讨了误导性广告、数字媒体的所有权、实体媒体和数字媒体的耐用性比较以及从流媒体服务中翻录或下载媒体的合法性等话题。
- 人们一直在讨论盗版的影响和道德问题，以及公司通过商业交易违背承诺和转移义务的问题。
- 总之，对话揭示了对 DRM 限制的不满、流媒体服务的局限性以及加强消费者保护措施的必要性。

## [门口效应：在不同地点之间移动如何导致记忆丧失](https://en.wikipedia.org/wiki/Doorway_effect)

- 门廊效应 "是一种心理现象，即在不同地点之间移动或通过门廊时会出现记忆缺失。
- 这是因为大脑为不同的环境创建了不同的记忆模型。
- 门道是事件的边界，会影响记忆的保持，导致错误记忆的增加，但对遗忘没有影响。

### [反应](https://news.ycombinator.com/item?id=39307677)

- 门廊效应是指人们在进入一个新的房间时经常忘记自己的目的的一种现象，这种现象在 Reddit 上引起了关于个人经历和克服环境变化造成的注意力分散的策略的讨论。
- Reddit 上的对话涉及相关主题，包括依赖语境的记忆回忆、空间存在、记忆联想以及盲人的经历。
- 一些评论对心理学研究的有效性表示怀疑，并强调了严格的实验设计和复制的重要性。
