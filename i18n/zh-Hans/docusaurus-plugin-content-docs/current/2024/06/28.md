## [“Python与苹果应用商店的拒绝作斗争”](https://lwn.net/SubscriberLink/979671/4fb7c1827536d1ae/)

- “从Python 3.11升级到3.12导致一些Python应用程序被苹果应用商店拒绝，因为在urllib解析器中存在字符串“itms-services”，这是在macOS沙盒应用程序中被禁止的。”
- “Eric Froemling的错误报告引发了Python开发者之间的讨论，讨论内容涉及如何适应应用商店的审核流程，考虑的解决方案包括混淆和分发级别的修补。”
- “达成了一致意见，在 Python 3.13 中添加一个名为 "--with-app-store-compliance" 的构建时选项，该选项将移除应用商店提交中存在问题的代码，突显了自由软件项目在不透明的审核过程中面临的挑战。”

### [反应](https://news.ycombinator.com/item?id=40815130)

- “由于二进制文件中包含“itms-services”URL 方案，Python 应用程序被苹果应用商店拒绝，同时使用 PyInstaller 构建的这些应用程序也会被 Windows Defender 标记。”
- “开发者认为，主要操作系统平台通过要求昂贵的代码签名证书来阻碍小型开发者，使应用程序的分发变得具有挑战性。”
- “建议的替代方案包括使用 Homebrew 安装 Mac 应用程序或使用 SignPath 提供的免费证书进行 Windows 构建，这突显了应用程序分发中与安全相关的拒绝问题。”

## [“捕捉引力波的新方法”](https://www.nature.com/articles/d41586-024-02003-6)

- “2015年，LIGO和Virgo天文台首次直接探测到引力波，这一发现彻底改变了物理学，使天文学家能够‘听到’宇宙事件。”
- “目前的探测器仅限于检测100至1000赫兹之间的频率，但新的仪器和技术旨在检测更广泛的频率范围，从兆赫兹到纳赫兹。”
- “五种新方法包括脉冲星计时阵列、微波望远镜、原子干涉仪、桌面探测器和量子晶体，每种方法针对不同的频率范围，并有望揭示更多的宇宙现象。”

### [反应](https://news.ycombinator.com/item?id=40820063)

- “第一代引力波探测器未能探测到引力波，否定了韦伯1987年的主张，但目前的干涉仪如LIGO、Virgo和KAGRA已经成功探测到了引力波。”
- “引力波证实了爱因斯坦的广义相对论，但量子引力理论仍未解决，弦理论和圈量子引力提供了潜在但具有挑战性的解释。”
- “未来的项目如LISA旨在增强探测能力，并且LIGO设施的公众参观活动也可用于教育目的。”

## [“帕金森病与肠道细菌的联系表明意想不到的简单治疗方法”](https://www.sciencealert.com/parkinsons-link-to-gut-bacteria-suggests-unexpected-simple-treatment)

- “研究人员已确定与帕金森病患者核黄素(维生素B2)和生物素(维生素B7)水平降低相关的特定肠道微生物，这表明补充B族维生素可能是一种潜在的治疗方法。”
- “这项由名古屋大学的西脇宏领导的研究发现，肠道细菌的变化影响了短链脂肪酸(SCFAs)和多胺的生成，这些物质对健康的肠道粘液层至关重要。”
- “研究结果发表在《npj帕金森病》期刊上，表明保持健康的肠道微生物群和减少环境毒素可能有助于预防帕金森病的症状。”

### [反应](https://news.ycombinator.com/item?id=40818987)

- “研究表明肠道细菌与帕金森病之间存在关联，某些细菌菌株可能会导致这种疾病。”
- “关于补充核黄素(B2)和生物素(B7)等维生素以缓解症状的潜在益处，已经出现了讨论。”
- “这项研究强调了肠道健康对神经系统疾病的重要性，但建议需要进行更多的研究，并且不应忽视现有的治疗方法。”
