# 2024-06-17

## [“MicroMac，一款售价低于5英镑的Macintosh”](https://axio.ms/projects/2024/06/16/MicroMac.html)

- “一个名为Pico-mac的项目使用Raspberry Pi RP2040微控制器来模拟Macintosh 128K，使其能够运行旧的Macintosh软件。”
- “RP2040的264KB RAM和2MB闪存足以容纳Mac的128KB内存、模拟器以及带有操作系统和软件的磁盘映像。”
- “该项目涉及构建一个Mac 128K模拟器，创建一个用于VGA视频输出的电路，并通过超频和解释器增强来优化性能，达到了大约1.4 MIPS。”

### [反应](https://news.ycombinator.com/item?id=40699684)

- “MicroMac，一款售价低于5英镑的Macintosh，因其低成本的DIY复古计算方式而备受关注。”
- “该项目涉及使用RP2040微控制器来模拟68k Macintosh，提供比仅在Linux上运行模拟器更真实的体验。”
- “讨论还涉及到在采购旧技术组件时面临的挑战和创造性解决方案，强调了社区的足智多谋以及对复古计算项目的兴趣。”

## [“如何在制造商不愿意修理时进行维修：将他们告上法庭”](https://blog.simonrumble.com/how-to-get-your-stuff-repaired-when-the-retailer-and-manufacturer-dont-wanna-take-em-to-court)

- “作者的伊莱克斯烤箱出现故障，尽管澳大利亚消费者法规定产品在合理使用寿命内自动享有保修，但他们仍被收取了160美元的诊断费。”

## [“NumPy 2.0”](https://numpy.org/devdocs/release/2.0.0-notes.html)

- “NumPy 2.0.0 是自2006年以来的第一个重大版本，由212位贡献者在11个月内开发完成，具有重要的新功能、性能改进以及对Python和C API的重大更改。”
- “关键更新包括新的可变长度字符串数据类型、加速的排序函数、以及公共和私有API之间的明确区分，同时还新增了用于创建自定义数据类型的公共C API。”
- “用户应注意重大变更和向后兼容性问题，并提供迁移指南以协助更新代码。”

### [反应](https://news.ycombinator.com/item?id=40699470)

- “NumPy 2.0 已经发布，引入了重大变化，包括弃用某些公共 API 和新增对字符串例程的支持。”
- “此次发布旨在解决NumPy中字符串操作的历史性弱点，可能有利于自然语言处理(NLP)和大型语言模型(LLM)的使用案例。”
- “与NumPy 1.x的兼容性大部分得以保持，除了“byte_bounds”函数，建议用户固定依赖项以避免在过渡期间出现问题。”

## [“布巴/基基效应”](https://en.wikipedia.org/wiki/Bouba/kiki_effect)

- “布巴/基基效应是一种心理联想，其中“布巴”与圆形联系在一起，而“基基”与尖锐形状联系在一起，这种现象在全球范围内都能观察到。”
- “这种现象最早在20世纪20年代被注意到，在婴儿和幼儿中尤为明显，表明它在语言学习中可能起到重要作用。”
- “fMRI研究显示了与这种声音形状匹配相关的大脑激活模式，提供了关于语言进化和声音象征的见解，尽管这种效果可能因文化和语言而异。”

### [反应](https://news.ycombinator.com/item?id=40699583)

- “雷焦艾米利亚教学法类似于蒙台梭利，强调跨模式的表达形式，例如将听觉体验转化为视觉体验。”
- “布巴-基基效应，指儿童自然地连接感官模式，这种能力在成人中往往会减弱。”
- “讨论包括这种减少是由于抑制还是大脑发育，以及语言和文化因素如何影响感知和命名惯例。”

## [“Quake 1 潜在的原始字体”](https://cohost.org/bekoha/post/2859948-quake-1-potential-or)

- “Quake 1 字体似乎源自一种名为“Visa”的较旧字体，该字体由拉斐尔·博古斯拉夫于1966年设计。”
- “‘Visa’ 出现在丹·X·索罗的《4,147种展示字体的Solotype目录》(1992年)和《模板字母：100种完整字体》(1988年)中。”
- “这一发现突显了Quake字体的历史根源，将其与1966年VGC全国字体设计比赛中的获奖设计联系起来。”

### [反应](https://news.ycombinator.com/item?id=40699459)

- “cohost.org上的辩论集中在Quake 1的字体是手工制作的还是源自现有字体，特别是在某些字母上存在显著差异。”
- “有些用户推测该字体可能受模板设计或新字体如Democratica的启发，而另一些用户则建议咨询原设计师以获得明确答案。”
- “讨论还探讨了字体的演变和涉及的创作过程，并对字体的使用权限持有一些怀疑态度。”
