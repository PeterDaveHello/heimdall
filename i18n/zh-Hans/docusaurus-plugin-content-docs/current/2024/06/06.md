# 2024-06-06

## [微软人工智能丑闻凸显全面网络隐私保护的必要性](https://spectrum.ieee.org/online-privacy)

- 巴拉特-拉加万（Barath Raghavan）和布鲁斯-施奈尔（Bruce Schneier）将网络隐私与捕鱼相提并论，强调了隐私泄露对整个生态系统的影响。
- 他们讨论了微软间谍活动的争议，以强调对数据保护采取整体方法的必要性。
- 作者主张在处理在线隐私问题时，应考虑到更广泛的数字环境。

### [反应](https://news.ycombinator.com/item?id=40592789)

- 文章对微软人工智能间谍丑闻进行了批评，呼吁更新隐私标准，并强调了监管信任度下降对经济和社会的影响。
- 报告将对 TikTok 隐私问题的关注与对美国公司类似问题的关注进行了对比，强调了两党在政治活动中对私人数据的滥用。
- 该文本强调亟需加强隐私法规，批评大型科技公司滥用数据和监控行为，并呼吁制定隐私标准。

## [太空探索技术公司的 "超重型 "飞船在墨西哥湾成功着陆](https://twitter.com/SpaceX/status/1798701489097183286)

## [从 OpenGL 到 Vulkan：在三个月内构建自定义游戏引擎](https://edw.is/learning-vulkan/)

- 作者记录了学习 Vulkan 和开发小型游戏引擎的三个月历程，尽管他之前没有 Vulkan 经验，但有一些 OpenGL 和游戏引擎方面的背景。
- 该引擎适合基于关卡的小型游戏，包括两个演示，可在 GitHub 上下载，并为从 OpenGL 过渡到 Vulkan 的初学者提供实用建议。
- 文章涉及着色器、同步和 UI 集成等技术方面，并讨论了 Vulkan 在小型 3D 桌面游戏中的优势，同时还反思了 OpenGL 不确定的未来和 WebGPU 的潜力。

### [反应](https://news.ycombinator.com/item?id=40595741)

- 文章探讨了作者学习 Vulkan 和开发游戏引擎的经验，提供了实用的编程建议，并强调了 YAGNI（You Aren't Gonna Need It）原则。
- 它强调了将 Rust 与 Vulkan、WGPU 和 Rend3 结合使用所面临的挑战，包括性能、并发性和社区支持问题。
- 讨论涵盖了 Vulkan 教程的复杂性，比较了 OpenGL 和 Direct3D 等不同的图形应用程序接口，研究了更高级抽象和工具的优势，指出了 Vulkan 的陡峭学习曲线和先进功能。

## [美国监管机构对微软、OpenAI 和 Nvidia 发起反垄断调查](https://www.nytimes.com/2024/06/05/technology/nvidia-microsoft-openai-antitrust-doj-ftc.html)

- 包括司法部和美国联邦贸易委员会在内的联邦监管机构已分工负责对人工智能行业的主要企业进行反垄断调查：微软、OpenAI 和 Nvidia。
- 美国联邦贸易委员会（FTC）将重点关注微软和 OpenAI，而司法部将调查英伟达（Nvidia），这表明监管部门对人工智能技术的审查力度加大。
- 这些调查旨在解决潜在的反垄断违法行为和对消费者的伤害，反映出美国在监督快速发展的人工智能产业方面做出了越来越大的努力。

### [反应](https://news.ycombinator.com/item?id=40593275)

- 美国已对 Nvidia、微软和 OpenAI 展开反垄断调查，重点是潜在的反竞争行为和公司所有权结构。
- 英伟达（Nvidia）因其在芯片分销领域的主导地位而备受关注，而微软则因入股 OpenAI 和收购 Inflection AI 的员工而备受质疑。
- 这些调查凸显了反垄断法的复杂性、更新的必要性以及对科技行业监管和消费者保护的广泛影响。

## [Balon 蛋白的发现揭示了细胞休眠和存活的关键机制](https://www.quantamagazine.org/most-life-on-earth-is-dormant-after-pulling-an-emergency-brake-20240605/)

- 研究人员发现了一种名为 "Balon "的蛋白质，它能迅速停止并重新启动细胞中的蛋白质生产，从而帮助细胞休眠。
- Balon 存在于北极永久冻土细菌和其他生物体中，它能在核糖体发挥作用的过程中打断核糖体，使其快速进入和退出休眠状态。
- 这种蛋白质存在于 20% 以上的细菌基因组中，突显了它在物种抵御气候变化方面的潜在作用，并为了解细胞适应和生存策略提供了新的视角。

### [反应](https://news.ycombinator.com/item?id=40592575)

- 卡拉-海伦娜-布埃诺（Karla Helena-Bueno）意外发现了北极细菌中的冬眠因子，凸显了偶然性在科学突破中的作用，将人类的直觉与机器学习的系统方法进行了对比。
- 讨论探讨了机器学习加强创新的潜力、人类冬眠和冷冻的可行性以及因纽特人的进化适应性。
- 该书批判了现代生活的复杂性和社会压力，强调人与人之间的联系、社会参与和目标感对真正幸福的重要性，而不是人为的解决方案。

## [调查七座圣米迦勒大教堂的排列：神话还是现实？](https://geospatial.netlify.app/posts/gds-2024-04-20-cathedrals/)

- 该研究项目调查了七座著名的供奉圣米迦勒的大教堂完美地排列在一条直线上的说法。
- 方法包括收集地理数据、使用 Python 库进行大地测量计算，以及使用 KeplerGl 实现数据可视化。
- 结果表明，这些大教堂在球面几何图形上并不对齐，但在墨卡托投影图上似乎是对齐的，均方根误差为 167.0303 千米。

### [反应](https://news.ycombinator.com/item?id=40595499)

- 这项调查研究了供奉圣米迦勒的七个宗教场所是否呈直线排列，对超常排列和巧合的看法不一。
- 讨论内容包括这些修道院和圣所的历史和奇迹起源、地图投影的挑战以及文化象征的作用。
- 对话最后对秘密、长期的阴谋持怀疑态度，并强调对此类理论中使用的数据和方法进行严格审查的重要性。

## [AeroSpace：受 i3 启发的 macOS 平铺窗口管理器现已进入公开测试版](https://github.com/nikitabobko/AeroSpace)

- AeroSpace 是受 i3 启发为 macOS 设计的全新平铺窗口管理器，目前处于公开测试阶段。
- 主要功能包括基于树形模式的手动平铺、纯文本配置、CLI（命令行界面）脚本功能，以及无需禁用 SIP（系统完整性保护）即可实现对多显示器的适当支持。
- AeroSpace 使用自己的虚拟工作空间，而不是 macOS Spaces，支持通过 Homebrew 安装进行自动更新，并强调以键盘为中心的使用方式和最小化 GUI（图形用户界面）。

### [反应](https://news.ycombinator.com/item?id=40596689)

- 讨论的重点是 AeroSpace，这是一款适用于 macOS 的平铺窗口管理器（TWM），因其窗口管理和多显示器支持而备受赞誉，但也因缺乏 i3/sway 的灵活性和需要键盘命令来调整布局而饱受批评。
- 与 Yabai 不同的是，AeroSpace 不需要禁用系统完整性保护（SIP），因此对用户更加友好，但用户也指出了与其他工具（如 Yabai、AutoRaise、Hammerspoon、Rectangle 和 Magnet）之间的问题。
- 用户讨论了在 macOS 上实现类似 i3 的功能所面临的挑战、超宽屏幕的好处以及 macOS 公证的复杂性，有些用户还提出了 tmux 或本地 macOS 功能等替代方案。

## [倡导针对使用公共数据进行人工智能貌相的法律保护](https://link.springer.com/article/10.1007/s13347-023-00616-9)

- 文章以社会控制、污名化和增加曝光率等风险为由，主张建立一项独特的法律权利，以保护个人免受未经其明确知情同意而使用公开数据进行人工智能特征分析的侵害。
- 该报告批评欧盟的《通用数据保护条例》（GDPR）对人工智能貌相缺乏足够的保障，并强调了一些研究，这些研究表明人工智能在从社交媒体数据中识别心理健康问题方面具有很高的准确性。
- 文章呼吁修订 GDPR 等法律框架以确立这一权利，强调隐私、知情同意和监管措施的重要性，以平衡人工智能特征分析的利弊。

### [反应](https://news.ycombinator.com/item?id=40597503)

- 讨论强调，由于技术的进步，智能手机和测速摄像头等监控手段得以广泛应用且价格低廉，因此迫切需要更新隐私法。
- 通过比较中国的社会信用体系和美国的信用评分体系，探讨了包括面部识别在内的监控技术的伦理影响，以及租房者保护和信用评分的复杂性。
- 会上提出了对数据隐私、人工智能剖析以及 GDPR 等现行法律有效性的担忧，并就实施隐私保护和人工智能生成内容的伦理问题展开了辩论。
