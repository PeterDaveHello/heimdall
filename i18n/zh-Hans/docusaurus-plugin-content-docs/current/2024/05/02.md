## [参议员要求美国联邦贸易委员会调查汽车制造商的位置数据共享情况](https://www.wyden.senate.gov/news/press-releases/wyden-markey-reveal-automakers-provide-detailed-location-information-to-law-enforcement-without-a-warrant-rarely-notify-car-owners-request-ftc-investigate-broken-promises-to-protect-drivers-privacy)

- 参议员怀登和马基敦促联邦贸易委员会调查主要汽车制造商在没有授权令的情况下与执法部门共享客户位置数据的行为。
- 很少有汽车制造商要求获得这些数据的授权，而其他汽车制造商在收到传票后会公布这些数据，这引起了人们对隐私和影响的担忧，尤其是在最高法院就生殖权利做出多布斯裁决之后。
- 这一要求凸显了参议员们对汽车制造商在处理位置数据时隐私保障不足的担忧。

### [反应](https://news.ycombinator.com/item?id=40222499)

- 联网汽车引发了隐私问题，重点是汽车制造商对位置数据的收集和共享。
- 建议消费者在购买汽车时谨慎行事，禁用特定的汽车功能。
- Wyden 参议员的报告强调了提高位置数据共享透明度和保障措施的必要性，主张加强隐私保护和消费者知识。

## [mRNA 疫苗有望防治胶质母细胞瘤](https://www.insideprecisionmedicine.com/topics/oncology/mrna-cancer-vaccine-reprograms-immune-system-to-tackle-glioblastoma/)

- 佛罗里达大学的研究人员通过对免疫系统进行重新编程，创造出一种针对胶质母细胞瘤的 mRNA 癌症疫苗，以对抗癌症。
- 在对四名成年人和十只患有脑肿瘤的宠物狗进行试验后，疫苗在刺激免疫系统对抗癌细胞方面取得了令人鼓舞的成果。
- 这种个性化疫苗利用患者的肿瘤细胞和脂质纳米颗粒进行递送，目前正进入脑癌的扩大一期临床试验阶段，有可能改变癌症免疫疗法。

### [反应](https://news.ycombinator.com/item?id=40231515)

- mRNA 癌症疫苗在治疗胶质母细胞瘤方面取得了可喜的成果，这凸显了监管机构在确保药品质量和管理索赔方面的作用。
- 讨论内容包括 mRNA 疫苗的益处和风险、癌症积极治疗面临的挑战以及就疫苗做出知情决策的重要性。
- 主题包括疫苗安全问题、公共卫生建议以及疾病控制与癌症发病率之间的潜在联系。

## [认知：用反语法重新定义元编程](https://ret2pop.nullring.xyz/blog/cognition.html)

- 该文件批判了 Lisp 的语法限制，并介绍了 Cognition，一种利用反语法的全新全后缀语言。
- 讨论了 Cognition 的发展，强调了标记化、分隔符、忽略字符、特定命令和字符，以及可定制的标记化分隔符和假名。
- 该书探讨了 VMACRO、递归词义、Brainfuck 方言，以及 Cognition 在字符串处理和符号人工智能中的应用，突出了其在编程中的语法自动化和哲学优雅。

### [反应](https://news.ycombinator.com/item?id=40231563)

- 文章介绍了一种新语言 "认知"（Cognition），侧重于用户可修改的语法，引发了读者对其清晰性和相关性的讨论。
- 它深入探讨了 Lisp 宏、AST、树结构、元编程以及 Cognition 如何动态地重新定义语法结构。
- 讨论涵盖了该项目的复杂性和创造性，包括对空白分隔词、可读性系统和任意元编程的使用的反馈，与会者对该项目的主流采用和实际效用发表了不同意见。

## [警惕 AWS S3 成本放大威胁](https://blog.limbus-medtec.com/the-aws-s3-denial-of-wallet-amplification-attack-bc5a97cc041d?gi=67ac052cc1ce)

- AWS S3 拒绝钱包放大攻击给在 AWS S3 上存储大型数据文件的公司带来了风险，由于数据传输成本计算方法不同，会造成意想不到的成本变化。
- 无论数据是否公开，这种攻击都会因收到的数据与账单金额不符而给公司带来大量意外支出。
- 缓解措施包括实施成本警报、跟踪 API 请求和控制对 S3 存储桶的访问，以防止恶意行为者利用潜在的财务漏洞。

### [反应](https://news.ycombinator.com/item?id=40227736)

- 用户担心 AWS S3 中的一个潜在漏洞会导致范围请求的出口成本上升，并强调了对计费方式、数据传输差异和意外费用的担忧。
- 讨论涉及支出限额、未经授权的收费、API 文档、数据缓存以及 AWS 和 Cloudflare 服务的比较，揭示了系统管理员在处理未经授权的请求时所面临的挑战。
- 对话强调了在 AWS 计费和管理问题不断的情况下利用 S3 Glacier 和 R2 等各种云服务的优势。

## [重新审视轮胎排放的毒性](https://phys.org/news/2024-05-toxicity-fresh-scrutiny-salmon-die.html)

- 文章强调了轮胎排放物的潜在危害，特别关注了对环境和人类健康构成风险的化学物质 6PPD。
- 研究人员发现 6PPD 与鱼类死亡之间存在联系，这促使监管机构对这种化学物质进行仔细检查。
- 由于轮胎排放对空气和水质的影响令人担忧，加利福尼亚州已着手对轮胎中的 6PPD 进行监管，促使制造商探索替代方案。

### [反应](https://news.ycombinator.com/item?id=40223253)

- 文章探讨了轮胎磨损对环境的影响，并建议对车辆征收重量税，以减少污染和道路损坏。
- 它探讨了轮胎磨损、车辆重量、路面恶化和行人安全之间的关系，强调了轮胎中的有害化学物质。
- 建议包括在家办公、利用远程办公和联合办公空间减少通勤时间、提倡使用替代交通方式以尽量减少对汽车的依赖，从而提高生活质量。

## [复兴 Neofetch：新叉确保社区发展](https://github.com/LorenDB/neofetch)

- Neofetch 是一款用于显示系统信息的命令行界面工具，由于缺乏活动已被归档。

### [反应](https://news.ycombinator.com/item?id=40222213)
