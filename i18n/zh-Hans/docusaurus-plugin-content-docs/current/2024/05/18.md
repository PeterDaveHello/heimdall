# 2024-05-18

## [Bend：利用 HVM2 进行 GPU 计算的高级语言](https://github.com/HigherOrderCO/Bend)

- Bend 是一种高级编程语言，针对 GPU 等大规模并行硬件进行了优化，提供快速对象分配、高阶函数、递归和连续过程。
- 它在 HVM2 运行时上运行，可根据内核数量实现近乎线性的加速，而无需明确的并行注释或手动线程管理。
- Bend 由 HigherOrderCO.com 创建，通过在 GPU 上高效执行复杂算法和模拟各种并发系统来简化并行编程。

### [反应](https://news.ycombinator.com/item?id=40390287)

- 目前正在讨论 Bend 编程语言及其 HVM2 实现，重点是 GPU 性能与 Python 和 Mojo 的比较。
- 用户正在评估基准、线性扩展、编译器效率和 Bend 的潜在应用，并就更清晰的免责声明、单核优化和增强代码生成提出了建议。
- Bend 的自动并行化功能在实用性和对开发社区的影响方面既受到好评，也受到质疑，人们对该项目的技术障碍和性能断言表示担忧。

## [Toon3D：创建手绘卡通的 3D 视图](https://toon3d.studio/)

- Toon3D 是一种通过片断刚性变形优化恢复手绘卡通场景三维结构的新技术。
- 它能从非几何一致的图像中重建摄像机姿势和密集几何图形，从而生成卡通场景的新视点。
- 该方法包括深度预测、图像标注、对齐、用户友好的注释工具、相机姿态估计、图像变形和高斯细化，以创建一个连贯的三维结构，从不同角度对卡通进行可视化。

### [反应](https://news.ycombinator.com/item?id=40389445)

- 讨论以《未来世界》和迪斯尼的《奥利弗与公司》等热门节目为例，探讨了三维动画在动画片中的应用。
- 它探讨了动画师在开发三维空间时遇到的挑战，以及如何整合人工智能和机器学习，将二维作品转化为三维模型。
- 人工智能对三维建模的潜在影响、制作视频场景所涉及的艺术决定、图像渲染新技术的应用，以及围绕人工智能在动画中的应用而持续进行的辩论，都得到了重点介绍。

## [ILGPU：利用 C# 和 F# 进行高性能 GPU 编程](https://github.com/m4rs-mt/ILGPU)

- ILGPU 是一款高性能 JIT 编译器，专为基于 .Net 语言的 GPU 程序而设计，将 C++ AMP 的灵活性与 CUDA 的性能相结合。
- 该库提供辅助功能和高级算法，并在 Discord 上拥有一个活跃的社区，同时还为初学者提供示例项目和构建指南。
- 要使用 ILGPU，请确保您拥有 Visual Studio 2022 和 .NET 6.0 SDK 工具链，并做好准备应对测试过程中可能出现的 XUnit/Visual Studio 挑战，为调试提供源链接和符号支持。

### [反应](https://news.ycombinator.com/item?id=40393873)

- ILGPU 支持用 C# 和 F# 编写 GPU 程序，提供了一种高级方法来提高性能优化。
- ComputeSharp 是 Windows 独占的 GPU 编程替代软件，是对 ILGPU 的补充。
- 在 GPU 编程方面，低级语言（C/C++）和高级语言之间的争论一直存在，其中 ILGPU 利用基于抽象堆栈的字节码来高效地针对 CUDA 和 OpenCL 编程。
