## [探索 Adobe Photoshop 1.0.1 源代码](https://computerhistory.org/blog/adobe-photoshop-source-code/)

- 计算机历史博物馆以展览、出版物和活动为特色，深入探讨计算机技术的历史。
- Adobe Photoshop 1.0.1 的 1990 年源代码由托马斯和约翰-诺尔精心制作，现在可用于非商业目的，凸显了该软件的精密设计。
- 博物馆提供了软件架构师 Grady Booch 的宝贵观点，展示了 Photoshop 早期演变的特殊视角。

### [反应](https://news.ycombinator.com/item?id=40368016)

- 讨论探讨了保存 90 年代 Photoshop、Illustrator 和 PostScript 等 Adobe 设计软件源代码的意义，强调了其历史重要性。
- 它将 Adobe 产品与 GIMP 和 Krita 等开源替代产品进行了比较，同时研究了用户界面、软件偏好和商标准则的演变。
- 强调了软件遗产等组织为保护作为后代文化遗产的源代码所做的努力。

## [血液中的蛋白质是早期癌症检测的关键](https://www.theguardian.com/society/article/2024/may/15/proteins-blood-cancer-warning-seven-years-study)

- 牛津大学的研究人员发现了 618 种与 19 种癌症有关的血液蛋白质，其中 107 种与 7 年后诊断出的癌症有关，这显示了早期癌症检测的潜力。
- 改变特定的血液蛋白质会影响罹患九种癌症的风险，这为癌症预防和早期诊断带来了希望。
- 这一发现标志着该领域专家在精确定位癌症高风险标志物和协助癌症早期检测方面取得了重大进展。

### [反应](https://news.ycombinator.com/item?id=40366443)

- 癌症筛查中的早期检测至关重要，并讨论了不同检测方法的益处和风险。
- 强调了加强医疗保健资源、透明的决策过程和改变生活方式对预防癌症的重要性。
- 此外，还讨论了创建实用的癌症检测血液生物标志物所面临的挑战，以及在保险范围内防止基因歧视的法律保障。

## [新的指数函数将 SiLU 和 SoftMax 的速度提高了 2 倍](https://github.com/ggerganov/llama.cpp/pull/7154)

- GitHub 上的 llama.cpp 代码库拥有 58.3k stars 和 8.3k forks。
- 贡献者 jart 正在增强 CPU 的 silu 和 softmax 函数，将 AVX512 和 AVX2 指令用于指数值计算，在基准测试中表现出更高的性能，ULP 误差小于 1。
- 作者强调了 vpermi2pd 在 Float64/双查找表中的使用，并指出 Zen4 或 icelake-client/tigerlake 等硬件可能无法从中获益，因为该表必须适合两个寄存器。

### [反应](https://news.ycombinator.com/item?id=40371612)

- 借鉴雷达信号处理器的经验，创建了新的指数函数，以加快 SiLU 和 SoftMax 的计算速度，同时保持精度。
- 利用 C 和 C++ 中的 SIMD 和硬件固有功能，通过查找表优化指数计算。
- 讨论内容包括非时态加载/存储指令、查找表限制、数据处理性能差异、llama.cpp 性能增强、ML 运行时比较、用于模型部署的 TensorFlow Lite 和 ONNX、机器人、动画、GPU 模型影响、CPU 速度增强和内存架构进步，并建议进行基准测试，以确定特定任务的最有效方法。

## [5 万年来前所未有的二氧化碳上升：人类活动是主要因素](https://today.oregonstate.edu/news/researchers-identify-fastest-rate-natural-carbon-dioxide-rise-over-last-50000-years)

- 俄勒冈州立大学（Oregon State University）的研究人员通过研究南极古冰，发现了过去 5 万年来二氧化碳水平最快速的自然增长。
- 美国国家科学院院刊》上发表的研究报告指出，目前大气中二氧化碳的上升速度比历史上任何时期都要快十倍，这主要是人类活动造成的。
- 这项研究对历史上的气候变化事件和当今气候变化的潜在影响提供了宝贵的见解，强调了大气中无与伦比的二氧化碳含量。

### [反应](https://news.ycombinator.com/item?id=40365882)

- 文章讨论了 5 万年来二氧化碳自然上升速度最快所带来的挑战，以及人工智能、核冲突和气候变化的风险。
- 报告强调了人类智慧在解决这些问题中的关键作用，并列举了避免核灾难的历史案例。
- 讨论内容包括不断上升的二氧化碳水平、平流层气溶胶注入等解决方案，以及对减排应对气候变化的紧迫性和可行性的不同看法，同时还涉及科幻小说的社会影响和有关 AGI 风险的辩论。
