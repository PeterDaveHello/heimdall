# 2024-05-23

## [泄露文件揭露 OpenAI 针对前员工的激进策略](https://www.vox.com/future-perfect/351132/openai-vested-equity-nda-sam-altman-documents-employees)

- 泄露的文件显示，OpenAI 在首席执行官山姆-奥特曼（Sam Altman）的领导下，采用激进的手段阻止前员工批评公司，威胁要取消他们的既得股权，除非他们签署限制性离职协议。
- 尽管 Altman 公开道歉并声称不知情，但有他和其他高管签名的内部文件表明，他们知道这些规定，这引发了人们对 OpenAI 透明度和可信度的担忧。
- 目前，OpenAI 正在更新其离职流程，以确保前员工的既得权益，并删除非贬损条款，但仍有更多工作要做，以消除人们的担忧。

### [反应](https://news.ycombinator.com/item?id=40447431)

- 泄露的文件显示，OpenAI 通过威胁前员工的既得股权，使用激进的策略来执行非贬损协议，导致了员工的反弹和 OpenAI 的后续保证。
- 这种情况引发了对合同谈判、股权补偿和非竞争条款的讨论，凸显了可执行性的复杂性和地区差异。
- 批评者认为，OpenAI 的行为揭示了更深层次的信任和透明度问题，对行业趋势、道德问题以及萨姆-奥特曼等知名人士的影响力产生了更广泛的影响。

## [记录证实，OpenAI 没有为 ChatGPT 克隆斯嘉丽-约翰逊的声音](https://www.washingtonpost.com/technology/2024/05/22/openai-scarlett-johansson-chatgpt-ai-voice/)

- 斯嘉丽-约翰逊（Scarlett Johansson）声称，OpenAI 在她拒绝首席执行官山姆-奥特曼（Sam Altman）的许可请求后，未经她同意复制了她的声音。
- 根据《华盛顿邮报》查阅的消息来源和文件，OpenAI 曾为一个秘密项目寻找声音温暖、引人入胜的非工会演员，以实现 ChatGPT 的人性化，但并没有明确要克隆约翰逊的声音。

### [反应](https://news.ycombinator.com/item?id=40448045)

- 据称，OpenAI 在未经斯嘉丽-约翰逊（Scarlett Johansson）同意的情况下为 ChatGPT 使用了一位声音酷似斯嘉丽-约翰逊（Scarlett Johansson）的配音演员，这引起了法律和道德方面的关注。
- 辩论强调了语音模仿法律的复杂性，提到了 Midler 诉 Ford 和 Tom Waits 诉 Frito-Lay 等案件，并强调了更新法律框架的必要性。
- 这场争议引发了人们对人工智能、配音演员以及未经明确同意使用相似声音的伦理问题的广泛关注。

## [WinDirStat：适用于 Windows 的开源磁盘使用情况查看器和清理工具](https://windirstat.net/)

- WinDirStat 是一款用于查看和清理 Microsoft Windows 磁盘使用情况的开源工具，提供目录列表、树状图和扩展名列表。
- Linux 的替代方案包括 KDirStat 和 QDirStat，而 MacOS X 用户则可以使用 Disk Inventory X 或 GrandPerspective。
- 该软件采用 GPLv2 许可，确保其保持自由和开源。

### [反应](https://news.ycombinator.com/item?id=40449767)

- 讨论比较了各种磁盘使用分析工具，重点是不同操作系统的功能、性能和可用性。
- WinDirStat 很受欢迎，而且是开源的，但因速度慢而受到批评，而 WizTree 速度更快，但存在安全问题，而且不是开源的。
- 用户强调了速度、安全性和可用性之间的权衡，并表示需要更好的文件管理功能和改进的可视化方法，如树状图和旭日图。

## [微软中断必应、Copilot、ChatGPT 和 DuckDuckGo 服务](https://www.bleepingcomputer.com/news/microsoft/microsoft-outage-affects-bing-copilot-duckduckgo-and-chatgpt-internet-search/)

- 微软的一次故障影响了必应、Copilot、DuckDuckGo 和 ChatGPT 等多个互联网服务。
- 这一事件凸显了在线服务的相互关联性及其对微软基础设施的依赖性。
- 有关更详细的讨论，用户已被引导至 Hacker News 上的特定主题。

## [Sierra 公司的《太空探索 II》软盘错误暴露了 AGI 源代码](https://lanceewing.github.io/blog/sierra/agi/sq2/2024/05/22/do-you-own-this-space-quest-2-disk.html)

- 2024 年，人们发现 Sierra On-Line 的《太空探索 II》的某些版本的 720KB 软盘上残留有 C 语言源代码，原因是格式化不当。
- 这一错误暴露了 Sierra AGI 解释器约 70% 的源代码，包括 75 个 C 源文件和 16 个汇编文件，直到 2016 年才被发现。
- 源代码现在可以在 GitHub 上获取，这为了解 Sierra 的开发流程提供了宝贵的资料，也为 AGILE 网络解释器等项目提供了帮助。

### [反应](https://news.ycombinator.com/item?id=40455758)

- 讨论强调了软件发行中的历史性错误，如在最终产品版本中包含敏感数据或非预期数据，1989 年的 DOS 游戏 "双龙 II：复仇 "就是一个例子。
- 它探讨了软盘的反盗版措施、消费级磁盘复制机的局限性以及读取磁性数据的现代工具（如 Greaseweazle）。
- 与会者分享了对老游戏的怀旧体验，讨论了源代码泄露的影响，并强调了扇区级图像存档的重要性。

## [掌握无人机飞行：与 FPVSIM 交互式初学者教程](https://fpvsim.com/drone-flying-101)

- FPVSIM 为初学者提供了详细的无人机飞行教程，涵盖了悬停、转弯、倾斜和协调转弯等关键概念。
- 教程强调了无人机和汽车导航的区别，重点是通过滚动、偏航和俯仰进行三维运动，并通过网络模拟器提供了实用课程。
- 用户可以访问手册、性能提示和高级配置等其他资源，并在 FPVSIM 的网络或桌面模拟器上继续练习，以增强学习效果和逼真的图形。

### [反应](https://news.ycombinator.com/item?id=40449278)

- "Drone Flying 101 - An Interactive Tutorial for Beginners "涵盖了 FPV（第一人称视角）和非 FPV（电影级）无人机的基础知识，重点介绍了大疆 Mavic 等非 FPV 无人机因 GPS 稳定功能而带来的用户友好性。
- 教程强调了选择合适发射机的重要性，推荐使用 OpenTX 或 EdgeTX 软件和 ExpressLRS 协议来提高性能，常用的发射机包括 Radiomaster 和 TBS Tango 2。
- 讨论探讨了无人机在现代战争中不断演变的使用情况，特别是在乌克兰，并涉及自主无人机、飞行员培训和反制措施的潜在未来。

## [Emacs Easy Draw：与 Org-Mode 集成的无缝绘图工具](https://github.com/misohena/el-easydraw)

- Emacs Easy Draw 是在 Emacs 中运行的绘图工具，需要 Emacs 27.2、图像和 SVG 支持、gzip/gunzip 和 libxml。
- 它与 Org-Mode 集成，允许用户直接在文档中创建和编辑图纸，并支持导出为 HTML 或 LaTeX 格式。
- 该工具使用 SVG 规范的一个子集，包括一个用于 CSS 和 HTML 编辑的取色器库，并以 GPLv3 许可，鼓励社区贡献。

### [反应](https://news.ycombinator.com/item?id=40447998)

- Emacs Easydraw 是一款专为 Emacs 开发的绘图工具，它是在 Windows 而非典型 Linux 环境下创建的。
- 用户发现它对小型绘图非常有用，而且有可能与 org-roam 等工作流程集成，不过目前还缺乏手写功能和协作功能。
- 随着算法的改进，有些人认为它可以与 Inkscape 相媲美，而且它还拥有一个方便的颜色选择器，使其优于许多在线工具。

## [汤姆-华兹因盗用嗓音赢得对 Frito-Lay 260 万美元的诉讼](http://tomwaitslibrary.info/biography/copyright/frito-lay/)

- 1988 年，汤姆-威茨（Tom Waits）起诉 Frito-Lay 公司在多力多滋的广告中使用了模仿他的歌曲 "Step Right Up "的声音，声称是虚假代言和声音盗用。
- 陪审团裁定 Waits 获赔 260 万美元，上诉后维持原判，为《兰哈姆法》和加州法律下的语音盗用开创了法律先例。
- Waits 批评了那些允许自己的音乐出现在广告中的艺术家，认为这有损于艺术的完整性，他的案例凸显了保护艺术声誉的重要性。

### [反应](https://news.ycombinator.com/item?id=40452655)

- 讨论探讨了使用仿声配音演员的法律和道德问题，提到了汤姆-华兹（Tom Waits）对 Frito-Lay 的诉讼以及斯嘉丽-约翰逊（Scarlett Johansson）与 OpenAI 的争议。
- 它探讨了声音模仿的复杂性、"出卖 "的概念，以及对艺术家作品许可的不断演变的看法，包括未经同意使用与约翰逊相似的声音所涉及的法律问题。
- 对话批评了企业的做法，尤其是 OpenAI 的做法，并探讨了人工智能生成的声音对知识产权和艺术完整性的影响这一更广泛的问题。

## [图灵机新突破：计算阿克曼级函数](https://www.sligocki.com//2024/05/22/bb-3-4-a14.html)

- 帕维尔-克罗皮茨（Pavel Kropitz）发现了一种3态4符号的图灵机（TM），它能够计算阿克曼级函数，在磁带上正好有（（2\uparrow^{15} 5）+14）个非零符号时停止。
- 这个 TM 于 2024 年 4 月 25 日被发现，是已知的第一个模拟如此复杂函数的 TM，其行为需要双重归纳才能证明。
- 这一发现也是 "归纳证明 "验证器的一个测试案例，旨在对此类证明进行标准化和验证。

### [反应](https://news.ycombinator.com/item?id=40453221)

- 这本书深入探讨了图灵机的复杂性，重点是 "海狸忙 "问题和一个新的图灵机程序 BB(3,4)，其复杂性超过了阿克曼函数。
- 它讨论了程序的模块化结构、长运行图灵机的特性以及科尔莫哥罗夫复杂性的含义，强调了分析这些机器所面临的挑战。
- 对话还批评了同行评审制度、数字信息来源的可靠性，并强调了科学研究中正确存档和引用实践的重要性。

## [在大幅涨价之际，Computershare放弃VMware，转投Nutanix AHV](https://www.theregister.com/2024/05/22/computershare_vm_migration_project/)

- 由于博通（Broadcom）收购 VMware 后价格大幅上涨，全球股票市场股份登记运营商 Computershare 正在放弃 VMware。
- VMware管理程序的成本增加了10至15倍，促使Computershare转向Nutanix AHV，预计将在一年内完成24,000台虚拟机的迁移。
- 此举凸显了博通面临的潜在问题，因为由于其高收入、高利润的定价策略，其他大客户也可能寻求替代方案。

### [反应](https://news.ycombinator.com/item?id=40452260)

- VMware 现在归 Broadcom 所有，由于大幅提价，它失去了一个拥有 24,000 台虚拟机的大客户，这反映了它专注于高收入、高利润业务的战略。
- 批评者认为，这种方法有可能将客户推向 Nutanix 等竞争对手或开源解决方案，这凸显了迁移大型 IT 系统的挑战和保持客户信任的重要性。
- 讨论还强调了锁定供应商的风险，并建议采用混合或多云方法来减少对单一管理程序供应商的依赖。
