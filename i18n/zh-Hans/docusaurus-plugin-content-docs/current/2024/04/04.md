## [谷歌推出 Jpegli：更快、更高效的 JPEG 编码库](https://opensource.googleblog.com/2024/04/introducing-jpegli-new-jpeg-coding-library.html)

- 谷歌推出了一个名为 Jpegli 的新 JPEG 编码库，在高质量设置下可将压缩率提高 35%。
- 与标准 JPEG 相比，Jpegli 采用了自适应量化启发式技术和增强型量化矩阵选择技术，因而更快、更有效、更美观。
- 测试表明，与传统的 JPEG 编解码器相比，Jpegli 能更有效地压缩高质量图像，从而可能带来更快、视觉效果更好的互联网体验。

### [反应](https://news.ycombinator.com/item?id=39920644)

- 谷歌苏黎世分公司推出了新的 JPEG 编码库 Jpegli，该库采用瑞士德语命名规则，侧重于提高图像质量和压缩效率。
- 讨论内容包括比较各种编解码器的图像质量、评估 AVIF 和 JPEG XL 等格式，以及关于 Google 选择不在 Chrome 浏览器中加入 JPEG XL 的争论。
- 用户考虑将 Jpegli 与 WebAssembly 集成以扩大其使用范围，强调优化现有格式而不是创建新格式，支持高位元深度编码。

## [PyTorch 的 ipex-llm：英特尔兼容 LLM 库](https://github.com/intel-analytics/ipex-llm)

- bigdl-llm 项目已转向 ipex-llm，这是一个专为在英特尔 CPU 和 GPU 上以最小延迟执行 LLM 而定制的 PyTorch 库。
- ipex-llm 基于 PyTorch 的英特尔扩展，可与各种工具和模型无缝集成，为英特尔硬件上的 LLM 模型提供优化和微调选项。
- 超过 50 个模型在 ipex-llm 上进行了优化，支持低位推理、自指定解码以及与社区库的兼容性，详细情况见文档中的基准测试、代码示例和安装指南；性能可能因使用和配置而异，尤其是在非英特尔设备上。

### [反应](https://news.ycombinator.com/item?id=39915594)

- 与 AMD 和 Nvidia 相比，GitHub 的讨论强调了英特尔 CPU 和 GPU 在 GPU 市场上的潜力，强调以具有竞争力的价格为人工智能工作负载提供更高的 VRAM 容量。
- 谈话表明，英特尔应集中精力在其推出的 GPU 中提高 VRAM 容量，优化内存性能，并以经济实惠的价格满足消费者对高内存容量的需求。
- 与会者对英特尔的软件支持和价值主张表示乐观，认为英特尔 GPU 与其他竞争对手相比前景广阔。

## [白宫提议为月球任务协调月球时间](https://www.bbc.co.uk/news/science-environment-68722032)

- 白宫已责成美国国家航空航天局（NASA）为月球开发一个新的时区--"协调月球时间"（LTC），因为不同的重力场强度会导致那里的时间流逝得更快。
- 由于地球上的原子钟在月球上运行得更快，这给飞行任务带来了时间上的挑战，因此 LTC 的目的是使航天器在月球上的飞行任务同步进行。
- 各个国家和组织正在合作，争取在 2026 年之前为美国载人登月任务建立 LTC，这对于在太空中进行有效的通信和导航至关重要。

### [反应](https://news.ycombinator.com/item?id=39916102)

- 白宫正在考虑为月球引入一个独特的时区--协调月球时间（CLT），以减少对地球时钟的依赖，并有可能利用原子钟来记录天体事件。
- 关于重新命名协调世界时（UTC）和建立世界时间标准、探索狭义相对论和广义相对论中的时间膨胀、影响太空旅行和引力相互作用等问题的讨论正在进行之中。
- 要理解时间膨胀、重力和光速等复杂概念，关键在于在不同参照系之间实现时间标准化、调整物理常数以及月球活动中精确计时的必要性等方面的挑战。

## [MOnSter 6502：MOS 6502 晶体管级复制品](https://monster6502.com/)

- MOnSter 6502 项目旨在复制 MOS 6502 微处理器，该处理器因其在重要计算机系统中的作用而闻名于世。
- 复制品由 3218 个晶体管和 1019 个电阻器等 4769 个元件组成，与原始芯片相比，其运行速度有所降低，并包含 LED 等新功能。
- 尽管还不能购买，但该项目已进行了约 8 年，正准备在 2023 年全面公开亮相，并在加利福尼亚州森尼韦尔的邪恶疯狂科学家实验室进行演示。

- MOnSter 6502 是 MOS 6502 微处理器的晶体管级工作复制品，由于电容增大，运行速度较慢，凸显了早期 CPU 中较老的计算机技术和晶体管的使用。
- 讨论围绕将 6502 芯片集成到苹果 ][ 等经典计算机中展开，重点是克服内存和视频输出障碍，并讨论调整时钟速度、显示器和模拟处理器。
- 该项目被视为艺术与历史技术进步的融合，展示了创意与历史意义的独特结合。
