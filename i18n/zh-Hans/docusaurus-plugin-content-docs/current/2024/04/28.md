- 此外，还提到了一个将 SVGs 编码为数据 URI 的网站，该网站获得了积极的反馈，并推荐了用于编辑的 BoxySVG 和用于根据描述生成 SVGs 的 ChatGPT，这引发了人们对人工智能生成图像与传统 SVGs 编辑工具的不同看法。

## [推理中大型语言模型的局限性](https://www.strangeloopcanon.com/p/what-can-llms-never-do)

- 大型语言模型（LLM）在复杂任务中表现出色，但在简单问题理解、文字游戏和长期推理方面却举步维艰，这表明人工智能模型的可靠性还有待提高。
- 建议结合人类智能、神经网络中的结构化记忆以及解决 "目标漂移 "等问题，以提高 LLM 在非常规任务中的表现。
- 强调需要了解人工智能的失效模式，提高可靠性，应对数据集的泛化和状态依赖性等挑战，以推进人工通用智能（AGI）和有效的训练技术。

### [反应](https://news.ycombinator.com/item?id=40179232)

- 大型语言模型（LLM）因其在单词拼写、逻辑推理和编程等任务中的缺陷而受到批评，使人们对其实现人工通用智能（AGI）的潜力产生怀疑。
- 虽然法律硕士在特定领域表现出熟练的能力，但他们在处理复杂任务和展示独立推理能力方面面临挑战。
- 重点是通过更好的提示、解决问题的培训和认识到他们的制约因素来提高 LLM 的性能，同时还要解决人工智能道德发展中必不可少的偏见、价值观和人为监督等问题。

## [Atari 2600 MovieCart：全彩电影和音频](https://github.com/lodefmode/moviecart)

- MovieCart 软件和硬件使用户能够为 Atari 2600 制作全长彩色电影和音频盒，分辨率为 80 x 262，音频为 4 位单声道。
- MovieCart 于 1977 年推出，在 Tindie 上提供预购工具包，允许以每秒 60 帧的速度控制亮度、音量和穿梭位置。
- MovieCart 上的每个字段都需要 2.5K 的数据，一部两小时的电影需要 50 多万个盒式磁带，并为各种用途提供了合理使用版权免责声明。

### [反应](https://news.ycombinator.com/item?id=40179553)

- Github 用户正在讨论在 Atari 2600 游戏机上使用全长彩色电影和音频盒，考虑为复古机器增加更多计算能力。
- 项目包括将 ARM 芯片集成到盒式计算机中，以及在老式系统上运行现代程序，展示复古计算的创新理念。
- 社区对一种潜在的压缩技术感到兴奋，这种技术可以在一个盒式磁带上存储整个电影资料库，从而促进这一领域的创造和创新。

## [美国国税局将决定更新直接报税网站](https://www.washingtonpost.com/business/2024/04/26/irs-direct-file/)

- 拜登政府宣布 "直接申报"（Direct File）免费报税网站取得成功，该网站未超出预算，并受到超过 140,000 个家庭的欢迎，他们在该网站上提交了 2023 年的报税表。
- 国税局在网站开发上投入了 1050 万美元，在运营成本上投入了 240 万美元，因此处理每份报税表的费用很低，并计划在今年春季决定是否延续该计划。
- 包括报税公司和共和党人在内的批评者对此表示担忧，而 GSA 报告称，大多数 Direct File 用户对该网站给予了积极评价。

### [反应](https://news.ycombinator.com/item?id=40182271)

- 讨论深入探讨了美国税收制度的复杂性，涉及隐私问题、政府参与报税及其对低收入家庭的影响。
- 与会者就简化税法、政府获取财务数据以及简化和公开报税程序的必要性交换了各种观点。
- 辩论探讨了利益冲突、金钱对税收政策的影响、国税局资金不足、免费报税网站的好处、私营报税公司的作用以及政府管理的报税系统的有效性。

## [让我们来填空：变形金刚的隐藏计算。](https://arxiv.org/abs/2404.15758)

- 正如研究中揭示的那样，变换器可以利用无意义的填充标记来提高它们在具有挑战性的算法任务中的表现。
- 这项研究解决了学习使用填充标记的困难，并提出了对大型语言模型可能进行隐蔽计算的担忧。
- 通过提供一个理论框架，本文确定了填充代币带来优势的情况，并提出无论具体内容如何，额外代币都能带来计算优势。

### [反应](https://news.ycombinator.com/item?id=40182695)

- 论文深入探讨了 Transformer 语言模型中的隐藏计算，强调了填充标记在影响模型性能方面的作用。
- 尽管目前的模型难以有效利用填充标记，但增加动词和整合填充标记有可能提高模型的推理和计算能力。
- 这项研究还引发了人们对与填充令牌使用相关的后门攻击的担忧，突出了正在进行的研究对于增强人工智能中的变形模型和应对未来潜在进步的重要意义。
