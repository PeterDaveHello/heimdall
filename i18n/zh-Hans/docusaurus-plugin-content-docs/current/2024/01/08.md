# 2024-01-08

## [介绍 Nino：简化工作任务的一体化应用程序](https://nino.app)

- Nino 是一款新的应用程序，旨在解决知识工作者应用程序过载的问题。
- 该应用程序将各种功能整合到一个平台中，提供 18 个模块，用于数据库管理、合成和通信等任务。
- Nino 优先考虑整合、离线模式以提高性能，并计划引入端到端加密以增强隐私和安全性。此外，用户还可以在自定义域或免费子域上发布作品。开发商积极寻求用户反馈，以进一步完善平台。

### [反应](https://news.ycombinator.com/item?id=38901504)

- Nino 应用程序已经推出，旨在通过将多种功能整合到一个平台上，为知识工作者简化超负荷的应用程序。
- 对 Nino 的反馈包括对可扩展性、数据库系统、用户体验、数据所有权和自托管选项的讨论。
- 其他主题还包括实时协作工具、为不同需求寻找合适的工具、应用程序的技术栈和架构，以及潜在的改进功能。

## [GitUI：用于高效版本控制的快速、可移植的 git GUI](https://github.com/extrawurst/gitui)

- GitUI 是一款基于终端的工具，为各种 git 任务提供友好的用户界面，包括索引、提交、差异、存储、归咎和日志记录。
- 它优先考虑速度、可移植性和开源性，与其他流行的 git GUI 相比，它在速度和内存使用方面的基准测试中表现更好。
- 1.0 版的路线图包括可视化分支结构和交互式 rebase 等功能，但也有不足之处，如缺乏对稀疏版本库、GPG 签名和 git-lfs 的支持。
- 为不同平台提供安装说明，并提供颜色主题和按键绑定的定制选项。
- 该项目鼓励支持和赞助，并从 lazygit、tig 和 GitUp 等工具中汲取灵感。

### [反应](https://news.ycombinator.com/item?id=38905019)

- 文章探讨了各种 Git 用户界面（UI）及其功能，以及用户对不同 GUI 工具的看法和体验。
- 一些用户喜欢图形用户界面，因为它易于使用、方便快捷，而另一些用户则认为命令行界面速度快、效率高。
- 我们将 GitUI 和 lazygit 等基于 Rust 的用户界面工具与 Magit、Fork 和 SourceTree 等其他流行工具进行了比较，对它们的实用性和必要性褒贬不一。

## [改版后的 HTMX Playground：在浏览器中体验 HTMX](https://lassebomh.github.io/htmx-playground/)

- 作者恢复了一个旧项目，允许用户在浏览器中尝试使用 HTMX。
- 该项目拥有一个基于 Django 的模拟服务器的代码沙盒，其设计与框架无关。
- 目前已知的一些问题包括移动支持有限、使用王牌编辑器而非摩纳哥编辑器以及错误输出不足。作者欢迎反馈意见以改进该项目。

### [反应](https://news.ycombinator.com/item?id=38906989)

- 作者开发了 HTMX Playground，这是一个在浏览器中探索 HTMX 的代码沙盒。
- 游戏场地包括一个模拟服务器，其灵感来自 Django，但与框架无关，允许用户在不编写专门代码的情况下尝试使用 HTMX。
- 虽然游戏场为测试和提供反馈提供了一个便捷的平台，但它对移动设备的支持有限，而且王牌编辑器还存在一些问题。

## [英国国家医疗服务系统调查 Palantir 的影响者活动是否违反合同](https://goodlawproject.org/nhs-to-investigate-palantir-influencer-campaign-as-possible-contract-breach/)

- 国家医疗服务系统正在调查 Palantir 是否违反了合同，在未经授权的情况下开展了秘密的影响者活动。
- Palantir 聘请公关和营销机构与社交媒体上有影响力的人联系，并向他们提供报酬，让他们在未经同意的情况下使用英国国家医疗服务系统（NHS）的名称和品牌，宣传其运营联邦数据平台的合同。

### [反应](https://news.ycombinator.com/item?id=38902983)

- 讨论围绕着数据分析公司 Palantir 与英国国家医疗服务系统（NHS）的潜在违约事件展开，引发了人们对英国国家医疗服务系统的选择和创新能力的质疑。
- 重点介绍了国家医疗服务体系不断增长的成本和支出，并与其他国家的医疗支出进行了比较。
- 报告提出了有关保守党政府的参与、Palantir 的做法、人工智能技术、外包合同、隐私、监控和预测性警务等方面的道德问题。

## [小脑在运动控制和认知增强中的作用](https://sarahconstantin.substack.com/p/what-does-the-cerebellum-do-anyway)

- 文章探讨了小脑在控制运动中的作用及其在运动技能中的意义。
- 它强调了小脑的独特特征，如普金杰细胞，以及它们如何有助于提高认知能力。
- 这项研究挑战了传统的学习概念，证明学习可以发生在小脑的单个细胞中。
- 难以准确估计距离和速度是小脑损伤的常见症状。
- 小脑受损的患者在完成需要转换注意力的任务时也很吃力。
- 小脑参与预期、预测和生成世界的预测模型。
- 它在快速预测行动方面起着至关重要的作用，并与大脑中参与认知过程的各个部分相连。
- 小脑受损会导致行动笨拙和不协调。
- 支持具身认知的概念，即认知功能与感觉运动功能密切相关。
- 文章最后提到了与具身认知有关的思维失调和普遍小脑转换理论的观点。

### [反应](https://news.ycombinator.com/item?id=38905898)

- 文章探讨了小脑在大脑中的功能和重要性，特别是它在协调、语言和运动学习方面的作用。
- 它讨论了如何使用隐喻和类比来理解大脑的复杂性，以及人工智能如何帮助释放大脑的全部潜能。
- 重点介绍了理解大脑所面临的挑战，尤其是与人工智能工程相比。

## [革新 VAX 架构：使用 FPGA 板的新实施方案](https://mail-index.netbsd.org/port-vax/2021/07/03/msg003899.html)

- 作者利用低端 FPGA 板成功开发并实施了一种新的 VAX（虚拟地址扩展）架构。

- 对话深入探讨了芯片架构、Apple Silicon 的性能、使用分布式处理器编程以及 VAX 系统的历史和所有权等话题。

## [未能采取行动：科学不端行为的后果与责任](https://statmodeling.stat.columbia.edu/2024/01/07/french-bio-lab-research-scandal/)

- 法国国家科学研究中心-巴黎索邦大学（CNRS-Université Sorbonne Paris Nord）的一个化学实验室被发现有多篇科学文章的数据被篡改和回收。
- 国家科学研究中心没有收回文章，而是要求更正，并对负有责任的科学家处以停职一个月的处罚。
- 作者认为，应及时调查研究不当行为，透明地报告研究结果，立即撤回存在严重错误的文章，并归还用于欺诈性研究的研究经费。他们强调了未能有效处理不当行为的负面影响，如阻碍科学进步、削弱公众信任以及使不诚实的科学家与诚实的科学家竞争。保护涉及不当行为的研究人员的机构受到了批评。

### [反应](https://news.ycombinator.com/item?id=38903145)

- 这些课文讨论了一系列主题，如骗局、欺诈、剽窃、功能失调的团队、操纵和研究不当行为。
- 他们强调了识别和避免骗局的重要性，以及剽窃和研究欺诈的后果。
- 此外，还探讨了公司道德行为的重要性，以及人类易受操纵性和被骗正常化的倾向。文章还涉及意识形态、制度和机构的挑战和缺陷。

## [Motūrus OS：适用于 x64 虚拟机的 Rust 优先操作系统，旨在实现简单性、安全性和性能](https://github.com/moturus/motor-os)

- Motūrus OS 是专为基于云的虚拟机工作负载设计的全新操作系统。
- 它的目标是比 Linux 更简单、更安全，同时达到相当或更好的性能。
- Motūrus OS 由 Rust 构建，目前支持基于 x64 KVM 的虚拟机，并提供微内核调度、基本文件系统、网络功能和简单的 httpd 等功能，但它仍处于开发阶段，还不适合用于生产。

### [反应](https://news.ycombinator.com/item?id=38907568)

- 讨论围绕操作系统展开，涉及 MotorOS 的开发和取代 Linux 所面临的挑战等话题。
- 此外，还讨论了 SteamDeck 游戏机和基于 Unix 的系统的安全性。
- 讨论揭示了不同操作系统的设计选择和安全影响，并强调了编程语言 Rust 在操作系统开发中取得的成功。

## [纽约时报》就侵犯版权和不正当竞争问题起诉微软和 OpenAI](https://katedowninglaw.com/2024/01/06/the-new-york-times-launches-a-very-strong-case-against-microsoft-and-openai/)

- 纽约时报》起诉微软和 OpenAI，称其侵犯版权、商标稀释和不正当竞争。
- 泰晤士报》称，微软和 OpenAI 利用其内容训练语言模型，并推出了转载《泰晤士报》内容的新闻服务，但未适当注明出处。
- 该诉讼提供了语言模型从《泰晤士报》内容中生成相同或不准确信息的证据，为一场重大的以人工智能为重点的法律纠纷奠定了基础，并有可能影响未来的联邦立法。

### [反应](https://news.ycombinator.com/item?id=38900197)

- 争论的焦点是 OpenAI 的 GPT 等人工智能模型在生成内容时缺乏适当的引用和归因，从而引发了人们对新闻文章中错误引用和缺乏参考文献的担忧。
- 讨论强调了寻找可靠来源的挑战、个人来源的重要性以及引用和来源对学术诚信的重要意义。
- 此外，还讨论了与搜索引擎使用受版权保护的材料有关的法律问题、人工智能生成的内容与人类撰写的内容相比是否值得信赖，以及《纽约时报》与 OpenAI 之间正在进行的诉讼，强调了人工智能生成的内容及其法律影响的复杂性和多面性。
