# 2024-01-25

## [Zed 代码编辑器实现开源，强调协作性](https://zed.dev/blog/zed-is-now-open-source)

- 代码编辑器 Zed 已转型为开源项目，以增强产品功能并与编程社区合作。
- Zed 现已在 GitHub 上发布，采用 Copyleft 和 Apache 2 许可，允许开发人员访问和修改代码。
- 该公司计划推出实时协作编码功能 Fireside Hacks，同时通过提供集成服务和对私人使用特定功能收费来保持可持续的商业模式。他们鼓励社区贡献，并提供参与指南。

### [反应](https://news.ycombinator.com/item?id=39119835)

- 讨论涉及各种代码编辑器，包括 Zed、Visual Studio Code (VSC)、Emacs 和其他替代工具。
- 用户分享有关提高性能、协作、自托管聊天服务器、启动时间和软件包管理的经验和建议。
- 对话涉及许可、与不同操作系统的兼容性、协作编辑选项的可用性，并强调了实时协作和效率对结对编程的重要性。

## [OpenAI 违背公开重要文件的承诺，引发关注](https://www.wired.com/story/openai-scrapped-promise-disclose-key-documents/)

- 非营利性人工智能研究实验室 OpenAI 违背了向公众披露重要文件的承诺，引发了人们对其透明度的担忧。
- 该公司成立的目的是让社会参与人工智能的开发，但当被要求提供管理文件和财务报表时，OpenAI 拒绝提供。
- 批评者认为，增加信息披露可以提高客户忠诚度和监管部门的信任度，而获取这些文件可以让人们深入了解 OpenAI 的漏洞、它与微软等利益相关者的关系，以及为稳定公司结构所做的任何修订。

### [反应](https://news.ycombinator.com/item?id=39121521)

- OpenAI 因其缺乏透明度和转向盈利而饱受批评，引发了人们对其造福人类承诺的担忧。
- 与会者对 OpenAI 的行动表示失望和怀疑，质疑其诚信度。
- 对话强调了人工通用智能（AGI）发展的影响、科技行业的利益驱动，以及对人工智能进行监管和公开讨论的必要性。

## [了解现代筹资：介绍开源初创企业筹资模拟器](https://www.fundingsimulator.com/)

- 作者开发了一个开源工具，帮助创始人更好地了解现代筹款以及他们可能遇到的潜在稀释问题。
- 该工具简化了了解初创企业融资情况的过程，允许用户添加付款后保险箱、定价回合和员工期权，以了解对所有权的影响。
- 作者认识到初创企业融资的复杂性，包括不同类型的保险箱、期权分配、按比例权利和最惠国条款，并欢迎工具用户提供反馈意见。

### [反应](https://news.ycombinator.com/item?id=39120647)

- 讨论围绕 Fundingsimulator.com 展开，这是一个帮助创始人了解筹资复杂性和潜在稀释的工具。
- 用户可就工具的功能提供反馈意见，并提出改进建议。
- 其他讨论还包括初创企业融资的挑战和益处、风险投资在企业成长不同阶段的相关性，以及技能和向成功创始人学习的重要性。此外，还探讨了初创企业所有权和财务的复杂性，以及联合创始人的潜在优势。

## [阿拉斯加航空公司首席执行官：在 737 Max 型飞机上发现松动的螺栓](https://www.nbcnews.com/business/business-news/alaska-airlines-found-more-loose-bolts-boeing-737-max-9-ceo-says-rcna135316)

- 阿拉斯加航空公司的波音 737 Max 9 飞机在最近险些发生灾难事件后，在内部检查中发现螺栓松动。
- 美国联邦航空管理局停飞了所有波音 Max 9 飞机，并启动了安全调查，导致阿拉斯加航空公司的航班取消和重新安排。
- 阿拉斯加航空公司首席执行官本-米尼库奇认为波音公司有责任改进其质量控制系统，该公司正派遣审计人员对生产线上的飞机进行检查。

### [反应](https://news.ycombinator.com/item?id=39115878)

- 阿拉斯加航空公司首席执行官已派出一个审计小组，对波音公司 Max 飞机的质量控制问题进行调查。
- 美国联邦航空局正在检查波音公司的质量流程，重点是门和螺栓，以及质量体系中的其他潜在问题。
- 关于联邦航空局对波音公司的尊重态度、政府收购该公司的可能性、独立审计和质量控制的重要性、工程师担任首席执行官的合适性、工会在安全方面的作用以及阿拉斯加航空公司和其他航空公司对联邦航空局的信任等问题，都存在争论。

## [制作 8kB 动画电影：将故事和动画与技术限制相结合](https://www.ctrl-alt-test.fr/2024/how-we-made-an-animated-movie-in-8kb)

- 此外，还讨论了时空旅行、人工智能以及压缩和动画制作 2D 和 3D 视觉效果所面临的挑战等主题。

## [破解小米空气净化器：利用 RFID 芯片黑客程序重置过滤器警告](https://unethical.info/2024/01/24/hacking-my-air-purifier/)

- 作者讲述了他们使用小米空气净化器的经历，这款净化器上有一个无法拆卸更换滤网的警告。
- 他们发现了一种方法，可以入侵过滤器上的 RFID 芯片，并使用从标签 UID 生成的密码重置过滤器。
- 文中提供了如何使用 NFC 手机重置过滤器的说明，以及兼容设备的列表和尝试过的成功读者的评论。

### [反应](https://news.ycombinator.com/item?id=39115837)

- 用户讨论了对空气净化器 DRM（数字版权管理）的不满，并分享了黑客绕过设备限制的经验。
- 有人担心，如果这种黑客攻击流行起来，假冒过滤器可能会增加。
- 此外，还提出了其他解决方案，如使用压差传感器代替 DRM，并呼吁废除 DMCA（《数字千年版权法》）和消费者的维修权。

## [国税局免费文件：与可信赖的合作伙伴一起免费报税](https://www.irs.gov/newsroom/irs-free-file-is-now-available-for-the-2024-filing-season)

- 国税局免费报税计划为符合条件的纳税人提供了免费使用由值得信赖的国税局合作伙伴提供的软件进行联邦报税的机会。
- 该计划的资格由年龄、收入和州居住地决定，大多数纳税人的调整后总收入限额为 79,000 美元。
- 收入上限较高的纳税人也可通过该计划提供的 "可填写表格 "选项免费申报。
- 该计划确保纳税人信息的安全，合作公司不得将其用于报税以外的任何目的。
- 提供移动访问，纳税人可使用智能手机或平板电脑报税。

### [反应](https://news.ycombinator.com/item?id=39119668)

- 讨论探讨了报税的不同方面，如可供选择的方案、政府软件和在线报税。
- 报告还涉及报税公司的作用、房东和软件工程师的扣除额、免税账户和递延纳税账户以及税法的复杂性。
- 对话强调了教育的必要性，比较了其他国家的税收制度，并倡导建立一个更简单、更高效的报税系统。

## [自体免疫疾病疗法的进展带来希望](https://www.nature.com/articles/d41586-024-00169-7)

- 科学家们在开发糖尿病、红斑狼疮和多发性硬化症等自身免疫性疾病的疗法方面正在取得进展。
- 有希望的方法包括：给不听话的细胞注射抗原、选择性地消灭有问题的细胞以及引入工程免疫细胞。
- 名为 "navacims "的纳米粒子和肝脏靶向抗原疗法分别在关闭自身免疫反应和逆转多发性硬化症症状方面显示出希望。此外，涉及嵌合自身抗体受体T细胞的疗法也显示出对红斑狼疮和其他疾病患者的显著改善。

### [反应](https://news.ycombinator.com/item?id=39116387)

- 科学家们正在探索使用抗原来训练不听话的细胞进行攻击，希望以此来治疗自身免疫性疾病，但每个人都对这种方法的有效性感到担忧。
- 一些自身免疫性疾病患者对更具针对性的治疗充满希望。
- 讨论围绕类风湿性关节炎药物增加感染和癌症风险、基于安全性和潜在副作用对某些药物的偏好、禁食作为一种潜在的治疗方法、肥大细胞疾病和过敏的其他治疗方法、当前过敏治疗方法的有效性以及饮食干预在治疗自身免疫性疾病中的有效性展开。
