# 2024-01-19

## [Ello 的消亡无广告社交网络令人失望的结局](https://waxy.org/2024/01/the-quiet-death-of-ellos-big-dreams/)

- 旨在成为 Facebook 无广告替代品的社交网络 Ello 突然关闭，让用户大失所望。
- 尽管 Ello 最初致力于隐私和用户数据保护，但在寻求发展和货币化资金的同时，它在维护自身价值方面也面临着挑战。
- 所有权和管理权的变更，再加上财务困难，最终导致该平台突然关闭，没有给用户保留其内容的机会。

### [反应](https://news.ycombinator.com/item?id=39043871)

- 文章和评论主题涉及与社交网络平台有关的各种话题，包括融资模式、货币化挑战和风险投资的影响。
- 讨论了成功平台（Ello、Mastodon）和失败平台（Bluesky）的例子，提供了现实世界的背景。
- 讨论深入探讨了用户增长的重要性、创始人的控制权以及在商业企业中坚持原则，同时探讨了不同融资和货币化方法的风险和挑战。外部投资者的负面影响和建立成功社交网络的困难也得到了认可。总之，对话揭示了社交媒体平台可持续性和财务模式的复杂性和不确定性。

## [通知团队成功迁移 PostgreSQL 数据库，停机时间仅为 11 秒](https://gds.blog.gov.uk/2024/01/17/how-we-migrated-our-postgresql-database-with-11-seconds-downtime/)

- GOV.UK Notify 团队使用 AWS 数据库迁移服务 (DMS) 成功迁移了 PostgreSQL 数据库，停机时间最短。
- 迁移过程包括设置 DMS 实例、创建目标数据库、执行完整的数据加载以及实施持续复制。
- 迁移过程中的停机时间仅为 11 秒，超过了 5 分钟以内的目标。该团队计划今后将其应用程序迁移到 AWS Elastic Container Service (ECS)。

### [反应](https://news.ycombinator.com/item?id=39048317)

- 讨论涉及 AWS RDS 蓝绿部署和数据库迁移中数据库降级的限制等主题。
- 还讨论了数据库迁移框架面临的挑战和 AWS 数据迁移服务（DMS）存在的问题。
- 讨论还涉及政府云服务、数据安全和主权问题，以及将 IT 服务外包给外国公司（尤其是公共部门）的利弊。

## [AlphaFold 能够发现新型抗抑郁迷幻药](https://www.nature.com/articles/d41586-024-00130-8)

- 蛋白质结构预测工具 AlphaFold 发现了数十万个潜在的迷幻分子，可用于开发新型抗抑郁药物。
- 最近的研究表明，AlphaFold 预测对药物发现很有价值，但仅靠预测结构有局限性，往往需要更详细的实验模型。
- DeepMind 分拆出来的 Isomorphic Labs 正在利用 AlphaFold 进行药物发现工作，并已与制药公司达成交易，这表明 AlphaFold 等工具有可能极大地帮助寻找新药。

### [反应](https://news.ycombinator.com/item?id=39049703)

- DeepMind开发的人工智能系统AlphaFold有可能通过识别与迷幻药中的5HT2A受体结合的分子，彻底改变药物的发现。
- AlphaFold 的模型可以准确预测与 GPCR 结合的分子，从而提高为 GPCR 寻找选择性配体的效率，并为精神疾病研究提供帮助。
- 讨论还涉及靶向 GPCR 在医学中的意义、对 AlphaFold 预测可靠性的担忧以及与使用研究化学品相关的潜在风险。

- 讨论内容包括调查数据的准确性、与不同浏览器和操作系统的兼容性、潜在的安全风险以及 WebGPU 与其他图形 API 相比的局限性。
- 一些用户称赞 WebGPU 的便利性和潜力，而另一些用户则提到其局限性以及正在对不同 GPU 进行的测试。
- 讨论内容包括将 WebGPU 用于图形和计算、潜在的性能优势，以及开发网络应用程序以替代本地代码。
- 其中提到 PixiJS 是一种优化性能的工具，虚幻引擎 5 支持 WebGPU。

## [美国机床工业的衰落：原因与后果](https://www.construction-physics.com/p/what-happened-to-the-us-machine-tool)
