# 2024-01-07

## [飞行途中发生面板事故后，美国联邦航空局下令对 170 多架波音 737 Max 9 进行检查](https://www.cnbc.com/2024/01/06/boeing-737-max-9-grounding-after-alaska-airlines-door-blows-midflight.html)

- 阿拉斯加航空公司一架波音 737 Max 9 飞机在飞行途中一块面板脱落，美国联邦航空局已强制要求 170 多架波音 737 Max 9 飞机停飞接受检查。
- 作为一项预防措施，阿拉斯加航空公司和美国联合航空公司已停飞了其全部波音 737 Max 9 飞机。
- 目前，美国国家运输安全委员会（NTSB）正在对这起事故进行调查，所幸没有人员伤亡报告。这起事故再次引发了人们对波音 737 Max 潜在结构故障的担忧，该机型在 2019 年发生两起致命坠机事故后已面临严格审查。

### [反应](https://news.ycombinator.com/item?id=38893909)

- 讨论涉及与航空安全有关的多个话题，包括波音 737 Max 9 飞机停飞和阿拉斯加航空公司的安全问题。
- 报告还强调航空业需要更严格的监管，并批评波音公司忽视安全和必要的更新。
- 讨论范围超出了航空业，还包括有关政府与企业关系、军费开支、公司文化以及某些企业衰落的争论。

## [美国国税局试行免费报税系统，面临反对和支持分歧](https://www.nytimes.com/2024/01/05/your-money/irs-tax-filing-free-online.html)

- 美国国税局正在测试一个免费报税系统，使 12 个州的居民能够直接在线提交联邦报税表，不收取任何费用。
- 这项试点计划是为中低收入纳税人设计的，他们的申报材料并不复杂，旨在简化申报程序。
- 商业报税公司反对这一举措，而国会两党议员对支持这一举措持不同意见。国税局在决定是否将直接报税系统扩大到更多的纳税人之前，将公布一份有关试验的公开报告。

### [反应](https://news.ycombinator.com/item?id=38892044)

- 美国的税收制度复杂而低效，引发了关于简化和改进必要性的讨论。
- 辩论的重点是政府对报税的参与、公共领域源代码的使用以及与其他国家税收系统的比较。
- 分析内容包括政府服务的效率、预算分配、私营实体的作用，以及对税收遵从、国税局、数据安全和纳税人教育的看法。

## [探索浏览器内的代码游乐场：Codapi 和轻量级 WASI 运行时的力量](https://antonz.org/in-browser-code-playgrounds/)

- 目前正在讨论浏览器内的代码游戏场，特别是与名为 Codapi 的开源工具有关的问题。
- 语言运行时可编译成 WebAssembly 系统接口（WASI）二进制文件，从而可在浏览器中使用轻量级 WASI 运行时运行。

- 提供了如何使用游戏场地的说明，并强调了在浏览器中运行代码的好处。

### [反应](https://news.ycombinator.com/item?id=38891177)

- 文章探讨了不同的浏览器内代码游戏场，这些游戏场使用户能够直接在网络浏览器中编写和执行代码。
- 例如，Scheme 语言的游戏场和 Python 游戏场都具有针对初学者编程教育的功能。
- 文章强调了在浏览器中运行具有大量执行时间或无限循环的代码的困难，并讨论了网络应用程序游乐场为演示目的展示沙盒产品的潜力。

## [Swieros 简介：多功能 CPU 仿真器、C 编译器和操作系统](https://github.com/rswier/swieros)

- Swieros 是一个基于 xv6 的轻量级快速类 Unix 内核、编译器和用户平台。
- 它旨在提供一个寓教于乐的研究环境。
- Swieros 包括一个支持用户/监管模式和虚拟内存的虚拟 CPU，以及一个用于即时编译和执行的快速编译器。
- Swieros 的其他功能还包括网络客户端和服务器、远程 OpenGL、图形用户界面应用程序等。
- 它可以在 Windows 和 Linux 操作系统上运行。

### [反应](https://news.ycombinator.com/item?id=38890540)

- 摘要讨论了 GitHub 上的一个项目，该项目由 CPU 仿真器、C 编译器和操作系统组成，赢得了用户的积极关注和赞誉。

- 总之，该项目被视为一项激情项目和宝贵的学习教育工具。

- 文章还深入探讨了公费广播和电视的道德困境和有效性，重点关注德国的媒体格局。文章涉及对极右政党、宣传和媒体两极分化的担忧。文章还讨论了各种政治团体参与德国媒体的情况，包括对左翼宣传和假新闻的指控。
