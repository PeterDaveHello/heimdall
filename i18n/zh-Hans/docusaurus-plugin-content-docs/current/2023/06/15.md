## 2023 年 6 月 15 日

### [纽约州参议院通过禁止非竞争性协议的规定](https://ogletree.com/insights/new-york-state-senate-passes-prohibitions-on-non-competes/)

- 纽约州参议院已经批准了两项法案，可以禁止或限制非竞争协议在该州的使用。
- S3100A 号法案将禁止雇主与雇员、独立承包商或其他为雇主工作的人签订非竞争协议。违反者将被处以最高 1 万美元的赔偿金。
- 第 S6748 号法案旨在防止建立垄断和限制贸易的行为。它将要求雇主有一个 "善意的基础 "来相信非竞争协议是可执行的，并且还将扩大非竞争协议的定义，以包括某些其他类型的协议。
- 这两项法案仍需在纽约州议会通过，并由州长 Kathy Hochul 批准才能成为法律。

#### [行业反应](http://news.ycombinator.com/item?id=36329483)

- 纽约州参议院通过了对非竞争协议的禁令，这是一个重要的进展。
- 这项禁令主要影响到金融和技术行业，特别是高频交易(HFT)公司和雇用软件专业人员的自营交易商店。
- 新法规定合同性非竞争协议无效，但不涉及限制与前同事工作的非关联条款。

### [我在 21 小时内启动了 Linux 293k 次](https://rwmj.wordpress.com/2023/06/14/i-booted-linux-292612-times/)

- Linux 6.4 中的一个错误导致它在启动时挂起，但只在大约 1000 次启动中出现。
- 作者在用 libguestfs 进行测试时注意到这个 bug，并决定进一步调查。
- 通过一个被称为 "分割 "的过程，作者通过运行 Linux 292,612 次，确定了引入该 bug 的具体提交。
- 这个帖子很特别，因为它强调了在像 Linux 这样复杂的软件系统中定位和修复一个 bug 所需要的坚韧和努力。

#### [行业反应](http://news.ycombinator.com/item?id=36325986)

- 一位开发者分享了他们在大型集成测试套件中调试一个罕见故障的经验，他们在一夜之间触发了大量的运行，并使用贝叶斯统计学来缩小故障点。
- 导致系统崩溃的 Linux 内核中的 bug 是通过涉及超过 292,000 个不同提交的分叉过程发现的。
- 该错误报告和 Linux 内核邮件列表上的讨论突出了软件开发中持久性、彻底测试和版本控制的重要性，同时讨论了寻找错误的各种工具和技术。

### [Reddit 停电将继续](https://old.reddit.com/r/ModCoord/comments/148ks6u/indefinite_blackout_next_steps_polling_your/)

- Reddit 宣布了一项政策变化，将影响第三方 Reddit 应用程序，只留下官方移动应用程序作为选择。
- 对此，许多子版块已经变黑以示抗议，包括 r/aww、r/music、r/videos 和 r/futurology 等热门版块。

- Reddit 首席执行官史蒂夫-赫夫曼(Steve Huffman)声称，停电没有对收入产生重大影响，但许多子版块都承诺在他们的要求得到满足之前保持停电。

### [GPT-4 的本地 JSON 输出](https://yonom.substack.com/p/native-json-output-from-gpt-4)

- OpenAI 为他们的 GPT 模型发布了一个名为函数调用的新功能，允许他们调用一个函数并返回 JSON 数据，而不是仅仅一个字符串。
- 这项功能目前适用于聊天模型 gpt-3.5-turbo-0613 和 gpt-4-0613。
- 一个名为 Recipe Creator 的演示应用程序展示了如何使用这个新功能来生成 JSON 响应，以提供烹饪菜肴的说明。
- 通过使用 JSON 模式，开发者可以定义他们想从 GPT 模型中接收的 JSON 数据的结构。
- 这个新功能减少了对提示工程的需求，并使其更容易生成正确的 JSON 输出。
- 它还允许使用 GPT 模型作为应用程序和无代码原型工具的智能后端 API 的可能性。
- JSON 模式可用于在单个 API 调用中实现更复杂的代理和思维过程。
- OpenAI 的 API 支持各种 JSON 模式功能，允许创建更复杂和高效的代理。
- 虽然不是所有的 JSON 模式功能都被支持，但支持的功能使开发者能够将复杂的策略嵌入到单个 API 调用中。

#### [行业反应](http://news.ycombinator.com/item?id=36330972)

- 对使用 GPT-4 来构建 SQL 查询和总结电子邮件而不进行适当的人工验证表示担忧。
- 担心开发者取消了人工审查，增加了及时注入和妥协代理的风险。
- OpenAI 缺乏关于安全应用开发和 LLM 安全实践的明确文件和教育。

### [总是同样的警告信号](https://www.science.org/content/blog-post/always-same-warning-signs)

- Laronde，一家生物技术公司，在复制去年帮助他们筹集了数亿美元的数据时面临重大问题。
- 该公司的循环-mRNA 技术，被称为 "无尽的 RNA"，被期望有更好的稳定性、分布和更持久的蛋白质表达，但没有人能够让它发挥作用。
- 同事们被拒绝访问原始数据，而管理层则将复制问题归咎于其他人，这与 Theranos 的丑闻类似。
- 警告信号包括只有一个人取得了成功的结果，搪塞合法的问题，以及丢失或隐藏重要数据。
- 这些警告信号不应该被忽视，因为它们表明公司或研究项目中存在潜在的问题，这可能导致经济损失和职业受损。

#### [行业反应](http://news.ycombinator.com/item?id=36325349)

- 与生物技术相比，计算机技术通常被视为更适合于启动资金，因为它更容易理解和评估。
- 生物技术公司承担着更多的研发风险，但根据病人数量和向医疗机构收费的潜力，其市场规模是有保证的。
- 与软件公司中更容易操纵的用户增长数字相比，生物技术中的科学尽职调查更难伪造。

### [血清素促进剂导致大脑功能连接增加](https://www.alphagalileo.org/en-gb/Item-Display/ItemId/234406?returnurl=https://www.alphagalileo.org/en-gb/Item-Display/ItemId/234406)

- 一项发表在《生物精神病学》上的新研究：认知神经科学和神经影像学研究了一种名为普鲁卡必利的 5-羟色胺助推剂对大脑的影响。
- 羟色胺受体，特别是 5-HT4 受体，存在于大脑中调节情绪和认知功能的区域。
- 研究发现，接受普鲁卡必利的参与者显示出大脑中功能连接的增加，特别是在认知网络和参与注意力和信息处理的区域之间。
- 这些发现表明，普鲁卡必利可能有治疗潜力，可以治疗情绪障碍和其他精神疾病中的认知缺陷。
- 该研究增加了越来越多的证据，即针对 5-羟色胺受体的药物可能是改善抑郁症和其他疾病的认知功能的一种新方法。

#### [行业反应](http://news.ycombinator.com/item?id=36321843)

- 一个领域的智力不一定意味着在其他领域的资质。
- 研究表明，5-羟色胺助推器会导致大脑功能连接的增加。
- 必须以谨慎和怀疑的态度对待评论中提出的建议和结论。
