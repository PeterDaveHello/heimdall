## [重新评估邓宁-克鲁格效应：自相关性的解释](https://economicsfromthetopdown.com/2022/04/08/the-dunning-kruger-effect-is-autocorrelation/)

- 邓宁-克鲁格效应（Dunning-Kruger effect）表明，缺乏技能的个人会高估自己的能力，这种效应正在被重新评估为一种由自相关引起的统计假象。
- 重复研究没有发现该效应的证据，因此质疑其作为一种心理现象是否存在。
- 对原始研究的批评突出了统计错误和偏见，强调需要更好地进行科学分析，并认识到邓宁-克鲁格效应的缺陷。

### [反应](https://news.ycombinator.com/item?id=38415252)

- 邓宁-克鲁格效应是一种心理现象，即能力较低的人倾向于高估自己的能力，而能力较高的人则倾向于低估自己。
- 围绕该效应的准确性和解释存在争论和批评，包括对方法缺陷、复制研究、自相关性和社会因素的讨论。
- 其中提到了自我评估的复杂性和冒名顶替综合症的存在，这与邓宁-克鲁格效应有关。

- 这篇文章谈到了获取技术方面的不平等及其对生活质量的影响。

## [嘘用 Haskell 简化 Shell 脚本](https://github.com/luke-clifton/shh)

- Shh 是 Haskell 中的一个库，可实现类似 shell 的编程，旨在取代 shell 脚本。
- 该库提供模拟 shell 环境的函数，并支持重定向、管道、并发和捕获进程输出等功能。

### [反应](https://news.ycombinator.com/item?id=38413064)

- Shh 是一款用 Haskell 编写的 shell 脚本工具，旨在取代大型 bash 脚本。
- 用户建议使用 Xonsh、Turtle、Shelly 和 Rash 等替代工具，以取代 bash 脚本。
- 讨论探讨了 bash 脚本的替代方案，包括 Lisps 和静态类型的函数式语言，以及使用 NixOS 的好处，即便于使用工具和库。

## [FLSHclust算法发现188个新的CRISPR基因，释放基因组编辑潜力](https://www.nature.com/articles/d41586-023-03697-w)

- 研究人员创建了一种名为 FLSHclust 的算法，它可以通过分析基因序列来识别罕见类型的 CRISPR 系统。
- 该算法通过研究数百万个基因组，发现了188个与CRISPR相关的新基因。
- 这些发现为基因组编辑和生物工程的新应用提供了潜力，但要充分了解这些系统及其用途，还需要开展更多的研究。

### [反应](https://news.ycombinator.com/item?id=38415851)

- 新的 CRISPR 系统具有基因组编辑的潜力，为定向基因编辑提供了更多选择，并为新型生物技术提供了潜力。
- 通过基于散列的快速定位敏感聚类算法，现在可以对大型序列数据库进行高效分析，从而发现罕见的生物系统。
- 然而，要充分理解基因编辑的影响，必须应对复杂的监管网络和挑战。

## [介绍 Flix：具有独特功能的函数式编程语言](https://flix.dev/)

- Flix 是一种编程语言，由奥胡斯大学和滑铁卢大学共同开发，开放源代码贡献者也提供了意见。
- 它是一种函数优先语言，从 OCaml、Haskell、Rust 和 Scala 中汲取灵感。
- Flix 提供了一系列功能，包括代数数据类型、模式匹配、可扩展记录、类型类、高级类型、多态效果、基于区域的局部突变、纯度反射和一流的 Datalog 约束。
- 该语言可编译成 JVM 字节码，并有一个用于开发的 Visual Studio Code 插件。
- Flix 的目标是提供其他编程语言所不具备的独特功能组合。

### [反应](https://news.ycombinator.com/item?id=38419263)

- Flix 编程语言的更新包括容错、代数效果和改进的软件包管理。
- 讨论了有争议的设计选择以及在函数式编程中保持平衡的必要性。
- 其中涉及局部突变的使用、函数式编程语言的历史以及纯函数的优势/挑战。
- 作者认为，编码风格和方法可以增强纯函数式语言。
- 还提到了 Rust 等其他语言和 Crumb 等项目。
- 讨论了 Flix 的能力及其与 Netflix 的联系。
- 探讨了使用英语作为编程语言的局限性。
