# 2023-08-31

## [我一半的打字都是自动完成的](https://github.com/eschluntz/compress)

- 在过去的一年里，该用户一直在使用一款工具，该工具可以分析他们的 Slack 消息，并为常用短语生成键盘快捷方式。
- 该工具通过解析 Slack 消息来识别经常重复的短语。
- 该工具可将日常用语转化为易于使用的键盘快捷键，从而为用户提供效率和便利。

### [反应](https://news.ycombinator.com/item?id=37326870)

- 该书的主要重点是通过自动化工具、快捷方式和缩写提高打字效率和生产力。
- 关于打字速度对编码工作效率的重要性，以及在工作和生活中如何在效率和放松之间取得平衡的问题，人们一直在争论不休。
- 探讨了键盘快捷键、备用键盘设置、文本扩展工具和听写系统等各种方法，并讨论了其潜在的优点和缺点。

## [我的种姓](https://www.cs.toronto.edu/~meel/opinions/caste.html)

- 帖子作者的身份是印度社会经济弱势群体 "其他落后阶层"（OBC）的成员。
- 他们强调了学术界代表性不足的问题，特别指出印度一流院校中缺少其他种族的教师。
- 该职位的总体目标是提高对寻求在高等教育中晋升或获得教职的其他落后阶层个人的认识，并为他们提供鼓励。

### [反应](https://news.ycombinator.com/item?id=37323760)

- 这套丛书讨论了印度的种姓制度和种姓歧视，重点是种姓制度对社区的影响及其在教育和就业等社会部门的普遍性。
- 对话深入探讨了平权行动、文化影响等复杂问题，并与种族主义进行了比较，从而提供了一个深入的视角。
- 报告强调了科技公司中可能存在的偏见做法，以及约会和婚姻平台对种姓数据的使用，表明种姓歧视具有广泛的影响。

## [DIY "仿生胰腺 "正在改变糖尿病护理](https://www.nature.com/articles/d41586-023-02648-9)

- 一个由 1 型糖尿病患者组成的社区开发了 DIY 开源软件，用于自动管理血糖水平，强调了个性化比商业系统更重要。
- 倡导者们正在推动美国食品及药物管理局批准一种开源算法，以提高这些 DIY 设备的可及性，但设备制造商之间的合作对于互操作性是必不可少的。
- 据报道，这些开源人工智能算法在管理糖尿病方面的优势凸显了用户选择的价值和开源社区的进步。

### [反应](https://news.ycombinator.com/item?id=37321028)

- 讨论涉及糖尿病管理的多个方面，如 DIY（自助）技术的利用、闭环系统、胰岛素泵使用方面的挑战、改进监测设备的必要性、饮食习惯的影响以及 DIY 社区在引领进步方面的影响。
- 对不同管理策略的潜在利弊进行了思考，强调了持续研究和为糖尿病患者提供支持的重要性。
- 首要重点是发现控制血糖水平和降低糖尿病相关风险的更佳解决方案。

## [微软在 Win11 中使用类似恶意软件的弹出式窗口让人们放弃 Chrome 浏览器](https://www.theverge.com/2023/8/30/23851902/microsoft-bing-popups-windows-11-malware)

- 微软正在通过弹出消息提示 Windows 11 用户将默认搜索引擎从谷歌切换到必应，即使他们正在使用 Chrome 浏览器。
- 这些策略让用户感到沮丧，因为他们认为这些弹出式窗口具有侵扰性，并认为他们不应该在自己购买的操作系统中看到广告。
- 微软已经意识到了这一反弹，并暂时停止了这些通知，同时试图解决这一问题。

### [反应](https://news.ycombinator.com/item?id=37321002)

- 讨论的焦点是用户对 Windows（特别是 Windows 11）的不满，由于弹出窗口、用户账户和程序可见性等问题，促使一些用户考虑使用 Mac 或 Linux 等替代品。
- 在各种系统选择中，Linux 因其易用性和功能性而备受赞誉，但在运行特定游戏方面也有明显的缺点。其他讨论领域包括硬件偏好、软件兼容性和 Windows Subsystem for Linux (WSL) 的局限性。
- 人们普遍对科技产品中的干扰和广告感到不满，并就平台限制、Windows 的未来、微软的行为以及潜在的法律或监管行动展开了辩论。对 Gmail 和谷歌地图 iPhone 应用程序的一些不满也表明了对谷歌产品的潜在不满。

## [Windows 11 系统组件使用默认浏览器打开欧洲链接](https://blogs.windows.com/windows-insider/2023/08/25/announcing-windows-11-insider-preview-build-23531-dev-channel/)

- 微软向开发者频道推出了 Windows 11 Insider 预览版 Build 23531，其中包含安全模式、文件资源管理器、HDR 背景和任务管理器的修复等更新。
- 开始 "菜单和任务栏上的搜索功能仍存在一些已知问题。我们鼓励开发人员下载最新的 Windows Insider SDK 和 NuGet 软件包，以保持更新。

- 其他话题包括在 Edge 上下载 Chrome 浏览器时遇到的挫折、微软对其用户缺乏尊重的看法，以及使用 Linux Mint 等替代选项的积极体验。

## [联合国称，东南亚数十万人被贩运到网上从事诈骗工作](https://www.ohchr.org/en/press-releases/2023/08/hundreds-thousands-trafficked-work-online-scammers-se-asia-says-un-report)

- 联合国人权办公室的一份报告显示，有组织犯罪团伙正迫使数十万东南亚人从事网上犯罪活动，如恋爱投资诈骗、加密货币欺诈和非法赌博。
- 这些犯罪组织侵犯受害者的人权，如威胁、酷刑、性暴力、强迫劳动和任意拘留。由于其隐蔽性，这种网络诈骗贩运的规模很难估计。
- 报告呼吁受影响的国家加强人权、强化治理、实施法治和打击腐败，以此作为应对这些在 COVID-19 大流行期间愈演愈烈的骗局的措施之一。

### [反应](https://news.ycombinator.com/item?id=37320975)

- 网上对话涉及东南亚人口贩运的报告，受害者被迫从事网上诈骗活动，可能涉及当地执法部门和政治网络。
- 讨论延伸到现有的刑事司法实践，重点是美国在预防犯罪、改造罪犯和解决囚犯待遇方面的系统性改进。
- 人工智能在诈骗中的作用受到了严格审查，同时还讨论了 Tinder 和 LinkedIn 等社交媒体平台在减少这些诈骗活动方面的责任。

## [联邦通信委员会拒绝废除要求互联网服务提供商列出每项月费的规定](https://arstechnica.com/tech-policy/2023/08/fcc-says-too-bad-to-isps-complaining-that-listing-every-fee-is-too-hard/)

- 美国联邦通信委员会（FCC）驳回了康卡斯特（Comcast）和其他互联网服务提供商的上诉，要求取消一项规定，要求他们公开所有月费。
- 互联网服务提供商认为，这一要求带来了行政上的挑战和不必要的复杂性，但联邦通信委员会强调，消费者在选择互联网服务时需要有透明度。
- 通过立法，联邦通信委员会将强制执行这些规则，要求互联网服务提供商在宽带标签上注明所有政府规定的费用。

### [反应](https://news.ycombinator.com/item?id=37326806)

- 美国联邦通信委员会（FCC）维持了一项规定，要求互联网服务提供商（ISP）向客户披露所有月费，以提高透明度，方便客户比较不同的服务提供商。
- 支持者认为，这项规定将揭露隐性收费，刺激互联网服务提供商之间的竞争，并使其承担责任。讨论还提出了地方政府征收的转嫁费用是否应被视为税收以及互联网服务提供商是否应将这些费用转嫁给客户的问题。
- 此外，联邦通信委员会还规定，互联网服务提供商必须在广告中预先披露所有费用和收费，以进一步提高透明度，这反映了公众对诚实定价和反对隐性收费的呼声。

## [破解 LG 显示器的 EDID](https://gist.github.com/kj800x/be3001c07c49fdb36970633b0bc6defb)

- 这篇文章讨论了 LG ULTRAGEAR 显示器在 Linux 操作系统上无法正常工作的问题。
- 通过探索扩展显示识别数据 (EDID) 文件并使用十六进制编辑器对其进行修改，可对该问题进行调查。
- 这个问题似乎与 Linux 上的显卡驱动程序有关，而不是显示器本身，这为解决类似问题提供了一个潜在的途径。

### [反应](https://news.ycombinator.com/item?id=37323604)

- 文章深入探讨了与显示器兼容性和在 Linux、macOS 和 Windows 等操作系统上操作扩展显示识别数据 (EDID) 有关的挑战和问题。
- 报告重点介绍了有关 EDID 覆盖、低延迟杜比视界、亮度调整、像素时钟错误和非贴片机的用户体验，并指出了处理 EDID 数据的复杂性和局限性。
- 讨论还揭示了 Linux 硬件兼容性问题以及在该平台上排除故障的细微差别。讨论特别强调了 LG 显示器的 EDID 问题以及不同标准在同一设备上共存的问题。
