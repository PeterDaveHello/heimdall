## [Zoom 条款现在允许对用户内容进行人工智能训练，且无需选择退出](https://explore.zoom.us/en/terms/)

- 这段话是 Zoom Video Communications 与其用户之间的服务条款协议。
- 它涵盖用户权利、服务和软件所有权、付款义务、税务要求、争议解决以及其他法律政策和准则等各个方面。
- 用户一旦接受协议，就必须遵守这些条款。

### [反应](https://news.ycombinator.com/item?id=37021160)

- Zoom 更新的条款和条件允许该公司在没有退出选项的情况下将用户内容用于人工智能培训，这引发了人们对隐私和可信度的担忧。
- 用户将 Zoom 的条款与视频会议替代产品 Jitsi Meet 进行了比较，并就 Jitsi Meet 处理大型电话会议的能力以及 Zoom 的隐私和可靠性问题展开了争论。
- 讨论还涉及视频聊天服务的性能和电池损耗问题、将 Zoom 用于远程医疗服务的潜在隐私问题，以及 Zoom 将客户数据用于开发人工智能模型的影响。对于 Zoom 更新条款的意义，各方意见不一。

## [Postgres 语言服务器](https://github.com/supabase/postgres_lsp)

- 该语言服务器专为 PostgreSQL 设计，为代码编辑器添加了自动完成和转到定义等功能。
- 它通过 libg_query 使用实际的 PostgreSQL 解析器，为开发人员提供了更多信心和能力，以跟上 PostgreSQL 的快速发展。
- pganalyze 和 psteinroe 在 libpg_query 和语言服务器方面的工作功不可没。

### [反应](https://news.ycombinator.com/item?id=37020610)

- Postgres 语言服务器（Postgres Language Server）是一款新工具，它通过使用实际的 Postgres 解析器为 Postgres 添加自动补全和文档等功能，改进了代码编辑器。
- 它旨在增强 SQL 和数据库工具，而现有工具中往往缺乏这种工具。
- 用户表达了对其他功能的兴趣，并讨论了现有工具的局限性。
- 另一个名为 "pg_langserver "的项目旨在为 PostgreSQL 提供类似的语言服务器功能，该项目欢迎各方贡献。
- 与 Postgres 领域其他团队的合作有助于 Postgres 语言服务器的开发。
- 该团队有时间限制，目前无法解决与 Windows 的兼容性问题。
- 他们愿意接受帮助，并建议使用远程 ssh 等替代选项，或者接受 Windows 是二等公民的事实。

## [用 "同行复制 "取代同行评审（2021 年）](https://blog.everydayscientist.com/replace-peer-review-with-peer-replication/)

- 作者批评目前科学出版界的同行评审程序存在缺陷，并提出了一种名为 "同行复制 "的新方法。
- 拟议的系统包括在接受和出版前将预印本发送给其他实验室进行复制。
- 作者认为，这种方法可以鼓励复制，提高科学的完整性，但也承认存在一些挑战，如复制失败后时间和判断力的增加。他们建议先由 "共享评论 "这样的组织进行试验。

### [反应](https://news.ycombinator.com/item?id=37021264)

- Hacker News 上的讨论重点是复制科学研究的挑战和潜在解决方案。
- 一些人认为复制的重要性，以确保研究结果的有效性，而另一些人则强调了实际和资金方面的限制。
- 人们对同行评审在确保研究有效性方面的有效性持怀疑态度，并对科学出版的可信度，包括欺诈行为的普遍性表示担忧。
- 人们认识到有必要加强学术界的问责制，改进现行的科学研究和出版制度。

## [元屏蔽加拿大的新闻链接](https://www.michaelgeist.ca/2023/08/metablockslinks/)

- 由于 C-18 法案，Facebook 和 Instagram 的母公司 Meta 在加拿大屏蔽了其平台上的新闻链接和分享，导致媒体行业蒙受损失。
- 该法案的支持者无视梅塔的警告，导致了对政府的批评。
- 文章探讨了搜索和社交媒体公司开发内容过滤功能的可能性，比较了澳大利亚和加拿大与这些公司进行交易谈判的方法。

### [反应](https://news.ycombinator.com/item?id=37025233)

- Facebook 在加拿大屏蔽新闻链接的决定引发了有关该平台作为新闻来源的作用的讨论。
- 一些人认为，Facebook 应优先考虑将人们联系起来，而不是充当新闻平台。
- 其他讨论要点包括社交媒体对社会的影响、对假新闻的担忧、与年龄有关的技术偏好以及加拿大媒体的影响力。


## [针对谷歌的反垄断审判将于 9 月开始](https://www.thebignewsletter.com/p/the-first-big-antitrust-trial-of)

- 谷歌目前正面临反垄断审判，政府认为谷歌垄断了普通搜索和搜索广告市场，市场份额高达 90-95%。
- 这次审判的结果可能会对科技行业的竞争产生重大影响。
- 谷歌声称其垄断是合法的，因为消费者更喜欢它的搜索引擎，但批评者认为其搜索引擎的质量已经下降。
- 该审判预计将持续两个月，谷歌还面临着与应用商店垄断和在线广告软件垄断有关的其他反垄断案件。
- 文章强调了 YouTube 过去行动的相关性，以及 AT&T 对新供应商和初创公司崛起的影响。

### [反应](https://news.ycombinator.com/item?id=37023117)

- 针对谷歌的反垄断审判正在讨论之中，对于谷歌的行为是否反竞争以及是否有足够证据证明其垄断行为，各方意见不一。
- 苹果和微软等大型科技公司的主导地位也被提及，这引起了人们对搜索引擎优化、算法自动化以及苹果和谷歌之间缺乏互操作性的担忧。
- 辩论围绕监管的必要性、对创新和竞争的影响以及拆分大公司的潜在后果展开。讨论中还提到了数据隐私和广告行为。
