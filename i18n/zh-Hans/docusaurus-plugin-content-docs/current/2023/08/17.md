# 2023-08-17

## [LK-99 不是超导体](https://www.nature.com/articles/d41586-023-02585-7)

- LK-99 曾被认为是一种室温超导体，但后来发现由于材料中含有硫化铜等杂质，它是一种绝缘体。
- 这一发现强调了利用单晶体精确检测材料特性的重要性。
- 推翻 LK-99 作为超导体的说法被认为是一个迅速而确凿的结果。

### [反应](https://news.ycombinator.com/item?id=37149349)

- 最近，人们对室温超导体的说法表示失望和怀疑。
- 参加讨论者对超导体技术的局限性和挑战表示担忧。
- 在评估科学主张时，强调了复制、同行评审和仔细评估的重要性。

## [全新的 Linux 版本，我称之为 Debian Linux Release (1993)](https://wiki.debian.org/DebianHistory?action=AttachFile&do=get&target=Debian-announcement-1993.txt)

- Ian A. Murdock 宣布从零开始开发的 Debian Linux 即将发布。

- 讨论了与其他 Linux 发行版相比，使用 Debian 的优势和局限性，以及某些显卡的性能问题和在家庭服务器上安装 Ubuntu 的问题。

## [如何节水？停止在帝国县种植紫花苜蓿](https://www.desertsun.com/story/opinion/contributors/valley-voice/2023/02/05/growing-alfalfa-in-imperial-county-and-california-wastes-water/69860506007/)

- 加利福尼亚州的紫花苜蓿种植造成大量水资源浪费，加剧了该州的供水危机。
- 这种做法主要有利于大型农业综合企业，可以低价出售水资源。
- 作者质疑在沙漠中种植紫花苜蓿的可持续性，并建议减少种植以缓解供水问题。

### [反应](https://news.ycombinator.com/item?id=37146398)

- 讨论的重点是水资源定价不当问题及其对农业的影响，主要是在加利福尼亚州。
- 探讨的主题包括农业用水的定价和使用、沙漠用水的困难、水权差异、水资源管理以及水价改革的潜在结果。
- 提出的解决方案包括为农民提供补偿、改变水权以及推广更可持续的农业实践。辩论还涉及对水资源短缺、用水效率低下以及农业对环境影响的担忧。

## [今天，我了解到你可以暂停 Windows 任务管理器，并移动应用程序](https://www.theverge.com/2023/8/16/23834125/microsoft-windows-task-manager-pause-shortcut)

- 按住 CTRL 键可让用户暂停 Windows 任务管理器，防止应用程序四处移动。
- 此快捷方式可用于 Windows 10、Windows 11 以及可能的旧版本操作系统。
- 虽然微软正在为 Windows 11 开发强制退出选项，但对于尚未升级的用户来说，CTRL 键快捷方式仍然很有价值。

### [反应](https://news.ycombinator.com/item?id=37146268)

- 用户对 Windows 任务管理器和其他 Windows 操作系统的设计和用户界面表示不满。
- 缺乏清晰的说明和功能的可发现性是不满意的主要原因。
- 用户将 Windows 与其他操作系统进行比较，并建议使用其他任务管理器选项，有些用户甚至因为对 Windows 的不满而改用 Linux。

## [根据随机数据重置系统时钟的 Windows 功能正在造成严重破坏](https://arstechnica.com/security/2023/08/windows-feature-that-resets-system-clocks-based-on-random-data-is-wreaking-havoc/)

- Windows 的 "安全时间播报 "功能会将时钟重置为不正确的时间，从而造成严重干扰，影响手机号码路由表等关键系统。
- 该功能旨在保持准确的系统时钟，已被确认为自 2016 年以来报告的时间重置的根本原因。
- 工程师和管理员一直对这一鲜为人知的功能所造成的问题感到沮丧，导致不便和服务不可用。

### [反应](https://news.ycombinator.com/item?id=37151220)

- Windows 的时间同步功能因其潜在的安全漏洞以及被利用和冒名顶替的风险而备受关注。
- 用户对该系统的功能表示不满，并敦促对其进行改进，如使用可信服务器、固定签名和其他计时机制。
- 由于担心 Windows 上时间同步的准确性、可靠性和安全性，一些用户选择转用 Linux。

## [Ts_zip：使用大型语言模型压缩文本](https://bellard.org/ts_server/ts_zip.html)

- ts_zip 工具是 ts_server 软件的一部分，可使用大型语言模型对文本进行压缩，提供比其他工具更高的压缩率。
- 不同大小的文本文件有不同的压缩率，还有特定型号压缩速度和内存要求的详细信息。
- 由于 RNN 结构和更快的性能，建议使用较小的 RWKV 模型进行文本压缩。

### [反应](https://news.ycombinator.com/item?id=37152978)

- Ts_zip 是一种利用大型语言模型 (LLM) 的文本压缩方法，具有超越传统压缩方法的潜力。
- 讨论了基于 LLM 的压缩算法的有效性，强调了在新数据上进行无偏测试的必要性。
- 探讨了将 LLM 纳入压缩规模所面临的挑战以及广泛使用 LLM 的不切实际性。
