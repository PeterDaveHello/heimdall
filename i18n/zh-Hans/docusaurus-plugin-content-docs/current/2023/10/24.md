# 2023-10-24

## [软件失灵](https://tonsky.me/blog/disenchantment/)

- 作者批评软件业效率低下，指出了软件速度慢和臃肿的问题。他们对优先考虑更快的硬件而不是优化软件的做法表示担忧。
- 作者主张改进软件架构和编程实践，强调稳定性和质量，以应对网络应用程序中出现的错误和不稳定性。
- 为了在行业内发起讨论和改进，作者支持促进和采用开源项目，以获得更好的软件解决方案。

### [反应](https://news.ycombinator.com/item?id=37985176)

- 文章提出了在推动产品快速发布的背景下，经济激励、市场压力和风险投资对软件质量的影响问题。
- 它讨论了盈利能力和软件质量之间的权衡，重点是复杂性、性能、错误的普遍性以及使用 C++ 等语言所面临的挑战。
- 这篇报道还对优先考虑开发人员效率而不是用户效率的做法提出了质疑，并特别提到，不使用 Docker 进行部署缺乏经济上的动力。

## [谷歌拉高开放互联网的阶梯，推动违宪监管](https://www.techdirt.com/2023/10/23/google-decides-to-pull-up-the-ladder-on-the-open-internet-pushes-for-unconstitutional-regulatory-proposals/)

- 谷歌建立了旨在保护儿童上网的新政策框架，引入了年龄验证和加强家长监督等措施。
- 批评者指出，这项政策可能会对小型企业造成不利影响，并限制互联网自由，这反映了人们对企业动机和潜在政治影响的担忧。
- 话题还延伸到 NSFW 内容、政府监管和社交媒体审查等话题，而其他科技新闻则报道了商标诉讼和解、罚款撤销和全市宽带计划。

### [反应](https://news.ycombinator.com/item?id=37990031)

- 辩论详细讨论了与网络监管和儿童安全有关的话题，强调了人们对谷歌通过安全浏览器可能采取的审查行动的担忧。
- 保护儿童上网的建议包括在操作系统上启动 "安全内容 "设置，强制要求家长安装互联网过滤器，以及验证儿童网站的安全性。
- 还提出了实施这些解决方案的显著挑战，包括与年龄验证措施、网站认证和平衡技术与其他社会因素有关的方面。

## [SumatraPDF 阅读器](https://github.com/sumatrapdfreader/sumatrapdf)

- SumatraPDF 是一款开源阅读器，支持 PDF、EPUB、MOBI、CBZ 等多种文件格式。
- 该软件适用于 Windows，采用 (A)GPLv3 和 BSD 许可。
- 它在 GitHub 上拥有超过 11k 个星级、1.6k 个分叉和来自 48 个不同贡献者的贡献。

### [反应](https://news.ycombinator.com/item?id=37993575)

- 该主题侧重于 SumatraPDF Reader 的优缺点，将其定位为 Adobe Acrobat 的 PDF 处理替代品。这包括其平台兼容性、轻量级占用空间和最新增强功能。
- 对话还涉及对无缝开源 PDF 编辑工具的需求，以及解析和渲染 PDF 文件的挑战，表明了 PDF 工作流程的复杂性。
- 虽然 SumatraPDF 因其性能和页面导航的便捷性而备受赞誉，但用户也遇到过在 Mac 上打印和冻结的问题。此外，用户还希望获得更多功能，如将标签拖入新窗口的功能。

## [马自达对 DIY 智能家居集成开发商发出终止令](https://www.thedrive.com/news/mazda-slaps-developer-with-cease-and-desist-for-diy-smart-home-integration)

- 马自达向一名开发人员发出了停止令，该开发人员创建了增强马自达汽车功能的开放源代码，称其侵犯了版权并复制了现有功能。
- 开发商遵从了命令，以避免法律诉讼，这让使用代码提升驾驶体验的马自达车主感到失望。
- 这一事件引发了关于网络安全、版权法和合理使用概念之间平衡的争论，以及汽车制造商可能无视客户对汽车定制化和自动驾驶的偏好。

### [反应](https://news.ycombinator.com/item?id=37990990)

- 马自达曾因不愿允许定制和第三方访问其车辆而面临争议，他们向创建各种集成的开发者发送了终止信和 DMCA 侵权通知。
- 汽车公司以法律、安全和版权为由为自己的行为辩解，但批评者认为，汽车公司是出于利润和责任的考虑而限制定制。
- 评论者对《数字千年版权法》表示不满，并提出了保护开发者和开放源代码的其他方法。

## [女子打赢与谷歌长达 12 年的官司](https://www.abc.net.au/news/2023-10-23/janice-duffy-wins-12-year-legal-battle-against-google/103008954)

- 阿德莱德的珍妮丝-达菲（Janice Duffy）博士结束了她与谷歌长达 12 年的法律斗争，赢得了对这家科技巨头的两起诽谤诉讼。
- 该案的核心是谷歌在达菲提出删除内容的要求后，仍将某网站的诽谤性内容发布到其搜索引擎页面上。
- 尽管和解细节是保密的，但这起案件强调了此类诉讼所带来的情感和经济压力，并可能促使搜索引擎公司承担更多责任，更加警惕地监控其内容。

### [反应](https://news.ycombinator.com/item?id=37983903)

- 一名妇女就诽谤性自动完成建议对谷歌提起了长达 12 年的诉讼，最终胜诉，确定了谷歌对这些建议的责任，并确立了在收到通知后删除这些建议的先例。
- 各种案件，包括澳大利亚的另一起案件，反映了目前关于搜索引擎对诽谤性内容的责任、"被遗忘权 "以及言论自由与名誉保护之间的平衡的争论。
- 讨论延伸到公共犯罪记录和隐私问题，进一步质疑搜索引擎在处理潜在诽谤内容方面的责任和监管。

## [抗量子计算加密算法的 Java 实现](https://github.com/mthiim/dilithium-java)

- 本文讨论了 Dilithium 算法的 Java 实现，该算法是 RSA（Rivest-Shamir-Adleman）和 ECC（椭圆曲线加密法）的后量子加密替代方案。
- 该实现支持三种安全级别，包括生成密钥、签名、验证以及序列化/反序列化密钥的功能。
- 我们明确指出，本实施方案不用于生产，也不提供任何担保，但代码可根据 Apache 2.0 许可证免费访问。

### [反应](https://news.ycombinator.com/item?id=37984404)

- 文章重点介绍用 Java 开发抗量子加密算法的情况，重点是目前缺乏可用于生产的库和前景广阔的前景。
- 讨论了量子计算的发展给数据保护带来的长期挑战以及实施这些新算法的潜在风险。
- 混合加密是后量子加密与传统加密的结合，被认为是一种潜在的解决方案，但其可靠性和对比特币等数字货币的影响令人担忧。

## [克利夫兰推出提供廉价宽带的计划](https://www.techdirt.com/2023/10/23/tired-of-being-ripped-off-by-monopolies-cleveland-launches-ambitious-plan-to-provide-citywide-dirt-cheap-broadband/)

## [macOS 如何管理 Apple Silicon 上的虚拟内核？](https://eclecticlight.co/2023/10/23/how-does-macos-manage-virtual-cores-on-apple-silicon/)

- 本文介绍 MacOS 在苹果硅芯片上对虚拟内核的管理，并介绍两种类型的 CPU 内核：效率 (E) 内核和性能 (P) 内核。
- macOS 会根据应用程序的服务质量 (QoS) 设置分配这些内核，决定如何使用这些内核来优化应用程序的性能。
- 它讨论了在苹果硅主机上使用虚拟机（VM）所面临的挑战，即无法有效分配内核以及缺乏相关的能效优势。

### [反应](https://news.ycombinator.com/item?id=37991295)

- 文章仔细研究了 macOS 如何在苹果硅上管理虚拟内核，以及 AMD 和 ARM 如何实现大-小处理，这表明科技行业已经为异构计算做好了准备。
- 它强调了操作系统（OS）调度程序在决定内核分配方面的重要作用；这可能与 macOS 上的游戏性能和 Docker 问题等问题有关。
- 文章还讨论了用户空间程序中的线程亲和性、网络浏览器中的书签和标签管理、Chrome 浏览器对标签组的使用，以及专用处理器中 FPGA 的能力，突出了核心架构的效率。

## [如何在没有外部能源的情况下将运行速度提高 50%（2020 年）](https://www.science.org/doi/10.1126/sciadv.aay1950)

- 一项研究表明，类似弹弓的外骨骼装置可将跑步速度提高 50%以上，这表明无动力机器人外骨骼在提高人体性能方面具有潜力。

- 关于增强跑步的物理要求和限制的讨论表明，需要对跑步外骨骼进行更多的工程创新。此外，所提到的各种文章和专利为该技术的扩展研究提供了大量资料。

### [反应](https://news.ycombinator.com/item?id=37989875)

- 文章深入探讨了使用弹簧装置提高跑步速度和效果的想法，并讨论了耐克 Vaporfly 和 Alphafly 鞋等流行款式。
- 它讨论了足底筋膜炎的潜在治疗方法，这是一种影响许多运动员的足部疼痛病症。
- 这篇报道进一步讨论了通过拉起脚踏板来提高自行车运动成绩的有效性，并引用了一项通过在脚上安装弹簧来提高跑步效率的研究。

## [1Password 在其内部 Okta 账户中检测到 "可疑活动](https://arstechnica.com/security/2023/10/1password-detects-suspicious-activity-in-its-internal-okta-account/)

### [反应](https://news.ycombinator.com/item?id=37993702)

- 摘要深入探讨了 Okta 内部账户的安全漏洞，引发了人们对该公司实施的安全措施的担忧。
- 对话就通过在线服务进行密码管理的利弊、软件漏洞的复杂性进行了深入讨论，并对自托管与使用第三方身份供应商进行了权衡。
- 它引起了人们对通过 Okta 影响广泛使用的密码管理器 1Password 的安全漏洞的关注，促进了对分散式密码管理库必要性的讨论和对集中式身份验证的批评。
