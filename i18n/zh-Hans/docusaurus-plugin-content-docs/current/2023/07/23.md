### [我因鳞癌而奄奄一息，而潜在的治疗方法却遥不可及](https://jakeseliger.com/2023/07/22/i-am-dying-of-squamous-cell-carcinoma-and-the-treatments-that-might-save-me-are-just-out-of-reach/)

- 身患鳞状细胞癌的作者强调了他们在获得潜在治疗方法方面所面临的挑战。
- 美国食品和药物管理局(FDA)在批准鳞状细胞癌等疾病的治疗方法时采取了谨慎的态度，这种态度被批评为阻碍了疾病的进展，延误了获得可能挽救生命的治疗方法。
- 人们对允许临终病人尝试实验性治疗的伦理和安全性展开了争论，呼吁在确保病人安全的同时，采用更灵活、更快捷的审批程序。

## [微软密钥遭破解：影响比我们想象的要大](https://www.wiz.io/blog/storm-0558-compromised-microsoft-key-enables-authentication-of-countless-micr)

- 微软和 CISA 披露了一起安全事件，一名来自中国的威胁行为者获取了一个私人加密密钥(MSA 密钥)，并用它伪造了访问令牌，影响了 Exchange Online 和 Outlook.com 的多个客户。
- 被泄露的 MSA 密钥比最初想象的更强大，威胁者可以利用它伪造各种 Azure Active Directory 应用程序的访问令牌。
- 虽然微软已采取措施降低风险，但由于令牌验证过程中缺乏日志记录，客户可能会发现很难检测到针对其应用程序使用伪造令牌的情况。应用程序所有者必须更新其 Azure SDK 并刷新其应用程序缓存，以防止出现密钥泄露漏洞。

### [业界反应](http://news.ycombinator.com/item?id=36823007)

- 身份提供商的签名密钥价值高、功能强大，类似于证书颁发机构(CA)的密钥。
- 微软密钥的泄露对云服务的信任和安全产生了重大影响，特别是在身份层方面。
- 该事件凸显了改进安全措施的必要性，如使用硬件安全模块(HSM)和更严格的密钥管理实践。
