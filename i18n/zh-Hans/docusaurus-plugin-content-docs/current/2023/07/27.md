# 2023-07-27

### [Google 已将 WEI 推入 Chromium](https://github.com/chromium/chromium/commit/6f47a22906b2899412e79a2727355efa9cc8f5bd)

- 谷歌正在为 Chromium 网络浏览器提出一项名为 "网络环境完整性"(WEI)的新功能。
- WEI 旨在解决网站创建和维护成本高昂，而用户却希望免费访问的问题。它允许网站根据特定条件控制访问和功能。
- 该提案因可能破坏网络的开放性和隐私性，以及进一步巩固谷歌在行业中的权力而受到批评。

### [业界反应](http://news.ycombinator.com/item?id=36876301)

- 谷歌在 Chrome 浏览器中实施的网络环境完整性(WEI)功能引发了有关控制、竞争和访问限制的争议。
- Mozilla 呼吁谷歌退出 W3C，并鼓励用户推广火狐等替代浏览器。
- 这场争论还涉及对不懂技术的用户、Safari 的类似功能以及隐私和滥用问题的讨论。

## [解读谷歌的网络环境完整性规范](https://vivaldi.com/blog/googles-new-dangerous-web-environment-integrity-spec/)

- 谷歌提出了一项名为 "网络环境完整性"(Web Environment Integrity)的新规范，旨在为网站提供一个应用程序接口，以确定所使用的浏览器和平台是否受到权威第三方的信任。
- Vivaldi 浏览器认为，该规范是对开放网络的一大威胁，可以比作网站的 DRM。
- 人们担心可能会出现偏见和排斥，因为决定浏览器可信度的测试者很可能是谷歌、微软和苹果等公司，这将使小厂商和 Linux 用户处于不利地位。

### [业界反应](http://news.ycombinator.com/item?id=36875940)

- 谷歌的网络环境完整性(WEI)规范引起了用户的关注，特别是其对安全性和用户自由的影响。
- WEI 的潜在实施引发了人们对网络环境中隐私、控制和限制的担忧，因为它可能导致网络体验受到更多限制，并有可能给大型科技公司带来垄断性控制。
- WEI在防止机器人方面的有效性及其限制用户定制和控制其设备的可能性是争论的焦点。

## [谷歌与开放网络](https://interpeer.io/blog/2023/07/google-vs-the-open-web/)

- 谷歌提出的 "网络环境完整性 "系统允许网站选择哪些设备和浏览器可以访问其服务，从而可能将某些设备和边缘群体排除在外。
- 该提案的动机似乎是谷歌希望通过排除机器人和实施可检测广告拦截器使用情况的证明来降低其广告网络的成本。
- 批评者认为，该提案可能会导致网络的包容性和开放性降低，类似于一个设备，而不是一个赋予用户权力的生成系统。

### [业界反应](http://news.ycombinator.com/item?id=36875164)

- 谷歌提议的网络环境完整性(WEI)引起了开发人员和技术爱好者的关注，因为它可能对广告拦截器和用户体验产生影响。
- 批评者认为，WEI 可能导致网络的开放性降低，并使谷歌对用户体验拥有更多控制权，有可能导致供应商锁定。
- 有人怀疑WEI在防止机器人方面的有效性，也有人担心谷歌对互联网的控制及其潜在动机。

- 还有人担心大型科技公司可能会进行垄断控制，并对浏览器的修改和扩展产生影响。

## [Stable Diffusion XL 1.0](https://techcrunch.com/2023/07/26/stability-ai-releases-its-latest-image-generating-model-stable-diffusion-xl-1-0/)

- Stability AI 发布了 Stable Diffusion XL 1.0，这是一种先进的文本到图像模型，可生成色彩鲜艳、对比度更好、光照更强的高质量图像。
- 该模型包含 35 亿个参数，可在数秒内生成高分辨率图像，因此可根据基本的自然语言处理提示进行定制，并能进行复杂的设计。
- Stable Diffusion XL 1.0 还支持内绘、外绘和图像到图像的提示，允许用户创建图像的详细变化。不过，该模型可能会被滥用，这涉及到伦理问题，Stability AI 正在积极解决这一问题。

### [业界反应](http://news.ycombinator.com/item?id=36880756)

- Stable Diffusion XL 1.0 已经发布，为微调和生成高质量图像提供了令人印象深刻的开箱即用工具。
- Stable Diffusion XL 的进步是由动漫色情社区推动的，但它也可用于各种应用，例如为图标或幻灯片创建一致的风格。
- Hugging Face 上提供了 Stable Diffusion XL 1.0 的模型权重，Clipdrop 和 ComfyUI 等网络工具也可用于生成图像。

### [人们过分强调 "减量、再用、再循环 "的循环方面](https://futurism.com/the-byte/scientists-recycling-backfired)

- 回收利用被过度强调，掩盖了过度生产和其他可持续减少废物战略的问题。
- 许多人错误地认为回收利用是废物管理的一种解决方案，导致人们对哪些物品可以回收利用产生混淆，并加剧了日益严重的废物生产危机。
- 不正确的回收做法会导致破坏性的下游影响，如海洋和陆地受到微塑料污染，温室气体排放增加。

### [业界反应](http://news.ycombinator.com/item?id=36873516)

- 研究强调了对 "减少、再利用、再循环 "口号中有效性顺序的误解，以及在回收箱中放置不可回收废物的普遍现象。
- 由于缺乏明确的回收准则，导致消费者无所适从，不同地区的回收做法也不一致。
- 文章强调，必须把重点放在减少和再利用废物上，因为仅靠回收不足以解决过量产生废物的问题。
