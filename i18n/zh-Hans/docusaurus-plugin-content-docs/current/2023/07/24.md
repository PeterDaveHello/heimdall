## [Llama2.c：在一个纯 C 文件中推理 llama 2](https://github.com/karpathy/llama2.c)

- 该代码可用于在 PyTorch 中训练 Llama 2 LLM 架构，然后将权重保存到原始二进制文件中。

## [罗马尼亚经济学家合法操纵彩票](https://thehustle.co/the-man-who-won-the-lottery-14-times/)

- 罗马尼亚经济学家斯特凡-曼德尔(Stefan Mandel)开发了一套合法 "玩弄 "彩票的系统，通过购买每一种可能的组合，他可以赢得多个大奖和较小的奖金。
- 曼德尔的方法包括分析概率论文和开发一种选号算法，以减少所需的组合数量，从而提高中奖率。
- 曼德尔和他的团队成功赢得了澳大利亚和美国的几项彩票，但他的策略引起了负面关注，最终导致彩票法的修改，以防止其他人复制他的方法。

#### [业界反应](http://news.ycombinator.com/item?id=36834415)

- 这位罗马尼亚经济学家使用合法的方法，通过创建一种算法来计算所有可能的数字组合，从而赢得彩票。
- 这位经济学家进行了一场豪赌，因为没有利润保证，而且有可能出现巨额亏损。
- 彩票机构可以考虑公布售出彩票的数量，以吸引更多的购买者，但一旦不再公布号码，可能会使其他人望而却步。
