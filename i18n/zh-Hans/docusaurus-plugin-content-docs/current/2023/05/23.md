## 2023 年 5 月 23 日

### [内存分配](https://samwho.dev/memory-allocation/)

- 内存分配是计算机编程中的一个关键概念。
- malloc 和 free 是帮助在程序中分配和释放内存的重要函数。
- 存在不同类型的分配器，但它们都跟踪内存的使用，并防止碎片化。

#### [行业反应](http://news.ycombinator.com/item?id=36029087)

- Delphi 7 的内存分配器有一个错误，由于 Windows 锁定了整个分配的 VAD 条目，并在内核中出现了内存不足的错误，导致了隐性错误。
- 文章涉及各种内存分配技术，包括好友分配器、板块分配器和线程本地存储，以及跨语言 FFI API 调用中的内存所有权和管理。
- 该文章提供了互动的视觉效果，以及在 C 语言编程中优化内存分配和处理内存泄露的建议，并对进一步的相关话题提出了建议。

### [美达公司因向美国传输数据被罚款 13 亿美元](https://www.wsj.com/articles/meta-fined-1-3-billion-over-data-transfers-to-u-s-b53dbb04)

- Meta 公司(原名 Facebook)因将用户数据转移到美国而被欧盟罚款 13 亿美元。
- 这是欧盟迄今为止最大的隐私处罚，它强调了数据隐私法规的重要性。
- 这一处罚提醒人们，公司必须遵守数据传输法规，以保护用户隐私。

#### [行业反应](http://news.ycombinator.com/item?id=36028845)

- Meta 公司因将数据转移到美国而违反欧盟数据保护法而被罚款 13 亿美元，引发了关于全球治理、数据隐私和国际法的讨论。
- Meta 公司为避免罚款而可能关闭其爱尔兰子公司的做法可能很复杂，遵守欧盟法规的成本和努力远远超过拟议的罚款。
- 对数据隐私法规有不同的意见，一些人批评对 Meta 等公司的低额罚款，而另一些人则认为，费用只是做生意的额外支出，只要公司进行适当的成本效益分析，就可以继续滥用用户隐私和数据。

### 取消惠普即时墨水防止墨盒被使用

- 惠普即时墨水的客户报告说，取消他们的订阅会导致他们的打印机拒绝使用从其他来源购买的墨盒。
- 据惠普称，惠普即时墨水是一项订阅服务，当你的打印机墨水不足时，会将墨水送到你的家门口。
- 惠普在他们的服务条款中指出，"即时墨水墨盒仍然是惠普的财产，当你的服务结束时，必须归还"。

#### [行业反应](http://news.ycombinator.com/item?id=36030156)

- 惠普即时墨水用户在取消订阅后或订阅期满后不能使用墨水。
- 由于用户租赁墨水而不是拥有墨水，限制了第三方墨盒和替代品的使用，因此引起了争议。
- 惠普的即时墨水订阅服务根据用户的打印页数收费，一些人认为这很方便，而另一些人则认为这是一种卑鄙的做法。

### [泄露的政府文件显示西班牙想禁止端到端加密](https://www.wired.com/story/europe-break-encryption-leaked-document-csa-law/)

- 许多欧盟成员国就是否应该保护端到端加密平台作为欧洲人行使基本隐私权的一种方式，或削弱其作用以防止犯罪分子在执法部门的范围之外进行通信展开辩论。

- 西班牙泄露的政府文件显示了禁止端到端加密的建议，批评者认为这侵犯了隐私权，可能无法有效减少犯罪或恐怖主义。
- 许多欧盟国家希望扫描人们的信息以寻找非法内容，这引起了人们对政府潜在的过度扩张和滥用的担忧。
- 专家说，加密是互联网的一个基本组成部分，限制使用端到端加密可能被视为对隐私的侵犯。

### [维基媒体基金会的遣散费](https://en.wikipedia.org/wiki/Wikipedia:Wikipedia_Signpost/2023-05-22/News_and_notes)

- 维基媒体基金会即将离任的首席执行官凯瑟琳-马赫在 2021 年获得了 623,286 美元的遣散费，除此之外还有工资。
- 首席运营官 Janeen Uzzell 在 2021 年获得了 324,748 美元的遣散费，这创造了基金会的新纪录。
- 基金会已经宣布，它将对各级工作人员采用新的标准化离职政策，即每工作一年支付一个月的离职金，最多不超过九个月(除非当地法律另有规定)--任何例外情况都需要人才与文化部主管和总法律顾问的联合建议，并由首席执行官最终批准。

#### [行业反应](http://news.ycombinator.com/item?id=36029115)

- 维基媒体基金会被批评在净资产充足的情况下向高层管理人员支付高薪和遣散费
- 维基媒体为遣散费辩护，认为在大流行期间和留住顶尖人才是必要的。
- 关于高管薪酬是否应该遵循非营利组织的市场价格动态的辩论，一些人呼吁对高管进行公平的补偿，但不要过度支付。

### [现代工作需要注意力--不断的提醒会偷走它](https://stackoverflow.blog/2023/05/22/modern-work-requires-attention-constant-alerts-steal-it/)

- 在现代工作环境中，不断的通知和干扰会偷走解决棘手问题所需的宝贵注意力。
- 聊天程序、电子邮件收件箱和项目管理应用程序的盛行，使当代工作空间成为一个嘈杂和苛刻的地方，导致工作满意度下降和倦怠。
- 为了对抗注意力的流失，个人和组织可以尝试一些技术，如时间框，设置通知和状态，以及通过重新思考工作流程和协作方法来减少通知。

#### [行业反应](http://news.ycombinator.com/item?id=36034929)

- 由于越来越复杂的技术堆栈，现代工作可能会让人分心，导致更多的交接和对实际工作的关注减少。
- 文章讨论了围绕沟通和专业化的不同组织理念，评论中提出了一些解决方案，如关闭通知或利用专注时间来提高生产力。
- 帖子讨论了远程工作和最大限度地提高生产力的技巧，包括安排所有的工作，关闭通知，对会议要有策略，异步工作，以及利用文档和工作流程工具。

### [恶意的 VSCode 扩展，安装量超过 45k](https://blog.checkpoint.com/securing-the-cloud/malicious-vscode-extensions-with-more-than-45k-downloads-steal-pii-and-enable-backdoors/)

- CloudGuard Spectral 在 Visual Studio Code(VSCode)市场上发现了几个恶意扩展，这些扩展允许窃取个人身份信息(PII)并实现对用户机器的远程壳访问。这些扩展程序已被下载超过 45,000 次。
- 这些恶意扩展包括 "Prettiest java"、"Theme Darcula dark "和 "python-vscode"，它们被伪装成有用的工具或主题，但包含有害代码，旨在窃取用户数据或注入恶意脚本。
- 发现这些扩展后，CloudGuard Spectral 向 VSCode 团队发出警报，导致这些扩展被从市场上删除。该事件强调了对所有开源组件进行仔细验证以避免安全风险的必要性。

#### [行业反应](http://news.ycombinator.com/item?id=36029020)

- 以色列公司 JFrog 在官方市场上发现了恶意的 Visual Studio Code 扩展，强调了验证开源组件和寻找更安全框架的必要性。
- VSCode 中缺乏适当的安全模型，使插件容易受到攻击，社会信号是不可靠的安全指标。
- 一些用户提出了一些解决方案，如明确选择进入/退出扩展遥测、沙箱或只使用受信任的作者。
