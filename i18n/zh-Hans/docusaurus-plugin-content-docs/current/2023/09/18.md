## [CatalaLang/catala：法律规范编程语言](https://github.com/CatalaLang/catala)

- Catala 是一种新的特定领域语言，可以根据法律文件创建算法，确保代码与法律之间的高度保真。
- 该语言的构建反映了法律的逻辑结构，使法律专业人士可以对其进行审查和认证；它甚至还有一个编译器，可以生成律师可读的 PDF 文件。
- Catala 项目以法国法律科技先驱皮埃尔-卡塔拉（Pierre Catala）的名字命名，是法国国家计算机科学研究所 Inria 领导的一项研究计划。然而，该编译器仍不稳定，功能有限。

### [反应](https://news.ycombinator.com/item?id=37546874)

- 对话的重点是使用 Catala（一种编程语言）指定法律和法律文件，强调其优点、缺点以及将英语翻译成形式逻辑系统的潜在挑战。
- 关于使用代码作为监管者、法律代码的复杂性、在法律协议中使用编码符号以及使用编程语言将意图编码到法律中的概念，存在着争论。
- 与会者讨论了撰写清晰法律文本的想法、特定领域语言在法律合同中的作用、软件开发与法律系统的比较，以及对此类编程语言名称选择的关切。

## [在家中以 BitTorrent 方式运行法律硕士课程](https://petals.dev/)

- Petals 是一个平台，用户可以使用消费级 GPU 或 Google Colab 在自己的设备上操作大型语言模型，如 Llama 2、Falcon 和 BLOOM。
- 该平台允许用户加入一个网络，为模型的不同部分提供服务，并根据不同的任务调整模型，同时提供 API 和 PyTorch 及抱脸变形器等灵活选项。
- Petals 的项目在 BigScience 研究研讨会上得到了高度评价，这进一步表明了它对语言模型开发领域的重大贡献。

### [反应](https://news.ycombinator.com/item?id=37546810)

- 文章通过汇集计算资源、创建衍生模型、利用参数高效微调和 LoRA 方法，探讨了在家中使用 BitTorrent 式方法运行大型语言模型 (LLM) 的潜力。
- 讨论了与训练大型模型相关的困难和成本，以及可能的解决方案，如水冷和改装旧服务器卡。此外，还讨论了分散计算和微调模型等概念。
- 文章还提到了 Petals，这是一种在低端设备上运行 LLM 的服务，参与者的评价褒贬不一。文章推测，未来可能使用代币和分散计算来支持开源 LLM。

## [AWS IPv4 资产现值 45 亿美元](https://toonk.io/aws-ipv4-estate-now-worth-4-5-billion/index.html)

- 亚马逊网络服务（AWS）的 IPv4 地址数量增加了 2700 万个，总数达到 1.28 亿个；这使得其 IPv4 资产价值约为 45 亿美元，比 3 年前有了大幅增长。
- 值得注意的是，由于 IPv4 地址越来越稀缺，成本越来越高，估计每个 IPv4 地址价值 35 美元。
- AWS 已计划按每个 IP 每小时 0.005 美元的费率向客户收取 IPv4 地址费用，预计每年可产生 5 亿至 10 亿美元的收入。

### [反应](https://news.ycombinator.com/item?id=37549745)

- 文章讨论了从 IPv4 向 IPv6 过渡过程中遇到的困难和挫折，包括对兼容性、网络升级和互联网服务提供商采用速度低于预期的担忧。
- 关于其他解决方案，如扩展 IPv4 或创建一个新版本，人们还在争论不休；关于 IPv6 地址的复杂性和可用性，以及迁移缺乏经济动力，人们也提出了批评。
- 帖子还强调了 IPv4 地址的短缺以及主要参与者可能采取的反竞争行为，表明需要一个更顺畅的转换过程和更方便用户的解决方案。

## [植物可以探测声音](https://www.economist.com/science-and-technology/2023/09/06/plants-dont-have-ears-but-they-can-still-detect-sound)

- 科学研究表明，植物可以探测声音并对声音做出反应，而且有可能自己发出声音。
- 各种实验表明，不同类型的声音可以影响植物的生长和应激化合物的产生。例如，据报道，一种亚洲灌木在听到佛教诵经声后叶片变大。

- 一项研究发现，Boquila trifoliolata 植物物种具有辨别声音的能力，但其背后的机制仍存在争议。
