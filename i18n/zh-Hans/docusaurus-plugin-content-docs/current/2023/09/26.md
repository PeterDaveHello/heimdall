# 2023-09-26

## [我们开始在 ChatGPT 中推出新的语音和图像功能](https://openai.com/blog/chatgpt-can-now-see-hear-and-speak)

- OpenAI 正在为其 ChatGPT 人工智能系统添加语音和图像功能，使其成为一个交互性更强的界面，用户可以讨论图像并进行语音对话。
- 这些功能应用广泛，包括讨论旅行照片、识别食品和帮助解决数学问题。
- 这些更新将在未来两周内提供给 Plus 和企业用户，在 iOS 和安卓系统上都可以使用语音功能，并且图像兼容所有平台。OpenAI 回应了与这些新功能相关的冒名顶替和隐私问题，声称他们已采取措施降低这些风险。

### [反应](https://news.ycombinator.com/item?id=37642335)

- OpenAI 的目标是根据用户对低延迟语音订购的反馈以及对更快语音响应和系统可中断性的要求，改进 ChatGPT 的语音和图像功能。
- 尽管对速度、准确性和生态影响存在担忧，但 ChatGPT 在机器人规划和任务中的潜在用途仍被视为大有可为。关于人工智能生成信息的可信度和价值，人们一直在争论不休。
- OpenAI 计划推出一个新的图像生成模型 DALLE-3，与此同时，人们还在讨论人工智能的商业模式、对互联网搜索的影响以及人工智能平台撤销网页浏览功能等问题。

## [团结最古老的社区宣布解散](http://farewell.bostonunitygroup.com/)

- 波士顿 Unity 小组（BUG）是全球首个官方 Unity 用户小组，该小组宣布将于 9 月 27 日举行最后一次活动，理由是对公司最近的发展方向和缺乏对开发者社区的支持感到失望。
- 组织者认为，他们的贡献应更好地用于在更广泛的地方发展社区内创造机会。
- 欢迎 BUG 成员加入波士顿游戏开发者聚会小组，继续与其他行业专业人士进行交流。最后一次 BUG 会议将是在 Zoom 上举行的社交活动。

### [反应](https://news.ycombinator.com/item?id=37648068)

- 由于 Unity 最近的价格变动，Unity 的长期社区波士顿 Unity 小组（BUG）已经解散，这表明人们对该公司失去了信任。
- 人们普遍讨论，由于审查制度、注重短期利润和产品可靠性下降等负面商业行为，人们对 Unity 和 Craftsman 等知名品牌的信任度下降。
- 开源游戏开发中的问题，如 Godot 等游戏引擎的局限性、人事变动、沟通失败以及 Unity 对独立开发者的支持存在问题等，也得到了强调。

## [节奏 0](https://en.wikipedia.org/wiki/Rhythm_0)

- "节奏 0 "是艺术家玛丽娜-阿布拉莫维奇（Marina Abramović）在 1974 年进行的一场长达 6 小时的表演，她允许观众使用提供的物品与她互动，测试公众行为的程度。
- 随着演出的进行，观众的行为从温和的互动升级到极端暴力，包括割破她的衣服、割破她的喉咙、用上膛的枪威胁她。
- 最后，阿布拉莫维奇站起身来，观众纷纷逃离，这部作品被誉为行为艺术史上的开山之作。

### [反应](https://news.ycombinator.com/item?id=37642610)

- 中心主题围绕对玛丽娜-阿布拉莫维奇的行为艺术 "节奏 0 "的解读，辩论了观众行为的概括性以及艺术描绘科学所不能描绘的人类视角的能力。
- 讨论还探讨了营销和背景对观众行为的影响，并思考了表演可能涉及的法律问题，对艺术作品背后的道德和议程提出了质疑。
- 辩论还涉及基督教伦理、艺术与法律相互交织的作用，以及艺术作品在引发思考和催生暴力行为方面的真实性和有效性。

## [根据新州法律，纽约雇主必须在招聘广告中注明薪资标准](https://english.elpais.com/economy-and-business/2023-09-17/new-york-employers-must-include-pay-rates-in-job-ads-under-new-state-law.html)

- 纽约最近的一项法律规定，雇主有义务在招聘广告中公布薪酬标准，旨在鼓励薪酬透明，解决与年龄、性别和种族有关的薪酬差距问题。
- 该法律适用于至少有四名员工的企业，其中包括向纽约管理人员汇报工作的远程员工。
- 批评者认为，小雇主可能难以遵守，而且该法律给企业带来了额外的行政负担。加利福尼亚州和科罗拉多州也制定了类似的薪酬透明法。

### [反应](https://news.ycombinator.com/item?id=37641583)

- 纽约新出台的一项法律规定，雇主必须在所有招聘广告中注明薪资标准。
- 讨论显示了不同的观点，一些人质疑该法律的效力，另一些人则认为它可以帮助求职者避免薪酬过低的工作，同时促进薪酬对等。
- 话题延伸到决定薪酬等级，考虑的因素包括软件工程师的价值和绩效、薪酬中性别歧视和种族主义的影响，以及公布薪酬细节的优势或挑战。

- 苹果公司正在远离 Perl 等不同的脚本环境，在代码库中转向 Swift，这引起了用户对这一变化的不满。
