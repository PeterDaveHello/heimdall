# 2023-09-15

## [边际用户的暴政](https://nothinghuman.substack.com/p/the-tyranny-of-the-marginal-user)

- 这篇文章批评了消费软件的衰落，OKCupid 等交友软件就是一个例子，原因是将重点转向了吸引新用户，而忽略了改善现有用户的体验。
- 提出了 "边缘用户 "的概念，即注意力集中时间短、喜欢简单内容的用户。人们认为，优先考虑这种用户类型会导致软件质量和用户体验下降。
- 作者指出，增强用户代理的工具通常由业余爱好者开发，经常被大公司收购或停用。

### [反应](https://news.ycombinator.com/item?id=37509507)

- 文章讨论了 OkCupid 转向以移动用户为中心的方法，据说这导致了对话质量的下降和更多表面约会应用程序的崛起。
- 报告还强调了增长模式在软件开发中的负面影响，批评这种模式只注重经济成就而忽视用户体验，并指出一味迎合眼前的满足感可能会导致产品的衰败。
- 文章关注用户界面设计对内容传输的影响、功利主义的挑战以及对资源消耗和不平等的日益关注等问题，提请人们注意设计最佳道德系统所面临的挑战。

## [Unity 似乎悄无声息地删除了跟踪 ToS 变更的 GitHub 仓库](https://www.gamerbraves.com/unity-silently-deletes-github-repo-that-tracks-terms-of-service-changes-and-updated-its-license/)

- 游戏开发平台 Unity 悄然删除了一个 GitHub 存储库，该存储库记录了其服务条款（ToS）的变更，以及允许开发人员使用旧版本游戏引擎的条款。
- 存储库的删除使得该网页不再可访问，最后一次可访问时间为 2022 年 7 月 16 日。
- Unity ToS 的主要变化包括取消了一项条款（2023 年 4 月），允许开发者按照更新时的条款使用当年的软件版本；这迫使用户必须遵守 Unity 的所有服务变化，例如最近的定价计划更新，对开发者的每个游戏安装收取费用。

### [反应](https://news.ycombinator.com/item?id=37516523)

- Unity 删除了跟踪其服务条款（ToS）修订情况的 GitHub 存储库，引发了人们对透明度和信任度的担忧。
- 对服务条款的这些追溯性修改让游戏开发商感到不满，因为他们的工作是建立在以前的成本结构基础上的。
- Unity 的行为引起了人们对其合法性和道德性的质疑，并与虚幻引擎进行了不利的比较，后者并没有征收类似的每次安装费。

## [PostgreSQL 16](https://www.postgresql.org/about/news/postgresql-16-released-2715/)

- PostgreSQL 全球开发小组宣布于 2023 年 9 月 14 日发布 PostgreSQL 16，这是开源数据库的最新更新。
- 增强功能包括：提高查询并行性、批量数据加载和逻辑复制的性能；扩展 SQL/JSON 语法；新增监控统计功能；改进策略管理的访问控制。
- 值得注意的新增功能包括支持客户端负载平衡、通过 SIMD 加速 CPU 和双向逻辑复制，由于加强了监控、访问控制和安全功能，这对开发人员和组织机构非常有利。

### [反应](https://news.ycombinator.com/item?id=37508471)

- PostgreSQL 16 是一个开源软件项目，它的发布增强了功能并改善了版本管理。
- 该项目因其在软件基础架构方面的影响力而闻名，得到了众多贡献者的支持。
- 当前讨论的重点领域包括性能优化、索引膨胀、复制以及与 PostgreSQL 有关的部署选项。

## [在 Git 仓库中，文件存放在哪里？](https://jvns.ca/blog/2023/09/14/in-a-git-repository--where-do-your-files-live-/)

- Git 将文件作为独立的对象存储在仓库内的 .git/objects 目录中，采用内容寻址存储，文件名是文件内容的哈希值。
- Git 并不存储单个版本的差异，每个版本都保存为一个完整的文件。要查找旧版本，可以使用 "git log "查找与文件改动相关的提交 ID。
- Git 使用 zlib 格式压缩对象。作者提供了一个使用 Git 检索博客文章旧版本的实用示例，并附有代码片段。

### [反应](https://news.ycombinator.com/item?id=37512888)

- 文章全面介绍了 Git，包括其独特功能、故障排除方法、内部工作原理以及使用有向无环图（DAG）跟踪变更。
- 尽管有观点支持 Git 在版本控制系统领域的灵活性和一致性，但也探讨了对 Git 用户界面的批评意见以及潜在的改进方案。
- 文章还讨论了 Git 的复杂性和可用性，将其受欢迎程度与其他选择进行了比较，并概述了未来创新和扩展的潜力。

- 该公司的首次公开募股获得了广泛的认可，吸引了苹果、三星和英特尔等科技巨头的投资，但由于今年早些时候在伦敦两地上市的希望破灭，该公司将只在美国上市。

### [反应](https://news.ycombinator.com/item?id=37509427)

- ARM公司的首次公开募股（IPO）已经开始，有猜测认为，潜在的价格上涨可能会促使客户转向开源的RISC-V架构。
- 目前正在讨论从 ARM 过渡到 RISC-V 的可能性，主要考虑因素是软件兼容性和硬件互操作性。
- 软银出售 21% 的股份，IPO 的战略定位是保持 ARM 的成功。苹果、谷歌和英伟达（Nvidia）等科技巨头可能会支持首次公开募股，以防止恶意收购。

## [被盗的梵高画作装在宜家家居的袋子里被归还](https://www.smithsonianmag.com/smart-news/dutch-art-detective-recovers-a-van-gogh-stolen-in-2020-180982896/)

- 文森特-梵高的早期作品《春天纽嫩的堂屋花园》在被盗三年半后从荷兰一家博物馆找回。
- 这幅价值在 320 万至 640 万美元之间的画作被匿名装在宜家的一个袋子里归还给了艺术侦探亚瑟-布兰德，归还者没有牵涉到失窃案。
- 最初的窃贼于 2021 年被捕，并被判处八年监禁。修复后的艺术品将在经过必要的检查和修复后再次展出。

### [反应](https://news.ycombinator.com/item?id=37510885)

- 在艺术侦探亚瑟-布兰德（Arthur Brand）经营的一个地下网络的协助下，被盗的梵高画作在一个宜家家居的袋子里被找回并归还。
- 这幅 2002 年被盗的画作一直在黑社会流传，它的归还说明了与拥有被盗艺术品有关的法律问题。
- 帖子中还讨论了有关绘画保护、艺术品修复的主题，以及有关艺术品失窃叙事的书籍和系列推荐。
