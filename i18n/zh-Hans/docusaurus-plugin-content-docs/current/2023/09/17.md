## [魂器将文件分割成加密片段](https://github.com/jesseduffield/horcrux)

- Horcrux 是一种开源工具，可将文件分割成加密部分，用户无需再调用密码来确保分布式存储的安全。
- 该技术采用沙米尔秘密共享方案，它将加密密钥分割成若干部分，只有预先确定的部分才能重建加密密钥。
- 该工具为需要加密大型敏感文件并通过多种渠道传播以降低拦截风险的特定用户提供服务。

### [反应](https://news.ycombinator.com/item?id=37534615)

- 讨论围绕几种加密工具和技术展开，如 Horcrux、Shamir Secret Sharing (SSS)、HTTP Basic Authentication、多项式采样和 Oblivious Transfer (OT)。
- 对话广泛讨论了这些工具的局限性、安全问题和潜在用例，强调了改进和方便用户使用的必要性。
- 一些无关的话题，如《哈利-波特》、J.K. 罗琳的政治观点和变性人权利，也被简要地纳入讨论。

## [生成图像动力学](https://generative-dynamics.github.io/)

- 研究论文详细介绍了一种将单一静态图像转化为循环视频或交互式动态场景的技术。
- 该方法包括学习从真实视频序列中提取的运动轨迹，预测每个像素的长期运动表示，称为神经随机运动纹理。
- 该模型可将神经随机运动纹理转化为用于生成视频的密集运动轨迹，从而实现与图片中物体的逼真互动。

### [反应](https://news.ycombinator.com/item?id=37536016)

- 生成图像动态技术是一种新工具，可以在图像中创建微妙的动态效果，类似于电影胶片，可以增强营销材料或摄影作品的效果。
- 该工具引发了人们对其潜在应用和实例的极大兴趣和讨论。
- 此外，还讨论了将类似技术融入视频游戏开发的可能性。

## [为 Let's Encrypt 提供动力的数据库服务器（2021 年）](https://letsencrypt.org/2021/01/21/next-gen-database-servers.html)

- Let's Encrypt 是一家为各种网站提供 TLS 证书的证书颁发机构，它增强了数据库服务器的性能和可扩展性。
- 该组织选择使用 AMD EPYC 芯片和戴尔的 PowerEdge R7525 服务器进行升级，从而将性能提高了一倍，并启用了 NVMe 驱动器，有效减少了 API 请求处理时间和数据库延迟。

### [反应](https://news.ycombinator.com/item?id=37536103)

- Let's Encrypt 采用 AMD EPYC 处理器和 NVMe SSD 增强了其数据库服务器，从而提高了性能和可扩展性，并大幅提升了每日证书发行量。
- 此外，还讨论了与其他数据库的比较以及与 AWS 托管相比的成本节约情况，从而引发了一场关于采用云服务还是投资物理硬件的辩论。
- 此外，还提出了租用专用服务器或使用主机托管服务等替代战略，强调在做出决定时需要考虑独特的情况和工作负载。
