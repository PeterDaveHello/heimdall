# 2023-12-28

## [美国 5 年来首个新核反应堆在沃格特尔核电站开始运行](https://www.eia.gov/todayinenergy/detail.php?id=61106)

- 佐治亚电力公司（Georgia Power）位于佐治亚州的沃格特尔（Vogtle）核电站已开始新核反应堆的商业运营，这是自 2016 年以来美国的首个核电站。
- 该反应堆名为 3 号机组，功率为 1,114 兆瓦，将与另一个反应堆 4 号机组一起投入运行，预计将于 2024 年 3 月开始运行。
- 两座新反应堆投入运行后，沃格特尔核电站将成为美国最大的核电站，并将有助于减少电力行业的碳排放。

### [反应](https://news.ycombinator.com/item?id=38785468)

- 摘要概述了讨论核电站、可再生能源、成本效益、安全问题和能源生产未来的对话。
- 它强调了核能、太阳能和风能等能源的挑战和优势，包括成本、效率和环境影响等因素。
- 对话还涉及铀矿开采、可再生能源的储存解决方案、政府对核能的支持、建设延误以及不同国家在能源生产中的作用等话题。

## [纽约时报》起诉 OpenAI 和微软侵犯版权](https://www.theverge.com/2023/12/27/24016212/new-york-times-openai-microsoft-lawsuit-copyright-infringement)

- 纽约时报》已对 OpenAI 和微软提起诉讼，指控它们未经许可使用该报的数百万篇文章来训练其语言模型。
- 泰晤士报》认为，这些人工智能模式与其内容直接竞争，导致收入损失，并损害了与读者的关系。
- 该出版物正在寻求数十亿美元的赔偿，并希望法院禁止 OpenAI 和微软在其人工智能模型中使用其内容。其他新闻机构，如英国广播公司（BBC）、有线电视新闻网（CNN）和路透社（Reuters），也阻止了 OpenAI 的网络爬虫，而包括 Politico 和美联社（Associated Press）在内的一些出版物已达成协议，允许将其内容用于人工智能训练。

### [反应](https://news.ycombinator.com/item?id=38781941)

- 纽约时报》起诉 OpenAI 和微软侵犯版权，引发了关于在训练语言模型时使用受版权保护材料的质疑。
- OpenAI 因没有适当承认内容创作者而受到批评，人们还在争论训练算法中的公平使用问题。
- 讨论涉及数据隐私、健康数据的使用、版权保护、创作者的报酬以及对传统新闻机构和开源机器学习的影响等话题。

## [瑞士甲状腺肿的兴衰：战胜碘缺乏症](https://www.lrb.co.uk/the-paper/v45/n23/jonah-goodman/a-national-evil)

- 20 世纪初，甲状腺肿和相关疾病在瑞士非常普遍，导致颈部肿胀，对家庭的健康和经济造成严重影响。
- 人们发现甲状腺肿的原因是缺碘，解决的办法是引入加碘盐，尽管最初遭到了反对。
- 加碘盐在瑞士的实施大大降低了甲状腺肿大的发病率，但由于欧根-比尔彻博士的反对，相关人员的成就黯然失色。

### [反应](https://news.ycombinator.com/item?id=38782954)

- 文章和评论主题涉及各种话题，包括甲状腺肿、碘补充、碘盐、氟化物、公共卫生倡议以及瑞士的历史健康问题。
- 评论中的讨论涉及碘缺乏症、干预措施的益处和风险、碘在甲状腺和其他健康状况中的作用、关于加碘盐的争论、关于水氟化和疫苗接种的争议以及甲状腺肿和克汀病在瑞士的历史。
- 对话强调了碘对公共卫生的重要性，以及在这些领域开展更多研究和了解的必要性。

## ["三角测量 "活动破解 iPhone 并揭露苹果硬件的秘密功能](https://arstechnica.com/security/2023/12/exploit-used-in-mass-iphone-infection-campaign-targeted-secret-hardware-feature/)

- 在长达四年的时间里，"三角测量 "活动针对并入侵了大量苹果手机，其中包括总部位于莫斯科的安全公司卡巴斯基的员工所拥有的手机。

- 苹果公司已经修补了该活动利用的四个关键零日漏洞，但在苹果的 M1 和 M2 CPU 中也发现了用于绕过内存保护的秘密硬件功能，这引起了人们对其他苹果设备安全性的担忧。

### [反应](https://news.ycombinator.com/item?id=38784073)

- 黑客利用先进的漏洞利用技术，针对 iPhone 实施了长达 4 年的攻击活动，最终通过一个未记录的硬件 GPU 寄存器绕过了内存保护。
- 揭示这一活动的文章以其清晰、全面的解释受到了读者的好评。
- 已经发表了许多文章，提供了有关该行动的深入技术信息和分析。

## [用 Python 和汇编中的 if 语句优化数字校验](https://andreasjhkarlsson.github.io//jekyll/update/2023/12/27/4-billion-if-statements.html)

- 作者使用大量 if 语句探讨了确定一个数字是奇数还是偶数的问题。
- 他们使用 Python 程序生成一个 C 程序，可以处理 8 位、16 位和 32 位整数的这一问题。
- 然而，在扩大规模时，它们会遇到编译器和可执行文件大小的限制。
- 为了克服这些限制，他们手动编写了 x86-64 汇编程序，并创建了一个函数来检查 32 位数字是偶数还是奇数。
- 该程序性能良好，能够处理大量数字。

### [反应](https://news.ycombinator.com/item?id=38790597)

- 对话中包含了幽默、怀疑和个人轶事等元素，使其具有吸引力和亲和力。

## [三角行动研究人员发现利用零日漏洞的 iOS 攻击链](https://securelist.com/operation-triangulation-the-last-hardware-mystery/111669/)

- 研究人员展示了他们对 "三角行动 "的发现，这是一个针对 iOS 16.2 及以下版本的 iOS 系统的攻击链。
- 这次攻击利用了零日漏洞，包括苹果公司 ADJUST TrueType 字体指令中的远程代码执行漏洞和 JavaScript 漏洞，以获得设备控制权并运行间谍软件。
- 攻击者利用苹果设计的 SoC 上的未知硬件功能绕过了基于硬件的内存保护，利用与 GPU 协处理器相关的未知 MMIO 块和特定寄存器来控制硬件功能和中断。

### [反应](https://news.ycombinator.com/item?id=38783112)

- 拥有高级研究能力的黑客们正在讨论一种名为 "三角测量行动 "的针对 iPhone 的潜在攻击，该行动利用了一种自定义哈希函数和零日漏洞来进行广告跟踪和 iPhone 云服务检测。
- 有关苹果公司内部人员参与的猜测引发了对硬件接口安全和后门存在的担忧，苹果芯片和 ARM 设计中的漏洞也受到了严格审查。
- 讨论内容包括 iPad 功能、SMS 短信、苹果为提高设备安全性所做的努力，以及网络浏览器漏洞、信息应用程序安全性和黑客攻击相关风险。重点关注苹果公司的安全措施、设备漏洞以及加强防御策略的必要性。

- 苹果公司仍在为 iPhone 6S 等旧设备提供安全更新，但这些更新只针对某些安全问题。

- 苹果公司的回应暗示，他们可能只在旧版本的 iOS 系统漏洞被主动利用时才打补丁。
- 这不禁让人担心 iOS 安全更新对旧设备的有效性。

### [反应](https://news.ycombinator.com/item?id=38788273)

- 文章讨论了 iOS 中的一个漏洞，该漏洞可导致数据外泄，但尚未得到修复。
- 它强调了与苹果工程师就错误报告进行沟通所面临的挑战。
- 讨论了谷歌 Pixel 手机和 iPhone SE 机型在更新支持和经济性方面的比较。
- 强调了就报废操作系统和安全更新进行明确沟通的重要性。
- 其中提到了关于为旧设备提供支持和安全更新的争论，以及终止对旧设备的安全支持的道德问题。
- 此外，还简要介绍了 Spotify 在 iOS 上的价格与网络价格的比较，以及在苹果设备上运行 Linux 的可能性。
