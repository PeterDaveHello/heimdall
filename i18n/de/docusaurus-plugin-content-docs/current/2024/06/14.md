## [„Lamini Memory Tuning: 10-mal weniger Halluzinationen“](https://www.lamini.ai/blog/lamini-memory-tuning)

- „Lamini Memory Tuning ist eine neue Methode, um Fakten in große Sprachmodelle (LLMs) einzubetten, was die faktische Genauigkeit erheblich verbessert und Halluzinationen reduziert.“
- „Es erreichte eine Genauigkeit von 95 % für einen Fortune-500-Kunden, verglichen mit 50 % bei anderen Methoden, und reduzierte Halluzinationen von 50 % auf 5 %.“
- „Die Methode beinhaltet das Abstimmen von Millionen von Expertenadaptern mit präzisen Fakten auf einem beliebigen Open-Source-LLM, wobei eine Optimierung auf null Fehler bei spezifischen Fakten angestrebt wird, während die allgemeine Leistung beibehalten wird.“

### [Reaktionen](https://news.ycombinator.com/item?id=40675577)

- „Lamini Memory Tuning ist eine Methode, die zahlreiche faktengestützte Low-Rank Adaptations (LoRAs) auf null Verlust trainiert und Retrieval-Augmented Generation (RAG) verwendet, um die geeigneten LoRAs für eine Abfrage auszuwählen.“
- „Die Methode beinhaltet eine Experten-Router-Schicht, um zu entscheiden, welche LoRAs zur Inferenzzeit integriert werden sollen, aber der Trainingsprozess für den Router ist unklar.“
- „Das Modell kombiniert Konzepte aus Mixture of Experts (MoE) und Cross-Attention, wobei die vortrainierten Gewichte des Large Language Model (LLM) konstant gehalten werden, während die Cross-Attention-Schicht trainiert wird, um die faktische Genauigkeit zu gewährleisten.“
