### Kompostierbare Lebensmittelbehälter können giftige Chemikalien in die Luft abgeben

Kompostierbare Fast-Food-Verpackungen können flüchtige PFAS freisetzen.
Eine neue Studie hat ergeben, dass kompostierbare Fast-Food-Verpackungen giftige Chemikalien, so genannte PFAS, in die Luft abgeben können. Einige kompostierbare Verpackungen sind möglicherweise nicht umweltfreundlich und setzen in Mülldeponien Methan frei. Einige Hersteller verzichten jedoch schrittweise auf PFAS in kompostierbaren Schalen, indem sie diese durch Beschichtungen auf Pflanzenbasis ersetzen. Wiederverwendbare Verpackungen sind die beste Lösung, um umweltfreundlich zu werden. Die Toxizität von Lebensmittelverpackungen wird zunehmend beachtet, und die Verwendung von biologisch abbaubaren Verpackungen wie Papier und Glas könnte sicherer sein. Forscher schlagen ein Verbot aller PFAS in Lebensmittelverpackungen vor. Der Gesamtgehalt an PFAS sinkt nach zwei Jahren Lagerung um bis zu 85 %, da sich Fluortelomeralkohole und andere PFAS verflüchtigen.

### Microsoft führt Windows Sandbox zum sicheren Testen von Anwendungen unter Windows ein

Windows Sandbox.
Microsoft hat Windows Sandbox eingeführt, eine sichere und effiziente Desktop-Umgebung für das sichere Testen von Anwendungen unter Windows. Die Sandbox ist unverfälscht, einwegfähig und nutzt hardwarebasierte Virtualisierung zur Kernel-Isolierung. Sie ist Teil von Windows, und alles, was für diese Funktion erforderlich ist, ist in Windows 10 Pro und Enterprise enthalten. Windows Sandbox ermöglicht es Benutzern zusammen mit WSL (Windows Subsystem für Linux), Windows in der Sandbox auszuführen, ohne VirtualBox/VMware Workstation zu verwenden. Es wird jedoch nicht von der Windows Home Edition unterstützt, und die Netzwerkverbindung muss über die Windows Sandbox-Konfigurationsdatei deaktiviert werden.

### Optimierung von Hash-Tabellen für effiziente reale Anwendungsfälle

Optimierung der offenen Adressierung.
