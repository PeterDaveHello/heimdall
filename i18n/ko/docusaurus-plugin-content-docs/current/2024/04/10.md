## [Go-MySQL-Server: Go의 MySQL 호환 엔진](https://github.com/dolthub/go-mysql-server)

- go-mysql-server는 Go로 작성된 MySQL 호환 데이터베이스 엔진으로, 사용자가 인메모리 데이터베이스 옵션을 포함하여 MySQL 방언 및 와이어 프로토콜을 사용하여 데이터 소스를 쿼리할 수 있습니다.
- 지원되는 주요 프로덕션 데이터베이스는 클라이언트 라이브러리 및 기능과 같은 다양한 측면에 대한 호환성을 갖춘 MySQL을 원활하게 대체하는 것을 목표로 하는 Git 스타일의 버전 관리가 특징인 Dolt입니다.
- 사용자는 인메모리 테스트 서버를 설정하고, 특정 인터페이스를 구현하여 사용자 정의 백엔드를 만들고, 기술 문서, 기여자 가이드라인 및 보안 정책에 액세스할 수 있으며, Apache 라이선스 2.0에 따라 라이센스가 부여됩니다.

### [반응](https://news.ycombinator.com/item?id=39983490)

- Dolt는 Go로 작성된 MySQL 호환 데이터베이스 엔진으로 복제, 버전 관리 및 성능 최적화와 같은 기능을 제공합니다.
- Dolt의 Postgres 호환성 확장, 메인스트림 데이터베이스용 버전 관리 시스템 개발, PostgreSQL 호환 버전인 Doltgres의 출시 가능성에 대한 논의가 진행 중입니다.
- Dolt에서 사용하는 go-mysql-server 라이브러리를 사용하면 트리거 및 저장 프로시저를 지원할 수 있지만 동시성 제한이 있습니다. Dolt는 실행 중인 서버 없이 상호작용을 테스트하는 데 권장되며 연결 프록시 역할을 할 수 있습니다.
