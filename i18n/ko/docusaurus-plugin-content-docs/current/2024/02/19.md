## [명함 크기의 던전 생성기 (2019)](https://gist.github.com/munificent/b1bcd969063da3e6c298be070a22b604)

## [모든 통계 테스트가 실제로 선형 모델인가요?](https://danielroelfs.com/blog/everything-is-a-linear-model/)

- 일반적인 통계 테스트는 본질적으로 선형 모델의 변형이며, 테스트 목적으로 R의 lm() 함수를 사용하는 데 중점을 둡니다.
- 이 블로그 게시물에서는 T 검정과 선형 모델을 사용하여 그룹 평균을 비교하고 R2 값을 계산하는 일원 표본 T 검정 방법에 대해 자세히 설명합니다.
- 잔차를 분석하고 결과를 예측하는 방법을 설명하고 다양한 시나리오에서 통계 테스트에서 도출된 값을 이해하는 것이 중요하다는 점을 강조합니다.

- 다양한 시스템에서의 선형성 개념, 엔지니어링에서 선형 모델의 실제 유용성, 데이터 분석에서 비선형 모델의 어려움에 대해 설명합니다.
- 이 강연에서는 비선형 시스템을 근사화하기 위해 선형 모델을 사용하는 방법을 살펴보고 다양한 분야에서 선형 모델과 비선형 모델의 사용법에 대해 논의합니다.

## [간단한 플라이 만들기...라텍스 필요 없음](https://tools.suckless.org/sent/)

- 각 단락은 슬라이드 역할을 하여 창에 맞게 콘텐츠를 자동으로 조정하고, UTF-8, 화살표 키 탐색, 파브필드 도구용 Xlib 및 Xft를 지원합니다.
- 간단한 편집, 최소한의 디자인 선택, 풍부한 콘텐츠를 위한 의도적인 결함을 갖춘 센트는 간단한 프레젠테이션 도구의 대안을 제공합니다.

- 선호도는 최소한의 도구부터 기능이 풍부한 도구까지 다양하며, 주로 마크다운 기반 솔루션에 집중되어 있습니다.
