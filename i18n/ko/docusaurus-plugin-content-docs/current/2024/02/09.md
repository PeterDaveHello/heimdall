## [Microsoft, Windows 11 인사이더 프리뷰 빌드에 Windows용 Sudo 도입](https://devblogs.microsoft.com/commandline/introducing-sudo-for-windows/)

- Microsoft는 Windows 11 인사이더 프리뷰 빌드 26052에서 사용자가 관리자 권한이 없는 콘솔 세션에서 관리자 권한 명령을 실행할 수 있는 Windows용 Sudo를 도입했습니다.
- Windows용 Sudo는 GitHub의 오픈 소스 프로젝트로, 새로운 관리자 콘솔을 열지 않고도 명령을 실행하려는 사용자에게 인체공학적인 솔루션을 제공합니다.
- 현재 버전의 Windows용 Sudo는 세 가지 구성 옵션을 지원하며, 권한 상승을 위해 UAC 대화 상자를 통해 사용자의 확인이 필요합니다. Microsoft는 향후 이 프로젝트의 보안 영향 및 오픈 소스 계획에 대한 설명서를 확장하고 더 많은 정보를 공유할 계획입니다.

### [반응](https://news.ycombinator.com/item?id=39305452)

- 이 토론에서는 Windows용 Sudo의 소개, 효과적인 솔루션을 위한 팀 통합 등 여러 Windows 관련 주제를 다룹니다.
- 'sudo'라는 이름의 사용과 혼동 가능성에 대한 우려가 제기되고 있습니다.
- GUI 개발을 위한 PowerShell의 기능, Windows 개발자 환경의 불만 사항, 보안 조치 및 소프트웨어 배포의 어려움에 대해서도 논의합니다.
