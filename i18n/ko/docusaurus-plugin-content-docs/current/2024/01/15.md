## [Vanna.ai: 채팅 인터페이스를 통한 SQL 쿼리 생성](https://github.com/vanna-ai/vanna)

- Vanna는 학습된 모델을 기반으로 SQL 쿼리를 생성하는 오픈 소스 Python 프레임워크로, 사용자가 질문을 하고 그 결과로 SQL 쿼리를 받을 수 있습니다.
