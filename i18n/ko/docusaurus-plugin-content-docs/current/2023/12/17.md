## [SSH3를 소개합니다: HTTP/3 및 QUIC을 통한 더 빠르고 안전한 셸 경험](https://github.com/francoismichel/ssh3)

- SSH3는 HTTP 메커니즘을 사용하여 더 빠르고 안전한 셸 환경을 제공하는 SSH 프로토콜의 개정된 버전입니다.
- 보안 채널 설정에는 QUIC+TLS1.3을, 사용자 인증에는 HTTP 인증을 사용합니다.
- SSH3는 더 빠른 세션 설정, 향상된 인증 방법(OAuth2.0, OpenID Connect), 포트 스캐닝 공격에 대한 견고성, UDP 포트 포워딩 및 최신 QUIC 프로토콜의 기능을 제공합니다.
- SSH 서버를 숨길 수 있으며 인증을 위한 X.509 인증서를 지원합니다.
- SSH3는 현재 실험 중이며 바이너리를 다운로드하거나 소스에서 코드를 컴파일하여 설치할 수 있습니다.
- SSH3를 사용하려면 SSH3 서버를 배포해야 하며, SSH3 클라이언트는 다양한 인증 방법을 지원합니다.

### [반응](https://news.ycombinator.com/item?id=38664729)
