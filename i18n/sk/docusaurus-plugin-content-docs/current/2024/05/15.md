## [Pico: Open source alternatíva k Ngroku pre Kubernetes pripravená na produkciu](https://github.com/andydunstall/pico)

- Pico je open-source tunelovací nástroj ako Ngrok, prispôsobený na produkčnú prevádzku a jednoduché nasadenie v systéme Kubernetes.
- Umožňuje službám registrovať koncové body a smerovať požiadavky prostredníctvom odchádzajúcich pripojení, čím sa odstráni nutnosť vystavovať verejné porty.
- Pico funguje ako cluster odolný voči chybám, čo umožňuje horizontálnu škálovateľnosť a nasadenie bez výpadkov, ideálne na hosťovanie za vyrovnávačmi záťaže HTTP.

### [Reakcie](https://news.ycombinator.com/item?id=40355744)

- Pico je open-source nástroj podobný nástroju Ngrok, ktorý je prispôsobený na spracovanie produkčnej prevádzky a funguje ako reverzný proxy server pre upstream služby, pričom ponúka odolnosť voči chybám a škálovateľnosť.
- Nedávno bol premenovaný na "Piko" z dôvodu podobnosti názvu so zavedeným textovým editorom a možno ho bezproblémovo nasadiť na platformách, ako sú Kubernetes a Docker, pričom podporuje nasadenie bez prestojov.
- Vývojárov zaujala všestrannosť systému Piko pri sieťových úlohách, ako je prístup k miestnym službám HTTP a integrácia s nástrojmi, ako sú napríklad tunely Cloudflare.
