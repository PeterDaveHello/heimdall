## [Computador: Ambiente de desktop avançado baseado em navegador](https://github.com/HeyPuter/puter)

- O "Puter" é um ambiente de desktop sofisticado de código aberto, acessível por meio de um navegador da Web, que se destaca por ser rico em recursos, rápido e personalizável.
- Ele serve como substituto para plataformas de armazenamento em nuvem ou atua como uma área de trabalho remota para servidores, criado com JavaScript básico e jQuery para otimizar o desempenho e o gerenciamento de pilha.
- O projeto visa ao desenvolvimento da Web, à computação em nuvem e aos sistemas distribuídos, oferecendo vários aplicativos, que podem ser implantados localmente ou em ambientes ativos, e promovendo uma comunidade acolhedora.

## [Solicitação de retorno dos controles físicos nos carros até 2026](https://arstechnica.com/cars/2024/03/carmakers-must-bring-back-buttons-to-get-good-safety-scores-in-europe/)

- O Euro NCAP está defendendo que os fabricantes de automóveis reintroduzam controles físicos para funções fundamentais nos veículos até 2026 para combater a condução distraída causada pelo uso excessivo de telas sensíveis ao toque.
- A organização sugere controles físicos separados para sinais de mudança de direção, luzes de emergência, limpadores de para-brisa, buzina e recursos de emergência para aumentar a segurança.
- Embora o Euro NCAP não tenha autoridade para impor o uso de controles físicos, uma classificação de segurança de cinco estrelas tem um peso significativo e poderia incentivar os fabricantes a implementar as mudanças sugeridas.

### [Reações](https://news.ycombinator.com/item?id=39598189)

- O Euro NCAP está pedindo aos fabricantes de automóveis que tragam de volta os controles físicos para funções cruciais, como piscas e limpadores de para-brisa, para aumentar a segurança.
- A discussão enfatiza os riscos associados às telas sensíveis ao toque, especialmente para controles vitais, e ressalta a importância de controles fáceis de usar e de fácil acesso nos veículos.
- As preocupações levantadas pelos usuários incluem distrações e acidentes causados por telas sensíveis ao toque, enfatizando a necessidade de encontrar um equilíbrio entre os controles digitais e físicos durante a condução.

## [Rust para sistemas incorporados: Benefícios, desafios e soluções](https://arxiv.org/abs/2311.05063)

- O artigo explora o uso da linguagem de programação Rust para o desenvolvimento de software incorporado em sistemas críticos de segurança, como dispositivos médicos e veículos autônomos.
- Ele enfatiza os recursos de segurança de memória do Rust e sua compatibilidade com as bases de código C existentes, mas também aponta desafios como suporte inadequado, limitações das ferramentas de teste de segurança estática de aplicativos e dificuldades na engenharia de código interoperável.
- Os desenvolvedores encontram obstáculos ao utilizar o Rust para sistemas incorporados, o que exige percepções e recomendações descritas no documento para enfrentar esses desafios.

### [Reações](https://news.ycombinator.com/item?id=39591694)

- O artigo destaca os desafios do uso do Rust para sistemas incorporados, principalmente em negociações de alta frequência e jogos, enfatizando a importância da segurança da memória para impedir bugs e táticas de trapaça.
- Ele discute questões como a confiança nas dependências do Rust, o gerenciamento de problemas relacionados à memória e a interface do Rust com o C no desenvolvimento incorporado, demonstrando o interesse no Rust para uso incorporado, apesar das preocupações com as restrições do ecossistema e a facilidade de integração.
- O Rust está ganhando força para aplicativos incorporados, mas ainda há preocupações com relação às limitações do ecossistema, à simplicidade da integração e à confiança nas ferramentas de conversão de código.
