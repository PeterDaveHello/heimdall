# 2023-07-01

## [O Twitter exige agora uma conta para ver os tweets](https://techcrunch.com/2023/06/30/twitter-now-requires-an-account-to-view-tweets/)

- O Twitter agora exige que os usuários tenham uma conta para ver os tweets, o que significa que se você não estiver conectado, você será solicitado a fazer o login.
- Esta alteração pode ser uma tática para tentar aumentar a base de utilizadores do Twitter, mas também pode ter um efeito contrário, reduzindo potencialmente a visibilidade dos tweets em motores de busca como o Google.
- Elon Musk, que foi anteriormente CEO do Twitter, mas ainda está envolvido nas operações, pode ser motivado por um desejo de impedir que as ferramentas de IA pesquisem no Twitter.

### [Reacções da indústria](http://news.ycombinator.com/item?id=36540957)

- O Twitter exige agora uma conta para ver os tweets devido a preocupações com os níveis extremos de recolha de dados da plataforma.
- Os utilizadores e os especialistas estão a debater a ética, a legalidade e a propriedade dos conteúdos gerados pelos utilizadores nas plataformas das redes sociais, bem como a necessidade de regulamentos e modelos de compensação.
- A iniciativa do Twitter e de outras plataformas de redes sociais de implementar restrições e rentabilizar os seus serviços levou os utilizadores a explorar alternativas descentralizadas e de fonte aberta para as redes sociais e os fóruns.

## [Hurl 4.0.0](https://hurl.dev/blog/2023/06/30/announcing-hurl-4.0.0.html)

- Hurl 4.0.0 é uma ferramenta de linha de comando que permite aos utilizadores executar pedidos HTTP definidos num formato de texto simples.
- A nova versão inclui relatórios HTML melhorados com uma cascata de pedidos detalhada, facilitando a análise e a depuração de pedidos.
- Recursos adicionais no Hurl 4.0.0 incluem novos filtros para decodificação e XPath, alterações na avaliação JSONPath e suporte para métodos HTTP personalizados.

### [Reacções da indústria](http://news.ycombinator.com/item?id=36539144)

- Os utilizadores apreciam a simplicidade do Hurl e consideram-no fácil de utilizar para testes de API e testes de integração.

## [MdBook - Uma ferramenta de linha de comando para criar livros com Markdown](https://rust-lang.github.io/mdBook/)

- MdBook é uma ferramenta de linha de comando que ajuda a criar livros com Markdown, facilitando a criação de documentação de produtos ou APIs, tutoriais e materiais de cursos.
- Tem características como suporte de pesquisa integrado, realce de sintaxe a cores para blocos de código e ficheiros de tema personalizáveis para formatar a saída.
- O MdBook é escrito em Rust, o que garante velocidade, segurança e simplicidade, e é utilizado pelo projeto de linguagem de programação Rust.

### [Reacções da Indústria](http://news.ycombinator.com/item?id=36528984)

- O MdBook é uma ferramenta de linha de comandos utilizada para criar livros com Markdown.
- É ideal para criar documentação de produtos ou API, tutoriais, materiais de curso ou qualquer coisa que exija uma apresentação limpa, facilmente navegável e personalizável.
- O MdBook é leve, fácil de instalar e tem suporte de pesquisa integrado, realce de sintaxe colorido para blocos de código em várias linguagens e a capacidade de renderizar a saída em vários formatos.
- Tem sido utilizado pelo projeto da linguagem de programação Rust, e o livro The Rust Programming Language é um exemplo do mdBook em ação.
- O mdBook é mantido ativamente e tem uma comunidade ativa de colaboradores.

## [Ficheiro para divórcio da LLVM](https://github.com/ziglang/zig/issues/16270)

- A proposta sugere a remoção das bibliotecas LLVM, Clang e LLD do projeto Zig.
- O Zig planeja implementar seus próprios passes de otimização e atrair contribuições de fabricantes de chips para melhorar o código da máquina.
- Embora a remoção das capacidades de compilação de C++ possa ter impacto em alguns projectos, o Zig pretende ter um IR amigo do contribuidor e um binário de compilador mais leve.

### [Reacções da indústria](http://news.ycombinator.com/item?id=36529456)

- A linguagem de programação Zig está a considerar afastar-se da sua dependência do LLVM, o que tem suscitado preocupações entre os utilizadores.
- Alguns utilizadores acreditam que a remoção do suporte para C++ e Objective-C no Zig pode prejudicar a sua popularidade e adoção.
- Há preocupações sobre a potencial perda de desempenho e os recursos necessários para construir e manter um novo backend de compilador para o Zig, se ele se afastar do LLVM.

## [Recipientes de plástico aquecidos no micro-ondas libertam microplásticos para os alimentos](https://pubs.acs.org/doi/10.1021/acs.est.3c01942)

### [Reacções da Indústria](http://news.ycombinator.com/item?id=36532812)

- Os recipientes de plástico usados no micro-ondas libertam microplásticos para os alimentos, o que pode ter efeitos negativos na saúde humana.
- Os plásticos rotulados como "seguros" ou "sem BPA" podem ainda conter substâncias nocivas, uma vez que mais de 50 químicos diferentes são utilizados como alternativas ao BPA.
- Os recipientes de vidro são uma alternativa mais segura para armazenar e aquecer alimentos, uma vez que não libertam microplásticos e são duráveis e fáceis de limpar.
