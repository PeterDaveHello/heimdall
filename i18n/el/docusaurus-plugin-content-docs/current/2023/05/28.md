- Υπάρχουν εναλλακτικές λύσεις στην τρέχουσα στοίβα ιστού, όπως οι Svelte, Astro και Gemini, που προσφέρουν ελαφριές και χαμηλής κατανάλωσης ενέργειας λύσεις.
- Το Servo ήταν απαραίτητο για την πρώιμη ανάπτυξη του Rust και εργάζεται για τη βελτίωση του Layout 2020 για την υποστήριξη flexbox, grids και μιας αξιόπιστης υλοποίησης float.
