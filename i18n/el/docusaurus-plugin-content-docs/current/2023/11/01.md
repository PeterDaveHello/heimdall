- Οι χρήστες MacBook Pro αντιμετωπίζουν σοβαρές αποτυχίες εκκίνησης λόγω σημαντικών σφαλμάτων στο macOS Sonoma και στο macOS Ventura 13.6, που οδηγούν σε επίμονες μαύρες οθόνες.
- Τα σφάλματα επηρεάζουν όλους τους χρήστες, συμπεριλαμβανομένων εκείνων που έχουν εγκαταστήσει ή δεν έχουν εγκαταστήσει το Asahi Linux, με τον εγκαταστάτη να ελέγχει πλέον αυτόματα για αυτά τα ζητήματα.
