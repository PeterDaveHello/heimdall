## [Δωρεάν πιλοτική φορολογική κατάθεση από την IRS θα είναι διαθέσιμη σε 13 κράτη](https://www.irs.gov/newsroom/irs-advances-innovative-direct-file-project-for-2024-tax-season-free-irs-run-pilot-option-projected-to-be-available-for-eligible-taxpayers-in-13-states)

- Η IRS παρουσίασε ένα πιλοτικό πρόγραμμα Direct File για τη φορολογική περίοδο του 2024, το οποίο επιτρέπει στους επιλέξιμους φορολογούμενους σε ορισμένες πολιτείες να υποβάλλουν απευθείας και δωρεάν τις ομοσπονδιακές φορολογικές δηλώσεις τους.
- Στις συμμετέχουσες πολιτείες περιλαμβάνονται η Αριζόνα, η Καλιφόρνια, η Μασαχουσέτη και η Νέα Υόρκη. Η υπηρεσία Direct File θα είναι φιλική προς τα κινητά τηλέφωνα και θα είναι δομημένη ως συνέντευξη, καλύπτοντας συγκεκριμένους τύπους εισοδήματος και φορολογικές πιστώσεις.
- Σκοπός του πιλοτικού προγράμματος είναι η αξιολόγηση της σκοπιμότητας και της αποτελεσματικότητας του προγράμματος Direct File για τη συλλογή κρίσιμων δεδομένων για την περαιτέρω ανάπτυξή του. Η IRS αναμένει πιο εκλεπτυσμένες λεπτομέρειες και αλλαγές πριν από την επίσημη έναρξη.
