- Οι συζητήσεις καλύπτουν θέματα όπως η πληρωμή για δημόσια δεδομένα, οι περιορισμοί πρόσβασης και η χρήση κλειδιών API για έλεγχο ταυτότητας και προστασία DDoS, καθώς και η αποτελεσματικότητα και η αναγκαιότητα του περιορισμού του ρυθμού και των εναλλακτικών μεθόδων ελέγχου ταυτότητας. Αναφέρονται επίσης οι ανησυχίες για την ασφάλεια που σχετίζονται με τα κλειδιά API.
