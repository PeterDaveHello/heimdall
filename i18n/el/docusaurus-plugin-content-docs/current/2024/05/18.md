## [Bend: Γλώσσα υψηλού επιπέδου για GPU Computing με HVM2](https://github.com/HigherOrderCO/Bend)

- Η Bend είναι μια γλώσσα προγραμματισμού υψηλού επιπέδου βελτιστοποιημένη για μαζικά παράλληλο υλικό, όπως οι GPU, προσφέροντας γρήγορες κατανομές αντικειμένων, συναρτήσεις υψηλότερης τάξης, αναδρομή και συνέχειες.
- Τρέχει στο πρόγραμμα εκτέλεσης HVM2, επιτρέποντας σχεδόν γραμμική επιτάχυνση με βάση τον αριθμό των πυρήνων, χωρίς να απαιτούνται ρητές παράλληλες επισημειώσεις ή χειροκίνητη διαχείριση νημάτων.
- Το Bend, το οποίο δημιουργήθηκε από την HigherOrderCO.com, βελτιώνει τον παράλληλο προγραμματισμό εκτελώντας αποτελεσματικά πολύπλοκους αλγορίθμους σε GPU και εξομοιώνοντας διάφορα ταυτόχρονα συστήματα.

## [ILGPU: C# και F# για προγραμματισμό GPU υψηλής απόδοσης](https://github.com/m4rs-mt/ILGPU)

- Το ILGPU είναι ένας υψηλής απόδοσης μεταγλωττιστής JIT σχεδιασμένος για προγράμματα GPU σε γλώσσες που βασίζονται στο .Net, συνδυάζοντας την ευελιξία της C++ AMP με τις επιδόσεις της CUDA.
- Η βιβλιοθήκη προσφέρει βοηθητικές συναρτήσεις, αλγορίθμους υψηλού επιπέδου και διαθέτει μια ενεργή κοινότητα στο Discord, μαζί με δείγματα έργων και οδηγίες κατασκευής για αρχάριους.
- Για να εργαστείτε με το ILGPU, βεβαιωθείτε ότι διαθέτετε το Visual Studio 2022 και μια αλυσίδα εργαλείων .NET 6.0 SDK και να είστε προετοιμασμένοι για πιθανές προκλήσεις του XUnit/Visual Studio κατά τη διάρκεια των δοκιμών, παρέχοντας υποστήριξη Source Link και Symbol για αποσφαλμάτωση.
