## [PythonはAppleのApp Storeでの拒否に苦慮している](https://lwn.net/SubscriberLink/979671/4fb7c1827536d1ae/)

- Python 3.11から3.12へのアップグレードにより、urllibパーサー内に「itms-services」という文字列が含まれているため、いくつかのPythonアプリがAppleのアプリストアで拒否されました。これは、macOSのサンドボックス化されたアプリでは禁止されています。
- エリック・フロムリングのバグ報告は、アプリストアのレビュー過程に対応する方法について、Python開発者間での議論を引き起こしました。難読化や配布レベルのパッチ適用などの解決策が検討されました。
- Python 3.13に「--with-app-store-compliance」というビルド時オプションを追加することで合意に達しました。これにより、アプリストアへの提出に問題となるコードが削除され、フリーソフトウェアプロジェクトが不透明なレビュー過程で直面する課題が浮き彫りになります。

### [反応](https://news.ycombinator.com/item?id=40815130)

- Pythonアプリはバイナリ内の「itms-services」URLスキームのためにApple App Storeから拒否されており、PyInstallerでビルドされた場合にはWindows Defenderもこれらのアプリをフラグします。
- 開発者たちは、主要なOSプラットフォームが高額なコード署名証明書を要求することで、小規模な開発者を妨げ、アプリの配布を困難にしていると主張しています。
- 提案された代替案には、Macアプリ用のHomebrewやWindowsビルド用のSignPathからの無料証明書を使用することが含まれており、アプリ配布におけるセキュリティ関連の拒否という広範な問題が浮き彫りにされています。

## [FCCの規則により、キャリアは60日後にすべての電話をアンロックする必要がある](https://techcrunch.com/2024/06/27/fcc-rule-would-make-carriers-unlock-all-phones-after-60-days/)

- FCCは、消費者の手続きを簡素化するために、購入から60日後にキャリアが携帯電話のロックを解除することを義務付ける規則を提案しています。
- FCC委員長のジェシカ・ローゼンウォーセルは、明確で全国的なアンロックルールを確立するための規則制定通知(NPRM)を発表し、7月18日から一般の意見募集を開始すると発表しました。
- この規則は一貫性と透明性を提供し、消費者がキャリアを変更しやすくすることを目的としています。

### [反応](https://news.ycombinator.com/item?id=40814728)

- FCCは、キャリアが60日後に電話をアンロックすることを義務付ける規則を提案しており、これにより電子廃棄物の削減とキャリア間の切り替えの容易化を目指しています。
- アメリカはキャリアロックされた電話を持っている点で独特であり、これによりキャリアは電話の機能やアップデートに対して大きな制御権を持っています。これは、支払い未納の場合にリモートでロックできるヨーロッパとは異なります。
- この提案は議論を引き起こしており、キャリアロックを反消費者的と見る人もいれば、信用が低い個人のために携帯電話の資金調達を助けると主張する人もいます。

## [重力波を捉える新しい方法](https://www.nature.com/articles/d41586-024-02003-6)

- 2015年にLIGOとVirgo観測所によって初めて重力波が直接検出されたことは、物理学に革命をもたらし、天文学者が宇宙の出来事を『聞く』ことを可能にしました。
- 現在の検出器は100～1,000 Hzの周波数に限定されていますが、新しい機器と技術はメガヘルツからナノヘルツの周波数範囲を検出することを目指しています。
- 5つの新しい方法には、パルサータイミングアレイ、マイクロ波望遠鏡、原子干渉計、デスクトップ検出器、量子結晶が含まれ、それぞれが異なる周波数範囲を対象としており、より多くの宇宙現象を解明することが期待されています。

### [反応](https://news.ycombinator.com/item?id=40820063)

- 最初の世代の重力波検出器は波を検出することに失敗し、ウェーバーの1987年の主張を信用できないものにしましたが、現在のLIGO、Virgo、KAGRAのような干渉計はそれらを成功裏に検出しています。
- 重力波はアインシュタインの一般相対性理論を確認しますが、重力の量子論は未解決のままであり、弦理論とループ量子重力が潜在的な説明を提供するものの、依然として困難を伴います。
- 将来のプロジェクトとしてLISAのようなものが検出能力を強化することを目指しており、LIGO施設の一般公開ツアーは教育目的で利用可能です。

## [パーキンソン病と腸内細菌の関連が予想外で簡単な治療法を示唆](https://www.sciencealert.com/parkinsons-link-to-gut-bacteria-suggests-unexpected-simple-treatment)

- 研究者たちは、パーキンソン病患者においてリボフラビン(ビタミンB2)およびビオチン(ビタミンB7)のレベル低下に関連する特定の腸内微生物を特定し、Bビタミンの補給が潜在的な治療法となる可能性を示唆しています。
- 名古屋大学の西脇宏教授が主導した研究によると、腸内細菌の変化が短鎖脂肪酸(SCFA)およびポリアミンの生成に影響を与え、これらは健康な腸粘液層にとって重要であることがわかった。
- この研究結果は、npj Parkinson's Disease に掲載されており、健康な腸内細菌叢を維持し、環境毒素を減らすことがパーキンソン病の症状から守るのに役立つ可能性があることを示しています。

### [反応](https://news.ycombinator.com/item?id=40818987)

- 研究によると、腸内細菌とパーキンソン病の間に関連があることが示唆されており、特定の細菌株がこの病状に寄与している可能性がある。
- リボフラビン(B2)やビオチン(B7)などのビタミンを補給することによる症状緩和の潜在的な利点について議論が浮上しています。
- この研究は、神経学的状態に対する腸の健康の重要性を強調していますが、さらなる研究が必要であり、現在の治療法を無視すべきではないと助言しています。
