## [NLRB判事が競業避止条項を不当労働行為と宣言](https://www.nlrbedge.com/p/in-first-case-of-its-kind-nlrb-judge)

- NLRB判事のサラ・カルピネンは、競業避止条項および同僚勧誘禁止条項が不当労働行為であると判断し、画期的な決定を下しました。
- この事件はJ.O. Mory, Inc.に関するもので、労働組合の組織者が「サルティング」(同僚の組織化)を理由に解雇され、裁判官がその組織者の復職と未払い賃金の支払いを命じました。
- これは、行政法判事(ALJ)がこれらの条項を不当労働行為と判断した初めてのケースであり、非監督労働者に対するこれらの条項に反対するNLRBの立場を強調しています。

### [反応](https://news.ycombinator.com/item?id=40696992)

- NLRBの判事は、競業避止条項が不当労働行為であると判断し、他の制限的な雇用慣行についての議論を引き起こしました。
- この判決は、求人勧誘禁止条項や、マネージャーが推薦状を提供することを制限する会社の方針についての議論を引き起こしました。これらは一部の人々によって、職業の流動性を妨げると主張されています。
- この決定は、雇用主と従業員の間の権力のバランスを取るための大きな議論の一部であり、公正な待遇と労働市場の流動性を確保するための労働者保護の強化を求める声が上がっています。

## [フランスの裁判所がGoogle、Cloudflare、Ciscoに対し、海賊行為を阻止するためにDNSを無効化するよう命じる](https://torrentfreak.com/google-cloudflare-cisco-will-poison-dns-to-stop-piracy-block-circumvention-240613/)

## [Raspberry Pi 5はTini-Mini-Micro PCには敵わない](https://louwrentius.com/the-raspberry-pi-5-is-no-match-for-a-tini-mini-micro-pc.html)

- 2023年秋に発売されたRaspberry Pi 5は、性能が向上していますが、SDカードストレージの制約や前モデルに比べてコストが高いという点で依然として制限があります。
- デル、レノボ、HPなどのブランドの中古1LミニPCは、処理能力、RAM容量、およびストレージオプション(SATAおよびNVME SSD)においてPi 5を上回ります。
- Raspberry Pi 5は省エネですが、中古のミニPCは性能が優れており、内蔵SSD/NVMEサポートがあるため、ホームサーバー用途にはより適しています。

### [反応](https://news.ycombinator.com/item?id=40697831)

- Raspberry Pi 5は、Broadcomとの特別な取引の終了により手頃な価格を失い、現在の市場で競争力が低下しています。
- ユーザーは、中古のミニPCにより良い価値を見出しており、同じ価格でより多くのパワーと機能を提供しています。
- Pi 5は教育やGPIO(汎用入出力)アプリケーションにおいて依然として強力ですが、中古のIntel NUCや新しいN100ミニPCのような代替品は、一般的なコンピューティングタスクにおいてより魅力的です。

- この会話には歴史的な背景、無能の役割、そして複雑なセルフサービスシステムや義務的なオフィス復帰ポリシーなどの現代の慣行が含まれており、これらが意図せずに妨害の原則と一致する可能性があります。

## [SQLiteは他のすべてのデータベースエンジンを合わせたよりも多く使用されている可能性があります](https://sqlite.org/mostdeployed.html)

- SQLiteは最も広く展開され、使用されているデータベースエンジンであり、Android、iOS、Mac、Windows 10、主要なウェブブラウザ、およびさまざまな人気のあるアプリケーションを含む数十億のデバイスやアプリケーションに存在しています。
- 40億台以上のスマートフォンにはそれぞれ数百のSQLiteデータベースが含まれており、世界中で1兆以上のアクティブなSQLiteデータベースが存在する可能性があります。
- SQLiteは、最も広く展開されているソフトウェアモジュールのトップ5の一つであり、zlibに次いで2番目に広く展開されているソフトウェアライブラリである可能性があり、デバイスごとに複数のインスタンスが存在するため、最も広く展開されているソフトウェアコンポーネントである可能性もあります。

### [反応](https://news.ycombinator.com/item?id=40693500)

- SQLiteは、その使いやすさ、信頼性、オフライン機能により、他のすべてのデータベースエンジンを合わせたものよりも広く使用されています。
- ユーザーはバックアップや共有のためにデータをダウンロードする際、より良いスキーマの強制を求めてjson+zipからgzipped SQLiteに移行しています。
- その人気にもかかわらず、SQLiteは貢献を受け付けていませんが、そのファイル形式はよく文書化されており安定しているため、iOSやAndroidを含む多くのアプリケーションに理想的です。
