# 2024-02-06

## [ヴェスヴィオ・チャレンジ2023」優勝者が機械学習で2000年前の巻物を解読](https://scrollprize.org/grandprize)

- ヴェスヴィオ・チャレンジ2023の大賞受賞者が発表され、ヘルクラネウム・パピルスから2000年前の巻物を読み解くことに成功した。
- この巻物は、音楽、食べ物、喜びなどのトピックについて論じており、さらなる研究のための貴重な洞察を与えてくれる。
- 優勝チームは機械学習モデルと自動セグメンテーションツールを活用したが、まだ改良が必要だ。
- 研究の過程では、X線トモグラフィと機械学習を使ってパピルスの巻物を分析した。
- 大会は協力と情報共有を促進した。
- 次の挑戦は、2024年ヴェスヴィオ・チャレンジと発表され、埋もれた巻物からさらに多くのテキストを解き明かし、古典の生活と文学への理解を深めることを目的としている。
- チャレンジの勝者は、ゲティ・ヴィラ美術館で祝賀会を行なう。

### [反応](https://news.ycombinator.com/item?id=39261861)

- 科学者たちは、デジタルのアンラッピングとAIのアルゴリズムによって古代の巻物を読むことに進歩を遂げ、考古学とテクノロジーの進歩につながる可能性がある。
- 手作業によるセグメンテーションや、より多くの巻物をスキャンすることの難しさ、そして資金不足が、この分野のさらなる発展を妨げている。
- 失われた写本が発見され、写本にエピクロス哲学が理解される可能性があることに興奮を覚えるが、正確さや検証に対する懸念は根強い。

## [SMSベースの認証：SIMスワップ攻撃で非難される企業](https://keydiscussions.com/2024/02/05/sim-swap-attacks-can-be-blamed-on-companies-embracing-sms-based-password-resets/)

- アカウントのログインやパスワードのリセットにSMSを利用しているセキュリティ企業は、SIMスワップ攻撃に対して責任を負うべきである。
- SIMスワップ攻撃は、犯罪者が通信事業者を騙して被害者の電話番号を被害者の端末に転送させ、SMSアカウントのログイン情報や機密情報へのアクセスを許可することで発生する。
- Apple、Google、PayPalのような有名企業は、セキュリティ上の欠陥が知られているにもかかわらず、SMSベースの認証を採用している。著者は、AuthyやGoogle Authenticatorのような、より安全なオプションの採用を主張している。

### [反応](https://news.ycombinator.com/item?id=39269327)

- この対談では、SMS、Yubikeys、TOTPアプリ、電子メールなど、さまざまなアカウント認証の方法とセキュリティ対策が検討された。
- SIMスワッピング攻撃への懸念から、アカウント関連の目的でSMSを利用することの信頼性と有効性については意見が分かれている。
- この議論では、さまざまな認証方法の限界と脆弱性が強調され、セキュリティとユーザーの利便性のバランスを達成するために、より厳格な規制と多要素認証の実装の改善が求められている。

## [ワシントン下院法案、警察の取り調べでの嘘禁止で虚偽自白に取り組む](https://www.seattletimes.com/seattle-news/politics/wa-house-would-make-it-illegal-for-police-to-lie-during-interrogations/)

- ワシントンで提案されている法案は、虚偽の自白の懸念に対処し、倫理的な法執行を促進するため、警察官が取り調べ中に嘘をつくことを禁止することを目的としている。

- 対談では、取り調べ中の欺瞞や公選弁護人が直面する課題など、警察の不正行為のさまざまな側面について掘り下げる。

- 法的代理権の重要性と、法律顧問なしに警察と関わることの危険性が強調され、人種や法執行機関に対する信頼の問題にも触れて議論される。

## [科学者、ハリケーン規模にカテゴリー6を追加することを提案](https://eos.org/articles/weve-already-seen-category-6-hurricanes-now-scientists-want-to-make-it-official)

- 最近の研究では、サフィア・シンプソン・ハリケーン風力スケールにカテゴリー6という新しいカテゴリーを加えることが提案されている。
- この新しいカテゴリーは、気候変動による熱帯低気圧の威力と破壊力の強化に対処するものである。
- この研究では、過去の暴風雨のデータ、潜在的な最大強度の分析、気候変動モデルを活用し、カテゴリー6の必要性を裏付けている。
- このカテゴリーを追加することで、気候変動に関連するリスクをより正確に表現し、ハリケーンに関連するリスクを一般に伝えるための会話を刺激することができるだろう。

### [反応](https://news.ycombinator.com/item?id=39268106)

- 科学者たちは、ハリケーンの強度が増していることから、新たにカテゴリー6を追加することを提案しているが、懐疑論者たちは調査方法とデータ操作を批判している。
- 科学研究、特に気候変動のような政治的にセンシティブなテーマにおいては、信頼できる一貫性のあるデータが極めて重要である。
- 風速に基づく現在の分類システムでは、高潮や洪水による潜在的な被害を伝えきれない可能性があり、インフラ、高潮、洪水リスクなどの要素を考慮した、より改善された格付けシステムの必要性が浮き彫りになっている。
