# 2024-02-25

## [RSSフィード導入減少におけるグーグルの役割](https://openrss.org/blog/how-google-helped-destroy-adoption-of-rss-feeds)

- グーグルが自社製品全体のRSSサポートを廃止したことは、RSSフィードの普及率低下に大きく寄与している。
- この結果、RSSフィードの信頼性に対するユーザーの信頼が低下し、コンテンツの集約や配信をRSSフィードに依存している人々に影響を与えている。
- ユーザーの信頼を回復し、インターネット上でのRSSフィードの継続的な利用を促進するためには、GoogleがRSS機能を優先し、支持することが不可欠である。

### [反応](https://news.ycombinator.com/item?id=39493770)

- RSSフィードとリーダーの衰退は、2023年にグーグル・リーダーが閉鎖された後に始まり、その結果、ユーザーは適切な代替品を見つけるのに苦労し、RSSの使用から離れた。
- Feedlyのようなプラットフォームが代替案として登場したものの、多くのユーザーはRSSフィードの利用には戻らず、アルゴリズムによるフィードやソーシャルメディアのようなクローズドなエコシステムの台頭にシフトの原因があるとした。
- この対談では、収益化戦略、RSSサービスを維持する上での課題、そして、RSSの関連性を維持するための継続的な努力にもかかわらず、グーグルの決定がインターネット・コンテンツ消費の進化にどのような影響を与えたかについて話している。

- 気候変動への対応の緊急性、情報源の信頼性、気温上昇の影響、再生可能エネルギーへの転換などが議論されている。
- また、人口増加の影響、気温測定の正確さ、産業公害の影響、気候変動の否定、人間活動と気候変動との関連、二酸化炭素排出量削減におけるハッカーやエンジニアの役割などについても議論されている。

## [都市緑化環境に優しい都市のための緑化](https://www.bbc.com/future/article/20240222-depaving-the-cities-replacing-concrete-with-earth-and-plants)

- フューチャー・アースやグリーン・ベンチャーといった団体が主導するデペービング・イニシアチブを通じ、世界の都市はコンクリートやアスファルトを樹木や土に置き換えることを選択している。
- デペービングの取り組みは、都市の回復力、生物多様性、日陰の提供、精神的健康を高めると同時に、洪水を減らし、より環境に優しい都市景観を促進することを目的としている。
- ロンドンとパリは、不透水性の路面を緑地に変えるというイニシアチブの先陣を切っており、このような環境変革には幅広い実施と政府の後押しが必要であることを強調している。

- 都市は、駐車場収入への依存や生息地の喪失といった問題に取り組み、持続可能性を高めるために、コンクリートを植物に置き換えることを考えている。
- ネズミの問題や、適切な廃棄物管理や殺鼠剤の代わりに天敵を捕食するなどの解決策について議論されるなか、地下水位を涵養するための伐採イニシアチブを検討している都市もある。
- 雨水の流出を最小限に抑え、持続可能な実践のために都市部の在来植生を優先させる努力とともに、環境上の利点とコスト削減のために、手入れの行き届いた芝生を緑地に置き換えることの利点を強調する。
