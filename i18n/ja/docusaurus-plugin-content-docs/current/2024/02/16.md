## [アップル、デジタル市場法の新規制によりEUでiPhoneのウェブアプリを意図的に削除](https://techcrunch.com/2024/02/15/apple-confirms-its-breaking-iphone-web-apps-in-the-eu-on-purpose/)

- この動きは、データの損失や通知機能の停止など、ユーザーにとって重大な問題を引き起こし、アップルがEUでPWAをサポートし続けることは現実的ではなくなっている。

### [反応](https://news.ycombinator.com/item?id=39388218)

- アップルは、規制要件を満たし、セキュリティ上の懸念に対処するため、EUでiPhoneのウェブアプリを意図的に壊していたことを認めた。
- 批評家たちは、この動きはアップルがApp Storeに代わるものとしてProgressive Web Apps（PWA）をサポートしていないことを示していると考えている。
- USB-Cケーブルへの不満、PWAとネイティブアプリの比較、アップルのセキュリティ対策への信頼、規制やアップルのApp Storeポリシーに関する議論など、さまざまなトピックで議論が展開された。
