# 2024-03-13

## [アップル、EUでウェブサイトからのアプリ直接ダウンロードを可能に](https://www.macrumors.com/2024/03/12/apple-announces-app-downloads-from-websites/)

- アップルは欧州連合（EU）のデベロッパー向けに新たなアップデートを発表し、ウェブページから直接アプリを配布したり、アプリ内プロモーションを作成したり、自社のカタログから独占的にアプリを提供したりできるようにした。
- 開発者は、特定の基準を遵守し、ウェブサイトからアプリを配布する継続的な義務に同意し、春にソフトウェア・アップデートを投稿する必要がある。
- これらの変更は、アップルがEUのデジタル市場法を遵守していることに沿ったものである。

### [反応](https://news.ycombinator.com/item?id=39678532)

- 対談では、EUの規制、プライバシーの問題、デベロッパーの手数料、消費者の選択、アプリストア部門の競争力など、アップルにまつわるさまざまなテーマについて掘り下げた。
- アップルの利益重視の戦略に対する批判から、クッキー同意通知に関する議論、アップルのセキュリティ・プロトコルに対する見解、デジタル市場法がアップルに及ぼす影響、個人ユーザーがテック企業に及ぼす影響まで、話題は多岐にわたる。
- この議論は、ユーザーのコントロール、プライバシー、セキュリティ、競争、そして技術分野における制約と自由の均衡を広くカバーしている。

- 対談では、AIハードウェアにおけるNvidiaの市場支配力、GoogleのTPUのようなハードウェアの販売における課題、Metaのオープンソースへの献身、そしてクローズドソース技術への転換の可能性についても触れられている。

## [ニューヨーク市警、警官のツイッターに対する令状なしの召喚状を取り下げ](https://hellgatenyc.com/nypd-warrantless-subpoena-copwatcher-social-media)

- ニューヨーク市警は、警官のソーシャルメディア・アカウントに対して令状なしの召喚状を発行したが、法廷での正当性を主張する挑戦に直面し、それを撤回した。
- マイケル・クランシー刑事は、ツイッターで召喚状について知らされ、法的支援を求めた。
- 法律専門家たちは、ニューヨーク市警が十分な正当性なしに令状なしの召喚状を発行する権限に疑問を呈した。これは、警察署がこのようなソーシャルメディアデータの召喚状を使用することへの懸念と、法廷でそれを支持することへの躊躇を反映したものである。

### [反応](https://news.ycombinator.com/item?id=39681920)

- この記事は、ニューヨーク市警が警官のツイッター・アカウントに対して令状なしの召喚状を発行したことについて調査し、政府関係者の説明責任と必要な法的調整についての議論を巻き起こしている。
- スイスの特徴的な民主主義、最近の反テロ法、そして課題について、民主主義の定義と効力、アメリカにおける超党派の欠如、警察の説明責任、適格免責のような法的ハードルを強調しながら論じている。
- 議論は、取り締まりにおける透明性、法定代理における弁護士の役割、職権乱用への懸念、外国との情報協力、法執行における資源配分、警察の説明責任を確保し政治的変化を促すための召喚状の法的意味合いなどにも及ぶ。

## [天気予報精度の進歩](https://ourworldindata.org/weather-forecasts)

- 技術の進歩によって天気予報の精度は大幅に向上し、現在の4日予報は30年前の1日予報と同等の信頼性を持つようになった。
- 正確な気象予報へのアクセスは、暴風雨、熱波、災害の早期警報によって人命を救い、生活を守るために不可欠である。
- 特に低所得国では、正確な気象予測へのアクセスに世界的な格差があり、気候変動への適応と異常気象に対する回復力のために、コミュニケーション、早期警報システム、人工知能（AI）のような最先端技術に投資する必要性が浮き彫りになっている。

### [反応](https://news.ycombinator.com/item?id=39678783)

- この議論では、さまざまな情報源から得られる天気予報の精度のばらつきについて掘り下げ、正確な予報のためには地元の気象予報士と生のデータが重要であることを強調している。
- 雨の正確な予測におけるSiriとApple Weatherの限界については、短期予測のためのレーダー追跡のような天気予報技術の進歩とともに議論されている。
- 気候変動が天気予報に与える影響、アップルによるダークスカイ買収への懸念、特に低所得国における正確な予報の重要性などが議論のポイントだ。

- 特定のGPUライブラリを必要とし、セットアップ、環境設定の保存、ユーザーフレンドリーなGPU情報表示のための対話型機能を提供し、Ubuntu、Fedora、Red Hat、CentOS、OpenSUSE、Arch Linuxなどのディストリビューションと互換性があります。
- GPLv3ライセンスに基づくNVTOPは、NVIDIA、AMD、IntelのGPUバッキングでソースからコンパイルすることができます。

- ユーザーからは、Linux GPUサポート、ビデオデコーディング、MacOSツールの利用可能性、AI/MLパフォーマンスプロファイリングや多様なGPUデータ読み取りライブラリの提案などについての問い合わせがある。

## [浮動小数点精度がマンデルブロ集合画像に与える影響を探る](https://github.com/ProfJski/FloatCompMandelbrot)

- FloatCompMandelbrotは、Boost::MultiprecisionとRayLibを利用したプログラムで、浮動小数点の精度がマンデルブロ集合の視覚にどのような影響を与えるかを紹介しています。
- 様々な浮動小数点実装の画像を比較することができ、不正確さがマンデルブロ集合の複雑さに与える影響を明らかにすることができます。
- 研究用に設計されたこのツールは、浮動小数点型がフラクタル画像表現にどのような影響を与えるかを分析・理解するための詳細なコントロールを提供する。

### [反応](https://news.ycombinator.com/item?id=39683455)

- github.com/profjskiのディスカッションでは、浮動小数点数の精度がマンデルブロ集合の画像に与える影響について掘り下げ、fused-multiply-addやディープズームレンダリングなどのテクニックを探求している。
- トピックは、周期検出、ポジット、4倍精度浮動小数点演算、数学とコンピュータ・サイエンスの融合などである。
- 高精度が要求されるマンデルブロ集合画像の精密な計算における課題と、浮動小数点演算の限界について概説し、トラブルシューティングにおける浮動小数点数の有用性と、XKCDのコミックを含む浮動小数点演算に関するリソースの引用を紹介する。
