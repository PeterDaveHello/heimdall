## [フリッパー・ゼロ：ハッカーとギークのための多用途マルチツール](https://flipperzero.one)

- フリッパー・ゼロは、ペンテスターやギークのために設計されたポータブル・マルチツール・デバイスで、無線プロトコル、アクセス・コントロール・システム、ハードウェアなどのデジタル・システムをハッキングするためのカスタマイズとオープンソース機能を提供する。
- 1.4インチのモノクロ液晶ディスプレイ、5ボタンの方向パッド、Bluetooth、NFC、赤外線などの各種接続オプションを備えている。
- このデバイスはRFIDカードの読み取りとエミュレーションが可能で、ストレージ用にmicroSDカードスロットを備え、自律的に制御することも、コンピューターやスマートフォンに接続して高度な機能を利用することもできる。

### [反応](https://news.ycombinator.com/item?id=39084137)

- フリッパー・ゼロは、IRリモコンとして機能するなど、さまざまな機能を備えた多用途マルチツールだ。
- IRブラスターを搭載した他の代替品や中国ブランドと比較して、その手頃な価格について議論が巻き起こっている。
- このデバイスは、iPhoneをクラッシュさせたり、アマゾンに禁止されたり、データ交換機能やプライバシーへの影響への懸念など、論争や制限に直面してきた。

## [GPT-3.5の脆弱性：スペルミスのフレーズでエラーメッセージが表示され、データポイズニングの懸念が生じる](https://iter.ca/post/gpt-crash/)

- ChatGPT 3.5のレスポンスの先頭に特定のフレーズを追加すると、一貫してエラーメッセージが表示されます。
- このエラーはGPT-4やGPT-3.5のAPIでは発生しません。
- このエラーは、OpenAIのGPTモデルで使用されているトークン化処理に起因していると思われます。
- 特定のフレーズは、Katalon Studio ソフトウェアの XML 設定ファイルに見られるスペルミスのオプション名です。
- GPT-3.5のトレーニングに使用されたデータセットがXMLファイルを除外している可能性があり、スペルミスのトークンにモデルが気づかず、無効なレスポンスを生成している可能性があります。
- この発見は、潜在的なデータポイズニングとGPT-3.5の要約機能の弱体化についての懸念を引き起こす。

### [反応](https://news.ycombinator.com/item?id=39086106)

- AI言語モデル、特に特定のトークンによる不具合、トークン生成、テキスト変換への影響に焦点を当てて議論する。
- 会話には、トレーニングセット、トークナイザー、GPT-3.5とGPT-4モデルの外観と機能などのトピックが含まれています。
- ユーザーは、視覚的な問題、スピード、APIを使ったGPT-4の有用性、そしてOpenAIのアプローチや思考を証明する能力に対する懐疑的な見方など、さまざまな意見を持っている。

## [科学者ら、癌の燃料となるタンパク質を標的とするペプチド化合物を開発](https://phys.org/news/2024-01-scientists-chaotic-protein-fueling-cancers.html)

- カリフォルニア大学リバーサイド校の研究者たちが、ガン細胞によく見られるMYCというタンパク質を標的とするペプチド化合物を開発した。
- 研究者たちは、ペプチドの構造を改変することによって、MYCとの相互作用能力を高め、MYCと強く結合できるようにした。
- この開発は、新しいガン治療法の開発に有望であり、研究者たちはペプチドの細胞内への侵入能力を向上させることに焦点を当てている。

### [反応](https://news.ycombinator.com/item?id=39084146)

- 科学者たちは、癌の75％に関与するMYCタンパク質の活性を阻害する分子を発見した。
- この発見により、がんに対する治療法の選択肢が生まれる可能性があるが、その有効性、送達方法、潜在的な副作用はまだわかっていない。
- これは有望な開発ではあるが、まだ初期段階であり、人体実験は行われていないことに注意する必要がある。がん研究は複雑な分野であり、成功が保証されているわけではないが、がん治療を進歩させる上で重要な役割を果たしている。

## [グーグルがインコグニートモードでユーザーを追跡していることを認める。](https://news.abplive.com/technology/google-incognito-new-disclaimer-canary-version-you-are-being-tracked-even-in-google-incognito-mode-new-disclaimer-is-up-privacy-lawsuit-1658159)

- グーグルはインコグニートモードの免責条項を更新し、プライバシーの主張にもかかわらずユーザーデータが収集されることを明確にした。
- 更新された免責条項では、同じ端末を使用している他のユーザーがユーザーの行動を見ることはないが、訪問したウェブサイトや利用したサービス（グーグル自身を含む）によってデータは収集されるとしている。
- これは、グーグルがインコグニート・モードであってもユーザーの行動を追跡しているという疑いを裏付けるものである。

### [反応](https://news.ycombinator.com/item?id=39078417)

- シークレット・モードにおけるグーグルの追跡行為と、ユーザーのプライバシーへの潜在的な影響については、現在も議論が続いている。
- 一部のユーザーは、シークレットモードでもグーグルが自分の行動を追跡していると主張し、他のユーザーは追跡の範囲や非シークレットセッションとの関連性に疑問を呈している。
- プライバシーを維持するための提案としては、代替ブラウザやプライバシー重視のツールを使用することなどが挙げられ、シークレットモードの制限についてグーグルからの明確な説明が求められている。

## [バッファローフィッシュは老化に逆らう：100歳を過ぎても元気な個体もいる](https://www.nationalgeographic.com/premium/article/buffalofish-oldest-hundred-years-mystery)

- アリゾナ州のアパッチ湖で行われた最近の研究で、淡水魚の一種であるバッファローフィッシュは100年以上生きることができ、年齢を重ねても強い免疫システムとストレス反応を持っていることが発見された。
- 以前は、バッファローフィッシュは20代半ばまでしか生きられないと考えられていた。
- この研究結果は、バファローフィッシュの寿命に関する研究が進めば、人間を含む脊椎動物の寿命を延ばすためのヒントが得られる可能性を示唆している。さらに、バッファローフィッシュの個体数は減少傾向にあり、現在その捕獲に対する規制はないため、この研究は保護活動の必要性を強調している。

### [反応](https://news.ycombinator.com/item?id=39078061)

- このスレッドでは、魚種の寿命、不老不死が社会に与える影響、延命技術の進歩、人口過剰への懸念など、さまざまなトピックが取り上げられている。
- 適応能力の重要性、社会の公平性、政府の介入の必要性を強調している。
- 会話には、真剣な議論とともにユーモアや皮肉も含まれる。

## [NPS Data APIで国立公園局のデータにアクセスする](https://www.nps.gov/subjects/digital/nps-data-api.htm)

- 国立公園局（NPS）データAPIは、開発者がアプリや地図、ウェブサイトなどのプロジェクトでNPSデータを使用するために利用できます。
- APIは、アラート、記事、キャンプ場、イベント、レッスンプラン、ニュースリリース、公園、ビジターセンターなど様々な情報を提供しています。
- 開発者は NPS Data API ウェブサイトで API リソースにアクセスし、API キーを登録することができる。
