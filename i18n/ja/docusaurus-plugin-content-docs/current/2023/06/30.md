## 2023 年 6 月 30 日

### [最高裁が大学入試におけるアファーマティブ・アクションを否定](https://www.latimes.com/politics/story/2023-06-29/supreme-court-strikes-down-affirmative-action-in-college-admissions)

- 最高裁は、人種を入学者選抜の判断材料とする大学のアファーマティブ・アクション(差別是正措置)政策を、人種による違法な差別であり、憲法修正第 14 条に違反するとの判決を下し、これを取り消した。
- ジョン・G・ロバーツ・ジュニア裁判長は、憲法は人種によって人を差別的に扱うことを禁じており、学生は人種によってではなく、個人としての経験に基づいて扱われるべきであると述べた。
- カリフォルニア州では、公立大学において人種を入試要因として使用することをすでに禁止しているため、今回の判決の影響は限定的なものになりそうだが、今回の判決により、私立大学にも禁止事項が拡大されることになる。

#### 業界の反応](http://news.ycombinator.com/item?id=36520658)

- 大学入試におけるアファーマティブ・アクション(差別是正措置)に関する最高裁判決は、不平等への対処の有効性について議論を巻き起こしている。
- 人種を唯一の焦点とすべきではなく、代わりに社会経済的要因を考慮すべきだという意見もある。
- この議論では、歴史的不正義に対処することの複雑さ、差別の背景を理解することの重要性、不利な立場にあるコミュニティの教育と機会を改善する必要性が強調されている。

### [Valve はもう AI が生成したコンテンツでゲームを公開する気はないのか](https://old.reddit.com/r/aigamedev/comments/142j3yt/valve_is_not_willing_to_publish_games_with_ai/)

- Valve は、知的財産権に関する潜在的な法的問題を理由に、AI が生成したコンテンツを含むゲームのパブリッシングを行う意思がなくなった。
- この決定は、あるデベロッパーが AI 生成アセットを含むゲームをリリースしようとしたところ、Valve からそのデベロッパーはアートアセットに必要な権利を持っていないというメッセージを受け取ったことによる。
- Valve のこの動きは、AI が生成したコンテンツにまつわる著作権侵害の潜在的な問題に慎重になっていることを示唆しており、ゲーム開発における AI の将来にも影響を与えるかもしれない。

#### 業界の反応](http://news.ycombinator.com/item?id=36522665)

- Valve は今後、学習データの所有権を証明しない限り、AI が生成したコンテンツを含むゲームを公開しない。
- 著作権違反による法的責任を回避するための決定だ。
- Valve の方針は著作権保護への配慮を反映したもので、人間が生成したコンテンツと機械が生成したコンテンツの違いについて疑問を投げかけている。

### [Awk プログラミング言語第 2 版](https://awk.dev/)

- 1988 年の初版以来、Awk 言語とコンピューティング世界の変化を反映した "The AWK Programming Language" の第 2 版が間もなくリリースされます。
- この本には、Awk と関連するトピックに関する歴史的な文書、コード・スニペット、エッセイなどが含まれ、読者にさらに興味深く有用な資料を提供します。
- Awk には複数の実装があり、読者は GitHub 上の Awk ソースや GNU のウェブサイト上の Gawk リリースにアクセスできるほか、Arnold Robbins がまとめた他の Awk 実装のリストも見つけることができる。

#### [業界の反応](http://news.ycombinator.com/item?id=36517525)

- The Awk Programming Language」の第 2 版がリリースされ、Brian Kernighan によって更新と再構成が行われました。
- この本の初期の章では、特に CSV ファイルを使った実践的な探索的データ処理を取り上げている。
- Gawk と awk に新しい "--csv" オプションが追加され、適切な CSV 入力モードが使えるようになり、引用符で囲まれたフィールドや複数行のフィールドの解析ができるようになります。

### [Python 標準ライブラリに隠された CLI ツール](https://til.simonwillison.net/python/stdlib-cli-tools)

- Python 標準ライブラリには、`python -m` コマンドを使ってターミナルから直 接アクセスできる隠れた CLI ツールが含まれています。
- これらのツールの例としては、ファイルを解凍したり、ローカルのウェブサーバを実行したり、JSON をきれいに印刷したり、カレンダーを生成したりする機能があります。
- これらのツールは、追加のソフトウェアをインストールすることなく、特定のタスクを素早く実行したい開発者にとって便利である。

#### [業界の反応](http://news.ycombinator.com/item?id=36515531)

- Python の標準ライブラリには、あまり知られていない CLI ツールが含まれています。
- 一例として、正規表現ベースのトークナイザである re.Scanner ツールがあるが、これは公式ドキュメントには記載されていない。
- これらの CLI ツールは、JSON のプリティプリントやシンプルな HTTP サーバーの実行など、様々なタスクに使用できる便利で強力な機能を提供します。

### [モンスター重力波を初めて発見](https://www.nature.com/articles/d41586-023-02167-7)

- 研究者らは、銀河系近傍のビーコン星であるパルサーを使って、初めてモンスター重力波を検出した。
- この波は 2015 年に検出された波よりもはるかに大きく、長く、波長は数十光年にも及ぶ。
- この発見は、宇宙に存在する何千組もの超大質量ブラックホールとその合体の証拠となる可能性があり、レーザー干渉計宇宙アンテナ(LISA)ミッションによってさらに研究が進められる。

#### [業界の反応](http://news.ycombinator.com/item?id=36520925)

- 科学者たちは重力波を検出し、アルバート・アインシュタインの一般相対性理論の主要な予言を確認した。
- 干渉計を用いたこれらの波の検出は、ブラックホールと中性子星の存在に対する強力な証拠となる。
- この画期的な発見により、宇宙における激しくエネルギッシュなプロセスの研究が可能になり、宇宙に対する我々の理解に影響を与えることになる。

### [アスパルテーム甘味料、WHO が発がんリスクの可能性を報告へ](https://www.theguardian.com/society/2023/jun/29/aspartame-artificial-sweetener-possible-cancer-risk-carcinogenic)

- 世界保健機関(WHO)のガン研究部門が、何千もの製品に使用されている人工甘味料アスパルテームを "ヒトに対する発ガン性の可能性がある "と宣言する準備をしていると報じられている。
- アスパルテームは 1980 年代から広く使用されており、入手可能な証拠を検討した規制当局によって世界的に使用が許可されている。
- アスパルテームが発がんリスクに与える潜在的影響について疑問を呈する既存の証拠はあるが、確固たる結論を出すにはさらなる研究が必要である。

#### [業界の反応](http://news.ycombinator.com/item?id=36519942)

- 広く消費されている人工甘味料であるアスパルテームは、フランスの研究に基づき、世界保健機関(WHO)によって発がんリスクの可能性があると分類される可能性がある。
- 専門家の間では、アスパルテームの発がん性の可能性と研究結果について議論がある。
- 現在進行中の議論は、アスパルテームやその他の物質のリスクレベルに関する明確なガイドラインとコミュニケーションの重要性を浮き彫りにしている。

## [法案 C-18：グーグル、オンラインニュース法をめぐりカナダでニュースリンクを削除へ](https://www.ctvnews.ca/politics/google-set-to-remove-news-links-in-canada-meta-ends-contract-for-journalism-fellowship-as-bill-c-18-fallout-continues-1.6461557)

- ビル C-18 と呼ばれる新しいオンラインニュース法のため、グーグルはカナダでニュースリンクを削除する。
- グーグルによるこの動きは、ハイテク企業がニュース出版社にコンテンツの対価を支払うことを義務付ける法律の要件に対応するものである。
- グーグルによるニュースリンクの削除は、テクノロジーに詳しい人々の間で関心と懸念を呼んでいる。

#### [業界の反応](http://news.ycombinator.com/item?id=36524662)

- カナダの法案 C-18 は、グーグルとメタを対象に、ニュースリンクの削除とニュース出版社へのロイヤリティ支払いを求めている。
- 批評家たちは、この法律はインターネットのオープンな性質を損ない、情報へのアクセスを妨害すると主張している。
- この法律は、「ニュース・コンテンツ」の定義、カナダの報道機関への影響、他人の著作物を収益化することの倫理的意味について疑問を投げかけている。

### [プログラミング言語の上を行く](https://whileydave.com/2023/06/27/programming-languages-going-above-and-beyond/)

- 著者は、既存のプログラミング言語の限界について論じ、プログラミング言語の改善の可能性を強調している。
- 著者は、コンパイル時に後条件の検証を可能にする、高度な静的チェック機能を提供する言語、Dafny を紹介している。
- Dafny は、大規模なテストを必要とせず、検証可能な正しい関数の開発を可能にし、ソフトウェア開発におけるユニークで強力なツールとなる。

#### [業界の反応](http://news.ycombinator.com/item?id=36519497)

- Dafny は形式検証をサポートするプログラミング言語であり、プログラムに関する特性を証明することで、ソフトウェアの信頼性と正しさを向上させることができる。
- Ada、SPARK、ACL2 のような他のツールや言語も形式検証をサポートしており、実世界で応用されている。
- 関数型プログラミング、静的型付け、ロバストなコードを書くことの実用性と利点についての議論。
