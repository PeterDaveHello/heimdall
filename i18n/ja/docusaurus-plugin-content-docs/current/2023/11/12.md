# 2023-11-12

## [エクスペリアンで驚くほど簡単になりすましが可能：脆弱なセキュリティ対策が露呈](https://krebsonsecurity.com/2023/11/its-still-easy-for-anyone-to-become-you-at-experian/)

- 著者のエクスペリアン・アカウントはハッキングされ、個人情報と別の電子メールアドレスを使って再作成することができた。
- アカウント作成時の認証プロセスは脆弱で、公的記録に基づく簡単に答えられるセキュリティ質問が用意されている。
- エクスペリアンは、アカウント変更の際、本人確認を必要とせずに通知メールのみを送信するため、セキュリティ上の脆弱性が生じる可能性がある。

### [反応](https://news.ycombinator.com/item?id=38232767)

- プライバシー、個人情報盗難、信用調査機関、サイバーセキュリティ、信用調査業界を取り巻く懸念や不満に焦点を当てたディスカッション。
- これらの問題に対処するため、より強力な規制、説明責任、代替システムの必要性を示唆している。
- また、なりすまし、クレジットスコアへの影響、詐欺の解決、本人確認、メールアドレスの取り違えなどの話題にも触れている。全体として、より良いプライバシー保護、セキュリティ対策、クレジット業界の改革が求められている。

## [カナートと呼ばれる古代の地下水システムは、乾燥地域に持続可能な水管理をもたらす](https://en.wikipedia.org/wiki/Qanat)

- カーリーゼとも呼ばれるカナートは、ポンプを使わずに地下水源から地表に水を運ぶ古代の地下水システムである。
- 約3000年前にイランで生まれたカナートは、北アフリカや中東のいくつかの国で、乾燥地帯に信頼できる水を供給するために利用されている。
- カナートには灌漑、飲料水供給、冷却、氷の貯蔵などさまざまな用途があり、持続可能で効果的な水管理システムであることが証明されており、砂漠化を防ぎ、土壌の塩分濃度をコントロールしている。

### [反応](https://news.ycombinator.com/item?id=38232243)

- このHacker Newsスレッドでは、古代ペルシャの技術であるカナートとヤフチャル、そしてその歴史的意義について探求している。
- ペルシャ語と文化的アイデンティティを指す用語、特に "ペルシャ語 "と "ペルシア語 "の使い分けについて掘り下げた。
- その他にも、カナートの故意による損傷に対する脆弱性、他の水輸送手段との比較、水不足によるカナートの衰退、各地域における水管理、「カナート」の語源、将来的な進歩の可能性などが取り上げられている。

## [匿名」データの幻想を覆す：なぜ私たちのプライバシーは危険にさらされているのか](https://www.eff.org/deeplinks/2023/11/debunking-myth-anonymous-data)

- 本書では、直接的な識別子を除去するだけの方法を批判し、間接的な識別子や個人属性を匿名化する、より高度なアプローチを提唱している。
- コメント欄では、プライバシーと有用性のバランス、真の匿名化を達成することの難しさ、仮名化されたデータに関連する潜在的なリスクについての懸念が提起されている。また、データ収集における真の匿名性を確保するための課題とともに、データプライバシーに関する法的・倫理的側面についても議論されている。

## [Microsoft Outlook、ユーザーデータをサーバーに送信で批判：プライバシーに関する懸念とGDPR違反](https://mailbox.org/en/post/warning-new-outlook-sends-passwords-mails-and-other-data-to-microsoft?nl=e)

- マイクロソフト・アウトルックの最新版は、パスワードや電子メールを含む機密ユーザー・データをマイクロソフト社のサーバーに送信し、ユーザーのプライバシーを損なう可能性があるとして反発を受けている。
- このようなデータ送信は、法人顧客向けのGDPR規制にも抵触する可能性があり、マイクロソフトは法的措置に直面するリスクを負うことになる。
- ドイツのITポータルサイト『Heise Online』は、新しいアウトルックの使用を強く勧め、潜在的なプライバシー・リスクを避けるために、サンダーバードやウェブメールの使用といった代替手段を提案している。
- ドイツ連邦データ保護・情報自由委員会もこの問題に警鐘を鳴らしており、欧州レベルでこの問題を取り上げ、ユーザーのプライバシーに関する懸念に対処する意向だ。

### [反応](https://news.ycombinator.com/item?id=38233694)

- アウトルックの最新バージョンは、パスワードや電子メールを含む個人データをマイクロソフト社のサーバーに送信しており、同社のデータ収集慣行に対する懸念を促している。
- ユーザーは、新しいメールプログラムに広告が含まれていることや、従来のメールクライアントからウェブアプリケーションへの移行に不満を抱いている。
- 職場でOutlookが広く使われているため、このような懸念や不満にもかかわらず、多くのユーザーは新バージョンを使うしかない。
