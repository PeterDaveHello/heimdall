### [内部告発者は社会の良心でありながら、深刻な苦しみを抱えている](https://covertactionmagazine.com/2023/04/25/whistleblowers-are-the-conscience-of-society-yet-suffer-gravely-for-trying-to-hold-the-rich-and-powerful-accountable-for-their-sins/)

- 内部告発者は、富裕層や権力者の責任を追及するため、社会にとって重要な存在ですが、その努力のために大きな被害を受けることも少なくありません。
- より多くの人々が不正行為に対して名乗り出ることを奨励するために、立法府が内部告発者を保護することが不可欠です。
- 著者の Ashley Gjøvik は、弁護士、活動家、作家、そして人権と公益を専門とする元通報者です。

#### [業界の反応](http://news.ycombinator.com/item?id=35762780)

- 権力構造は、自分たちに逆らう内部告発者を容認しにくく、折檻や組織内からの支持の欠如につながる。
- 内部告発に関する議論には、倫理、情報機関の役割、PRISM のようなプログラムの合法性と道徳性などが含まれる。

### [マイクロソフトは、デバイスの修理を容易にするための法案を静かに支持した](https://grist.org/technology/microsoft-right-to-repair-quietly-supported-legislation-to-make-it-easier-to-fix-devices-heres-why-thats-a-big-deal/)

- マイクロソフトは、ワシントン州で公正修理法(Fair Repair Act)を静かに支持し、機器の修理をより身近なものにしようとする修理権法案に対する典型的なハイテク企業の反対を退けました。
- 修理擁護団体によると、マイクロソフトがこの法案を支持したことで、他の大手ハイテクメーカーが初めて交渉のテーブルに着くことになったといいます。
- フェア・リペア法は、議員の反対によりワシントン州上院で頓挫しましたが、マイクロソフトがデバイスの修理に関与したことは、修理権に対する業界の姿勢に変化をもたらしたと言えます。

#### [業界の反応](http://news.ycombinator.com/item?id=35758842)

- Microsoft は、Surface Laptop 4 に交換可能な SSD を搭載するなど、自社のハードウェアの修理を容易にすることを公約しており、これは、Apple や Google を含むテック企業が、修理権運動を受けて同様の公約を行う幅広いトレンドの一部となっています。
- Windows 11 は、広告の存在が大きく、Windows 10 よりも使い勝手の悪い UI、ハードウェア要件、プリインストールされたガラクタウェアの多さなどの批判にさらされていますが、可変リフレッシュレートのサポートや Intel Thread Director など、バックエンドの改善も行われています。

### [Just Simply - ドキュメントで物事がいかにシンプルであるかを言うのをやめる](https://justsimply.dev/)

- 技術文書で「簡単」という言葉を使うと、苦労している読者を見下し、不快にさせることがあります。
- 技術文書を読んでいる人は、たいてい新しい概念を学ぼうとしているのですから、「簡単であるべきだ」と暗示をかけて気分を悪くさせるのは、何の役にも立ちません。
- ドキュメントでコードに対する興奮や熱意を共有することは重要ですが、慇懃無礼と解釈されるような言葉を使わず、そうすることが可能なのです。

#### [業界の反応](http://news.ycombinator.com/item?id=35759449)

- 明確で簡潔な文書が重要ですが、親しみやすいトーンも効果的です
- シンプル」「簡単」といった大げさな言葉を使うと、分かりやすさが阻害され、誤解を招くことがある

### [エンジニアが「永遠の化学物質」を除去する水ろ過を開発](https://news.ubc.ca/2023/03/22/new-ubc-water-treatment-zaps-forever-chemicals-for-good/)

- ブリティッシュ・コロンビア大学のエンジニアは、水源に存在するすべての PFAS 粒子を捕捉して破壊できる独自の吸着材を用いて、「永遠の化学物質」とも呼ばれる PFAS を飲料水から除去する効果的な水処理法を開発しました。
- Madjid Mohseni 博士と彼のチームが開発した技術は、PFAS 粒子を最大 99%捕捉し、再生して再利用できる可能性があるため、有毒廃棄物という環境問題を解消することができます。
- この新しい水ろ過技術は、PFAS を除去できる高価な解決策を実施する資源がない小規模コミュニティの人々にとって特に有益であり、分散型および家庭内水処理において特に有用であると考えられます。

#### [業界の反応](http://news.ycombinator.com/item?id=35764476)

- フォーエバーケミカルは分解されにくく、環境中に蓄積される可能性があり、人や動物の健康被害との関連性が指摘されている
- 新システムは、プラズマ技術を利用して大流量の水をリアルタイムで処理するもので、飲料水、排水、農業排水の処理に役立つという。

### [ピュアシュバイブル](https://github.com/dylanaraps/pure-sh-bible)

- "Pure sh Bible "は、スクリプトから不要な依存関係を取り除き、高速化することを目的とした、外部プロセスに代わる POSIX sh のコレクションです。
- 本書では、文字列、ファイル、ファイルパス、ループ、変数、条件式、算術演算子に関する話題を取り上げ、一般的な Unix コマンドの代替となる例を紹介しています。
- 本書では、テキストの書式設定や色の設定には、tput よりも生のエスケープシーケンスの使用を推奨しています。

#### [業界の反応](http://news.ycombinator.com/item?id=35765707)

- より複雑なスクリプトには Python や Perl を使うことを推奨するユーザーもいれば、シェルスクリプトを有力な選択肢として擁護するユーザーもいます
- eval の使用やシェル特有のエスケープシーケンスに依存するような危険な行為を避けることの重要性についても言及されています。

### [FAA がエンジニアを制圧、ボーイング・マックスの飛行を継続させる](https://apnews.com/article/boeing-max-crashes-faa-ea8fac0ad2758b08c58d2d64756fbf37)

- 運輸省の監察官事務所による調査では、今後同様の事故を防ぐために、重要かつ緊急な安全上の決定がどのようになされるかを FAA が文書化するよう勧告しています。

### [開発中のお気に入りの C コンパイラのフラグ](https://nullprogram.com/blog/2023/04/29/)

- この記事では、GCC、Clang、MSVC の開発者向けに、警告、静的および動的なランタイムチェック、デバッグ情報に焦点を当てた重要な C コンパイラフラグについて説明します。
- Wall -Wextra や-Wdouble-promotion などのフラグを有効にすることで、静的にバグを検出することができます。
- アドレスサニタイザーや未定義動作サニタイザーなどのサニタイザーを使用すると、動的なランタイムエラーやデータレースを検出することができます。

#### [業界の反応](http://news.ycombinator.com/item?id=35758898)

- フラグの中には、パフォーマンスの向上、シンボルの再エクスポート、C コンパイラのアンブレイクを目的としたものがある
- 開発者は、C および C++プログラムのメモリバグを見つけるために、開発プロセスの早い段階で頻繁にサニタイザーを使用することを推奨しており、特定のフラグと設定が強調されています。
