## [傷跡の組織は関係を摩耗させる (2013)](https://gist.github.com/gtallen1187/27a585fcf36d6e657db2)

- 人間関係は、個人的にも仕事的にも、維持するのが難しいものです。
- 傷跡組織、つまり蓄積された小さな迷惑が、人間関係を破綻させる主な原因であり、単一の破滅的な出来事ではない。
- 人間関係に瘢痕組織を作らないためには、長引く反感をゼロにすることが必要です。コミュニケーション、妥協、そして両者が耳を傾け、妥協する姿勢が、長期的な関係を維持するために必要不可欠です。

### [業界の反応](http://news.ycombinator.com/item?id=36093954)

- この投稿では、定期的なコミュニケーションと問題への対峙を通じて、人間関係の「瘢痕組織」を予防し、個人の成長とオープンなコミュニケーションにつなげるという考え方が述べられています。
- コメントでは、人間関係を長続きさせるためには、気持ちを確認すること、早い段階で問題に対処すること、非暴力的なコミュニケーションを用いること、共有する価値観、約束、信頼に焦点を当てることが重要であると論じています。
- 著者は、共感、妥協、尊重の重要性を強調し、小さな問題の積み重ねが人間関係の破綻を招くと述べています。

## [Rust で書かれた並列ブラウザエンジン「Servo」](https://servo.org/)

- Servo は、プログラミング言語である Rust で書かれた Web レンダリングエンジンです。
- Layout と呼ばれる Servo のレイアウトエンジンは、2020 年に書き直されました。
- 最近の Servo のテストプロセスの改善により、開発者がより貢献しやすくなりました。

### [業界の反応](http://news.ycombinator.com/item?id=36092852)

- Rust と Servo は密接な関係にあり、Rust は Servo と並行して開発され、Servo をそのアプリケーションとして使用しています。
- 現在のウェブスタックには、Svelte、Astro、Gemini といった、軽量かつ低消費電力なソリューションを提供する代替品があります。
- Servo は Rust の初期開発に欠かせない存在で、Layout 2020 を改良して flexbox、グリッド、信頼性の高い float の実装をサポートするよう取り組んでいます。
