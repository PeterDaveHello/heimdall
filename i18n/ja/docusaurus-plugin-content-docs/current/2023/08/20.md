# 2023-08-20

## [アムステルダム、大音量の車に対する「騒音カメラ」の導入へ](https://nltimes.nl/2023/08/11/amsterdam-use-noise-cameras-loud-cars)

- アムステルダムは、市内で騒々しい車の問題に対処するため、「騒音カメラ」を導入した。
- この標識は後に騒音カメラに置き換えられ、違反したドライバーに自動的に罰金を科すことになる。
- 騒音標識はまず2カ所でテストされ、今月中により多くの騒音計と標識を設置する予定だ。この取り組みは、市内での騒音障害を最小限に抑えることを目的としている。ロッテルダムもアムステルダムと提携して同様の実験を行っている。

### [反応](https://news.ycombinator.com/item?id=37187837)

- アムステルダム市は、市内の自動車やオートバイによる過度の騒音問題に取り組むため、「騒音カメラ」の導入を計画している。
- この問題をめぐる議論の焦点は、大音量のオートバイの必要性と影響であり、安全上の利点を主張する者もいれば、その迷惑性を批判する者もいる。
- また、騒音公害、歩行者の行動、車両の改造、より静かで電気的な代替手段の促進など、より広範な議論も行われている。
- 住民たちは、より静かな都市環境を作るために、ガスエンジン車の禁止と電動モペッドの普及を主張している。

## [古い本はなぜあんなにいい香りがするのだろう？](https://scienceswitch.com/2023/08/19/why-do-old-books-smell-so-good/)

- 科学者たちは、古い本が放つ独特の匂いを研究し、放出される揮発性有機化合物（VOC）を使って、その年代、状態、歴史を割り出している。
- 古書の臭いは、紙やその他の有機化合物が分解され、ベンズアルデヒド、バニリン、エチルベンゼン、2-エチルヘキサノールといった特定の化合物が放出されることによって生じる。
- VOC分析は、図書館員や歴史家が保存の必要性を特定し、孤児となった本を元の図書館と照合するのに役立つ。

### [反応](https://news.ycombinator.com/item?id=37188015)

- Hacker Newsでは、古い本の香り、テクノロジーと結びついたノスタルジー、テクノロジーの進歩、健康への懸念、物理的な本と電子書籍の間で進行中の議論など、さまざまなトピックについて議論している。
- デジタルノーズのコンセプトと匂いの合成の可能性についても簡単に触れている。

## [Rivet - Nomad と Rust によるオープンソースのゲームサーバー管理](https://github.com/rivet-gg/rivet)

- Rivetはオープンソースのゲームサーバー管理ツールで、インフラ経験のないゲーム開発者向けに専用サーバーのデプロイを簡素化するように設計されています。
- 主にRustとLuaを使って作られており、他のツールやライブラリも統合されている。
- Rivetは、エッジプロキシ、キャッシング、分析、オーバーレイネットワーク、Redis Luaスクリプトなど、さまざまな機能をサポートしている。
- Apache 2.0でライセンスされ、アクセスしやすく、包括的なゲームサーバー・マネージャーを目指しています。
- Rivetはネットワーク・ロジックを扱わず、既存のツールと連携して動作する。
- 2022年1月からクローズドベータを開始し、複数のタイトルで数百万人の月間アクティブユーザーをサポートしている。

### [反応](https://news.ycombinator.com/item?id=37188659)

- ディスカッションでは、ゲーム開発インフラの課題とオープンソースツールの利点に焦点を当てる。
- オープンソースのゲームサーバー管理ツールであるRivetは、専用サーバーを効率的に展開できることで高く評価されている。
- 対談では、ゲーム開発にKubernetesとNomadを使用することの利点と欠点も探られ、ある開発者は自身のゲームにRivetを活用することに興味を示した。
- Rivetチームは、ローカル開発における強化の必要性を認識し、オープンソースの機能、Nakama、HashiCorpのライセンス変更、Bevyのサポートなどのトピックについて議論している。

## [セルブライト社、自社技術の秘密保持を警察に要請](https://techcrunch.com/2023/08/19/cellebrite-asks-cops-to-keep-its-phone-hacking-tech-hush-hush/)

## [WD、自己ワイプ式サンディスクSSDに関する質問への回答を拒否](https://www.theverge.com/23837513/western-digital-sandisk-ssd-corrupted-deleted-questions)

- ウエスタンデジタルのサンディスク製SSDがデータ損失の原因になっていると報じられているが、同社はこの問題についての問い合わせに回答していない。
- The Verge』や『Ars Technica』の従業員を含め、データ損失の報告が複数出ている。
- ウエスタンデジタル社は問題を認識しているにもかかわらずSSDの販売を続けており、同社の広報責任者はこの問題に対処することを拒否している。
- ウェスタン・デジタルがアマゾンの偽レビューを使い、検索結果を操作して故障を軽視しているとの疑惑が持ち上がっている。
- 同社は現在、サンディスクSSDの不具合をめぐる複数の集団訴訟に直面している。
- ウェスタン・デジタルはまた、製品ラインナップに劣悪なドライブを含めようとしたり、NASディスクに不必要な警告を表示したりするなど、欺瞞的な行為で告発されている。

### [反応](https://news.ycombinator.com/item?id=37188736)

- オンライン・フォーラムでは、SSDやHDDのようなストレージ・ソリューションの信頼性や品質に関する議論が行われており、ユーザーはさまざまなブランドやモデルに関する経験や懸念を共有している。
- ファームウェアの問題や保証クレームなどが取り上げられ、テクノロジー業界における信頼できる効率的な顧客サービスの重要性が強調されている。
- ウェスタンデジタルやサンディスクのようなブランドは、顧客からの苦情や集団訴訟による評判の低下に直面しており、ユーザーは信頼性で知られるSKハイニックスやサムスンのような代替ブランドを探すようになる。
