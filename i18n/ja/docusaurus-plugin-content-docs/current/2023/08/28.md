# 2023-08-28

## [胚性幹細胞をよりよく模倣するためにヒト細胞を再プログラムする新しい方法](https://www.uwa.edu.au/news/Article/2023/August/Scientists-find-way-to-wipe-a-cells-memory-to-reprogram-it-as-a-stem-cell)

- オーストラリアの科学者たちは、ヒト細胞を胚性幹細胞のように機能させるTNTリプログラミングと呼ばれる新しい方法を開発した。
- この方法は、細胞の記憶を消し、人工多能性幹細胞（iPS細胞）と胚性幹細胞（ES細胞）の違いを減らすことを目的としている。
- 研究者たちによれば、この画期的な進歩は細胞治療と生物医学研究を前進させる可能性があるという。

### [反応](https://news.ycombinator.com/item?id=37286860)

- 研究者たちは、ヒト細胞を初期化する新しい方法を考案し、より胚性幹細胞に近づけ、研究目的の人工多能性幹細胞（iPS細胞）を改良する可能性があることを明らかにした。
- 対談では、不健康なライフスタイルに起因する健康問題への薬物使用の可能性や、肥満が癌発生率に与える影響について掘り下げている。
- 医療介入のために幹細胞を保存すること、人間の寿命を延ばすこと、老化を食い止め、その影響を逆転させるための進行中の研究についての議論がある。

## [フォントはどこから来たのか？主にMonotypeです。](https://thehustle.co/where-do-fonts-come-from/)

- フォント業界を支配するMonotype社は、競争して生計を立てようと奮闘する独立系フォントデザイナーに難題を突きつけている。
- Monotypeの市場支配と買収は、独立したデザイナーが成功することを難しくしている。
- 独立系デザイナーもまた、大企業によるフォントの無断使用の問題に直面している。
- Monotype社は、同社のMyFontsマーケットプレイスにサブスクリプションモデルを導入する予定だが、これはMonotype社に主に利益をもたらすのではないかという懸念が持ち上がっている。
- Monotype社は、フォントの発見にAIを活用することを模索しているが、フォントデザイナーは、AIではフォントの背後にある人間のストーリーや意味を再現することはできないと主張している。

### [反応](https://news.ycombinator.com/item?id=37283860)

- フォントの起源、著作権に関する問題、独立系デザイナーへの支援など、フォントに関するさまざまな側面を網羅した議論が展開される。
- 参加者は、さまざまなオペレーティング・システムでのフォント・レンダリングや、デザイン目的でのフォントの選択と組み合わせについて語り合う。
- また、オープンソースと商用フォントの可用性、品質、効果に焦点を当てながら、フォント業界における独占と未公開株式会社の影響についても探求している。

## [111,111.1メートルは確実に緯度1度である](https://gis.stackexchange.com/a/2964/5599)

- 地理情報システム（GIS）スタックエクスチェンジは、地図製作者、地理学者、GIS専門家が質問し、答えるためのプラットフォームです。
- あるユーザーが、組込みシステムでの作業のために、緯度と経度の座標をメートル単位でオフセットするアルゴリズムを探している。
- Vincentyの直接式は、その速度と正確さで知られているため、この目的に役立つ可能性のあるアルゴリズムとして提案されている。

### [反応](https://news.ycombinator.com/item?id=37284487)

- この議論では、測定単位としての緯度1度の信頼性と、緯度経度座標の限界を探る。
- メーターの歴史と再定義を掘り下げ、メートル法とインペリアル法といった異なる計測システムの利点と欠点を浮き彫りにしている。
- アメリカにおけるメートル法への抵抗や、帝国制がもたらす混乱を取り上げる一方、海里の実在性と有用性についても論じている。さらに、単位換算やさまざまな文脈における測定の複雑さにも触れている。

## [グラスは食べられる](https://en.wikipedia.org/wiki/I_Can_Eat_Glass)

- 「I Can Eat Glass "は、インターネットの黎明期にイーサン・モリックによって始められた言語学的プロジェクトである。
- このプロジェクトは、このフレーズをさまざまな言語に翻訳することを目的としており、その結果、150以上の翻訳が生まれた。
- オリジナルのウェブページはすでに公開されていないが、このプロジェクトは人気を博し、観光客にユーモラスな翻訳を提供するインターネットミームとなった。

### [反応](https://news.ycombinator.com/item?id=37283636)

- この記事では、「I Can Eat Glass」という翻訳プロジェクトに焦点を当て、さまざまな言語でのこのフレーズの翻訳を探っている。
- 対談には、逸話や歴史的背景、翻訳間の比較などが盛り込まれ、愉快で軽快な語り口となっている。
- さらに、インターネット上のノスタルジー、ミーム・アーカイブ、ガラスを食べることの安全性といった話題にも触れており、議論に深みを与えている。

## [Python Type Hints - \*argsと\*\*kwargs (2021)](https://adamj.eu/tech/2021/05/11/python-type-hints-args-and-kwargs/)

## [スクラムは癌である](https://twitter.com/svpino/status/1695806027256475777)

- このツイートの著者はスクラムを強く批判し、それがソフトウェアチームを非効率にすると主張している。
- 否定的な意見の裏付けとなる2つの逸話がある。1つは、ポーカーをプランニングツールとして使用したこと、もう1つは、プロセスを追加することで効率が向上するという信念についてである。

### [反応](https://news.ycombinator.com/item?id=37289151)

- ソフトウェア開発チームにおけるスクラムとアジャイル手法の導入と有効性をめぐる議論がある。

- 議論されている課題には、有能なプログラマーの発掘と確保、スクラムの枠組みの中での仕事量やプレッシャーに関する懸念などがある。

## [米国にいる息子がネットで政府を批判し、母親がバングラデシュで逮捕される](https://www.nbcnews.com/news/asian-america/mother-arrested-bangladesh-son-us-criticizes-government-online-rcna101392)

- バングラデシュの母親が、息子のフェイスブックへの政府批判の投稿が原因で逮捕された。
- 米国で博士課程に在籍するその息子は、母親が拘束されたことへの不満をあらわにした。

- バングラデシュの母親が、息子がネット上で政府を批判したために逮捕された。
- ネット上での批判を理由に個人を逮捕するという問題は、バングラデシュに限ったことではなく、地政学的な意味合いを持つかもしれない。
