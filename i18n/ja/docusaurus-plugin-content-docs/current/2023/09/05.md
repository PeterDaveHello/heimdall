# 2023-09-05

## [VSCodium - VSCodeのオープンソースバイナリ](https://vscodium.com)

- VSCodiumは、マイクロソフトのVisual Studio Code (VS Code)のフリーでオープンソースの代用品で、コミュニティによって運営され、プロプライエタリなライセンスや遠隔測定/トラッキングを排除したVS Codeのバイナリを提供します。
- MITライセンスのVSコードへのアクセスのしやすさは、VSCodiumのおかげでより便利になり、Windows、Mac OS、Linuxのような複数のプラットフォームで、直接ダウンロードするか、パッケージマネージャを通してインストールするオプションがあります。
- VSCodiumは、Visual Studio Codeから移行するユーザーのために、Flatpakアプリインストールオプションと詳細なドキュメントも提供します。

### [反応](https://news.ycombinator.com/item?id=37381335)

- この対談では、ソフトウェアの信頼性、オープンソースプロジェクトの脆弱性、マイクロソフトの慣行、代替ソフトウェアに関する考察など、さまざまなトピックについて話し合われた。
- 参加者は、検証、サプライチェーンセキュリティ、データプライバシー、デジタル著作権管理（DRM）、ライセンス、拡張機能の互換性といった重要な側面を強調した。
- この議論は、プロプライエタリな慣行と、Visual Studio Codeのようなアプリケーションに見られるオープンソースの精神との間のストレスを強調するものである。

## [潮汐エネルギーは再生可能ではない](https://cs.stanford.edu/people/zjl/tide.html)

- このリストには、低温および高温超伝導、超流動、電子トンネリングの理論など、さまざまなトピックを論じた科学論文が含まれている。
- LK-99の重要性と限界、ブラウン運動の背後にある説明、クルックス放射計を駆動するプロセスについても検証している。
- このリストでは、熱力学の誤解に挑戦し、地球の磁場が超伝導体である可能性を探り、潮汐エネルギーが再生可能でない理由を論じ、相対性理論を理解し、E=mc2を導き出す方法を提示している。

- 主なトピックは、持続可能性への障害、人口増加、出生率、将来のエネルギー需要を満たすための戦略、エネルギー消費が環境の安定と経済発展に与える影響などである。
- その他の分野としては、不平等の持続、エネルギー採掘が地球の自転と気候に及ぼす影響、風パターンがエネルギー発電に及ぼす影響、不動産と石油生産産業のダイナミクスなどがある。
