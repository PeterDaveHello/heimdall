## [私の知る最悪のプログラマー](https://dannorth.net/2023/09/02/the-worst-programmer/)

- ダン・ノースのブログ記事は、開発者の生産性を測るという複雑な問題に取り組み、個々のタスクの完了から累積的なビジネスインパクトへの重要な視点の転換を強調している。
- ティム・マッキノンというプログラマーは、孤独な作業よりもグループでの問題解決に重点を置いていたため、従来の生産性指標では常に劣勢に立たされていた。
- ティムを追放せよという経営陣の圧力にもかかわらず、ノースはチーム全体の有効性と生産性を高める上で極めて重要な役割を果たしたと弁明し、生産性は複雑なシステムにおける具体的なビジネスインパクトによって測られるべきであると強調した。

- このポストでは、知識の共有、批判や社内政治の管理、知覚がキャリアアップに与える影響の理解など、ソフトウェア開発業界における重要なトピックを探求している。
- プロジェクト管理のためのスクラムや、ジュニア／シニアエンジニア、テクニカルリード、プロダクトオーナー、プロダクトマネージャーの役割など、業界における具体的な役割や方法論が語られている。
- また、効果的なコミュニケーションとコラボレーションの重要性を強調し、協力的な職場環境を培っている。社内政治の複雑さ、組合結成の是非、コード貢献の課題についても論じている。

## [納税準備会社：無料申告に反対するロビー活動に9000万ドル](https://www.opensecrets.org/news/2023/09/tax-prep-companies-lobbying-against-free-file-face-scrutiny-from-lawmakers/)

- ターボタックスを所有するインテュイット社やH&Rブロック社などの大手税務申告会社は、政府が運営する無料の税務申告システムに反対するロビー活動を行ったとして、監視の対象となっている。
- これらの企業は、IRSが独自の納税準備ソフトを開発するのを阻止するために、2003年以来ロビー活動に総額9000万ドル以上を費やしてきた。
- 2019年12月現在、Free File Allianceの覚書の補遺により、その妨害は取り除かれ、IRSは2024年に無料申告サービスの計画を立てることができるようになった。税務申告業界のロビー活動に対する調査は、複数の議会民主党議員によって要求されている。

### [反応](https://news.ycombinator.com/item?id=37363616)

- 米国の税務申告会社は、無料申告オプションに反対するロビー活動に9000万ドルを費やし、税制の複雑さと税務申告会社の役割について議論を巻き起こしている。
- ロビー活動、特にインテュイットのような企業の影響力、業界崩壊の可能性、そして政府が運営する税務申告システムの潜在的な利点についての議論がある。
- また、税務調査会社の国有化案、税制簡素化の意味合い、ロー対ウェイド事件に関する最高裁判決の関連性と影響といったトピックにも取り組んでいる。

## [再帰的要約がLLMの長期対話記憶を可能にする](https://arxiv.org/abs/2308.15022)

- 本論文では、大規模言語モデル（LLM）を用いて要約や記憶を再帰的に作成することで、オープンドメイン対話システムにおける長期記憶を強化する戦略を提案する。
- この方法によって、LLMはより小さな対話の文脈を保持・想起し、その後に以前の記憶とそれに続く文脈を利用して新しい記憶を作り出すことができる。
- 実験では、この方法によって生成されたロングコンテキストの会話において、より一貫した応答が得られることが実証された。著者らは後日、コードとスクリプトを公開する予定である。

### [反応](https://news.ycombinator.com/item?id=37363362)

- 議論の焦点は、言語モデルの制約と可能性、主に推論と数学的課題における能力である。
- 談話にはAIモデルの能力も含まれ、パフォーマンス向上のための特定のツールやシステムの必要性を疑問視している。
- 会話AIシステムにおける長期記憶の役割や、再帰的要約法の効率性についても、代替戦略や一般的な実装とともに取り上げられている。
