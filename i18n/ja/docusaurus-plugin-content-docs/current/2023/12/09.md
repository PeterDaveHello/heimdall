# 2023-12-09

## [アップル、Beeper Miniのアクセスを遮断、障害とプライバシーへの懸念に飛び火](https://techcrunch.com/2023/12/08/apple-cuts-off-beeper-minis-access-after-launch-of-service-that-brought-imessage-to-android/)

- AndroidユーザーにiMessageへのアクセスを提供していた新興企業Beeperは、Appleがそのアクセスをブロックしたため、ユーザーがBeeper Miniアプリ経由でテキストを送信できない障害が発生した。
- Beeper社のCEOであるエリック・ミギコフスキー氏によると、アップル社はこのアプリを無効化することに成功し、Beeper Miniの将来に懸念を抱かせたという。
- ミギコフスキーはアップルの動機に疑問を呈した。このアプリは、特に暗号化されていないグリーンバブル・テキストに関連して、iPhoneユーザーのセキュリティを強化する方法とみなされていたからだ。

### [反応](https://news.ycombinator.com/item?id=38574888)

- ディスカッションでは、デバイスやサービスに対するアップルの管理、暗号化の慣行、広告がユーザーに与える影響など、さまざまなトピックについて掘り下げている。
- メッセージングアプリの相互運用性と、代替メッセージングプラットフォームが直面する課題についても検討する。
- これらの問題に対するさまざまな視点が提示され、ハイテク業界の複雑さを物語っている。

## [イスラエルのWhatsAppグループが親パレスチナ派の声を封殺](https://jackpoulson.substack.com/p/inside-the-pro-israel-information)

- J-Ventures Global Kibbutz Groupは、投資家、ハイテク企業幹部、活動家、政府高官で構成される親イスラエルのWhatsAppグループで、世論を形成し、イスラエル批判に対抗することを目的としている。

- このことは、ハイテク部門とイスラエル政府、そしてイスラエル擁護派との密接な関係を浮き彫りにしており、市民的自由を擁護する団体や言論の自由を擁護する人々の間に懸念を抱かせている。

- 一国家対二国家の解決策、民族国家、ハマスとリクードの役割、民間人の犠牲者、歴史的な出来事、紛争の複雑さなどが議論されている。
- アラブ連盟、カタールやトルコによるハマスへの資金提供、民間人支援、ターゲティングの正確さ、イスラエルの政治家の発言、パレスチナ民間人の扱いといったトピックも議論されている。要約では、紛争の複雑さと、強制力のある解決策を見出すことの難しさを認めると同時に、個人攻撃や扇動的な言葉を使わず、敬意ある言論の重要性を強調している。

## [隠されたメカニズムに悩まされる列車：ポーランドのハッカーが重大問題を発見](https://badcyber.com/dieselgate-but-for-trains-some-heavyweight-hardware-hacking/)

- ポーランドのある列車メーカーは、メンテナンス後に隠れたメカニズムが原因で列車が運行できなくなるという重大な問題に遭遇した。
- ポーランドのハッカー集団であるドラゴン・セクター・チームが支援を要請され、徹底的な分析とリバース・エンジニアリングの結果、列車故障の原因となったコードを特定した。
- 彼らはソフトウェアのロックを解除するツールを開発し、複数の列車セットの修理に成功したが、当局によるこの問題への対処はほとんど行われていない。この件は、CERT Polskaによって法執行機関に報告されている。

### [反応](https://news.ycombinator.com/item?id=38567687)

- ポーランドのある列車保守会社は、列車の機関車にコードをインストールし、競合する修理施設に長時間留置しておくと運行できなくなるようにしていたが、これは法的な意味合いと利益相反を伴う悪質な行為とみなされている。
- 所有する機器のソフトウェアを変更する能力、中間管理職が違法行為に関与すること、鉄道の運行を妨害した場合の潜在的な法的影響について議論がある。
- ディスカッションは、妨害行為、インターフェイスの標準化、F1レースでのソフトウェア使用、産業機械のソフトウェア開発における課題、相互運用性の問題、ソフトウェアの品質低下、ハイテク産業における文化的問題、ソフトウェアの透明性と再現性、アップルのデバイス修理をめぐる論争、妨害行為の告発、ベンダーのロックインに対する懸念、ニューラグ鉄道の安全ロックアウトをめぐる論争などのトピックに広がっていく。

## [ベライゾンのミス：ストーカーが被害者のデータを入手](https://arstechnica.com/tech-policy/2023/12/verizon-fell-for-fake-search-warrant-gave-victims-phone-data-to-stalker/)

- ベライゾン・ワイヤレス社は、ストーカー被害者の住所と電話履歴を、警察官を装って偽の捜索令状を送ったストーカー容疑者に誤って提供した。
- この事件により、ストーカーは逮捕され、ストーカー行為と詐欺罪で告訴された。
- ベライゾンは、法的要求には秘密厳守で法律を遵守して処理すると主張しているが、今回の事件は機密情報の保護に失敗したことを示している。

### [反応](https://news.ycombinator.com/item?id=38576530)

- ベライゾンは、偽の捜査令状に引っかかったために、被害者の電話データをストーカーと誤って共有し、デジタル署名されたファイルではなく、紙のプリントアウトを使用することのリスクを浮き彫りにした。
- 信頼できる認証局（CA）から証明書を取得することの複雑さと、HTTPSのような安全なプロトコルの限定的な採用について、法的文書の本人確認と認可に関連して議論する。
- 改善案としては、データプライバシーを権利として認めること、暗号署名を導入すること、透明性と説明責任を高めるために裁判所が自らのウェブサイトで令状を公開することなどが挙げられる。

## [アメリカ栗基金、病害抵抗性種子の配布を中止](https://tacf.org/darling-58/)

- アメリカ栗基金（TACF）は、栗枯れ病菌によって壊滅的な打撃を受けたアメリカ栗の木の回復に尽力している。
- TACFは、伝統的な育種やバイオテクノロジーなどさまざまな方法を用いて、病気に強く、遺伝的に多様な栗を開発してきた。
- このため、TACFは支援を打ち切り、病害耐性を向上させる別の方法を模索することになった。

### [反応](https://news.ycombinator.com/item?id=38573765)

- アメリカン・チェストナット財団は、ダーリングD58系統の遺伝的欠陥が問題となり、木の代謝コストが増加したため、病害抵抗性種子の流通を停止した。
- この事件は、野外試験を実施する前に、遺伝子組み換えの徹底的な検証の必要性を強調している。
- とはいえ、アメリカ栗の木の復元を目指した研究活動は続けられており、アメリカ栗の木の植樹に対する人々の関心も高まっている。しかし、入手可能な種子の多様性が限られていることが懸念されている。
- 種子の流通を担当する組織は、この分野での業務を停止し、リソースを他に振り向ける意向だ。
