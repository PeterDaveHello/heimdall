# 2023-12-21

## [VW、ポルシェ、アウディがテスラのEV充電規格を採用へ](https://www.theverge.com/2023/12/19/24008426/volkswagen-audi-porsche-tesla-nacs-ev-charging-plug)

- フォルクスワーゲン、ポルシェ、アウディは、テスラの電気自動車（EV）充電規格を採用し、VWの顧客にテスラのスーパーチャージャー・ネットワークへのアクセスを提供する計画だ。
- テスラの充電プラグの採用は、2025年までにアダプター・ソリューションとともに実施される見込みだ。
- テスラのスーパーチャージャー・ネットワークは、その信頼性と可用性で有名であり、他のサードパーティの充電ステーションよりも優れている。

### [反応](https://news.ycombinator.com/item?id=38709319)

- フォルクスワーゲン、ポルシェ、アウディはテスラの電気自動車用充電プラグを使用する予定だが、業界には支払いと電力線通信の標準プロトコルが存在しない。
- プラグ・アンド・チャージ・システムの利便性と安全性については議論が続いており、標準的な支払い方法を求める擁護派と、安全性に懸念を示す擁護派がいる。
- EV充電のための標準化されたユーザーフレンドリーな決済システムが必要であり、タップ・ツー・ペイ技術、ダイレクトカーペイメント、QRコード決済システムなどのソリューションが考えられる。

- Mambaモデルは、入力に基づいた選択メカニズムを持つ状態空間モデルを扱い、ハードウェアの効率性、新しいアーキテクチャ、潜在的なアプリケーションについて考察している。また、知識の蒸留と最小記述長の原則についても言及している。

## [PowerInfer：コンシューマー向けGPUで大規模言語モデルを高速処理](https://github.com/SJTU-IPADS/PowerInfer)

- PowerInferは、コンシューマーグレードのGPU上で動作する大規模言語モデル（LLM）用の高速推論エンジンです。
- 適応予測子とニューロンを意識したスパース演算子を使用し、活性化の局所性を活用することで効率を最適化する。
- PowerInferは高いトークン生成率を達成し、既存のモデルを凌駕し、精度を維持します。PowerInferは、ローカル展開をサポートし、低遅延推論を実現するように設計されている。コードとデータは順次公開され、引用と謝辞が推奨される。

### [反応](https://news.ycombinator.com/item?id=38708585)

- Hacker News』のディスカッションでは、大規模な言語モデルの最適化技術や、計算規制の法的意味合いなど、さまざまなテーマについて掘り下げている。
- モバイルゲームにおける広告やリワードに関連する弊害や、ゲームやAIのためのデュアルGPUセットアップの使用についても論じている。
- その他のトピックとしては、ディープラーニングモデルを高速化するPowerInferプロジェクト、行列計算のためのllama.cppライブラリ、CPUのみのソリューションに関する議論、マイクロソフトのPower関連製品やPowerPointの買収に関する言及などがある。

## [ワイヤーフロー：このオープンソースツールでユーザーフロープロトタイプを簡単に作成できる](https://wireflow.co)

- ユーザーは、Excalidraw、Tldraw、Figmaなどの代替ワイヤーフローツールについて議論し、設計プロセスにおけるさまざまなワイヤーフローツールの経験や課題を共有しています。
- Wireflowを他のツールと統合することを勧めるユーザーもいれば、開発に貢献したいと表明するユーザーもいる。全体として、Wireflowに対する反応はまちまちであり、その機能性や有用性についても議論が続いている。

## [ENPP1タンパク質が乳がん転移のキープレイヤーであることが判明](https://news.stanford.edu/2023/12/20/breast-cancer-metastasis-off-switch-revealed/)

- 研究者らは、乳癌の転移を制御するENPP1と呼ばれるタンパク質を同定した。
- ENPP1の高値は、免疫療法に対する抵抗性と転移リスクの増加と関連している。
- この発見は、より優れた免疫療法の開発につながり、現在の治療に対する患者の反応を予測するのに役立つ可能性がある。さらに、ENPP1を標的とすることで、既存の治療法の有効性が高まる可能性もある。ENPP1は、肺癌や膵癌のような他の "低温 "腫瘍にも関連するかもしれない。

### [反応](https://news.ycombinator.com/item?id=38715230)

- スタンフォード大学とカリフォルニア大学サンフランシスコ校の研究者らが、マウスにおける乳がん転移の「オン／オフスイッチ」の可能性を発見した。
- 別の研究では、乳がんの転移におけるENPP1タンパク質と免疫療法に対する抵抗性の相関が確認され、新たな標的療法の可能性が示された。
- これらの知見の妥当性や意義については議論が続いており、研究者間の潜在的なバイアスについての懸念も指摘されている。さらに、がん研究においてマウスをモデルとして使用することの限界が認められ、直面する課題が浮き彫りになっている。

## [キー認証によるiMessageのセキュリティ強化](https://support.apple.com/en-gb/HT213465)

- iMessage Contact Key Verificationは、iMessageの会話の信頼性を確保するためにAppleが提供するセキュリティ機能です。
- ユーザーは、認証コードを比較して連絡先を手動で確認するか、連絡先が共有する公開認証コードを使用することができます。
- この機能を使用するには、互換性のあるソフトウェアのバージョンと必要な設定が有効になっていることが重要です。ユーザーは「会話の詳細」で認証状況を確認でき、問題があればアップルに問い合わせることができる。

### [反応](https://news.ycombinator.com/item?id=38714674)

- アップルはiMessage Key Verificationを導入し、ユーザーが連絡先の身元を確認できるようにした。
- ユーザーへの普及や、検証プロセスにおける潜在的な不便さについての懸念が提起されている。
- 暗号鍵の検証の重要性と標的型攻撃のリスクについて説明する。
- 政府による監視と、それが暗号化ツールの使用に与える影響が議論されている。
- ゼロデイ攻撃から逃れるためには、OSを定期的にアップデートすることが重要だ。
- アンドロイド端末は脆弱とされ、令状なしの監視の常態化が議論されている。
- アップルのポリシーに不満を表明し、暗号化キーの認証にサードパーティ製の代替手段を提案するユーザーもいる。
- アップルのプラットフォーム上での検証方法の実装について、ユーザーエクスペリエンスや潜在的な欠点に関する懸念とともに議論する。
- iMessageとiCloud Keychainのセキュリティと暗号化について触れている。
- 全体として、セキュリティとアカウント管理に対するアップルのアプローチに不満がある。

## [スノーデン暴露の衝撃：オンライン・プライバシーの課題と進歩](https://blog.torproject.org/tor-in-2023/)

- エドワード・スノーデン氏による大量監視に関する情報公開は、オンラインプライバシーに対する人々の意識に大きな影響を与えた。
- 政府や企業のプライバシー侵害は世論を操作し、社会の重要な決定に影響を与える可能性があります。
- この認識の変化は、より強力なプライバシー保護に対する顧客の要求の高まりに火をつけ、消費者が自分のデータをよりコントロールできるようにする法律の制定につながった。
- プライバシーを優先事項として売り込んでいるハイテク企業もあるが、その根本的なビジネスモデルは依然としてデータ収集に重点を置いている。

- 議論の中心は、Tor、HTTPS暗号化、DNSSEC、データ暗号化などの一般的なツールを含む、オンライン・プライバシーのさまざまな側面です。
- HTTPS-everywhereやTorなど様々な方法の有効性や安全性については議論があり、政府の監視や潜在的な脆弱性についての懸念もある。
- I2PやFreenetのような代替ネットワークがTorに代わるものとして言及され、米国政府の信頼性や、小さな犯罪を対象とする法執行機関への懸念も取り上げられている。情報の保存と送信におけるCPUの役割も疑問視されている。

## [除草剤パラコートをめぐる法廷闘争が勃発：労働者がパーキンソン病を訴える](https://www.latimes.com/environment/story/2023-12-20/california-workers-say-herbicide-is-giving-them-parkinsons)

- 歴史的な例を挙げて、特定の物質に関連する健康リスクに対する懸念の欠如を強調している。
- 対談では、特定の食事がもたらす潜在的リスク、客観的な情報を提供するジャーナリストの役割、農薬パラコートの安全性などに触れながら、家族の健康問題についての個人的な経験も披露された。
