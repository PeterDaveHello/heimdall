# 2024-03-07

## [ஐஓஎஸ் அச்சுறுத்தல் சர்ச்சைக்கு மத்தியில் ஆப்பிள் காவிய விளையாட்டு டெவலப்பர் கணக்கை முடிக்கிறது](https://techcrunch.com/2024/03/06/apple-terminates-epic-games-developer-account-calling-it-a-threat-to-the-ios-ecosystem/)

- பயன்பாட்டு பொருளாதாரத்தில் ஆப்பிளின் ஆதிக்கத்தை எபிக் சவால் செய்த பின்னர், ஆப்பிள் எபிக் கேம்ஸின் டெவலப்பர் கணக்கை ஐஓஎஸ் சுற்றுச்சூழல் அமைப்புக்கு அச்சுறுத்தலாக மேற்கோள் காட்டி நிறுத்தியுள்ளது.
- ஆப்பிள் அதன் முன்மொழியப்பட்ட விதிகளை விமர்சித்து போட்டியிட முயற்சித்ததற்காக பதிலடி கொடுப்பதாக எபிக் நம்புகிறது, அதே நேரத்தில் ஆப்பிள் எபிக் ஒப்பந்தக் கடமைகளை மீறியதாகக் கூறுகிறது.
- இந்த சண்டை தொழில்நுட்ப நிறுவனங்களுக்கு இடையிலான சக்தி இயக்கவியல் மற்றும் மோதல்களை எடுத்துக்காட்டுகிறது, இது ஆப் ஸ்டோர் சந்தையின் சிக்கல்களை வெளிச்சம் போட்டுக் காட்டுகிறது.

## [டெக்ஸ்காம் ஸ்டெலோ குளுக்கோஸ் பயோசென்சார் அமைப்பை ஓவர்-தி-கவுண்டர் பயன்பாட்டிற்கு எஃப்.டி.ஏ அங்கீகரிக்கிறது](https://www.fda.gov/news-events/press-announcements/fda-clears-first-over-counter-continuous-glucose-monitor)

- டெயில்விண்ட் சிஎஸ்எஸ் பதிப்பு 4.0 ஐ அறிமுகப்படுத்துகிறது, இது மேம்பட்ட செயல்திறன் மற்றும் குறைக்கப்பட்ட அளவுக்காக ரஸ்டில் புதுப்பிக்கப்பட்ட இயந்திரத்தைக் கொண்டுள்ளது.
- மேம்பட்ட @import மேலாண்மை, விற்பனையாளர் முன்னொட்டு மற்றும் CSS மாறிகள் போன்ற CSS-சொந்த கூறுகள் போன்ற புதிய செயல்பாடுகள் அறிமுகப்படுத்தப்படுகின்றன.
- புதுப்பிப்பு வழக்கற்றுப்போன வகுப்புகள், வரவிருக்கும் ஜாவாஸ்கிரிப்ட் உள்ளமைவு மற்றும் அடுத்த வெளியீட்டிற்கான வளர்ச்சியில் உள்ள ஒரு முழுமையான CLI ஆகியவற்றை நீக்குவதை உள்ளடக்கியது.

### [எதிர்வினைகள்](https://news.ycombinator.com/item?id=39617953)

- விவாதம் Tailwind CSS v4.0 ஐ ஏற்றுக்கொள்வதன் நன்மை தீமைகளை எடுத்துக்காட்டுகிறது, வலை அபிவிருத்தியில் அதன் செயல்திறன் மற்றும் பயனர் நட்பு குறித்து பயனர்களிடமிருந்து கலவையான எதிர்வினைகளைக் காட்டுகிறது.
- பயனர்கள் Rust, Svelte, Vue மற்றும் Tailwind போன்ற பல்வேறு வலை அபிவிருத்தி கட்டமைப்புகளை ஒப்பிடுவதையும், மாறும் வகையில் இணைக்கப்பட்ட வகுப்பு பெயர்கள் மற்றும் SCSS கோப்புகளில் ஆழமான கூடு போன்ற சவால்களையும் தொடுகிறார்கள்.
- டெயில்விண்ட் CSS இன் நன்மைகள், பராமரிப்பு, குறியீடு வாசிப்புத்திறன் மற்றும் குறுக்கு-கட்டமைப்பு இணக்கத்தன்மை உள்ளிட்ட வலியுறுத்தப்படுகின்றன, HTML/CSS இருப்பு மற்றும் டெயில்விண்ட் v4 மேம்பாடுகளை நிர்வகிப்பதற்கான சிறந்த நடைமுறைகள் பற்றிய நுண்ணறிவுகள், கட்டமைப்பு மாறுபாடுகள், மேலதிகங்கள் மற்றும் கருப்பொருள்களுக்கான சமூக தீர்வுகளை நிவர்த்தி செய்கின்றன.
