## [மரிஜுவானாவை மறுவகைப்படுத்த அமெரிக்க டிஇஏ முன்மொழிகிறது](https://apnews.com/article/marijuana-biden-dea-criminal-justice-pot-f833a8dae6ceb31a8658a5d65832a3b8)

- கூட்டாட்சி மரிஜுவானா சட்டங்களை மறுபரிசீலனை செய்வதை ஜனாதிபதி பைடன் ஆதரிக்கிறார் மற்றும் வைத்திருந்த தண்டனைகளைக் கொண்ட நபர்களுக்கு கருணை காட்டியுள்ளார்.

## [CSS கட்டத்தில் இசைக் குறியீட்டை ஆராய்தல்](https://cruncher.ch/blog/printing-music-with-css-grid/)

- வலையில் மேம்பட்ட இசை குறியீடு, அணுகல் மற்றும் மென்மைத்தன்மைக்காக கட்டம் போன்ற இசை ஊழியர்களை உருவாக்க ஸ்கிரிப்ட் முன்மாதிரி CSS கட்டத்தைப் பயன்படுத்துகிறது.
- CSS கட்டம் மற்றும் தரவு பண்புகள் மூலம் ஒரு தண்டில் இசை சின்னங்களை எவ்வாறு நிலைநிறுத்துவது, CSS ஐப் பயன்படுத்தி தளவமைப்பு மற்றும் இடைவெளியை மாற்றுவது மற்றும் கணினியின் கட்டுப்பாடுகள் மற்றும் சாத்தியமான மேம்பாடுகளை எடுத்துக்காட்டுகிறது.
- கூடுதல் அம்சங்களில் டிரம்ஸ் மற்றும் பியானோவுக்கான ஸ்டேவ் ஸ்பிலிட்டிங், மல்டி-ஸ்டேவ் ரெண்டரிங் மற்றும் ஜாஸ் தரத்திற்கான இடமாற்றக்கூடிய முன்னணி தாளை வழங்குகிறது.

- மிக சமீபத்திய வெளியீடு பதிப்பு 1.2.0 ஆகும், இது README, மூல களஞ்சியம், API ஆவணங்கள், பங்களிப்பு வழிகாட்டுதல்கள், அடிக்கடி கேட்கப்படும் கேள்விகள் மற்றும் TOR தொடர்பான விவரங்கள் போன்ற பல்வேறு ஆதாரங்களை வழங்குகிறது.

### [எதிர்வினைகள்](https://news.ycombinator.com/item?id=40214763)

- கட்டுரை ஆர்ட்டியை ஆராய்கிறது, ரஸ்டில் குறியிடப்பட்ட ஒரு டோர் செயல்படுத்தல், அதன் வளர்ச்சி, நிதி மற்றும் சோதனை பயன்பாட்டிற்கான பரிந்துரைகள் பற்றிய நுண்ணறிவுகளைப் பகிர்ந்து கொள்கிறது.
- ஆர்த்தியின் தற்போதைய நிலை மற்றும் சாத்தியமான பாதுகாப்பு பலவீனங்கள் குறித்து ஒரு விவாதம் வெளிப்படுகிறது, இது பயனர்களுக்கான முக்கியமான பரிசீலனைகளுக்கு கவனத்தை ஈர்க்கிறது.
- ஆர்டி இன் ரஸ்ட் டோர் செயல்படுத்தல்களில் ஒரு தனித்துவமான முன்னோக்கை வழங்குகிறது, அதன் வளர்ச்சி அம்சங்கள் மற்றும் பாதுகாப்பு தாக்கங்கள் காரணமாக ஆர்வத்தை ஈர்க்கிறது.
