## [Ada apa, Python? GIL dihapus, kompiler baru, optparse tidak digunakan lagi](https://www.bitecode.dev/p/whats-up-python-the-gil-removed-a)

- Pada bulan Juli 2023, beberapa pembaruan dan pengembangan diumumkan untuk Python.
- Global Interpreter Lock (GIL) dalam Python telah dihapus, yang dapat meningkatkan kinerja dan memungkinkan pemanfaatan lebih baik dari beberapa core.
- Python sekarang memiliki kompiler baru yang disebut LPython.
- Pydantic 2 telah dirilis dengan peningkatan dalam kecepatan dan stabilitas.
- Modul getopt dan optparse sudah tidak digunakan lagi.
- Cython 3.0 sekarang memiliki dukungan yang lebih baik untuk Python murni.
- Proposal baru yang disebut PEP 722 telah dibuat untuk menentukan ketergantungan dalam skrip file tunggal.
- Dukungan Python VSCode telah ditingkatkan untuk kinerja yang lebih cepat.
- Aplikasi cat berbasis terminal baru yang disebut textual-paint telah diperkenalkan.

#### [Reaksi](https://news.ycombinator.com/item?id=36935041)

- Diskusi ini berpusat pada pengoptimalan kode Python untuk kinerja multi-threaded dan kemungkinan penghapusan Global Interpreter Lock (GIL) di Python.
- Berbagai perspektif dibagikan, termasuk saran untuk menggunakan alat bantu seperti `gunicorn` dan `gevent` untuk multiprosesing dan multithreading.
- Alternatif lain seperti menggunakan bahasa seperti C++ atau Rust untuk performa yang lebih baik juga bisa dipertimbangkan.
- Saran untuk pengoptimalan disediakan, seperti memanfaatkan caching atau memori bersama dengan Redis atau memcached.
- Kekhawatiran tentang masalah seperti kebuntuan dan penggunaan sumber daya diangkat, bersama dengan saran untuk bahasa atau teknologi alternatif.
- Kelemahan kinerja Python dan solusi potensial seperti menggunakan bahasa lain atau memanfaatkan kompiler JIT juga dibahas.
- Kompatibilitas bahasa pemrograman yang berbeda, tantangan transisi, dan kompleksitas desain dan penggunaan bahasa pemrograman disebutkan.
- Potensi manfaat dan kerugian dari penghapusan GIL di Python masih diperdebatkan, dengan berbagai pendapat tentang peningkatan kinerja dan potensi masalah konkurensi.
