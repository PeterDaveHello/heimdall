## [Panduan Pemrograman Modul Kernel Linux](https://sysprog21.github.io/lkmpg/)

- Panduan Pemrograman Modul Kernel Linux adalah sebuah buku gratis yang mengajarkan pembaca cara menulis modul kernel, yang memperluas fungsionalitas kernel tanpa perlu me-reboot sistem.
- Buku ini mencakup berbagai topik, termasuk header, driver perangkat, sistem berkas, panggilan sistem, penangan interupsi, dan penjadwalan tugas.
- Penulis merekomendasikan untuk mengekstrak, mengkompilasi, dan memuat semua contoh dari konsol. Buku ini memperkenalkan cara-cara baru untuk menangani pengaturan modul kernel, termasuk penggunaan makro module_init dan module_exit, dan menjelaskan opsi lisensi untuk modul kernel. Sebagai tambahan, modul dapat mengambil argumen baris perintah menggunakan variabel global dan makro module_param().

#### [Reaksi Industri](http://news.ycombinator.com/item?id=35782630)

- eBPF adalah alternatif yang lebih baru dan sedikit lebih mudah/aman untuk beberapa tugas modul kernel.
- Panduan ini merupakan titik awal yang baik untuk belajar tentang modul kernel, tetapi mungkin tidak cukup untuk menulis driver yang lebih kompleks.

## [faulTPM: Mengungkap Rahasia Terdalam AMD fTPM](https://arxiv.org/abs/2304.14717)

- Para peneliti di Fraunhofer Institute menemukan kerentanan pada fTPM (TPM berbasis firmware) AMD yang mereka beri nama faulTPM.
- Dengan mengeksploitasi kerentanan ini, penyerang dapat membaca informasi sensitif dari TPM, termasuk rahasia seperti kunci enkripsi dan kata sandi.
- AMD telah merilis pembaruan firmware untuk mengatasi masalah ini, dan pengguna disarankan untuk memperbarui sistem mereka sesegera mungkin.

#### [Reaksi Industri](http://news.ycombinator.com/item?id=35787195)

- Bagian komentar membahas berbagai topik teknologi, termasuk keterbatasan teknologi, pemutusan dukungan perangkat keras Windows 11 yang sewenang-wenang, dan kelayakan beralih ke distro Linux dari Windows.
- Para peneliti mengungkapkan kerentanan dalam implementasi fTPM AMD yang dapat dieksploitasi dalam serangan chip senilai $3 untuk mengeksfiltrasi data, menyoroti pentingnya mengevaluasi model ancaman dan menerapkan langkah-langkah keamanan tambahan selain mengandalkan fTPM atau pelindung khusus TPM.

## [Jsonformer: Menghasilkan keluaran terstruktur dari LLM](https://github.com/1rgs/jsonformer)

- Jsonformer adalah pendekatan baru untuk menghasilkan JSON terstruktur dari model bahasa untuk memastikan bahwa JSON yang dihasilkan selalu benar secara sintaksis dan sesuai dengan skema yang ditentukan.
- Jsonformer mengisi token tetap selama proses pembuatan dan hanya mendelegasikan pembuatan token konten ke model bahasa, sehingga lebih efisien dan tahan banting daripada pendekatan yang ada.
- Jsonformer dibangun di atas pustaka transformer HuggingFace dan mendukung subset dari skema JSON sekaligus fleksibel dan dapat diperluas.

#### [Reaksi Industri](http://news.ycombinator.com/item?id=35790092)

- Generator skema JSON serupa lainnya seperti Clownfish dan Guardrails memerlukan pendekatan yang lebih kompleks untuk skema yang kompleks
