## [Percontohan pengajuan pajak gratis yang dikelola IRS akan tersedia di 13 negara bagian](https://www.irs.gov/newsroom/irs-advances-innovative-direct-file-project-for-2024-tax-season-free-irs-run-pilot-option-projected-to-be-available-for-eligible-taxpayers-in-13-states)

## [Membuat CRDT Lebih Efisien](https://jakelazaroff.com/words/making-crdts-98-percent-more-efficient/)

- Artikel blog ini membahas strategi untuk meningkatkan efisiensi Collaborative Real-Time Editors (CRDTs), khususnya untuk editor pixel art yang menggunakan teknik seperti kode heksa, menyimpan UUID (Universal Unique Identifiers) dan warna dalam tabel, dan menerapkan run-length encoding.
- Selain itu, buku ini menguraikan pendekatan kompresi data seperti run-length dan pengkodean biner untuk mengurangi ukuran data dan membahas cara-cara unik untuk mengompresi data status CRDT melalui beragam teknik pengoptimalan yang menghasilkan pengurangan ukuran sebesar 98%.
- Lebih lanjut disebutkan tentang pustaka CRDT yang telah mengintegrasikan teknik optimasi ini.

### [Reaksi](https://news.ycombinator.com/item?id=37915934)

- Artikel ini menyarankan untuk meningkatkan efisiensi Conflict-free Replicated Data Types (CRDT) melalui skema kompresi dan metode alternatif untuk menghasilkan UUID (Universally Unique Identifiers).
- Ini membahas tentang konversi struktur data berbasis JSON ke format biner, kompresi lebih lanjut menggunakan algoritme seperti brotli dan LZ4, dan potensi menggunakan kompresi LZ4 untuk transfer data melalui kabel.
- Kemungkinan mengeksploitasi algoritme kompresi gambar lossless, seperti PNG pada CRDT juga dieksplorasi, sementara masalah performa pada demo didiskusikan untuk perbaikan.
