# 2023-08-20

## [阿姆斯特丹將使用「噪音攝像頭」對付聲音太大的汽車](https://nltimes.nl/2023/08/11/amsterdam-use-noise-cameras-loud-cars)

- 阿姆斯特丹已經實施了「噪音攝像頭」來解決城市中車輛嘈雜的問題。
- 電動路標已被放置，以提醒駕駛員如果他們的車輛聲音太大，這些標誌稍後將被噪音攝像頭取代，這些攝像頭將自動向違規司機處以罰款。
- 噪音標誌最初將在兩個地點進行測試，並計劃在一個月內安裝更多的聲級計和標誌。這一舉措旨在最大限度地減少城市的噪音干擾。鹿特丹也與阿姆斯特丹合作進行類似的實驗。

### [反應](https://news.ycombinator.com/item?id=37187837)

- 阿姆斯特丹計劃使用「噪音攝像機」來解決城市中汽車和摩托車噪音過大的問題。
- 圍繞這個問題的爭論集中在大聲摩托車的必要性和影響上，一些人認為它們的安全益處，另一些人則批評它們的煩惱。
- 關於噪音污染、行人行為、車輛改裝以及推廣更安靜和電動的替代品也有更廣泛的討論。
- 居民們主張禁止汽油動力汽車和推廣電動輕便摩托車，以創造一個更安靜的城市環境。

## [為什麼舊書聞起來這麼好聞？](https://scienceswitch.com/2023/08/19/why-do-old-books-smell-so-good/)

- 科學家們正在研究舊書的獨特氣味，並使用釋放的揮發性有機化合物（VOC）來確定它們的年齡，狀況和歷史。
- 舊書的氣味是由紙張和其他有機化合物分解引起的，釋放出特定的化合物，如苯甲醛、香蘭素、乙苯和 2-乙基己醇。
- VOC 分析可以説明圖書館員和歷史學家確定保存需求，並將孤立的圖書與其原始圖書館相匹配。

### [反應](https://news.ycombinator.com/item?id=37188015)

- 駭客新聞討論探討了各種主題，包括舊書的氣味、與技術相關的懷舊、技術進步、健康問題以及實體書和電子書之間正在進行的辯論。
- 簡要提到了數位鼻子的概念和合成氣味的可能性。

## [鉚釘 – 使用 Nomad 和 Rust 進行開源遊戲伺服器管理](https://github.com/rivet-gg/rivet)

- Rivet 是一種開源遊戲伺服器管理工具，旨在為沒有基礎設施經驗的遊戲開發人員簡化專用伺服器的部署。
- 它主要使用 Rust 和 Lua 構建，並整合了其他工具和庫。
- 鉚釘支持各種功能，例如邊緣代理、緩存、分析、疊加網路和 Redis Lua 腳本。
- 它是在Apache 2.0下獲得許可的，旨在成為一個可訪問且全面的遊戲伺服器管理器。
- 鉚釘不處理網路邏輯，但與現有工具結合使用。
- 它自 2022 年 1 月以來一直處於封閉測試階段，支援多個遊戲的數百萬月活躍使用者。

### [反應](https://news.ycombinator.com/item?id=37188659)

- 討論的重點是遊戲開發基礎設施的挑戰和開源工具的優勢。
- Rivet是一種開源遊戲伺服器管理工具，因其有效部署專用伺服器的能力而備受推崇。
- 對話還探討了使用 Kubernetes 和 Nomad 進行遊戲開發的好處和缺點，開發人員表示有興趣將 Rivet 用於他們自己的遊戲。
- Rivet 團隊承認需要增強本地開發，並討論了開源功能、Nakama、HashiCorp 許可證更改和 Bevy 支援等主題。

## [Cellebrite要求員警對其技術保密](https://techcrunch.com/2023/08/19/cellebrite-asks-cops-to-keep-its-phone-hacking-tech-hush-hush/)

- 電話竊聽技術公司Cellebrite要求執法機構對其技術使用保密，這引發了對透明度和被告權利的擔憂。
- 批評者認為，披露對於評估證據和保護隱私很重要。
- 政府承包商哈裡斯公司（Harris Corporation）也對其手機監控工具提出了類似的保密要求。

### [反應](https://news.ycombinator.com/item?id=37194272)

- 討論的參與者對Cellebrite的可靠性和潛在濫用表示擔憂，Cellebrite是執法機構用於從智能手機中提取數據的軟體。
- 某些法醫技術的有效性和測謊儀測試的使用受到質疑，對其有效性表示懷疑。
- 對話強調了確保數據安全的挑戰，通過Cellebrite獲取證據的法律影響，並提到了蘋果與FBI的爭議，提出了對設備安全和數據保護的擔憂。

## [WD拒絕回答我們關於其自擦拭閃迪SSD的問題](https://www.theverge.com/23837513/western-digital-sandisk-ssd-corrupted-deleted-questions)

- 據報導，西部數據的SanDisk SSD導致數據丟失，該公司對有關該問題的詢問沒有反應。
- 已經出現了多份數據丟失報告，包括來自The Verge和Ars Technica的員工。
- 儘管意識到這個問題，西部數據仍在繼續銷售SSD，該公司的公關主管拒絕解決這個問題。
- 有人指控西部數據正在使用虛假的亞馬遜評論並操縱搜尋結果來淡化失敗。
- 該公司現在正面臨針對閃迪SSD故障的多起集體訴訟。
- 西部數據還被指控採取欺騙行為，例如試圖在其產品陣容中包含低於標準的驅動器，並在其NAS磁碟上觸發不必要的警告。

### [反應](https://news.ycombinator.com/item?id=37188736)

- 在線論壇正在舉辦有關SSD和HDD等存儲解決方案的可靠性和質量的討論，使用者分享他們對各種品牌和型號的經驗和擔憂。
- 固件問題和保修索賠是討論的主題之一，突出了技術行業中可靠和高效的客戶服務的重要性。
- 西部數據和閃迪等品牌因客戶投訴和集體訴訟而面臨聲譽損害，導致使用者探索SK海力士和三星等以其可靠性而聞名的替代品牌。
