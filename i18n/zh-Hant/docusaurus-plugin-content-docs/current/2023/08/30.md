# 2023-08-30

## [ISP不應該監管在線言論，無論它有多糟糕](https://www.eff.org/deeplinks/2023/08/isps-should-not-police-online-speech-no-matter-how-awful-it)

- 電子前沿基金會（EFF）認為，互聯網服務提供者（ISP），特別是一級服務提供者，不應充當內容監管機構，並警告可能導致潛在的權力濫用和審查。
- EFF引用了一級ISP颶風電氣（Hurricane Electric），該公司部分拒絕為一個充滿爭議的論壇中斷流量的服務，並將其描述為未來可能阻止法律言論的審查行為的先例。
- EFF倡導執法部門和法院執行法律和隱私措施，而不是ISP監管內容，堅持認為保護人權不應依賴於充當言論員警的公司。

### [反應](https://news.ycombinator.com/item?id=37313349)

- 討論圍繞一系列主題展開，包括對網路言論的監管、私營公司在控制話語方面的作用以及政府對言論監管的干預。
- 還談到了阻止訪問特定網站的潛在後果，以及保護言論自由的首要地位。
- 其他相關主題包括極端主義內容、互聯網自由以及託管供應商和互聯網服務提供者 （ISP） 的職責，出席者表達了不同的觀點並就這些問題的優點進行了辯論。

## [添加到 ssh（1） 中的擊鍵定時混淆](https://undeadly.org/cgi?action=article;sid=20230829051257)

- Damien Miller 在 ssh（1） 中引入了擊鍵定時混淆，旨在通過定期發送互動式流量和發送錯誤的擊鍵來掩蓋擊鍵之間的時序。
- 這一新功能已通過增強SSH（安全外殼）協定而整合，預計不久將在其他系統中採用。
- 它主要被視為安全更新，預計將包含在下一個OpenBSD版本中，這是一個免費的開源操作系統，以其專注於安全功能而聞名。

### [反應](https://news.ycombinator.com/item?id=37307708)

- 本文強調了SSH密碼安全性的各個方面，引入了擊鍵定時混淆作為提高安全性的新策略。
- 它闡明瞭偽裝密碼和提供用戶反饋的不同方法，而不會使安全性受到威脅，強調了密碼遮罩，管理器和在密碼輸入期間停止螢幕共用的重要性。
- 它還介紹了備用身份驗證方法，如 SSH 金鑰、某些設備的限制、託管服務，並探討了與加密和網路監控相關的主題。

## [為什麼Python贏了？](https://news.ycombinator.com/item?id=37308747)

- 作者在 2013 年開始了他們的 JavaScript 程式設計之旅，然後過渡到其他語言，如 Python，並最終將 Ruby 作為他們的首選語言。
- 作者觀察到語言流行度的轉變，注意到Ruby曾經與Python相提並論，但使用量有所下降。
- 出於好奇，作者正在尋找這種觀察到的趨勢背後的原因，導致 Ruby 的使用和普及減少。

### [反應](https://news.ycombinator.com/item?id=37308747)

- 該討論強調了Python在過去十年中越來越受歡迎，超過了Ruby和Perl等其他語言，主要是因為它對學術界的吸引力，簡單性和易於學習。
- Python的專業領域，特別是在數據環境中，以及它的培育社區，是其地位的其他因素。
- 此外，還討論了各種主題，如類型檢查、語言比較、Python 的顯式與隱式行為以及打包/依賴管理挑戰，表明 Python 的多功能性、豐富的庫和跨行業集成是其受歡迎程度的關鍵屬性。

## [Griffin – 一家完全受監管、API 驅動的銀行，擁有 Clojure](https://www.juxt.pro/blog/clojure-in-griffin/)

- Griffin是一家受監管的銀行服務企業，提供基於API的解決方案，並將Clojure用於其平臺，理由是諸如不變性和審計日誌等優點。
- 他們的平臺在AWS上的Kubernetes上運行，FoundationDB作為其資料庫。他們應用事件溯源模型，並使用異步 HTTP 處理程式 Netty 來處理付款回應。
- 該公司強調招聘遠端Clojure編碼員和對分散式系統的嚴格測試，以避免競爭條件和錯誤。他們採用類似於生成測試的測試方法，以實現更好的錯誤控制。

### [反應](https://news.ycombinator.com/item?id=37313183)

- 對話包括Clojure程式設計語言在銀行業的應用以及API銀行在英國的擴展。
- 還討論了對新技術平臺Griffin的懷疑，以及加密貨幣的優缺點。
- 該對話討論了金融服務中隱私和安全的重要性，並評估了程式設計語言中類型系統的效率。

## [英特爾展示採用矽光子學的 8 核 528 線程處理器](https://www.servethehome.com/intel-shows-8-core-528-thread-processor-with-silicon-photonics/)

- 英特爾在 Hot Chips 2023 活動中推出了採用矽光子學的 8 核 528 線程處理器。該處理器包括直接網狀到網狀光纖結構，專為處理超稀疏數據工作負載而定製。
- 該處理器採用精簡指令集電腦 （RISC） 指令集架構 （ISA） 設計，優化了其簡單性和能效。
- 該產品採用多晶元封裝，採用嵌入式多晶元互連橋接（EMIB）。它仍在開發中，目前還沒有商業名稱。

### [反應](https://news.ycombinator.com/item?id=37315802)

- 英特爾推出了一款利用矽光子學的新型 8 核 528 線程處理器，提高了計算效率，類似於索尼的 Cell 處理器，但設計規格不同。
- 在決定開發內部晶元之後，蘋果已經遠離英特爾處理器，以獲得更大的控制權並鞏固其平臺。
- 圍繞SMT（同步多線程）設計的性能和安全性的討論正在進行中，特別關注英特爾和AMD的Zen SMT技術，以及是否需要禁用SMT。

## [可存取調色板：停止將 HSL 用於顏色系統 （2021）](https://wildbit.com/blog/accessible-palette-stop-using-hsl-for-color-systems)

- Postmark開發了一種新的工具，可訪問的調色板，以處理其先前顏色系統中亮度和對比度的不一致。
- 該工具使用CIELAB和LCh色彩空間，可以更精確地表示人類如何感知色彩。
- 用戶可以利用無障礙調色板來創建具有一致亮度和可預測對比度的調色板;但是，流行的設計工具（如 Figma、Sketch 和 Adobe XD）目前不支援這些色彩空間。

### [反應](https://news.ycombinator.com/item?id=37307473)

- 這些主題側重於顏色系統、顏色表示、可訪問性和設計注意事項，突出了創建視覺上引人注目且易於訪問的設計的局限性和挑戰。
- 討論包括關於不同色彩空間、演算法和資源的有效性以及它們是否符合可訪問性指南的辯論。
- 涵蓋的其他主題包括有關廣播和流媒體服務中顏色質量的問題，以及在各種媒體上準確表示顏色的障礙。

## [iFixit請願政府有權入侵麥當勞霜淇淋機](https://www.404media.co/activists-petition-government-for-right-to-hack-mcdonalds-mcflurry-machines-to-repair-them/)

- iFixit和Public Knowledge要求國會圖書館員允許駭客入侵麥當勞的McFlurry機器進行維修，認為這些機器通常不可靠，允許數位鎖破將促進診斷和維修。
- 目前，泰勒與麥當勞特許經營權簽訂了獨家服務合同，但對“數位千年版權法”（DMCA）的擬議豁免可以讓特許經營權進行自己的維修。
- 這一運動發生在泰勒和Kytch之間的訴訟中，Kytch是一家公司，該公司創造了一種解碼機器錯誤代碼的設備，並尋求將維修豁免擴大到包括商業工業設備。

### [反應](https://news.ycombinator.com/item?id=37311239)

- 對話主要涉及麥當勞霜淇淋機的可靠性問題，科技行業的道德影響以及麥當勞與機器供應商泰勒獨家交易背後的潛在財務收益。
- 人們對某些零件和供應商的使用、安全隱患以及如何對待餐飲業的低工資工人提出了擔憂。
- 使用者對泰勒的產品及其壟斷地位表示不滿，建議使用Carpigiani的機器等替代品。

## [新研究發現微塑料滲透到身體的所有系統中，改變行為](https://www.sustainableplastics.com/news/microplastics-found-pass-blood-brain-barrier-even-though-ingested-drinking-water)

- 羅德島大學的Jaime Ross教授進行的一項研究表明，微塑膠可以穿透身體的所有系統，甚至大腦，導致哺乳動物的行為改變。
- 這項以小鼠為試驗物件的研究發現，暴露於微塑膠會導致肝臟和腦組織中行為的變化和免疫標誌物的交替。
- 該研究表明，迫切需要對與微塑膠暴露相關的健康影響進行更多調查，特別是在哺乳動物中。

### [反應](https://news.ycombinator.com/item?id=37306427)

- 對話探討了微塑膠在人體內的存在，它們對肥胖、智商水準下降和生育率降低等健康問題的可能影響，以及進一步研究的必要性。
- 對話討論了導致肥胖的多個變數、潛在的解決方案以及飲食的作用。有人提出塑膠與肥胖之間可能存在相關性。討論還擴展到食品包裝中使用的塑膠的影響。
- 這場辯論強調了一次性塑膠對環境的影響，逐步淘汰它們的困難，以及個人選擇、文化觀點以及政治和經濟影響的重要性。強調了解決微塑膠的有害影響和尋求可持續替代品的緊迫性。

## [BGP 錯誤處理中的嚴重缺陷](https://blog.benjojo.co.uk/post/bgp-path-attributes-grave-error-handling)

- 邊界閘道協定 （BGP） 對於互聯網上的網路流量路由至關重要，但發現其錯誤處理存在重大缺陷，可能導致網路中斷和互聯網中斷。
- 作者是一名專業的BGP漏洞測試人員，他在多家供應商的BGP系統中發現了這些問題，並對他們的安全回應表示不滿。
- 作者主張供應商需要對解決安全問題（例如提供漏洞賞金）承擔更多責任，並批評直接向供應商報告安全漏洞的無效性，建議將客戶通知作為替代方案。

### [反應](https://news.ycombinator.com/item?id=37305800)

- 這篇文章批評了BGP（邊界閘道協定）的錯誤處理存在缺陷，並強調了發現這些缺陷的研究人員的爭議。
- 它回顧了log4j漏洞對網路安全的影響，並討論了網路運營商和安全研究人員的責任。
- BGP 的錯誤處理功能正常，但需要增強功能，因為最近存在一個導致互聯網範圍中斷的漏洞。

## [Fomos：實驗性操作系統，使用 Rust 構建](https://github.com/Ruddle/Fomos)

- Fomos是一個使用Rust構建的實驗性操作系統（OS），旨在簡化應用程式和操作系統之間的關係。在Fomos中，應用程式僅被視為一個功能，並通過上下文結構與操作系統聯絡。
- Fomos OS包含沙箱，檢測，調試和應用程式的透明重啟等屬性。為了處理舊版本的應用程式，操作系統在上下文中構造新功能。
- 挑戰仍然存在，包括安全性和調度方式。未來的計劃包括擴展功能，如永久存儲、GPU 支援、網路連接和應用程式通信的抽象結構。

### [反應](https://news.ycombinator.com/item?id=37316309)

- 討論的重點是Fomos，這是一種將應用程式作為功能處理的實驗性操作系統，在嵌入式系統和AWS（亞馬遜網路服務）Lambda運行時中具有潛在的應用程式。
- 出席者對Fomos的安保/安全表示關切;建議使用潛在的解決方案，例如靜態分析（檢查程式代碼而不執行程序代碼）和看門狗計時器（在發生故障時執行操作的系統主管）。
- 討論了沙箱的主題，封裝程式以限制其操作範圍，在操作系統中，突出了相關挑戰和各種可執行格式的優勢。
