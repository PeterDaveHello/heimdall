## [AI實時人體全身照片生成器](https://generated.photos/human-generator/)

- 人體產生器是一種 API 工具，它利用先進的 AI 演算法即時生成超逼真的人物全身照片。
- 該工具在種族、年齡和體型方面提供了最大的多樣性，並支援不同的姿勢和服裝選擇。
- 它免費用於非商業用途，並提供其他功能，例如生成變體和動畫。
- Human Generator是一套產品的一部分，其中包括Face Generator，Face Gallery，Anonymizer和Datasets。
- 用戶可以聯繫公司，以解決有關生成內容的使用、安全和存儲的任何問題或疑慮。

### [反應](https://news.ycombinator.com/item?id=37237583)

- 本文探討了使用生成對抗網路 （GAN） 創建逼真的人體圖像，以及在可擴展性和圖像品質方面比較擴散模型和 GAN。
- 討論了圍繞deepfake技術的道德問題和濫用的可能性，以及對使用AI為邪惡目的生成類似人類圖像的網站的擔憂。
- 還涵蓋了版權對人工智慧生成藝術的法律影響、使用者對人工智慧圖像生成工具的體驗以及與人工智慧生成的數據相關的潛在問題、偏見和擔憂。

## [超過3000歐元的現金支付將在荷蘭被取締](https://www.rijksoverheid.nl/onderwerpen/geldzaken/vraag-en-antwoord/grote-aankopen-met-contant-geld)

## [咖啡渣使混凝土加固 30%](https://www.rmit.edu.au/news/all-news/2023/aug/coffee-concrete)

- 澳大利亞的工程師發現了一種通過加入烘焙過的二手咖啡渣來提高混凝土強度的方法。
- 這種技術涉及將廢棄咖啡渣轉化為生物炭，從而使混凝土強度提高 30%。
- 通過利用這種方法，不僅可以提高混凝土的性能，還可以減少送往垃圾填埋場的廢物量。
- 該團隊打算與不同的行業合作，並進行現場試驗以實施他們的發現。
- 這項創新有可能大大增強有機廢物的回收利用，並有助於保護沙子等寶貴資源。

### [反應](https://news.ycombinator.com/item?id=37234404)

- 據澳大利亞RMIT大學的科學家稱，在混凝土中添加咖啡渣可以將其強度提高30%。
- 這一發現對碳封存和減少建築業碳排放的潛力具有重要意義。
- 然而，人們擔心對缺失的碳徵稅會產生意想不到的後果和不正當的激勵措施。
- 討論還探討了使用高額費用作為威懾而不是徹底禁止的問題，以及這種方法的利弊。
- 關於實施碳稅或污染稅以應對氣候變化存在爭議，對其影響有不同的看法。
- 還討論了咖啡渣在混凝土生產中的潛在好處及其作為堆肥劑的用途。
- 在混凝土中使用咖啡渣的物流、能源需求和潛在的環境影響是考慮的主題。
- 對話中提到了傳統混凝土和廢物管理挑戰的替代方案。
