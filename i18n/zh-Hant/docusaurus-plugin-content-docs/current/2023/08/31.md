# 2023-08-31

## [我自動化了一半的打字](https://github.com/eschluntz/compress)

- 在過去的一年裡，使用者一直在使用一種工具來分析他們的 Slack 消息併為常用短語生成鍵盤快捷鍵。
- 該工具通過解析 Slack 消息來識別經常重複的短語。
- 該工具通過將常規短語轉換為易於訪問的鍵盤快捷鍵，為使用者提供效率和便利。

### [反應](https://news.ycombinator.com/item?id=37326870)

- 本文的主要重點是通過自動化工具、快捷方式和縮寫提高打字效率和生產力。
- 關於打字速度在編碼生產力中的重要性，以及在工作和生活中的效率和放鬆之間取得平衡，一直在爭論。
- 探討了各種方法，如鍵盤快捷鍵、備用鍵盤設置、文本擴展工具和聽寫系統，並討論了它們的潛在優點和缺點。

## [DIY「仿生胰腺」正在改變糖尿病護理](https://www.nature.com/articles/d41586-023-02648-9)

- 一個由管理1型糖尿病的個人組成的社區開發了DIY開源軟體來自動化血糖水準管理，強調個人化優於商業系統的重要性。
- 宣導者正在推動FDA批准開源演算法，以提高這些DIY設備的可訪問性，但設備製造商之間的合作對於互操作性是必要的。
- 據報導，這些開源AI演算法在管理糖尿病方面的好處突出了用戶選擇的價值和開源社區的進步。

### [反應](https://news.ycombinator.com/item?id=37321028)

- 討論涉及糖尿病管理的幾個方面，例如DIY（DIY）技術的利用，閉環系統，涉及胰島素泵使用的挑戰，改進監測設備的必要性，飲食習慣的影響以及DIY社區在帶頭進步方面的影響力。
- 人們正在思考不同管理策略的潛在優點和缺點，強調了持續研究和支援糖尿病患者的重要性。
- 首要重點是發現增強的解決方案來控制血糖水準並降低與糖尿病相關的風險。

## [Microsoft正在Win11中使用類似惡意軟體的彈出視窗來讓人們放棄Chrome。](https://www.theverge.com/2023/8/30/23851902/microsoft-bing-popups-windows-11-malware)

- Microsoft通過彈出消息提示Windows 11用戶，建議將預設搜尋引擎從Google切換到Bing，即使他們正在使用Chrome。
- 這些策略引起了使用者的挫敗感，因為他們認為此類彈出視窗具有侵入性，並認為他們不應該在他們購買的操作系統中受到廣告的影響。
- Microsoft已經認識到這種強烈反對，並在他們嘗試解決問題時暫時停止了這些通知。

### [反應](https://news.ycombinator.com/item?id=37321002)

- 討論集中在使用者對Windows（特別是Windows 11）的挫敗感上，由於彈出視窗，用戶帳戶和程式可見性的問題，促使一些人考慮Mac或Linux等替代方案。
- 在系統選擇中，Linux因其易用性和功能性而受到稱讚，但注意到運行特定遊戲的缺點。其他討論的領域包括硬體首選項、軟體相容性和 Windows 子系統 for Linux （WSL） 的限制。
- 人們對科技產品中的中斷和廣告普遍感到不滿，關於平臺限制、Windows的未來、Microsoft的行為以及潛在的法律或監管行動的爭論。對Gmail和谷歌地圖的iPhone應用程式的一些挫敗感也表明瞭對谷歌產品的潛在不滿。

## [Windows 11 系統元件使用預設瀏覽器打開歐洲的連結](https://blogs.windows.com/windows-insider/2023/08/25/announcing-windows-11-insider-preview-build-23531-dev-channel/)

- Microsoft 已將適用於 Windows 11 的預覽體驗成員預覽版 23531 推出到開發人員頻道，其中包含安全模式、文件資源管理器、HDR 背景和任務管理器的修復等更新。
- 任務欄上的「開始」功能表和搜索功能繼續存在一些已知問題。建議開發人員下載最新的 Windows 預覽體驗成員 SDK 和 NuGet 程式包以保持最新狀態。
- 重新啟動后，開發頻道現在接收 23000 個系列版本，因此桌面浮水印是預發佈版本的常規功能。

### [反應](https://news.ycombinator.com/item?id=37320786)

- 討論集中在歐洲經濟區（EEA）對Microsoft的Windows 11的批評上，因為它偏向於Edge瀏覽器，忽略了使用者的偏好。
- 出席者還就蘋果等其他科技公司的行為進行了辯論，暗示需要更廣泛的監管來防止限制使用者選擇和隱私的壟斷行為。
- 其他主題包括下載Chrome on Edge的挫敗感，Microsoft對其使用者缺乏尊重，以及Linux Mint等替代選項的積極體驗。

## [聯合國稱，數十萬人被販運到東南亞成為網路詐騙者](https://www.ohchr.org/en/press-releases/2023/08/hundreds-thousands-trafficked-work-online-scammers-se-asia-says-un-report)

- 聯合國人權高專辦的一份報告顯示，有組織犯罪團夥正在迫使數十萬東南亞人從事在線犯罪活動，如浪漫投資詐騙、加密欺詐和非法賭博。
- 這些犯罪組織使其受害者遭受人權侵犯，如威脅、酷刑、性暴力、強迫勞動和任意拘留。由於其秘密性質，這種在線詐騙販運的規模難以估計。
- 報告呼籲受影響國家加強人權，加強治理，實施法治，並打擊腐敗，作為應對這些在COVID-19大流行期間升級的騙局的一部分。

### [反應](https://news.ycombinator.com/item?id=37320975)

- 在線對話涉及東南亞人口販運的報告，受害者被迫從事在線詐騙活動，可能涉及當地執法和政治網路。
- 辯論延伸到現有的刑事司法實踐，重點是系統性改進，以防止犯罪，改造罪犯，並解決美國的囚犯待遇問題。
- 人工智慧在詐騙中的作用受到審查，同時也討論了Tinder和LinkedIn等社交媒體平臺減輕這些詐騙活動的責任。

## [FCC拒絕廢除要求ISP列出每個月費的規則](https://arstechnica.com/tech-policy/2023/08/fcc-says-too-bad-to-isps-complaining-that-listing-every-fee-is-too-hard/)

- 聯邦通信委員會（FCC）拒絕了康卡斯特和其他ISP提出的上訴，要求取消要求他們披露所有月費的規則。
- ISP認為，這一要求帶來了管理挑戰和不必要的複雜性，但FCC強調消費者在選擇互聯網服務時需要透明。
- 通過立法，FCC將執行這些規則，要求ISP將所有政府徵收的費用納入寬頻標籤。

### [反應](https://news.ycombinator.com/item?id=37326806)

- 聯邦通信委員會（FCC）堅持一項規則，要求互聯網服務提供者（ISP）向客戶披露所有月費，以提高透明度並促進供應商之間的比較。
- 支持者認為，這一規則將暴露隱藏的指控，刺激ISP之間的競爭，並追究他們的責任。討論還提出了一個問題，即地方政府徵收的轉嫁費是否應被視為稅收，以及ISP是否應將這些費用轉嫁給客戶。
- 此外，FCC裁定，ISP必須在其廣告中預先披露所有費用和收費，以提供進一步的透明度，反映公眾對定價誠實和反對隱藏費用的看法。

- 這筆資金籠罩在懷疑之中，引發了關於該倡議背後真正動機的辯論;A16Z澄清說，這些贈款不需要公平，旨在支援開源開發人員，鼓勵蓬勃發展的生態系統。

## [入侵LG顯示器的EDID瀏覽器](https://gist.github.com/kj800x/be3001c07c49fdb36970633b0bc6defb)

- 本文解決了LG ULTRAGEAR顯示器在Linux作業系統上無法正常工作的問題。
- 通過瀏覽擴展顯示標識資料 （EDID） 檔並使用十六進位編輯器對其進行修改來調查此問題。
- 這個問題似乎與Linux上的顯卡驅動程序有關，而不是顯示器本身，這表明瞭解決類似問題的潛在途徑。

### [反應](https://news.ycombinator.com/item?id=37323604)

- 本文深入探討了與Linux、macOS和 Windows 等作業系統上的顯示器相容性和擴展顯示標識資料 （EDID） 操作相關的挑戰和問題。
- 它強調了有關EDID覆蓋，低延遲杜比視界，亮度調整，圖元時鐘錯誤和非發佈機器的用戶體驗，並指出了處理EDID數據的複雜性和局限性。
- 討論還揭示了Linux上的硬體相容性問題以及平臺上故障排除的細微差別。它特彆強調了LG顯示器的EDID問題以及單個設備上不同標準的共存。
