## [谷歌努力遏制持續的垃圾郵件攻擊](https://www.searchenginejournal.com/google-search-overwhelmed-by-massive-spam-attack/504527/)

- 谷歌目前正遭受大規模的垃圾郵件攻擊，該攻擊已經持續了好幾天。
- 垃圾郵件發送者正在利用谷歌演算法的弱點，特別是在本地搜索和長尾關鍵詞方面。
- 垃圾郵件發送者正在使用連結網路來説明谷歌查找和索引他們的頁面，並且多個域正在為數十萬個關鍵字進行排名。
- 垃圾郵件頁面會自動重定向到其他域，除非使用GoogleIP位址訪問。
- 谷歌已經承認了這個問題，並正在積極努力解決它。

### [反應](https://news.ycombinator.com/item?id=38738619)

- 討論圍繞著幾個主題展開，包括谷歌為打擊針對谷歌機器人的垃圾郵件攻擊所做的努力、圍繞定向廣告的辯論、對谷歌搜尋結果質量的擔憂，以及關於替代搜尋引擎的討論。
- 參與者對充斥著廣告的網站表示沮喪，並推測谷歌的廣告選擇演算法。
- 對話闡明瞭與垃圾郵件、搜尋引擎演算法、搜尋結果質量和在線廣告相關的問題。

## [WebGPU 測試現已在Safari技術預覽中開放](https://webkit.org/blog/14879/webgpu-now-available-for-testing-in-safari-technology-preview/)

- WebGPU 是一種新的 API，可在 Web 上實現高性能 3D 圖形和計算，現在可在 Safari 技術預覽中進行測試。
- WebGPU 允許在 Web 內容中使用 GPU 功能，並且是用 JavaScript 編寫的。
- 要啟用 WebGPU，需要在 Safari 偏好設置中打開某些功能標誌。
- WebGPU API 是通過 JavaScript 訪問的，類似於 WebGL。
- GPU 設備、畫布上下文、渲染管道和繪製調用等不同元件用於創建和配置圖形輸出。
- WebGPU 還引入了一種稱為 WGSL 的新著色語言。
- 我們鼓勵使用者試用 WebGPU 並報告他們遇到的任何錯誤或問題。

### [反應](https://news.ycombinator.com/item?id=38737028)

- 討論涵蓋了一系列 Web 技術主題，例如 Safari 中的 WebGPU 實現、在 WebGPU 中使用二進位格式以及在 Xcode 中集成 WGSL 支援。
- 它還包括關於在手機和瀏覽器上運行語言模型、Safari 和 Chrome 的用戶體驗以及 Safari 的性能和市場份額的討論。
- 人們猜測蘋果的戰略舉措、潛在的訴訟和反壟斷行動，以及他們對Web API的投資。還提到了 Epiphany 和 Midori 網路瀏覽器，以及它們各自的問題。
