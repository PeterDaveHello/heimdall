## [Meta推出無縫通信 AI 模型，實現富有表現力的快速翻譯](https://ai.meta.com/research/seamless-communication/)

- Meta 創建了一系列名為 Seamless Communication 的 AI 研究模型，用於語言翻譯。
- 這些模型包括 SeamlessExpressive，它專注於保持語音的表達和細微差別，SeamlessStreaming 用於低延遲的近實時翻譯，以及 SeamlessM4T v2 作為通用翻譯的基礎模型。
- Meta 將開放式研究和合作放在首位，並已採取措施降低翻譯中的毒性。用戶可以訪問其他資源並下載模型。

## [Noiselith：輕鬆離線生成圖像](https://noiselith.com/)

- Noiselith 是由 VoyagerX， Inc. 開發的工具，它使用戶能夠在不依賴互聯網連接的情況下在其設備上生成圖像。
- 該工具提供隱私、無限的圖像生成和快速的處理速度。

## [遠端工作成為美國勞動力市場的永久固定裝置](https://www.cnbc.com/2023/11/30/return-to-office-is-dead-stanford-economist-says-heres-why.html)

- 遠端工作正在成為美國勞動力市場的永久固定裝置，工人被召回辦公室的比例趨於穩定。
- 帶薪在家工作的比率一直保持在28%左右的高位，而大流行前的水準為7%。
- 向遠端工作的轉變是由員工對無通勤、靈活的時程表和更好的工作與生活平衡的偏好推動的，而公司則享受到更低的成本和更高的員工保留率。由於技術進步和年輕首席執行官的偏好，預計遠端工作的普及將進一步增長。

### [反應](https://news.ycombinator.com/item?id=38486907)

- 遠端工作是一個討論的話題，對其有效性和挑戰有不同的看法。
- 有人認為遠端工作對生產力有好處，但強調有效的溝通至關重要。
- 遠端工作與面對面工作的優缺點是有爭議的。

- 對富人徵稅的不同觀點、財富集中對社會的影響以及「白手起家」億萬富翁的定義都存在爭議。
- 還討論了億萬富翁對政治的影響，利用慈善基金會作為避稅場所，以及秘密億萬富翁的存在。

## [研究發現耳鳴與隱藏的神經損傷之間存在聯繫](https://scitechdaily.com/tinnitus-linked-to-hidden-undetected-auditory-nerve-damage-a-step-towards-a-cure/)

- 馬薩諸塞州眼耳科醫院的研究人員發現耳鳴與未被發現的聽覺神經損傷之間存在聯繫，挑戰了以前對這種疾病的理解。
- 該研究表明，聽力測試正常的人仍然會出現聽覺神經喪失，這表明以神經再生為中心的新療法的潛力。
- 耳鳴影響全球超過10%的成年人，會對生活質量產生重大影響，導致睡眠障礙、社交退縮、焦慮和抑鬱。研究人員的目標是開發針對耳鳴根本原因的治療方法，以緩解其癥狀。

### [反應](https://news.ycombinator.com/item?id=38495720)

- 對話的參與者討論了他們患有耳鳴的經歷，耳鳴的特點是耳朵不斷響起或嗡嗡聲。
- 探討了耳鳴的潛在原因，包括肌肉動作、下頜問題和神經系統損傷。
- 提到了治療耳鳴的各種方法，例如聽白雜訊、保留聽力和利用特定療法。

## [Mozilla 推出 Llamafile：在電腦上本地運行 LLM](http://simonwillison.net/2023/Nov/29/llamafile/#atom-everything)

- Mozilla 的創新團隊和 Justine Tunney 創建了 llamafile，這是一種在您自己的計算機上運行大型語言模型 （LLM） 的新方法。
- llamafile 是一個多 GB 的檔，其中包含 LLM 的模型權重和運行模型所需的代碼。
- 使用者可以使用 Cosmopolitan Libc 執行 llamafile，它編譯了一個適用於多個操作系統和硬體架構的二進位檔。
- 通過運行 llamafile 可執行檔，用戶可以啟動 Web 伺服器並在瀏覽器中與模型進行交互，從而在沒有 Internet 連接的情況下實現 LLM 的本地使用。

### [反應](https://news.ycombinator.com/item?id=38489533)

- Llamafile 是一種新工具，允許使用者在自己的計算機上運行語言模型 （LM），為 LLM 提供單一的檔分發和執行系統。
- 雖然 Llamafile 對某些用戶來說很方便，但它可能不是性能最好的方法，尤其是對於需要 GPU 加速的模型。
- 建議使用 ExLLAmA v2 + elx2 量化和 tensorrt-llm 等替代方案以獲得更好的性能。
- 用戶討論運行 LLM 的硬體要求和選項，包括使用 Macbook M2 Air 和 LM Studio。
- 關於運行 LLM 的最佳方式，眾說紛紜，一些人認為將模型與可執行檔一起嵌入僅適用於演示，而另一些人則更喜歡 LM Studio 等工具，因為它們易於使用。
