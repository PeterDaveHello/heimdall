## [重新評估鄧寧-克魯格效應：自相關作為解釋](https://economicsfromthetopdown.com/2022/04/08/the-dunning-kruger-effect-is-autocorrelation/)

- 鄧寧-克魯格效應表明，不熟練的人高估了自己的能力，它正在被重新評估為由自相關引起的統計偽影。
- 複製研究沒有發現這種效果的證據，質疑它作為一種心理現象的存在。
- 對原始研究的批評突出了統計錯誤和偏見，強調需要更好的科學分析和對鄧寧-克魯格效應缺陷的認識。

### [反應](https://news.ycombinator.com/item?id=38415252)

- 鄧寧-克魯格效應是一種心理現象，能力較低的人往往會高估自己的能力，而能力較高的人往往會低估自己。
- 圍繞效應的準確性和解釋存在爭論和批評，包括對方法缺陷、重複研究、自相關和社會因素的討論。
- 提到了自我評估的複雜性和冒名頂替者綜合症的存在，這與鄧寧-克魯格效應有關。

## [FLSHclust演算法發現188個新的CRISPR基因，釋放基因組編輯潛力](https://www.nature.com/articles/d41586-023-03697-w)

- 研究人員創建了一種演算法FLSHclust，該演算法可以通過分析基因序列來識別罕見類型的CRISPR系統。
- 該演算法通過檢查數百萬個基因組發現了188個與CRISPR相關的新基因。
- 這些發現為基因組編輯和生物工程的新應用提供了潛力，但需要更多的研究來充分瞭解這些系統及其用途。

### [反應](https://news.ycombinator.com/item?id=38415851)

- 新的CRISPR系統已被發現具有基因組編輯的潛力，為靶向基因編輯提供了更多選擇，併為新型生物技術提供了潛力。
- 現在，使用基於快速局部敏感哈希的聚類演算法可以對大型序列資料庫進行高效分析，從而能夠發現稀有生物系統。
- 然而，必須解決複雜的調控網路和挑戰，才能充分理解基因編輯的影響。

## [介紹Flix：一種具有獨特功能的函數式程式設計語言](https://flix.dev/)

- Flix是由奧胡斯大學和滑鐵盧大學開發的一種程式設計語言，由開源貢獻者提供意見。
- 它是一種功能優先的語言，從 OCaml、Haskell、Rust 和 Scala 中汲取靈感。
- Flix提供了一系列功能，包括代數數據類型、模式匹配、可擴展記錄、類型類、高級類型、多態效應、基於區域的局部突變、純度反射和一流的數據記錄約束。
- 該語言編譯為 JVM 位元組碼，並具有用於開發的 Visual Studio Code 外掛程式。
- Flix旨在提供其他程式設計語言所沒有的獨特功能組合。

### [反應](https://news.ycombinator.com/item?id=38419263)

- 對 Flix 程式設計語言的更新包括容錯、代數效應和改進的包管理。
- 討論了有爭議的設計選擇和函數式程式設計中平衡的必要性。
- 涉及局部突變的使用、函數式程式設計語言的歷史以及純函數的優勢/挑戰。
- 作者認為，編碼風格和方法可以增強純函數式語言。
- 提到了 Rust 等其他語言和 Crumb 等專案。
- 討論了Flix的能力及其與Netflix的聯繫。
- 解決了使用英語作為程式設計語言的局限性。
