## [研究發現，改善深度睡眠可以預防失智](https://www.monash.edu/news/articles/improving-deep-sleep-may-prevent-dementia,-study-finds)

- 莫納什大學最近的一項研究發現，缺乏深度睡眠（稱為慢波睡眠）與老年人患失智的風險增加之間存在聯繫。
- 研究表明，即使60歲以上的人深度睡眠每年下降1%，也會導致失智風險增加27%。
- 該研究對346名60歲及以上的失智患者進行了17年的監測，表明改善或維持慢波睡眠可能是失智的可能預防措施。

### [反應](https://news.ycombinator.com/item?id=38097184)

- 在線討論的重點是通過關注咖啡因的影響來改善深度睡眠和預防失智，特別是對具有緩慢代謝基因CYP1A2的個體。
- 提出了各種策略，包括減少咖啡因攝入量、完成定期鍛煉、保持睡眠衛生和服用特定補充劑。然而，關於基因檢測公司的隱私問題和咖啡因的健康風險被提出來。
- 大家一致認為，不同的個體可能需要根據其獨特的生物因素制定不同的睡眠策略，但所有人都可以從運動、良好的營養和保持睡眠衛生中受益。

- 這些錯誤與較新的 MacOS 版本管理刷新率的方式有關，對配備 ProMotion 顯示幕的 MacBook Pro 機型（14 英寸和 16 英寸版本）產生了重大影響;在升級和啟動序列中合併時，可能會導致黑屏，需要設備固件更新 （DFU） 恢復。

### [反應](https://news.ycombinator.com/item?id=38101328)

- Asahi Linux 是一個志願者團隊，通過識別可能導致 Mac 無法啟動的錯誤來證明其能力，從而超越了 Apple 的專業團隊。
- 該小組的調查結果引發了關於蘋果是否需要投入更多資源進行錯誤修復的討論，以及參與這些調查的志願者應該如何獲得補償。
- 人們擔心在配備Apple Silicone或T2安全晶片的Mac上自動加密數據、由此導致的潛在數據丟失以及隨後對定期數據備份和更新延遲的建議。

## [C++23：刪除垃圾回收支援](https://www.sandordargo.com/blog/2023/11/01/cpp23-garbage-collection)

- C++23打算完全消除對垃圾回收的支援，該功能大部分未實現且不受支援。
- 這一決定被視為整理語言的積極措施，並將導致從 std：： 庫中刪除特定名稱，如 declare_reachable、undeclare_reachable 等。
- 儘管發生了這些變化，現有的垃圾回收器，主要是那些用 C++ 程式設計並運行其他語言的機器的垃圾回收器，不會受到影響，因為它們不依賴於這個標準。

### [反應](https://news.ycombinator.com/item?id=38097984)

- 即將推出的 C++23 標準可能不再支援垃圾回收，導致開發者社區對自動記憶體管理的要求存在不同意見。
- 辯論延伸到C++的複雜性，其不可預見的編碼行為以及消除過多功能的潛在後果，語言障礙使非英語參與者難以理解。
- 對 C++ 和 Rust 進行了比較，表明 Rust 可以作為通常由 C++ 執行的任務的替代方案，從而放大了理解各種系統功能的重要性。

- 有報導稱，Apple 的 macOS Sonoma 上的“grep”功能出現故障，正如開發人員在 Apple 開發者論壇上所強調的那樣。
- 用戶報告了對該錯誤的不同體驗;雖然有些人證實了它的發生，但其他人沒有遇到已發現的問題。
- 一位用戶建議的一種可能的解決方法是在Mac上使用 ssh，並設置不同的 LANG（確定系統語言和區域設置的系統變數）。

### [反應](https://news.ycombinator.com/item?id=38104554)

- 開發人員和研究人員發現，在更新macOS的同時，還要確保數據分析流程的完整性，這些流程通常與遺留代碼相關聯。
- 有人提出了 Docker、MacPorts 和 Nix 等解決方案，以及從 Homebrew 過渡到 MacPorts 以管理 macOS 軟體包，但發現具有挑戰性且工作量增加。
- 由於 macOS 更新、工具和過時應用程式的問題，一些人建議使用 RedHat Enterprise Linux、Rocky Linux 和 Almalinux 等 Linux 發行版。用戶對蘋果過時系統的挫敗感以及對更好的開發人員支援的渴望浮出水面。
