# 2023-10-27

## [雲電腦](https://oxide.computer/blog/the-cloud-computer)

- Oxide宣佈他們聲稱是世界上第一台商用雲計算機的全面上市，並獲得了4400萬美元的A輪融資。
- 該公司的願景涉及可購買的雲計算機，而不是目前的租賃模式，宣導軟體和硬體共同設計的機架級範式。
- Oxide 的雲端電腦旨在減少延遲和噪音，消除不必要的佈線，並提供整合的開源解決方案，而無需單獨許可。

### [反應](https://news.ycombinator.com/item?id=38023891)

- Oxide.computer是一家銷售伺服器和開源軟體的公司，將自己定位為傳統雲供應商和獨立伺服器公司之間的橋樑。他們計劃在即將推出的產品中使用下一個AMD晶元系列。
- 儘管人們擔心使用上一代伺服器和Solaris的衰落，但Oxide已經獲得了AMD的積極反應，並展示了他們在市場上的熟練程度，引發了圍繞潛在市場顛覆的討論。
- 這場辯論還引發了關於開源固件優勢以及擁有與租用硬體的質疑。最終，Oxide尋求提供一個集成的硬體/軟體環境，作為昂貴的雲服務的替代品。

- 討論涉及網頁設計的各個方面，包括移動優先設計的缺點影響桌面使用者，以及旨在提供卓越、包容性在線體驗的“hinternet”理念。
- 它解決了使用者的偏好和需求、市場的局限性以及前端工程師遇到的挑戰，強調了前端開發的艱巨任務。
- 現代網頁設計批評，如模糊的選項和功能，得到了解決，加強了以使用者為中心的設計、可用性、可訪問性、同行評審在網頁設計中的重要性。

## [I2P：端到端加密和匿名互聯網](https://github.com/PurpleI2P/i2pd)

- GitHub 儲存庫“PurpleI2P/i2pd”專用於不可見互聯網協定 （I2P） 用戶端的 C++ 實現，該用戶端支援匿名和安全加密的通信。
- 該存儲庫包含用於開發和利用i2pd用戶端的代碼、文檔和資源。
- 該專案根據 BSD 3 條款許可證獲得許可，由貢獻者社區積極維護。

### [反應](https://news.ycombinator.com/item?id=38026736)

- 爭論的焦點是I2P，這是一個加密和匿名的互聯網專案，旨在避免單一實體對使用者匿名的控制。儘管如此，I2P並沒有受到太大的歡迎，經常被資金更慷慨的Tor專案所忽視。
- 用戶建議，在ISP在硬體和對等互連方面投入更多資金之前，VPN對於增強連接仍然至關重要。它還研究了從點對點架構到集中式架構的轉換及其可能的缺點。
- 重點介紹了 I2P 在種子下載方面的速度和實用性與 Tor 和 VPN 等替代品的比較。討論還提到了 Java 在其他去中心化系統（如 Freenet、PerfectDark 和 Maidsafe）中的潛在性能和跨平臺功能。

## [獨家硬木可能被非法採伐](https://www.chalmers.se/en/current/news/75-percent-of-exclusive-hardwood-may-be-illegally-harvested,c3863187/)

- 查爾姆斯理工大學（Chalmers University of Technology）的一項研究顯示，超過75%的ipê（一種用於甲板的搶手熱帶木材）可能是從巴西的主要產區非法採伐的。
- 該研究發現，市場上有16%的木材是在沒有適當許可證的情況下砍伐的，流通中的木材數量超過了官方的生產統計數據。
- 該研究強調了消費者在確保可持續木材採購方面發揮的關鍵作用，他們質疑木材的來源並在購買前要求提供檔。

### [反應](https://news.ycombinator.com/item?id=38025998)

- 對話探討了木材在建築和地板中與替代材料的對比，並評估了非法採伐對環境的影響。
- 討論者還考慮了區塊鏈技術在追蹤木材來源方面的潛力、未經授權的木材採伐和森林砍伐問題，以及各種類型地板的優缺點。
- 建議採取《瀕危野生動植物種國際貿易公約》（CITES）等監管執法措施，以減少非法木材需求，並提出植樹和建立原產地文件等解決方案。

## [帶GPS的鳥飛入颱風](https://newatlas.com/biology/bird-typhoon-ride/)

- 一隻雄性海鳥在颱風上方不尋常的 700 英里飛行中倖存下來，達到 15,420 英尺的高度和 106 英里/小時的速度，這與它們的正常飛行模式有很大不同。
- 這隻鳥的旅程，以及其他13隻海鳥的旅程，使用科學家附帶的GPS生物記錄器進行跟蹤，突出了氣候變化和極端天氣對海鳥種群的潛在影響。
- 更多的研究對於瞭解中上層鳥類如何適應這些氣候變化至關重要。這項研究的結果發表在《生態學》雜誌上。

### [反應](https://news.ycombinator.com/item?id=38030705)

- 一隻裝有GPS的鳥類成功穿越颱風回到其繁殖地，展示了鳥類的導航能力和GPS技術在動物研究中的潛力。
- 對話深入探討了各種主題，包括鳥類在磁場中的導航、鳥類生理學、雲的形成、風暴中的行為，以及鳥類飛行對未來運輸技術的影響。
- 討論還涉及使用旋風處理垃圾、颱風中鳥類的存活率、氣候變化對海鳥種群的影響，以及關於氣候變化是否直接導致颶風或僅僅加劇颶風頻率和嚴重程度的爭論。
