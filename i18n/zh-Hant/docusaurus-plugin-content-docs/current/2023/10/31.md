## [私募股權正在吞噬美國經濟](https://www.theatlantic.com/ideas/archive/2023/10/private-equity-publicly-traded-companies/675788/)

- 私募股權公司收購上市公司並將其轉變為私人實體的趨勢日益增長，這引起了人們的關注，因為這使得市場的很大一部分對投資者、監管機構和媒體的透明度降低。
- 這種向私募股權的轉變是由放鬆管制推動的，導致了潛在的有害後果，如裁員、服務品質下降和破產，而主要使公司的合作夥伴受益。
- 政府已提出修正案，以增加對私募股權顧問的披露要求，但可能需要採取更有力的措施來處理潛在的系統性風險並驗證更高回報的聲明。

### [反應](https://news.ycombinator.com/item?id=38069197)

- 美國私募股權的興起引發了一些問題，例如普通投資者接觸高增長公司的機會很少，企業家的機會更少，以及財富集中。
- 爭論的主要焦點包括某些行為的道德和合法性、將住房視為投資的觀點、401k 退休計劃的缺陷、散戶投資者面臨的挑戰以及退休儲蓄的充足性。
- 人們的擔憂還集中在退休計劃的複雜性、對 401k 計劃的信任、供款的可負擔性以及住房和醫療保健成本對退休儲蓄的影響上，這些通常被認為是不充分的。

## [Raspberry Pi 5 沒有硬體視頻編碼，只有 HEVC 解碼](https://www.raspberrypi.com/news/introducing-raspberry-pi-5/#comment-1594055)

- Raspberry Pi 基金會宣佈發佈 Raspberry Pi 5，這是一款具有重大升級的單板電腦，包括更快的處理器、改進的圖形功能和增強的連接選項。
- 新的Raspberry Pi操作系統，名為“Bookworm”，伴隨著硬體的發佈，引發了關於系統功能和潛在改進的大量討論。
- 使用者對新 Pi 5 中缺乏 TrustZone 安全性和 USB-C 支援表示擔憂，Raspberry Pi 工作人員目前正在澄清和解決這些問題。

- Raspberry Pi 5 缺乏硬體視頻編碼功能，導致需要視頻相關任務的使用者感到失望。批評者對關注更昂貴的新車型而不是使現有車型更經濟、更方便表示擔憂。
- Raspberry Pi 基金會似乎將工業企業使用者置於教育和業餘愛好者之上，引發了關於 Beaglebone Black 和 Orange Pi 等替代單板電腦的討論。
- 爭論的焦點包括使用迷你HDMI埠，取消了對HEVC視頻編碼的硬體支持，推測這是由於業務和許可成本，以及Raspberry Pi 5與競爭對手之間的功耗性能比較。

## [德國法院宣佈「請勿跟蹤」具有法律約束力](https://www.vzbv.de/urteile/gericht-untersagt-datenschutzverstoesse-von-linkedin)

- 德國消費者聯合會（vzbv）在針對LinkedIn的訴訟中獲勝，指控他們違反數據隱私法規。
- 該判決指出，LinkedIn不得告知使用者它忽略了“請勿跟蹤”信號，允許個人選擇不跟蹤他們的瀏覽行為。
- 該裁決還禁止LinkedIn在未經同意的情況下在不同的網站上自動填充使用者個人資料，並限制他們向非會員發送未經請求的電子郵件。

- 德國最近的一項法院判決在法律上承認了“請勿跟蹤”（DNT）信號，禁止LinkedIn無視它或在其他平臺上設置使用者個人資料。
- 該裁決可能會影響其他網站，並可能消除每個網站上對唯一 cookie 通知的需求，從而緩解使用者對跟蹤 cookie 的挫敗感。
- 圍繞 DNT 的對話涉及其實施、有效性以及網站尊重信號的法律要求，引發了關於是否應將 DNT 視為拒絕跟蹤的辯論。

### [反應](https://news.ycombinator.com/item?id=38072495)

- 軟體公司 Threads Software Limited 對 Meta 提出質疑，聲稱在 Meta 的社交媒體平臺上使用“Threads”名稱侵犯了商標權。
- Threads Software Limited 表示，它於 2012 年註冊了該名稱的商標，並於 2014 年開始運營名為 Threads 的產品。Meta 試圖從 Threads Software 購買功能變數名稱“threads.app”，但沒有成功。
- Meta於 2023 年 7 月推出了其“Threads”社交媒體平臺，隨後將 Threads Software Limited 從其 Facebook 平台上刪除。這場法律爭論的結果尚未確定，嚴重依賴兩個實體的法律策略和資源。
