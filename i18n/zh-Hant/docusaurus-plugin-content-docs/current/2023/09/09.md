## [包子 v1.0.0](https://bun.sh/blog/bun-v1.0)

- Bun 1.0 是一個工具包，旨在簡化 JavaScript 和 TypeScript 開發，取代多個工具並提供與現有 Node.js 應用程式的相容性。
- 它帶有TypeScript，JSX支援和內置的Web API，使其成為一個功能強大的JavaScript運行時，據報導每秒可以處理比Node.js更多的請求。
- 其他功能包括 WebSockets 支援和包管理。開發Bun 1.0的公司位於三藩市，目前正在招聘員工。

### [反應](https://news.ycombinator.com/item?id=37434117)

- 討論集中在 Bun 上，這是 Node.js 生態系統中的一個工具，它同時支援 CommonJS 和 ES 模組，用戶強調了優點和缺點。
- 對話還闡明瞭使用 Discord 和 GitHub 作為開源專案的交流平臺。
- 將Bun工具與Deno和Vite等其他工具進行比較，對其穩定性和許可提出了其他擔憂。

## [也許 Rust 不是大規模併發使用者空間軟體的好工具](https://bitbashing.io/async-rust.html)

- 作者闡明瞭Async Rust的目的，Async Rust是一種專注於現代併發的程式設計語言，區分並行性和併發性，並概述了代碼拆分的局限性。
- 由於競爭條件和死鎖的風險，基於互斥的併發被認為是有害的;因此，建議使用通道或佇列進行線程間通信。但是，對於高級併發問題，線程可能不夠用，這促使 Rust 的“異步/等待”模型。
- 儘管有這些優點，但 Async Rust 仍存在挑戰，例如確保編譯器對對對象的安全性和 Arc（原子引用計數）的普遍使用。Rust 的大規模併發使用者空間軟體應用程式的適用性仍然值得懷疑。

### [反應](https://news.ycombinator.com/item?id=37435515)

- 該演講涵蓋了圍繞 Rust 程式設計的各種主題，例如記憶體安全、併發性、圖形開發、生產力和記憶體管理。
- 參與者的分享經驗闡明瞭使用 Rust 的優勢和挑戰，以及它在各種用例中的適用性。
- 該交互還提供了與其他語言（如 Python 和 Go）的比較，並討論了繼承和抽象等多種程式設計概念，提供了對 Rust 的優勢、劣勢和不斷增強努力的看法。

## [FTC法官裁定Intuit的「免費」TurboTax廣告誤導了消費者](https://www.theverge.com/2023/9/8/23864538/turbotax-intuit-ftc-deceptive-practices-free-tax-filing)

- 聯邦貿易委員會（FTC）裁定，TurboTax的母公司Intuit通過宣傳其報稅服務“免費”來誤導消費者。
- 在 FTC 訴訟後做出裁決並支付 1.41 億美元後，Intuit 現在被要求明確披露任何限制其免費產品供應的條款。
- 鑒於在嘗試查找Intuit的實際免費報稅服務「免費報稅」時遇到的困難，國稅局正在開發一項政府支持的免費報稅服務，該服務將在2024年報稅季節進行試點。

### [反應](https://news.ycombinator.com/item?id=37437311)

- 討論圍繞著使用者對Intuit的TurboTax產品的不滿，因為未公開的費用，欺騙性的促銷和缺乏透明度。
- 替代報稅選項，如FreeTaxUSA和Cash App Tax，建議作為更透明和更具成本效益的解決方案。
- 遊說影響、稅收制度腐敗、誤導性廣告和制度改革呼籲等話題突出了這些用戶投訴的更大背景，並暗示需要對納稅申報系統進行重大改革。

## [由於空氣污染，蜜蜂難以找到鮮花](https://www.birmingham.ac.uk/news/2023/bees-struggle-to-find-flowers-because-of-air-pollution)

- 英國生態與水文中心和多所大學的一項新研究表明，空氣污染阻礙了蜜蜂尋找花朵的能力，因為臭氧會降低花朵的香味。
- 研究發現，蜜蜂的氣味識別能力降低了90%，這表明對野花豐度和作物產量有潛在的不利影響，因為蜜蜂在對糧食安全至關重要的授粉中起著至關重要的作用。
- 該研究還表明，臭氧可能會影響昆蟲的其他氣味影響行為，例如配偶吸引，因此有必要對空氣污染採取行動以保護糧食生產和生物多樣性。

### [反應](https://news.ycombinator.com/item?id=37434381)

- 據稱，空氣污染導致蜜蜂活動減少和難以找到花朵，導致昆蟲種群減少。其他因素可能包括交通和噪音污染，以及蜜蜂資源的減少。
- 一些出席者指出，蜜蜂活動和野生動物有所增加，這可能是由於有利的條件或為培育昆蟲友好環境所做的努力。
- 討論指出了污染對蜜蜂對特定植物的吸引力的可能影響，導致蜜蜂適應污染的進化壓力的暗示，以及通過催化轉換器解決車輛污染的創新解決方案。
