# 2023-09-15

## [邊緣使用者的暴政](https://nothinghuman.substack.com/p/the-tyranny-of-the-marginal-user)

- 這篇文章批評了消費者軟體的衰落，例如OKCupid等約會應用程式，原因是重點轉向以改善現有用戶體驗為代價來吸引新使用者。
- 引入了「邊緣使用者」的概念，即注意力短暫、喜歡簡單內容的使用者。這種使用者類型的優先順序被視為導致軟體品質和使用者代理的下降。
- 作者指出，增強使用者代理的工具通常由業餘愛好者開發，經常被大公司收購和停產。

### [反應](https://news.ycombinator.com/item?id=37509507)

- 這篇文章討論了OkCupid向以移動使用者為中心的方法的轉變，據報導，這導致了對話品質的下降和更多表面約會應用程式的興起。
- 它還強調了軟體開發中增長模式的負面影響，批評他們關注經濟成功而不是用戶體驗，並指出迎合即時滿足可能導致產品垮臺。
- 本文關注設計最佳道德體系的挑戰，重點關注使用者介面設計對內容傳輸的影響、功利主義的挑戰以及對資源消耗和不平等的日益關注等問題。

## [Unity 似乎默默地刪除了跟蹤 ToS 更改的 GitHub 儲存庫。](https://www.gamerbraves.com/unity-silently-deletes-github-repo-that-tracks-terms-of-service-changes-and-updated-its-license/)

- 遊戲開發平臺Unity悄悄地刪除了一個GitHub存儲庫，該存儲庫跟蹤其服務條款（ToS）的變化以及允許開發人員使用舊版本遊戲引擎的條款。
- 刪除存儲庫后，該網頁將無法再訪問，最後一次可用時間為 2022 年 7 月 16 日。
- Unity ToS 的主要變化包括刪除了一項條款（2023 年 4 月），該條款允許開發人員根據更新時的條款使用當年的軟體版本;這迫使使用者遵守Unity的所有服務更改，例如最近的定價計劃更新，向開發人員收取每個遊戲安裝的費用。

### [反應](https://news.ycombinator.com/item?id=37516523)

- Unity已經刪除了跟蹤其服務條款（ToS）修訂的GitHub存儲庫，引發了對透明度和信任的擔憂。
- 這些對 ToS 的追溯性更改讓遊戲開發人員感到不安，他們的工作基於以前的成本結構。
- Unity的行為引起了有關其合法性和道德規範的質疑，並且對虛幻引擎進行了不利的比較，虛幻引擎沒有收取類似的每次安裝費用。

## [在 Git 儲存庫中，您的文件位於何處？](https://jvns.ca/blog/2023/09/14/in-a-git-repository--where-do-your-files-live-/)

- Git 將檔作為不同的物件存儲在儲存庫內的 .git/objects 目錄中，採用內容尋址存儲，其中檔名是文件內容的哈希。
- Git 不儲存單個版本的差異;每個版本都另存為一個完整的檔。要查找舊版本，可以使用“git log”來查找與檔更改相關的提交 ID。
- Git 使用 zlib 格式壓縮物件。作者提供了一個使用 Git 檢索舊版本的博客文章的實際示例，並附有代碼片段。

### [反應](https://news.ycombinator.com/item?id=37512888)

- 本文全面概述了 Git，包括其獨特的功能、故障排除方法、內部工作原理以及利用有向無環圖 （DAG） 來跟蹤更改。
- 對 Git 使用者介面的批評與潛在的改進一起被探索，儘管有論點支援它在版本控制系統領域的靈活性和一致性。
- 這篇文章還討論了 Git 的複雜性和可用性，將其受歡迎程度與其他選項進行了比較，並概述了未來創新和擴展的潛力。

### [反應](https://news.ycombinator.com/item?id=37510885)

- 被盜的梵古畫作被找回並裝在宜家包中歸還，通過藝術偵探亞瑟·布蘭德（Arthur Brand）運營的地下網路提供便利。
- 這幅畫於2002年被盜，一直在犯罪的黑社會中流傳，它的歸還說明瞭與擁有被盜藝術品有關的法律問題。
- 帖子中還討論了有關繪畫保護，藝術修復的主題，以及對有關藝術品盜竊敘事的書籍和系列的建議。
