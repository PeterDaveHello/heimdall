## [ZenHammer：AMD Zen 平臺容易受到 Rowhammer 的攻擊](https://comsec.ethz.ch/research/dram/zenhammer/)

- ZenHammer 研究表明，儘管 TRR 緩解措施，但基於 AMD Zen 的平臺仍容易受到 Rowhammer 攻擊，與 Intel Coffee Lake 相比，Zen 3 系統顯示出更高的漏洞。
- 該研究還對DDR5設備進行了測試，表明瞭潛在的位翻轉風險，該研究的模糊器工具可在Github上訪問，以進行其他調查。
- 需要進一步的研究來衡量 DDR5 設備上漏洞的全部範圍，該研究在研究資助的支援下，按照負責任的披露協定與 AMD 共用。

### [反應](https://news.ycombinator.com/item?id=39819599)

- 基於 AMD Zen 的平臺容易受到 Rowhammer 攻擊，這凸顯了使用 ECC 記憶體來增強安全性的必要性。
- 在 AMD 平台上實施 ECC 記憶體會帶來挑戰和成本，並且 CPU 型號的支持級別各不相同。
- 本文強調了監控和解決記憶體錯誤的重要性，並強調了 ECC 記憶體錯誤的重要性，並強調了 ECC 記憶體檢測和緩解對 Ryzen CPU 的攻擊方面的功效。
