## [jnv 簡介：帶有 jq 過濾器編輯功能的互動式 JSON 查看器](https://github.com/ynqa/jnv)

- jnv 是一個用於導航 JSON 檔的工具，由於 Rust 綁定，它提供了一個互動式 JSON 查看器和 jq 過濾器編輯器，無需安裝 jq。
- 用戶可以通過 Homebrew 或 Cargo 安裝 jnv，並受益於語法突出顯示、自動完成和鍵綁定等功能，以便於導航。
- 該工具支援從 stdin、檔或 URL 讀取 JSON，允許自訂編輯模式、縮進、提示顯示和建議清單長度等設置，同時提供 JSON 可視化控制的範例和選項。

### [反應](https://news.ycombinator.com/item?id=39759325)

- 這篇文章介紹了「jnv」，這是一個利用 jq 的互動式 JSON 過濾器，用戶可以在其中討論其性能並建議 fx、jiq 和 ijq 等替代方案。
- 使用者分享反饋，提出改進想法，並提到使用 fzf 等外掛程式進行互動式查詢和過濾 JSON 數據。
- 討論的重點是用於互動式查詢和過濾 JSON 數據的各種工具和外掛程式，展示了社區對增強此類功能的興趣。

## [研究：腦細胞中脂肪堆積與阿爾茨海默氏症風險有關](https://medicalxpress.com/news/2024-03-root-alzheimer-fat-buildup-brain.html)

- 《自然》雜誌上的一項研究表明，腦細胞中的脂肪堆積，而不是β-澱粉樣蛋白或tau蛋白，可能是阿爾茨海默病的主要驅動因素。
- 研究人員認為，負責將更多脂肪輸送到腦細胞中的APOE4基因變異可能會增加患阿爾茨海默氏症的風險。
- 研究結果表明，澱粉樣蛋白的堆積可能會引發腦細胞中的脂肪堆積，從而可能導致阿爾茨海默氏症的進展。

### [反應](https://news.ycombinator.com/item?id=39760333)

- 研究表明，腦細胞中的脂肪堆積可能是阿爾茨海默病的主要誘因，導致人們專注於開發靶向藥物。
- 在阿爾茨海默氏症的討論中，辯論涵蓋了遺傳影響、飲食脂肪影響和對健康信念的錯誤資訊。
- 主題還包括脂肪堆積與阿爾茨海默氏症之間的聯繫，GLP1類似物等治療方法，以及基因檢測和健康生活方式選擇對疾病管理的重要性。

## [Long Covid 中的腦霧與受損的腦血管有關](https://www.sciencenews.org/article/long-covid-brain-fog-blood-brain-barrier-damage)

- 科學家發現了長期 COVID 患者腦霧的生物學解釋，將其歸因於通過 MRI 掃描顯示的大腦血管滲漏。
- 這些患者大腦中受損的血管使異物滲入腦組織，導致認知障礙。
- 這一突破可能會導致開發治療方法，以防止或消除患有腦霧的長期COVID患者的血腦屏障惡化。

### [反應](https://news.ycombinator.com/item?id=39762776)

- 討論深入探討了 Covid-19 的長期影響，例如「長期 Covid」腦霧和與自身免疫相關的健康問題，
- 關於腸道健康、自身免疫性疾病和 Covid 後認知癥狀之間的潛在聯繫的研究正在進行中，爭論 Long Covid 與其他病毒后綜合征相比的獨特性，
- 人們努力瞭解 Covid-19 對大腦的影響，區分錯誤歸因的癥狀和真正的病毒相關損害，一位評論者建議 Covid-19 是地方病，大流行正在消退。
