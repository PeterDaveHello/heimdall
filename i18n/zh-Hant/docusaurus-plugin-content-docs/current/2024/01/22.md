## [Flipper Zero：面向駭客和極客的多功能工具](https://flipperzero.one)

- Flipper Zero 是一款專為滲透測試人員和極客設計的便攜式多功能工具設備，為駭客攻擊數字系統（如無線電協定、訪問控制系統和硬體）提供定製和開源功能。
- 它具有 1.4 英寸單色 LCD 顯示幕、5 按鈕方向鍵以及藍牙、NFC 和紅外線等各種連接選項。
- 該設備可以讀取和類比RFID卡，具有用於存儲的microSD卡插槽，並且可以自主控制或連接到計算機或智慧手機以獲得高級功能，使硬體探索和開發更加愉快和方便。

- Flipper Zero 是一款多功能多功能設備，具有多種功能，包括充當紅外遙控器。
- 與其他替代品和手機上裝有紅外發射器的中國品牌相比，它引發了關於其可負擔性的討論。
- 該設備面臨爭議和限制，包括iPhone崩潰，被亞馬遜禁止，以及對數據交換能力和隱私影響的擔憂。

## [GPT-3.5 漏洞：拼寫錯誤的短語導致錯誤消息並引發數據中毒問題](https://iter.ca/post/gpt-crash/)

- 在對 ChatGPT 3.5 的響應頂部添加特定短語會持續觸發錯誤消息。
- GPT-4 或使用 GPT-3.5 API 時不會發生此錯誤。
- 該錯誤可能是由 OpenAI 的 GPT 模型中使用的標記化過程引起的。
- 特定短語是在 Katalon Studio 軟體的 XML 設定檔中找到的拼寫錯誤的選項名稱。
- 用於訓練 GPT-3.5 的數據集可能排除了 XML 檔，導致模型不知道拼寫錯誤的令牌並產生無效回應。
- 這一發現引發了人們對潛在數據中毒和破壞 GPT-3.5 摘要能力的擔憂。

### [反應](https://news.ycombinator.com/item?id=39086106)

- 討論的重點是 AI 語言模型，尤其是具有特定標記的故障、標記生成及其對文本轉換的影響。
- 對話包括訓練集、分詞器以及 GPT-3.5 和 GPT-4 模型的外觀和功能等主題。
- 用戶對視覺問題、GPT-4 與 API 的速度和實用性的看法不一，以及對 OpenAI 的方法和證明思維的能力持懷疑態度。

## [科學家開發肽化合物以靶向助長癌症的蛋白質](https://phys.org/news/2024-01-scientists-chaotic-protein-fueling-cancers.html)

- 加州大學河濱分校的研究人員創造了一種肽化合物，該化合物靶向癌細胞中常見的蛋白質MYC。
- 通過修改肽的結構，研究人員增強了其與MYC相互作用的能力，使其能夠與蛋白質牢固結合。
- 這一發展在開發新的癌症治療方法方面顯示出前景，研究人員專注於提高肽進入細胞的能力。

### [反應](https://news.ycombinator.com/item?id=39084146)

- 科學家們已經確定了一種分子，可以阻斷MYC蛋白的活性，MYC蛋白是75%的癌症的燃料。
- 這一發現為癌症提供了一種潛在的治療選擇，但其有效性、遞送方法和潛在的副作用仍然未知。
- 雖然這是一個很有希望的發展，但重要的是要注意它仍處於早期階段，尚未在人體上進行測試。癌症研究是一個複雜的領域，不能保證成功，但它在推進癌症治療方面發揮著至關重要的作用。

## [谷歌承認在隱身模式下跟蹤使用者，儘管有隱私聲明](https://news.abplive.com/technology/google-incognito-new-disclaimer-canary-version-you-are-being-tracked-even-in-google-incognito-mode-new-disclaimer-is-up-privacy-lawsuit-1658159)

- 谷歌更新了其隱身模式的免責聲明，以明確表示儘管聲稱存在隱私要求，但用戶數據仍會被收集。
- 更新後的免責聲明指出，雖然使用同一設備的其他人不會看到用戶的活動，但訪問的網站和使用的服務（包括谷歌本身）仍將收集數據。
- 這證實了谷歌即使在用戶處於隱身模式時也會跟蹤用戶活動的懷疑，因此建議使用者在使用它時要小心。

### [反應](https://news.ycombinator.com/item?id=39078417)

- 關於谷歌在隱身模式下的跟蹤做法以及對使用者隱私的潛在影響，一直存在爭議。
- 一些用戶聲稱，即使在隱身模式下，谷歌仍然會跟蹤他們的活動，而另一些使用者則質疑跟蹤的程度及其與非隱身會話的聯繫。
- 維護隱私的建議包括使用替代瀏覽器或面向隱私的工具，並且呼籲谷歌就隱身模式的局限性進行更清晰的溝通。

## [水牛魚抗衰老：有些活過 100 歲並保持健康](https://www.nationalgeographic.com/premium/article/buffalofish-oldest-hundred-years-mystery)

- 最近在亞利桑那州阿帕奇湖進行的一項研究發現，水牛魚是一種淡水魚，可以活 100 多年，並且隨著年齡的增長具有強大的免疫系統和應激反應。
- 以前，人們認為水牛只能活到 20 多歲。
- 研究結果表明，對水牛長壽的進一步研究可以為延長包括人類在內的脊椎動物的壽命提供見解。此外，該研究強調了保護工作的必要性，因為水牛魚種群正在下降，目前沒有關於其捕撈的規定。

### [反應](https://news.ycombinator.com/item?id=39078061)

- 討論主題涵蓋了各種主題，例如魚類的壽命、不朽對社會的影響、延長壽命技術的進步以及人口過剩問題。
- 它強調了適應性、社會公平和政府干預的潛在需求的重要性。
- 談話包括幽默和諷刺的評論以及嚴肅的討論。
