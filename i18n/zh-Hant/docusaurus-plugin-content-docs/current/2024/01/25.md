# 2024-01-25

## [Zed 代碼編輯器開源，強調協作](https://zed.dev/blog/zed-is-now-open-source)

- Zed 是一個代碼編輯器，它已經過渡到一個開源專案，以增強產品並與程式設計社區互動。
- Zed 現在在 GitHub 上以 copyleft 和 Apache 2 許可證提供，允許開發人員訪問和修改代碼。
- 該公司計劃推出Fireside Hacks，這是一種即時協作編碼功能，同時通過提供整合服務和對特定功能的私人使用收費來保持可持續的商業模式。他們鼓勵社區貢獻，併為參與提供指南。

### [反應](https://news.ycombinator.com/item?id=39119835)

- 討論涵蓋了各種代碼編輯器，包括 Zed、Visual Studio Code （VSC）、Emacs 和其他替代方案。
- 使用者分享有關提高性能、協作、自託管聊天伺服器、啟動時間和包管理的經驗和建議。
- 對話涉及許可、與不同操作系統的相容性、協作編輯選項的可用性，並強調了實時協作和效率對結對程式設計的重要性。

## [OpenAI 背棄披露關鍵文件的承諾，引發擔憂](https://www.wired.com/story/openai-scrapped-promise-disclose-key-documents/)

- 非營利性人工智慧研究實驗室OpenAI違背了向公眾披露關鍵文件的承諾，引發了對其透明度的擔憂。
- 該公司成立的目的是讓社會參與人工智慧的發展，但當被要求提供管理文件和財務報表時，OpenAI 拒絕提供。
- 批評人士認為，增加披露可以提高客戶忠誠度和監管信任度，訪問這些檔可以深入瞭解OpenAI的漏洞，與Microsoft等利益相關者的關係，以及為穩定其公司結構所做的任何修訂。

### [反應](https://news.ycombinator.com/item?id=39121521)

- OpenAI 因其缺乏透明度和向盈利能力轉變而面臨批評，這引發了人們對其造福人類的承諾的擔憂。
- 參與者對OpenAI的行為表示失望和懷疑，質疑其完整性。
- 對話強調了通用人工智慧 （AGI） 發展的影響、科技行業以利潤為導向的重點，以及對人工智慧進行監管和公開討論的必要性。

## [瞭解現代融資：介紹開源創業融資模擬器](https://www.fundingsimulator.com/)

- 作者開發了一個開源工具，以幫助創始人更好地瞭解現代籌款以及他們可能經歷的潛在稀釋。
- 該工具允許使用者添加投后保險箱、定價輪次和員工選項，以查看對擁有權的影響，從而簡化瞭解啟動資金的過程。
- 作者認識到啟動資金所涉及的複雜性，包括不同類型的保險箱、期權分配、按比例分配的權利和最惠國條款，並歡迎該工具用戶的反饋。

### [反應](https://news.ycombinator.com/item?id=39120647)

- 討論圍繞著 Fundingsimulator.com 展開，這是一種幫助創始人瞭解籌款複雜性和潛在稀釋的工具。
- 使用者提供有關該工具功能的反饋並提出改進建議。
- 其他討論包括創業融資的挑戰和好處，風險投資在業務增長不同階段的相關性，以及技能和向成功創始人學習的重要性。還探討了初創公司的擁有權和財務複雜性以及擁有聯合創始人的潛在優勢。

## [駭客入侵小米空氣凈化器：使用RFID晶元駭客重置篩檢程式警告](https://unethical.info/2024/01/24/hacking-my-air-purifier/)

- 作者講述了他們使用小米空氣凈化器的經歷，該空氣凈化器具有不可拆卸的更換篩檢程式警告。

### [反應](https://news.ycombinator.com/item?id=39115837)

- 用戶討論了空氣凈化器中DRM（數字版權管理）的挫敗感，並分享了駭客入侵設備以繞過限制的經驗。

- 建議使用其他解決方案，例如使用差壓感測器代替DRM，並呼籲廢除DMCA（數位千年版權法）和消費者的維修權。

## [自身免疫性疾病治療的進展點燃了希望](https://www.nature.com/articles/d41586-024-00169-7)

- 科學家們在開發糖尿病、狼瘡和多發性硬化症等自身免疫性疾病的療法方面取得了進展。
- 有前途的方法包括對流氓細胞施用抗原，選擇性地消除有問題的細胞，以及引入工程免疫細胞。
- 稱為navacims的納米顆粒和肝臟靶向抗原療法分別在關閉自身免疫反應和逆轉多發性硬化症癥狀方面顯示出希望。此外，涉及嵌合自身抗體受體 T 細胞的療法已顯示出狼瘡和其他疾病患者的顯著改善。

- 科學家們正在探索使用抗原來訓練流氓細胞進行攻擊，以期治癒自身免疫性疾病，但每個人都擔心有效性。
