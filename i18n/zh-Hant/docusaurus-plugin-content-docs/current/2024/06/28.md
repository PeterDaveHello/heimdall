## [Python 面臨 Apple App Store 的拒絕問題](https://lwn.net/SubscriberLink/979671/4fb7c1827536d1ae/)

- 從 Python 3.11 升級到 3.12 導致一些 Python 應用程式被 Apple 的應用程式商店拒絕，原因是 urllib 解析器中存在字串 "itms-services"，這在 macOS 的沙盒應用程式中是被禁止的。
- 「Eric Froemling 的錯誤報告引發了 Python 開發者之間的討論，討論如何適應應用程式商店的審核流程，考慮了像混淆和發佈層級修補等解決方案。」
- 已達成共識，將在 Python 3.13 中新增一個名為 "--with-app-store-compliance" 的建置時選項，這將移除對應應用程式商店提交的問題代碼，突顯了自由軟體專案在面對不透明審查過程時所面臨的挑戰。

### [反應](https://news.ycombinator.com/item?id=40815130)

- 由於二進位檔中包含了 "itms-services" URL 方案，Python 應用程式被 Apple App Store 拒絕上架，並且這些應用程式在使用 PyInstaller 打包時也會被 Windows Defender 標記為有問題。
- 開發者認為主要的作業系統平台通過要求昂貴的程式碼簽署證書，正在阻礙小型開發者，這使得應用程式的分發變得困難。
- 建議的替代方案包括使用 Homebrew 來管理 Mac 應用程式，或使用 SignPath 提供的免費證書來進行 Windows 構建，這突顯了應用程式分發中與安全相關的拒絕問題。

## [捕捉重力波的新方法](https://www.nature.com/articles/d41586-024-02003-6)

- 2015年，LIGO和Virgo天文台首次直接探測到重力波，這一發現徹底改變了物理學，使天文學家能夠「聆聽」宇宙事件。
- 目前的探測器僅限於100至1,000赫茲之間的頻率，但新的儀器和技術旨在偵測更廣泛的範圍，從兆赫茲到奈赫茲的頻率。
- “五種新方法包括脈衝星計時陣列、微波望遠鏡、原子干涉儀、桌面探測器和量子晶體，每種方法針對不同的頻率範圍，並有望揭示更多宇宙現象。”

### [反應](https://news.ycombinator.com/item?id=40820063)

- 第一代重力波探測器未能探測到波，推翻了韋伯1987年的主張，但目前的干涉儀如LIGO、Virgo和KAGRA已成功探測到它們。
- “重力波證實了愛因斯坦的廣義相對論，但量子重力理論仍未解決，弦理論和迴圈量子重力提供了潛在但具有挑戰性的解釋。”
- 未來像 LISA 這樣的專案旨在提升偵測能力，LIGO 設施也提供公眾參觀以供教育用途。

## [帕金森病與腸道細菌的關聯暗示出意想不到且簡單的治療方法](https://www.sciencealert.com/parkinsons-link-to-gut-bacteria-suggests-unexpected-simple-treatment)

- 研究人員已經確定了與帕金森氏症患者體內核黃素(維生素B2)和生物素(維生素B7)水平降低相關的特定腸道微生物，這表明補充維生素B可能是一種潛在的治療方法。
- 這項研究由名古屋大學的西脇宏領導，發現腸道細菌的變化影響了短鏈脂肪酸(SCFAs)和多胺的生成，這對於維持健康的腸道黏液層至關重要。
- 這項研究結果發表在《npj Parkinson's Disease》期刊上，顯示維持健康的腸道微生物群並減少環境毒素，可能有助於預防帕金森氏症的症狀。

### [反應](https://news.ycombinator.com/item?id=40818987)

- 一項研究表明腸道細菌與帕金森氏症之間存在關聯，指出某些細菌菌株可能會促成該疾病的發生。
- 關於補充維生素如核黃素(B2)和生物素(B7)以緩解症狀的潛在益處，已經展開了討論。
- 這項研究強調了腸道健康對神經系統疾病的重要性，但建議需要進行更多的研究，並且不應忽視現有的治療方法。
