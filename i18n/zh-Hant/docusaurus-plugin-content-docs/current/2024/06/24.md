## [蘋果被發現違反歐盟競爭規則](https://www.theguardian.com/technology/article/2024/jun/24/apple-breach-eu-competition-rules-digital-markets-act)

- 歐洲委員會發現蘋果違反了旨在保護較小競爭對手並使消費者能夠找到更便宜應用程式的新歐盟法律。
- 蘋果公司有12個月的時間來遵守這些規定，否則將面臨最高達其全球收入10%的罰款，這是根據三月份啟動的調查結果所做出的決定。
- 這標誌著自《數位市場法》(DMA)頒布以來，對蘋果進行的第三次不合規調查，重點關注蘋果的費用和限制開發者引導消費者選擇替代購買選項的做法。

### [反應](https://news.ycombinator.com/item?id=40773883)

- 蘋果被發現違反歐盟競爭規則，引發了在美國採取類似行動的呼聲。
- 批評者認為，蘋果對應用程式發佈和開發者費用的控制抑制了競爭，特別是對於防止開發者告知用戶在 App Store 之外有更便宜支付選項的反引導規則感到擔憂。
- 蘋果公司正在等待歐盟法院對要求為開發者提供免費 iOS API 的條款合憲性做出裁決，因為有聲稱其遵守歐盟法律只是表面功夫。

## [Cosmopolitan v3.5.0](https://github.com/jart/cosmopolitan/releases/tag/3.5.0)

- Cosmopolitan Libc v3.5.0 已經發布，使 C 語言成為一種無需解釋器或虛擬機器即可一次編譯、隨處運行的語言。
- 主要改進包括更高品質的記憶體映射實作、重新編寫的記憶體管理器，以及更新至 5.4.6 版本的 Lua。
- 引入了新的 C++ STL 功能，例如 ctl::string 小字串優化、ctl::set、ctl::map 和 ctl::unique_ptr，以及各種錯誤修正和優化。

### [反應](https://news.ycombinator.com/item?id=40773139)

- Cosmopolitan v3.5.0 已經發布，因其能夠創建高度可移植的可執行檔，無需修改即可在多個作業系統和架構上運行，從而引起了廣泛關注。
- 新版本擁有更佳的效能，據稱 Cosmopolitan Libc 在許多命令列介面(CLI)程式中速度是 Musl Libc 的兩倍，並提供強大的執行緒支援，使其適合高效能的 AI 軟體。
- “此版本包含對 WebAssembly (WASM) 的支援，並且具有如「實際可攜執行檔」(APE)標頭等功能，這增強了跨平台相容性，包括對 Android 的支援。”

## [LINQPad – .NET 程式設計師的遊樂場](https://www.linqpad.net/)

- LINQPad 是一個多功能工具，適用於 .NET 程式設計師，可以即時測試 C#/F#/VB 程式片段、查詢資料庫，並利用 REPL(讀取-求值-輸出循環)的優勢來自動化腳本。
- 它支援 C# 12 和 .NET 8，允許在 LINQ 或 SQL 中進行互動式資料庫查詢，並包含多個資料庫的驅動程式，如 SQL Server、Oracle 和 MySQL。
- LINQPad 提供了進階功能，如 AI 自動完成、整合除錯和豐富的輸出格式，使其成為快速編碼和動態開發的強大工具。

### [反應](https://news.ycombinator.com/item?id=40770171)

- LINQPad 因其易用性而備受推崇，特別是在 .NET 開發者中，用於編寫腳本、除錯和執行 LINQ 查詢。
- 用戶欣賞其如 SQL Server 資料庫連線等功能，但對其僅限於 Windows 平台及高昂成本感到不滿。
- 提到的替代方案包括 WinMerge、NetPad、dotnetfiddle.net 和 RoslynPad，有些使用者希望能夠跨平台支援並整合到 Visual Studio 中。

## [汽車經銷商在軟體供應商遭受網路攻擊後恢復使用筆和紙](https://apnews.com/article/car-dealerships-cyberattack-cdk-outage-3f7c81f6be0e212172b33cdc9f49feba)

- 由於汽車經銷商關鍵軟體供應商 CDK Global 遭受網路攻擊，北美的汽車經銷商經歷了重大干擾。
- 2024年6月19日的攻擊事件導致持續的停電，進而造成受影響的經銷商出現延誤並需進行人工處理。
- 主要汽車公司如 Stellantis、福特和 BMW 確認了影響，但正在使用替代方法繼續運營，而 CDK Global 正在努力恢復其系統。

### [反應](https://news.ycombinator.com/item?id=40776951)

- 在其軟體供應商 CDK Global 遭受網路攻擊後，汽車經銷商正在恢復使用筆和紙，因為這些攻擊中斷了他們的數位運營。
- “由於零件和服務部門的庫存資訊儲存在受損的CDK系統中，網路攻擊對這些部門的影響比新車銷售更為嚴重。”
- 這起事件突顯了高度依賴單一軟體供應商的產業的脆弱性，強調了需要更好的網路安全措施以及可能更多元化的軟體解決方案。

## [交通噪音傷害兒童的大腦](https://www.bbc.com/future/article/20240621-how-traffic-noise-pollution-harms-childrens-health-and-development)

- 兒童對交通噪音污染極為脆弱，這會對他們的健康和學習能力產生負面影響。
- 研究顯示，包括 Arline Bronzaft 在 1975 年的研究和 2022 年在巴塞隆納進行的研究，噪音污染會損害兒童的閱讀表現、工作記憶和注意力持續時間。
- “像布宜諾斯艾利斯和巴塞隆納這樣的城市正在實施改善綠地和降低車速限制等措施來對抗噪音污染，巴塞隆納的‘超級街區’等計畫優先考慮行人和綠地而非汽車。”

### [反應](https://news.ycombinator.com/item?id=40769880)

- 一項研究顯示，學校的道路交通噪音與兒童工作記憶和注意力的發展較慢有關。
- 潛在的解決方案包括減少學校周圍的交通量以及對教室進行隔音處理，以減輕環境噪音的影響。
- 討論還涉及更廣泛的議題，如汽車依賴、噪音污染，以及安靜、適合步行的城市環境的優勢。

## [CentOS Linux 7 將於星期日達到生命週期終止(EOL)](https://www.redhat.com/en/topics/linux/centos-linux-eol)

- 「CentOS Linux 7 將於 2024 年 6 月 30 日達到生命週期終止(EOL)，這意味著在此日期之後，它將不再接收更新或安全補丁。」
- 「Red Hat 提供多種遷移工具和服務，例如 Convert2RHEL、Red Hat Insights 和 Red Hat Consulting，以協助用戶轉換到 Red Hat Enterprise Linux (RHEL)。」
- 「RHEL 提供與 CentOS 類似的使用者體驗，並具備額外的功能、支援和安全性，且與主要的雲端服務提供商和混合雲部署相容。」

### [反應](https://news.ycombinator.com/item?id=40773749)

- CentOS Linux 7 將於星期日達到生命週期終止(EOL)，促使用戶尋找替代方案。
- SUSE 提供了一個解決方案，通過將 CentOS 7 的更新倉庫切換到 SUSE 的倉庫，避免了破壞性的遷移，而 Oracle Linux 是另一個選項，儘管一些使用者對此持謹慎態度。
- 許多使用者正在轉向 Rocky Linux、AlmaLinux 或 Debian 等替代方案，有些人在升級過程中面臨挑戰，並考慮使用容器來減少作業系統變更的影響。
