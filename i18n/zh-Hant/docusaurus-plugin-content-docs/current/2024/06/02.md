## [Signal 威脅歐盟因隱私妥協提案而退出](https://twitter.com/mer__edith/status/1796508893822238881)

- Signal宣佈強烈反對一項新提案，稱他們寧願離開歐盟市場，也不願妥協他們的隱私標準。
- 該公司認為，該提案將迫使他們在合規性和用戶隱私之間做出選擇，他們將其比作“安全瓶中的監控酒”。

### [反應](https://news.ycombinator.com/item?id=40551260)

- Signal計劃退出歐盟市場，以維護其隱私標準，以回應一項要求客戶端掃描加密內容的法規草案。

- 辯論強調了安全加密的挑戰、監控法的潛在濫用以及廣泛數據處理的道德影響，強調需要提高公眾意識並動員起來反對侵入性法律。

- 討論集中在 2000 年代早期的電腦硬體上，特別是像賽揚 300A 和 AMD Duron 這樣的超頻 CPU，以及提高性能所涉及的 DIY 精神。

- Haoran Xu 和 Fredrik Kjolstad 的論文“Copy-and-Patch Compilation”介紹了一種新的編譯技術，該技術使用稱為範本的預組裝代碼變體將高級語言和位元組碼快速編譯為二進位代碼。
- 該技術通過兩個編譯器進行演示，一個用於高級類 C 語言，另一個用於 WebAssembly，該技術顯示出可以忽略不計的編譯成本，並且在速度和效率方面明顯優於 LLVM 和 Google 的 Liftoff 等現有編譯器。
- 該方法在增強資料庫系統和Web瀏覽器的運行時編譯方面具有潛在的應用，表明編譯技術取得了重大進步。

### [反應](https://news.ycombinator.com/item?id=40553448)

- “複製和修補”技術是一種快速編譯方法，它通過複製目標代碼並使用運行時值對其進行修補來生成高效的代碼。
- 這種方法正在 Python 的新 JIT（Just-In-Time）編譯器和 LuaJIT 的 Deegen 工具包等專案中採用，這表明它越來越受歡迎。
- 該技術涉及使代碼可寫以進行轉換，然後可執行，遵守 W^X（寫 XOR 執行）等安全規則，並且是開發人員社區中積極討論和實現的主題。
