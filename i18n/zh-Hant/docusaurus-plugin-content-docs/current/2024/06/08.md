# 2024-06-08

## [阿波羅8號宇航員威廉·安德斯在聖胡安群島附近飛機失事身亡](https://www.fox13seattle.com/news/william-anders-wa-plane-crash)

- 2024 年 6 月 7 日，退役的阿波羅 8 號宇航員威廉·安德斯在華盛頓州聖胡安群島附近的一次飛機失事中喪生，當時他正在駕駛他的老式空軍 T-34 導師。
- 安德斯是歷史性的阿波羅8號任務的一部分，該任務拍攝了標誌性的“地球升起”照片，並在空軍預備役和通用動力公司擔任董事長兼首席執行官方面有著傑出的職業生涯。
- 他的妻子、六個孩子和13個孫子孫女倖存下來，他也是安德斯基金會和遺產飛行博物館的創始人。

### [反應](https://news.ycombinator.com/item?id=40614227)

- 阿波羅8號宇航員威廉·安德斯（William Anders）以標誌性的“地球升起”照片而聞名，他捲入了華盛頓的飛機失事，引發了對各種話題的討論。
- 關鍵點包括「地球升起」照片的文化影響，追求激情和安全之間的平衡，以及美國聯邦航空局的BasicMed計劃，該計劃允許年長的飛行員在較少的體檢下飛行。
- 辯論還涵蓋了公共場所的監管、對飛行等活動的年齡相關限制，以及飛行與駕駛的相對危險。

## [揭穿神話：PID 0 在 Unix 系統中的真正作用](https://blog.dave.tf/post/linux-pid0/)

- David Anderson 在 2024 年 6 月 5 日的博客文章中解決了對 Unix 進程 ID （PID） 和使用者 ID （UID） 的誤解，尤其是 PID 0 的角色。
- 與普遍的看法相反，PID 0 在用戶空間中是隱藏的，它負責內核初始化、調度和電源管理，而不是一些消息來源聲稱的虛擬記憶體管理。
- Anderson 提供了歷史背景，並詳細介紹了Linux內核的初始化過程，強調了進程和線程識別的複雜性以及內核和用戶空間 PID之間的區別。

### [反應](https://news.ycombinator.com/item?id=40613287)

- 討論探討了對在線評論的過度自信如何影響人工智慧語言模型，並強調了對自信陳述持懷疑態度的重要性。
- 它涵蓋了 PID 0 在操作系統中的作用、記憶體管理和進程調度等技術主題，強調了權威驗證的必要性。
- 其他主題包括編輯維琪百科的挑戰、用戶介面首選項（如深色模式）以及有關 POSIX kill 功能的技術細節。

- 討論重點介紹了 Xv6，一個來自麻省理工學院的類 Unix 操作系統，因為它在教授操作系統基礎知識方面的有效性。

- 該主題還提到了替代操作系統設計，如 SerenityOS、Redox，以及教育工具，如 PintOS、XINU 和 Ultibo for Raspberry Pi，強調了操作系統設計的演變和教育方面。

## [揭開互聯網“後室”傳奇的起源和影響](https://blog.archive.org/2024/06/01/the-backrooms-of-the-internet-archive/)

- 傑森·斯科特（Jason Scott）的文章「互聯網檔案館的後室」（The Backrooms of the Internet Archive）深入探討了“後室”圖像的起源和文化影響，這張令人毛骨悚然的照片成為互聯網傳奇。
- 這張圖片在 2019 年獲得了一個關於廣闊、令人不安的空間的背景故事，激發了專注於閾限空間的 subreddit、視頻和遊戲等創意內容。
- 這張照片的來源之謎最近被解開了：它是在2002年威斯康星州奧什科什的一家前傢俱店裝修期間拍攝的，互聯網檔案館的Wayback Machine在保存其歷史方面發揮了關鍵作用。

### [反應](https://news.ycombinator.com/item?id=40618079)

- Hacker News使用者正在爭論“The Backrooms”圖像的來源，該圖像最初被認為是張貼在留言板上的，可能來自互聯網檔案館。
- 發現過程包括對 4chan 檔案和元數據分析的廣泛搜索，導致 2011 年的 Twitter 帖子得到了 Wayback Machine 的確認。
- 討論突出了數位保存和在線內容的短暫性等更廣泛的問題。

- 討論還批評了首席執行官的激進反應，並強調了更廣泛的問題，例如法律糾紛的高成本以及社交媒體在應對企業威脅方面的作用。

## [南極研究站如何在極端條件下維持供水](https://brr.fyi/posts/south-pole-water-infrastructure)

- 2023年8月，作者檢查了南極研究站的水基礎設施，特別是麥克默多站和南極站。
- 麥克默多站使用絕緣管道和伴熱電纜來獲取和處理液態海水，而南極站則使用“羅德韋爾”系統來融化積雪並形成一個地下湖進行抽水。
- 南極站的供水系統是能源密集型的，利用附近發電廠的廢熱，包括緊急凈化系統和用於遠端工作的便攜式解決方案，突出了這些系統對人類在南極洲生存的重要性。

### [反應](https://news.ycombinator.com/item?id=40615530)

- Hacker News 討論的重點是維護南極水利基礎設施的複雜性，強調從零度以下加熱水的能源密集型過程。
- 關鍵點包括使用煤油基JP8噴氣燃料供電，麥克默多站從核能到柴油動力的歷史轉變，以及廢物管理對環境的影響。
- 評論者對資源有限的生活（例如兩分鐘淋浴）的實用性以及車站運營的歷史和技術細節很感興趣。

## [波音 737-800 勉強避免了由於已知的自動油門軟體故障而造成的災難](https://www.independent.co.uk/travel/tui-boeing-flight-bristol-disaster-avoided-b2558536.html)

- 人們對波音公司的安全實踐、成本削減措施和管理決策提出了更廣泛的擔憂，引發了關於自動化和手動控制之間責任平衡的辯論。

## [基於切比雪夫近似的嵌入式系統中的高效函數計算](https://www.EmbeddedRelated.com/showarticle/152.php)

- Jason Sachs 的博客文章深入探討了切比雪夫近似，這是一種在資源受限的嵌入式系統中高效計算函數的技術。
- 這篇文章將切比雪夫多項式與泰勒級數進行了對比，突出了它們的效率和數值穩定性，並提供了用於實際實現的 Python 代碼示例。
- 實際應用包括將切比雪夫多項式擬合到正弦等函數，並將其用於感測器數據估計，因此強烈推薦用於嵌入式系統。

### [反應](https://news.ycombinator.com/item?id=40614338)

- 切比雪夫近似是一種利用傅里葉變換和遞歸定義有效逼近函數的數學技術。
- 主要優點包括以 \（O（n \log n）\） 時間為計算係數，並在 \（O（n）\） 時間內執行乘法和微分等操作，從而提高 chebfun/chebpy 等工具的速度。
- 切比雪夫方法用途廣泛，應用於金融和機器視覺等領域，有時可以與卷積神經網路 （CNN） 的精度相媲美。
