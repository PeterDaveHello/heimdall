# 2024-02-13

## [Cloudflare 在審判中戰勝了專利流氓 Sable](https://blog.cloudflare.com/cloudflare-defeats-patent-troll-sable-at-trial)

- Cloudflare 在針對專利流氓 Sable IP 和 Sable Networks 的審判中取得了勝利。
- 陪審團得出結論，Cloudflare 沒有侵犯所主張的專利，並宣佈 Sable 的專利權利要求無效。
- 這一勝利可歸因於 Cloudflare 的法律團隊、外部法律顧問和 Jengo 項目參與者的努力，該專案獎勵個人提供證據，使專利流氓的主張無效。
- Cloudflare成功地使Sable的三項專利無效，這限制了他們向其他公司提起訴訟的能力。
- 此案凸顯了與專利訴訟相關的高成本和風險，並向專利流氓發出了警告，即Cloudflare不會輕易被嚇倒。

### [反應](https://news.ycombinator.com/item?id=39344956)

- 討論涵蓋了專利制度的多個方面，例如專利品質、專利流氓、軟體專利、智慧財產權稅、專利期限以及專利在創新中的作用。
- 改進建議包括激勵公司對無效專利進行辯護，增加對專利局的資助，懲罰惡意專利申請，以及解決公司創建空殼實體的問題。
- 關於專利的目的和有效性存在爭議，一些人主張取消專利，另一些人則強調專利在保護發明人和促進發展方面的作用。還討論了專利對製藥和技術等不同行業的影響。

## [AMD 資助 ROCm 上 AMD GPU 的開源 CUDA 實施](https://www.phoronix.com/review/radeon-cuda-zluda)

- AMD 為開發名為 ZLUDA 的開源嵌入式 CUDA 實現提供了資金支援，該實現使 NVIDIA CUDA 應用程式能夠在 AMD Radeon GPU 上運行，而無需修改原始程式碼。
- ZLUDA 最初是為 Intel 顯卡開發的，現已適應在 AMD GPU 上使用，但它並非完全故障安全，並且缺乏對 NVIDIA OptiX 的全面支援。
- ZLUDA 的開源版本允許識別實際的 Radeon 顯卡字串，而不是通用的“圖形設備”標籤。

### [反應](https://news.ycombinator.com/item?id=39344815)

- 討論圍繞著AMD試圖在GPU市場上與NVIDIA競爭，主要是在機器學習和軟體相容性方面。
- 人們對CUDA的壓倒性主導地位以及模仿它的困難表示擔憂，強調開放標準的必要性。
- 該對話還討論了 AMD 的軟體和驅動程式支持挑戰，並強調了操作系統在 AI 領域的重要性。人們普遍呼籲提高行業的可訪問性、協作和競爭。
