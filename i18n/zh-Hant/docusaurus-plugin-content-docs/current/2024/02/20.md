## [htmz：使用輕量級 HTML 微框架製作模組化 Web UI](https://leanrada.com/htmz/)

- htmz 是一個極簡的 HTML 微框架，使用戶能夠使用純 HTML 構建模組化 Web 介面，無需後端依賴。
- 受 htmx 和其他 Web 架構的啟發，htmz 可以使用基本的 HTML 無縫交換頁面部分，僅更新鏈接點擊的特定部分。
- 它不是一個傳統的 JavaScript 框架，而是一個緊湊的代碼片段，利用瀏覽器的固有功能來檢索和解釋 HTML，為增強功能提供可擴充性。

### [反應](https://news.ycombinator.com/item?id=39429370)

- 討論深入探討了各種 Web 開發工具（如 htmz、htmx、Vue.js）和 hack（如 htmy），重點介紹了它們的優勢、局限性和潛在應用。
- 使用者就平衡功能和配置、iframe 的使用、HTML 標準、JavaScript 依賴性以及網站在沒有 JavaScript 的情況下優雅降級以實現可訪問性的重要性進行辯論。
- 一些使用者欽佩這些工具的簡單性和有效性，而另一些使用者則對它們的實用性、對Web標準的合規性、安全性、可訪問性以及大型JavaScript捆綁包對用戶體驗的影響提出了質疑。
