# 2024-02-18

## [雨果獎因對中國的擔憂而將科幻作家排除在外](https://www.nbcnews.com/news/world/science-fiction-authors-excluded-hugo-awards-china-rcna139134)

- 去年，包括尼爾·蓋曼（Neil Gaiman）和保羅·韋默（Paul Weimer）在內的多位科幻作家被排除在雨果獎之外，原因是擔心冒犯中國以及有關中國、臺灣和西藏的敏感政治內容。
- 洩露的電子郵件揭露了排除背後的原因，引發了圍繞獎項管理透明度和信任的爭議和批評。
- 正在採取措施解決雨果獎組織的強烈反對和提高透明度。

### [反應](https://news.ycombinator.com/item?id=39415234)

- 在中國舉辦雨果獎引發了科幻作家的不滿，因為他們擔心政府的干預和審查制度。
- 這一決定引發了關於自我審查、中國科幻文學品質以及威權政權對文化活動的影響的辯論。
- 討論還集中在利用技術來對抗威權主義、孤立主義和全球依賴性，強調地緣政治、審查制度和文學認可之間的錯綜複雜的聯繫。

## [Ute Tribe 將在科羅拉多州建造大型太陽能發電場](https://www.ksut.org/news/2024-02-15/ute-mountain-ute-tribe-will-construct-one-of-the-largest-solar-farms-in-the-u-s)

- Ute Mountain Ute Tribe 計劃在科羅拉多州建造 Sun Bear 太陽能發電場，該發電場將成為美國最大的太陽能發電場之一，擁有 220 萬塊太陽能電池板。
- 該太陽能發電場將產生756兆瓦的電力，預計將為當地創造500多個就業機會，耗資超過10億美元，旨在連接到西部地區電力局的電力線。
- 主席曼努埃爾·哈特（Manuel Heart）對部落向可再生能源的轉變及其將提供的潛在經濟效益充滿熱情。

### [反應](https://news.ycombinator.com/item?id=39409653)

- 討論包括尤特部落的太陽能農場專案、加州的輸電成本、太陽能電池板的可持續性以及關於核電可行性和成本效益的辯論。
- 它還涵蓋了美洲原住民部落的自治權、經濟挑戰、能源生產對環境的影響以及比特幣挖礦在能源消耗中的作用。
- 此外，它還解決了向100%可再生能源的過渡、乙太坊從PoW到PoS的轉變，並比較了太陽能、核能和煤炭能源。

- 探索 btree 索引中的隨機分佈對快取位置和頁面效率的影響，以及對優化資料庫性能和企業軟體中 ORM 工具約束的見解。

## [我們必須重新開始：重寫代碼以優化性能](https://zed.dev/blog/we-have-to-start-over)

- 該團隊強調高效的核心元件構建，同時保持對非關鍵方面的適應性，強調了代碼重寫、從以前的反覆運算中學習和提高性能的好處和障礙。

- 討論強調了將可訪問性集成到 Zed.dev 的 UI 框架中的挑戰，並強調了可訪問性在協作工具中的重要性。
- 指出工程中對可訪問性的關注不足，遠端工作流工具的出現以及文本編輯器的比較，同時解決了提高代碼編輯效率的必要性。
- 使用者與不同的編輯者交流經驗，討論功能優勢，並探索潛在的未來進步，如 AST 操作和 SIMD，強調代碼編輯工具中的用戶體驗、性能、靈活性和跨平臺相容性等關鍵方面。

## [蘋果從歐盟刪除網路應用程式引發反壟斷問題](https://open-web-advocacy.org/blog/its-official-apple-kills-web-apps-in-the-eu/)

- 蘋果正在從歐盟刪除 Web 應用程式 （PWA），以遵守《數位市場法》，並表示它不會嚴重影響使用者。
- 這一決定被認為是為了限制競爭並維持蘋果在App Store的主導地位，無視蘋果的安全性和低Web App使用率的理由。
- 批評人士認為，蘋果正在逃避監管，導致人們對其動機的擔憂，可能會危及公平競爭和iOS上網路應用程式的存在。

### [反應](https://news.ycombinator.com/item?id=39408196)

- 蘋果已決定停止在歐盟支援網路應用程式，這引發了人們對保護其App Store收入的猜測。
- 對話探討了應用分發的變化、變現趨勢以及Apple法規對開發者和用戶的影響。
- 關於 Web 應用程式與原生應用程式的有效性、iOS 設備的工作限制以及對 Apple 對 Web 功能的權力的擔憂存在爭議，重點是 PWA 和原生應用程式之間的競爭。
