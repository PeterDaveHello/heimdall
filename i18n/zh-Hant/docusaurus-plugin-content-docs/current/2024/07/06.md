# 2024-07-06

## [「道德經，烏蘇拉·勒瑰恩譯(1997)」](https://github.com/nrrb/tao-te-ching/blob/master/Ursula%20K%20Le%20Guin.md)

- “老子所著的《道德經》，由厄休拉·K·勒瑰恩翻譯，探討了關於存在本質、領導力和智慧的深刻哲學概念。”
- 「主要主題包括簡單、謙遜和無為而為的重要性，強調真正的力量和智慧來自於與自然之道(道)相契合。」
- 這段文字分為81章，每章都提供了如何和諧生活、有效領導以及通過悖論和隱喻理解生活深層真理的見解。

### [評論](https://news.ycombinator.com/item?id=40886419)

- 「厄休拉·勒瑰恩的《道德經》版本並非直接翻譯，而是基於現有的翻譯和音譯進行的詮釋，因為她不懂中文。」
- 文言文的內在模糊性使其具有多重解釋的可能性，因此在任何單一翻譯中都難以完全捕捉其全部意義。
- 討論強調了理解《道德經》哲學本質的重要性，以及在沒有原文語言知識的情況下翻譯此類文本的局限性。

## [Radio Garden](https://radio.garden/)

- Radio Garden 讓使用者透過旋轉虛擬地球來探索全球的直播電台，提供了一種獨特且互動的體驗。
- 該平台具有可自訂的設定，包括字體大小、顏色和主題，並支援各種螢幕尺寸和裝置，以提供最佳化的使用者體驗。
- 它確保與舊版瀏覽器的相容性，並包含使用者友好的設計元素，如易於使用的播放按鈕和載入動畫。

### [評論](https://news.ycombinator.com/item?id=40887359)

- 「Radio Garden 讓使用者能夠收聽全球的電台，但使用者創建的 WinAmp 播放清單遇到了許多串流無法連線的問題。」
- Radio-browser.info 提供了一個包含約 50,000 個電台的 JSON 檔案，作為尋求公共串流完整列表的替代方案。
- 「用戶分享了其他替代方案，如 WebSDR 和 radiooooo.com，不過英國用戶可能會因為版權問題而遇到限制。」

## [DuckDB 社群擴充套件](https://duckdb.org/2024/07/05/community-extensions.html)

- “DuckDB 推出了社群擴充套件儲存庫，簡化了擴充套件的安裝和維護，減輕了開發者的負擔。”
- 自從 0.3.2 版本以來，擴充功能可以透過簡單的指令安裝和載入，支援各種作業系統和處理器架構，每週約有六百萬次下載。
- 這個儲存庫允許開發者發佈擴充功能，而不需要處理編譯和分發的複雜性，提升了開發者的體驗並鼓勵社群貢獻。

### [評論](https://news.ycombinator.com/item?id=40885238)

- “DuckDB 社群擴充功能已引入對 WebAssembly (WASM) 的支援，提升其在網頁環境中的功能和效能。”
- 「shellfs 擴充功能允許使用 shell 命令進行輸入/輸出，提升了 DuckDB 作為命令列工具的實用性。」
- 「儘管這些下載是通過簽名檢查進行驗證的，但由於 DuckDB 下載擴展二進位檔案，仍然存在供應鏈安全的擔憂。」

## [複製和貼上右鍵選單項目有時會在不應該被禁用的情況下被禁用](https://bugzilla.mozilla.org/show_bug.cgi?id=1863246)

- 提出的修正方案正在進行中，預計未來的版本將解決這個問題，這突顯了在維持瀏覽器功能方面持續面臨的挑戰。

## [正確測試並發資料結構](https://matklad.github.io/2024/07/05/properly-testing-concurrent-data-structures.html)

- 這篇文章介紹了一個名為 loom 的 Rust 函式庫，該函式庫專為測試無鎖資料結構而設計，並通過一個簡單的範例來探討其概念。
- 它強調了測試並發資料結構的挑戰，展示了一個損壞的並發計數器和各種測試方法，包括基於屬性測試(PBT)和管理執行緒。
- 這篇文章詳細解釋了如何建立一個受控的測試環境來進行並發操作，為從事 Rust 多執行緒和原子操作的開發者提供了寶貴的見解。

### [評論](https://news.ycombinator.com/item?id=40890035)

- 在 Rust 中測試並發資料結構涉及詳細建模記憶體模型和追蹤執行緒寫入，像是 Temper 和 Loom 這樣的函式庫可以幫助這個過程。
- Shuttle 提供隨機化測試方法，為錯誤檢測提供概率保證，而其他工具如 JetBrains 的 Lincheck 和 Relacy Race Detector 分別用於 Kotlin/Java 和 C++。
- 討論強調了可重現測試的重要性以及在並行程式設計中確保執行緒安全和同步的挑戰。
