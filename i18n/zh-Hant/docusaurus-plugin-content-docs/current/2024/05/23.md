## [洩露的檔揭露了 OpenAI 對前雇員的侵略性策略](https://www.vox.com/future-perfect/351132/openai-vested-equity-nda-sam-altman-documents-employees)

- 洩露的文件顯示，由首席執行官山姆·阿爾特曼（Sam Altman）領導的OpenAI使用激進的策略來阻止前員工批評公司，威脅要撤銷他們的既得股權，除非他們簽署限制性退出協定。
- 儘管阿爾特曼公開道歉並聲稱自己無知，但帶有他和其他高管簽名的內部檔表明，他們知道這些規定，這引發了人們對OpenAI透明度和可信度的擔憂。
- OpenAI現在正在更新其離職流程，以確保前雇員的既得股權並刪除非貶低條款，儘管仍有更多工作需要解決。

### [反應](https://news.ycombinator.com/item?id=40447431)

- 洩露的文件顯示，OpenAI 使用激進的策略通過威脅前員工的既得股權來執行非貶低協議，導致 OpenAI 的強烈反對和隨後的保證。
- 這種情況引發了關於合同談判、股權補償和競業禁止條款的討論，凸顯了可執行性的複雜性和地區差異。
- 批評者認為，OpenAI的行為揭示了更深層次的信任和透明度問題，對行業趨勢、道德問題以及山姆·奧特曼（Sam Altman）等知名人物的影響產生了更廣泛的影響。

## [記錄證實，OpenAI 沒有為 ChatGPT 克隆 Scarlett Johansson 的聲音](https://www.washingtonpost.com/technology/2024/05/22/openai-scarlett-johansson-chatgpt-ai-voice/)

- 斯嘉麗·詹森（Scarlett Johansson）聲稱，在她拒絕了首席執行官山姆·奧特曼（Sam Altman）的許可請求后，OpenAI在未經她同意的情況下複製了她的聲音。

### [反應](https://news.ycombinator.com/item?id=40448045)

- OpenAI 因涉嫌在未經 ChatGPT 同意的情況下使用 Scarlett Johansson 的配音演員而受到審查，這引發了法律和道德問題。
- 這場辯論凸顯了聲音模仿法的複雜性，並引用了米德勒訴米德勒案等案例。Ford 和 Tom Waits 訴 Frito-Lay，並強調了更新法律框架的必要性。
- 這場爭議引起了人們對人工智慧、配音演員的更廣泛影響的關注，以及在未經明確同意的情況下使用聽起來相似的聲音的道德考慮。

## [WinDirStat：適用於 Windows 的開源磁碟使用方式查看器和清理工具](https://windirstat.net/)

- WinDirStat是一個開源工具，用於查看和清理Microsoft Windows上的磁碟使用方式，提供目錄清單，樹狀圖和擴展清單。
- Linux 的替代品包括 KDirStat 和 QDirStat，而 MacOS X 使用者可以使用 Disk Inventory X 或 GrandPerspective。
- 該軟體根據 GPLv2 獲得許可，確保其保持免費和開源。

### [反應](https://news.ycombinator.com/item?id=40449767)

- 討論比較了各種磁碟使用方式分析工具，重點關注不同操作系統的功能、性能和可用性。
- WinDirStat 很受歡迎且開源，但因速度慢而受到批評，而WizTree速度更快，但存在安全問題並且不是開源的。
- 用戶強調了速度、安全性和可用性之間的權衡，並表示需要更好的檔管理功能和改進的可視化方法，如樹狀圖和旭日圖。

## [Microsoft 中斷中斷了 Bing、Copilot、ChatGPT 和 DuckDuckGo 服務](https://www.bleepingcomputer.com/news/microsoft/microsoft-outage-affects-bing-copilot-duckduckgo-and-chatgpt-internet-search/)

## [Sierra 的 Space Quest II 軟盤錯誤暴露了 AGI 原始程式碼](https://lanceewing.github.io/blog/sierra/agi/sq2/2024/05/22/do-you-own-this-space-quest-2-disk.html)

- 2024年，由於格式不正確，發現 Sierra On-Line 的 Space Quest II 的某些版本在其 720KB 軟盤上殘留了 C 原始程式碼。
- 這個錯誤暴露了 Sierra 大約 70% 的 AGI 解釋器原始程式碼，包括 75 個 C 源檔和 16 個彙編檔，直到 2016 年才被注意到。
- 原始碼現在可以在 GitHub 上找到，它提供了對 Sierra 開發過程的寶貴見解，並為 AGILE 基於 Web 的解釋器等專案提供了説明。

### [反應](https://news.ycombinator.com/item?id=40455758)

- 討論強調了軟體分發中的歷史錯誤，例如在最終產品版本中包含敏感或意外數據，例如1989年的DOS遊戲“雙龍II：復仇”。
- 它研究了軟盤的反盜版措施、消費級磁碟複印機的局限性，以及用於讀取磁性級數據的 Greaseweazle 等現代工具。
- 參與者分享了對舊遊戲的懷舊體驗，討論了原始程式碼洩露的影響，並強調了部門級映像對存檔的重要性。

## [湯姆·韋茨（Tom Waits）贏得了對Frito-Lay的260萬美元訴訟，理由是聲音盜用](http://tomwaitslibrary.info/biography/copyright/frito-lay/)

- 1988 年，Tom Waits 起訴 Frito-Lay Inc. 在 Doritos 廣告中使用了對他的歌曲“Step Right Up”的類似聲音的模仿，聲稱虛假代言和聲音盜用。
- 陪審團判給 Waits 260 萬美元，上訴維持原判，為《蘭哈姆法案》和加州法律下的聲音盜用確立了法律先例。
- Waits批評那些允許他們的音樂出現在廣告中的藝術家，認為這破壞了藝術的完整性，他的案件凸顯了保護藝術聲譽的重要性。

### [反應](https://news.ycombinator.com/item?id=40452655)

- 討論探討了使用聲音相似的配音演員的法律和道德問題，參考了 Tom Waits 對 Frito-Lay 的訴訟和 Scarlett Johansson 與 OpenAI 的爭議。
- 它研究了聲音模仿的複雜性、“出賣”的概念，以及對藝術家許可其作品的不斷變化的看法，包括未經同意使用類似於約翰遜的聲音的法律影響。
- 該對話批評了企業的做法，尤其是 OpenAI 的做法，並解決了對人工智慧生成的聲音影響智慧財產權和藝術完整性的廣泛擔憂。

## [新的圖靈機突破性計算阿克曼級函數](https://www.sligocki.com//2024/05/22/bb-3-4-a14.html)

- 帕維爾·克羅皮茨（Pavel Kropitz）發現了一個3態、4個符號的圖靈機（TM），它能夠計算阿克曼級函數，在磁帶上精確地停止\（（2 \uparrow^{15} 5） + 14 \）個非零符號。
- 這個TM於2024年4月25日被發現，是第一個已知的類比如此複雜功能的TM，其行為需要雙重感應來證明。
- 這一發現還充當了「歸納證明」驗證器的測試用例，旨在標準化和驗證此類證明。

### [反應](https://news.ycombinator.com/item?id=40453221)

- 本文深入探討了圖靈機的複雜性，重點介紹了忙碌的海狸問題和一個新的圖靈機程式BB（3,4），它在複雜度上超過了阿克曼函數。
- 它討論了程式的模組化結構、長時間運行的圖靈機的特性以及柯爾莫哥羅夫複雜性的影響，強調了分析這些機器的挑戰。
- 對話還批評了同行評審制度、數位資訊源的可靠性，並強調了科學研究中正確存檔和引用實踐的重要性。

## [Computershare 在價格大幅上漲的情況下拋棄 VMware 轉而使用 Nutanix AHV](https://www.theregister.com/2024/05/22/computershare_vm_migration_project/)

- 全球股票市場股票註冊運營商 Computershare 正在放棄 VMware，原因是 Broadcom 收購 VMware 後價格大幅上漲。
- VMware 虛擬機管理程式的成本增加了 10 到 15 倍，促使 Computershare 改用 Nutanix AHV，預計一年內將完成 24,000 個虛擬機的遷移。
- 此舉凸顯了博通的潛在問題，因為其他大客戶也可能因其高收入、高利潤的定價策略而尋求替代方案。

### [反應](https://news.ycombinator.com/item?id=40452260)

- 由於價格大幅上漲，現在由博通（Broadcom）擁有的VMware失去了一個擁有24,000台虛擬機的主要客戶，這反映了專注於高收入、高利潤業務的戰略。
- 批評者認為，這種方法可能會將客戶推向 Nutanix 或開源解決方案等競爭對手，這凸顯了遷移大型 IT 系統的挑戰以及維護客戶信任的重要性。
- 該討論還強調了供應商鎖定的風險，並建議採用混合或多雲方法來減少對單個虛擬機管理程式供應商的依賴。
