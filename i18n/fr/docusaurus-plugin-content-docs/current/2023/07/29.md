## [Cap'n Proto 1.0](https://capnproto.org/news/2023-07-28-capnproto-1.0.html)

- Cap'n Proto, un protocole de messagerie, a publié la version 1.0 de son implémentation de référence C++.
- Le protocole est utilisé en production depuis de nombreuses années, ce qui témoigne de sa fiabilité.
- Cette version comprend des optimisations pour améliorer les performances.
- Des modifications ont été apportées pour permettre l'annulation de RPC.
- La version 2.0 prévoit des révisions de l'API C++ et de la bibliothèque de la boîte à outils KJ C++ afin d'améliorer l'expérience des développeurs.
- Ces changements nécessiteront un compilateur prenant en charge le C++20 et pourraient rendre KJ plus compatible avec les coroutines.
- Le format de sérialisation et le protocole RPC ne feront l'objet d'aucune modification rétroactive.
- Une version 2.0 officielle n'est pas attendue avant quelques années.

## [Le mode No-GIL arrive pour Python](https://lwn.net/Articles/939568/)

- Le Python Steering Council étudie une proposition visant à rendre le Global Interpreter Lock (GIL) optionnel dans CPython.
- Cela permettrait un mode sans GIL en Python, améliorant potentiellement la concurrence.
- Le Conseil prend des précautions pour assurer la compatibilité ascendante et éviter un scénario de scission entre Python 2 et 3.
- Ils discutent des exigences en matière de compatibilité ABI et de l'impact sur la compatibilité ascendante.
- Certains commentateurs se sont inquiétés des problèmes de compatibilité potentiels et ont recommandé d'autres approches pour améliorer la concurrence dans Python.

#### [Réactions](https://news.ycombinator.com/item?id=36914401)

- Python introduit un mode No-GIL (Global Interpreter Lock), comme le confirme un post sur lwn.net.
- Le verrouillage global de l'interpréteur est un mécanisme de Python qui permet à un seul thread d'exécuter le bytecode Python à la fois.
- Le mode No-GIL vise à supprimer cette limitation et à permettre à plusieurs threads d'exécuter simultanément le bytecode Python.
- Les informations sur le mode No-GIL ont été initialement partagées sur ycombinator.com.
