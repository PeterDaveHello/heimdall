- Este subliniată importanța amplasării corecte a pietrelor, a proiectării pereților și a stabilității structurilor de piatră pentru integritatea structurală.
- Este discutată utilizarea gabioanelor ca alternativă la zidurile de piatră uscată pentru reținerea apei în grădini.

## [Mail-in-a-Box: Configurați-vă cu ușurință propriul server de e-mail securizat](https://github.com/mail-in-a-box/mailinabox)

- Mail-in-a-Box este un proiect open-source care permite persoanelor fizice să își configureze cu ușurință și în siguranță propriul server de e-mail pe o mașină Ubuntu.
- Promovează descentralizarea, inovarea și confidențialitatea pe web, oferind în același timp o configurare automată și verificabilă.
- Caracteristicile includ webmail, filtrare spam, gestionare DNS, backup-uri, firewall și protecție împotriva intruziunilor. A câștigat popularitate în comunitatea de hackeri și are un forum de asistență disponibil.

### [Reacții](https://news.ycombinator.com/item?id=38406478)

- Mail-in-a-Box este un software server de e-mail popular pentru găzduirea e-mailurilor personale, cu probleme minime raportate în ceea ce privește livrarea e-mailurilor.
- Unii utilizatori au întâmpinat dificultăți cu anumiți furnizori, cum ar fi Microsoft Outlook.
- Sunt sugerate opțiuni alternative pentru găzduirea serverelor SMTP și sunt discutate avantajele găzduirii proprii a e-mailului, inclusiv confidențialitatea și securitatea.
