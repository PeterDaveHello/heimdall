## [„Google Chrome are o API specială ascunsă, accesibilă doar de pe \*.google.com”](https://twitter.com/lcasdev/status/1810696257137959018)

- „Google Chrome oferă tuturor site-urilor \*.google.com acces extins la utilizarea CPU, GPU și a memoriei sistemului și a filelor, precum și informații detaliate despre procesor și un canal de înregistrare în fundal.”
- „Acest acces API este exclusiv pentru site-urile \*.google.com, ridicând potențiale probleme de confidențialitate și securitate în rândul utilizatorilor și dezvoltatorilor.”

### [Reacții](https://news.ycombinator.com/item?id=40918052)

- „Google Chrome are o API ascunsă accesibilă doar de pe \*.google.com, probabil folosită pentru Google Meet, care oferă informații detaliate despre sistem, cum ar fi utilizarea CPU/GPU/RAM.”
- „Această exclusivitate ridică îngrijorări cu privire la comportamentul anticoncurențial și la confidențialitatea utilizatorilor, deoarece alte site-uri web nu pot accesa aceste informații.”
- „API-ul este integrat în Chrome și nu este listat în chrome://extensions, stârnind dezbateri despre etica și legalitatea unor astfel de practici.”

## [„Concepția greșită despre zombi în știința calculatoarelor teoretice”](https://scottaaronson.blog/?p=8106)

- „Postarea pe blog a lui Scott Aaronson clarifică concepțiile greșite comune în știința calculatoarelor teoretică, în special diferența dintre calculabilitate și complexitate.”
- „El folosește exemple, cum ar fi o funcție care determină existența lui Dumnezeu, pentru a ilustra că calculabilitatea se referă la funcții, nu la întrebări individuale, și abordează neînțelegerile legate de problema P vs. NP.”
- „Aaronson explică faptul că calculabilitatea se referă la existența unui program care să mapeze intrările la ieșiri, nu la dificultatea de a crea acel program, și discută funcția Busy Beaver pentru a evidenția valorile necalculabile versus valorile calculabile.”

### [Reacții](https://news.ycombinator.com/item?id=40912684)

- „Știința teoretică a calculatoarelor se ocupă adesea cu concepte care implică infinitul, făcându-le neintuitive, cum ar fi complexitatea Kolmogorov și problema opririi.”
- „Problema P=NP rămâne un subiect de dezbatere filosofică, unii comparând complexitatea acesteia cu ipoteza continuumului (CH).”
- „Distincția dintre computabilitate și demonstrabilitate este subliniată, arătând că o funcție poate fi computabilă chiar dacă implementarea sa exactă este în prezent nedeterminată.”
