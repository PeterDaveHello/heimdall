## 2023 年 6 月 10 日

### [Reddit CEO、ドラマ仕立ての AMA で Apollo 開発者への攻撃を倍増させる](https://techcrunch.com/2023/06/09/reddit-ceo-doubles-down-on-attack-on-apollo-developer-in-drama-filled-ama/)

- Reddit の CEO である Steve Huffman は、API 価格の改定という同社の不評な決定を擁護し続け、Apollo を含むサードパーティアプリの閉鎖を招きました。- ハフマンは、Apollo の開発者である Christian Selig が、新しい API 価格によって Apollo アプリの運用が不可能になると Selig が強調したにもかかわらず、非効率的に運用し、良い「API」ユーザーでないと非難しました。- コミュニティの反発や数千の subreddit からの抗議は、ハフマンや Reddit に API 変更による影響を懸念させるものではなく、同社は決定を延期したり再考したりするつもりはない。

#### [業界の反応](http://news.ycombinator.com/item?id=36264232)

- Reddit の CEO は Apollo 開発者をターゲットにしたドラマ仕立ての AMA で反発にあい、Reddit の立場が悪くなった。- Reddit は、ユーザー数が少ないにもかかわらず、IPO による収益計画を達成しようとしており、完全な透明性を確立する必要がある。- Reddit のユーザーとモデレーターは、利益を重視する同社に憤慨し、利益だけでなくコミュニティを重視する「Reddit のライバル」を求めています。

### [ArchiveTeam は 11.2B 以上の Reddit のリンクを保存しています](https://old.reddit.com/r/DataHoarder/comments/142l1i0/archiveteam_has_saved_over_108_billion_reddit/)

- ArchiveTeam は 108 億以上の Reddit リンクを保存しており、Reddit の API コストシステム変更によりサブレディットが消滅する前にアーカイブしようとしています。- 同グループはすでに 108 億 1000 万件のリンクをアーカイブしており、Windows、MacOS、Linux のデバイスを持つ人々が ArchiveTeam Warrior を使って投稿やコメントをアーカイブするための支援を必要としています。- この取り組みは、API コストの変更が解決されない場合に備えて、閉鎖された subreddit の投稿にアクセスできる状態を維持することを目的としています。

#### [業界の反応](http://news.ycombinator.com/item?id=36254172)

- ArchiveTeam は、最近の Reddit API コスト変更に抗議して非公開になる可能性のあるサブレディットの古い投稿を保存するため、11.2B 以上の Reddit リンクを保存しました。- Reddit は最近、ボランティアモデレーターのサードパーティアプリを公式アプリに置き換えることで、物議を醸しています。- ArchiveTeam は Reddit のダウンロードを完了し、サイトの知識と会話を保存しています。

### [2023 年 6 月のデータ・ダンプが欠落しています](https://meta.stackexchange.com/a/390023/6212)

- Stack Overflow は、通常 3 ヶ月ごとに更新される 2023 年 6 月のデータダンプを公開しておらず、上級幹部は Archive.org へのアップロードを無効にしています。- これは、大規模言語モデル(LLM)を構築する企業によるデータの悪用を防ぐための戦略を練っているためで、ガードレールを設置できるまでデータダンプを一時的に停止しています。- Stack Overflow は、データを利用する企業に補償を求めており、AI 大手にはトレーニングデータを請求する予定であり、Stack Exchange ネットワークでユーザーが投稿した既発表コンテンツは CC-BY-SA でライセンスされています。

#### [業界の反応](http://news.ycombinator.com/item?id=36257523)

- Stack Exchange の 2023 年 6 月のデータダンプの行方不明は、中央集権的なプラットフォームやストレージに依存することへの懸念を示しています。- 分散型コミュニケーションや、データをフォークして新しいコミュニティを作ることが代替案として議論されているが、ライセンスや帰属の要件などの課題がある。- 機械学習モデルの API アクセスを制限するという Stack Overflow の決定は、AI の進歩を妨げる恐れがあると懸念するユーザーの間で論争を引き起こし、一方でコミュニティを保護する会社の権利を擁護する人もいます。

### [手遅れになる前に Reddit のデータをアーカイブする](https://xavd.id/blog/post/archive-your-reddit-data/)

- Reddit の API に対する今後の変更により、一部のサブレディットが無期限に暗転し、サードパーティのクライアントに影響を与える可能性があります。- ナレッジストアとしての Reddit の非金銭的価値は貴重であり、ユーザーは reddit-user-to-sqlite という Python ツールを使って個人データをアーカイブすることができます。- このツールにより、ユーザーは構造化された検索可能なデータを 1 つのファイルに保存し、Reddit の検索機能に頼ることなく、自分のコメントや投稿の全文検索に使用することができます。

#### [業界の反応](http://news.ycombinator.com/item?id=36259930)

- ディスカッションサービス利用者が、エコーチェンバーやスパムへの懸念とともに、Reddit のアカウントやカルマの価値について議論しています。- 投稿では、Hacker News や Reddit のようなウェブサイトにおけるアップボートやダウンボートの目的や、ユーザーが投稿履歴を削除することによる将来の読者への影響について議論されています。- ユーザーは、Reddit の投稿やコメントを削除するためのテクニックやツールを共有していますが、貴重な情報を失うことや、永久に検索可能な知識のリポジトリがないことへの懸念を表明する人もいます。

### [3M、「実存的」な 143 億ドルの永久化学物質訴訟で裁判に臨む](https://www.bloomberg.com/news/articles/2023-06-02/3m-heads-to-trial-in-existential-143-billion-pfas-litigation)

- 3M は、4,000 件の訴訟のうち、自治体の水源を汚染した、癌に関連するとされる「永久化学物質」の浄化に対する推定責任額 1,430 億ドルの最初の裁判に直面しています。- パーおよびポリフルオロアルキル物質(PFAS)と呼ばれるこの化学物質は、何十年も日常製品に使用されているが、自然に分解されることはなく、土壌、水、人体に蓄積されることがある。- 6 月 5 日から連邦裁判所で始まるこの裁判は、同社の将来に大きな影響を与える可能性があり、PFAS 汚染に対する法的・社会的懸念が高まっていることを浮き彫りにしています。

#### [業界の反応](http://news.ycombinator.com/item?id=36259163)

- 3M は、人体に有害であることが知られているにもかかわらず、まだ合法的に使用されていた有毒な PFAS 化学物質を中心に、143 億ドルの訴訟に直面しています。- PFAS は日常製品に使用されており、この裁判は、特に公衆衛生に関して、化学物質の使用と透明性を規制することの重要性を強調しています。- この裁判は、地球上の生命の存続を脅かすものとして「存立危機事態」とみなされており、解決策として、会社の解体や国有化が提案されています。
