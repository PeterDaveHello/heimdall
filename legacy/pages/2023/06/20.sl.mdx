#### [Odzivi industrije](http://news.ycombinator.com/item?id=36388821)- Slint je deklarativni nabor orodij za grafični vmesnik, napisan predvsem v jeziku Rust, ki je zasnovan za namizno in vgrajeno uporabo.

- Orodje ponuja podporo API za več programskih jezikov, vključno s C++ in JavaScriptom.
- Uporabniki so pohvalili Slintovo zasnovo, enostavnost uporabe in povratne informacije o grafičnem vmesniku v realnem času pri uporabi razširitve VS Code. Nekateri ga primerjajo z Microsoftovim orodjem Windows Presentation Foundation.
- En uporabnik je izrazil skrb glede optimizacije za velikost binarnih datotek na mikrokrmilniku z omejenim pomnilnikom flash.
- Ekipa Slint je potrdila skrb in omenila načrte za optimizacijo ustvarjene kode za nadaljnje zmanjšanje velikosti binarne kode.
- Drug uporabnik je izrazil zanimanje za knjižnico grafičnega vmesnika, namenjeno majhnim vgrajenim sistemom, ki ne porabijo veliko pomnilnika flash ali RAM.
- Ekipa Slint je odgovorila, da je to njihov cilj z orodjem.
- Uporabniki so priporočili alternativne knjižnice grafičnega vmesnika, kot sta fltk-rs in imgui, ki bi lahko bolje ustrezale posebnim potrebam.
- Uporabniki so razpravljali o optimizaciji velikosti in zmogljivosti kode za mikrokrmilnike s predlogi, kot so prilagoditev nastavitev codegen v programu Rust in eksperimentiranje s pragovi za inliniranje.
- Uporabniki so se pogovarjali o GRPC, Rustu kot zaledju za uporabniški vmesnik in integraciji Flutterja z Rustom.
- Uporabniki so izrazili zanimanje za boljše primerjalne razdelke na spletnem mestu Slint in predlagali druga ogrodja za uporabniški vmesnik, o katerih bi bilo treba razmisliti.
- Uporabniki so razpravljali o strukturi cen ogrodja Slint in zahtevali večjo jasnost informacij o cenah.
- Uporabniki so predlagali vključitev vodnika za uvajanje v delo in izboljšanje primerov galerije gradnikov na spletnem mestu Slint.
- Uporabniki so postavili vprašanja o ravnanju z besedilom, podpori za dostopnost, integraciji nativnih gradnikov in načrtih za API Python.
- Uporabniki so se strinjali z imenom Slint in njegovo povezavo s skupino.
