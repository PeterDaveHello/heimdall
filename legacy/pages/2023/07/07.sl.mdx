#### Odpustili so skoraj vse preostalo osebje Evernote

- Evernote, priljubljena aplikacija za beleženje zapiskov, je odpustila velik del osebja.
- Podjetje se je soočalo s finančnimi težavami in s težavo konkuriralo drugim aplikacijam za beleženje zapiskov.
- Ta novica je pomembna, ker kaže, da je Evernote morda v težavah in da bodo morali uporabniki aplikacije razmisliti o drugih možnostih.

#### [Odzivi industrije](http://news.ycombinator.com/item?id=36609641)

- Evernote je po prevzemu s strani podjetja Bending Spoons odpustil precejšen del preostalega osebja, kar vzbuja skrb glede sposobnosti podjetja, da financira nove funkcije.
- Uporabniki izražajo razočaranje nad tem, da si Evernote prizadeva spremeniti brezplačne uporabnike v plačljive stranke, in poudarjajo pomen preprostosti, zanesljivosti in prenosljivosti aplikacije za vodenje beležk.
- Alternativne aplikacije za beležke, kot sta Obsidian in OneNote, so omenjene kot uspešni konkurenti Evernote, uporabniki pa razpravljajo o tveganjih in slabostih zagonskih podjetij, financiranih s tveganim kapitalom.

### [GPT-4 API Splošna razpoložljivost](https://openai.com/blog/gpt-4-api-general-availability)

- Organizacija OpenAI je objavila splošno razpoložljivost vmesnika GPT-4 API, njihovega najzmogljivejšega modela, za vse plačljive stranke API. Ta zelo zahtevan API omogoča razvijalcem dostop do inovativnih izdelkov, ki uporabljajo GPT-4.
- OpenAI je predstavil tudi API za dokončanje klepeta, ki se je izkazal za zmogljivega in prilagodljivega za široko paleto nalog dokončanja. Predstavlja 97 % uporabe API in ponuja boljše rezultate s strukturiranim vmesnikom za pozive.
- Starejši modeli v vmesniku API za dokončanje bodo 4. januarja 2024 zastareli in nadomeščeni z novimi modeli. Razvijalci, ki uporabljajo te starejše modele, bodo morali nadgraditi svojo integracijo, da bi si zagotovili nadaljnji dostop do API.

#### [Odzivi industrije](http://news.ycombinator.com/item?id=36621120)

- Organizacija OpenAI je objavila API GPT-4, ki je zdaj na voljo vsem plačljivim uporabnikom.
- Modeli ChatGPT, kot sta gpt-3.5-turbo in text-davinci-003, bodo sčasoma odpravljeni v korist API GPT-4.
- GPT-4 v primerjavi s prejšnjimi različicami ponuja izboljšano zmogljivost in zmožnosti, razvijalci pa so pohvalili njegov kakovostni preskok na različnih področjih.

### [Cloud Backed SQLite](https://sqlite.org/cloudsqlite/doc/trunk/www/index.wiki)

- Sistem "Cloud Backed SQLite" (CBS) omogoča shranjevanje podatkovnih zbirk v računih za shranjevanje v oblaku, tako da jih lahko odjemalci za shranjevanje berejo in pišejo, ne da bi prenesli celotno podatkovno zbirko.
- Sistem CBS trenutno podpira Azure Blob Storage in Google Cloud Storage, z možnostjo dodajanja podpore za druge sisteme shranjevanja v oblaku.
- CBS za shranjevanje podatkovnih zbirk SQLite v oblaku uporablja bloke s fiksno velikostjo in manifestno datoteko, ki opisuje, kako se bloki sestavijo v podatkovne zbirke. Zagotavlja tudi vmesnike API in orodja ukazne vrstice za nalaganje, prenašanje in upravljanje podatkovnih zbirk v oblaku.

#### [Odzivi industrije](http://news.ycombinator.com/item?id=36610595)

- "Cloud Backed SQLite" omogoča serviranje velikih podatkovnih zbirk SQLite v kosih in poizvedovanje po njih z zahtevami HTTP.
- Knjižnica, imenovana sql.js-httpvfs, opravi večino dela za drobljenje in nalaganje podatkovne zbirke na strani GitHub.
- Ljudje so navdušeni nad hitrostjo in učinkovitostjo podatkovne zbirke SQLite kot možnosti podatkovne zbirke samo za branje za velike zbirke podatkov brez režijskih stroškov za strežniško infrastrukturo.
