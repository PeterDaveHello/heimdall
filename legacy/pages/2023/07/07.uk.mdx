### [Хмарний SQLite](https://sqlite.org/cloudsqlite/doc/trunk/www/index.wiki)

- Система "Cloud Backed SQLite" (CBS) дозволяє зберігати бази даних в облікових записах хмарного сховища, що дозволяє клієнтам сховища читати і записувати їх без завантаження всієї бази даних.
- Наразі CBS підтримує Azure Cloud Storage та Google Cloud Storage, з потенційною можливістю додавання підтримки інших хмарних систем зберігання даних.
- CBS використовує блоки фіксованого розміру для зберігання баз даних SQLite в хмарі, з файлом маніфесту, який описує, як блоки збираються в бази даних. Він також надає API та інструменти командного рядка для завантаження, вивантаження та керування базами даних у хмарі.

#### [Реакція галузі](http://news.ycombinator.com/item?id=36610595)

- "Cloud Backed SQLite" дозволяє обслуговувати великі бази даних SQLite частинами і запитувати їх за допомогою HTTP-запитів діапазону.
- Бібліотека під назвою sql.js-httpvfs виконує більшу частину роботи з розбиття бази даних на частини та завантаження її на GitHub Pages.
- Люди в захваті від швидкості та ефективності SQLite як бази даних, доступної лише для читання, для великих наборів даних без накладних витрат на серверну інфраструктуру.
