## 25. märts 2023

### GitHub uuendab RSA SSH Hosti võtme pärast kokkupuudet

- GitHub uuendab RSA SSH host-võtit pärast avalikus repositooriumis avalikustatud privaatvõtme avaldamist
- ECDSA või Ed25519 võtmeid kasutavad kliendid ei pea midagi ette võtma, kuid RSA võtmeid kasutavad kliendid peavad uuendama oma ~/.ssh/known_hosts faili.
- GitHub ei ole veel kasutusele võtnud SSH-võtmete tühistamise mehhanismi, mis võib ohu korral olla potentsiaalne probleem.
- Reaalse riski-/ohtumudeli olemasolu on oluline mõistlike turvakontrollide loomiseks.
- Giti autorid peaksid koodi terviklikkuse tagamiseks allkirjastama oma kommiteid oma isikliku võtmega, mitte usaldama hostingupakkujaid.
- GitHub avaldas kogemata oma võtme, mis õõnestas usalduse põhimõtet ja jättis kogu suhtluse alates selle loomisest kompromiteerituks
- GitHub julgustab parandama sertifikaate, mida kasutatakse koodirepositooriumidesse sisselogimiseks
- GitHub ja GitLab toetavad mõlemad kahefaktorilist autentimist (2FA) oma käsurea liidestes (CLI).
- Hiljutine intsident toob esile paljude fintech-ettevõtete võimalikud turvaaugud, sealhulgas pime usaldus GitHubi hostivõtmete suhtes
- GitHub paljastas lühiajaliselt õnnetuse tõttu privaatvõtme, kuid võttis ennetavaid meetmeid, et tühistada ja uuesti kasutusele võtta uued võtmed ja sertifikaadid ning rakendada tõhustatud kontrolle.
- [Arutelu Discussion Serviceis](http://news.ycombinator.com/item?id=35285390) või [Loe originaalteksti](https://github.blog/2023-03-23-we-updated-our-rsa-ssh-host-key/).

### USA kohus keelab Internet Archive'i raamatute laenutusprogrammi

- USA kohtuniku otsuse kohaselt rikkus Internet Archive (IA) autoriõiguse seadust, kui ta laenutas ilma loata autoriõigusega kaitstud raamatute digitaalseid koopiaid.
- IA väitis, et see on vabastatud "õiglase kasutamise" tõttu, kuid kohtunik leidis, et nende raamatute kopeerimine ja laenutamine ei ole midagi muutvat.
- IA-le anti korraldus lõpetada autoriõigusega kaitstud raamatute levitamine ilma loata.
- Hachette, HarperCollins ja Penguin Random House'i kirjastuste poolt IA skaneerimise ja digitaalse laenutamise vastu algatatud kohtumenetlus.
- IA täidab kohtu korraldust ja lõpetab laenutamisprogrammi.
- Otsus mõjutab tõenäoliselt kasutajaid, kes sõltuvad IAst, et saada juurdepääs mujal kättesaamatutele tekstidele, eriti neid, kellele traditsioonilised raamatukogud on kättesaamatud.

### Ühendkuningriigi toiduainete inflatsioon on suurim alates 1977. aastast, mis tekitab muret.

- Toiduainete inflatsioon Ühendkuningriigis tõusis veebruaris 18,2%ni, mis on kõrgeim määr alates 1977. aastast, sest köögiviljade hinnad on tõusnud, teraviljad, teatavad valgud, munad ja õli on kallinenud.
- Ühendkuningriigi supermarketid võitlevad toidupuuduse, kõrgete energiakulude ja ülemaailmsete tarneahelaprobleemidega, mis põhjustavad tühje riiuleid ja hinnatõusu.
- Toiduainete hinnatõus tekitab muret haavatavatele elanikkonnarühmadele, sest mõnedel inimestel on raskusi toidu ostmisega.
- Tõusvad energiakulud ja tarneahela häired on ühed tegurid, mida nimetatakse paljudes riikides toiduainete hinnatõusu põhjustava inflatsioonisurve all.
- Inflatsiooni algpõhjus on kõrgenenud energiahinnad, kuna energia mõjutab tootmist ja transporti, mis omakorda mõjutab toiduainete hinda.

### Framework toob turule suure jõudlusega, kohandatava sülearvuti koos uuendatava graafikaga

- Framework annab välja oma uue toote, Framework Laptop 16, millel on uuendatav graafika ja kohandatav sisendtekk.
- Sisendtekk võimaldab kolme suurusega sisendmoodulite vahetamist: Small, Medium (Numpad Modules) ja Large (Keyboard Modules) koos erinevate kohandamisvõimalustega igaühele.
- Expansion Bay süsteem võimaldab uuendatavat graafikat, pakkudes samas tohutut disaini paindlikkust, et tulla toime tulevaste muudatustega graafikaprotsessorite mehaanilistes, termilistes ja elektrilistes nõuetes.
- Expansion Bay PCIe x8 liides võimaldab suure võimsusega ja kiireid kasutusviise, näiteks kahte M.2 SSD-kaarti, millega saab lisada 16TB salvestusruumi või muid arendajate loodud mooduleid.
- Framework Laptop 16 sisaldab ka Framework Laptop 13-st pärit Expansion Card süsteemi, mis võimaldab kasutajatel valida, milliseid porte nad sooviksid mõlemal küljel koos muude funktsioonide, näiteks kiire salvestusruumi lisamisega.
- Kõigi kolme süsteemi arendusdokumentatsioon: Sisendmoodulid, laienduspesa moodulid ja laienduskardid on saadaval GitHubis, Framework Marketplace on tulevikus avatud ka kolmandate osapoolte moodulite tootjatele.
- Framework Laptop 16 on loodud selleks, et seda saaks kohandada vastavalt kasutaja vajadustele, olgu see siis mängimiseks, arendamiseks, Linuxi kasutamiseks või muudeks jõudlust nõudvateks töödeks.
