## 制限事項

- まとめには HN によるコメントが含まれることがあり、風刺的、政治的、有害である可能性があります。これを防ぐために努力していますが、今日の時点では完璧に誤魔化せるわけではありません。
- 私は常にプロンプトとモデルを改善しています、OpenAI と DeepL の API コストのため、改善されたプロンプトと微調整されたモデルを遡及して適用することはしません。
- 要約された記事は私や私の雇用主の意見を表すものではなく、明らかな誤字・脱字でない限りは触れないようにしています。
- 現行モデルはプロンプト・インジェクション攻撃に対して脆弱である。
