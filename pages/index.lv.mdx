## Ierobežojumi

- Kopsavilkumā var būt iekļauti HN komentāri, kas var būt satīriski, politiski vai kaitīgi. Es strādāju pie tā, lai to novērstu, taču pagaidām tas nav pilnīgi droši.
- Es nepārtraukti uzlaboju uzsaukumus un modeļus, uzlabotos uzsaukumus un precizētos modeļus mēs nepiemēroam ar atpakaļejošu datumu OpenAI un DeepL API izmaksu dēļ.
- Apkopotais raksts neatspoguļo manu vai mana darba devēja viedokli; es tiem nepieskaršos, ja vien tā nav acīmredzama pārrakstīšanās vai nepareizs formāts.
- Pašreizējais modelis ir neaizsargāts pret Prompt Injection uzbrukumiem.
