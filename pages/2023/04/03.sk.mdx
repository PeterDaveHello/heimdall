### Microsoft predstavuje Windows Sandbox na bezpečné testovanie aplikácií v systéme Windows

Windows Sandbox.
Spoločnosť Microsoft predstavila Windows Sandbox, bezpečné a efektívne pracovné prostredie na bezpečné testovanie aplikácií v systéme Windows. Sandbox je nedotknutý, jednorazový a na izoláciu jadra využíva hardvérovú virtualizáciu. Je súčasťou systému Windows a všetko potrebné pre túto funkciu je súčasťou systémov Windows 10 Pro a Enterprise. Windows Sandbox spolu s WSL (Windows Subsystem for Linux) umožňuje používateľom spustiť systém Windows v pieskovisku bez použitia VirtualBoxu/VMware Workstation. Nie je však podporovaný vo verzii Windows Home a jeho sieťové pripojenie je potrebné vypnúť pomocou konfiguračného súboru Windows Sandbox.
[HN](http://news.ycombinator.com/item?id=35413594), [článok](https://learn.microsoft.com/en-us/windows/security/threat-protection/windows-sandbox/windows-sandbox-overview).

Analýza algoritmu Twitteru odhaľuje, že Twitter znižuje poradie tweetov o Ukrajine.
