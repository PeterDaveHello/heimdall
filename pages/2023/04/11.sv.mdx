### Xylazin, en farlig ny drog som driver på Kanadas opioidkris.

[Original](https://macleans.ca/society/what-is-xylazine-the-dangerous-new-drug-fuelling-canadas-opioid-crisis/).
Xylazin, ett kraftfullt lugnande medel som vanligtvis används på stora djur, är ett nytt farligt ämne som har förstärkt giftigheten i Kanadas drogförsörjning och som hittats i 1 350 narkotikabeslag i Ontario och British Columbia. Xylazin är känt av vissa användare som "tranq" eller "zombiedrog" och kombineras ibland med fentanyl, som för närvarande inte har något motgift. Xylazin används också som fyllmedel för att skapa fler droger, vilket ökar deras styrka och pris för mänsklig konsumtion. De fysiska reaktionerna på xylazin varierar och det är svårt att avgöra om någon har fått i sig det. För att mildra effekterna föreslår experter att man har en reglerad, säker läkemedelsförsörjning och fler skadeförebyggande roller.

### Använd GitHub Actions för att göra din GitHub-profil dynamisk.

[Original](https://www.bengreenberg.dev/posts/2023-04-09-github-profile-dynamic-content/).
GitHub Actions kan omvandla ett statiskt Markdown-dokument till en dynamisk profil genom att skrapa data från webbplatser och uppdatera profilen med den senaste informationen med hjälp av skript och arbetsflöden, vilket visas i ett exempel med Ruby. En GitHub-profil kan vara den första kontaktpunkten för potentiella arbetsgivare, samarbetspartners eller bidragsgivare, vilket gör det viktigt att ha en profil som sticker ut och visar upp en individs färdigheter och intressen. GitHub Actions gör det möjligt att automatisera en mängd olika uppgifter, inklusive att köra skript, och kan schemaläggas för att köras regelbundet. Samma principer kan tillämpas med hjälp av andra språk, till exempel JavaScript, TypeScript eller Python.
