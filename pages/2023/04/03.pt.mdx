### Os recipientes compostáveis para alimentos podem libertar produtos químicos tóxicos para o ar

<forte>Acondicionamento composto de comida rápida pode emitir PFAS voláteis.</strong>

Contudo, alguns fabricantes estão a eliminar progressivamente os PFAS em recipientes compostáveis, substituindo-os por revestimentos de base vegetal. A toxicidade das embalagens alimentares está a receber uma atenção crescente, e a utilização de embalagens biodegradáveis como papel e vidro pode ser mais segura. Os investigadores sugerem uma proibição de todos os PFAS nas embalagens de alimentos. Os níveis totais de PFAS caem até 85% após dois anos de armazenamento devido à volatilização de álcoois fluorotelómeros e outros PFAS.

### A Microsoft apresenta o Windows Sandbox para testes seguros de aplicações no Windows

<forte>Windows Sandbox.</strong>
A Microsoft introduziu o Windows Sandbox, um ambiente de trabalho seguro e eficiente para testes seguros de aplicações no Windows. O Sandbox é imaculado, descartável, e utiliza virtualização baseada em hardware para isolamento de kernel. Faz parte do Windows, e tudo o que é necessário para esta funcionalidade está incluído no Windows 10 Pro e Enterprise. Windows Sandbox, juntamente com WSL (Windows Subsystem for Linux), permite aos utilizadores executar o Windows no sandbox sem utilizar a VirtualBox/VMware Workstation. No entanto, não é suportado na edição Windows Home, e a sua ligação de rede precisa de ser desactivada utilizando o ficheiro de configuração do Windows Sandbox.
