### Liberator を使って Apple Notes を Json として抽出する。

- Apple Notes Liberator は、Apple Notes からデータを抽出し、JSON として保存することができる新しいツールです。
- このツールは、Apple Notes を主なメモ書きアプリケーションとして使用している研究者が、構造化されていないメモを取り込むのに役立ちます。
- ユーザーからは、Apple Notes がシンプルなメモアプリから、共有メモや複数同時編集のための CRDT のようなアルゴリズムを持つ現在のデータベース構造へと進化したことが強調されました。
- このツールは、ノートを一括してエクスポートしたり、ノートのエコシステム外で使用できる形式に抽出することが困難であるという Apple Notes の問題に対処しています。
- このプログラムは、抽出された情報を、Apple Notes を表すオブジェクトの配列を含む notes.json ファイルに保存し、将来的には、さらなるデータ型や CSV や HTML などの出力形式をサポートする予定です。
- HN で議論する](http://news.ycombinator.com/item?id=35316679)または[原文を読む](https://github.com/HamburgChimps/apple-notes-liberator)

### Blendergpt を使った自然言語コマンドで Blender を操作する。

- BlenderGPT は、OpenAI の GPT-4 が生成する自然言語コマンドを使って Blender を操作できる AI ツールですが、生成されたコードが必ずしも正しいとは限らないため、反復作業が必要です。
- このツールは、ノンプログラマーが簡単なデザインやアニメーションを作成するのに便利ですが、Blender の API の知識が必要な、より複雑で精密な作業には十分でない場合があります。
- AI と自動化の発展により、ソフトウェア開発の自動化が進み、未熟な労働者が機械に取って代わられる可能性があり、仕事の将来やソフトウェア産業で成功するために必要なスキルについて疑問が呈されています。
- 特定の作業が自動化されることで、より複雑で創造的な作業に時間を割くことができ、業界や経済全体に利益をもたらすという意見もあります。
- プログラミングのための自然言語インターフェースの可能性についての議論は、必要とされる具体性や精度の欠如を中心に展開されますが、明示的な記号的アプローチや、フレームワークメーカーによって「認可された」大きなビジネスロジックブロックをつなぎ合わせるために LLM を使用することを提案する人もいます。
- OpenAI の GPT は、ChatGPT の助けを借りて自然言語のクエリから Blender コマンドを生成できるようになり、3D モデリングソフトウェアでデザインするための UI やその他のユースケースに革命を起こす可能性があります。
- GitHub の BlenderGPT リポジトリには、OpenAI の GPT-4 で Blender を制御するための自然言語コマンドを可能にする拡張機能がホストされていますが、ユーザーはコードを再度実行する必要があり、有効な OpenAI API キーが必要になる場合があります。
- HN で議論する](http://news.ycombinator.com/item?id=35320571) または [原文を読む](https://github.com/gd3kr/BlenderGPT)

### Blake3 による部分的なファイル重複排除のための Sqlite の利用

- あるユーザーが SQLite を使って部分的なファイル重複排除を行ったことをきっかけに、別の重複排除方法や、ゲームのファイル更新を処理する際の課題についての議論が始まった。
- 解決策として、クラスタ複製された SQLite やコンテンツ定義のチャンキングが挙げられ、全ファイルベースのストレージと比較した重複排除の分析も共有された。
- また、開発者がコンテンツを更新する際のユースケースに関心が寄せられ、ブロックレベルとファイル全体の重複排除、Steamworks 開発者向けのアライメントの考慮など、技術的な詳細が議論された。
- 筆者は、BLAKE3 ハッシュ関数と複雑なクエリを用いて類似ファイルをマッチングさせるアプローチを採用し、よりシンプルな代替案よりも容量を節約することに成功した。
- 他のコメントでは、データセットがメモリを超えた場合のファイルの重複排除や、システム管理データの保存など、SQLite の型破りな使用例を紹介しています。
- HN で議論する](http://news.ycombinator.com/item?id=35317419) または [原文を読む](https://sqlite.org/forum/forumpost/7fecf11e42c71a91?raw)
