### Ekstrak Catatan Apple sebagai Json Menggunakan Liberator

- Apple Notes Liberator adalah sebuah alat baru yang memungkinkan pengguna untuk mengekstrak data dari Apple Notes dan menyimpannya sebagai JSON.
- Alat ini membantu para peneliti yang menggunakan Apple Notes sebagai aplikasi pencatatan utama mereka untuk menangkap memo yang tidak terstruktur.
- Para pengguna menyoroti evolusi Apple Notes dari aplikasi pencatatan sederhana menjadi struktur basis data saat ini dengan algoritme mirip CRDT untuk catatan bersama dan beberapa pengeditan bersamaan.
- Alat ini mengatasi masalah di Apple Notes yang menyulitkan untuk mengekspor catatan secara massal atau mengekstraknya ke dalam format yang dapat digunakan di luar ekosistem Notes.
- Program ini menyimpan informasi yang diekstrak dalam file notes.json yang berisi serangkaian objek yang mewakili Apple Notes, dengan dukungan yang direncanakan untuk tipe data lebih lanjut dan format keluaran seperti CSV dan HTML di masa mendatang.
- [Diskusikan di HN](http://news.ycombinator.com/item?id=35316679) atau [Baca Teks Asli](https://github.com/HamburgChimps/apple-notes-liberator)

### Panduan : Plugin Chatgpt Openai dengan Llama

- OpenAI telah merilis dukungan untuk plugin untuk ChatGPT, yang memungkinkan model bahasa untuk berinteraksi dengan dunia luar menggunakan API.
- Panduan ini mendemonstrasikan bukti konsep menggunakan plugin chatgpt-retrieval milik OpenAI dengan model bahasa LLaMA milik Meta.
- Keterbatasan dari pendekatan ini termasuk perbedaan dalam model dan kesulitan dalam menggeneralisasi ke plugin lain.
- Panduan langkah demi langkah ini mencakup pengaturan penyimpanan data, mengunggah file, mengatur LLaMA, menggunakan LLaMA untuk meminta data khusus, dan bereksperimen dengan pengaturan yang berbeda.
- Tim lastmile ai sedang menjajaki kebutuhan akan standar Protokol Plugin Model Foundation untuk menstandarkan interaksi antara model foundation dan plugin/ekstensi.

### Menggunakan Sqlite untuk Deduplikasi Berkas Parsial dengan Blake3

- Seorang pengguna membagikan penggunaan SQLite yang tidak konvensional untuk deduplikasi file parsial, yang memicu diskusi tentang metode deduplikasi alternatif dan tantangan dalam menangani pembaruan file untuk game.
- Solusi yang memungkinkan termasuk SQLite yang direplikasi cluster dan chunking yang ditentukan konten, dan analisis deduplikasi dibagikan untuk dibandingkan dengan penyimpanan berbasis seluruh file.
- Ketertarikan diungkapkan dalam kasus penggunaan yang mungkin bagi pengembang yang memperbarui konten, dan detail teknis dibahas seperti deduplikasi tingkat blok versus deduplikasi seluruh file dan pertimbangan penyelarasan untuk pengembang Steamworks.
- Pendekatan penulis melibatkan penggunaan fungsi hash BLAKE3 dan kueri yang rumit untuk mencocokkan file yang serupa, terbukti lebih hemat ruang daripada alternatif yang lebih sederhana.
- Komentator lain berbagi kasus penggunaan yang tidak biasa untuk SQLite, termasuk deduplikasi file ketika kumpulan data melebihi memori dan menyimpan data administrasi sistem.
