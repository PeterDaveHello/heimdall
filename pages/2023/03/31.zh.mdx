## 2023 年 3 月 31 日

### 美国禁令法案引起了人们对 VPN 和 TikTok 的关注

一项由白宫支持的、旨在监管互联网的美国禁令法案可能使 VPN 成为非法，给予最高 100 万美元的罚款和 20 年的监禁，并被视为对公民自由的攻击。该法案得到了两党的支持，重点关注与中国有关的国家安全问题，并限制与中国的商业往来。欧盟可能不会效仿美国的趋势，因为各国都有自己的法律结构，TikTok 的禁令可能导致年轻人的不满和对美国政府的报复。该禁令的批评者认为，它威胁到个人隐私以及言论和表达自由。
[Hacker News Link](http://news.ycombinator.com/item?id=35366955), [Original Post](https://www.youtube.com/watch?v=FWQGA_n5Z4M).

### 对 Y Combinator 对初创企业的价值看法不一

关于 Y Combinator（YC）对初创企业的价值，人们的看法不一，取决于个人的观点和需求。有些人重视 YC 的网络和指导，有些人则批评其建议和对短期目标的关注。YC 的声誉和品牌可以帮助筹集资金，但其不设上限的 SAFE 交易和强调不惜一切代价扩大规模可能不适合所有创始人。社区建设和个人联系在远程模式中的重要性也有争议。归根结底，YC 的价值取决于创始人利用其资源最大化成功机会的能力。
[Hacker News Link](http://news.ycombinator.com/item?id=35373019), [Original Post](https://acecreamu.substack.com/p/is-y-combinator-worth-the-money).

### Scratch：受欢迎的儿童可视化编程语言

Scratch 是一种针对儿童的流行的可视化编程语言，它使用彩色块而不是基于文本的编码来帮助他们学习编程概念。该程序包括基本的编程原理，如变量、列表、if/else 和循环控制流结构以及基于事件的并发性。Scratch 有一个简单的、用户友好的方法，鼓励孩子们以创造性的方式探索计算，同时也非常注重同伴互动和分享工作以获得反馈。Scratch 社区是世界上最大的编码社区之一，有超过 1.03 亿用户，在该平台上分享了 1.23 亿个项目。虽然有人担心不受监管的社交媒体网络和儿童玩视频游戏而不是将 Scratch 用于教育目的，但该工具激发了许多人对编程的兴趣，项目范围从初级编码到高级编程技能。
[Hacker News Link](http://news.ycombinator.com/item?id=35373052), [Original Post](https://scratch.mit.edu/about).

### LED 照明技术的挑战与权衡

LED 灯泡存在快速失效、显色性和光谱相似性指数差、电源设计不良、过热以及室内照明成本较高等挑战。恒定电流电源对 LED 照明来说是必要的，而且灯具的设计需要与白炽灯不同，以便散热。人们对 LED 灯泡的可靠性和耐用性感到担忧，尽管高质量的选择可以在较高的价位上获得。在将 LED 灯泡与调光器开关整合并确保兼容性方面存在挑战。LED 照明的闪烁和缺乏生动性，导致了新的分类系统和可调整的 LED 灯泡的发展。总的来说，LED 技术有潜力，但需要继续发展和创新。
[Hacker News Link](http://news.ycombinator.com/item?id=35371750), [Original Post](https://nymag.com/strategist/article/led-light-bulbs-investigation.html).

### 超过一半的新 NPM 包是垃圾邮件

超过一半的新提交的 npm 包是垃圾邮件，其中许多包含恶意网站的链接或提供虚假的免费产品，如 Fortnite V-Bucks。垃圾邮件的增加被认为与编码训练营有关，在那里，学生们被鼓励发布软件包，即使它们什么都不做。回应者提出了自动化测试流程和信任网模式等解决方案。这个问题凸显了软件开发行业道德规范的复杂性，以及言论自由和私有产权之间的紧张关系。安全研究人员已经发现了整个 npm 注册表中存在的垃圾邮件包，并正在进一步调查。
[Hacker News Link](http://news.ycombinator.com/item?id=35370728), [Original Post](https://blog.sandworm.dev/one-in-two-new-npm-packages-is-seo-spam-right-now).

### 美国提出对技术自由和隐私有威胁的法案

美国政府提出了一项法案，将赋予其前所未有的权力，禁止任何它认为是 "威胁国家安全 "的外国技术，让一些人认为这可能导致类似中国的审查权力，而且比最初想象的要广泛得多。关于禁止中国拥有的社交媒体应用程序也有争论，一些人认为这是一个走向审查信息的滑坡，而另一些人则认为有必要防止中国公司窃取美国技术和为间谍目的收集信息。RESTRICT 法案是一项拟议的法律，可以用来禁止 TikTok 和 VPN，批评者认为它可能违反第一修正案。共和党参议员还提议禁止联邦雇员使用中国公司拥有的社交媒体应用程序，因为他们担心间谍活动和监控。一些评论家认为，该立法相当于政府的保护主义，而另一些人则认为，社交媒体的发展损害了心理健康，助长了对公众舆论的操纵。
[黑客新闻链接](http://news.ycombinator.com/item?id=35369075), [原创文章](https://www.vice.com/en/article/4a3ddb/restrict-act-insanely-broad-ban-tiktok-vpns).

### 开源 3D 渲染引擎用于教育和整合

一个 StratusGFX 开源 3D 渲染引擎已经被一个开发者发布，供人们将其渲染代码整合到其他引擎中或作为教育资源使用。它的功能包括实时反射、阴影和完整的 PBR 着色支持，但它目前只支持 OpenGL。它的创造者正在考虑将渲染器移植到 Vulkan，以提高跨平台的兼容性。StratusGFX 引擎的设计是为了最大限度地提高真实性，但对于不那么强大的硬件来说，它具有回退的妥协能力，其架构可在 Github 上用于 Windows 和 Linux。该引擎处于预发布/beta-release 阶段，预计会有大量的错误和不稳定。
[Hacker News Link](http://news.ycombinator.com/item?id=35370284), [Original Post](https://github.com/KTStephano/StratusGFX).

### 谷歌否认使用 OpenAI 的 ChatGPT 来训练 Bard

谷歌否认使用 OpenAI 的 ChatGPT 来训练 Bard，一个新的语言模型。人们对用另一个商业控制的 LLM 的输出来训练一个大型语言模型的道德问题表示关注。OpenAI 的服务条款禁止通过其 ML 输出来训练竞争对手的模型，但谷歌不受此约束。一名谷歌前雇员声称他们使用了 OpenAI 的数据，但谷歌否认了这一点。人工智能产生的材料是没有版权的，对其他模型输出的训练是允许的。有些人认为，偷窃 ChatGPT 在道德上是合理的，因为它是团体来源的，但在其他人工智能生成的数据上训练模型可能会执行坏数据。
[Hacker News Link](http://news.ycombinator.com/item?id=35370915), [Original Post](https://twitter.com/steventey/status/1641267979399704576).

### Twitter 的 API 被忽视，用户的不满情绪不断增加

据报道，Twitter 的 API 没有得到维护，导致了用户的不稳定问题。一些用户报告说，由于科学论文内容的减少和煽动性或攻击性内容比例的增加，他们对该服务感到疏远。许多社交媒体平台使用算法推送，但默认值很重要，因为它们影响用户行为。推特的用户遇到了内部服务器错误和网站访问困难等问题，导致一些人转而使用其他平台，如 Mastodon。Twitter 最近的不稳定问题导致一些用户减少使用。尽管有这些担忧，一些人认为，在马斯克的领导下，Twitter 的增长表明，无论是否有忠实的用户，该平台都将继续蓬勃发展。埃隆-马斯克限制访问 Twitter 的 API 的决定引起了争议，批评者指责他故意将 Twitter 推向绝境。现在，Twitter 的 API 实际上已经无人维护，只有一名员工定期在 Twitter 开发者论坛上发帖，并且只关注广告 API。
[Hacker News Link](http://news.ycombinator.com/item?id=35370152), [Original Post](https://snarfed.org/the-twitter-api-is-now-effectively-unmaintained).
