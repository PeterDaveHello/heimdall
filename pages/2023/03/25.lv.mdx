## 2023. gada 25. marts

### GitHub atjaunina RSA SSH hostu atslēgu pēc atklāšanas

- GitHub atjaunina RSA SSH host atslēgu pēc tam, kad publiskajā repozitorijā tika atklāta privātā atslēga.
- Klientiem, kas izmanto ECDSA vai Ed25519 atslēgas, nav jārīkojas, bet tiem, kas izmanto RSA, jāatjaunina ~/.ssh/known_hosts fails.
- GitHub vēl nav ieviesis SSH atslēgu atsaukšanas mehānismu, kas varētu būt potenciāla problēma kompromitēšanas gadījumā.
- Lai izveidotu saprātīgu drošības kontroli, ir ļoti svarīgi izmantot reālu riska/apdraudējuma modeli.
- Git autoriem būtu jāparaksta savi ieraksti ar savu privāto atslēgu, lai nodrošinātu koda integritāti, nevis jāuzticas mitināšanas pakalpojumu sniedzējiem.
- GitHub nejauši publicēja savu atslēgu, tādējādi apdraudot uzticības principu un padarot kompromitētu visu saziņu kopš tās izveides.
- GitHub rosina uzlabot sertifikātus, ko izmanto, lai pieteiktos kodu krātuvēs.
- Gan GitHub, gan GitLab atbalsta divu faktoru autentifikāciju (2FA), izmantojot komandrindas saskarnes (CLI).
- Nesenais incidents izgaismo potenciālās drošības nepilnības daudzos finanšu tehnoloģiju uzņēmumos, tostarp aklo uzticēšanos GitHub hostu atslēgām.
- GitHub negadījuma dēļ uz īsu brīdi atklāja privāto atslēgu, taču veica proaktīvus pasākumus, lai atsauktu un atkārtoti izvietotu jaunas atslēgas un sertifikātus un ieviestu uzlabotas kontroles.
- [Apspriest HN](http://news.ycombinator.com/item?id=35285390) vai [Lasīt oriģinālo tekstu](https://github.blog/2023-03-23-we-updated-our-rsa-ssh-host-key/).

### ASV tiesa aizliedz interneta arhīva grāmatu aizdošanas programmu

- Saskaņā ar ASV tiesneša lēmumu Internet Archive (IA) pārkāpa autortiesību likumu, bez atļaujas aizdodot ar autortiesībām aizsargātu grāmatu digitālās kopijas.
- IA apgalvoja, ka tas ir atbrīvots no prasības "godīgas izmantošanas" dēļ, bet tiesnesis nekonstatēja, ka grāmatu kopēšana un aizdošana nav nekas pārveidojošs.
- IA tika uzdots pārtraukt ar autortiesībām aizsargātu grāmatu izplatīšanu bez atļaujas.
- Hachette, HarperCollins un Penguin Random House izdevēji iesniedza prasību tiesā pret IA skenēšanas un digitālās aizdošanas praksi.
- IA izpildīs tiesas rīkojumu un pārtrauks aizdošanas programmu.
- Lēmums, visticamāk, ietekmēs lietotājus, kuri paļaujas uz IA, lai piekļūtu tekstiem, kas nav pieejami citur, jo īpaši tos, kuriem tradicionālās bibliotēkas nav pieejamas.

### Apvienotās Karalistes pārtikas inflācija sasniedz augstāko līmeni kopš 1977. gada, radot bažas

- Pārtikas inflācija Apvienotajā Karalistē februārī pieauga līdz 18,2 %, kas ir augstākais rādītājs kopš 1977. gada, jo sadārdzinājās dārzeņi, pieauga graudu, atsevišķu olbaltumvielu, olu un eļļas cenas.
- Apvienotās Karalistes lielveikali cīnās ar pārtikas trūkumu, augstām enerģijas izmaksām un globālās piegādes ķēdes problēmām, kas noved pie tukšiem plauktiem un cenu kāpuma.
- Pārtikas cenu inflācija rada bažas neaizsargātām iedzīvotāju grupām, jo daži cilvēki cīnās, lai iegādātos pietiekami daudz pārtikas.
- Pieaugošās enerģijas izmaksas un piegādes ķēdes traucējumi ir vieni no faktoriem, kas tiek minēti kā spiedīgs inflācijas spiediens, kas izraisa pārtikas cenu pieaugumu daudzās valstīs.
- Galvenais inflācijas cēlonis ir paaugstinātas enerģijas cenas, jo enerģija ietekmē ražošanu un transportēšanu, kas savukārt ietekmē pārtikas produktu cenas.

### Framework laiž klajā augstas veiktspējas, pielāgojamu klēpjdatoru ar modernizējamu grafiku

- Uzņēmums Framework izlaiž savu jauno produktu - klēpjdatoru Framework Laptop 16, kam ir modernizējama grafika un pielāgojama ievades platforma.
- Ieejas bloku var nomainīt pret trīs izmēru ieejas moduļiem: Mazs, vidējs (Numpad moduļi) un liels (tastatūras moduļi) ar dažādām pielāgošanas iespējām katram no tiem.
- Expansion Bay sistēma ļauj uzlabot grafiku, vienlaikus nodrošinot arī milzīgu dizaina elastību, lai nākotnē varētu pielāgoties GPU mehāniskajām, termiskajām un elektriskajām prasībām.
- PCIe x8 interfeiss Expansion Bay sistēmā ļauj izmantot jaudīgus un ātrdarbīgus izmantošanas gadījumus, piemēram, dubulto M.2 SSD karti, kas var pievienot 16 TB datu glabāšanas vietu, vai citus izstrādātāju izveidotos moduļus.
- Framework Laptop 16 ir arī paplašināšanas karšu sistēma no Framework Laptop 13, kas ļauj lietotājiem izvēlēties, kurus portus viņi vēlas katrā pusē, kā arī pievienot citu funkcionalitāti, piemēram, ātrgaitas datu glabāšanas sistēmu.
- Visu trīs sistēmu izstrādātāju dokumentācija: Ieejas moduļi, paplašināšanas ligzdu moduļi un paplašināšanas kartes ir pieejamas vietnē GitHub, un Framework Marketplace nākotnē tiks atvērts trešo pušu moduļu ražotājiem.
- Framework Laptop 16 ir veidots tā, lai to varētu pielāgot lietotāja vajadzībām - gan spēlēm, gan izstrādei, gan Linux lietošanai, gan citiem darbiem, kas prasa lielu veiktspēju.

### OpenAI ChatGPT un GPT-3 saskaras ar datu izpaušanu un noplūdi

- OpenAI ChatGPT piedzīvoja datu izpaušanas incidentu, kas ļāva lietotājiem 9 stundu laikā redzēt citu lietotāju sensitīvos datus un ar maksājumiem saistīto informāciju par 1,2 % ChatGPT Plus abonentu.

- OpenAI arī cieta no drošības pārkāpuma, kura rezultātā tika atklāti klientu dati, tostarp vārdi, uzvārdi, e-pasta adreses un API lietošanas preferences.
- Daži lietotāji ir ziņojuši par problēmām ar OpenAI kļūdu ziņošanas kanāliem, un šis incidents rada bažas par Redis kā galvenās datubāzes izmantošanu un OpenAI infrastruktūras uzticamību no drošības un atbalsta viedokļa.
- Izstrādātāji un programmētāji apspriež atbildību par atvērtā koda bibliotēku koda kvalitātes un drošības nodrošināšanu, un šis incidents uzsver, cik svarīgi ir rūpīgi pārbaudīt ārējās atkarības.
- OpenAI ir veikusi pasākumus incidentu novēršanai, tostarp labojusi kļūdu Redis klienta bibliotēkā, uzlabojusi savas sistēmas, mainījusi visas API atslēgas un žetonus, kā arī uzlabojusi testēšanas, apgalvojumu, žurnāla brīdinājumu un atkļūdošanas procesus.
