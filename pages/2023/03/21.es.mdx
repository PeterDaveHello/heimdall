### [Web fingerprinting is worse than I thought](https://www.bitestring.com/posts/2023-03-19-web-fingerprinting-is-worse-than-I-thought.html?ref=hackernewsgpt.com)

- El web fingerprinting es una técnica utilizada por los sitios web para identificar y rastrear a usuarios individuales mediante el uso de información sobre su navegador web y la configuración de su dispositivo, sin depender de cookies u otro tipo de almacenamiento persistente.
- El fingerprinting se vende como un servicio a los sitios web de comercio electrónico para la prevención del fraude y la seguridad del sitio web.
- Este artículo detalla cómo una empresa llamada FingerprintJS está vendiendo fingerprinting como servicio a otros sitios web, y presenta pruebas que demuestran la eficacia del fingerprinting y las mejores estrategias para protegerse contra él.
- Usar el Navegador Tor o Firefox con resistFingerprinting=true proporciona más protección que usar navegadores basados en Chromium como Google Chrome o Brave.
- Las VPN no ofrecen protección contra el fingerprinting.

### [Asahi Linux lanza un controlador de GPU actualizado con mayor rendimiento y compatibilidad](https://asahilinux.org/2023/03/road-to-vulkan/?ref=hackernewsgpt.com)

- El equipo de Asahi Linux ha publicado la versión 0.3 de su controlador de GPU, que incluye enormes mejoras de rendimiento, mayor compatibilidad con Mesa y soporte explícito de sincronización, lo que aumenta el paralelismo CPU-GPU.
- Asahi Linux lleva un año trabajando en controladores de GPU de código abierto para Apple Silicon y recientemente ha diseñado una API de espacio de usuario para cada clase de GPU, que es la interfaz entre las partes de núcleo y espacio de usuario del controlador.
- El sistema de seguimiento de lotes se desarrolló para gestionar el envío de lotes que no están activos pero que se envían para aumentar la eficiencia.
- El nuevo controlador también admite el envío de múltiples VM y trabajos, la capacidad de ejecutar sombreadores de cálculo e incluye soporte de stub para comandos blit, al tiempo que se han corregido errores de seguridad de la memoria en el componente del programador DRM.

### [Mailchimp pone en la lista negra a los usuarios que abren las herramientas de desarrollo del navegador](http://news.ycombinator.com/item?id=35235732&ref=hackernewsgpt.com)

- Los usuarios de MailChimp han denunciado haber sido bloqueados de sus cuentas tras abrir las herramientas de desarrollo de su navegador mientras editaban correos electrónicos.
- Los comentaristas señalaron que la prohibición podría estar relacionada con vulnerabilidades de secuencias de comandos en sitios cruzados causadas por el uso de HTML no descifrado o malformado en la consola o con API que interactúan con el cortafuegos de aplicaciones web del sistema.
- Algunos especularon que Mailchimp intentaba evitar que los usuarios inspeccionaran las plantillas de correo electrónico o realizaran ingeniería inversa en las campañas de correo electrónico, o que el mecanismo de bloqueo era para evitar que los usuarios utilizaran bloqueadores de anuncios.
- El asunto ha suscitado preocupación por las restricciones a las herramientas de desarrollo y las implicaciones para los desarrolladores, a quienes mecanismos de seguridad demasiado agresivos o cautelosos pueden negar herramientas esenciales para su trabajo.

### [Petals permite el uso colaborativo de modelos lingüísticos de más de 100B](https://petals.ml/?ref=hackernewsgpt.com)

- Petals permite el uso colaborativo de grandes modelos lingüísticos (por ejemplo, BLOOM-176B) cargando una pequeña parte del modelo y asociándose con otros que sirvan las demás partes.
- Los usuarios pueden ejecutar la inferencia o el ajuste fino, con una inferencia de lote único que funciona a ≈ 1 seg por token, y una inferencia paralela que alcanza cientos de tokens/seg.
- Petals ofrece más flexibilidad que las API clásicas de modelos lingüísticos, con la posibilidad de emplear cualquier método de ajuste fino y muestreo, ejecutar rutas personalizadas a través del modelo o ver sus estados ocultos. Utiliza PyTorch para una mayor flexibilidad.
- Actualmente está en lista de espera, pero los usuarios pueden probarlo en ColabDocs en GitHub o unirse al canal de Discord para seguir su desarrollo.

### [Así que has instalado `fzf` - ¿y ahora qué?](https://andrew-quinn.me/fzf/?ref=hackernewsgpt.com)

- Fzf es una herramienta de línea de comandos que permite la búsqueda difusa de archivos y directorios.
- Atajos de teclado como Ctrl+R y Alt+C pueden mejorar la eficiencia de la búsqueda.
- Fzf puede utilizarse en combinación con otras herramientas UNIX, como grep y vi, para mejorar la experiencia de búsqueda.
- Combinando fzf con rg, una rápida herramienta grep recursiva, se pueden realizar búsquedas aún más rápidas.
