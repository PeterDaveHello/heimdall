### [ウェブフィンガープリントは思ったより悪い](https://www.bitestring.com/posts/2023-03-19-web-fingerprinting-is-worse-than-I-thought.html?ref=hackernewsgpt.com)

- Web フィンガープリンティングは、クッキーやその他の永続的なストレージを使用せずに、異なるブラウジングセッションや Web サイト間でユーザーを識別するために使用される方法です。ユーザーのウェブブラウザとハードウェア構成を調査することで動作し、数百万の中からユニークな ID を生成することができます。
- 多くのウェブサイトがフィンガープリント・ライブラリを使用してユニークな ID を生成しており、フィンガープリントを他のウェブサイトへのサービスとして販売している企業もある。
- FingerprintJS はそのような企業の 1 つで、オープンソースの JavaScript フィンガープリント・ライブラリを多くのウェブサイトに販売しています。ユーザーがブラウザのキャッシュや他のサイトデータをすべて消去しても、FingerprintJS は同じビジター ID を生成し、ユーザーの前回の訪問に結びつけることができる。
- フィンガープリンティングはユーザーのプライバシーを著しく損ないますが、それを防ぐ方法があります。Firefox の resistFingerprinting=true と Tor Browser はフィンガープリンティングに対してより安全であり、Brave ブラウザはセッションごとにフィンガープリントをランダム化する。しかし、Brave はまだ Google の Chromium エンジンをベースにしており、Google の独占を助長しています。VPN は IP アドレスをマスクするだけなので、フィンガープリンティングには役立ちません。

### [アドビの AI アートジェネレーター：ホタル](https://www.adobe.com/sensei/generative-ai/firefly.html?ref=hackernewsgpt.com)

- Adobe Firefly は、デジタルイメージング、写真、イラスト、アートワーク、グラフィックデザイン、ビデオ、マーケティング、ソーシャルメディア、3D モデリングのためのコンテンツ生成に使用できる、アドビ製品に搭載されるクリエイティブな生成 AI モデルファミリーです。
- このジェネレーティブ AI により、クリエイターは日常的な言葉を使って、並外れた新しいコンテンツを素早く生成し、すでに知っている大好きなツールを使って作成したものを編集し、わずかな言葉やスケッチからカスタムベクター、ブラシ、テクスチャーを生成できるようになります。
- アドビは、クリエイターを中心に据えて、クリエイティブなジェネレーティブ AI を責任を持って開発することを約束し、Firefly の進化とともに、クリエイティブなプロセスをサポートし改善する技術を構築するために、クリエイティブコミュニティと協力し続けます。
- Firefly がユニークなのは、アドビが単なるテキストから画像への AI ジェネレーターにとどまらないことを意図しているからです。
- Firefly のような AI ジェネレーターは、人々が想像し、実験し、アイデアを実現する新しい方法を提供することで、創造性を高めることができます。
- ジェネレーティブ AI を使えば、誰でもビデオ、ドキュメント、デジタル体験、そして豊かな画像やアートを作ることができます。
- Firefly は Creative Cloud に組み込まれており、結果を素早く生成し、独自のビジョンに合うようにカスタマイズするためのツールを提供します。
- Firefly で作成されたコンテンツには、生成 AI を使用して作成されたことを示す Content Credentials タグが含まれます。
- 現在、Firefly は Adobe Stock、オープンライセンスワーク、著作権の切れたパブリックドメインコンテンツのデータセットで学習していますが、アドビはクリエイターが自身のアセットで機械学習モデルを学習する方法を検討する予定です。

### [ボブ・メトカーフがチューリング賞を受賞](https://amturing.acm.org/?2023=&ref=hackernewsgpt.com)

- Bob Metcalfe が、50 年前に開発した Ethernet の発明、標準化、商業化に対して、2022 年 ACM A.M. Turing Award の受賞者に決定しました。

- 現在、イーサネットは世界中の有線ネットワーク通信の主要な導管であり、10Mbps から 400Gbps までのデータレートに対応し、800Gbps や 1.6Tbps の技術も登場しています。

### [Petals: Run 100B+ Language Models Collaboratively at Home](https://petals.ml/?ref=hackernewsgpt.com)

- Petals は、BLOOM-176B のような大規模な言語モデルの協調実行を可能にします。ユーザーはモデルの一部をロードし、推論や微調整のために残りの部分を提供する他のユーザーとチームを組みます。
- シングルバッチ推論は 1 ステップあたり 1 秒で実行され、オフロードに比べて最大 10 倍速いため、チャットボットのような対話型アプリケーションに有効です。
- Petals は PyTorch の柔軟性を生かし、微調整やサンプリング方法、モデルを通るカスタムパス、隠れた状態へのアクセスなどを可能にします。
