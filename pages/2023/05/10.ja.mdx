## 2023 年 5 月 10 日

### [アップル、Final Cut Pro と Logic Pro を iPad に対応](https://www.apple.com/newsroom/2023/05/apple-brings-final-cut-pro-and-logic-pro-to-ipad/)

- Final Cut Pro と Logic Pro for iPad は、ビデオや音楽のクリエイターのために設計されたパワフルなツールで、究極のモバイルスタジオ体験を提供します。
- これらのアプリケーションは、iPad の即時性と直感的なマルチタッチジェスチャーでワークフローを向上させることができるタッチファーストインターフェイスを導入しています。
- Final Cut Pro for iPad は、ビデオクリエイターのための新しいツールを導入し、ビデオの撮影、編集、仕上げ、共有を可能にします。Logic Pro for iPad は、作曲、ビートメイキング、録音、編集、ミキシングのための洗練されたツールの完全コレクションを提供します。

#### [業界の反応](http://news.ycombinator.com/item?id=35873732)

- Logic Pro X for iPad のリリースは、タブレットでの音楽制作にとって画期的なことですが、最終的なオーディオのミキシングと制作に関しては、ユーザーはデバイスの制限に注意する必要があります。
- Apple は、同社のアプリ、特に Final Cut Pro と Logic Pro を、それぞれ月額 4.99 ドルまたは年額 49 ドルのサブスクリプションモデルに移行しています。

### [言語モデルでニューロンを説明できる](https://openai.com/research/language-models-can-explain-neurons-in-language-models)

### [10 年後に学ぶ DNS](https://jvns.ca/blog/2023/05/08/new-talk-learning-dns-in-10-years/)

- RubyConf Mini のスピーカーは「10 年で DNS を学ぶ」と題して、難しい概念を学ぶための構造化されたアプローチについて語りました。

- 講演者は DNS を学ぶために、DNS とは何かを議論すること、dig や Wireshark などのツールでスパイすること、混乱して質問すること、仕様を読むこと、実験をすることを勧めています。

#### [業界の反応](http://news.ycombinator.com/item?id=35870654)

### [ビタミン D の日常摂取でがん死亡率が減少](https://www.dkfz.de/en/presse/pressemitteilungen/2023/dkfz-pm-23-27-Reduced-cancer-mortality-with-daily-vitamin-D-intake.php)

- ビタミン D を毎日摂取することで、集団におけるがん死亡率を 12％低減できることが、ドイツがん研究センターが約 10 万 5000 人を対象に実施した 14 の研究の評価で明らかになりました。
- ビタミン D 欠乏症は世界中で広く見られ、がん患者さんでは頻繁に診断される。
- ビタミン D3 を毎日投与することでより優れた効果が得られるのは、腫瘍の成長を抑制する活性物質である 1,25-ジヒドロキシビタミン D というホルモンのバイオアベイラビリティがより定期的に得られるためであると説明されている。本研究は、がん死亡予防におけるビタミン D3 投与の可能性を明らかにし、効果的な治療法に関する今後の研究につながる可能性があります。

#### [業界の反応](http://news.ycombinator.com/item?id=35876062)

- 最適なビタミン D 濃度は 60 ～ 70ng/ml 程度ですが、欧米では多くの人が欠乏していると言われています。
- ビタミン D を補給すると、特にビタミン D 欠乏症の人のがん死亡率が低下するが、高用量では効果がない。

### [Syncthing: A continuous file synchronization program](https://syncthing.net/)

- Syncthing は、2 台以上のコンピュータ間でファイルをリアルタイムに同期させ、不正なアクセスから保護するプログラムです。
- データは各自のコンピュータにのみ保存されるため、危険にさらされる可能性のある中央サーバーは存在しません。
- Syncthing は様々なプラットフォームで利用できるオープンソースのプログラムで、使いやすく、デバイス間でデータを同期するための強力なツールとなっています。

#### [業界の反応](http://news.ycombinator.com/item?id=35879039)

- ユーザーは Syncthing の信頼性を高く評価していますが、Apple の機能制限のために優れた iOS アプリがないことに不満を表明しています。
- ユーザーは、iCloud と同様の API を統合するなどの解決策を提案し、Nextcloud、Seedvault、ワーキングコピーなど他のバックアップオプションについても議論しています。

### ["Tinder for Canceling Meetings"](https://meetcala.com/)

- "会議をキャンセルするための Tinder "として、Cala という新しいアプリが紹介されています。
- このアプリは、参加者全員が匿名で左にスワイプして会議をキャンセルすることができ、全員が左にスワイプした場合のみ投票内容が明らかになる。
- Cala は、ユーザーが不必要な会議を避け、集中力を高めることを目的として設計されています。

#### [業界の反応](http://news.ycombinator.com/item?id=35868685)

- 会議の直前キャンセルに関する議論では、組織内で生産的な会議を行うための良好なコミュニケーションとガイドラインの必要性が強調されています。
- Cala の iOS アプリでは、参加者が「スワイプ」して会議をキャンセルすることができますが、一部のユーザーは、アプリをやみくもに使用すると、参加者から有益な最新情報や重要な決定を奪うことになると指摘し、直接フィードバックを提供したり、会議のコストプラグインを作成したりする代替案を提案しています。
