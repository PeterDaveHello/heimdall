## [Slack 向蜂窝架构的迁移](https://slack.engineering/slacks-migration-to-a-cellular-architecture/)

- Slack 已从单体架构过渡到基于单元的架构，以增强冗余并最大限度地减少站点故障的影响。
- 迁移的起因是一个可用区的网络中断导致 Slack 用户出错。
- Slack 的新蜂窝架构可在单个可用区内隔离故障，并重定向用户请求，以确保服务的持续可用性。

### [反应](https://news.ycombinator.com/item?id=37274871)

- Slack 采用了蜂窝式架构和筒仓策略，从而提高了性能和可扩展性。
- 该公司还利用多个可用区来确保服务的高可用性。
- Slack 微服务架构的一大挑战是使用不同的编程语言，这可能会导致兼容性问题。
- 用户对 Slack 的性能表示不满，尤其是与 Microsoft Teams 相比。
- 有人建议考虑使用 IRC 等替代品，它们可能提供更好的性能和可靠性。

## [90% 的 "环保 "纸吸管含有微量有毒的永久性化学物质](https://scienceswitch.com/2023/08/27/90-of-eco-friendly-paper-straws-contain-traces-of-toxic-forever-chemicals/)

- 安特卫普大学（University of Antwerp）的一项研究显示，90% 的环保纸吸管含有有害的 "永久化学品"--全氟辛烷磺酸（PFAS）。
- 全氟辛烷磺酸与各种健康问题有关，如高胆固醇、免疫反应减弱、甲状腺疾病和癌症风险增加。
- 研究发现，在 39 种受测吸管品牌中，69% 的吸管含有全氟辛烷磺酸，其中纸质吸管的全氟辛烷磺酸含量最高。纸吸管上的防水涂层很可能是使用 PFAS 的原因之一。
- 这对 "纸吸管自动比塑料吸管更健康、更可持续 "的假设提出了质疑，要求开展进一步研究，以确定纸吸管在通常使用过程中是否会渗出全氟辛烷磺酸。

### [反应](https://news.ycombinator.com/item?id=37275856)

- 辩论的焦点是吸管对环境的影响以及不锈钢吸管或可重复使用的塑料吸管等替代品。
- 人们对环保纸吸管中是否含有有害化学物质以及因其防水涂层而面临的回收难题表示担忧。
- 讨论还强调了塑料污染、不同类型稻草的能源消耗等更广泛的问题，以及需要全面的解决方案来解决这一问题。

## [n8n.io - 功能强大的工作流程自动化工具](https://n8n.io)

- n8n 是一款工作流程自动化工具，旨在让技术人员更轻松高效地创建复杂的自动化程序。
- 它提供友好的用户界面，并可使用 JavaScript 对自动化进行自定义。

- 它将可扩展性和可定制的错误处理放在首位，可免费自行托管，也可在云中使用，价格实惠。
- n8n 社区为用户提供支持和资源，该平台的功能和灵活性备受赞誉。

- 强调了解软件工具的许可和商业模式非常重要。
- 讨论了 n8n.io 的集成能力，用户就如何更好地与其他平台集成提出了改进建议。
- 该主题包括关于 "n8n "发音、与其他工具的比较以及这些工具在工作流程自动化方面的功能和可靠性的讨论。

## [准将 64 上的 Linux](https://github.com/onnokort/semu-c64)

- Semu-c64 是一个允许在 Commodore C64 上使用 semu 仿真器分叉运行 Linux 的项目。
- 它需要一个内存扩展单元，运行速度非常慢，在真正的 C64 上启动 Linux 估计需要一周时间。
- 代码可使用 make 和 mos-c64-clang 生成，模拟器可通过创建 .d64 文件并加载 semu 可执行文件在 VICE EMU 上运行。

### [反应](https://news.ycombinator.com/item?id=37277907)

- 一位开发人员使用 RISC-V 仿真器和 LLVM 移植程序，成功地在 Commodore 64 上运行 Linux。
- 关于编译器的效率和 Commodore 64 处理器的局限性存在争议。
- 有些人对在真实硬件上运行该项目感兴趣，但这需要一个罕见且难找的 RAM 扩展单元。
