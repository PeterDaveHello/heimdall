- 重点介绍了为 Windows 构建和打包基于 Fortran 的软件包的困难。

- Conda 和 conda-forge 社区在应对这些挑战方面发挥着重要作用。
- Python 打包工具（如 distutils 和 setuptools）有其局限性，Meson 被用作构建工具。为 Windows 构建和打包基于 Fortran 的软件包尤其困难。
- 我们成功构建了 Windows 上的 SciPy 定制工具链，在此对贡献者表示感谢。

## [缅因州选民批准《汽车维修权法》，汽车制造商面临挑战](https://www.404media.co/voters-overwhelmingly-pass-car-right-to-repair-law-in-maine/)

- 尽管遭到汽车制造商的反对，缅因州选民还是通过了一项投票措施，赋予消费者维修汽车的权利。
- 该措施要求汽车制造商对诊断系统进行标准化，并允许车主和独立维修机构访问数据。
- 继去年马萨诸塞州通过类似措施后，此次胜利加强了全国范围内的维修权保护运动。

### [反应](https://news.ycombinator.com/item?id=38195509)

- 缅因州选民通过了一项投票措施，赋予车主和独立维修机构从汽车制造商处获取数据和标准化诊断系统的权利。
- 这对一直反对此类立法的汽车制造商来说是一个挫折。
- 这项法律被认为是必要的，因为制造商越来越多地使用无线方式访问汽车诊断系统，这限制了独立维修人员的访问。
