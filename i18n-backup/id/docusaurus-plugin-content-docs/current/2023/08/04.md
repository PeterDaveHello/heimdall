## [Levitasi magnetik tekanan lingkungan suhu ruangan yang berhasil dari LK-99](https://arxiv.org/abs/2308.01516)

- Para peneliti telah mensintesis kristal LK-99 yang mampu melakukan levitasi magnetik pada suhu kamar.

## [AWS akan mulai mengenakan biaya untuk alamat IPv4 publik](https://aws.amazon.com/blogs/aws/new-aws-public-ipv4-address-charge-public-ip-insights/)

- Mulai 1 Februari 2024, AWS akan menerapkan biaya $0,005 per alamat IP per jam untuk alamat IPv4 publik di semua layanan AWS.
- Perubahan ini dimotivasi oleh semakin berkurangnya alamat IPv4 dan dimaksudkan untuk mempromosikan adopsi IPv6.
- AWS juga memperkenalkan Public IP Insights, sebuah fitur gratis yang memungkinkan pengguna untuk memantau dan menganalisis penggunaan alamat IPv4 publik.
- Selama 12 bulan pertama sejak 1 Februari 2024, AWS Free Tier untuk EC2 akan menyertakan 750 jam penggunaan alamat IPv4 publik per bulan.

### [Reaksi](https://news.ycombinator.com/item?id=36989798)

- AWS telah menerapkan biaya untuk alamat IPv4 publik, yang menyebabkan ketidakpuasan di antara para pengguna.
- Para pengguna telah memuji solusi Google Cloud dan menyarankan alternatif lain seperti menggunakan instance NAT atau VM Linux.
- Frustrasi muncul terkait biaya dan desain internal VPC di AWS, serta kekhawatiran tentang kenaikan harga alamat IP Elastis.
