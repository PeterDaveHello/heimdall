- ผลการทดลองและการเปรียบเทียบกับแบบจําลองอื่น ๆ มีไว้เพื่อสนับสนุนสมมติฐานนี้
- นอกจากนี้ยังมีการสํารวจผลกระทบของการค้นพบเหล่านี้กับ GPT-3.5-Turbo ซึ่งอาจมีสถาปัตยกรรมที่คล้ายคลึงกัน
