# 2023-08-31

## [タイピングの半分を自動化した](https://github.com/eschluntz/compress)

- このユーザーは過去1年間、Slackのメッセージを分析し、よく使われるフレーズのキーボードショートカットを生成するツールを活用してきた。
- このツールは、Slackメッセージを解析して、頻繁に繰り返されるフレーズを認識する。
- このツールは、日常的なフレーズを簡単にアクセスできるキーボードショートカットに変換することで、ユーザーに効率性と利便性を提供します。

### [反応](https://news.ycombinator.com/item?id=37326870)

- 自動化ツール、ショートカット、略語などを通じて、タイピングの効率と生産性を高めることに主眼を置いている。
- コーディングの生産性におけるタイピング速度の重要性、仕事と生活の両方における効率とリラックスのバランスを取ることの重要性については、現在も議論が続いている。
- キーボードショートカット、代替キーボードセットアップ、テキスト拡張ツール、ディクテーションシステムなど、さまざまな方法が検討され、それらの潜在的な利点と欠点が議論される。

## [私のカースト](https://www.cs.toronto.edu/~meel/opinions/caste.html)

- この投稿の著者は、インドの社会経済的に不利なグループであるその他後進階級（OBC）の一員である。

## [DIYの「バイオニック膵臓」が糖尿病治療を変える](https://www.nature.com/articles/d41586-023-02648-9)

- 1型糖尿病を管理する個人のコミュニティが、血糖値管理を自動化するDIYオープンソースソフトウェアを開発した。
- 擁護者たちは、こうしたDIY機器へのアクセシビリティを高めるために、オープンソースアルゴリズムのFDA認可を求めているが、相互運用性のためには機器メーカー間の協力が必要である。
- 糖尿病管理におけるこれらのオープンソースAIアルゴリズムの利点の報告は、ユーザーの選択の価値とオープンソースコミュニティ内の進歩を強調している。

### [反応](https://news.ycombinator.com/item?id=37321028)

- DIY（Do-It-Yourself）技術の活用、クローズドループシステム、インスリンポンプ使用に関する課題、モニタリング装置の改良の必要性、食習慣の影響、進歩の先陣を切るDIYコミュニティの影響力など、糖尿病管理のいくつかの側面について議論する。
- さまざまな管理戦略の潜在的な利点と欠点についての考察があり、継続的な研究と糖尿病患者への支援の重要性が強調されている。
- 包括的な焦点は、血糖値をコントロールし、糖尿病に関連するリスクを軽減するための強化された解決策を発見することである。

## [マイクロソフトはWin11でマルウェアのようなポップアップを使い、クロームを捨てさせようとしている](https://www.theverge.com/2023/8/30/23851902/microsoft-bing-popups-windows-11-malware)

- マイクロソフトはウィンドウズ11のユーザーに対し、クロームを使用している場合でも、デフォルトの検索エンジンをグーグルからビングに切り替えるようポップアップメッセージで促している。
- このようなポップアップを押しつけがましいと考えたり、購入したオペレーティング・システム内で広告にさらされるべきではないと考えたりするためだ。
- マイクロソフトは反発を認識し、問題の解決を試みる間、これらの通知を一時的に停止した。

### [反応](https://news.ycombinator.com/item?id=37321002)

- 議論の中心は、ウィンドウズ、特にウィンドウズ11に対するユーザーの不満であり、ポップアップ、ユーザーアカウント、プログラムの可視性などの問題から、マックやリナックスなどの代替案を検討するよう促している。
- システムの選択肢の中で、Linuxは使いやすさと機能性で賞賛されているが、特定のゲームを実行するための欠点が指摘されている。その他、ハードウェアの好み、ソフトウェアの互換性、Windows Subsystem for Linux（WSL）の制限などが議論されている。
- ハイテク製品の中断や広告に対する不満が蔓延しており、プラットフォームの抑制、ウィンドウズの将来、マイクロソフトの行動、潜在的な法的措置や規制措置についての議論がある。GmailやGoogle MapsのiPhoneアプリに対する不満も、グーグル製品に対する潜在的な不満を示している。

## [Windows 11のシステムコンポーネントは、デフォルトのブラウザを使用してヨーロッパのリンクを開きます。](https://blogs.windows.com/windows-insider/2023/08/25/announcing-windows-11-insider-preview-build-23531-dev-channel/)

- マイクロソフトはWindows 11のInsider Preview Build 23531をDev Channelにロールアウトし、セーフモード、ファイルエクスプローラー、HDR背景、タスクマネージャーの修正などのアップデートを満載した。
- スタートメニューとタスクバーの検索機能については、いくつかの既知の問題が残っている。開発者は最新のWindows Insider SDKとNuGetパッケージをダウンロードして、最新の状態を保つことが推奨される。
- 再起動後、Dev Channelは23000シリーズのビルドを受信するようになり、そのためプレリリースビルドの定番機能であるデスクトップの透かしが表示されるようになった。

### [反応](https://news.ycombinator.com/item?id=37320786)

- 議論の中心は、欧州経済領域（EEA）におけるマイクロソフトのウィンドウズ11が、ユーザーの嗜好を無視してエッジ・ブラウザに偏っているという批判である。
- 参加者はまた、アップルなど他のハイテク企業の行動についても議論し、ユーザーの選択とプライバシーを制限する独占的な慣行を防ぐための厳格な規制の必要性を広く示唆した。
- その他のトピックとしては、Edge上でChromeをダウンロードする際の不満、マイクロソフトのユーザーに対する敬意の欠如、Linux Mintのような代替オプションのポジティブな経験などがある。

## [東南アジアで数十万人が人身売買され、オンライン詐欺師として働かされていると国連が発表](https://www.ohchr.org/en/press-releases/2023/08/hundreds-thousands-trafficked-work-online-scammers-se-asia-says-un-report)

- 国連人権事務所の報告書は、組織化された犯罪組織が数十万人の東南アジア人を恋愛投資詐欺、暗号詐欺、違法賭博などのオンライン犯罪行為に強制的に巻き込んでいることを明らかにしている。
- これらの犯罪組織は、脅迫、拷問、性的暴力、強制労働、恣意的な拘留などの人権侵害を被害者に課している。このオンライン詐欺人身売買の規模を見積もるのは、その秘密主義的な性質ゆえに困難である。
- 報告書は、COVID-19の流行中にエスカレートしたこうした詐欺への対応の一環として、被害国に対し、人権を強化し、統治を強化し、法の支配を徹底し、汚職に取り組むよう求めている。

### [反応](https://news.ycombinator.com/item?id=37320975)

- このネット上の話題は、東南アジアにおける人身売買の報告に関するもので、そこでは被害者がネット詐欺に加担させられており、現地の法執行機関や政治ネットワークが関与している可能性がある。
- 議論は既存の刑事司法慣行にも及び、米国における犯罪防止、犯罪者の更生、囚人処遇の制度的改善に焦点が当てられている。
- また、TinderやLinkedInのようなソーシャルメディア・プラットフォームの詐欺行為を軽減する責任についても議論されている。

## [FCC、ISPに各月額料金の記載を義務付ける規則の廃止を拒否](https://arstechnica.com/tech-policy/2023/08/fcc-says-too-bad-to-isps-complaining-that-listing-every-fee-is-too-hard/)

- 米連邦通信委員会（FCC）は、コムキャスト社をはじめとするISP各社に対し、すべての月額料金の開示を義務づける規則を撤廃するよう求めていた訴えを却下した。
- ISPは、この要件は管理上の課題と不当な複雑さをもたらすと主張したが、FCCは消費者がインターネット・サービスを選択する際の透明性の必要性を強調した。
- 法律により、FCCはこれらの規則を施行し、ISPは政府が課すすべての料金をブロードバンドラベルに含める必要がある。

### [反応](https://news.ycombinator.com/item?id=37326806)

- 米連邦通信委員会（FCC）は、インターネット・サービス・プロバイダー（ISP）に対し、透明性を高めプロバイダー間の比較を容易にするため、すべての月額料金を顧客に開示することを義務付ける規則を支持した。
- 支持者は、このルールは隠れた料金を明らかにし、ISP間の競争を刺激し、ISPに説明責任を果たさせるものだと主張している。また、この議論では、地方自治体が課すパススルー料金を税金とみなすべきかどうか、ISPはこれを顧客に転嫁すべきかどうかという問題も提起されている。
- さらにFCCは、ISPは広告ですべての料金や手数料を前もって明らかにし、さらなる透明性を提供しなければならないとの裁定を下した。

## [LGモニターのEDIDをハックする](https://gist.github.com/kj800x/be3001c07c49fdb36970633b0bc6defb)

- この記事は、LG ULTRAGEARモニターがLinuxオペレーティングシステム上で正しく機能しない問題を取り上げています。
- この問題は、拡張ディスプレイ識別データ(EDID)ファイルを調査し、16進エディタを使って修正することで調査される。
- この問題は、モニターそのものではなく、Linuxのグラフィックカード・ドライバーに関連しているようで、同様の問題を解決できる可能性を示唆している。

### [反応](https://news.ycombinator.com/item?id=37323604)

- この記事では、Linux、macOS、Windowsなどのオペレーティングシステムにおけるモニタの互換性と拡張ディスプレイ識別データ（EDID）の操作に関する課題と問題を掘り下げている。
- EDIDオーバーライド、低遅延ドルビービジョン、輝度調整、ピクセルクロックエラー、非ポスティングマシンに関するユーザー体験を取り上げ、EDIDデータの取り扱いの複雑さと限界を指摘している。
- このディスカッションでは、Linuxにおけるハードウェアの互換性の問題や、プラットフォーム上でのトラブルシューティングのニュアンスも浮き彫りにされている。特に、LGモニターのEDIDに関する問題や、1つのデバイス上で異なる規格が共存していることが強調されている。
