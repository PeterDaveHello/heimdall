# 2023-08-17

## [LK-99は超伝導体ではない](https://www.nature.com/articles/d41586-023-02585-7)

- LK-99は、かつては室温超伝導体だと信じられていたが、硫化銅などの不純物により絶縁体であることが判明した。
- この発見は、材料特性の精密な検査に単結晶を用いることの意義を強調している。
- LK-99が超伝導体であることを否定したことは、迅速かつ決定的な成果と考えられる。

### [反応](https://news.ycombinator.com/item?id=37149349)

- 最近の常温超伝導の主張は、失望と懐疑の目で見られている。
- ディスカッションの参加者は、超電導技術の限界と課題について懸念を表明している。
- 科学的主張の評価においては、再現、査読、慎重な評価の重要性が強調される。

## [HtmxはGitHub Acceleratorの一部です。](https://htmx.org/posts/2023-06-06-htmx-github-accelerator/)

- htmxはGitHub Open Source Acceleratorの第1期生に選ばれました。
- このプログラムは、htmx開発者に、成功したオープンソース開発者と協力し、そこから学ぶ機会を提供します。
- htmxでの作業をフルタイムの仕事にすることを目標に、htmxはhtmx 2.0に取り組む予定で、アクセラレータから他のオープンソースプロジェクトを探求することを奨励している。

### [反応](https://news.ycombinator.com/item?id=37144985)

- この対談では、ウェブ開発ライブラリであるHTMXの人気と利点について、そのシンプルさやモバイル機器でのパフォーマンスなどを含めて探ります。
- 参加者は、HTMXと他の技術との統合や、フロントエンド開発のためのドメイン固有言語（DSL）としての利用など、ウェブ開発のパラダイムへの潜在的な影響について議論する。
- 対談では、複雑なアプリケーションにReactを使用する際の課題、ウェブ開発フレームワークの役割、ウェブフレームワークにおけるシンプルさの重要性、HTMXとHATEOAS（Hypermedia as the Engine of Application State）の関係についても触れている。

## [Debian Linux Release (1993) と呼んでいる、真新しい Linux リリース。](https://wiki.debian.org/DebianHistory?action=AttachFile&do=get&target=Debian-announcement-1993.txt)

- Ian A. Murdock はスクラッチから開発された Debian Linux の次期リリースを発表します。
- マードック氏は、Linuxコミュニティから特定のパッケージや機能に関する提案やフィードバックを求めている。
- Debian Linux リリースは、より洗練され、最新で、インストールと保守がより簡単で、メニューシステム、広範な文書、インターネットを利用しないユーザがアップグレードパッケージを受け取り、追加パッケージにアクセスするためのオプションがあることを目指しています。

### [反応](https://news.ycombinator.com/item?id=37147617)

- Debian Linux のリリースとその影響、その創設者である Ian Murdock 氏の不運な死など、さまざまなトピックが議論されています。
- 他の Linux ディストリビューションと比較して Debian を使用する利点と制限について、特定のグラフィックカードのパフォーマンス問題やホームサーバーへの Ubuntu のインストールと共に議論します。
- また、Debianの安定性からDebianの使用を推奨し、ベンゾジアゼピン系薬剤とその効果、サプリメントとしてのGABAの使用についても論じている。さらに、アルコール離脱の治療法としてハロペリドールが挙げられている。

## [どうすれば水を節約できるか？インペリアル郡でのアルファルファ栽培の中止](https://www.desertsun.com/story/opinion/contributors/valley-voice/2023/02/05/growing-alfalfa-in-imperial-county-and-california-wastes-water/69860506007/)

- カリフォルニア州におけるアルファルファの栽培は、大幅な水の浪費につながり、州の水供給危機を悪化させている。
- この慣行は、主にアグリビジネスの大企業に利益をもたらし、水を安く売ることを可能にする。
- 著者は、砂漠でのアルファルファ栽培の持続可能性に疑問を呈し、水供給問題を緩和するために栽培を減らすことを提案している。

### [反応](https://news.ycombinator.com/item?id=37146398)

- カリフォルニア州を中心に、水のミスプライスとその農業への影響について議論する。
- 農業用水の価格設定と利用、砂漠の水利用の難しさ、水利権の不一致、水資源管理、水価格改革の潜在的な成果などのトピックが検討された。
- 解決策として提案されているのは、農家への補償、水利権の変更、より持続可能な農業の推進などである。また、水不足、非効率的な水使用、農業が環境に与える影響などに関する懸念も議論されている。

## [今日、Windowsタスクマネージャーを一時停止してアプリを移動できることを知った。](https://www.theverge.com/2023/8/16/23834125/microsoft-windows-task-manager-pause-shortcut)

- CTRLキーを押し続けることで、Windowsタスクマネージャーを一時停止し、アプリが動き回るのを防ぐことができる。
- このショートカットは、ウィンドウズ10、ウィンドウズ11、そして潜在的に古いバージョンのOSで使用できる。
- マイクロソフトはウィンドウズ11用の強制終了オプションを開発中だが、CTRLキーのショートカットはまだアップグレードしていないユーザーにとっては貴重なものだ。

### [反応](https://news.ycombinator.com/item?id=37146268)

- ユーザーはWindowsタスクマネージャーや他のWindowsオペレーティングシステムのデザインやユーザーインターフェイスに不満を表明している。
- 明確な指示の欠如と機能の発見しやすさが、不満の主な点である。
- ユーザーはウィンドウズと他のオペレーティング・システムを比較し、別のタスク・マネージャー・オプションを提案する。

## [ランダムなデータに基づいてシステムクロックをリセットするWindowsの機能が大混乱を引き起こしている](https://arstechnica.com/security/2023/08/windows-feature-that-resets-system-clocks-based-on-random-data-is-wreaking-havoc/)

- ウィンドウズのセキュア・タイム・シーディング機能が、時計を誤った時刻にリセットすることで重大な混乱を引き起こし、携帯電話番号のルーティング・テーブルのような重要なシステムに影響を与えている。
- この機能は正確なシステムクロックを維持するためのものだが、2016年以降に報告されたタイムリセットの根本的な原因であると指摘されている。
- エンジニアや管理者は、このあまり知られていない機能によって引き起こされる問題に苛立ち、不便やサービスの利用不能を招いてきた。

### [反応](https://news.ycombinator.com/item?id=37151220)

- ウィンドウズの時刻同期は、潜在的なセキュリティの脆弱性、悪用やなりすましの危険性から精査されている。
- ユーザーはシステムの機能性に不満を表明しており、信頼できるサーバーの使用、固定署名、代替の計時メカニズムなどの改善を求めている。
- ウィンドウズでの時刻同期の正確性、信頼性、セキュリティに対する懸念から、Linuxへの乗り換えを選択したユーザーもいる。

## [Ts_zip大規模言語モデルを用いたテキスト圧縮](https://bellard.org/ts_server/ts_zip.html)

- ts_serverソフトウェアの一部であるts_zipユーティリティは、Large Language Modelsを使用したテキスト圧縮を可能にし、他のツールよりも高い圧縮率を提供します。
- 様々なサイズのテキストファイルは、圧縮率が異なり、特定のモデルの圧縮速度とメモリ要件の詳細が表示されます。
- RNN構造と高速性能により、テキスト圧縮には小型のRWKVモデルが推奨される。

### [反応](https://news.ycombinator.com/item?id=37152978)

- Ts_zipは大規模言語モデル（LLM）を利用したテキスト圧縮手法で、従来の圧縮手法を凌駕する可能性を秘めている。
- LLMベースの圧縮アルゴリズムの有効性について議論し、新しいデータに対する偏りのないテストの必要性を強調する。
- LLMを圧縮サイズに含めることの課題と、広く使用することの非現実性を探る。
